# Workshop Notebooks

This section contains notebooks for hands-on workshop sessions. They are not listed in the main navigation but can be accessed via direct URL.

All notebooks support execution via the embedded code runner (thebelab), Binder, or Code Engine — the same environment used for tutorials.

To add a workshop notebook, place an `.mdx` file in the `docs/workshop/` directory.
