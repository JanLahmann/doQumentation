# סביבות מעבדה מקוונות {#online-lab-environments}

בנוסף להורדה והתקנה של Qiskit לעבוד על המחשב המקומי שלך, קיימות מספר אפשרויות למי שמעדיף לעבוד בסביבת פיתוח מקוונת.

מספר ספקים מציעים סביבות Jupyter מקוונות עם Qiskit מותקן מראש לקבלה מהירה ונוחה. ספקים מומלצים אלה כוללים את ספק שירותי הענן OVHcloud ואת פלטפורמת הענן לפיתוח תוכנה קוונטית qBraid.

## סביבות Notebook מומלצות עם Qiskit מותקן מראש {#recommended-notebook-environments-with-qiskit-preinstalled}

ניתן להשתמש בסביבות Jupyter notebook מקוונות נוספות כגון IBM Watson&reg; Studio, Google Colab ו-Microsoft Azure Machine Learning Studio לעבודה עם Qiskit על ידי ביצוע ההוראות המתאימות.

### Notebooks של OVHcloud AI {#ovhcloud-ai-notebooks}

משתמשי OVHcloud Public Cloud ו-OVHcloud Startup יכולים להשתמש ב-notebooks הקוונטיים של OVHcloud כדי לגשת ל-Qiskit ולהגיש משרות ל-QPUs של IBM&reg; עם חשבון IBM Quantum&reg; או IBM Cloud&reg; שלהם. למידע נוסף, בקר בדף [תיעוד ומדריכים של OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Notebooks של OVHcloud AI אינם זמינים בחלק מהאזורים, כגון ארצות הברית. בנוסף, ממשקי המשתמש עשויים להיראות שונים באזורים שונים.
</Admonition>

בצע את הצעדים הבאים כדי להתחיל:

1. [צור חשבון IBM Cloud.](./cloud-setup)
1. [הגדר חשבון OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [צור פרויקט OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. בצע את הצעדים ב-[הפעל את ה-AI Notebook הראשון שלך,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) ובחר **Qiskit** כמסגרת העבודה.

### qBraid Lab {#qbraid-lab}

משתמשי Qiskit יכולים להשתמש בסביבות Python המוגדרות מראש של qBraid על ידי ביצוע ההוראות הבאות. למידע נוסף, ראה את [תיעוד qBraid.](https://docs.qbraid.com/home/introduction)

1. [הגדר חשבון qBraid חינמי.](https://docs.qbraid.com/home/account)
3. [צור סביבת Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. הפעל את qBraid Lab, לחץ על סמל **ENVS** מצד ימין, לחץ על **+ ADD** בראש סרגל הצד החדש, וחפש את Qiskit. לחץ על הגרסה שברצונך להשתמש בה, ואז לחץ על **Install**.
4. צור Jupyter notebook על ידי לחיצה על **Python 3 [Qiskit]** בחלק ה-Notebook.
4. ניתן להתקין חבילות Python נוספות (כגון Qiskit v1.2) בתא notebook על ידי הרצת `%pip install [package_name].` למידע נוסף, ראה את הנושא [פקודות Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## סביבות Notebook נוספות למשתמשי Qiskit {#additional-notebook-environments-for-qiskit-users}

עבור פלטפורמות שאין בהן Qiskit מוגדר מראש, הוראות ההתקנה דומות, למעט שצריך להתקין את Qiskit באופן ידני. תיצור חשבון, תפתח פרויקט חדש (או notebook או workspace, בהתאם לפלטפורמה), ותתקין את Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [הגדר ניסיון חינמי של Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [צור פרויקט Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. כדי להתקין את חבילות Qiskit והתלויות הנדרשות, פתח notebook חדש והרץ את הפקודה הבאה:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [צור חשבון Google.](https://support.google.com/accounts/answer/27441?hl)
2. עבור לדף הנחיתה של [Google Colab](https://colab.research.google.com/) והיכנס עם חשבון Google שלך.
3. [צור Colab notebook](https://colab.research.google.com/#create=true) והתקן את Qiskit על ידי הרצת הפקודה הבאה:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [צור חשבון Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [צור workspace.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [צור Jupyter notebook ב-workspace שלך.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. מתוך ה-notebook, הרץ את הפקודה הזו כדי להתקין את Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
