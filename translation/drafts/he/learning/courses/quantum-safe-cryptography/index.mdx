# סקירה כללית {#overview}

!["תמונת גיבור"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

מבוא מעשי לקריפטוגרפיה עמידה בפני קוונטום הוא קורס מקוון חינמי ב-IBM Quantum&reg; Learning המשמש כמבוא למושגי היסוד בקריפטוגרפיה עמידה בפני קוונטום.

הקורס נוצר עבור מפתחים המעוניינים לשדרג את אבטחת היישומים שלהם, ומציע שיעורים רב-ממדיים ודוגמאות קוד חי ואינטראקטיבי בארבעה תחומי ליבה של קריפטוגרפיה: פונקציות גיבוב קריפטוגרפיות, קריפטוגרפיה בעלת מפתח סימטרי, קריפטוגרפיה בעלת מפתח אסימטרי, וקריפטוגרפיה עמידה בפני קוונטום.

בסיום הקורס, המפתחים ילמדו כיצד נוף סיכוני הסייבר מתפתח, ויחקרו גישות עכשוויות העומדות לעצב את עידן הקוונטום.
