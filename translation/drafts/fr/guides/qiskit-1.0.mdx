# Guide de migration Qiskit 1.0 {#qiskit-10-migration-guide}

Qiskit 1.0 apporte de nouvelles garanties de stabilité et des améliorations de performances, mais il rompt la compatibilité avec toutes les versions précédentes de Qiskit. Consulte les guides suivants pour mettre à jour ton environnement et ton code afin de les faire fonctionner avec Qiskit v1.0.

- [Installer le nouveau paquet Qiskit v1.0](/guides/qiskit-1.0-installation) pour mettre à niveau les environnements Python et les dépendances de projet.
- [Changements de fonctionnalités de Qiskit v1.0](/guides/qiskit-1.0-features) pour mettre à jour ton code.
- [Modifications incompatibles du paquet Qiskit v1.0](/guides/metapackage-migration) pour comprendre la nouvelle et l'ancienne structure de paquets.

Pour en savoir plus sur les avantages de Qiskit v1.0, consulte le [blog IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
