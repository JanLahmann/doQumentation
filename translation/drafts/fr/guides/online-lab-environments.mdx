# Environnements de lab en ligne

En plus de télécharger et d'installer Qiskit pour travailler sur ta machine locale, plusieurs options sont disponibles pour ceux qui préfèrent travailler dans un environnement de développement en ligne.

Plusieurs fournisseurs proposent des environnements Jupyter en ligne avec Qiskit préinstallé pour une prise en main rapide et pratique. Ces fournisseurs recommandés incluent le fournisseur de services cloud OVHcloud et la plateforme cloud de développement logiciel quantique qBraid.

## Environnements de notebook recommandés avec Qiskit préinstallé {#recommended-notebook-environments-with-qiskit-preinstalled}

D'autres environnements de notebooks Jupyter en ligne comme IBM Watson&reg; Studio, Google Colab et Microsoft Azure Machine Learning Studio peuvent être utilisés pour travailler avec Qiskit en suivant les instructions appropriées.

### Notebooks AI OVHcloud {#ovhcloud-ai-notebooks}

Les utilisateurs d'OVHcloud Public Cloud et d'OVHcloud Startup peuvent utiliser leurs notebooks quantiques OVHcloud pour accéder à Qiskit et soumettre des jobs aux QPUs IBM&reg; avec leur compte IBM Quantum&reg; ou IBM Cloud&reg;. Pour en savoir plus, consulte la page [Documentation et tutoriels OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Les notebooks AI OVHcloud ne sont pas disponibles dans certaines régions, comme les États-Unis. De plus, les interfaces utilisateur peuvent différer selon les régions.
</Admonition>

Suis ces étapes pour commencer :

1. [Crée un compte IBM Cloud.](./cloud-setup)
1. [Configure un compte OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Crée un projet OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Suis les étapes de [Lancement de ton premier notebook AI,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) en sélectionnant **Qiskit** comme framework.

### qBraid Lab {#qbraid-lab}

Les utilisateurs de Qiskit peuvent utiliser les environnements Python préconfigurés de qBraid en suivant ces instructions. Pour plus d'informations, consulte la [documentation qBraid.](https://docs.qbraid.com/home/introduction)

1. [Crée un compte qBraid gratuit.](https://docs.qbraid.com/home/account)
3. [Crée un environnement Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Lance qBraid Lab, clique sur l'icône **ENVS** à droite, clique sur **+ ADD** en haut de la nouvelle barre latérale, et recherche Qiskit. Clique sur la version que tu veux utiliser, puis clique sur **Install**.
4. Crée un notebook Jupyter en cliquant sur **Python 3 [Qiskit]** dans la section Notebook.
4. Tu peux installer des packages Python supplémentaires (comme Qiskit v1.2) dans une cellule de notebook en exécutant `%pip install [package_name].` Pour plus d'informations, consulte la rubrique [Commandes Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Autres environnements de notebooks pour les utilisateurs de Qiskit {#additional-notebook-environments-for-qiskit-users}

Pour les plateformes sans Qiskit préconfiguré, les instructions de configuration sont similaires, sauf que tu dois installer Qiskit manuellement. Tu devras créer un compte, démarrer un nouveau projet (ou notebook ou espace de travail, selon la plateforme), et installer Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Configure l'essai gratuit de Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Crée un projet Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Pour installer les packages Qiskit et les dépendances nécessaires, ouvre un nouveau notebook et exécute la commande suivante :
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Crée un compte Google.](https://support.google.com/accounts/answer/27441?hl)
2. Accède à la [page d'accueil de Google Colab](https://colab.research.google.com/) et connecte-toi avec ton compte Google.
3. [Crée un notebook Colab](https://colab.research.google.com/#create=true) et installe Qiskit en exécutant la commande suivante :
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Crée un compte Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Crée un espace de travail.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Crée un notebook Jupyter dans ton espace de travail.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Depuis le notebook, exécute cette commande pour installer Qiskit :
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
