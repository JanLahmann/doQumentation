# Vue d'ensemble {#overview}

!["Hero image"](/learning/images/modules/quantum-mechanics/hero.avif)

Cette collection de modules Qiskit en classe est conçue pour s'intégrer dans un cours de mécanique quantique de niveau licence. L'objectif est d'aider les enseignants et les étudiants à tirer parti des ordinateurs quantiques pour explorer expérimentalement des sujets essentiels à un cours de mécanique quantique, tels que l'incertitude et la superposition.
Chaque module doit être commencé en classe avec un soutien de groupe et de l'instructeur, mais peut être complété de façon indépendante ou en petits groupes. Toutes les activités peuvent être réalisées en bien moins de temps que celui alloué gratuitement dans le cadre du Plan Open disponible pour tous les utilisateurs. Des options de simulateur sont également intégrées, au cas où des étudiants épuiseraient accidentellement leur temps de calcul quantique gratuit.

Les enseignants peuvent demander des versions de ces notebooks avec des corrigés et des conseils sur leur placement dans les programmes courants en remplissant ce [rapide sondage](https://ibm.biz/classrooms_instructor_key_request) sur la façon dont les notebooks sont utilisés.

## Prérequis recommandés {#recommended-background}

Chaque module contient des recommandations sur l'endroit dans un programme où chaque module pourrait être le plus efficace. En général, les étudiants tireront le meilleur parti de ces modules s'ils disposent de :

- Quelques notions d'algèbre linéaire (les matrices $2 \times 2$ devraient suffire)
- Une certaine familiarité avec Python, bien que la plupart des notebooks puissent simplement être exécutés tels quels
