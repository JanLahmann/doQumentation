## استرداد نتائج المهام من IBM Quantum {#retrieve-job-results-from-ibm-quantum}

يحتفظ IBM Quantum تلقائيًا بنتائج كل مهمة لتتمكن من استردادها لاحقًا. استخدم هذه الميزة لمتابعة برامجك الكمومية عبر إعادات تشغيل النواة ومراجعة النتائج السابقة. يمكنك الحصول على معرّف المهمة برمجيًا عبر طريقة `job_id`، أو يمكنك الاطلاع على جميع مهامك المُرسَلة ومعرّفاتها في [صفحة Workloads](https://quantum.cloud.ibm.com/workloads).

للبحث عن مهمة برمجيًا، استخدم طريقة [`QiskitRuntimeService.jobs`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#jobs). تُعيد هذه الطريقة افتراضيًا أحدث المهام المُرسَلة، لكن يمكنك أيضًا تصفيتها حسب اسم الجهاز، وتاريخ الإنشاء، وغير ذلك. تجد في الخلية التالية كيفية البحث عن أي مهام أُرسلت خلال الثلاثة أشهر الماضية. يجب أن تكون قيمة المعامل `created_after` كائنًا من نوع [`datetime.datetime`](https://docs.python.org/3.8/library/datetime.html#datetime.datetime).

```python
# Added by doQumentation — required packages for this notebook
```

```python
import datetime
from qiskit_ibm_runtime import QiskitRuntimeService

three_months_ago = datetime.datetime.now() - datetime.timedelta(days=90)

service = QiskitRuntimeService()
jobs_in_last_three_months = service.jobs(created_after=three_months_ago)
jobs_in_last_three_months[:3]  # show first three jobs
```

يمكنك أيضًا التصفية حسب الجهاز، وحالة المهمة، والجلسة، وغيرها. لمزيد من المعلومات، راجع [`QiskitRuntimeService.jobs`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#jobs) في توثيق API.

بمجرد حصولك على معرّف المهمة، استخدم طريقة [`QiskitRuntimeService.job`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#job) لاستردادها.

```python
# Get ID of most recent successful job for demonstration.
# This will not work if you've never successfully run a job.
successful_job = next(
    j for j in service.jobs(limit=1000) if j.status() == "DONE"
)
job_id = successful_job.job_id()
print(job_id)
```

## حفظ النتائج على القرص {#save-results-to-disk}

قد ترغب أيضًا في حفظ النتائج على القرص. لفعل ذلك، استخدم مكتبة JSON المدمجة في Python مع المُشفِّرات المتوفرة في Qiskit Runtime.

```python
import json
from qiskit_ibm_runtime import RuntimeEncoder

with open("result.json", "w") as file:
    json.dump(retrieved_job.result(), file, cls=RuntimeEncoder)
```

يمكنك بعدها تحميل هذه المصفوفة من القرص في نواة منفصلة.

```python
from qiskit_ibm_runtime import RuntimeDecoder

with open("result.json", "r") as file:
    result = json.load(file, cls=RuntimeDecoder)

result
```
