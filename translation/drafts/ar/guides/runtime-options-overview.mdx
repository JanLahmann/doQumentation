# مقدمة إلى الخيارات {#introduction-to-options}

يمكنك تمرير خيارات إلى الأدوات البدائية لتخصيصها وفق احتياجاتك. يركّز هذا القسم على خيارات أدوات Qiskit Runtime البدائية. على الرغم من أن واجهة طريقة `run()‎` مشتركة بين جميع التطبيقات، إلا أن خياراتها تختلف. راجع مراجع API المقابلة للاطلاع على معلومات حول خيارات [`qiskit.primitives`](https://docs.quantum.ibm.com/api/qiskit/primitives#primitives) و[`qiskit_aer.primitives`](https://qiskit.github.io/qiskit-aer/apidocs/aer_primitives.html).

## نظرة عامة {#overview}

<span id="options-structure"></span>
### البنية {#structure}

عند استدعاء الأدوات البدائية، يمكنك تمرير الخيارات إما عبر فئة خيارات أو قاموس (dictionary). الخيارات الأكثر استخداماً، مثل `resilience_level`، تكون في المستوى الأول. أما الخيارات الأخرى فتُجمَّع في فئات مختلفة، كفئة `execution`. راجع قسم [تعيين خيارات الأداة البدائية](/guides/specify-runtime-options#pass-options) للاطلاع على التفاصيل الكاملة.

<span id="options-defaults"></span>
### القيم الافتراضية {#defaults}

إن لم تحدّد قيمةً لأحد الخيارات، يُعطى قيمة خاصة هي `Unset` وتُستخدم القيمة الافتراضية للخادم. وبهذا تظل القيمة الافتراضية ثابتة بصرف النظر عن إصدار الكود الذي تستخدمه.

تسرد جداول قسم [ملخص فئات الخيارات](#options-classes) القيم الافتراضية.

<span id="options-precedence"></span>
## تعيين الخيارات {#set-options}

يمكن تعريف الخيارات قبل إنشاء الأداة البدائية وتمريرها إليها، وستقوم الأداة بإنشاء نسخة منها. يمكن إجراء ذلك إما كقاموس متداخل أو باستخدام فئات الخيارات. علاوةً على ذلك، يمكن تغيير خيارات الأداة البدائية بعد إنشائها. استخدم الأسلوب الأنسب لتطبيقك. راجع [تحديد الخيارات](specify-runtime-options) للتفاصيل الكاملة.

## توافق الميزات {#feature-compatibility}

نظرًا للاختلافات في عملية تصريف الجهاز، لا يمكن استخدام بعض ميزات Runtime معًا في مهمة واحدة. انقر على التبويب المناسب للاطلاع على قائمة الميزات غير المتوافقة مع الميزة المحددة:

<Tabs>
  <TabItem value="Dynamic" label="الدوائر الديناميكية">
  غير متوافقة مع:
  - Gate-folding ZNE
  - PEA
  - PEC
  - الفصل الديناميكي

  ملاحظات أخرى:
  - يمكن استخدامها مع Gate twirling للبوابات غير الشرطية.
  - متوافقة مع الأبواب الكسرية عند استخدام `qiskit-ibm-runtime` v0.42.0 أو أحدث.

  </TabItem>
  <TabItem value="DD" label="الفصل الديناميكي">
  غير متوافق مع الدوائر الديناميكية.

  </TabItem>

  <TabItem value="Fractional" label="الأبواب الكسرية">
  غير متوافقة مع:
  - Gate twirling
  - PEA
  - PEC

  متوافقة مع الدوائر الديناميكية عند استخدام `qiskit-ibm-runtime` v0.42.0 أو أحدث.

  </TabItem>

  <TabItem value="ZNE" label="Gate-folding ZNE">
    غير متوافق مع:
  - الدوائر الديناميكية
  - PEA
  - PEC

  قد لا يعمل عند استخدام بوابات مخصصة.
  </TabItem>
  <TabItem value="Twirling" label="Gate twirling">
  غير متوافق مع الأبواب الكسرية أو مع الامتدادات.

  ملاحظات أخرى:
  - يمكن استخدامه مع الدوائر الديناميكية مع البوابات غير الشرطية.
  - لا يعمل مع المشابكات غير Clifford.

  </TabItem>

  <TabItem value="PEA" label="PEA">
    غير متوافق مع:
  - الدوائر الديناميكية
  - الأبواب الكسرية
  - Gate-folding ZNE
  - PEC
  </TabItem>

  <TabItem value="PEC" label="PEC">
    غير متوافق مع:
  - الدوائر الديناميكية
  - الأبواب الكسرية
  - Gate-folding ZNE
  - PEA
  </TabItem>
