في هذا الدرس التطبيقي، نُنفّذ [نمط Qiskit](https://quantum.cloud.ibm.com/docs/guides/intro-to-patterns) الذي يوضّح كيفية المعالجة اللاحقة للعيّنات الكمومية الضوضائية للوصول إلى تقريب لحالة الطاقة الأدنى لهاميلتونية شبكة فيرميونية تُعرف بنموذج أندرسون أحادي الشائبة. سنتبع نهج القطري الكمومي القائم على العيّنات (SQD) لمعالجة عيّنات مأخوذة من مجموعة من حالات فضاء كريلوف الأساسية ذات ``16`` Qubit على فترات زمنية متزايدة. تتحقق هذه الحالات على الجهاز الكمومي باستخدام تقنية تروتر للتطور الزمني. ولمراعاة أثر الضوضاء الكمومية، تُستخدم تقنية استرداد التهيئة. بافتراض حالة ابتدائية جيدة وتناثر حالة الطاقة الأدنى، [ثبت أن هذا النهج يتقارب بكفاءة](https://arxiv.org/abs/2501.09702).

يمكن وصف النمط في أربع خطوات:

1. **الخطوة 1: التخطيط للمسألة الكمومية**
    - توليد مجموعة من حالات فضاء كريلوف الأساسية (أي دوائر التطور الزمني بتقريب تروتر) على فترات زمنية متزايدة لتقدير حالة الطاقة الأدنى
2. **الخطوة 2: تحسين المسألة**
    - تحويل الدوائر الكمومية للعمل على الجهاز المستهدف
3. **الخطوة 3: تنفيذ التجارب**
    - أخذ العيّنات من الدوائر باستخدام أداة ``Sampler``
4. **الخطوة 4: المعالجة اللاحقة للنتائج**
   - حلقة استرداد التهيئة الذاتية التوافق
       - المعالجة اللاحقة لكامل مجموعة عيّنات السلاسل الثنائية، مستعينًا بالمعرفة المسبقة بعدد الجسيمات ومتوسط إشغال المدارات المحسوب في آخر تكرار
       - إنشاء دُفعات فرعية من السلاسل الثنائية المُستردة باحتمالية
       - إسقاط هاميلتونية الشبكة الفيرميونية وقطريتها على كل فضاء جزئي مُختار
       - حفظ أدنى طاقة للحالة الأساسية عبر كل الدُفعات وتحديث متوسط إشغال المدارات

### الخطوة 1: تخطيط المسألة إلى دائرة كمومية {#step-1-map-problem-to-a-quantum-circuit}

أولًا، سنُنشئ هاميلتونيتَي الجسم الواحد والجسمَين اللتين تصفان نموذج أندرسون أحادي الشائبة أحادي البُعد (SIAM) مع ``7`` مواقع حمام (``8`` إلكترونات في ``8`` مدارات). يُستخدم هذا النموذج لوصف الشوائب المغناطيسية المدمجة في المعادن.

ثم سنُنشئ دوائر تروتر ذات ``16`` Qubit المستخدمة لتوليد فضاء كريلوف الكمومي الجزئي.

```python
import numpy as np

n_bath = 7  # number of bath sites

V = 1  # hybridization amplitude
t = 1  # bath hopping amplitude
U = 10  # Impurity onsite repulsion
eps = -U / 2  # Chemical potential for the impurity

# Place the impurity on the first qubit
impurity_index = 0

# One body matrix elements in the "position" basis
h1e = -t * np.diag(np.ones(n_bath), k=1) - t * np.diag(np.ones(n_bath), k=-1)
h1e[impurity_index, impurity_index + 1] = -V
h1e[impurity_index + 1, impurity_index] = -V
h1e[impurity_index, impurity_index] = eps

# Two body matrix elements in the "position" basis
h2e = np.zeros((n_bath + 1, n_bath + 1, n_bath + 1, n_bath + 1))
h2e[impurity_index, impurity_index, impurity_index, impurity_index] = U
```

بعد ذلك، سنُولّد فضاء كريلوف الكمومي الجزئي بمجموعة من الدوائر الكمومية ذات تقريب تروتر. هنا نُنشئ دوال مساعدة لتوليد الحالة الابتدائية (المرجعية) والتطور الزمني للأجزاء أحادية الجسم وثنائية الجسم من الهاميلتونية. للاطلاع على وصف أكثر تفصيلًا لهذا النموذج وكيفية تصميم الدوائر، يُرجى الرجوع إلى [الورقة البحثية](https://arxiv.org/abs/2501.09702).

```python
import ffsim
import scipy
from qiskit import QuantumCircuit, QuantumRegister
from qiskit.circuit.library import CPhaseGate, XGate, XXPlusYYGate

n_modes = n_bath + 1
nelec = (n_modes // 2, n_modes // 2)

dt = 0.2
Utar = scipy.linalg.expm(-1j * dt * h1e)

# The reference state
def initial_state(q_circuit, norb, nocc):
    """Prepare an initial state."""
    for i in range(nocc):
        q_circuit.append(XGate(), [i])
        q_circuit.append(XGate(), [norb + i])
    rot = XXPlusYYGate(np.pi / 2, -np.pi / 2)

    for i in range(3):
        for j in range(nocc - i - 1, nocc + i, 2):
            q_circuit.append(rot, [j, j + 1])
            q_circuit.append(rot, [norb + j, norb + j + 1])
    q_circuit.append(rot, [j + 1, j + 2])
    q_circuit.append(rot, [norb + j + 1, norb + j + 2])

# The one-body time evolution
free_fermion_evolution = ffsim.qiskit.OrbitalRotationJW(n_modes, Utar)

# The two-body time evolution
def append_diagonal_evolution(dt, U, impurity_qubit, num_orb, q_circuit):
    """Append two-body time evolution to a quantum circuit."""
    if U != 0:
        q_circuit.append(
            CPhaseGate(-dt / 2 * U),
            [impurity_qubit, impurity_qubit + num_orb],
        )
```

أنشئ ``d`` حالة متطوّرة زمنيًا تُحدّد فضاء كريلوف الكمومي الجزئي. اخترنا هنا ``d=8``. يتقارب الخطأ الناتج عن أخذ العيّنات من حالات فضاء كريلوف مع زيادة ``d``. لاحظ أن خصائص هذه المسألة تتيح تصريفًا فعّالًا للتطور أحادي الجسم باستخدام `OrbitalRotationJW`؛ غير أنه في الحالة العامة، يمكن استخدام [PauliEvolutionGate](https://quantum.cloud.ibm.com/docs/api/qiskit/qiskit.circuit.library.PauliEvolutionGate) من Qiskit لتطبيق التطور الزمني بتقريب تروتر للهاميلتونية الكاملة.

```python
# Generate the initial state
qubits = QuantumRegister(2 * n_modes, name="q")
init_state = QuantumCircuit(qubits)
initial_state(init_state, n_modes, n_modes // 2)
init_state.draw("mpl", scale=0.4, fold=-1)

d = 8  # Number of Krylov basis states
circuits = []
for i in range(d):
    circ = init_state.copy()
    circuits.append(circ)
    for _ in range(i):
        append_diagonal_evolution(dt, U, impurity_index, n_modes, circ)
        circ.append(free_fermion_evolution, qubits)
        append_diagonal_evolution(dt, U, impurity_index, n_modes, circ)
    circ.measure_all()
```

### الخطوة 2: تحسين المسألة {#step-2-optimize-the-problem}

بعد إنشاء الدوائر ذات تقريب تروتر، سنُحسّنها لعتاد مستهدف. نحتاج أولًا لاختيار جهاز العتاد قبل التحسين. سنستخدم Backend وهميًا بـ 127 Qubit من ``qiskit_ibm_runtime`` لمحاكاة جهاز حقيقي. للتشغيل على وحدة معالجة كمومية حقيقية، ما عليك سوى استبدال الـ Backend الوهمي بـ Backend حقيقي. اطّلع على [مستندات Qiskit IBM Runtime](https://quantum.cloud.ibm.com/docs/guides/get-started-with-primitives#get-started-with-sampler) لمزيد من المعلومات.

```python
from qiskit_ibm_runtime.fake_provider.backends import FakeSherbrooke

backend = FakeSherbrooke()
```

بعد ذلك، سنحوّل الدوائر إلى الـ Backend المستهدف باستخدام Qiskit.

```python
from qiskit.transpiler import generate_preset_pass_manager

# The circuit needs to be transpiled to the AerSimulator target
pass_manager = generate_preset_pass_manager(optimization_level=3, backend=backend)
isa_circuits = pass_manager.run(circuits)
```

### الخطوة 3: تنفيذ التجارب {#step-3-execute-experiments}

بعد تحسين الدوائر لتنفيذها على العتاد، أصبحنا جاهزين لتشغيلها على العتاد المستهدف وجمع العيّنات لتقدير طاقة الحالة الأدنى. نستخدم هنا ``SamplerV2`` من ``qiskit-ibm-runtime`` لمحاكاة عيّنات ضوضائية مأخوذة من الـ Backend `ibm_sherbrooke`. ثم نجمع الإحصاءات من كل حالة أساسية في قاموس إحصاءات موحّد ونرسم أكثر 20 سلسلة ثنائية شيوعًا في أخذ العيّنات.

***ملاحظة: [Qiskit Aer](https://qiskit.github.io/qiskit-aer/index.html) مطلوب لمحاكاة العيّنات من الدوائر المحوَّلة.***

```python
from qiskit.primitives import BitArray
from qiskit.visualization import plot_histogram
from qiskit_ibm_runtime import SamplerV2 as Sampler

# Sample from the circuits
noisy_sampler = Sampler(backend, options={"simulator": {"seed_simulator": 24}})
job = noisy_sampler.run(isa_circuits, shots=500)

# Combine the counts from the individual Trotter circuits
bit_array = BitArray.concatenate_shots([result.data.meas for result in job.result()])

plot_histogram(bit_array.get_counts(), number_to_keep=20)
```

### الخطوة 4: المعالجة اللاحقة للنتائج {#step-4-post-process-the-results}

الآن، نُشغّل خوارزمية SQD باستخدام الدالة `diagonalize_fermionic_hamiltonian`. راجع [توثيق API](../apidocs/qiskit_addon_sqd.fermion.rst#qiskit_addon_sqd.fermion.diagonalize_fermionic_hamiltonian) لشرح وسيطات هذه الدالة.

```python
from qiskit_addon_sqd.fermion import SCIResult, diagonalize_fermionic_hamiltonian

# List to capture intermediate results
result_history = []

def callback(results: list[SCIResult]):
    result_history.append(results)
    iteration = len(result_history)
    print(f"Iteration {iteration}")
    for i, result in enumerate(results):
        print(f"\tSubsample {i}")
        print(f"\t\tEnergy: {result.energy}")
        print(f"\t\tSubspace dimension: {np.prod(result.sci_state.amplitudes.shape)}")

rng = np.random.default_rng(24)
result = diagonalize_fermionic_hamiltonian(
    h1e,
    h2e,
    bit_array,
    samples_per_batch=300,
    norb=n_modes,
    nelec=nelec,
    num_batches=3,
    max_iterations=10,
    symmetrize_spin=True,
    callback=callback,
    seed=rng,
)
```

الآن، نرسم النتائج. يوضّح الرسم الأول أنه بعد بضعة تكرارات، نحصل على طاقة الحالة الأدنى الدقيقة.

هذا المثال صغير بما يكفي لاستكشاف فضاء هيلبرت الكامل، كما يتضح من رسائل الطباعة أعلاه. تذكّر أن فضاء هيلبرت الكامل له بُعد `(num_orbitals choose nelec_a) * (num_orbitals choose nelec_b)`. إذن لهذه المسألة: `(8 choose 4)**2 = 4900`. تزداد أبعاد الفضاء الجزئي بسبب تحسين استرداد التهيئة، وأيضًا لأن خوارزمية SQD تنقل التهيئات المهمة من تكرار إلى آخر. بحلول آخر تكرار، نُجري القطرية على فضاء هيلبرت الكامل.

يوضّح الرسم الثاني متوسط إشغال كل مدار مكاني عبر حلول كل الدُفعات. نرى أنه باحتمالية عالية يحتوي كل مدار على إلكترون واحد.

```python
import matplotlib.pyplot as plt
```
