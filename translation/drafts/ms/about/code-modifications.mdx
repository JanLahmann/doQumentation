# Pengubahsuaian Kod Automatik

doQumentation secara automatik buat sikit pengubahsuaian pada kandungan tutorial
dan panduan Qiskit dari sumber asal supaya pengalaman interaktif jadi lebih
lancar. Halaman ni dokumentasikan setiap pengubahsuaian supaya korang boleh
faham dengan tepat apa yang dah berubah berbanding dokumentasi IBM Quantum asal.

## Salinan notebook (Buka dalam Colab / Binder / Code Engine) {#notebook-copies-open-in-colab-binder-code-engine}

Bila korang klik **Buka dalam Colab**, **Buka dalam JupyterLab**, atau **Buka
dalam Code Engine**, korang akan dapat salinan notebook asal dengan tambahan
berikut:

### 1. Sel notis persediaan (markdown) {#1-setup-notice-cell-markdown}

Sel blockquote disisipkan di bahagian paling atas untuk menerangkan bahawa
doQumentation dah tambah sel persediaan automatik. Ia juga ada pautan balik ke
halaman ni.

### 2. Sel prasyarat (kod) {#2-prerequisites-cell-code}

Sel kod disisipkan selepas notis yang:

- **Pasang pakej yang diperlukan** (`qiskit`, `qiskit-aer`, `qiskit-ibm-runtime`,
  `pylatexenc`, ditambah mana-mana pakej khusus tutorial yang dikesan melalui
  pengimbasan import). Pemasangan akan **dilangkau** kalau pakej dah ada
  (contohnya pada Binder atau Code Engine yang memang dah pra-pasang).
- **Sediakan templat kelayakan yang dikomentari** untuk IBM Quantum, supaya
  pengguna yang nak jalankan pada perkakasan sebenar boleh nyahkomen dan isi
  kunci API mereka.

Pada Google Colab, sel ni akan jalankan secara automatik semasa notebook dibuka
melalui flag metadata `cell_execution_strategy: setup`.

### 3. Penulisan semula laluan imej {#3-image-path-rewrites}

Laluan imej relatif (`/docs/images/...`, `/learning/images/...`) ditulis semula
supaya berfungsi dengan betul dalam persekitaran notebook yang berdiri sendiri.

## Halaman MDX (pemaparan dalam pelayar) {#mdx-pages-in-browser-rendering}

Tutorial yang dipaparkan pada laman web ni ditukar daripada notebook `.ipynb`
atau fail `.mdx` dari sumber asal. Transformasi berikut digunakan:

- **Baris `pip install`** ditambah pada blok kod Python yang mengimport pakej
  pihak ketiga, membolehkan pelaksanaan satu klik melalui thebelab.
- **Bahagian Tinjauan Tutorial IBM**: Nota ditambah untuk menjelaskan bahawa
  tinjauan tu kepunyaan IBM Quantum dan ada pautan ke GitHub Issues doQumentation
  untuk maklum balas khusus laman web.
- **Widget maklum balas**: Widget "Adakah ini membantu?" ditambah di bahagian
  bawah setiap tutorial, dijejaki melalui analitik Umami yang mesra privasi.
- **Pembetulan sintaks MDX**: Pendakap keriting, hierarki tajuk, dan isu
  keserasian JSX diperbetulkan secara automatik untuk pemaparan Docusaurus.
- **OpenInLabBanner**: Banner interaktif disisipkan di bawah tajuk dengan butang
  untuk buka notebook dalam Colab, Binder, atau Code Engine.

## Apa yang TIDAK diubah {#what-is-not-modified}

- **Kandungan tutorial itu sendiri** (penjelasan, logik kod, output) tidak pernah
  diubah.
- **Atribusi pengarang asal** dipelihara melalui frontmatter dan fail NOTICE
  (lesen Apache 2.0 / CC BY-SA 4.0).
- **Tiada kod telemetri atau penjejakan** yang disisipkan ke dalam notebook.
  Analitik (Umami) hanya berjalan pada laman web doQumentation, bukan dalam
  notebook yang dieksport.

## Kod sumber {#source-code}

Semua transformasi dilaksanakan dalam
[`scripts/sync-content.py`](https://github.com/JanLahmann/doQumentation/blob/main/scripts/sync-content.py).
