# Gambaran keseluruhan pelan yang tersedia

IBM Quantum Platform menawarkan pelan Open (akses percuma) dan pelan berbayar, seperti yang diterangkan di bawah. Lihat [pelan akses IBM Quantum](https://www.ibm.com/quantum/pricing) untuk maklumat penuh.

Jika anda mahu akses kepada berbilang pelan, anda boleh mencipta berbilang instance dan menetapkan pelan berbeza kepada setiap satu. Contohnya, anda boleh mempunyai instance Pelan Open dan instance Pelan Pay-As-You-Go. Anda boleh menetapkan instance pilihan untuk digunakan dalam pemula `QiskitRuntimeService` atau kaedah `QiskitRuntimeService.save_account()`. Contoh:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Pelan Open

Jalankan litar kuantum anda pada QPU terbaik di dunia secara percuma (sehingga 10 minit masa kuantum dalam tempoh gelinding 28 hari). Anda boleh menjejak penggunaan anda pada halaman [papan pemuka Platform](https://quantum.cloud.ibm.com/) dan [Workloads](https://quantum.cloud.ibm.com/workloads). Pelan Open hanya tersedia di rantau us-east.

Pelan ini disyorkan bagi mereka yang sedang belajar pengkomputeran kuantum dan meneroka teknologi kuantum IBM.

## Pelan Flex

Beli terlebih dahulu bilangan minit tertentu (sekurang-kurangnya 400) untuk menjalankan kerja kuantum, untuk digunakan dalam tempoh satu tahun dari tarikh pembelian. Anda boleh membeli lebih banyak minit pada bila-bila masa.

Selepas membeli masa, anda perlu memperuntukkan minit kepada instance sedia ada atau baharu pada Pelan Flex untuk akaun anda. Tetapkan atau kemas kini had peruntukan pada [halaman Instances](https://quantum.cloud.ibm.com/instances) atau tetapkan peruntukan semasa anda [mencipta instance Pelan Flex baharu.](/guides/instances) Jika anda memperuntukkan semua minit kepada instance sedia ada, anda tidak boleh mencipta instance baharu untuk Pelan Flex sehingga anda membeli lebih banyak minit atau mengeluarkan beberapa minit daripada peruntukan instance lain. Lihat [Tetapkan had peruntukan instance](allocation-limits) untuk maklumat lanjut.

Jika anda memerlukan lebih daripada 10,000 minit, anda patut mempertimbangkan langganan Pelan Premium. Jika anda tidak mahu membeli 400 minit, pertimbangkan Pelan Pay-As-You-Go.

[Hubungi IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) untuk mendaftar Pelan Flex.

## Pelan Pay-As-You-Go

Jalankan litar kuantum pada QPU terbaik di dunia dan bayar hanya untuk masa kuantum yang anda gunakan.

Pelan ini disyorkan bagi mereka yang menjalankan projek penyelidikan utiliti kuantum dan menguji model perniagaan dengan akses yang fleksibel. Jika anda akan menggunakan sekurang-kurangnya 400 minit dalam tahun hadapan, adalah disyorkan agar anda menggunakan Pelan Flex.

## Pelan Premium

Jalankan litar kuantum pada QPU terbaik di dunia menggunakan langganan masa kuantum peringkat enterprise. Selain itu, beberapa fungsi, seperti [Qiskit Functions](functions) dan [Qiskit Transpiler as a Service](qiskit-transpiler-service), tersedia untuk pengguna Pelan Premium.

Pelan ini disyorkan bagi mereka yang ingin melaksanakan peta jalan kuantum strategik dan membangunkan algoritma serta aplikasi kuantum pada skala besar. [Hubungi IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) untuk maklumat lanjut.

## Pelan On-Prem

Akses sistem kuantum dalam premis yang sepenuhnya berdedikasi, yang diselenggarakan oleh IBM Quantum.

Perkhidmatan ini disyorkan bagi mereka yang meneroka algoritma dan aplikasi kuantum-klasikal serta memerlukan kawalan tinggi ke atas sumber dan data mereka. [Hubungi IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) untuk maklumat lanjut.
