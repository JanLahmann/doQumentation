# Panoramica

!["Immagine hero"](/learning/images/courses/quantum-machine-learning/hero.svg)

Questo corso esplora come il quantum computing possa svolgere un ruolo entusiasmante nel machine learning. Il corso offre una panoramica degli approcci classici al machine learning, presupponendo al contempo una certa familiarità con questi metodi. L'obiettivo principale è capire come il quantum computing possa integrarsi nei flussi di lavoro di machine learning esistenti. Gli argomenti trattati includono la codifica dei dati, i kernel quantistici e i circuiti quantistici variazionali.
