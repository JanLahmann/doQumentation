# Überblick {#overview}

!["Titelbild"](/learning/images/courses/quantum-machine-learning/hero.svg)

Dieser Kurs untersucht, welche spannende Rolle Quantencomputing im Bereich Machine Learning spielen kann. Der Kurs bietet eine Auffrischung klassischer Machine-Learning-Ansätze, setzt dabei aber eine gewisse Vertrautheit mit diesen Methoden voraus. Im Mittelpunkt steht die Frage, wie Quantencomputing in bestehende Machine-Learning-Workflows integriert werden kann. Themen sind unter anderem Datenkodierung, Quantenkerne und variationelle Quantenschaltkreise.
