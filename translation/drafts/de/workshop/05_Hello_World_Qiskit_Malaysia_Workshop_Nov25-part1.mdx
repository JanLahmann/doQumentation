## Qiskit installieren <a id="install-qiskit"></a>

Du kannst eine Online-Jupyter-Lab-Umgebung verwenden (siehe Anleitung [Online-Lab-Umgebungen](quantum.cloud.ibm.com/docs/en/guides/online-lab-environments)) oder Qiskit lokal installieren.

Folge der Qiskit-Installationsanleitung [Qiskit SDK und den Qiskit Runtime Client installieren](https://quantum.cloud.ibm.com/docs/en/guides/install-qiskit), um die folgenden Schritte abzuschließen:

	- Qiskit inklusive der zusätzlichen Visualisierungspakete installieren: `pip install qiskit[visualization]`

	- qiskit-ibm-runtime installieren: `pip install qiskit-ibm-runtime`

	- jupyter installieren: `pip install jupyter`
Überprüfe, dass die Python-Version in deiner Umgebung python>=3.10 ist, um sicherzustellen, dass sie mit der neuesten Qiskit-Version kompatibel ist:
