# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) sind offizielle Erweiterungspakete, die spezialisierte Quantencomputing-Algorithmen und -Techniken bereitstellen. Jedes Addon konzentriert sich auf einen bestimmten Bereich und enthält Tutorials mit ausführbarem Code.

## Verfügbare Addons

| Addon | Paket | Beschreibung |
|-------|-------|--------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Reduziere Circuit-Breite und -Tiefe durch Schneiden von Gates und Drähten |
| **SQD** | `qiskit-addon-sqd` | Sample-basierte Quantumdiagonalisierung zur Schätzung des Grundzustands |
| **OBP** | `qiskit-addon-obp` | Operator-Rückwärtspropagierung zur Reduzierung der Circuit-Tiefe |
| **MPF** | `qiskit-addon-mpf` | Multi-Produkt-Formeln für verbesserte Hamiltonian-Simulation |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximierte Quantenkompilierung mit Tensornetzwerken |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption zur Fehlerminderung |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones zur Reduzierung des PEC-Sampling-Overheads |

Alle Tutorials können direkt in deinem Browser mit Binder oder auf deinem eigenen Jupyter-Server ausgeführt werden.

Für die API-Dokumentation besuche die einzelnen Addon-Repos auf [GitHub](https://github.com/Qiskit?q=qiskit-addon).
