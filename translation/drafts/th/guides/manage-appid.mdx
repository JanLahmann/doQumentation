# จัดการผู้ใช้ ID provider {#manage-id-provider-users}

สามารถผสานรวมกับ external identity provider (IDP) เพื่อยืนยันตัวตนผู้ใช้ภายนอกกับบัญชี IBM Cloud อย่างปลอดภัย โดยการใช้ external provider สามารถให้วิธีสำหรับผู้ใช้ในบริษัทในการใช้ single sign-on (SSO) ได้ เชื่อมต่อ cloud account กับ external IDP ได้ด้วยวิธีใดวิธีหนึ่งต่อไปนี้:

- ใช้ [IBM Cloud&reg; SAML service provider (SP)](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-cloud-saml)
- ใช้ [App ID](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-appid) (อาจมีค่าใช้จ่าย)
- [Federate ผู้ใช้ไปยัง IBM&reg; products ทั้งหมดด้วย IBMid](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-ibmid)

<Admonition type="tip">
หลังจากเชื่อมต่อ IDP แล้ว สามารถตั้งค่า dynamic rules เพื่อไม่ต้องเพิ่มผู้ใช้ทุกคนในทุก group ด้วยตนเองเมื่อเข้าร่วม ดู [Enabling authentication from an external identity provider](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#app-id-dynamic-rules) สำหรับคำแนะนำ
</Admonition>
