# เริ่มต้นใช้งาน Sample-based quantum diagonalization (SQD)
<details>
<summary><b>เวอร์ชันของแพ็กเกจ</b></summary>

โค้ดในหน้านี้พัฒนาโดยใช้ข้อกำหนดต่อไปนี้
แนะนำให้ใช้เวอร์ชันเหล่านี้หรือใหม่กว่า

```
qiskit-addon-sqd~=0.10.0
pyscf~=2.8.0; sys.platform != 'win32'
```
</details>
<Admonition type="caution" title="หมายเหตุสำหรับผู้ใช้ Windows">

ตัวอย่างนี้ใช้แพ็กเกจ `pyscf` ซึ่งเป็นไลบรารีจำลองเคมีด้วย Python ที่ไม่รองรับบน Windows โดยตรง ดังนั้นตัวอย่างด้านล่างจะไม่สามารถทำงานได้

เพื่อรันโค้ดด้านล่าง ผู้ใช้ Windows ต้องสร้าง Python environment โดยใช้ Windows Subsystem for Linux ตามที่แนะนำในคู่มือติดตั้ง [`pyscf`](https://pyscf.org/user/install.html)

</Admonition>

คู่มือนี้สาธิตตัวอย่างการทำงานแบบง่ายเพื่อเริ่มต้นใช้แพ็กเกจ `qiskit-addon-sqd` ในตัวอย่างนี้ คุณสามารถใช้ SQD เพื่อหาค่าประมาณของสถานะพื้นของโมเลกุล $N_2$ ที่สมดุล

เริ่มต้นด้วยการทบทวนว่า Hamiltonian ของอิเล็กตรอนที่มีปฏิสัมพันธ์ทั่วไปมีรูปแบบดังนี้:

$$ \hat{H} = \sum_{\substack{pr\\ \sigma}}h_{pr} \hat{a}_{p\sigma}^\dagger \hat{a}_{r\sigma} + \sum_{\substack{prqs\\ \sigma\tau}} \frac{1}{2}\left(pr|qs\right)\hat{a}_{p\sigma}^\dagger \hat{a}_{q\tau}^\dagger \hat{a}_{s\tau}\hat{a}_{r\sigma} $$

โดยที่ $\hat{a}_{p\sigma}^\dagger$ และ $\hat{a}_{p\sigma}$ คือตัวดำเนินการสร้างและทำลาย Fermionic ที่สัมพันธ์กับออร์บิทัลที่ $p$ ที่มีสปิน $\sigma$ และ $h_{pr}$ กับ $\left(pr|qs\right)$ คืออินทิกรัลอิเล็กทรอนิกส์แบบหนึ่งและสองตัว
## เตรียมข้อมูลโมเลกุล {#prepare-molecule-information}

เริ่มต้นด้วยการระบุโมเลกุลและคุณสมบัติของมันโดยใช้ `pyscf` และอินทิกรัลอิเล็กทรอนิกส์ที่เก็บใน `n2_fci.txt`
<Admonition type="note">
ตัวอย่างนี้รับข้อมูลที่สร้างไว้ล่วงหน้าในไฟล์ชื่อ `n2_fci.txt` ซึ่งประกอบด้วยอินทิกรัลอิเล็กทรอนิกส์แบบหนึ่งและสองตัว เพื่อรันโค้ดด้านล่างบนเครื่องท้องถิ่น ให้คัดลอกและวางข้อมูลลงในไฟล์ที่มีชื่อเดียวกัน ตัวอย่างนี้ยังต้องติดตั้งแพ็กเกจ `pyscf` ด้วย
</Admonition>
