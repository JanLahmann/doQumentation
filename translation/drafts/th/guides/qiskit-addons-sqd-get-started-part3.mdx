## รับตัวอย่างจาก ansatz {#obtain-samples-from-an-ansatz}

เมื่อโหลดข้อมูลโมเลกุลเข้ามาแล้ว ขั้นต่อไปคือกำหนด ansatz Circuit และปรับให้เหมาะสมสำหรับการประมวลผล ซึ่งต้องพิจารณาสมมาตรของปัญหา การเลือก ansatz และการปรับให้ทำงานบนฮาร์ดแวร์ควอนตัม

อย่างไรก็ตาม การตัดสินใจเหล่านี้ขึ้นอยู่กับปัญหาเฉพาะและอยู่นอกขอบเขตของคู่มือนี้ แต่เพื่อสาธิตขั้นตอนการทำงานของ SQD เราจะสร้างชุดนับแบบสุ่มสำหรับการวนซ้ำการกู้คืนค่าคอนฟิกูเรชั่นเพื่อประมวลผลภายหลัง ซึ่งจำลองข้อมูลการวัดของ Circuit 32 Qubit ที่สุ่มตัวอย่าง 10,000 ช็อต หลังจากสร้างข้อมูลนับแล้ว เมธอด `recover_configurations()` จำเป็นต้องแปลงข้อมูลนับให้เป็นเมทริกซ์ของ bitstring ที่วัดได้ในแต่ละช็อต รวมถึงอาร์เรย์ของความน่าจะเป็นสำหรับแต่ละสถานะที่วัดได้

<Admonition type="note">
การสร้างตัวอย่างเทียมนี้ใช้เป็นเพียงวิธีสาธิตเครื่องมือของ `qiskit-addon-sqd` เท่านั้น แพ็กเกจนี้ออกแบบมาเพื่อใช้เป็นส่วนหนึ่งของขั้นตอนการทำงานขนาดใหญ่กว่า ซึ่งมีการกำหนด ansatz หรือ Circuit อื่น ๆ ปรับให้เหมาะสม และประมวลผลโดยใช้ Sampler primitive โค้ดด้านล่างมีโค้ดที่คอมเมนต์ไว้เพื่อแสดงให้เห็นว่าขั้นตอนการทำงานทั่วไปจะเป็นอย่างไรเมื่อได้รับ Circuit ansatz ที่แปลงเป็น ISA circuit แล้ว
</Admonition>

## เรียกใช้การวนซ้ำการกู้คืนค่าคอนฟิกูเรชั่น {#run-configuration-recovery-loop}

ตัวอย่างการวัดสามารถปรับปรุงได้โดยการทำซ้ำการกู้คืนค่าคอนฟิกูเรชั่นและการไดอะโกนัลไลซ์ชุดตัวอย่างย่อยเพื่อประมาณ ground state จนกว่าจะลู่เข้า

ก่อนอื่น กำหนดตัวเลือกต่อไปนี้:

- `ITERATIONS`: จำนวนการวนซ้ำการกู้คืนค่าคอนฟิกูเรชั่นแบบ self-consistent
- `NUM_BATCHES`: จำนวนชุดตัวอย่างย่อยที่จะไดอะโกนัลไลซ์
- `SAMPLES_PER_BATCH`: จำนวนตัวอย่างที่จะรวมในแต่ละ batch
- `MAX_DAVIDSON_CYCLES`: จำนวนการวนซ้ำสำหรับตัวแก้ eigenstate

```python
# SQD options
ITERATIONS = 5

# Eigenstate solver options
NUM_BATCHES = 1
SAMPLES_PER_BATCH = 500
MAX_DAVIDSON_CYCLES = 200
```

ต่อไป เพื่อพล็อตการลู่เข้า ให้กำหนดอาร์เรย์เพื่อเก็บการประมาณพลังงาน ground state ค่าคาดหวังของ $\langle S \rangle ^2$ และการครอบครอง orbital ของโมเลกุล

```python
# Self-consistent configuration recovery loop
energy_hist = np.zeros((ITERATIONS, NUM_BATCHES))  # energy history
spin_sq_hist = np.zeros((ITERATIONS, NUM_BATCHES))  # spin history
occupancy_hist = []
avg_occupancy = None
```

ตอนนี้ให้เรียกใช้การวนซ้ำการกู้คืนค่าคอนฟิกูเรชั่น การวนซ้ำแต่ละรอบประกอบด้วยสามขั้นตอน:

1. ใช้ฟังก์ชัน `recover_configurations()` เพื่อรับเมทริกซ์ bitstring และอาร์เรย์ความน่าจะเป็นที่ปรับปรุงแล้วตามค่าเฉลี่ยการครอบครอง orbital
1. ใช้ `postselect_and_subsample()` เพื่อรวบรวม batch ของตัวอย่างย่อยสำหรับการไดอะโกนัลไลซ์
1. จากนั้นใช้ batch ของตัวอย่างย่อยเป็นอาร์กิวเมนต์ให้กับฟังก์ชัน `solve_fermion()` เพื่อรับการประมาณ ground state

สิ่งสำคัญที่ต้องทราบคือวิธีจัดการการวนซ้ำแรกของการวนซ้ำการกู้คืนค่าคอนฟิกูเรชั่น เนื่องจากค่าเฉลี่ยการครอบครอง orbital ยังไม่พร้อมใช้งาน จึงมีการเรียกใช้เฉพาะฟังก์ชัน `postselect_and_subsample()` เท่านั้น ซึ่งจะลบตัวอย่างที่ไม่ใช่ physical (ตัวอย่างที่มี Hamming weight ไม่ถูกต้อง) ก่อนเรียกใช้ตัวแก้ eigenstate `solve_fermion()` หลังจากนั้น ค่าเฉลี่ยการครอบครอง orbital จะถูกคำนวณจากทุก batch และใช้เป็น input ให้กับเมธอด `recover_configurations()` ซึ่งจะพลิก bit แต่ละตัวแบบ probabilistic ตามค่าเฉลี่ยนี้ ดูข้อมูลเพิ่มเติมในส่วน **II-A** ของข้อมูลเสริมในบทความ [SQD](https://arxiv.org/abs/2405.05068)

```python
from qiskit_addon_sqd.configuration_recovery import recover_configurations
from qiskit_addon_sqd.subsampling import postselect_and_subsample
from qiskit_addon_sqd.fermion import (
    bitstring_matrix_to_ci_strs,
    solve_fermion,
)

for i in range(ITERATIONS):
    print(f"\nStarting configuration recovery iteration {i}")
    # On the first iteration, we have no orbital occupancy information from the
    # solver, so we just post-select from the full bitstring set based on Hamming weight.
    if avg_occupancy is None:
        bitstring_matrix_tmp = bitstring_matrix_full
        probs_array_tmp = probs_array_full

    # If there is average orbital occupancy information, use it to refine the full set of noisy configurations
    else:
        bitstring_matrix_tmp, probs_array_tmp = recover_configurations(
            bitstring_matrix_full,
            probs_array_full,
            avg_occupancy,
            num_alpha,
            num_beta,
            rand_seed=rng,
        )

    # Throw out configurations with an incorrect particle number in either the spin-up or spin-down systems
    batches = postselect_and_subsample(
        bitstring_matrix_tmp,
        probs_array_tmp,
        hamming_right=num_alpha,
        hamming_left=num_beta,
        samples_per_batch=SAMPLES_PER_BATCH,
        num_batches=NUM_BATCHES,
        rand_seed=rng,
    )

    # Run eigenstate solvers in a loop. This loop should be parallelized for larger problems.
    e_tmp = np.zeros(NUM_BATCHES)
    s_tmp = np.zeros(NUM_BATCHES)
    occs_tmp = []
    coeffs = []
    for j in range(NUM_BATCHES):
        strs_a, strs_b = bitstring_matrix_to_ci_strs(batches[j])
        print(f"Batch {j} subspace dimension: {len(strs_a) * len(strs_b)}")
        energy_sci, coeffs_sci, avg_occs, spin = solve_fermion(
            batches[j],
            core_hamiltonian,
            electron_repulsion_integrals,
            open_shell=open_shell,
            spin_sq=spin_sq,
            max_cycle=MAX_DAVIDSON_CYCLES,
        )
        energy_sci += nuclear_repulsion_energy
        e_tmp[j] = energy_sci
        s_tmp[j] = spin
        occs_tmp.append(avg_occs)
        coeffs.append(coeffs_sci)

    # Combine batch results
    avg_occupancy = tuple(np.mean(occs_tmp, axis=0))

    # Track optimization history
    energy_hist[i, :] = e_tmp
    spin_sq_hist[i, :] = s_tmp
    occupancy_hist.append(avg_occupancy)
    print()
```

### แสดงผลลัพธ์ {#visualize-the-results}

สุดท้าย ผลลัพธ์สามารถแสดงภาพได้โดยการตรวจสอบพลังงานที่ประมาณและค่าเฉลี่ยการครอบครอง orbital ในแต่ละการวนซ้ำของการวนซ้ำการกู้คืนค่าคอนฟิกูเรชั่น กราฟแรกแสดงให้เห็นว่าหลังจากการวนซ้ำไม่กี่รอบ พลังงาน ground state ถูกประมาณในระดับความแม่นยำประมาณ `27 mH` (ความแม่นยำทางเคมีโดยทั่วไปถือว่าอยู่ที่ `1 kcal/mol` $\approx$ `1.6 mH`) จำไว้ว่าการสาธิตนี้ใช้ noise ล้วน ๆ และความสามารถในการประมาณพลังงานในระดับนี้มาจากความรู้เบื้องต้นเกี่ยวกับโมเลกุลและโครงสร้างอิเล็กทรอนิกส์ของมัน

กราฟที่สองแสดงค่าเฉลี่ยการครอบครองของ spatial orbital แต่ละตัวหลังจากการวนซ้ำครั้งสุดท้าย สังเกตว่าทั้งอิเล็กตรอน spin-up และ spin-down ครอบครอง orbital ห้าตัวแรกด้วยความน่าจะเป็นสูงในผลลัพธ์

```python
import matplotlib.pyplot as plt

# Chemical accuracy (+/- 1 milli-Hartree)
chem_accuracy = 0.001
exact_energy = -109.04667

# Data for energies plot
x1 = range(ITERATIONS)
min_e = [np.min(e) for e in energy_hist]
e_diff = [abs(e - exact_energy) for e in min_e]
yt1 = [1.0, 1e-1, 1e-2, 1e-3, 1e-4]

# Data for avg spatial orbital occupancy
y2 = occupancy_hist[-1][0] + occupancy_hist[-1][1]
x2 = range(len(y2))

fig, axs = plt.subplots(1, 2, figsize=(12, 6))

# Plot energies
axs[0].plot(x1, e_diff, label="energy error", marker="o")
axs[0].set_xticks(x1)
axs[0].set_xticklabels(x1)
axs[0].set_yticks(yt1)
axs[0].set_yticklabels(yt1)
axs[0].set_yscale("log")
axs[0].set_ylim(1e-4)
axs[0].axhline(
    y=chem_accuracy,
    color="#BF5700",
    linestyle="--",
    label="chemical accuracy",
)
axs[0].set_title("Approximated Ground State Energy Error vs SQD Iterations")
axs[0].set_xlabel("Iteration Index", fontdict={"fontsize": 12})
axs[0].set_ylabel("Energy Error (Ha)", fontdict={"fontsize": 12})
axs[0].legend()

# Plot orbital occupancy
axs[1].bar(x2, y2, width=0.8)
axs[1].set_xticks(x2)
axs[1].set_xticklabels(x2)
axs[1].set_title("Avg Occupancy per Spatial Orbital")
axs[1].set_xlabel("Orbital Index", fontdict={"fontsize": 12})
axs[1].set_ylabel("Avg Occupancy", fontdict={"fontsize": 12})

print(f"Exact energy: {exact_energy:.5f} Ha")
print(f"SQD energy: {min_e[-1]:.5f} Ha")
print(f"Absolute error: {e_diff[-1]:.5f} Ha")
plt.tight_layout()
plt.show()
```
