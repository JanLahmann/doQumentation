## ตระกูลโปรเซสเซอร์ที่เลิกใช้งานแล้ว {#retired-processor-families}

- **Canary**: ตระกูล Canary ประกอบด้วยดีไซน์ขนาดเล็กที่มี Qubit ตั้งแต่ 5 ถึง 16 ตัว
- **Falcon**: ตระกูล Falcon ของอุปกรณ์เสนอแพลตฟอร์มที่มีคุณค่าสำหรับ Circuit ขนาดกลาง และยังทำหน้าที่เป็นแพลตฟอร์มที่มีคุณค่าสำหรับการสาธิตการปรับปรุงประสิทธิภาพและความสามารถในการขยายก่อนที่จะนำไปใช้บนอุปกรณ์ขนาดใหญ่กว่า
- **Egret**: Egret นำนวัตกรรม tunable coupler มาสู่แพลตฟอร์ม 33 Qubit ส่งผลให้ Gate สองQubit ทำงานได้เร็วขึ้นและมีความเที่ยงตรงสูงขึ้น
- **Hummingbird**: โดยใช้เลย์เอาต์ Qubit แบบ heavy-hexagonal ตระกูล Hummingbird รองรับ Qubit ได้สูงสุด 65 ตัว
- **Osprey**: Osprey มีขนาดใหญ่เกือบสี่เท่าของ Eagle ที่ 433 Qubit ขนาดชิปที่ใหญ่ขึ้นต้องการการปรับปรุงเพิ่มเติมในด้านบรรจุภัณฑ์อุปกรณ์ รวมถึงสาย flex แบบกำหนดเองในตู้เย็น cryostat เพื่อรองรับข้อกำหนด I/O ที่มากขึ้นภายในพื้นที่การเดินสายเดิม
- **Eagle**: ตระกูลโปรเซสเซอร์ Eagle ใช้เทคโนโลยีบรรจุภัณฑ์ที่ขยายขนาดได้ดีกว่ารุ่นก่อนหน้า โดยเฉพาะอย่างยิ่ง สัญญาณส่งผ่านชั้นชิปหลายชั้น ทำให้มี I/O ความหนาแน่นสูงโดยไม่สูญเสียประสิทธิภาพ

## ดึง job จาก QPU ที่เลิกใช้งานแล้ว {#retrieve-a-job-from-a-retired-qpu}

ใช้โค้ดต่อไปนี้เพื่อดึง job จาก QPU ที่เลิกใช้งานแล้ว

<Admonition type="note">
การส่ง Cloud Resource Name (CRN) ในโค้ดต่อไปนี้เป็นตัวเลือก เมื่อส่งเข้ามา จะมีเพียง Backend และ job จาก instance นั้นเท่านั้นที่พร้อมใช้งาน หากไม่ระบุ instance จะสามารถใช้ Backend และ job ทั้งหมดในทุก instance ในบัญชีของคุณได้ ในกรณีนี้ เมื่อระบุ Backend จะค้นหาและใช้ instance ที่สามารถเข้าถึง Backend นั้นได้
</Admonition>

```python
from qiskit_ibm_runtime import QiskitRuntimeService

# Load your IBM Quantum account. Replace "<IBM Cloud CRN>" with your desired instance's CRN.
service = QiskitRuntimeService(channel="ibm_quantum_platform", instance="<IBM Cloud CRN>")

# Retrieve a single job by id
job = service.job(<job_id>)

# Retrieve a batch of jobs. Filtering options can be found in the QiskitRuntimeService.jobs api reference
jobs = service.jobs(backend_name=<backend_name>)
```

<Admonition type="note" title="แพ็กเกจ provider ที่เลิกใช้งานแล้ว">
  `service.jobs()` คืนค่า job ที่รันจากทั้ง `qiskit-ibm-runtime` และแพ็กเกจ `qiskit-ibm-provider` ที่เลิกใช้งานแล้ว job ที่ส่งโดยแพ็กเกจ `qiskit-ibmq-provider` รุ่นเก่า (ที่เลิกใช้งานเช่นกัน) ไม่สามารถใช้งานได้อีกต่อไป
</Admonition>
