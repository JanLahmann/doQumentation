# Logging สำหรับ Qiskit Runtime {#logging-for-qiskit-runtime}

บริการ IBM Cloud&reg; เช่น Qiskit Runtime จะสร้าง platform logs ที่ใช้ตรวจสอบกิจกรรมที่ผิดปกติและการดำเนินการสำคัญในบัญชี รวมถึงแก้ไขปัญหาต่างๆ

สามารถใช้ [IBM Cloud Logs Routing](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about) ซึ่งเป็น platform service เพื่อกำหนดเส้นทาง platform logs ในบัญชีไปยังปลายทางที่ต้องการโดยการกำหนดค่า tenant ที่ระบุว่า platform logs ถูกส่งไปที่ใด สำหรับข้อมูลเพิ่มเติม ดู [About Logs Routing](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about)

สามารถใช้ [IBM Cloud Logs](https://cloud.ibm.com/docs/cloud-logs) เพื่อแสดงภาพและแจ้งเตือน platform logs ที่สร้างในบัญชีและถูกกำหนดเส้นทางโดย IBM Cloud Logs Routing ไปยัง IBM Cloud Logs instance

## ตำแหน่งที่สร้าง platform logs {#locations-where-platform-logs-are-generated}

IBM Cloud Container Registry สร้าง platform logs ที่แสดงใน logging instances ตารางต่อไปนี้แสดงตำแหน่งที่เปิดใช้งานการเก็บรวบรวม Container Registry service logs โดยอัตโนมัติ

| ตำแหน่ง                   | มี platform services logs ให้ใช้งานหรือไม่? |
|----------------------------|---------------------------------------|
| Washington, D.C. (us-east) |                  ใช่                  |
| Frankfurt (eu-de)          |                  ใช่                  |

## Platform logs ที่ถูกสร้าง {#platform-logs-that-are-generated}

IBM Qiskit Runtime สร้าง platform services logs ที่แสดงใน logging instances สำหรับกรณีเฉพาะต่อไปนี้:

- เมื่อสถานะของ job เปลี่ยนแปลง

## การดู logs จากหน้า Observability {#viewing-logs-from-the-observability-page}

สำหรับข้อมูลเกี่ยวกับการเปิด IBM Cloud Logs interface ดู **Launching the UI** ใน [IBM Cloud Logs documentation](https://cloud.ibm.com/docs/cloud-logs?topic=cloud-logs-instance-launch)

## Fields ตามประเภท log {#fields-by-log-type}

สำหรับข้อมูลเกี่ยวกับ fields ที่รวมอยู่ใน platform log ทุกรายการ ดู [Fields that are included in platform logs](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about-platform-logs#about-platform-logs-2)

fields ต่อไปนี้รวมอยู่ใน log record

| Field           | ประเภท     | คำอธิบาย                                                              |
|-----------------|----------|--------------------------------------------------------------------------|
| logSourceCRN    | Required | กำหนดบัญชีและ flow log instance ที่เผยแพร่ log    |
| saveServiceCopy | Required | กำหนดว่า IBM บันทึกสำเนาของ record ไว้เพื่อวัตถุประสงค์ด้านการดำเนินงานหรือไม่ |
| message         | Required | คำอธิบายของ log ที่สร้างขึ้น                                |
