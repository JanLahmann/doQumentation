# แนะนำ options

 คุณสามารถส่ง options ให้กับ primitives เพื่อปรับแต่งให้ตรงตามความต้องการ ส่วนนี้จะเน้นที่ options ของ Qiskit Runtime primitive แม้ว่า interface ของ method `run()` ของ primitives จะเหมือนกันทุก implementation แต่ options ของแต่ละตัวไม่เหมือนกัน ดู API references ที่เกี่ยวข้องสำหรับข้อมูลเกี่ยวกับ options ของ [`qiskit.primitives`](https://docs.quantum.ibm.com/api/qiskit/primitives#primitives) และ [`qiskit_aer.primitives`](https://qiskit.github.io/qiskit-aer/apidocs/aer_primitives.html)

## ภาพรวม

<span id="options-structure"></span>
### โครงสร้าง

เมื่อเรียก primitives คุณสามารถส่ง options โดยใช้ options class หรือ dictionary ก็ได้ Options ที่ใช้บ่อย เช่น `resilience_level` จะอยู่ในระดับแรก ส่วน options อื่นจะจัดกลุ่มไว้ในหมวดหมู่ต่างๆ เช่น `execution` ดูรายละเอียดทั้งหมดในส่วน [ตั้งค่า primitive options](/guides/specify-runtime-options#pass-options)

<span id="options-defaults"></span>
### ค่าเริ่มต้น

ถ้าคุณไม่ระบุค่าสำหรับ option ใด มันจะได้ค่าพิเศษ `Unset` และจะใช้ค่า default ของเซิร์ฟเวอร์แทน ดังนั้นค่า default จะเหมือนกันไม่ว่าจะใช้ code version ไหน

ตารางในส่วน [สรุป Options classes](#options-classes) จะแสดงค่า default ไว้

<span id="options-precedence"></span>
## ตั้งค่า options

สามารถกำหนด options ก่อนสร้าง primitive แล้วส่งให้กับ primitive ซึ่งจะสำเนาค่าเหล่านั้น ทำได้ทั้งในรูปแบบ nested dictionary หรือใช้ options classes นอกจากนี้ยังเปลี่ยน options ของ primitive ได้หลังจากสร้างแล้ว ใช้วิธีที่เหมาะกับแอปพลิเคชันของคุณ ดูรายละเอียดทั้งหมดในหน้า [Specify options](specify-runtime-options)

<span id="options-table"></span>
## Options ที่ใช้ได้

ตารางต่อไปนี้แสดง options จาก `qiskit-ibm-runtime` เวอร์ชันล่าสุด หากต้องการดู options เวอร์ชันเก่า ให้ไปที่ [API reference ของ `qiskit-ibm-runtime`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime) แล้วเลือกเวอร์ชันก่อนหน้า

<Tabs>
  <TabItem value="Estimator" label="Estimator">
<details>
<summary>
### `default_shots`
</summary>

จำนวน shots ทั้งหมดที่ใช้ต่อ Circuit ต่อ configuration

**ตัวเลือก**: Integer >= 0

**ค่าเริ่มต้น**: None

[เอกสาร API `default_shots`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/options-estimator-options#default_shots)
</details>

<details>
<summary>
### `default_precision`
</summary>

ความแม่นยำ default ที่ใช้กับ PUB หรือการเรียก `run()` ที่ไม่ระบุค่า

**ตัวเลือก**: Float > 0

**ค่าเริ่มต้น**: 0.015625 (1 / sqrt(4096))

[เอกสาร API `default_precision`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/options-estimator-options#default_precision)
