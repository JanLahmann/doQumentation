# บทนำ {#introduction}

## บทนำสู่ Qiskit และ IBM Quantum {#introduction-to-qiskit-and-ibm-quantum}

ยินดีต้อนรับสู่เอกสารประกอบของ Qiskit, แพ็กเกจที่เกี่ยวข้อง และ IBM Quantum&reg; Platform เอกสารนี้ประกอบด้วยคู่มือการใช้งานเพื่อเริ่มต้นกับเครื่องมือของเรา, tutorials สำหรับ use cases เฉพาะที่มีตัวอย่างครบวงจร และคอลเลกชัน API references

Qiskit มอบ framework แบบ modular และขยายได้สำหรับการวิจัยและพัฒนาควอนตัมในด้านอัลกอริทึม, high-performance computing และ quantum information science ด้วย Qiskit นักวิจัยสามารถสร้าง, ปรับแต่ง และรัน quantum workflows ด้วย addons เฉพาะทาง, เครื่องมือซอฟต์แวร์ และทรัพยากรมากมาย ผ่าน IBM Quantum Platform ผู้ใช้สามารถเข้าถึงบริการ Quantum Compute เช่น Qiskit Runtime และ Qiskit Functions Catalog เพื่อรัน workloads อย่างมีประสิทธิภาพบนกลุ่มคอมพิวเตอร์ควอนตัมของ IBM&reg;

![กราฟิกแสดงองค์ประกอบต่าง ๆ ของ IBM Quantum Platform และ Qiskit](/docs/images/guides/index/overview-image.svg)

นอกเหนือจาก Qiskit และแพ็กเกจที่เกี่ยวข้อง ยังมี [Qiskit ecosystem](https://www.ibm.com/quantum/ecosystem) ซึ่งเป็นแคตตาล็อกของโปรเจกต์ open-source ที่เชื่อมต่อกับ Qiskit เพื่อขยายฟังก์ชันการทำงาน
