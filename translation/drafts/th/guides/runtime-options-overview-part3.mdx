## ความเข้ากันได้ของฟีเจอร์ {#feature-compatibility}

เนื่องจากความแตกต่างในกระบวนการ compilation บนอุปกรณ์ ฟีเจอร์ runtime บางอย่างจึงไม่สามารถใช้ร่วมกันใน job เดียวได้ คลิกแท็บที่เหมาะสมเพื่อดูรายการฟีเจอร์ที่ไม่เข้ากันกับฟีเจอร์ที่เลือก:

<Tabs>
  <TabItem value="Dynamic" label="Dynamic circuits">
  ไม่เข้ากันกับ:
  - Gate-folding ZNE
  - PEA
  - PEC
  - Dynamical decoupling

  หมายเหตุอื่น:
  - สามารถใช้กับ gate twirling สำหรับ gate ที่ไม่มีเงื่อนไขได้
  - เข้ากันได้กับ fractional gates เมื่อใช้ `qiskit-ibm-runtime` v0.42.0 หรือใหม่กว่า

  </TabItem>
  <TabItem value="DD" label="Dynamical decoupling">
  ไม่เข้ากันกับ dynamic circuits

  </TabItem>

  <TabItem value="Fractional" label="Fractional gates">
  ไม่เข้ากันกับ:
  - Gate twirling
  - PEA
  - PEC

  เข้ากันได้กับ dynamic circuits เมื่อใช้ `qiskit-ibm-runtime` v0.42.0 หรือใหม่กว่า

  </TabItem>

  <TabItem value="ZNE" label="Gate-folding ZNE">
    ไม่เข้ากันกับ:
  - Dynamic circuits
  - PEA
  - PEC

  อาจไม่ทำงานเมื่อใช้ custom gates
  </TabItem>
  <TabItem value="Twirling" label="Gate twirling">
  ไม่เข้ากันกับ fractional gates หรือ stretches

  หมายเหตุอื่น:
  - สามารถใช้กับ dynamic circuits ที่มี gate ไม่มีเงื่อนไขได้
  - ไม่ทำงานกับ non-Clifford entanglers

  </TabItem>

  <TabItem value="PEA" label="PEA">
    ไม่เข้ากันกับ:
  - Dynamic circuits
  - Fractional gates
  - Gate-folding ZNE
  - PEC
  </TabItem>

  <TabItem value="PEC" label="PEC">
    ไม่เข้ากันกับ:
  - Dynamic circuits
  - Fractional gates
  - Gate-folding ZNE
  - PEA
  </TabItem>

</Tabs>
