# ประเภทของ Processor

ประเภทของ processor ตั้งชื่อตามคุณสมบัติทางเทคโนโลยีที่ใช้ในการสร้าง ประกอบด้วย family และ revision โดย Family (เช่น Heron) หมายถึงขนาดและความสามารถของวงจรที่รองรับบนชิป ซึ่งกำหนดจากจำนวน Qubit และกราฟการเชื่อมต่อเป็นหลัก ส่วน Revision (เช่น r3) คือรูปแบบการออกแบบที่แตกต่างกันภายใน family เดียวกัน ซึ่งมักนำมาซึ่งการปรับปรุงประสิทธิภาพหรือข้อแลกเปลี่ยนต่าง ๆ

<Admonition type="tip" title="Native gates และ operations">
เพื่อดู native gates และ operations ของ backend ให้ใช้[โค้ดนี้](/guides/get-qpu-information#native-gates-with-qiskit) สามารถดู native gates และ operations ทั้งหมดได้ใน[ตารางนี้](/guides/qpu-information#native-gates)
</Admonition>

เมื่อมีการเพิ่ม processor family ใหม่ family เก่าจะถูกยกเลิก ดูข้อมูลเพิ่มเติมได้ที่ส่วน [Retired processor families](retired-qpus#families)

## Nighthawk

![Nighthawk processor icon](/docs/images/guides/processor-types/nighthawk.svg)

ด้วย grid topology Nighthawk นำเทคโนโลยีจาก Heron มาใช้ (coupler, การส่งสัญญาณ และ coherence) พร้อมเพิ่มการเชื่อมต่อเพื่อรองรับ workload ของแอปพลิเคชันที่หลากหลายมากขึ้น

*   [ดู Nighthawk processors ที่ใช้งานได้](https://quantum.cloud.ibm.com/computers?processorType=Nighthawk)

<Admonition type="info" title="Revisions">

  `r1` (ธันวาคม 2025) เวอร์ชันแรกของ Nighthawk มี 120 Qubit

</Admonition>

## Heron

![Heron processor icon](/docs/images/guides/processor-types/heron.svg)

Heron มี 156 Qubit เป็นการอัปเกรด Eagle-sized จาก Egret โดยนำนวัตกรรมการส่งสัญญาณที่เคยใช้ใน Osprey มาพัฒนาต่อ สัญญาณที่ทำให้สามารถควบคุม two-qubit และ single-qubit แบบรวดเร็วและแม่นยำสูงนั้น ส่งผ่านสายเคเบิล flex ความหนาแน่นสูง

*   [ดู Heron processors ที่ใช้งานได้](https://quantum.cloud.ibm.com/computers?processorType=Heron)

<Admonition type="info" title="Revisions">
  `r3` (กรกฎาคม 2025) Heron r3 เป็นผลลัพธ์จากการปรับปรุงกระบวนการผลิตแบบมุ่งเป้า ซึ่งส่งผลโดยตรงต่อ coherence ความแม่นยำของ Gate และประสิทธิภาพการวัดผล

  `r2` (กรกฎาคม 2024) นี่คือ revision ของ Heron processor รุ่นดั้งเดิม ชิปได้รับการออกแบบใหม่ให้มี 156 Qubit ในโครงตาข่าย heavy-hexagonal ยังคงใช้ประโยชน์จากนวัตกรรมของ Heron รุ่นแรก พร้อมเพิ่มฟีเจอร์ TLS mitigation ใหม่ที่ควบคุมสภาพแวดล้อม TLS บนชิป ช่วยเพิ่ม coherence และความเสถียรทั่วทั้งชิป

  `r1` (ธันวาคม 2023) เวอร์ชันแรกของ Heron มี 133 Qubit

</Admonition>
