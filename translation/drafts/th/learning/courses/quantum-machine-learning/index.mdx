# ภาพรวม

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

คอร์สนี้สำรวจว่า quantum computing สามารถมีบทบาทที่น่าตื่นเต้นใน machine learning ได้อย่างไร คอร์สจะมีการทบทวน classical machine learning แต่จะถือว่าผู้เรียนมีความคุ้นเคยกับวิธีการเหล่านี้อยู่บ้าง จุดเน้นหลักคือวิธีที่ quantum computing สามารถเข้ากับ workflow ของ machine learning ที่มีอยู่แล้วได้ หัวข้อได้แก่ data encoding, quantum kernels และ variational quantum circuits
