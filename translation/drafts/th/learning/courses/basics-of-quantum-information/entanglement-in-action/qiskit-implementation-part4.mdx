#### ภาพเชิงเรขาคณิต {#geometric-picture}

สามารถมองกลยุทธ์ที่อธิบายไว้ข้างต้นในเชิงเรขาคณิตได้ ซึ่งอาจช่วยให้เข้าใจความสัมพันธ์ระหว่างมุมต่างๆ ที่ Alice และ Bob เลือกสำหรับการดำเนินการของตนได้ดีขึ้น

สิ่งที่ Alice ทำในทางปฏิบัติคือการเลือกมุม $\alpha$ ขึ้นอยู่กับคำถาม $x$ ของเธอ แล้วจึงใช้ $U_{\alpha}$ กับ Qubit ของเธอและวัดผล
ในทำนองเดียวกัน Bob เลือกมุม $\beta$ ขึ้นอยู่กับ $y$ แล้วใช้ $U_{\beta}$ กับ Qubit ของเขาและวัดผล
เราเลือก $\alpha$ และ $\beta$ ดังนี้

$$
\begin{aligned}
\alpha & = \begin{cases}
0 & x=0\\
\pi/4 & x=1
\end{cases}\\[4mm]
\beta & = \begin{cases}
\pi/8 & y = 0\\
-\pi/8 & y = 1
\end{cases}
\end{aligned}
$$

แต่ในตอนนี้ ให้สมมติว่า $\alpha$ และ $\beta$ เป็นค่าใดก็ได้
เมื่อ Alice เลือก $\alpha$ เธอกำลังกำหนดฐานออร์โธนอร์มัลของเวกเตอร์ที่มีลักษณะดังนี้

![Basis for Alice](/learning/images/courses/basics-of-quantum-information/entanglement-in-action/qiskit-implementation/alpha-basis.avif)

Bob ก็ทำเช่นเดียวกัน แต่ใช้มุม $\beta$ ของเขา

![Basis for Bob](/learning/images/courses/basics-of-quantum-information/entanglement-in-action/qiskit-implementation/beta-basis.avif)

สีของเวกเตอร์สอดคล้องกับคำตอบของ Alice และ Bob คือสีน้ำเงินสำหรับ $0$ และสีแดงสำหรับ $1$

ตอนนี้ ถ้าเรารวม ($3$) และ ($4$) เข้าด้วยกัน เราจะได้สูตร

$$
\langle \psi_{\alpha} \otimes\psi_{\beta} \vert \phi^+ \rangle
= \frac{1}{\sqrt{2}} \langle \psi_{\alpha} \vert \psi_{\beta} \rangle;
$$

ซึ่งใช้ได้กับจำนวนจริง $\alpha$ และ $\beta$ ทุกค่า

เมื่อทำการวิเคราะห์แบบเดียวกับที่เราทำข้างต้น แต่ให้ $\alpha$ และ $\beta$ เป็นตัวแปร เราจะได้ผลดังนี้

$$
  \begin{aligned}
  & \bigl(U_{\alpha} \otimes U_{\beta}\bigr) \vert \phi^+\rangle\\[1mm]
  & \qquad =
    \vert 00 \rangle \langle \psi_{\alpha} \otimes \psi_{\beta}\vert \phi^+\rangle
    + \vert 01 \rangle \langle \psi_{\alpha} \otimes\psi_{\beta + \pi/2}\vert \phi^+\rangle \\
  & \qquad \qquad + \vert 10 \rangle \langle \psi_{\alpha+\pi/2} \otimes \psi_{\beta}\vert \phi^+\rangle
    + \vert 11 \rangle \langle \psi_{\alpha+\pi/2} \otimes \psi_{\beta+\pi/2}\vert \phi^+\rangle\\[2mm]
  & \qquad = \frac{
        \langle \psi_\alpha \vert \psi_\beta \rangle \vert 00\rangle
      + \langle \psi_\alpha \vert \psi_{\beta+\pi/2} \rangle \vert 01\rangle
      + \langle \psi_{\alpha+\pi/2} \vert \psi_\beta \rangle \vert 10\rangle
      + \langle \psi_{\alpha+\pi/2} \vert \psi_{\beta+\pi/2} \rangle \vert 11\rangle
      }{\sqrt{2}}.
  \end{aligned}
$$

เราสรุปได้เป็นสองสูตรนี้

$$
\begin{aligned}
\operatorname{Pr}(a = b)  & = \frac{1}{2} \vert \langle \psi_\alpha \vert \psi_\beta \rangle \vert^2
+ \frac{1}{2} \vert \langle \psi_{\alpha+\pi/2} \vert \psi_{\beta+\pi/2} \rangle \vert^2
= \cos^2(\alpha - \beta)\\[2mm]
\operatorname{Pr}(a \neq b) & = \frac{1}{2} \vert \langle \psi_\alpha \vert \psi_{\beta+\pi/2} \rangle \vert^2
+ \frac{1}{2} \vert \langle \psi_{\alpha+\pi/2} \vert \psi_\beta \rangle \vert^2
= \sin^2(\alpha - \beta).
\end{aligned}
$$

สมการเหล่านี้สามารถเชื่อมโยงกับภาพข้างต้นได้โดยการจินตนาการว่าเราซ้อนทับฐานที่ Alice และ Bob เลือกเข้าด้วยกัน

#### สำรวจกลยุทธ์ {#explore-the-strategy}

เมื่อ $(x,y) = (0,0),$ Alice และ Bob เลือก $\alpha = 0$ และ $\beta = \pi/8$ และเมื่อเราซ้อนทับฐานของพวกเขา จะได้ภาพดังนี้

![Alice and Bob bases case 1](/learning/images/courses/basics-of-quantum-information/entanglement-in-action/qiskit-implementation/strategy00.avif)

มุมระหว่างเวกเตอร์สีแดงคือ $\pi/8$ ซึ่งเท่ากับมุมระหว่างเวกเตอร์สีน้ำเงินสองตัว
ความน่าจะเป็นที่ผลลัพธ์ของ Alice และ Bob จะตรงกันคือค่าโคไซน์กำลังสองของมุมนี้

$$
\cos^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2 + \sqrt{2}}{4},
$$

ส่วนความน่าจะเป็นที่ผลลัพธ์ไม่ตรงกันคือค่าไซน์กำลังสองของมุมนี้

$$
\sin^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2 - \sqrt{2}}{4}.
$$

เมื่อ $(x,y) = (0,1),$ Alice และ Bob เลือก $\alpha = 0$ และ $\beta = -\pi/8$ และเมื่อเราซ้อนทับฐานของพวกเขา จะได้ภาพดังนี้

มุมระหว่างเวกเตอร์สีแดงยังคงเป็น $\pi/8$ เช่นเดียวกับมุมระหว่างเวกเตอร์สีน้ำเงิน
ความน่าจะเป็นที่ผลลัพธ์ของ Alice และ Bob จะตรงกันยังคงเป็นค่าโคไซน์กำลังสองของมุมนี้

$$
\cos^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2 + \sqrt{2}}{4},
$$

ส่วนความน่าจะเป็นที่ผลลัพธ์ไม่ตรงกันคือค่าไซน์กำลังสองของมุมนี้

$$
\sin^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2 - \sqrt{2}}{4}.
$$

เมื่อ $(x,y) = (1,0),$ Alice และ Bob เลือก $\alpha = \pi/4$ และ $\beta = \pi/8$ และเมื่อเราซ้อนทับฐานของพวกเขา จะได้ภาพดังนี้

ฐานเปลี่ยนไปแต่มุมไม่เปลี่ยน — มุมระหว่างเวกเตอร์ที่มีสีเดียวกันยังคงเป็น $\pi/8$
ความน่าจะเป็นที่ผลลัพธ์ของ Alice และ Bob จะตรงกันคือ

$$
\cos^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2 + \sqrt{2}}{4},
$$

และความน่าจะเป็นที่ผลลัพธ์ไม่ตรงกันคือ

$$
\sin^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2 - \sqrt{2}}{4}.
$$

เมื่อ $(x,y) = (1,1),$ Alice และ Bob เลือก $\alpha = \pi/4$ และ $\beta = -\pi/8$ เมื่อเราซ้อนทับฐานของพวกเขา จะเห็นว่าสิ่งที่แตกต่างออกไปเกิดขึ้น

ความน่าจะเป็นที่ผลลัพธ์ของ Alice และ Bob จะตรงกันยังคงเป็นค่าโคไซน์กำลังสองของมุมนี้ แต่ค่าที่ได้คือ

$$

$$
\sin^2\Bigl(\frac{3\pi}{8}\Bigr) = \frac{2 + \sqrt{2}}{4}.
$$

### หมายเหตุ {#remarks}

แนวคิดพื้นฐานของการทดลองแบบ CHSH game ที่ entanglement นำไปสู่ผลทางสถิติที่ขัดแย้งกับการใช้เหตุผลแบบคลาสสิกล้วนๆ นั้น มาจาก John Bell ซึ่งเป็นชื่อของ Bell states
ด้วยเหตุนี้ ผู้คนจึงมักเรียกการทดลองประเภทนี้ว่า *Bell tests*
บางครั้งผู้คนยังอ้างถึง *Bell's theorem* ซึ่งสามารถระบุได้หลายรูปแบบ แต่แก่นของมันคือกลศาสตร์ควอนตัมไม่เข้ากันกับที่เรียกว่า *local hidden variable theories*
CHSH game เป็นตัวอย่างที่ชัดเจนและเรียบง่ายโดยเฉพาะของ Bell test และสามารถมองได้ว่าเป็นการพิสูจน์หรือการสาธิต Bell's theorem

CHSH game เป็นวิธีทดสอบทฤษฎีข้อมูลควอนตัมเชิงทดลอง
เราสามารถทำการทดลองที่ใช้งาน CHSH game และทดสอบกลยุทธ์ประเภทที่อิงบน entanglement ที่อธิบายไว้ข้างต้น
สิ่งนี้ทำให้เรามั่นใจในระดับสูงว่า entanglement นั้นมีจริง — และต่างจากวิธีที่คลุมเครือหรือกวีนิพนธ์ที่เราคิดขึ้นเพื่ออธิบาย entanglement CHSH game ให้วิธีที่เป็นรูปธรรมและทดสอบได้สำหรับการ *สังเกต* entanglement
รางวัลโนเบลสาขาฟิสิกส์ปี 2022 ยอมรับความสำคัญของงานในแนวทางนี้ โดยมอบรางวัลให้กับ Alain Aspect, John Clauser (ตัว C ใน CHSH) และ Anton Zeilinger สำหรับการสังเกต entanglement ผ่าน Bell tests บนโฟตอนที่พันกัน

### การใช้งานใน Qiskit {#qiskit-implementation}

เราสามารถใช้งาน CHSH game พร้อมกับกลยุทธ์ควอนตัมที่กำหนดไว้ข้างต้นใน Qiskit ได้ดังนี้

ก่อนอื่น นี่คือนิยามของเกมเอง ซึ่งอนุญาตให้เสียบกลยุทธ์ใดก็ได้เป็น argument

```python
def chsh_game(strategy):
    # This function runs the CHSH game, using the strategy (a function
    # from two bits to two bits), returning 1 for a win and 0 for a loss.

    # Choose x and y randomly
    x, y = random.randint(0, 1), random.randint(0, 1)

    # Use the strategy to determine a and b
    a, b = strategy(x, y)

    # Decide if the strategy wins or loses
    if (a != b) == (x & y):
        return 1  # Win
    return 0  # Lose
```

ตอนนี้เราจะสร้างฟังก์ชันที่ส่งออก Circuit ขึ้นอยู่กับคำถามสำหรับ Alice และ Bob เราจะให้ Qubit ใช้ชื่อดีฟอลต์เพื่อความเรียบง่าย และจะใช้ Gate $R_y(\theta)$ ที่มีอยู่แล้วสำหรับการกระทำของ Alice และ Bob

```python
def chsh_circuit(x, y):
    # This function creates a `QuantumCircuit` implementing the quantum
    # strategy described above (including the e-bit preparation).

    qc = QuantumCircuit(2, 2)

    # Prepare an e-bit
    qc.h(0)
    qc.cx(0, 1)
    qc.barrier()

    # Alice's actions
    if x == 0:
        qc.ry(0, 0)
    else:
        qc.ry(-pi / 2, 0)
    qc.measure(0, 0)

    # Bob's actions
    if y == 0:
        qc.ry(-pi / 4, 1)
    else:
        qc.ry(pi / 4, 1)
    qc.measure(1, 1)

    return qc
```

ตอนนี้เราจะสร้าง job โดยใช้ Aer simulator ที่รัน Circuit หนึ่งครั้งสำหรับคู่อินพุต $(x,y)$ ที่กำหนด

```python
def quantum_strategy(x, y):
    # This function runs the appropriate quantum circuit defined above
    # one time and returns the measurement results

    # Setting `shots=1` to run the circuit once
    result = AerSimulator().run(chsh_circuit(x, y), shots=1).result()
    statistics = result.get_counts()

    # Determine the output bits and return them
    bits = list(statistics.keys())[0]
    a, b = bits[0], bits[1]
    return a, b
```

NUM_GAMES = 1000

เราสามารถกำหนดกลยุทธ์แบบคลาสสิกและดูว่ามันทำงานได้ดีแค่ไหน
นี่เป็นเพียงกลยุทธ์หนึ่ง — สามารถทดสอบกลยุทธ์อื่นๆ ได้โดยการเปลี่ยนโค้ด — แต่เป็นหนึ่งในกลยุทธ์คลาสสิกที่เหมาะสมที่สุด

```python
def classical_strategy(x, y):
    # This function implements just one example of an optimal classical
    # strategy for the CHSH game. Other classical strategies can be
    # implemented by changing the bit values assigned to a and b.

    # Alice's answer
    if x == 0:
        a = 0
    elif x == 1:
        a = 1

    # Bob's answer
    if y == 0:
        b = 1
    elif y == 1:
        b = 0

    return a, b
```

มาเล่นเกม 1000 ครั้งอีกครั้งเพื่อดูว่ามันทำงานได้ดีแค่ไหน

```python
NUM_GAMES = 1000
TOTAL_SCORE = 0

for _ in range(NUM_GAMES):
    TOTAL_SCORE += chsh_game(classical_strategy)

print("Fraction of games won:", TOTAL_SCORE / NUM_GAMES)
```
