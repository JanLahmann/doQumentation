# ภาพรวม

!["Hero image"](/learning/images/courses/quantum-chem-with-vqe/hero.svg)

ยินดีต้อนรับสู่ **Quantum chemistry with VQE** variational quantum eigensolver (VQE) คือ quantum algorithm ที่มีการประยุกต์ใช้ในเคมีควอนตัมและการจำลอง ในคอร์สเบื้องต้นนี้ คุณจะได้เรียนรู้ building blocks พื้นฐานของ VQE, options ในการ implement และการประยุกต์ใช้บางอย่าง เนื้อหาส่งถึงผู้เรียนผ่านการผสมผสานระหว่างข้อความ วิดีโอ และโค้ด

หัวข้อหลักที่ครอบคลุม:

    - Hamiltonians
    - Ansaetze
    - Classical optimizers
    - การหาพลังงาน ground state โดยใช้ Qiskit patterns
    - การหาความยาวพันธะโดยใช้ Qiskit patterns

คอร์สนี้เน้นที่การประยุกต์ใช้กับเคมีควอนตัม แม้ว่า building blocks เดียวกันหลายอย่าง (ansatz circuits, classical optimizers และ Qiskit patterns framework) สามารถนำไปใช้กับสาขาอื่น ๆ ที่หลากหลายนอกเหนือจากเคมีควอนตัม

## แหล่งเรียนรู้เพิ่มเติม

บทนำสู่ VQE นี้ออกแบบสำหรับผู้เรียนระดับกลางที่ต้องการฝึกทักษะเทคนิคการคำนวณควอนตัม ไม่ต้องมีพื้นฐาน VQE มาก่อน แต่ควรคุ้นเคยกับ Qiskit และกลศาสตร์ควอนตัมกับคณิตศาสตร์ที่เกี่ยวข้อง (เช่น operators) ในระดับหนึ่ง เราแนะนำให้สละเวลาประมาณ 6-10 ชั่วโมงในการทำเนื้อหานี้ให้เสร็จสมบูรณ์ ขึ้นอยู่กับประสบการณ์ก่อนหน้า

    - คอร์ส [Basics of Quantum Information](/learning/courses/basics-of-quantum-information/index) อาจมีประโยชน์ในการให้พื้นฐานและหลักการของ quantum information และการคำนวณ
