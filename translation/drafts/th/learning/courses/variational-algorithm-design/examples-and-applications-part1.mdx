## นิยามปัญหา {#problem-definitions}

สมมติว่าเราต้องการใช้ variational algorithm หาค่า eigenvalue ของ observable ต่อไปนี้:

$$
\hat{O}_1 = 2 II - 2 XX + 3 YY - 3 ZZ,
$$

observable นี้มีค่า eigenvalue ดังนี้:

$$
\left\{
\begin{array}{c}
\lambda_0 = -6 \\
\lambda_1 = 4 \\
\lambda_2 = 4 \\
\lambda_3 = 6
\end{array}
\right\}
$$

และ eigenstate:

$$
\left\{
\begin{array}{c}
|\phi_0\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)\\
|\phi_1\rangle = \frac{1}{\sqrt{2}}(|00\rangle - |11\rangle)\\
|\phi_2\rangle = \frac{1}{\sqrt{2}}(|01\rangle - |10\rangle)\\
|\phi_3\rangle = \frac{1}{\sqrt{2}}(|01\rangle + |10\rangle)
\end{array}
\right\}
$$

## Custom VQE {#custom-vqe}

เราจะสำรวจวิธีสร้าง VQE instance ด้วยตัวเองก่อน เพื่อหาค่า eigenvalue ต่ำสุดของ $\hat{O}_1$ ซึ่งจะรวมเทคนิคหลากหลายที่เราเรียนมาตลอดคอร์สนี้

```python
def cost_func_vqe(params, ansatz, hamiltonian, estimator):
    """Return estimate of energy from estimator

    Parameters:
        params (ndarray): Array of ansatz parameters
        ansatz (QuantumCircuit): Parameterized ansatz circuit
        hamiltonian (SparsePauliOp): Operator representation of Hamiltonian
        estimator (Estimator): Estimator primitive instance

    Returns:
        float: Energy estimate
    """
    pub = (ansatz, hamiltonian, params)
    cost = estimator.run([pub]).result()[0].data.evs

    return cost
```

```python
from qiskit.circuit.library.n_local import n_local
from qiskit import QuantumCircuit

import numpy as np

reference_circuit = QuantumCircuit(2)
reference_circuit.x(0)

variational_form = n_local(
    num_qubits=2,
    rotation_blocks=["rz", "ry"],
    entanglement_blocks="cx",
    entanglement="linear",
    reps=1,
)

raw_ansatz = reference_circuit.compose(variational_form)
raw_ansatz.decompose().draw("mpl")
```

### เพิ่ม reference state {#add-reference-state}

### เปลี่ยน initial point {#change-the-initial-point}
เมื่อเห็นผลของการเพิ่ม reference state แล้ว เราจะสำรวจผลที่เกิดขึ้นเมื่อเลือก initial point $\vec{\theta_0}$ ต่างกัน โดยเฉพาะอย่างยิ่งเราจะใช้ $\vec{\theta_0}=(0,0,0,0,6,0,0,0)$ และ $\vec{\theta_0}=(6,6,6,6,6,6,6,6,6)$

จำไว้ว่า ตามที่กล่าวถึงตอนแนะนำ reference state solution ที่ ideal จะพบเมื่อ parameter ทั้งหมดเป็น $0$ ดังนั้น initial point แรกควรใช้จำนวน evaluation น้อยกว่า

```python
import time

start_time = time.time()

x0 = [0, 0, 0, 0, 6, 0, 0, 0]

x0_1_result = minimize(
    cost_func_vqe, x0, args=(raw_ansatz, observable_1, estimator), method="COBYLA"
)

end_time = time.time()
execution_time = end_time - start_time
```

```python
print("INITIAL POINT 1:")

print(f"""Time (s): {execution_time}""")
```

### ทดลองกับ optimizer ต่าง ๆ {#experimenting-with-different-optimizers}

เราสามารถปรับ optimizer โดยใช้ argument `method` ของ SciPy `minimize` โดยดูตัวเลือกเพิ่มเติม[ได้ที่นี่](https://docs.scipy.org/doc/scipy/reference/generated/scipy.optimize.minimize.html) เดิมเราใช้ constrained minimizer (`COBYLA`) ในตัวอย่างนี้ เราจะลองใช้ unconstrained minimizer (`BFGS`) แทน

```python
import time

start_time = time.time()

result = minimize(
    cost_func_vqe, x0, args=(raw_ansatz, observable_1, estimator), method="BFGS"
)

end_time = time.time()
execution_time = end_time - start_time
```

```python
print("CHANGED TO BFGS OPTIMIZER:")
print(f"""Number of iterations: {result.nfev}""")
print(f"""Time (s): {execution_time}""")
```

```text
CHANGED TO BFGS OPTIMIZER:
Number of iterations: 117
Time (s): 0.31656408309936523
```
