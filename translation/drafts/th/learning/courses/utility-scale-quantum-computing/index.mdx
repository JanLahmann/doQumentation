# ภาพรวม

!["Hero image"](/learning/images/courses/utility-scale-quantum-computing/hero.svg)

คอร์ส event replay นี้ประกอบด้วย 14 บทเรียนและแล็บที่พัฒนาและดำเนินการโดย IBM Quantum&reg; ร่วมกับมหาวิทยาลัยโตเกียว บทเรียนเหล่านี้ครอบคลุมหัวข้อสำคัญหลากหลายด้านการประมวลผลเชิงควอนตัม โดยมุ่งเน้นไปที่การสร้างการคำนวณในระดับ utility-scale เป็นหลัก ผลลัพธ์ที่ได้มีความใกล้เคียงกับงานวิจัยที่ตีพิมพ์บนหน้าปก Nature เมื่อเดือนมิถุนายน 2023
