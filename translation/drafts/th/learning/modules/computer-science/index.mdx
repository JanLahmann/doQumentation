# ภาพรวม

!["Hero image"](/learning/images/modules/computer-science/hero.avif)

คอลเลกชัน Qiskit in-classroom modules นี้ออกแบบมาสำหรับหลักสูตรระดับปริญญาตรีในสาขาวิทยาการคอมพิวเตอร์ การประมวลผลควอนตัม หรือกลศาสตร์ควอนตัมเบื้องต้น เป้าหมายคือช่วยให้ผู้สอนและนักเรียนสามารถใช้คอมพิวเตอร์ควอนตัมในการสำรวจโปรโตคอลและอัลกอริทึมควอนตัมพื้นฐานด้วยการทดลองจริง เช่น quantum teleportation, quantum key distribution (QKD) และอัลกอริทึม Deutsch-Jozsa และ Bernstein-Vazirani

แต่ละโมดูลควรเริ่มต้นในชั้นเรียนโดยมีการทำงานเป็นกลุ่มและการสนับสนุนจากผู้สอน แต่สามารถทำให้เสร็จได้อย่างอิสระหรือในกลุ่มเล็ก กิจกรรมทั้งหมดสามารถทำเสร็จได้ในเวลาที่น้อยกว่าเวลาฟรีที่กำหนดไว้ภายใต้ Open Plan ที่เปิดให้ผู้ใช้ทุกคน นอกจากนี้ยังมีตัวเลือก Simulator ในตัวสำหรับกรณีที่นักเรียนใช้เวลาการประมวลผลควอนตัมฟรีหมดโดยบังเอิญ

ผู้สอนสามารถขอเวอร์ชันของ notebooks เหล่านี้พร้อมเฉลยและแนวทางการจัดวางในหลักสูตรทั่วไปได้โดยการกรอก[แบบสอบถามสั้นๆ](https://ibm.biz/classrooms_instructor_key_request) นี้เกี่ยวกับวิธีการใช้งาน notebooks

## พื้นฐานที่แนะนำ {#recommended-background}

แต่ละโมดูลมีคำแนะนำว่าควรใช้โมดูลนั้นที่จุดใดในหลักสูตรจึงจะมีประสิทธิภาพสูงสุด โดยทั่วไปแล้ว นักเรียนจะได้รับประโยชน์สูงสุดจากโมดูลเหล่านี้หากมีพื้นฐานดังนี้:

- มีพื้นฐานพีชคณิตเชิงเส้นบ้าง (เมทริกซ์ $2 \times 2$ น่าจะเพียงพอ)
- มีความคุ้นเคยกับ Python บ้าง แม้ว่า notebooks ส่วนใหญ่จะสามารถรันได้ตามที่เขียนไว้
