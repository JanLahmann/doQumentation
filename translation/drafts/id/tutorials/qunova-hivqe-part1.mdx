## Latar Belakang {#background}

Menghitung energi reaksi kimia secara akurat sangat penting untuk kemajuan ilmiah di bidang ilmu material, rekayasa kimia, penemuan obat, dan bidang lainnya. Di antara berbagai sistem kimia, sistem Li-S mendapat perhatian besar dalam memahami dan mengembangkan komposisi baterai baru. Tutorial ini memberikan pengalaman langsung dalam menghitung potential energy surface (PES) disosiasi ikatan Li-S dari sistem $Li_2S$ dengan menghapus satu atom litium menggunakan perhitungan HiVQE. Hasilnya dapat dibandingkan dengan perhitungan referensi (CASCI) maupun metode klasik seperti Hartree-Fock (HF) untuk masalah 20 Qubit.

Perhitungan HiVQE akan dilakukan dengan opsi yang didefinisikan di bawah ini. Menggunakan basis sto3g untuk $Li_2S$, terdapat 19 orbital spasial dengan 22 elektron. Untuk menjalankan kasus (10o,10e) dengan perhitungan HiVQE, kamu dapat mendefinisikan 10 orbital aktif dan enam orbital beku. Pada setiap iterasi, 100 shots akan digunakan untuk men-sample konfigurasi elektron yang dihasilkan oleh Circuit kuantum ExcitationPreserving (`epa`) dengan entanglement `circular` dan dua pengulangan (`reps`). Jumlah maksimum iterasi diatur menjadi 30 untuk memastikan terminasi iterasi dengan konvergensi energi.

```python
molecule_options = {
    "basis": "sto3g",
    "active_orbitals": list(range(5, 15)),
    "frozen_orbitals": list(range(5)),
}

hivqe_options = {
    "shots": 100,
    "max_iter": 30,
    "ansatz": "epa",
    "ansatz_entanglement": "circular",
    "ansatz_reps": 2,
}
```

### Langkah 2 dan 3: Optimalkan masalah untuk eksekusi pada hardware kuantum dan jalankan menggunakan fungsi HiVQE Chemistry {#langkah-2-dan-3-optimalkan-masalah-untuk-eksekusi-pada-hardware-kuantum-dan-jalankan-menggunakan-fungsi-hivqe-chemistry}

Siapkan loop `for` untuk menjalankan perhitungan HiVQE dengan geometri dan opsi yang telah didefinisikan. Job-job disubmit dalam loop `for`. Dalam tutorial ini, kamu akan mensubmit enam geometri dan mengambil hasilnya ketika semuanya sudah selesai. Dalam eksekusi fungsi utama, kamu perlu mendefinisikan `max_states` dan `max_expansion_states` untuk mengontrol ukuran maksimum matriks subspace dan mengontrol berapa banyak state yang dapat dihasilkan menggunakan metode ekspansi CI klasik per iterasi. ID job fungsi akan disimpan dalam dictionary dengan label geometri masing-masing untuk melacak dan memproses output lebih lanjut.

```python
info_jobid = {}
for dis, geom in str_geometries.items():
    hivqe_run = hivqe.run(
        geometry=geom,
        backend_name="",
        max_states=40000,
        max_expansion_states=100,
        molecule_options=molecule_options,
        hivqe_options=hivqe_options,
    )
    status = hivqe_run.status()
    info_jobid[dis] = hivqe_run.job_id
print(info_jobid)
```

```python
rhf_result = {}
casci_result = {}

cas_list = molecule_options["active_orbitals"]
distance_ref = []
for dis, geom in str_geometries.items():
    distance_ref.append(dis)
    mole = gto.M(atom=geom, basis=molecule_options["basis"])
    mole.verbose = 0
    # RHF energy
    mf = scf.RHF(mole).run()
    mo_occ = mf.mo_occ
    num_elecs_as = int(sum([mo_occ[idx] for idx in cas_list]))

    rhf_result[str(dis)] = mf.e_tot
    # CASCI energy
    casci_solver = mcscf.CASCI(mf, len(cas_list), num_elecs_as)
    orbs = mcscf.addons.sort_mo(casci_solver, mf.mo_coeff, cas_list, base=0)
    casci_solver.kernel(orbs)
    casci_result[str(dis)] = casci_solver.e_tot
    print(
        f"d={dis:4.3} RHF Energy: {mf.e_tot:14.10}, CASCI Energy: {casci_solver.e_tot:14.10}"
    )
```
