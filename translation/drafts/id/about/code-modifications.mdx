# Modifikasi Kode Otomatis {#automatic-code-modifications}

doQumentation secara otomatis menerapkan sejumlah kecil modifikasi pada konten
tutorial dan panduan Qiskit dari sumber aslinya untuk memastikan pengalaman
yang lancar dan interaktif. Halaman ini mendokumentasikan setiap modifikasi
supaya kamu bisa memahami persis apa yang berubah dibandingkan dokumentasi
IBM Quantum yang asli.

## Salinan notebook (Buka di Colab / Binder / Code Engine) {#notebook-copies-open-in-colab-binder-code-engine}

Saat kamu klik **Open in Colab**, **Open in JupyterLab**, atau **Open in Code
Engine**, kamu akan menerima salinan notebook asli dengan tambahan berikut:

### 1. Sel pemberitahuan setup (markdown) {#1-setup-notice-cell-markdown}

Sebuah sel blockquote disisipkan di bagian paling atas yang menjelaskan bahwa
doQumentation telah menambahkan sel setup otomatis. Sel ini juga menyertakan
tautan kembali ke halaman ini.

### 2. Sel prasyarat (kode) {#2-prerequisites-cell-code}

Sebuah sel kode disisipkan setelah pemberitahuan yang:

- **Menginstal paket-paket yang dibutuhkan** (`qiskit`, `qiskit-aer`, `qiskit-ibm-runtime`,
  `pylatexenc`, ditambah paket-paket spesifik tutorial yang terdeteksi melalui
  pemindaian impor). Instalasi akan **dilewati** jika paket sudah ada
  (misalnya di Binder atau Code Engine yang sudah pre-installed).
- **Menyediakan template kredensial yang dikomentari** untuk IBM Quantum, supaya
  pengguna yang ingin menjalankan di hardware nyata bisa mengaktifkan komentar
  dan mengisi API key mereka.

Di Google Colab, sel ini otomatis berjalan saat notebook dibuka melalui flag
metadata `cell_execution_strategy: setup`.

### 3. Penulisan ulang path gambar {#3-image-path-rewrites}

Path gambar relatif (`/docs/images/...`, `/learning/images/...`) ditulis ulang
supaya bisa berfungsi dengan benar di lingkungan notebook yang berdiri sendiri.

## Halaman MDX (rendering di browser) {#mdx-pages-in-browser-rendering}

Tutorial yang ditampilkan di website ini dikonversi dari notebook `.ipynb`
atau file `.mdx` dari sumber aslinya. Transformasi berikut diterapkan:

- **Baris `pip install`** ditambahkan ke blok kode Python yang mengimpor
  paket pihak ketiga, memungkinkan eksekusi satu klik melalui thebelab.
- **Bagian IBM Tutorial Survey**: Sebuah catatan ditambahkan yang menjelaskan
  bahwa survei tersebut milik IBM Quantum dan menyertakan tautan ke GitHub Issues
  doQumentation untuk umpan balik terkait situs.
- **Widget umpan balik**: Widget "Was this helpful?" ditambahkan di bagian bawah
  setiap tutorial, dilacak melalui analitik Umami yang ramah privasi.
- **Perbaikan sintaks MDX**: Kurung kurawal, hierarki heading, dan masalah
  kompatibilitas JSX diperbaiki secara otomatis untuk rendering Docusaurus.
- **OpenInLabBanner**: Banner interaktif disisipkan di bawah judul dengan
  tombol untuk membuka notebook di Colab, Binder, atau Code Engine.

## Yang TIDAK dimodifikasi {#what-is-not-modified}

- **Konten tutorial itu sendiri** (penjelasan, logika kode, output) tidak pernah
  diubah.
- **Atribusi penulis asli** dipertahankan melalui frontmatter dan file NOTICE
  (lisensi Apache 2.0 / CC BY-SA 4.0).
- **Tidak ada kode telemetri atau pelacakan** yang disisipkan ke dalam notebook.
  Analitik (Umami) hanya berjalan di website doQumentation, bukan di notebook
  yang diekspor.
