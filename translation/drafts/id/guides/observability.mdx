# Event pelacakan aktivitas untuk Qiskit Runtime

Layanan IBM Cloud&reg;, seperti Qiskit Runtime, menghasilkan event pelacakan aktivitas.

Event pelacakan aktivitas melaporkan aktivitas yang mengubah status layanan di IBM Cloud. Kamu bisa menggunakan event ini untuk menyelidiki aktivitas yang tidak biasa, tindakan kritis, dan memenuhi persyaratan audit regulasi.

Kamu bisa menggunakan IBM Cloud Activity Tracker Event Routing, sebuah layanan platform, untuk mengarahkan event audit di akunmu ke tujuan yang kamu pilih dengan mengonfigurasi target dan rute yang menentukan ke mana event pelacakan aktivitas dikirim. Untuk informasi lebih lanjut, lihat [Tentang IBM Cloud Activity Tracker Events Routing.](https://cloud.ibm.com/docs/atracker?topic=atracker-about)

Kamu bisa menggunakan IBM Cloud Logs untuk memvisualisasikan dan memberi peringatan pada event yang dihasilkan di akunmu dan diarahkan oleh IBM Cloud Activity Tracker Event Routing ke instans IBM Cloud Logs.

## Lokasi pembuatan event pelacakan aktivitas {#locations-where-activity-tracking-events-are-generated}

Qiskit Runtime mengirim event pelacakan aktivitas di region berikut:

- Washington ("us-east")
- Frankfurt ("eu-de")

## Melihat event pelacakan aktivitas untuk Qiskit Runtime {#viewing-activity-tracking-events-for-qiskit-runtime}

Kamu bisa menggunakan IBM Cloud Logs untuk memvisualisasikan dan memberi peringatan pada event yang dihasilkan di akunmu dan diarahkan oleh IBM Cloud Activity Tracker Event Routing ke instans IBM Cloud Logs.
