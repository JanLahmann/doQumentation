# Lihat biaya

Kapan saja, kamu bisa melihat berapa banyak biaya yang sudah dikeluarkan oleh workload yang terkait dengan sebuah instance.

## Lihat biaya instance {#view-instance-cost}

Untuk mengetahui seberapa banyak sebuah instance sudah digunakan, dari [halaman Instances](https://quantum.cloud.ibm.com/instances), klik instance tersebut untuk membuka halaman detailnya. Kamu bisa klik tautan yang ada di **Billing overview** untuk detail biaya dan penagihan secara lengkap.

**Instance usage** menampilkan nilai yang berbeda, tergantung jenis paket yang terkait dengan instance tersebut:

- Untuk paket Premium dan Open, ini adalah penggunaan real-time selama 28 hari terakhir (bergulir), hingga waktu saat ini pada hari ini.
- Untuk paket Pay-As-You-Go, ini adalah semua penggunaan sejak instance dibuat.
- Untuk paket Flex, ini adalah penggunaan sejak awal kontrak.

## Lihat penggunaan workload {#view-workload-usage}

Untuk mengetahui seberapa banyak penggunaan yang dikeluarkan oleh workload tertentu, buka [halaman Workloads](https://quantum.cloud.ibm.com/workloads) dan lihat kolom "Usage" untuk workload tersebut.
