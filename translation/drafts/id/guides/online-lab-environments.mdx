# Lingkungan lab online

Selain mengunduh dan menginstal Qiskit untuk bekerja di mesin lokalmu, ada beberapa opsi tersedia bagi yang lebih suka bekerja di lingkungan pengembangan online.

Beberapa penyedia menawarkan lingkungan Jupyter online dengan Qiskit yang sudah terinstal untuk onboarding yang cepat dan praktis. Penyedia yang direkomendasikan ini termasuk penyedia layanan cloud OVHcloud dan platform cloud pengembangan perangkat lunak kuantum qBraid.

## Lingkungan notebook yang direkomendasikan dengan Qiskit sudah terinstal {#recommended-notebook-environments-with-qiskit-preinstalled}

Lingkungan Jupyter notebook online lainnya seperti IBM Watson&reg; Studio, Google Colab, dan Microsoft Azure Machine Learning Studio bisa digunakan untuk bekerja dengan Qiskit dengan mengikuti instruksi yang sesuai.

### Notebook AI OVHcloud {#ovhcloud-ai-notebooks}

Pengguna OVHcloud Public Cloud dan OVHcloud Startup dapat menggunakan notebook kuantum OVHcloud mereka untuk mengakses Qiskit dan mengirimkan job ke QPU IBM&reg; dengan akun IBM Quantum&reg; atau IBM Cloud&reg; mereka. Untuk informasi lebih lanjut, kunjungi halaman [Dokumentasi dan tutorial OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Notebook AI OVHcloud tidak tersedia di beberapa wilayah, seperti Amerika Serikat. Selain itu, antarmuka pengguna mungkin terlihat berbeda di wilayah yang berbeda.
</Admonition>

Ikuti langkah-langkah berikut untuk memulai:

1. [Buat akun IBM Cloud.](./cloud-setup)
1. [Siapkan akun OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Buat proyek OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Ikuti langkah-langkah di [Luncurkan AI Notebook pertamamu,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) pilih **Qiskit** untuk framework-nya.

### qBraid Lab {#qbraid-lab}

Pengguna Qiskit dapat menggunakan lingkungan Python yang sudah dikonfigurasi dari qBraid dengan mengikuti instruksi berikut. Untuk informasi lebih lanjut, lihat [Dokumentasi qBraid.](https://docs.qbraid.com/home/introduction)

1. [Buat akun gratis qBraid.](https://docs.qbraid.com/home/account)
3. [Buat lingkungan Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Luncurkan qBraid Lab, klik ikon **ENVS** di sebelah kanan, klik **+ ADD** di bagian atas sidebar baru, dan cari Qiskit. Klik versi yang ingin kamu gunakan, lalu klik **Install**.
4. Buat Jupyter notebook dengan mengklik **Python 3 [Qiskit]** di bagian Notebook.
4. Kamu bisa menginstal paket Python tambahan (seperti Qiskit v1.2) di sel notebook dengan menjalankan `%pip install [package_name].` Untuk informasi lebih lanjut, lihat topik [Perintah Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Lingkungan notebook tambahan untuk pengguna Qiskit {#additional-notebook-environments-for-qiskit-users}

Untuk platform tanpa Qiskit yang sudah dikonfigurasi, instruksi pengaturannya serupa, kecuali kamu harus menginstal Qiskit secara manual. Kamu akan membuat akun, memulai proyek baru (atau notebook atau workspace, tergantung platformnya), dan menginstal Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Siapkan uji coba gratis Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Buat Proyek Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Untuk menginstal paket Qiskit dan dependensi yang dibutuhkan, buka notebook baru dan jalankan perintah berikut:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Buat akun Google.](https://support.google.com/accounts/answer/27441?hl)
2. Buka [halaman landing Google Colab](https://colab.research.google.com/) dan masuk dengan akun Google-mu.
3. [Buat notebook Colab](https://colab.research.google.com/#create=true) dan instal Qiskit dengan menjalankan perintah berikut:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Buat akun Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Buat workspace.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Buat Jupyter notebook di workspace-mu.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Dari notebook, jalankan perintah ini untuk menginstal Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
