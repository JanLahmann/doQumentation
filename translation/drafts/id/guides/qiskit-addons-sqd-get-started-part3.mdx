## Dapatkan sampel dari ansatz {#obtain-samples-from-an-ansatz}

Setelah informasi molekul dimuat, kamu perlu menentukan Circuit ansatz dan mengoptimalkannya untuk eksekusi. Ini mencakup mempertimbangkan simetri masalahmu, pilihan ansatz, serta mengoptimalkannya agar berjalan di perangkat keras kuantum.

Namun, pilihan-pilihan ini bersifat spesifik terhadap masalah dan berada di luar cakupan panduan ini. Sebagai gantinya, untuk mendemonstrasikan alur kerja SQD, kita akan membuat sekumpulan counts acak untuk diproses oleh loop pemulihan konfigurasi. Ini mensimulasikan data pengukuran dari Circuit 32-qubit yang disampling dengan 10.000 shots. Setelah data count dibuat, metode `recover_configurations()` mengharuskanmu mengonversi data count menjadi matriks bitstring yang diukur di setiap shot, serta array probabilitas untuk setiap state yang diukur.

## Jalankan loop pemulihan konfigurasi {#run-configuration-recovery-loop}

Sampel pengukuran kemudian dapat diperhalus dengan mengulangi pemulihan konfigurasi dan mendiagonalisasi sekumpulan sub-sampel untuk mendekati ground state hingga konvergen.

Pertama, tentukan opsi-opsi berikut:

- `ITERATIONS`: Jumlah iterasi pemulihan konfigurasi yang konsisten-diri
- `NUM_BATCHES`: Jumlah kumpulan sub-sampel yang akan didiagonalisasi
- `SAMPLES_PER_BATCH`: Jumlah sampel yang disertakan di setiap batch
- `MAX_DAVIDSON_CYCLES`: Jumlah iterasi untuk solver eigenstate

```python
# SQD options
ITERATIONS = 5

# Eigenstate solver options
NUM_BATCHES = 1
SAMPLES_PER_BATCH = 500
MAX_DAVIDSON_CYCLES = 200
```

Selanjutnya, untuk memplot konvergensi, definisikan array untuk menyimpan aproksimasi energi ground state, nilai ekspektasi dari $\langle S \rangle ^2$, dan ocupansi orbital molekul.

```python
# Self-consistent configuration recovery loop
energy_hist = np.zeros((ITERATIONS, NUM_BATCHES))  # energy history
spin_sq_hist = np.zeros((ITERATIONS, NUM_BATCHES))  # spin history
occupancy_hist = []
avg_occupancy = None
```

Sekarang, jalankan loop pemulihan konfigurasi. Setiap loop terdiri dari tiga langkah:

1. Gunakan fungsi `recover_configurations()` untuk mendapatkan matriks bitstring yang diperhalus dan array probabilitas berdasarkan rata-rata ocupansi orbital.
1. Gunakan `postselect_and_subsample()` untuk mengumpulkan batch sub-sampel yang akan didiagonalisasi.
1. Kemudian gunakan batch sub-sampel sebagai argumen ke fungsi `solve_fermion()` untuk mendapatkan aproksimasi dari ground state.

Penting untuk memperhatikan cara menangani iterasi pertama dari loop pemulihan konfigurasi. Karena rata-rata ocupansi orbital belum tersedia, hanya fungsi `postselect_and_subsample()` yang dipanggil. Ini menghapus sampel yang tidak fisis (sampel dengan Hamming weight yang salah) sebelum menjalankan solver eigenstate, `solve_fermion()`. Setelah itu, rata-rata ocupansi orbital dihitung di semua batch dan digunakan sebagai input ke metode `recover_configurations()`, yang membalik bit individual secara probabilistik berdasarkan rata-rata ini. Lihat Bagian **II-A** dari informasi tambahan dalam [paper SQD](https://arxiv.org/abs/2405.05068) untuk informasi lebih lanjut.

```python
from qiskit_addon_sqd.configuration_recovery import recover_configurations
from qiskit_addon_sqd.subsampling import postselect_and_subsample
from qiskit_addon_sqd.fermion import (
    bitstring_matrix_to_ci_strs,
    solve_fermion,
)

for i in range(ITERATIONS):
    print(f"\nStarting configuration recovery iteration {i}")
    # On the first iteration, we have no orbital occupancy information from the
    # solver, so we just post-select from the full bitstring set based on Hamming weight.
    if avg_occupancy is None:
        bitstring_matrix_tmp = bitstring_matrix_full
        probs_array_tmp = probs_array_full

    # If there is average orbital occupancy information, use it to refine the full set of noisy configurations
    else:
        bitstring_matrix_tmp, probs_array_tmp = recover_configurations(
            bitstring_matrix_full,
            probs_array_full,
            avg_occupancy,
            num_alpha,
            num_beta,
            rand_seed=rng,
        )

    # Throw out configurations with an incorrect particle number in either the spin-up or spin-down systems
    batches = postselect_and_subsample(
        bitstring_matrix_tmp,
        probs_array_tmp,
        hamming_right=num_alpha,
        hamming_left=num_beta,
        samples_per_batch=SAMPLES_PER_BATCH,
        num_batches=NUM_BATCHES,
        rand_seed=rng,
    )

    # Run eigenstate solvers in a loop. This loop should be parallelized for larger problems.
    e_tmp = np.zeros(NUM_BATCHES)
    s_tmp = np.zeros(NUM_BATCHES)
    occs_tmp = []
    coeffs = []
    for j in range(NUM_BATCHES):
        strs_a, strs_b = bitstring_matrix_to_ci_strs(batches[j])
        print(f"Batch {j} subspace dimension: {len(strs_a) * len(strs_b)}")
        energy_sci, coeffs_sci, avg_occs, spin = solve_fermion(
            batches[j],
            core_hamiltonian,
            electron_repulsion_integrals,
            open_shell=open_shell,
            spin_sq=spin_sq,
            max_cycle=MAX_DAVIDSON_CYCLES,
        )
        energy_sci += nuclear_repulsion_energy
        e_tmp[j] = energy_sci
        s_tmp[j] = spin
        occs_tmp.append(avg_occs)
        coeffs.append(coeffs_sci)

    # Combine batch results
    avg_occupancy = tuple(np.mean(occs_tmp, axis=0))

    # Track optimization history
    energy_hist[i, :] = e_tmp
    spin_sq_hist[i, :] = s_tmp
    occupancy_hist.append(avg_occupancy)
    print()
```

### Visualisasikan hasilnya {#visualize-the-results}

Terakhir, hasilnya dapat divisualisasikan dengan memeriksa energi yang diaproksimasi dan rata-rata ocupansi orbital di setiap iterasi dari loop pemulihan konfigurasi. Plot pertama menunjukkan bahwa setelah beberapa iterasi, energi ground state diestimasi dalam kisaran sekitar `27 mH` (akurasi kimia umumnya diterima sebagai `1 kcal/mol` $\approx$ `1.6 mH`). Ingat bahwa demonstrasi ini menggunakan noise murni, dan kemampuan untuk mendekati energi hingga tingkat ini berasal dari pengetahuan sebelumnya tentang molekul dan struktur elektroniknya.

Plot kedua menunjukkan rata-rata ocupansi setiap orbital spasial setelah iterasi terakhir. Perhatikan bahwa elektron spin-up maupun spin-down keduanya menempati lima orbital pertama dengan probabilitas tinggi dalam solusinya.

import matplotlib.pyplot as plt
