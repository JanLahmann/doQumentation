# Gunakan IBM Cloud Platform APIs untuk mengakses instans

Panduan ini ditujukan bagi siapa saja yang menggunakan IBM Cloud Platform APIs untuk menyediakan, memperbarui, atau mengelola akses ke instans. Panduan ini memberikan detail tentang [parameter](#parameters) yang bisa kamu atur saat menyediakan instans, serta kumpulan [kolom ekstensi](#extensions) yang dikembalikan saat kamu mengakses suatu instans.

<Admonition type="note">
Lihat dokumentasi IBM Cloud&reg; ResourceController API untuk informasi lebih lanjut.

Beberapa halaman yang relevan:
- [Buat (sediakan) instans resource baru](https://cloud.ibm.com/apidocs/resource-controller/resource-controller#create-resource-instance)
- [Perbarui instans resource](https://cloud.ibm.com/apidocs/resource-controller/resource-controller#update-resource-instance)
- [Dapatkan instans resource](https://cloud.ibm.com/apidocs/resource-controller/resource-controller#get-resource-instance)
- [Dapatkan daftar semua instans resource](https://cloud.ibm.com/apidocs/resource-controller/resource-controller#list-resource-instances)

    Saat mendapatkan daftar semua instans resource, perlu diketahui bahwa kamu bisa memfilter hanya instans Qiskit Runtime dengan `resource_id = "b6049020-80f4-11eb-a0f7-e35ec9b4054f"`

</Admonition>

## `parameters` {#parameters}

Dalam body saat menyediakan atau memperbarui instans:

- **instance_limit_seconds** (string atau null, `^[0-9]+$`): Batas yang ditetapkan pada instans. Untuk Open Plan: jika dihilangkan, nilai defaultnya adalah 10 menit.
.
- **usage_allocation_seconds** (string atau null, `^[0-9]+$`): Waktu yang dialokasikan untuk instans, digunakan oleh penjadwal fair-share untuk menentukan prioritas dalam antrean, berdasarkan penggunaan semua QPU. (Ini tidak berlaku untuk instans Pay-As-You-Go.)
- **backends** (array berisi daftar backend (string atau null) atau array dengan "ANY"): daftar izin nama Backend yang tersedia untuk instans ini. Default: `["ANY"]`, artinya semua Backend tersedia di paket ini. `[]` berarti tidak ada Backend.

## `extensions` {#extensions}

Dalam respons saat menyediakan atau saat mendapatkan detail suatu instans:

Properti instans tambahan yang disumbangkan oleh layanan dan direpresentasikan sebagai pasangan kunci-nilai. Properti ini tetap ada bersama instans di bawah properti extensions.

- **instance_limit_seconds** (integer atau null, `>= 0`): Batas waktu yang ditetapkan pada instans.
- **usage_allocation_seconds** (integer atau null, `>= 0`): Waktu yang dialokasikan untuk instans, digunakan oleh penjadwal fair-share untuk menentukan prioritas dalam antrean, berdasarkan penggunaan semua QPU.
- **backends** (Wajib; array berisi daftar backend (string) atau array dengan "ANY" (string)): Daftar izin nama Backend yang tersedia untuk instans ini. Default: `["ANY"]`, artinya semua Backend tersedia di paket ini. `[]` berarti tidak ada Backend.
