# Memulai dengan Sample-based quantum diagonalization (SQD) {#getting-started-with-sample-based-quantum-diagonalization-sqd}
<details>

</details>
<Admonition type="caution" title="Catatan untuk pengguna Windows">

Contoh ini menggunakan paket `pyscf`, yaitu library simulasi kimia berbasis Python, yang tidak didukung secara native di Windows; oleh karena itu, contoh-contoh di bawah ini tidak akan berfungsi.

Untuk menjalankan potongan kode di bawah ini, pengguna Windows perlu membuat lingkungan Python menggunakan Windows Subsystem for Linux, seperti yang disarankan dalam [panduan instalasi `pyscf`.](https://pyscf.org/user/install.html)

</Admonition>

Panduan ini menunjukkan contoh kerja sederhana untuk memulai dengan paket `qiskit-addon-sqd`. Dalam contoh ini, kamu bisa menggunakan SQD untuk mendapatkan aproksimasi ground state dari molekul $N_2$ pada keadaan keseimbangan.

Untuk memulai, ingat bahwa Hamiltonian elektron berinteraksi secara generik memiliki bentuk:

$$ \hat{H} = \sum_{\substack{pr\\ \sigma}}h_{pr} \hat{a}_{p\sigma}^\dagger \hat{a}_{r\sigma} + \sum_{\substack{prqs\\ \sigma\tau}} \frac{1}{2}\left(pr|qs\right)\hat{a}_{p\sigma}^\dagger \hat{a}_{q\tau}^\dagger \hat{a}_{s\tau}\hat{a}_{r\sigma} $$

di mana $\hat{a}_{p\sigma}^\dagger$ dan $\hat{a}_{p\sigma}$ adalah operator kreasi dan anihilasi Fermionic yang terkait dengan orbital ke-$p$ dengan spin $\sigma$, serta $h_{pr}$ dan $\left(pr|qs\right)$ adalah integral elektronik satu-bodi dan dua-bodi.
## Menyiapkan informasi molekul {#prepare-molecule-information}

Untuk memulai, tentukan molekul dan propertinya menggunakan `pyscf` serta integral elektronik yang tersimpan di `n2_fci.txt`.
<Admonition type="note">
Contoh ini menggunakan data yang sudah disiapkan sebelumnya dalam file bernama `n2_fci.txt`, yang berisi integral elektronik satu-bodi dan dua-bodi. Untuk menjalankan sel kode di bawah ini pada mesin lokal, salin dan tempel data ke dalam file dengan nama yang sama. Contoh ini juga membutuhkan paket `pyscf` untuk diinstal.
</Admonition>
