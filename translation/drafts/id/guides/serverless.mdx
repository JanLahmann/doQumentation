# Apa itu Qiskit Serverless? {#what-is-qiskit-serverless}

Qiskit Serverless menyediakan antarmuka sederhana untuk menjalankan beban kerja di seluruh sumber daya kuantum-klasik. Ini mencakup penerapan program ke IBM Quantum&reg; Platform dan menjalankan beban kerja dari jarak jauh, serta manajemen sumber daya yang mudah untuk kasus penggunaan multi-cloud dan superkomputer berbasis kuantum.

Pengguna premium dapat membangun, menerapkan, dan menjalankan beban kerja mereka dari jarak jauh di komputasi klasik yang tersedia melalui IBM Quantum Platform.

<Admonition type="note">
    Ini adalah fitur eksperimental yang hanya tersedia untuk pengguna IBM Quantum Premium Plan, Flex Plan, dan On-Prem (melalui IBM Quantum Platform API) Plan.
</Admonition>

## Kapan sebaiknya pakai Qiskit Serverless? {#when-should-i-use-qiskit-serverless}

Qiskit Serverless membantu mengelola sumber daya klasik dan kuantum di seluruh alur kerja [pola Qiskit](./intro-to-patterns). Ini mencakup beberapa contoh berikut:

- Memparalelkan tugas klasik, seperti pra-pemrosesan dan pasca-pemrosesan
- Mempertahankan beban kerja yang berjalan lama di cloud, bahkan jika laptop kamu dimatikan
- Menerapkan program yang dapat digunakan kembali

## Apa saja yang termasuk? {#what-is-included}

Sumber daya komputasi berikut sudah termasuk secara default di lingkungan Qiskit Serverless. Pengguna dapat menghubungi [dukungan](/guides/support) untuk meningkatkan salah satu dari ini.

- 16 core vCPU
- 64GB RAM
- Maksimum dua pekerjaan aktif per pengguna dalam satu waktu

## Instal Qiskit Serverless {#install-qiskit-serverless}

Untuk menggunakan Qiskit Serverless di IBM Quantum Platform, instal paket-paket berikut:

```bash
pip install qiskit-ibm-catalog qiskit_serverless
```

`qiskit_ibm_catalog` menyediakan alat sisi klien untuk mengunggah dan menjalankan program jarak jauh, sementara `qiskit_serverless` menyediakan alat sisi server untuk mendistribusikan komputasi dan mengelola data. Paket-paket ini memerlukan `python` 3.11+. Untuk pengguna dan organisasi yang ingin menjalankan Qiskit Serverless di infrastruktur kustom, ikuti panduan [Pengaturan infrastruktur cloud](https://qiskit.github.io/qiskit-serverless/).

<Admonition type="tip">
Saat ini, tabel beban kerja IBM Quantum hanya mencerminkan beban kerja Qiskit Runtime. Gunakan `job.status()` untuk melihat status terkini beban kerja Qiskit Serverless kamu.
</Admonition>
