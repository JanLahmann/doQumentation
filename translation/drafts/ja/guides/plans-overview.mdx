# 利用可能なプランの概要 {#overview-of-available-plans}

IBM Quantum Platform には、以下に説明するとおり、Open（無料アクセス）および有料プランがございます。詳細については、[IBM Quantum アクセスプラン](https://www.ibm.com/quantum/pricing)をご参照ください。

複数のプランにアクセスしたい場合は、複数のインスタンスを作成し、それぞれに異なるプランを割り当てることができます。たとえば、Open Plan のインスタンスと Pay-As-You-Go Plan のインスタンスを同時に持つことが可能です。優先して使用するインスタンスは、`QiskitRuntimeService` のイニシャライザーまたは `QiskitRuntimeService.save_account()` メソッドで設定できます。例：

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Open Plan {#open-plan}

世界最高水準の QPU 上で量子回路を無料で実行できます（28 日間のローリングウィンドウあたり最大 10 分の量子時間）。使用状況は [Platform ダッシュボード](https://quantum.cloud.ibm.com/)および[ワークロード](https://quantum.cloud.ibm.com/workloads)ページで追跡できます。Open Plan は us-east リージョンのみで利用可能です。

このプランは、量子コンピューティングを学習中の方や IBM の量子技術を探索されている方にお勧めです。

## Flex Plan {#flex-plan}

量子ジョブの実行に使用する一定数の分（最低 400 分）を事前購入し、購入から 1 年以内に使用できます。追加の分はいつでも購入可能です。

時間を購入した後、アカウントの Flex Plan 上の既存または新規インスタンスに分を割り当てる必要があります。割り当て制限は[インスタンスページ](https://quantum.cloud.ibm.com/instances)で設定・更新するか、[新しい Flex Plan インスタンスを作成する](/guides/instances)際に設定できます。すべての分を既存のインスタンスに割り当てた場合、追加の分を購入するか別のインスタンスの割り当てから分を削除するまで、Flex Plan の新しいインスタンスを作成することはできません。詳細については、[インスタンス割り当て制限の設定](allocation-limits)をご覧ください。

10,000 分以上必要な場合は、代わりに Premium Plan のサブスクリプションをご検討ください。400 分の購入を希望されない場合は、代わりに Pay-As-You-Go Plan をご検討ください。

Flex Plan にサインアップするには、[IBM にお問い合わせ](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum)ください。

## Pay-As-You-Go Plan {#pay-as-you-go-plan}

世界最高水準の QPU 上で量子回路を実行し、使用した量子時間分のみお支払いいただけます。

このプランは、量子ユーティリティの研究プロジェクトを実施中の方や、柔軟なアクセスでビジネスモデルをテストされている方にお勧めです。今後 1 年間で少なくとも 400 分を使用する予定がある場合は、代わりに Flex Plan の使用をお勧めします。

## Premium Plan {#premium-plan}

エンタープライズ向け量子時間サブスクリプションを使用して、世界最高水準の QPU 上で量子回路を実行できます。また、[Qiskit Functions](functions) や [Qiskit Transpiler as a Service](qiskit-transpiler-service) などの一部の機能は、Premium Plan ユーザーのみご利用いただけます。

このプランは、戦略的な量子ロードマップを実行し、量子アルゴリズムおよびアプリケーションを大規模に開発したい方にお勧めです。詳細については、[IBM にお問い合わせ](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum)ください。

## On-Prem Plan {#on-prem-plan}

IBM Quantum によってサービスおよびメンテナンスが行われる、完全に専用のオンプレミス量子システムにアクセスできます。

このサービスは、古典量子アルゴリズムおよびアプリケーションを探索中で、リソースとデータに対する高い制御が必要な方にお勧めです。詳細については、[IBM にお問い合わせ](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum)ください。
