# Qiskit 1.0 移行ガイド {#qiskit-10-migration-guide}

Qiskit 1.0 では新しい安定性の保証とパフォーマンスの改善が導入されていますが、以前のすべてのバージョンの Qiskit との互換性は失われています。Qiskit v1.0 に対応するために、環境およびコードを更新する方法については以下のガイドをご参照ください。

- [新しい Qiskit v1.0 パッケージのインストール](/guides/qiskit-1.0-installation): Python 環境とプロジェクトの依存関係をアップグレードする方法について説明します。
- [Qiskit v1.0 の機能変更](/guides/qiskit-1.0-features): コードを更新する方法について説明します。
- [Qiskit v1.0 のパッケージの破壊的変更](/guides/metapackage-migration): 新旧のパッケージ構成を理解するための説明です。

Qiskit v1.0 のメリットの詳細については、[IBM&reg; ブログ](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary)をご覧ください。
