# 行動規範

## 私たちの誓約 {#our-pledge}

Qiskitコミュニティは、すべての個人を尊重と敬意をもって扱うという価値観に専念しています。オープンで歓迎的な環境を育むために、参加者、講演者、スポンサー、ボランティア、オンライン貢献者、IBM従業員を含むすべての参加者は、年齢、外見、障害、民族性、性自認と表現、体型、経験レベル、国籍、人種、宗教、カースト、性的自認と指向にかかわらず、すべての人にハラスメントのない体験を提供することで、互いのコミュニティに対して礼節を示すことが期待されます。期待される行動は、Qiskitコミュニティ内でのオンラインおよびオフラインの関与の両方に適用されます。

## 適用範囲 {#scope}

この行動規範の目的は、Qiskitオープンソースコミュニティにおける貢献者および参加者の価値観と行動を定め、実施することです。行動規範は、プロジェクトスペース内と、個人がQiskitオープンソースコミュニティに関与する公共スペースの両方に適用されます。例としては、Qiskitイベントへの参加、オンラインプロジェクトへの貢献、Slack上のコメント、または公式プロジェクトのメールアドレスの使用、公式ソーシャルメディアアカウントによる投稿、オンラインまたはオフラインイベントでの任命された代表者としての行動など、プロジェクトやコミュニティの代表が含まれます。プロジェクトの代表については、プロジェクトメンテナーによってさらに定義・明確化される場合があります。

## 私たちの基準 {#our-standards}

ポジティブな環境の創出に貢献する行動の例:

- 歓迎的で包括的な言語を使用すること
- 異なる視点、経験、文化を尊重すること
- 建設的な批判を穏やかに受け入れること
- コミュニティにとって最善のことに集中すること
- 他のコミュニティメンバーへの共感を示すこと
- 周囲と仲間の参加者に気を配り、他者の話に耳を傾けること
- すべての参加者の貢献を大切にすること
- 対立の前に協力を優先すること
- コミュニティメンバーが無意識に人種差別的、性差別的、カースト差別的、または偏った発言やジョークをした際に、その場で指摘すること

「皮肉」や「冗談」として提示された場合でも、参加者による許容できない行動の例:

- 性的な言語や画像の使用、歓迎されない身体的接触、性的な注目や働きかけ
- トローリング、侮辱的・軽蔑的なコメント、個人的または政治的攻撃
- 公開または非公開のハラスメント（不快または品位を傷つける言語を含む）
- 明示的な許可なく、物理的または電子的なアドレスなど、他者の個人情報を公開すること。これには、本人の同意なく、その人のアイデンティティのいかなる側面についても「アウティング」することが含まれます。
- 「ドクシング」：引用されたすべてのユーザーの明示的な同意なく、特にアイデンティティSlackチャンネル、プライベートチャット、または公開イベントからのスクリーンショットや引用を公開すること。
- スパム行為への関与（事前のIBMの承認なく、未承諾のメッセージ、LLM（大規模言語モデル）の出力、広告、またはプロモーションコンテンツをコミュニティメンバーに繰り返し送信するなど）。
- 職業上の場において不適切と合理的に判断されうるその他の行為

## 責任と実施 {#responsibilities-and-enforcement}

Qiskitコミュニティ全体が、Qiskitコミュニティのイベントやスペースにおいて行動規範の条件を守る責任を持ち、違反を目にした際には報告する責任があります。IBMの内部Qiskitチームは、最終的に許容できる行動の基準を明確にし、実施する責任を持ち、許容できない行動のいかなる事例に対しても適切かつ公正な是正措置を講じることが期待されます。

参加者または貢献者が否定的または有害な行動に従事した場合、IBMは適切と判断するあらゆる措置を講じます。これには、警告の発令、返金なしのイベントからの排除、コメントの削除、将来のイベントやオンラインコミュニティからの永久禁止、または地元の法執行機関への連絡などが含まれますが、これらに限定されません。IBMは、この行動規範に沿っていないコメント、コミット、コード、ウィキ編集、課題、その他の貢献を削除、編集、または拒否する権利と責任を持ち、不適切、脅迫的、攻撃的、または有害と判断するその他の行動に対して、貢献者または参加者を一時的または永久に禁止することができます。

行動規範の違反を目にした場合:

1. 可能であれば、その行動が不適切であることを当該者に伝え、やめるよう、またはメッセージやコメントを編集・削除するよう求めてください。
2. 当該者がすぐに行動をやめない、または問題を是正しない場合、あるいは発言しにくい場合は、モデレーターに報告し、適切であれば匿名の[行動規範違反フォーム](https://airtable.com/shrl5mEF4Eun1aIDm)に記入してください。
3. Qiskitコミュニティは、フォームの送信を受け取った後、調査を開始します。報告の際は、状況をより深く理解し解決するために使用できる関連する詳細情報、リンク、スクリーンショット、文脈、またはその他の情報を含めてください。
4. 行動規範の違反がイベントで発生し、即時対応が必要な場合、または今後のイベントに参加する個人に関する懸念がある場合は、イベント固有の行動規範ドキュメントに記載されているイベントのオンコール行動規範担当者に連絡してください。担当者に直接話しにくい場合は、行動規範違反フォームに記入し、イベントの詳細を含めて、行動規範執行委員会がイベントのオンコール行動規範担当者に連絡できるようにしてください。
5. IBM従業員が、イベント、Slackチャンネル、オープンソースフォーラムなど、いかなる場所でも行動規範の違反を目撃した場合、行動規範違反報告書を提出することが彼らの責任です。

この行動規範は、IBMビジネス行動指針やIBMビジネスパートナー行動規範などの既存のIBM企業ポリシーに取って代わるものではありません。IBM従業員はIBMビジネス行動指針に従わなければなりません。IBMビジネスパートナーはIBMビジネスパートナー行動規範に従わなければなりません。同僚のIBM社員の行動に懸念を持つIBM従業員は、違反したIBM社員のマネージャーへの働きかけやIBM懸念・不服申し立て制度の活用を含む、IBM独自の社内HR報告プロトコルに従うべきです。IBMビジネスパートナーの行動に懸念を持つIBM従業員は、tellibm@us.ibm.comに通知してください。
