# 行動規範 {#code-of-conduct}

## 私たちの誓い {#our-pledge}

Qiskitコミュニティは、すべての個人を敬意と尊厳をもって扱うという価値観に専念しています。オープンで歓迎的な環境を促進するために、参加者、講演者、スポンサー、ボランティア、オンライン貢献者、IBM従業員を含むすべての参加者は、年齢、外見、障害、民族、ジェンダーアイデンティティと表現、体格、経験レベル、国籍、人種、宗教、カースト、性的アイデンティティと指向に関係なく、ハラスメントのない体験を作り出すことで、互いにそしてコミュニティに対して礼儀を示すことが期待されています。期待される行動は、Qiskitコミュニティ内のオンラインとオフラインの両方の活動に適用されます。

## 適用範囲 {#scope}

この行動規範の目的は、Qiskitオープンソースコミュニティの貢献者と参加者の価値観と行動を定義し、実施することです。行動規範は、プロジェクトスペース内と、個人がQiskitオープンソースコミュニティに関わっている公共の場の両方に適用されます。例としては、Qiskitイベントへの参加、オンラインプロジェクトへの貢献、Slackでのコメント、プロジェクトやコミュニティの代表としての活動（公式のプロジェクトメールアドレスの使用、公式ソーシャルメディアアカウントからの投稿、オンラインまたはオフラインイベントでの任命された代表としての行動など）が含まれます。プロジェクトの代表は、プロジェクトメンテナーによってさらに定義・明確化される場合があります。

## 私たちの基準 {#our-standards}

ポジティブな環境の構築に貢献する行動の例：

- 歓迎的で包括的な言葉の使用
- 異なる視点、経験、文化の尊重
- 建設的な批判を快く受け入れること
- コミュニティにとって最善のことに集中すること
- 他のコミュニティメンバーへの共感を示すこと
- 周囲や他の参加者に注意を払い、他者の意見に耳を傾けること
- すべての参加者の貢献を尊重すること
- 対立の前に協力に取り組むこと
- コミュニティメンバーによる意図しない人種差別的、性差別的、カースト差別的、または偏見のあるコメントやジョークが発生した場合に指摘すること

「皮肉」や「冗談」として提示された場合でも、参加者による容認できない行動の例：

- 性的な言葉やイメージの使用、および不快な身体的接触、性的な注目、またはアプローチ
- トローリング、侮辱的/軽蔑的なコメント、個人的または政治的攻撃
- 攻撃的または侮辱的な言葉を含む公的または私的なハラスメント
- 明示的な許可なく、物理的または電子的な住所など他者の個人情報を公開すること。これには、本人の同意なくアイデンティティのあらゆる側面を「暴露」することが含まれます。
- 「ドクシング」—アイデンティティSlackチャンネル、プライベートチャット、または公開イベントからのスクリーンショットや引用を、引用されたすべてのユーザーの明示的な同意なく公開すること。
- スパム活動への関与。例えば、IBMの事前承認なしにコミュニティメンバーに対して未承諾のメッセージ、LLM（大規模言語モデル）の出力、広告、またはプロモーションコンテンツを繰り返し送信すること。
- 専門的な環境で合理的に不適切と見なされるその他の行為

## 責任と実施 {#responsibilities-and-enforcement}

Qiskitコミュニティ全体が、Qiskitコミュニティのイベントとスペースにおいて行動規範の条件を遵守し、違反を目撃した場合に報告する責任を負っています。IBM内のQiskit内部チームが、許容される行動の基準を明確にし、実施する最終的な責任を負い、容認できない行動のあらゆる事例に対して適切かつ公正な是正措置を講じることが期待されています。

参加者または貢献者がネガティブまたは有害な行動に関わった場合、IBMは適切と判断する措置を講じます。これには、警告の発行、返金なしでのイベントからの退去、コメントの削除、将来のイベントやオンラインコミュニティからの恒久的な追放、または地域の法執行機関への通報が含まれますが、これらに限定されません。IBMは、この行動規範に沿わないコメント、コミット、コード、Wiki編集、Issue、その他の貢献を削除、編集、または拒否する権利と責任を有し、不適切、脅迫的、攻撃的、または有害と見なすその他の行動を理由に、貢献者または参加者を一時的または恒久的に追放する権利を有します。

行動規範違反を目撃した場合：

1. あなたが快適であれば、本人にその行為が不適切であることを伝え、やめるよう求めるか、メッセージやコメントを編集または削除するよう求めてください。
2. 本人が直ちに行動を止めないか問題を是正しない場合、またはあなたが発言するのが不快な場合は、モデレーターに報告し、適切であれば匿名の[行動規範違反報告フォーム](https://airtable.com/shrl5mEF4Eun1aIDm)に記入してください。
3. Qiskitコミュニティは、フォームの入力を受け取り次第、調査を開始します。報告の際は、状況の理解と解決に役立つ関連する詳細、リンク、スクリーンショット、文脈、その他の情報を含めてください。
4. 行動規範違反がイベントで発生し、即座の対応が必要な場合、または今後のイベントに参加する個人に関する懸念がある場合は、イベント固有の行動規範文書に記載されているイベントの当番行動規範連絡先に連絡してください。連絡先と直接話すのが不快な場合は、行動規範違反報告フォームに記入し、イベントの詳細を含めてください。そうすることで、行動規範執行委員会がイベントの当番連絡先に連絡できます。
5. IBM従業員が、イベント、Slackチャンネル、オープンソースフォーラムなどで行動規範違反を目撃した場合、行動規範違反報告を提出するのはその従業員の責任です。

この行動規範は、IBMビジネス行動ガイドラインやIBMビジネスパートナー行動規範など、既存のIBM企業ポリシーに優先するものではありません。IBM従業員はIBMビジネス行動ガイドラインに従う必要があります。IBMビジネスパートナーはIBMビジネスパートナー行動規範に従う必要があります。IBM従業員が同僚のIBM従業員の行動について懸念がある場合は、違反者のマネージャーへの報告やIBM Concerns and Appealsへの連絡を含むIBM独自の内部HR報告プロトコルに従ってください。IBM従業員がIBMビジネスパートナーの行動について懸念がある場合は、tellibm@us.ibm.com に通知してください。
