# オープンソース {#open-source}

2016年以来、IBM Quantum&reg;はクラウドを通じて量子コンピューティングリソースをすべての人に公開することで、コミュニティ主導のイノベーションの力を活用してきました。
この画期的な取り組みには、量子ソフトウェアスタックの主要な部分をオープンソース化し、汎用性の高い量子コンピューティングフレームワークであるQiskitを導入することが含まれています。
現在、Qiskitはその活気あるコミュニティのおかげもあり、最も人気の高い量子フレームワークとなっています。
Qiskitコミュニティは、量子コンピューティングを通じてソフトウェア、研究、教育、ビジネスを発展させるために協力する、開発者、科学者、愛好者からなる活動的で歓迎的なグループです。
この革新的な技術をオープンかつアクセスしやすいものにすることで、イノベーションを加速し、新たな応用を発見することができます。

量子技術が進化するにつれて、オープンソースの[Qiskitエコシステム](https://www.ibm.com/quantum/ecosystem)プロジェクトおよび開発者からの貢献が、その未来を形作る上で非常に重要になっています。

Qiskitコミュニティに参加することで、量子技術の発展に貢献することができます。
熟練した量子プログラマーであっても、量子の世界への第一歩を踏み出したばかりであっても、Qiskitは限界を押し広げるために必要なサポートとインスピレーションを提供しています。
プルリクエストでのコードから[Slack](https://qisk.it/join-slack)でのコミュニティサポートまで、あらゆる種類の貢献を歓迎しており、誰もが量子技術の発展に役割を果たすことができます。
