# データのセキュリティ保護 {#securing-your-data}

IBM Quantum&reg; Platform を使用する際にデータを安全に管理するためには、どのようなデータが保存・暗号化されているか、また保存されたデータをどのように削除できるかを正確に把握することが重要です。

## 機密データの保護 {#protecting-your-sensitive-data}

IBM Cloud&reg; に保存されたデータは、ランダムに生成されたキーを使用して保存時に暗号化されます。

## データの削除 {#deleting-your-data}

サービスインスタンスを削除すると、ジョブ、結果、パラメーター、プログラムなど、そのインスタンスに関連するすべてのコンテンツが削除されます。インスタンスを削除するには、[インスタンスページ](https://quantum.cloud.ibm.com/instances)から削除したいインスタンスを見つけ、オーバーフローメニューをクリックして **Delete** をクリックしてください。削除の確認を求めるメッセージが表示されます。

Manager ロールをお持ちの場合は、個別のジョブとそのデータも削除できます。アクセスポリシー、グループ、ロールの詳細については、[ドキュメント](/guides/access-groups)をご参照ください。

## データ保持ポリシー {#data-retention-policy}

IBM は、サービスの現在のユーザーである間、（削除されていない）データを3年間保持します。サービスの利用を停止した後、IBM がデータをどのくらいの期間保持するかについては、IBM Terms サイトに掲載されている [Qiskit Runtime データシート](https://www.ibm.com/support/customer/csol/terms/?id=BB6EE1D06BDE11EBA199CCFE56FA6D9D)をご参照ください。
