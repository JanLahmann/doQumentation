## 廃止されたプロセッサーファミリー {#retired-processor-families}

- **Canary**: Canaryファミリーは、5量子ビットから16量子ビットまでの小規模な設計で構成されていました。
- **Falcon**: Falconファミリーのデバイスは、中規模回路向けの優れたプラットフォームを提供し、より大型のデバイスへ展開する前にパフォーマンスとスケーラビリティの改善を実証する場としても活用されました。
- **Egret**: Egretは、チューナブルカプラーの革新技術を33量子ビットのプラットフォームに導入し、より高速かつ高忠実度の2量子ビットゲートを実現しました。
- **Hummingbird**: ヘビー・ヘキサゴナル量子ビットレイアウトを採用したHummingbirdファミリーは、最大65量子ビットをサポートしました。
- **Osprey**: OspreyはEagleのほぼ4倍の規模となる433量子ビットを備えていました。チップサイズの大型化に伴い、デバイスパッケージングのさらなる改善や、同一配線フットプリント内でより多くのI/O要件に対応するためのクライオスタット内カスタムフレキシブルケーブルが必要でした。
- **Eagle**: Eagleプロセッサーファミリーは、それ以前の世代よりもスケーラブルなパッケージング技術を採用しました。特に、信号が複数のチップ層を通過することで、性能を損なうことなく高密度のI/Oを実現しました。

## 廃止されたQPUからジョブを取得する {#retrieve-a-job-from-a-retired-qpu}

廃止されたQPUからジョブを取得するには、以下のコードをご使用ください。

<Admonition type="note">
以下のコードにCloud Resource Name（CRN）を渡すことはオプションです。渡した場合、そのインスタンスのバックエンドとジョブのみが利用可能になります。インスタンスを指定しない場合、アカウント内のすべてのインスタンスにわたるすべてのバックエンドとジョブが利用可能になります。この場合、バックエンドが指定されると、そのバックエンドにアクセスできるインスタンスが検索されて使用されます。
</Admonition>

```python
from qiskit_ibm_runtime import QiskitRuntimeService

# Load your IBM Quantum account. Replace "<IBM Cloud CRN>" with your desired instance's CRN.
service = QiskitRuntimeService(channel="ibm_quantum_platform", instance="<IBM Cloud CRN>")

# Retrieve a single job by id
job = service.job(<job_id>)

# Retrieve a batch of jobs. Filtering options can be found in the QiskitRuntimeService.jobs api reference
jobs = service.jobs(backend_name=<backend_name>)
```

<Admonition type="note" title="Deprecated provider packages">
  `service.jobs()` returns jobs run from both `qiskit-ibm-runtime` and the deprecated `qiskit-ibm-provider` package. Jobs submitted by the older (also deprecated) `qiskit-ibmq-provider` package are no longer available.
</Admonition>
