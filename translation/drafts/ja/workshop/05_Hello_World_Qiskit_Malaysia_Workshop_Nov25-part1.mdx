## Qiskitのインストール <a id="install-qiskit"></a>

オンラインのjupyter lab環境（ガイド参照：[Online lab environments](quantum.cloud.ibm.com/docs/en/guides/online-lab-environments)）を使用するか、ローカルにQiskitをインストールすることもできます。

Qiskitインストールガイド[Install the Qiskit SDK and the Qiskit Runtime client](https://quantum.cloud.ibm.com/docs/en/guides/install-qiskit)に従って、以下の手順を完了させてください：

	- 視覚化パッケージを含むQiskitをインストールする：`pip install qiskit[visualization]`

	- qiskit-ibm-runtimeをインストールする：`pip install qiskit-ibm-runtime`

	- jupyterをインストールする：`pip install jupyter`
使用するPythonのバージョンがpython>=3.10であることを確認して、最新のQiskitバージョンとの互換性を保護してください：
