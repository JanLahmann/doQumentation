# 概要

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

量子安全暗号の実践的な入門は、IBM Quantum&reg; Learning で提供される無料のオンラインコースで、量子安全暗号の基礎概念を学ぶための入門教材です。

このコースは、アプリケーションのセキュリティを近代化することに関心を持つ開発者を対象に作成されています。暗号学の4つのコア領域（暗号ハッシュ関数、共通鍵暗号、公開鍵暗号、量子安全暗号）について、マルチモーダルなレッスンとインタラクティブなライブコード例を交えて学ぶことができます。

このコースを受講することで、開発者はサイバーセキュリティリスクの状況がどのように変化しているかを学び、量子時代を形作ることが期待される最新のアプローチを探求できます。
