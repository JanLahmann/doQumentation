# Онлайн-середовища для роботи

Окрім завантаження та встановлення Qiskit на локальній машині, є кілька варіантів для тих, хто воліє працювати в онлайн-середовищі розробки.

Деякі провайдери пропонують онлайн-середовища Jupyter з попередньо встановленим Qiskit для швидкого та зручного старту. До рекомендованих провайдерів належать хмарний провайдер OVHcloud і хмарна платформа розробки квантового програмного забезпечення qBraid.

## Рекомендовані середовища для ноутбуків із попередньо встановленим Qiskit {#recommended-notebook-environments-with-qiskit-preinstalled}

Додаткові онлайн-середовища для Jupyter-ноутбуків, зокрема IBM Watson&reg; Studio, Google Colab і Microsoft Azure Machine Learning Studio, можна використовувати для роботи з Qiskit, дотримуючись відповідних інструкцій.

### AI-ноутбуки OVHcloud {#ovhcloud-ai-notebooks}

Користувачі OVHcloud Public Cloud і OVHcloud Startup можуть використовувати квантові ноутбуки OVHcloud для доступу до Qiskit та відправки завдань на QPU IBM&reg; за допомогою акаунту IBM Quantum&reg; або IBM Cloud&reg;. Щоб дізнатися більше, відвідай сторінку [Документація та навчальні матеріали OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
AI-ноутбуки OVHcloud недоступні в деяких регіонах, зокрема у Сполучених Штатах. Крім того, інтерфейс користувача може відрізнятися залежно від регіону.
</Admonition>

Виконай такі кроки, щоб розпочати:

1. [Створи акаунт IBM Cloud.](./cloud-setup)
1. [Налаштуй акаунт OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Створи проєкт OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Виконай кроки у розділі [Запустити перший AI Notebook,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) вибравши **Qiskit** як фреймворк.

### qBraid Lab {#qbraid-lab}

Користувачі Qiskit можуть використовувати попередньо налаштовані Python-середовища qBraid, дотримуючись цих інструкцій. Докладніше читай у [документації qBraid.](https://docs.qbraid.com/home/introduction)

1. [Зареєструй безкоштовний акаунт qBraid.](https://docs.qbraid.com/home/account)
3. [Створи лабораторне середовище.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Запусти qBraid Lab, натисни іконку **ENVS** праворуч, клацни **+ ADD** у верхній частині нової бічної панелі та знайди Qiskit. Клацни на версію, яку хочеш використовувати, а потім натисни **Install**.
4. Створи Jupyter-ноутбук, клацнувши **Python 3 [Qiskit]** у розділі Notebook.
4. Ти можеш встановити будь-які додаткові Python-пакети (наприклад, Qiskit v1.2) у клітинці ноутбука, виконавши команду `%pip install [package_name].` Докладніше читай у розділі [Команди Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Додаткові середовища для ноутбуків для користувачів Qiskit {#additional-notebook-environments-for-qiskit-users}

Для платформ без попередньо налаштованого Qiskit інструкції з налаштування схожі, але тобі потрібно встановити Qiskit вручну. Ти створюєш акаунт, починаєш новий проєкт (або ноутбук чи робочий простір, залежно від платформи) і встановлюєш Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Налаштуй безкоштовну пробну версію Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Створи проєкт Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Щоб встановити необхідні пакети та залежності Qiskit, відкрий новий ноутбук і виконай таку команду:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Створи акаунт Google.](https://support.google.com/accounts/answer/27441?hl)
2. Перейди на [цільову сторінку Google Colab](https://colab.research.google.com/) і увійди за допомогою свого акаунту Google.
3. [Створи ноутбук у Colab](https://colab.research.google.com/#create=true) і встанови Qiskit, виконавши таку команду:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Створи акаунт Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Створи робочий простір.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Створи Jupyter-ноутбук у своєму робочому просторі.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. З ноутбука виконай цю команду для встановлення Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
