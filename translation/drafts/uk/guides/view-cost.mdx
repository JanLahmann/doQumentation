# Перегляд вартості

У будь-який момент ти можеш переглянути, скільки коштів витрачено на робочі навантаження, пов'язані з екземпляром.

## Перегляд витрат екземпляра {#view-instance-cost}

Щоб дізнатись, наскільки інтенсивно використовувався екземпляр, на [сторінці Instances](https://quantum.cloud.ibm.com/instances) натисни на екземпляр, щоб відкрити сторінку з його деталями. Ти можеш перейти за посиланням **Billing overview** для отримання повної інформації про витрати та виставлення рахунків.

**Instance usage** відображає різні значення залежно від типу плану, до якого прив'язаний екземпляр:

- Для планів Premium і Open — це використання в реальному часі за останні 28 днів (ковзне вікно) до поточного моменту поточного дня.
- Для планів Pay-As-You-Go — це все використання з моменту створення екземпляра.
- Для планів Flex — це використання з початку дії контракту.

## Перегляд використання робочого навантаження {#view-workload-usage}

Щоб дізнатись, скільки ресурсів витратило конкретне робоче навантаження, відкрий [сторінку Workloads](https://quantum.cloud.ibm.com/workloads) і перегляньте стовпець «Usage» для відповідного навантаження.
