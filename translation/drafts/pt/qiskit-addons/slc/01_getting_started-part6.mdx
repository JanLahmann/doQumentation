### Visualize o SLC para inspeção manual

Após mesclar os limites para trás e os limites para frente ajustados, o comportamento dos SLCs combinados fica claro:

- A função acima nos informa que uma partição é escolhida na qual ocorre a transição dos limites para trás para os limites para frente ajustados.
- Podemos ver abaixo que os SLCs agora contêm limites parciais para trás e limites parciais para frente ajustados.

```python
for p in "XYZ":
    display(
        draw_shaded_lightcone(
            boxed_circuit,
            merged_bounds,
            noise_model_paulis,
            pauli_filter=p,
            scale=0.15,
            fold=-1,
            idle_wires=False,
            wire_order=wire_order,
            measure_arrows=False,

    )
```
