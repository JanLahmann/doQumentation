#### Prévia dos limites mesclados sem taxas de ruído aprendidas

A função `merged_bounds` determina o ponto no circuito em que a transição dos limites para trás para os limites para frente minimiza o viés total estimado no observável desejado. Esse viés é calculado como a soma das contribuições dos limites para trás para todos os locais de ruído antes desse ponto, mais as contribuições dos limites para frente para todos os locais de ruído após ele. Atualmente, isso é feito de forma uniforme para todos os qubits.

> **Nota Importante**: O ponto de transição dos limites para frente para os limites para trás depende das taxas de ruído aprendidas.

```python
merged_bounds = merge_bounds(
    boxed_circuit,
    forward_bounds_tighter,
    backward_bounds,
    noise_model_rates,
)
```
