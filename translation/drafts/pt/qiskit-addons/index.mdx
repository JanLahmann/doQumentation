# Qiskit Addons

Os [Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) são pacotes de extensão oficiais que fornecem algoritmos e técnicas especializadas de computação quântica. Cada addon foca em uma área específica e inclui tutoriais com código executável.

## Addons Disponíveis

| Addon | Pacote | Descrição |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Reduza a largura e a profundidade do Circuit cortando Gates e fios |
| **SQD** | `qiskit-addon-sqd` | Diagonalização Quântica Baseada em Amostras para estimativa do estado fundamental |
| **OBP** | `qiskit-addon-obp` | Retropropagação de Operadores para reduzir a profundidade do Circuit |
| **MPF** | `qiskit-addon-mpf` | Fórmulas Multi-Produto para simulação aprimorada de Hamiltoniano |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Compilação Quântica Aproximada com redes tensoriais |
| **PNA** | `qiskit-addon-pna` | Absorção de Ruído Propagado para mitigação de erros |
| **SLC** | `qiskit-addon-slc` | Cones de Luz Sombreados para reduzir a sobrecarga de amostragem PEC |

Todos os tutoriais podem ser executados diretamente no seu navegador usando o Binder, ou no seu próprio servidor Jupyter.

Para a documentação da API, visite os repositórios individuais dos addons no [GitHub](https://github.com/Qiskit?q=qiskit-addon).
