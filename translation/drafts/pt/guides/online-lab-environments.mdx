# Ambientes de laboratório online

Além de baixar e instalar o Qiskit para trabalhar na sua máquina local, há várias opções disponíveis para quem prefere trabalhar em um ambiente de desenvolvimento online.

Vários provedores oferecem ambientes Jupyter online com o Qiskit pré-instalado para uma configuração rápida e conveniente. Os provedores recomendados incluem o provedor de serviços em nuvem OVHcloud e a plataforma de desenvolvimento de software quântico em nuvem qBraid.

## Ambientes de notebook recomendados com Qiskit pré-instalado {#recommended-notebook-environments-with-qiskit-preinstalled}

Ambientes de notebook Jupyter online adicionais, como IBM Watson&reg; Studio, Google Colab e Microsoft Azure Machine Learning Studio, podem ser usados para trabalhar com o Qiskit seguindo as instruções apropriadas.

### Notebooks de IA da OVHcloud {#ovhcloud-ai-notebooks}

Usuários do OVHcloud Public Cloud e do OVHcloud Startup podem usar seus notebooks quânticos da OVHcloud para acessar o Qiskit e enviar jobs para QPUs da IBM&reg; com sua conta IBM Quantum&reg; ou IBM Cloud&reg;. Para saber mais, visite a página de [Documentação e tutoriais da OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Os notebooks de IA da OVHcloud não estão disponíveis em algumas regiões, como os Estados Unidos. Além disso, as interfaces de usuário podem ter aparência diferente em regiões distintas.
</Admonition>

Siga estas etapas para começar:

1. [Crie uma conta IBM Cloud.](./cloud-setup)
1. [Configure uma conta OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Crie um projeto OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Siga as etapas em [Lance seu primeiro Notebook de IA,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) selecionando **Qiskit** como framework.

### qBraid Lab {#qbraid-lab}

Usuários do Qiskit podem usar os ambientes Python pré-configurados do qBraid seguindo estas instruções. Para mais informações, consulte a [Documentação do qBraid.](https://docs.qbraid.com/home/introduction)

1. [Configure uma conta gratuita no qBraid.](https://docs.qbraid.com/home/account)
3. [Crie um ambiente Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Inicie o qBraid Lab, clique no ícone **ENVS** à direita, clique em **+ ADD** no topo da nova barra lateral e pesquise por Qiskit. Clique na versão que deseja usar e depois clique em **Install**.
4. Crie um notebook Jupyter clicando em **Python 3 [Qiskit]** na seção Notebook.
4. Você pode instalar pacotes Python adicionais (como o Qiskit v1.2) em uma célula do notebook executando `%pip install [package_name].` Para mais informações, consulte o tópico de [comandos Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Ambientes de notebook adicionais para usuários do Qiskit {#additional-notebook-environments-for-qiskit-users}

Para plataformas sem o Qiskit pré-configurado, as instruções de configuração são semelhantes, exceto que você deve instalar o Qiskit manualmente. Você vai criar uma conta, iniciar um novo projeto (ou notebook ou workspace, dependendo da plataforma) e instalar o Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Configure o teste gratuito do Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Crie um projeto no Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Para instalar os pacotes e dependências necessários do Qiskit, abra um novo notebook e execute o seguinte comando:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Crie uma conta Google.](https://support.google.com/accounts/answer/27441?hl)
2. Acesse a [página inicial do Google Colab](https://colab.research.google.com/) e entre com sua conta Google.
3. [Crie um notebook no Colab](https://colab.research.google.com/#create=true) e instale o Qiskit executando o seguinte comando:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Crie uma conta Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Crie um workspace.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Crie um notebook Jupyter no seu workspace.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. No notebook, execute este comando para instalar o Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
