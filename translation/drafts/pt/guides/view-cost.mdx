# Ver o custo

A qualquer momento, você pode ver quanto custo foi gerado pelos workloads associados a uma instância.

## Ver o custo da instância {#view-instance-cost}

Para determinar quanto uma instância foi utilizada, acesse a [página de Instâncias](https://quantum.cloud.ibm.com/instances), clique na instância para abrir a página de detalhes. Você pode clicar no link associado a **Visão geral de cobrança** para obter detalhes completos de custo e cobrança.

**Uso da instância** reporta um valor diferente dependendo do tipo de plano ao qual a instância está associada:

- Para planos Premium e Open, este é o uso em tempo real dos últimos 28 dias (período móvel), até o momento atual do dia atual.
- Para planos Pay-As-You-Go, este é todo o uso desde o momento em que a instância foi criada.
- Para planos Flex, este é o uso desde o início do contrato.

## Ver o uso do workload {#view-workload-usage}

Para determinar quanto uso um workload específico gerou, abra a [página de Workloads](https://quantum.cloud.ibm.com/workloads) e veja a coluna "Usage" do workload.
