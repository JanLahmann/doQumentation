# Protegendo seus dados

Para garantir que você possa gerenciar seus dados com segurança ao usar a IBM Quantum&reg; Platform, é importante saber exatamente quais dados são armazenados e criptografados, e como você pode excluir quaisquer dados armazenados.

## Protegendo seus dados sensíveis {#protecting-your-sensitive-data}

Os dados que você armazena na IBM Cloud&reg; são criptografados em repouso usando uma chave gerada aleatoriamente.

## Excluindo seus dados {#deleting-your-data}

Excluir uma instância de serviço remove todo o conteúdo associado a ela, como seus jobs, resultados, parâmetros e programas. Para excluir uma instância, acesse a [página de Instâncias,](https://quantum.cloud.ibm.com/instances) localize a instância que deseja remover, clique no menu de opções e, em seguida, clique em **Delete**. Você será solicitado a confirmar a exclusão.

Se você tiver a função de Manager, também poderá excluir um job individual e seus dados. Veja mais sobre políticas de acesso, grupos e funções na [documentação](/guides/access-groups).

## Política de retenção de dados {#data-retention-policy}

A IBM reterá seus dados (não excluídos) por três anos enquanto você for um usuário ativo do serviço. Para obter informações sobre por quanto tempo os dados são retidos pela IBM após você parar de usar o serviço, consulte o [Qiskit Runtime data sheet](https://www.ibm.com/support/customer/csol/terms/?id=BB6EE1D06BDE11EBA199CCFE56FA6D9D), disponível no site IBM Terms.
