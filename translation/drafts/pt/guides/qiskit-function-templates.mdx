# Introdução aos templates de Qiskit Function

[Os templates de Qiskit Function](https://github.com/qiskit-community/qiskit-function-templates) são uma coleção de código aberto com exemplos de código realistas que dão acesso a alguns dos fluxos de trabalho de pesquisa mais recentes, aproveitando ferramentas como o Qiskit SDK e os [Qiskit addons](/guides/addons). Esses fluxos de trabalho pré-construídos extraem informações de experimentos práticos e os empacotam de forma que você possa personalizar e reutilizar em sua própria pesquisa, sem precisar começar do zero. Todos os templates podem ser facilmente implantados no [Qiskit Serverless](./serverless), onde você pode aproveitar facilmente a computação clássica como parte do seu fluxo de trabalho.

Existem dois tipos de templates:

- *Implementações de templates*: Exemplos de código especializados que implementam técnicas avançadas de experimentos do mundo real em áreas como química e física.
- *Templates base*: Referências para desenvolver sua própria Qiskit Function original. Esses templates destacam as melhores práticas em desenvolvimento de interface, formatação de código, testes unitários e muito mais.

## Implementações de templates {#template-implementations}

As implementações de templates de Qiskit Function são organizadas por área de aplicação. A coleção inclui atualmente um template de física para simulação Hamiltoniana usando o [Qiskit addon AQC-Tensor](./qiskit-addons-aqc) e um template de química para estrutura eletrônica com o modelo de solvente implícito usando o [Qiskit addon SQD](./qiskit-addons-sqd).
Recursos para começar a usar esses dois templates estão disponíveis nos seguintes links:

- Simulação de estrutura eletrônica com modelo de solvente implícito: [arquivos-fonte do template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/chemistry/sqd_pcm) e [guia](./function-template-chemistry-workflow)
- Simulação Hamiltoniana: [arquivos-fonte do template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/physics/hamiltonian_simulation) e [guia](./function-template-hamiltonian-simulation)

## Templates base {#base-templates}

Os *templates base* são referências para desenvolver sua própria Qiskit Function original. Eles foram projetados para ajudar você a dar o pontapé inicial no processo de desenvolvimento, ilustrando as melhores práticas em desenvolvimento de interface, formatação de código, tratamento de E/S, mensagens de erro, checkpoints, testes unitários e muito mais. O repositório é exposto como um [template do GitHub](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-template-repository) para que a implementação de CI/CD possa ser facilmente estendida.

Existem atualmente dois templates: um [template de circuit function](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/circuit_function_template.py) e um [template de application function](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/application_function_template.py). O template de circuit function é construído de forma similar à [IBM Circuit Function](./ibm-circuit-function) e o template de application function ilustra como estruturar um fluxo de trabalho de aplicação típico.
