# Visão geral dos planos disponíveis {#overview-of-available-plans}

O IBM Quantum Platform oferece planos Open (acesso gratuito) e planos pagos, conforme descrito abaixo. Veja [IBM Quantum access plans](https://www.ibm.com/quantum/pricing) para mais detalhes.

Se você quiser acesso a múltiplos planos, pode criar múltiplas instâncias e atribuir planos diferentes a elas. Por exemplo, você pode ter uma instância do Open Plan e uma instância do Pay-As-You-Go Plan. Você pode definir instâncias preferidas para usar no inicializador do `QiskitRuntimeService` ou no método `QiskitRuntimeService.save_account()`. Exemplo:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Open Plan {#open-plan}

Execute seus circuitos quânticos nos melhores QPUs do mundo gratuitamente (até 10 minutos de tempo quântico por janela móvel de 28 dias). Você pode acompanhar seu uso no [painel da Plataforma](https://quantum.cloud.ibm.com/) e nas páginas de [Workloads](https://quantum.cloud.ibm.com/workloads). O Open Plan está disponível apenas na região us-east.

Este plano é recomendado para quem está aprendendo computação quântica e explorando a tecnologia quântica da IBM.

## Flex Plan {#flex-plan}

Pré-compre uma certa quantidade de minutos (mínimo de 400) para execução de jobs quânticos, a serem utilizados dentro de um ano a partir da compra. Você pode comprar mais minutos a qualquer momento.

Após a compra do tempo, você deve alocar os minutos em instâncias existentes ou novas no Flex Plan da sua conta. Defina ou atualize os limites de alocação na [página de Instâncias](https://quantum.cloud.ibm.com/instances) ou defina a alocação ao [criar uma nova instância do Flex Plan.](/guides/instances) Se você alocar todos os minutos nas instâncias existentes, não será possível criar uma nova instância para o Flex Plan até que você compre mais minutos ou remova minutos da alocação de outra instância. Veja [Definir limites de alocação de instâncias](allocation-limits) para saber mais.

Se você precisar de mais de 10.000 minutos, considere assinar um Premium Plan. Se não quiser comprar 400 minutos, considere o Pay-As-You-Go Plan.

[Entre em contato com a IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) para se inscrever no Flex Plan.

## Pay-As-You-Go Plan {#pay-as-you-go-plan}

Execute circuitos quânticos nos melhores QPUs do mundo e pague apenas pelo tempo quântico que você utilizar.

Este plano é recomendado para quem está realizando projetos de pesquisa de utilidade quântica e testando modelos de negócios com acesso flexível. Se você for usar pelo menos 400 minutos no próximo ano, é recomendado que você utilize o Flex Plan.

## Premium Plan {#premium-plan}

Execute circuitos quânticos nos melhores QPUs do mundo usando uma assinatura corporativa de tempo quântico. Além disso, algumas funcionalidades, como [Qiskit Functions](functions) e [Qiskit Transpiler as a Service](qiskit-transpiler-service), estão disponíveis para usuários do Premium Plan.

Este plano é recomendado para quem deseja executar um roteiro quântico estratégico e desenvolver algoritmos e aplicações quânticas em escala. [Entre em contato com a IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) para saber mais.

## On-Prem Plan {#on-prem-plan}

Acesse um sistema quântico on-premises completamente dedicado, com manutenção e suporte fornecidos pelo IBM Quantum.

Este serviço é recomendado para quem está explorando algoritmos e aplicações clássico-quânticas e precisa de alto controle sobre seus recursos e dados. [Entre em contato com a IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) para saber mais.
