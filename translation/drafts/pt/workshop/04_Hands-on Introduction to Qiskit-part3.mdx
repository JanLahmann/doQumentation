## 2.5 Escalar para grandes números de Qubits

Na computação quântica, o trabalho em escala de utilidade é crucial para o progresso na área. Esse tipo de trabalho requer que as computações sejam realizadas em uma escala muito maior; trabalhando com Circuits que podem usar mais de 100 Qubits e mais de 1000 Gates. Este exemplo dá um pequeno passo nessa direção, escalando o problema GHZ para $n=10$ Qubits. Ele usa o fluxo de trabalho de padrões Qiskit e termina medindo o valor esperado $\langle Z_0 Z_i \rangle $.

### Step 1. Map the problem

Escreva uma função que retorna um `QuantumCircuit` que prepara um estado GHZ de $n$ Qubits (essencialmente um estado de Bell estendido), depois use essa função para preparar um estado GHZ de 10 Qubits e colete os observáveis a serem medidos.

```python
def get_qc_for_n_qubit_GHZ_state(n: int) -> QuantumCircuit:

    qc = QuantumCircuit(n) 
    qc.h(0)
    for i in range(n-1):
        qc.cx(i, i+1)
    return qc
n = 10
qc_n_GHZ = get_qc_for_n_qubit_GHZ_state(n)
qc_n_GHZ.draw("mpl")
```

Neste gráfico, observamos que $\langle Z_0 Z_i \rangle$ oscila em torno do valor 1, mesmo que em uma simulação ideal todos os $\langle Z_0 Z_i \rangle$ deveriam ser 1.

Como você pode ver, os resultados de experimentos com 10 Qubits são bons, mas ainda apresentam alguns erros. Uma forma de melhorar os resultados é implementar o estado GHZ de maneira mais eficiente.

Normalmente, o estado GHZ é implementado com uma sequência de Gates CNOT em forma de escada. No entanto, você pode implementar o estado GHZ de forma mais eficiente, reduzindo a profundidade de 2 Qubits de `n` para `n/2` ou menos.
<div class="alert alert-info">

Uma métrica importante para avaliar o quão precisos serão os resultados, ou o quão pouco ruído um Circuit terá, é a profundidade de Gates de 2 Qubits. Isso ocorre porque as taxas de erro para Gates de 2 Qubits (~10 vezes maiores do que as de Gates de 1 Qubit) dominam os erros do Circuit inteiro. Use o código a seguir para obter a profundidade de Gates de 2 Qubits de um Circuit.

```
qc.depth(lambda x: x.operation.num_qubits == 2)
```

</div>

```python
def better_ghz(n):
    "fan out"
    s = int(n / 2)
    qc = QuantumCircuit(n)
    qc.h(s)
    for m in range(s, 0, -1):
        qc.cx(m, m - 1)
        if not (n % 2 == 0 and m == s): 
            qc.cx(n - m - 1, n - m)
    return qc

better_ghz(n).draw("mpl")
```
