# Visão Geral

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Este curso é destinado a pessoas que já têm alguma experiência com computação quântica e desejam aprender a escrever algoritmos variacionais usando os primitivos do Qiskit Runtime. Também é adequado para profissionais de computação quântica que desejam ampliar seus conhecimentos e habilidades na área de algoritmos variacionais.

## Exame {#exam}

Para conquistar sua medalha de design de algoritmos variacionais, faça o exame no IBM&reg; Training. Este exame deve ser realizado após a leitura das lições deste curso. Após ser aprovado, você receberá uma notificação da Credly informando que ganhou uma medalha.
