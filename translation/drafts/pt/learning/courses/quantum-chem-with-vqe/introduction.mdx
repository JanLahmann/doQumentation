Antes de começar, por favor, preencha esta breve [pesquisa pré-curso](https://your.feedback.ibm.com/jfe/form/SV_ePXDhFWwqj4Q0HY), que é importante para nos ajudar a aprimorar nossa oferta de conteúdo e a experiência do usuário.

Este curso é uma introdução focada no solucionador quântico variacional (VQE), incluindo a aplicação do VQE à química. O curso é voltado para profissionais técnicos de computação quântica que têm interesse em aplicar computação quântica a problemas químicos, bioquímicos ou biofísicos. Este curso aborda elementos importantes de um cálculo VQE, incluindo:

* Hamiltoniano
* Ansatz
* Otimizador clássico

Todos esses elementos são combinados para construir cálculos VQE nos Qiskit Runtime Primitives, principalmente o Estimator. Para mais informações sobre os primitivos, consulte a [documentação](/guides/primitives). Ao configurar um cálculo VQE, seguiremos o framework padrão de padrões do Qiskit:

* Mapear entradas clássicas para um problema quântico
* Otimizar o problema para execução quântica
* Executar usando os primitivos do Qiskit Runtime
* Pós-processar e retornar o resultado em formato clássico

Algumas explicações sobre o framework serão encontradas ao longo deste curso, mas para uma explicação completa com exemplos de padrões do Qiskit, consulte [esta documentação](/guides/intro-to-patterns).

Recomendamos planejar dedicar aproximadamente 6 a 10 horas para concluir este curso. Os vídeos e as leituras provavelmente levam duas horas. O tempo restante é para executar e modificar células de código, responder perguntas de quiz e ler a documentação vinculada.

O IBM Quantum&reg; tem outras ofertas de capacitação relacionadas ao VQE. Se você já está familiarizado com os componentes do VQE e simplesmente quer colocá-lo em prática, confira nosso tutorial: [Estimativa de energia do estado fundamental da cadeia de Heisenberg com VQE](/tutorials/spin-chain-vqe).
