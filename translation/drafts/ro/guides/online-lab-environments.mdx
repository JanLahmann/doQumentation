# Medii de laborator online

Pe lângă descărcarea și instalarea Qiskit pentru a lucra pe mașina ta locală, există mai multe opțiuni disponibile pentru cei care preferă să lucreze într-un mediu de dezvoltare online.

Mai mulți furnizori oferă medii Jupyter online cu Qiskit preinstalat, pentru o integrare rapidă și convenabilă. Printre furnizorii recomandați se numără furnizorul de servicii cloud OVHcloud și platforma cloud de dezvoltare software cuantică qBraid.

## Medii de notebook recomandate cu Qiskit preinstalat

Medii suplimentare de notebook Jupyter online, precum IBM Watson&reg; Studio, Google Colab și Microsoft Azure Machine Learning Studio, pot fi utilizate pentru a lucra cu Qiskit urmând instrucțiunile corespunzătoare.

### Notebook-uri AI OVHcloud

Utilizatorii de OVHcloud Public Cloud și OVHcloud Startup pot folosi notebook-urile lor cuantice OVHcloud pentru a accesa Qiskit și a trimite job-uri către QPU-urile IBM&reg; cu contul lor IBM Quantum&reg; sau IBM Cloud&reg;. Pentru mai multe informații, vizitează pagina [Documentație și tutoriale OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Notebook-urile AI OVHcloud nu sunt disponibile în unele regiuni, cum ar fi Statele Unite. De asemenea, interfețele utilizator pot arăta diferit în regiuni diferite.
</Admonition>

Urmează acești pași pentru a începe:

1. [Creează un cont IBM Cloud.](./cloud-setup)
1. [Configurează un cont OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Creează un proiect OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Urmează pașii din [Lansează primul tău AI Notebook,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) selectând **Qiskit** pentru framework.

### qBraid Lab

Utilizatorii Qiskit pot folosi mediile Python preconfigurate de qBraid urmând aceste instrucțiuni. Pentru mai multe informații, consultă [Documentația qBraid.](https://docs.qbraid.com/home/introduction)

1. [Configurează un cont gratuit qBraid.](https://docs.qbraid.com/home/account)
3. [Creează un mediu Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Lansează qBraid Lab, dă clic pe pictograma **ENVS** din dreapta, dă clic pe **+ ADD** în partea de sus a noii bare laterale și caută Qiskit. Dă clic pe versiunea pe care vrei să o folosești, apoi dă clic pe **Install**.
4. Creează un notebook Jupyter dând clic pe **Python 3 [Qiskit]** în secțiunea Notebook.
4. Poți instala orice pachete Python suplimentare (cum ar fi Qiskit v1.2) într-o celulă de notebook rulând `%pip install [package_name].` Pentru mai multe informații, consultă subiectul [Comenzi Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Medii de notebook suplimentare pentru utilizatorii Qiskit

Pentru platformele fără Qiskit preconfigurat, instrucțiunile de configurare sunt similare, cu excepția faptului că trebuie să instalezi Qiskit manual. Va trebui să îți creezi un cont, să pornești un proiect nou (sau notebook sau spațiu de lucru, în funcție de platformă) și să instalezi Qiskit.

### IBM Watson Studio

1. [Configurează versiunea de probă gratuită Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Creează un proiect Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Pentru a instala pachetele Qiskit și dependențele necesare, deschide un notebook nou și rulează următoarea comandă:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab

1. [Creează un cont Google.](https://support.google.com/accounts/answer/27441?hl)
2. Accesează [pagina de start Google Colab](https://colab.research.google.com/) și autentifică-te cu contul tău Google.
3. [Creează un notebook Colab](https://colab.research.google.com/#create=true) și instalează Qiskit rulând următoarea comandă:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio

1. [Creează un cont Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Creează un spațiu de lucru.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Creează un notebook Jupyter în spațiul tău de lucru.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Din notebook, rulează această comandă pentru a instala Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
