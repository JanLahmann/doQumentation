## Performanță

Această secțiune prezintă calculele de referință demonstrate ale HI-VQE pentru un caz cu 24 de qubiți pentru Li2S, un caz cu 40 de qubiți pentru o moleculă N2 și un caz cu 44 de qubiți pentru un sistem FeP-NO.
#### Curba suprafeței de energie potențială de disociere pentru o moleculă Li2S cu 24 de qubiți

Curba PES este prezentată cu referința FCI și estimarea inițială din RHF, împreună cu eroarea de energie față de referința FCI.

![Imagine care arată că HI-VQE produce soluții în limita acurateței chimice față de o curbă PES de referință clasică pentru sistemul Li2S.](/docs/images/guides/qunova-chemistry/Li2S_PES.avif)

Calculele au fost efectuate cu următoarele geometrii și opțiuni.

```python
# This cell is hidden from users
backend_name = service.least_busy(operational=True, min_num_qubits=38).name
```

```python
# Define Li2S geometries
Li2S_geoms = {
    "Li2S_1.51": "S        -1.239044    0.671232   -0.030374;Li       -1.506327    0.432403   -1.498949;Li       -0.899996    0.973348    1.826768;",
    "Li2S_2.40": "S        -1.741432    0.680397    0.346702;Li       -0.529307    0.488006   -1.729343;Li       -1.284307    0.989409    2.177209;",
    "Li2S_3.80": "S        -2.707255    0.674298    0.909161;Li        0.079218    0.552012   -1.671656;Li       -0.927010    0.931502    1.557063;",
}

# Configure some options for the job.
molecule_options = {
    "basis": "sto3g",
}
hivqe_options = {
    "shots": 100,
    "max_iter": 20,
}

results = []
for geom in ["Li2S_1.51", "Li2S_2.40", "Li2S_3.80"]:
    # Run HI-VQE
    job = function.run(
        geometry=Li2S_geoms[geom],
        backend_name=backend_name,  # can use any device with at least 38 qubits
        max_states=2000,
        max_expansion_states=10,
        molecule_options=molecule_options,
        hivqe_options=hivqe_options,
    )
    results.append(job.result())
```

Punctele roșii reprezintă rezultatele calculului HI-VQE pentru șase geometrii diferite, iar trei geometrii corespunzătoare valorilor de 1,51, 2,40 și 3,80 Angstromi sunt furnizate ca intrare în celula de mai sus.
#### Curba PES de disociere pentru o moleculă N2 cu 40 de qubiți

Molecula de azot a fost identificată ca un sistem multireference cu contribuții mari de energie de corelație dincolo de starea Hartree-Fock. Am efectuat un calcul de referință pentru molecula N2 cu baza cc-pvdz, (20o,14e) folosind selecția orbitală activă homo-lumo. Numărul spațiului activ complet (CAS) pentru a reprezenta această problemă este 6.009.350.400. Nu este posibil să obții soluția problemei de valori proprii (pentru energie și structură electronică) cu acest număr de stări folosind un desktop performant (16cpu/64GB). Cu HI-VQE, poți căuta eficient subspațiul stărilor CAS pentru a găsi rezultate cu acuratețe chimică, economisind semnificativ resursele de calcul. Graficele următoare arată curba PES a calculului HI-VQE cu 40 de qubiți pentru disocierea moleculei N2.

![Imagine care arată că HI-VQE produce soluții în limita acurateței chimice față de o curbă PES de referință clasică pentru sistemul N2.](/docs/images/guides/qunova-chemistry/N2_PES_40qubits.avif)
#### Curba PES de disociere pentru fier(II)-porfirina penta-coordonată cu un sistem NO cu 44 de qubiți

Un alt sistem chimic interesant este complexul fier(II)-porfirina (FeP) cu un ligand de oxid nitric (NO) coordonat, care reprezintă un sistem metaloporfirinic relevant biologic ce joacă roluri esențiale în diverse procese fiziologice. În acest exemplu, HI-VQE a fost utilizat pentru a estima curba exactă a suprafeței de energie potențială a interacțiunii intermoleculare dintre FeP și NO (energia stării fundamentale pentru geometrii separate diferit). Sistemul combinat are 450 de orbitali și 202 de electroni (450o,202e) cu baza 6-31g(d) în total. Selecția orbitală activă homo-lumo a fost utilizată pentru a calcula cazul mai mic față de cazul real cu (22o,22e). Din rezultatele de referință următoare, am reușit să atingem acuratețea chimică (>&nbsp;1,6 mHa) față de un calcul de chimie cu calculator clasic de ultimă generație CASCI(DMRG) (22o,22e) de referință.

## Valori de referință

- Dimensiunea exactă a matricei reprezintă numărul de determinanți pentru soluția exactă, cum ar fi FCI și CASCI.
- Calculul HI-VQE eșantionează și calculează subspațiul acesteia (adică dimensiunea matricei HI-VQE).
- Timpul total include timpul de execuție QPU și rulările Qiskit Function cu CPU.
- Acuratețea este estimată din diferența de energie față de soluția exactă.
