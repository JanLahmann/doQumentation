
Apoi Alice criptează secretul lung folosind cheia publică a lui Bob, iar secretul criptat îi este comunicat lui Bob.

```python
Alice_encrypted_secret = public_key_Bob.encrypt(
    Alice_long_secret,
    padding.OAEP(
        mgf=padding.MGF1(algorithm=hashes.SHA256()),
        algorithm=hashes.SHA256(),
        label=None,
    ),
)
print("Alice's secret encrypted")
```

Bob decriptează secretul criptat primit de la Alice folosind cheia sa privată.

```python
Bob_decrypted_secret = private_key_Bob.decrypt(
    Alice_encrypted_secret,
    padding.OAEP(
        mgf=padding.MGF1(algorithm=hashes.SHA256()),
        algorithm=hashes.SHA256(),
        label=None,
    ),
)

assert Alice_long_secret == Bob_decrypted_secret, "Secrets do not match!"

# if we get here they match
print("Secrets match")
```

În final, atât Alice cât și Bob aplică separat o [Funcție de Derivare a Cheii](https://en.wikipedia.org/w/index.php?title=Key_derivation_function&oldid=1165650237) (KDF) convenită în prealabil asupra secretului lung, pentru a deriva cheia simetrică.

De reținut că procesul implică un protocol de hashing și utilizarea unui <DefinitionTooltip definition="Regarding password hashing, the idea that salt enhances/makes a food unique is the same idea of how adding salt in this context makes our password unique, distinctive, and hard to reproduce.">*salt*</DefinitionTooltip> aleatoriu, care asigură unicitatea și impredictibilitatea cheii simetrice partajate în cazul în care secretul lung este refolosit vreodată (lucru nerecomandat). Cu toate acestea, *salt*-ul în sine nu trebuie să fie secret, iar odată generat aleatoriu — de exemplu de Alice în acest exemplu — poate fi transmis lui Bob alături de secretul lung criptat.

Vom presupune, prin urmare, că atât Alice cât și Bob au acces la același *salt* aleatoriu.

```python
def key_derivation_function(secret, salt):
    hkdf = HKDF(
        algorithm=hashes.SHA256(),
        length=32,  # Desired key length
        salt=salt,
        info=None,
        backend=None,
    )
    return hkdf.derive(secret)

common_salt = urandom(16)  # Random salt accessible to both Alice and Bob

symmetric_key_Alice = key_derivation_function(Alice_long_secret, common_salt)
symmetric_key_Bob = key_derivation_function(Bob_decrypted_secret, common_salt)

assert symmetric_key_Alice == symmetric_key_Bob, "Derived keys do not match!"
print(
    f"A symmetric key of length {len(symmetric_key_Alice)*8} bits was successfully derived by both Alice and Bob!"
)
```

#### Semnături digitale cu RSA {#digital-signatures-with-rsa}

Vom extinde acum scenariul de comunicare confidențială dintre Alice și Bob de mai sus pentru a include și validarea, cu ajutorul unei [semnături digitale](https://en.wikipedia.org/w/index.php?title=Digital_signature&oldid=1161173191).

Ca și înainte, Alice îi va trimite lui Bob un mesaj confidențial ce înglobează o cheie simetrică, dar ea va semna digital și mesajul, astfel încât Bob, la primirea acestuia, să poată verifica că Alice a fost cea care l-a inițiat și că conținutul mesajului nu a fost modificat în timpul transmisiei.

Mai general, este de dorit să se permită validarea fără a compromite confidențialitatea, astfel încât orice parte interesată să poată verifica <DefinitionTooltip definition="Integrity is maintained when a message is protected against unauthorized change.">integritatea</DefinitionTooltip>, <DefinitionTooltip definition="Authenticity is the proof that a message is from the source it claims to be from.">autenticitatea</DefinitionTooltip> și să stabilească non-repudierea în raport cu o anumită comunicare, chiar dacă acea parte nu are acces la mesajul în clar propriu-zis.

Vom considera acest cadru general, care implică următorii pași:

- Mai întâi, atât Bob cât și Alice își fac cheile publice disponibile printr-un canal deschis.
- Apoi, Alice criptează textul clar folosind cheia publică a lui Bob, creând un text cifrat.
- Următor, Alice creează un hash al textului cifrat cu o *funcție hash* și criptează în continuare textul cifrat hashat rezultat folosind cheia sa privată. Acest hash criptat reprezintă *semnătura*.
- Apoi, Alice transmite atât textul cifrat cât și semnătura printr-un canal deschis.
- Bob folosește cheia publică a lui Alice pentru a decripta semnătura, obținând textul cifrat hashat.
- Următor, deoarece Bob are acces și la textul cifrat propriu-zis, folosește aceeași funcție hash utilizată de Alice pentru a recrea o a doua instanță a textului cifrat hashat. Dacă aceasta corespunde celei obținute prin decriptarea semnăturii lui Alice, mesajul este validat, chiar dacă textul cifrat în sine nu a fost încă decriptat.
- În final, Bob, după ce a validat mesajul, decriptează textul cifrat folosind propria sa cheie privată.

Importăm din nou câteva module utile din biblioteca ``cryptography``.
Ca și înainte, Alice intenționează să îi trimită în mod securizat lui Bob o cheie simetrică, dar dorește și să o semneze digital. În acest caz, avem nevoie de chei publice atât pentru Alice cât și pentru Bob. Prin urmare, primul pas este ca atât Alice cât și Bob să își creeze propriul set de chei folosind RSA și să își publice cheia publică.

```python
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.asymmetric import padding, rsa
from cryptography.hazmat.primitives.asymmetric.utils import Prehashed

# Generate keys for Bob
bob_private_key = rsa.generate_private_key(public_exponent=65537, key_size=2048)
bob_public_key = bob_private_key.public_key()

# Generate keys for Alice
alice_private_key = rsa.generate_private_key(public_exponent=65537, key_size=2048)
alice_public_key = alice_private_key.public_key()

print("Private and Public keys generated for Bob and Alice.")
```

În pasul următor, ca și înainte, Alice folosește cheia publică a lui Bob pentru a cripta cheia simetrică și pregătește textul cifrat.

```python
# Alice encrypts the message using Bob's public key
ciphertext = bob_public_key.encrypt(
    symmetric_key,
    padding.OAEP(
        mgf=padding.MGF1(algorithm=hashes.SHA256()),
        algorithm=hashes.SHA256(),
        label=None,
    ),
)

print("ciphertext of symmetric key: ", ciphertext)
```
