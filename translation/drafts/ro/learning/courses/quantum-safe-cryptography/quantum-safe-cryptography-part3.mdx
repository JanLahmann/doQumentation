### Ilustrarea criptării LWE în Python

Următorul exemplu simplu arată utilizarea <DefinitionTooltip definition="Learning With Errors. O tehnică matematică care ascunde informații secrete prin introducerea de erori în ecuații, făcându-le mai greu de rezolvat.">LWE</DefinitionTooltip> pentru criptare și decriptare. Bob îi va trimite un mesaj criptat lui Alice.

Mai întâi, Alice și Bob convin asupra **parametrilor problemei**. Aceștia sunt explicați în detaliu în secțiunea matematică de mai sus, dar pe scurt avem nevoie de $n, q, N, \chi$.

Începem cu câțiva dintre parametrii de bază

- $N$ reprezintă numărul de eșantioane
- $q$ este un modul
- n este cunoscut ca parametrul de securitate, sau dimensiunea vectorului.

Toți acești parametri sunt, în principiu, informații publice.

```python
import numpy as np

n = 8
q = 127
N = int(1.1 * n * np.log(q))
sigma = 1.0
print(f"n={n},q={q},N={N},sigma={sigma}")
```

Avem nevoie și de o modalitate de a introduce erori

Aici $\chi$ reprezintă erorile pe care vrem să le introducem - folosim o distribuție Gaussiană discretă pe $\mathbb{Z}_{q}$ caracterizată prin media 0 și deviația standard $\sigma$.

Această definiție a erorilor de introdus este, de asemenea, publică.

Deci, în Python, avem nevoie de o funcție simplă care aproximează contribuțiile aleatorii de eroare/zgomot $\varepsilon$ extrase dintr-o distribuție Gaussiană discretă $\chi$ cu deviație standard $\sigma$.

Sunt afișate câteva valori exemplu, astfel încât să poți vedea distribuția în jurul lui 0.

```python
def chi(stdev, modulus):
    return round((np.random.randn() * stdev**2)) % modulus

# print some examples
sd = 2
m = 1000
for x in range(10):
    print("chi = ", chi(sd, m))
```

Apoi, Alice trebuie să-și genereze perechea de chei.

Ea își configurează **cheia privată** alegând $n$ valori între 0 și $q$.

```python
# Alice's private key
alice_private_key = np.random.randint(0, high=q, size=n)
print(f"Alice's private key: {alice_private_key}")
```

Alice își configurează acum **cheia publică**, alegând vectori aleatori, care sunt apoi combinați cu erorile generate.

<details>
<summary>Matematică</summary>

Mai precis, ea trebuie să genereze eșantionul $A_{s,\chi}$.

În consecință, ea alege aleator vectori $\mathbf{a} \in \mathbb{Z}_{q}^{n}$ și erori $\varepsilon$ din $\chi$ pentru a construi $N$ eșantioane $(\mathbf{a}, b)$.

</details>

```python
# Alice's Public Key
alice_public_key = []

# N is the number of values we want in the key
for i in range(N):
    # Get n random values between 0 and <q
    a = np.random.randint(0, high=q, size=n)
    # get an error to introduce
    epsilon = chi(sigma, q)
    #  calculate dot product (ie like array multiplication)
    b = (np.dot(a, alice_private_key) + epsilon) % q
    # value to be added to the key -
    sample = (a, b)
    alice_public_key.append(sample)

print(f"Alice's public key: {alice_public_key}")
```

Alice poate acum să-și partajeze cheia publică.

Bob o poate folosi acum pentru a-i trimite lui Alice un mesaj criptat.

Mesajul lui Bob este un singur bit.

```python
# Encryption
bob_message_bit = 1
print(f"Bob's message bit={bob_message_bit}")
```

Pentru a cripta mesajul, Bob trebuie să selecteze un număr arbitrar de eșantioane aleator din cheia publică a lui Alice pentru a forma textul cifrat.

În acest scop, el creează o mască, un vector binar aleator $r$ de lungime $N$.

```python
# a list of N values between 0 and <2 - ie 0 or 1
r = np.random.randint(0, 2, N)
print(r)
```

Luăm acum această mască și o aplicăm intrării relevante din cheia publică a lui Alice, calculând o sumă a valorilor găsite.

```python
sum_ai = np.zeros(n, dtype=int)
sum_bi = 0

for i in range(N):
    sum_ai = sum_ai + r[i] * alice_public_key[i][0]
    sum_bi = sum_bi + r[i] * alice_public_key[i][1]
sum_ai = [x % q for x in sum_ai]
# sum_bi = sum_bi
ciphertext = (sum_ai, (bob_message_bit * int(np.floor(q / 2)) + sum_bi) % q)
print(f"ciphertext is: {ciphertext}")
```

În final, Bob transmite textul cifrat, pe care Alice îl poate decripta folosind cheia sa privată.

```python
# Decryption
adots = np.dot(ciphertext[0], alice_private_key) % q
b_adots = (ciphertext[1] - adots) % q

decrypted_message_bit = round((2 * b_adots) / q) % 2

print(
    f"original message bit={bob_message_bit}, decrypted message bit={decrypted_message_bit}"
)

assert bob_message_bit == decrypted_message_bit
```

Deoarece acest protocol funcționează bit cu bit pentru criptarea unor șiruri de biți mai lungi, pur și simplu repetăm operațiile într-o buclă. Cele ce urmează arată un scenariu în care Bob dorește să transfere 16 biți criptați.

```python
bob_message_bits = np.random.randint(0, 2, 16)
print(f"Bob's message bits are : {bob_message_bits}")
decrypted_bits = []

for ib in range(len(bob_message_bits)):
    bob_message_bit = bob_message_bits[ib]

    r = np.random.randint(0, 2, N)

    sum_ai = np.zeros(n, dtype=int)
    sum_bi = 0
    for i in range(N):
        sum_ai = sum_ai + r[i] * alice_public_key[i][0]
        sum_bi = sum_bi + r[i] * alice_public_key[i][1]
    sum_ai = [x % q for x in sum_ai]

    ciphertext = (sum_ai, (bob_message_bit * int(np.floor(q / 2)) + sum_bi) % q)

    adots = np.dot(ciphertext[0], alice_private_key) % q
    b_adots = (ciphertext[1] - adots) % q

    decrypted_message_bit = round((2 * b_adots) / q) % 2
    assert decrypted_message_bit == bob_message_bit

    decrypted_bits.append(decrypted_message_bit)

print(f"Decrypted message bits = {np.array(decrypted_bits)}")
```
