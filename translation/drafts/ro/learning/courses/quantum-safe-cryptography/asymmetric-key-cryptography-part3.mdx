
Va trebui să putem calcula <DefinitionTooltip definition="Greatest Common Divisor - the largest positive whole number that can divide two or more of the given whole numbers.">cel mai mare divizor comun</DefinitionTooltip> al două numere întregi, deoarece va fi necesar pentru a testa dacă două numere întregi sunt <DefinitionTooltip definition="Two numbers that have a greatest common divisor of 1. For example, 3 and 13 are coprime because only 1 can divide both of them.">coprime</DefinitionTooltip>.

Vom explica o modalitate simplă de a calcula aceasta, dar este mult mai eficient cu numere întregi mai mari să folosești funcția Python `math.gcd`.

```python
import math

# Example function to compute the gcd (greatest common divisor)
def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b)

# let's calculate some examples using algorithm
n1 = gcd(50, 10)
n2 = gcd(99, 33)
n3 = gcd(59, 9)

# do the same with the python library call

m1 = math.gcd(50, 10)
m2 = math.gcd(99, 33)
m3 = math.gcd(59, 9)

# Confirm they are the same
assert n1 == m1
assert n2 == m2
assert n3 == m3

# They are - print out the values for explanation
print("gcd(50,10) =", m1)
print("gcd(99,33) =", m2)
print("gcd(59,9) =", m3)
```

Prima fază a fluxului de lucru RSA este generarea cheilor. Aceasta este inițiată prin alegerea a două numere prime, care trebuie să rămână secrete pentru entitatea ce generează cheile.

```python
# Choosing two prime numbers and keep them secret
p = 13
q = 19
print("The secret prime numbers p and q are:", p, q)
```

Apoi se calculează <DefinitionTooltip definition="The modulus is the remainder left over after dividing by the value specified. modulo is the operation of doing this.">modulul</DefinitionTooltip> $n$, care este pur și simplu produsul celor două numere prime alese. Acest modul va fi publicat ca parte a cheii publice.

```python
# Calculate n which is the modulus for both the public and private keys
n = p * q
print("modulus n (p*q)=", n)
```

Funcția <DefinitionTooltip definition="The totient is a number, or numbers, that are not greater than the given number, and are relatively prime to it, having no factors in common.">totientului</DefinitionTooltip> lui Euler <DefinitionTooltip definition="The greek symbol, phi, often used to indicate Euler's totient.">$\varphi(n)$</DefinitionTooltip> este calculată în continuare, deoarece este necesară pentru operația de <DefinitionTooltip definition="Given an integer $a$, and modulus $m$, the remainder from ax/m is 1. This is only the case for coprimes.">invers multiplicativ modular</DefinitionTooltip> folosită pentru determinarea cheilor în RSA. $phi$ este de asemenea ținut secret și este de obicei eliminat după generarea cheilor.

```python
# Compute Euler's totient function, φ(n) and keep it secret
phi = (p - 1) * (q - 1)
print("The secret Euler's function (totient) [phi(n)]:", phi)
```

Suntem acum gata să calculăm cheile publică și privată. În RSA, fiecare dintre acestea este specificată printr-un tuplu de două numere întregi. Prima intrare din fiecare tuplu este un număr întreg distinct, iar a doua intrare este modulul $n$ comun ambelor chei.

Prima intrare din cheia publică poate fi orice număr întreg mai mare decât 1 care este coprim cu $phi$. Două numere întregi sunt coprime dacă <DefinitionTooltip definition="Greatest Common Divisor - the largest positive whole number that can divide two or more of the given whole numbers.">cel mai mare divizor comun</DefinitionTooltip> al lor este 1. Prin urmare, folosim funcția `math.gcd` pentru a găsi un număr întreg $e$ coprim cu $phi$.

```python
# Choose an integer e such that e and φ(n) are coprime
e = 2
while e < phi:
    if math.gcd(e, phi) == 1:
        break
    else:
        e += 1
print("Public Key (e):", e)
```

Cheia privată necesită un număr întreg $d$, care este <DefinitionTooltip definition="Also known as reciprocal, a number, which when multiplied by the number given, results in a value of 1.">inversul multiplicativ</DefinitionTooltip> al lui $e$ modulo $phi$; adică satisface <DefinitionTooltip definition="Consistency, or equivalence ie. all behaving in the same way.">relația de congruență</DefinitionTooltip> $d*e\equiv 1 \pmod{\varphi(n)}$. Pentru această ilustrare simplă în care lucrăm cu numere întregi mici, putem parcurge numerele întregi pozitive pentru a găsi un $d$ corespunzător. În situații reale, pentru acest scop se folosește <DefinitionTooltip definition="An efficient way of calculating the greatest common divisor of two whole numbers. It improves on the original Euclidean algorithm.">algoritmul euclidian extins</DefinitionTooltip>, care este eficient din punct de vedere computațional.

```python
# Compute a value for d such that (d * e) % φ(n) = 1
d = 1
while True:
    if (d * e) % phi == 1:
        break
    else:
        d += 1
print("Private Key (d):", d)
```

Formăm acum tuplurile $(e, n), (d, n)$, care constituie respectiv cheia publică și cheia privată. Cheia publică este apoi publicată, în timp ce cheia privată este ținută secretă.

```python
# Public and Private Key pair
public = (e, n)
private = (d, n)

print(f"The Public key is {public} and Private Key is {private}")
```

Criptarea și decriptarea în RSA folosesc [operația de exponențiere modulară](https://en.wikipedia.org/w/index.php?title=Modular_exponentiation&oldid=1160764843). Mai mult, cheile publică și privată sunt complementare, în sensul că oricare dintre ele poate fi folosită pentru a cripta un mesaj pe care cealaltă îl poate decripta.

Ilustrăm aici cazul în care cheia publică $(e,n)$ este folosită pentru criptare, iar cheia privată $(d, n)$ este folosită pentru decriptare, definind o funcție Python pentru fiecare operație.

Criptăm și decriptăm apoi un mesaj întreg $msg$.

```python
# Encryption function
def encrypt(plain_text):
    return (plain_text**e) % n

# Decryption function
def decrypt(cipher_text):
    return (cipher_text**d) % n
