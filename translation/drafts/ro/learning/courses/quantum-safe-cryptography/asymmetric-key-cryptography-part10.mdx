### Ilustrarea DSA în Python {#illustration-of-the-dsa-in-python}
În acest exemplu, vom folosi numere prime mici pentru a ilustra procesul DSA într-un scenariu în care Alice trimite un mesaj semnat lui Bob. Vom folosi numere întregi mici în acest exemplu. Un scenariu din lumea reală ar utiliza numere prime mult mai mari, de ordinul a 2048-3072 de biți.

<Admonition type="note">
Poți încerca să rulezi din nou acest exemplu cu valori diferite pentru a vedea cum se comportă algoritmul, dar asigură-te că începi să rulezi de la prima celulă din această secțiune.
</Admonition>

Începem prin a importa bibliotecile necesare și a alege parametrii noștri.
Parametrii întregi $p$, $q$, $g$ sunt informații publice și satisfac următoarele reguli:

- $p$, $q$ sunt ambele prime
- $(p-1) \mod\ q = 0$
- $g \ge 2$
- $g \le (p-2)$
- $g^{(p-1)/q} \mod\ p \neq 1$

```python
from random import randint

# parameter generation: select the primes q, p and generator g:
# EXPERIMENT with the values, they must meet certain rules
# this example code does not verify p,q are prime

q = 11
p = 23
g = 4

assert (p - 1) % q == 0
assert g >= 2
assert g <= (p - 2)
assert (pow(g, (p - 1) / q) % p) != 1

print(f"Public information is good: q={p}, p={q}, g={g}")
```

Apoi semnatarul, Alice, își generează cheia privată.

Cheia privată, k (`alice-private-key` în codul Python) trebuie să satisfacă:
 - $k \ge 2$
 - $k \le (q-1)$

```python
# Alice chooses an integer randomly from {2..q-1}
# EXPERIMENT with the values

alice_private_key = randint(2, q - 1)
# alice_private_key =

assert alice_private_key >= 2
assert alice_private_key <= (q - 1)

print(f"Alice's private key is {alice_private_key}")
```

Alice își păstrează cheia privată secretă.

Apoi, Alice își calculează cheia publică folosind exponențierea modulară. Inversarea acestui pas pentru a recupera cheia privată este o instanță a DLP, deci dificil de calculat pe calculatoare clasice atunci când se folosesc numere prime mari.

Din explicația matematică de mai sus, știm că aceasta poate fi calculată folosind formula:

$y = g^{x} \mod\ p$

```python
alice_public_key = pow(g, alice_private_key, p)
# Alternatively can use (g ** alice_private_key) % p

print(f"Alice's public key is {alice_public_key}")
```

Ca de obicei, Alice pune cheia sa publică la dispoziție printr-un canal care nu trebuie neapărat să fie secret.

Acum Alice poate semna un mesaj.

Pentru schema de semnătură digitală, trebuie să generăm un hash $H(m)$ al mesajului $m$ care urmează să fie semnat.

Să presupunem că Alice și Bob sunt de acord să folosească un anumit algoritm de hashing cu lungimea hash-ului $N$ egală cu numărul de biți din $q$. În acest exemplu simplu, vom limita rezultatele funcției noastre mock de hash la $q$. Funcția hash de aici este trivială, generând pur și simplu un număr întreg aleatoriu.

```python
hash_dict = {}

def mock_hash_func(input_message):
    print(input_message)
    if input_message not in hash_dict:
        hash_dict[input_message] = randint(1, q)
    return hash_dict[input_message]

alice_message = "Inspection tomorrow!"
alice_hash = mock_hash_func(alice_message)  # In reality, you'd use a hash function
print(f"Alice's message hash is: {alice_hash}")
```

În continuare, Alice trebuie să genereze un număr secret aleatoriu per-mesaj $k$, precum și inversul său multiplicativ modulo $q$ pentru a calcula semnătura.

Un algoritm simplu de forță brută poate fi folosit pentru a calcula inversul modular. Este însă mai bine să folosești funcția Python built-in `pow`, așa cum este arătat mai jos.

```python
# brute-force implementation to find modular inverse
def modular_inverse(k, q):
    for i in range(0, q):
        if (k * i) % q == 1:
            return i
    print(f"error! no inverse found! for {k},{q}")
    return 0

# Let's compare this algorithm with the standard python 'pow' function

n1 = modular_inverse(3, 7)
n2 = modular_inverse(4, 11)
n3 = modular_inverse(7, 5)
m1 = pow(3, -1, 7)
m2 = pow(4, -1, 11)
m3 = pow(7, -1, 5)

assert n1 == m1
assert n2 == m2
# assert(n3==m3)

print(f"modular_inverse(3,7) = {m1}")
