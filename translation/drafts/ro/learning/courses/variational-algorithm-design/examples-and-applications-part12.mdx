Orice optimizator clasic poate fi folosit pentru a minimiza funcția de cost. Pe un sistem cuantic real, un optimizator proiectat pentru peisaje de funcții de cost non-liniare funcționează de obicei mai bine. Aici folosim [rutina COBYLA](https://docs.scipy.org/doc/scipy/reference/optimize.minimize-cobyla.html#optimize-minimize-cobyla) din SciPy prin intermediul funcției minimize.

Deoarece executăm iterativ multe apeluri către Runtime, folosim o sesiune pentru a executa toate apelurile în cadrul unui singur bloc. Mai mult, pentru QAOA, soluția este codificată în distribuția de ieșire a circuitului ansatz legat cu parametrii optimi din minimizare. Prin urmare, vom avea nevoie de un primitiv Sampler, pe care îl vom instanția cu aceeași sesiune.
Și rulăm rutina noastră de minimizare:

```python
result = minimize(
    cost_func, x0, args=(max_ansatz, max_hamiltonian, estimator), method="COBYLA"
)
```
