# Prezentare generală {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Acest curs este destinat persoanelor care au deja experiență cu calculul cuantic și doresc să învețe cum să scrie algoritmi variaționali folosind primitivele Qiskit Runtime. Este de asemenea potrivit pentru profesioniștii din domeniul calculului cuantic care doresc să-și extindă cunoștințele și abilitățile în domeniul algoritmilor variaționali.

## Examen {#exam}

Pentru a obține insigna pentru Variational algorithm design, susține examenul pe platforma IBM&reg; Training. Acest examen este destinat să fie susținut după parcurgerea lecțiilor din acest curs. După ce vei trece examenul, vei fi notificat de Credly că ai obținut o insignă.
