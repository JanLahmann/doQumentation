### Exercițiul 1: Estimarea valorii $r_x$ {#exercise-1-estimate-r_x-value}
Reamintește-ți că computerele cuantice IBM&reg; măsoară de-a lungul axei $z$ (uneori formulat „în baza $z$" sau „în baza computațională"). Totuși, folosind rotații înainte de măsurare, putem măsura și proiecția statului cuantic pe axa x. Mai precis, dacă rotim sistemul nostru astfel încât lucrurile care indicau anterior spre $x$ să indice acum spre $z$, atunci putem păstra același hardware de măsurare de-a lungul lui $z$, dar să aflăm informații despre statul care tocmai indica spre $x$ cu puțin timp în urmă. Acesta este modul în care majoritatea computerelor cuantice (și toate computerele cuantice IBM) efectuează măsurători de-a lungul mai multor axe.

Cu această înțelegere, încearcă să scrii cod pentru a estima valoarea lui $r_x,$ folosind tomografia statului cuantic.

__Soluție__:

```python
# create a 1-qubit quantum state psi from theta, varphi parameters
qc = QuantumCircuit(1, 1)
qc.u(theta, varphi, 0.0, 0)

qc.h(0)
qc.measure(0, 0)

qc.draw(output="mpl")
```

![Output of the previous code cell](/learning/images/courses/utility-scale-quantum-computing/teleportation/extracted-outputs/fcecbe31-9196-4804-8afa-e23e313880df-0.avif)

```python
# Define backend
backend = AerSimulator()
nshots = 10000

# Transpile to backend
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

# Run the job
sampler = Sampler(mode=backend)
job = sampler.run([isa_qc], shots=nshots)
result = job.result()

# Extract counts data
counts = result[0].data.c.get_counts()
print(counts)

# Plot the counts in a histogram
plot_histogram(counts)
```

```text
{'1': 5925, '0': 4075}
```

```python

print("rx = ", rx, " and approx of rx = ", rx_approx)
```


### Exercițiul 2: Estimarea valorii $r_y$ {#exercise-2-estimate-r_y-value}
Folosind aceleași argumente logice ca mai înainte, putem roti sistemul înainte de măsurare pentru a afla informații despre $r_y$.
Încearcă să scrii cod singur pentru a estima valoarea lui $r_y$ folosind tomografia statului cuantic. Poți porni de la exemplul anterior, dar aplică rotații diferite. (Pentru mai multe informații despre diferitele Gate-uri folosite, inclusiv ```sdg```, consultați [referința API.](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.SdgGate))

__Soluție__:

```python
# create a 1-qubit quantum state psi from theta, varphi parameters
qc = QuantumCircuit(1, 1)
qc.u(theta, varphi, 0.0, 0)

qc.sdg(0)
qc.h(0)
qc.measure(0, 0)

qc.draw(output="mpl")
```

```python
backend = AerSimulator()
```
