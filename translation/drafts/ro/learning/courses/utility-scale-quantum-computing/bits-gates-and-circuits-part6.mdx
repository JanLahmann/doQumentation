### 4.2 Porțile cuantice cu mai mulți qubiți {#42-multi-qubit-quantum-gates}
#### Poarta CNOT {#cnot-gate}

O poartă CNOT (din engleza „controlled NOT" sau CX) este o poartă cu doi qubiți, ceea ce înseamnă că acțiunea ei implică doi qubiți simultan: qubit-ul de control și qubit-ul țintă. CNOT inversează qubit-ul țintă numai când qubit-ul de control este $|1\rangle$.

| Intrare (țintă, control) | Ieșire (țintă, control) |
|:-----------:|:------------:|
| 00          | 00           |
| 01          | 11           |
| 10          | 10           |
| 11          | 01           |

Hai să simulăm mai întâi acțiunea acestei porți cu doi qubiți când q0 și q1 sunt ambii $|0\rangle$ și să obținem statevector-ul de ieșire. Sintaxa Qiskit folosită este ```qc.cx(qubit control, qubit țintă)```.

```python
# Create a circuit with two quantum registers and two classical registers
qc = QuantumCircuit(2, 2)

# Apply the CNOT (cx) gate to a |00> state.
qc.cx(0, 1)  # Here the control is set to q0 and the target is set to q1.

# Draw the circuit
qc.draw(output="mpl")
```

```python

print(out_vector)
```


Conform așteptărilor, aplicarea unei porți CNOT pe $|00\rangle$ nu a schimbat starea, deoarece qubit-ul de control era în starea $|0\rangle$.
Hai să revenim la operația noastră CNOT. De data aceasta vom aplica o poartă CNOT pe $|01\rangle$ și să vedem ce se întâmplă.

```python
qc = QuantumCircuit(2, 2)

# q0=1, q1=0
qc.x(0)  # Apply a X gate to initialize q0 to 1
qc.cx(0, 1)  # Set the control bit to q0 and the target bit to q1.

# Draw the circuit
qc.draw(output="mpl")
```


Prin aplicarea unei porți CNOT, starea $|01\rangle$ a devenit acum $|11\rangle$.

Hai să verificăm aceste rezultate rulând circuitul pe un simulator.

```python
qc.measure(0, 0)
qc.measure(1, 1)

# Draw the circuit
qc.draw(output="mpl")
```

### 4.3 Entanglementul cuantic și execuția pe un dispozitiv cuantic real {#43-quantum-entanglement-and-execution-on-a-real-quantum-device}

Hai să începem prin a introduce o stare entanglată specifică care este deosebit de importantă în computația cuantică, după care vom defini termenul „entanglat":

$$
\frac{1}{\sqrt{2}}|00\rangle + \frac{1}{\sqrt{2}}|11\rangle
$$
și această stare se numește **stare Bell**.

O stare entanglată este o stare $|\psi_{AB}\rangle$ formată din stările cuantice $|\psi_A\rangle$ și $|\psi_B\rangle$ care nu poate fi reprezentată ca un produs tensorial al stărilor cuantice individuale.
