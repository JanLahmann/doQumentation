## 3. Strategia 2. Arbore echilibrat de qubiți {#3-strategy-2-balanced-tree-of-qubits}

Următoarea idee este să găsim un arbore echilibrat de qubiți. Folosind arborele în loc de lanț, adâncimea circuitului ar trebui să fie mai mică. Înainte de aceasta, eliminăm nodurile cu erori de citire „proaste" și muchiile cu erori de gate „proaste" din graful de cuplare.

```python
BAD_READOUT_ERROR_THRESHOLD = 0.1
BAD_ECRGATE_ERROR_THRESHOLD = 0.1
bad_readout_qubits = [
    q
    for q in range(backend.num_qubits)
    if backend.target["measure"][(q,)].error > BAD_READOUT_ERROR_THRESHOLD
]
bad_ecrgate_edges = [
    qpair
    for qpair in backend.target["ecr"]
    if backend.target["ecr"][qpair].error > BAD_ECRGATE_ERROR_THRESHOLD
]
print("Bad readout qubits:", bad_readout_qubits)
print("Bad ECR gates:", bad_ecrgate_edges)
```

```text
Bad readout qubits: [19, 28, 41, 72, 91, 114, 120]
Bad ECR gates: []
```

```python
g = backend.coupling_map.graph.copy().to_undirected()
g.remove_edges_from(
    bad_ecrgate_edges
)  # remove edge first (otherwise might fail with a NoEdgeBetweenNodes error)
g.remove_nodes_from(bad_readout_qubits)
```

Încercăm să creăm o stare GHZ cu 16 qubiți ca înainte.

N = 16

Apelăm funcția `betweenness_centrality` pentru a găsi un qubit pentru nodul rădăcină. Nodul cu cea mai mare valoare a centralității de betweenness se află în centrul grafului. Referință: https://www.rustworkx.org/tutorial/betweenness_centrality.html

Sau îl poți selecta manual.

```python
# central = 65 #Select the center node manually
c_degree = dict(rx.betweenness_centrality(g))
central = max(c_degree, key=c_degree.get)
central
```

```text

```

Pornind de la nodul rădăcină, generăm un arbore prin căutare în lățime (BFS). Referință: https://qiskit.org/ecosystem/rustworkx/apiref/rustworkx.bfs_search.html#rustworkx-bfs-search

```python
class TreeEdgesRecorder(rx.visit.BFSVisitor):
    def __init__(self, N):
        self.edges = []
        self.N = N

    def tree_edge(self, edge):
        self.edges.append(edge)
        if len(self.edges) >= self.N - 1:
            raise rx.visit.StopSearch()

vis = TreeEdgesRecorder(N)
rx.bfs_search(g, [central], vis)
best_qubits = sorted(list(set(q for e in vis.edges for q in (e[0], e[1]))))
# print('Tree edges:', vis.edges)
```
