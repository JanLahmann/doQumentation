## Fundal {#background}

Acest tutorial demonstrează cum să rezolvi o problemă de optimizare binară de ordin superior (HOBO) folosind [Optimization Solver, o funcție Qiskit de la Q-CTRL Fire Opal](/guides/q-ctrl-optimization-solver). Exemplul prezentat în acest tutorial este o problemă de optimizare concepută pentru a găsi energia stării de bază a unui model Ising cu 156 qubiți și legături aleatoare, care conține termeni cubici. Optimization Solver poate fi utilizat pentru probleme de optimizare generale ce pot fi definite ca o funcție obiectiv.

Optimization Solver automatizează complet pașii de implementare conștienți de hardware pentru rezolvarea problemelor de optimizare pe hardware cuantic și, prin utilizarea [Performance Management](/guides/q-ctrl-performance-management) pentru execuția cuantică, obține soluții precise la scară utilă. Pentru o prezentare detaliată a fluxului de lucru complet al Optimization Solver și a rezultatelor de benchmarking, consultă [manuscrisul publicat](https://arxiv.org/abs/2406.01743).

Acest tutorial parcurge următorii pași:

1. Definirea problemei ca funcție obiectiv
2. Rularea algoritmului hibrid folosind Fire Opal Optimization Solver
3. Evaluarea rezultatelor

## Pasul 2: Rulează algoritmul hibrid folosind Fire Opal Optimization Solver {#step-2-run-the-hybrid-algorithm-using-the-fire-opal-optimization-solver}
Acum folosește Qiskit Function Optimization Solver pentru a rula algoritmul. În fundal, Optimization Solver se ocupă de maparea problemei la un algoritm cuantic hibrid, rularea circuitelor cuantice cu supresie a erorilor și efectuarea optimizării clasice.

```python
# Authenticate to the Qiskit Functions Catalog
catalog = QiskitFunctionsCatalog(
    token=token,
    instance=instance,
)

# Load the function
solver = catalog.load("q-ctrl/optimization_solver")
```

Verifică că device-ul ales are cel puțin 156 de qubiți.

```python
# Specify the target backend name
backend_name = "<CHOOSE_A_BACKEND>"
```

Solver-ul acceptă o reprezentare sub formă de șir de caractere a funcției obiectiv.

```python
# Convert the objective function to string format
spin_glass_poly_as_str = srepr(spin_glass_poly)
```

```python
# Run the problem
spin_glass_job = solver.run(
    problem=spin_glass_poly_as_str,
    run_options={"backend_name": backend_name},
)
```

Poți folosi familiarele [API-uri Qiskit Serverless](/guides/serverless) pentru a verifica statusul sarcinii Qiskit Function:

```python
# Get job status
spin_glass_job.status()
```

Solver-ul returnează un dicționar cu soluția și metadatele asociate, cum ar fi șirul de biți al soluției, numărul de iterații și maparea variabilelor la șirul de biți. Pentru o definiție completă a intrărilor și ieșirilor Solver-ului, consultă [documentația]().

```python
# Poll for results
result = spin_glass_job.result()
```

```python
# Get the final bitstring distribution and set the number of shots
distribution = result["final_bitstring_distribution"]
```

def plot_cost_histogram(
    costs, probabilities, distribution, qubit_count, bitstring_cost
):
    """Plots a histogram comparing the cost distributions of Q-CTRL Solver and random sampling."""

    # Set figure DPI for higher resolution and font size for labels
    plt.rcParams["figure.dpi"] = 300
    plt.rcParams.update({"font.size": 6})  # Set default font size to 6

    # Define labels and colors for the plot
    labels = ["Q-CTRL Solver", "Random Sampling"]
    colors = ["#680CE9", "#E04542"]

    # Calculate total shots (total number of bitstrings in the distribution)
    shots = sum(distribution.values())

    # Generate random bitstrings for comparison (random sampling)
    rng = np.random.default_rng(seed=0)
    random_array = rng.integers(
        0, 2, size=(shots, qubit_count)
    )  # Generate random bitstrings (0 or 1 for each qubit)
    random_bitstrings = ["".join(row.astype(str)) for row in random_array]

    # Compute the cost for each random bitstring
    random_costs = [bitstring_cost(k) for k in random_bitstrings]

    # Set uniform probabilities for the random sampling
    random_probabilities = (
        np.ones(shape=(shots,)) / shots
    )  # Equal probability for each random bitstring

    # Find the minimum and maximum costs for binning the histogram
    min_cost = np.min(costs)
    max_cost = np.max(random_costs)

    # Create a histogram plot with a smaller figure size (4x2 inches)
    fig, ax = plt.subplots(nrows=1, ncols=1, figsize=(4, 2))

    # Plot histograms for the Q-CTRL solver and random sampling costs
    _, _, _ = ax.hist(
        [costs, random_costs],  # Data for the two histograms
        np.arange(min_cost, max_cost, 2),  # Bins for the histogram
        weights=[
            probabilities,
            random_probabilities,
        ],  # Probabilities for each data set
        label=labels,  # Labels for the legend
        color=colors,  # Colors for each histogram
        histtype="stepfilled",  # Filled step histogram
        align="mid",  # Align bars to the bin center
        alpha=0.8,  # Transparency
    )

    # Set the x and y labels for the plot
    ax.set_xlabel("Cost")
    ax.set_ylabel("Probability")

    # Add the legend to the plot
    ax.legend()

    # Show the plot
    plt.show()
