# Addon-uri Qiskit

[Addon-urile Qiskit](https://github.com/Qiskit?q=qiskit-addon) sunt pachete oficiale de extensie care oferă algoritmi și tehnici de calcul cuantic specializate. Fiecare addon se concentrează pe un domeniu specific și include tutoriale cu cod executabil.

## Addon-uri disponibile

| Addon | Pachet | Descriere |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Reduce lățimea și adâncimea Circuit-ului prin tăierea Gate-urilor și a firelor |
| **SQD** | `qiskit-addon-sqd` | Diagonalizare cuantică bazată pe eșantionare pentru estimarea stării fundamentale |
| **OBP** | `qiskit-addon-obp` | Propagare inversă a operatorilor pentru a reduce adâncimea Circuit-ului |
| **MPF** | `qiskit-addon-mpf` | Formule cu produse multiple pentru simulări Hamiltoniane îmbunătățite |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Compilare cuantică aproximativă cu rețele de tensori |
| **PNA** | `qiskit-addon-pna` | Absorbție propagată a zgomotului pentru atenuarea erorilor |
| **SLC** | `qiskit-addon-slc` | Conuri de lumină umbrite pentru reducerea overhead-ului de eșantionare PEC |

Toate tutorialele pot fi executate direct în browser-ul tău folosind Binder sau pe propriul tău server Jupyter.

Pentru documentația API, vizitează repo-urile individuale ale addon-urilor pe [GitHub](https://github.com/Qiskit?q=qiskit-addon).
