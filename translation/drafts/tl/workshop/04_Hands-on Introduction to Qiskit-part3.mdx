## 2.5 Palawakin sa malaking bilang ng qubit

Sa quantum computing, ang trabaho sa antas ng utility ay mahalaga para sa pag-unlad sa larangan. Ang ganitong trabaho ay nangangailangan ng mga kalkulasyon na gagawin sa mas malaking sukat; nagtatrabaho sa mga circuit na maaaring gumamit ng mahigit 100 qubit at mahigit 1000 gate. Ang halimbawang ito ay gumagawa ng maliit na hakbang sa direksyong iyon sa pamamagitan ng pagpapalawak ng GHZ na problema sa $n=10$ na qubit. Ginagamit nito ang Qiskit patterns workflow at nagtatapos sa pagsukat ng expectation value na $\langle Z_0 Z_i \rangle $.

### Step 1. I-map ang problema

Sumulat ng function na nagbabalik ng `QuantumCircuit` na naghahanda ng $n$-qubit GHZ state (essentially isang extended Bell state), pagkatapos gamitin ang function na iyon para maghanda ng 10-qubit GHZ state at mangolekta ng mga observable na susukat.

```python
def get_qc_for_n_qubit_GHZ_state(n: int) -> QuantumCircuit:

    qc = QuantumCircuit(n) 
    qc.h(0)
    for i in range(n-1):
        qc.cx(i, i+1)
    return qc
n = 10
qc_n_GHZ = get_qc_for_n_qubit_GHZ_state(n)
qc_n_GHZ.draw("mpl")
```

### Step 4. I-post-process ang mga resulta

Para mas maunawaan ang gawi ng mga entangled quantum state sa tunay na hardware, sinusuri natin ang mga pairwise correlation sa pagitan ng mga qubit sa Z basis. Partikular, tinitingnan natin ang mga expectation value na ⟨Z₀Zᵢ⟩, na sumusukat kung gaano kalakas ang korelasyon ng qubit 0 sa bawat isa pang qubit i. Sa partikular, ipo-plot natin ang:
$$
\langle Z_i Z_0 \rangle / \langle Z_1 Z_0 \rangle 
$$
<div class="alert alert-success">

Anong mga halaga ng $\langle Z_i Z_0 \rangle / \langle Z_1 Z_0 \rangle $ ang inaasahan mong makita sa plot?

Mga opsyon:

a) Bumababa habang dinadagdagan ang $i$

b) Pare-pareho sa 1

c) Maliliit na pagbabago sa paligid ng 1

d) Alternating na 1 at 0 para sa kakaibang at pantay na halaga ng $i$

</div>

```python
data = list(range(1, len(operators) + 1))  # Distance between the Z operators
result = job.result()[0]
values = result.data.evs  # Expectation value at each Z operator.
values = [
    v / values[0] for v in values
]  # Normalize the expectation values to evaluate how they decay with distance.

plt.plot(data, values, marker="o", label=f"{n}-qubit GHZ state")
plt.xlabel("Distance between qubits $i$")
plt.ylabel(r"$\langle Z_i Z_0 \rangle / \langle Z_1 Z_0 \rangle $")

plt.show()
```

Tulad ng makikita mo, ang mga resulta ng 10 qubit na eksperimento ay maganda ngunit mayroon pa ring ilang error. Isang paraan para mapabuti ang mga resulta ay ang mas mahusay na pagpapatupad ng GHZ state.

Kadalasan, ipinapatupad ang GHZ state gamit ang isang sunud-sunod na sequence ng CNOT gate na parang hagdan. Gayunpaman, maaari mong ipatupad ang GHZ state nang mas mahusay, na binibabaang ang 2-qubit depth mula sa `n` hanggang `n/2` o mas kaunti pa.

```python
def better_ghz(n):
    "fan out"
    s = int(n / 2)
    qc = QuantumCircuit(n)
    qc.h(s)
    for m in range(s, 0, -1):
        qc.cx(m, m - 1)
        if not (n % 2 == 0 and m == s): 
            qc.cx(n - m - 1, n - m)
    return qc

better_ghz(n).draw("mpl")
```
