## I-install ang Qiskit <a id="install-qiskit"></a>

Puwede kang gumamit ng online jupyter lab environment (tingnan ang gabay na [Online lab environments](quantum.cloud.ibm.com/docs/en/guides/online-lab-environments)) o i-install ang Qiskit nang lokal.

Sundin ang Qiskit installation guide na [I-install ang Qiskit SDK at ang Qiskit Runtime client](https://quantum.cloud.ibm.com/docs/en/guides/install-qiskit) para makumpleto ang mga sumusunod na hakbang:

	- I-install ang Qiskit kasama ang mga karagdagang visualization package: `pip install qiskit[visualization]`

	- I-install ang qiskit-ibm-runtime: `pip install qiskit-ibm-runtime`

	- I-install ang jupyter: `pip install jupyter`
Siguraduhing ang bersyon ng Python na ginagamit mo sa iyong environment ay python>=3.10, para matiyak na compatible ito sa pinakabagong bersyon ng Qiskit:
