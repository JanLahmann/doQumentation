# Mga online na lab environment

Bukod sa pag-download at pag-install ng Qiskit para gamitin sa iyong lokal na makina, may ilang pagpipilian para sa mga gustong magtrabaho sa isang online na development environment.

Maraming provider ang nag-aalok ng online na Jupyter environment na may naka-preinstall na Qiskit para sa mabilis at maginhawang onboarding. Kasama sa mga inirerekomendang provider na ito ang cloud service provider na OVHcloud at ang quantum software development cloud platform na qBraid.

## Mga inirerekomendang notebook environment na may naka-preinstall na Qiskit {#recommended-notebook-environments-with-qiskit-preinstalled}

Ang mga karagdagang online na Jupyter notebook environment tulad ng IBM Watson&reg; Studio, Google Colab, at Microsoft Azure Machine Learning Studio ay maaaring gamitin kasama ang Qiskit sa pamamagitan ng pagsunod sa naaangkop na mga tagubilin.

### OVHcloud AI notebooks

Ang mga gumagamit ng OVHcloud Public Cloud at OVHcloud Startup ay maaaring gamitin ang kanilang OVHcloud quantum notebooks para ma-access ang Qiskit at mag-submit ng mga trabaho sa IBM&reg; QPUs gamit ang kanilang IBM Quantum&reg; o IBM Cloud&reg; account. Para matuto pa, bisitahin ang pahina ng [OVHcloud Documentation and tutorials](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Hindi available ang mga OVHcloud AI notebook sa ilang rehiyon, tulad ng United States. Bukod dito, maaaring magkakaiba ang hitsura ng mga user interface sa iba't ibang rehiyon.
</Admonition>

Sundin ang mga hakbang na ito para makapagsimula:

1. [Gumawa ng IBM Cloud account.](./cloud-setup)
1. [Mag-set up ng OVHcloud account.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Gumawa ng OVH Public Cloud project.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Sundin ang mga hakbang sa [Launch your first AI Notebook,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) at piliin ang **Qiskit** para sa framework.

### qBraid Lab

Ang mga gumagamit ng Qiskit ay maaaring gamitin ang mga preconfigured na Python environment ng qBraid sa pamamagitan ng pagsunod sa mga tagubiling ito. Para sa karagdagang impormasyon, tingnan ang [qBraid Docs.](https://docs.qbraid.com/home/introduction)

1. [Mag-set up ng libreng qBraid account.](https://docs.qbraid.com/home/account)
3. [Gumawa ng Lab environment.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. I-launch ang qBraid Lab, i-click ang icon na **ENVS** sa kanan, i-click ang **+ ADD** sa itaas ng bagong sidebar, at hanapin ang Qiskit. I-click ang bersyon na gusto mong gamitin, pagkatapos ay i-click ang **Install**.
4. Gumawa ng Jupyter notebook sa pamamagitan ng pag-click sa **Python 3 [Qiskit]** sa seksyon ng Notebook.
4. Maaari kang mag-install ng karagdagang mga Python package (tulad ng Qiskit v1.2) sa isang notebook cell sa pamamagitan ng pagpapatakbo ng `%pip install [package_name].` Para sa karagdagang impormasyon, tingnan ang paksa na [Pip (magic) commands](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Mga karagdagang notebook environment para sa mga gumagamit ng Qiskit {#additional-notebook-environments-for-qiskit-users}

Para sa mga platform na walang naka-preconfigure na Qiskit, magkapareho ang mga tagubilin sa pag-setup, maliban na kailangan mong i-install ang Qiskit nang manu-mano. Gagawa ka ng account, magsisimula ng bagong project (o notebook o workspace, depende sa platform), at i-install ang Qiskit.

### IBM Watson Studio

1. [Mag-set up ng libreng trial sa Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Gumawa ng Watson Studio Project.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Para i-install ang mga kinakailangang Qiskit package at dependencies, magbukas ng bagong notebook at patakbuhin ang sumusunod na command:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab

1. [Gumawa ng Google account.](https://support.google.com/accounts/answer/27441?hl)
2. Pumunta sa [Google Colab landing page](https://colab.research.google.com/) at mag-sign in gamit ang iyong Google account.
3. [Gumawa ng Colab notebook](https://colab.research.google.com/#create=true) at i-install ang Qiskit sa pamamagitan ng pagpapatakbo ng sumusunod na command:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio

1. [Gumawa ng Azure account.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Gumawa ng workspace.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Gumawa ng Jupyter notebook sa iyong workspace.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Mula sa notebook, patakbuhin ang command na ito para i-install ang Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
