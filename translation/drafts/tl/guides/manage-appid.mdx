# Pamahalaan ang mga gumagamit ng ID provider

Maaari kang mag-integrate sa iyong panlabas na identity provider (IDP) para ligtas na ma-authenticate ang mga panlabas na gumagamit sa iyong IBM Cloud account. Sa pamamagitan ng panlabas na provider, maibibigay mo sa mga gumagamit sa iyong kumpanya ang paraan para gumamit ng single sign-on (SSO). Maaari mong ikonekta ang iyong cloud account sa isang panlabas na IDP sa isa sa mga sumusunod na paraan:

- Gamitin ang [IBM Cloud&reg; SAML service provider (SP)](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-cloud-saml)
- Gamitin ang [App ID](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-appid) (maaaring magkaroon ng bayad)
- [I-federate ang iyong mga gumagamit sa lahat ng IBM&reg; produkto gamit ang IBMid](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-ibmid)

<Admonition type="tip">
Pagkatapos ikonekta ang iyong IDP, maaari kang mag-set up ng mga dynamic na panuntunan para hindi mo na kailangang manu-manong idagdag ang bawat gumagamit sa bawat grupo kapag sila ay sumali. Tingnan ang [Enabling authentication from an external identity provider](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#app-id-dynamic-rules) para sa mga tagubilin.
</Admonition>
