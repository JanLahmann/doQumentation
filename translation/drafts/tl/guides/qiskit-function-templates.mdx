# Panimula sa mga Qiskit Function template

Ang [mga Qiskit Function template](https://github.com/qiskit-community/qiskit-function-templates) ay isang open-source na koleksyon ng mga makatotohanang halimbawa ng code na nagbibigay sa iyo ng access sa ilan sa mga pinakabagong research workflow na gumagamit ng mga tool tulad ng Qiskit SDK at [mga Qiskit addon](/guides/addons). Ang mga pre-built na workflow na ito ay kumukuha ng impormasyon mula sa mga praktikal na eksperimento at inilalagay ang mga ito sa isang paraan na maaari mong i-customize at gamitin muli sa iyong sariling research nang hindi na kailangang magsimula sa simula. Lahat ng template ay madaling ma-deploy sa [Qiskit Serverless](./serverless) kung saan madali mong magagamit ang classical compute bilang bahagi ng iyong workflow.

Mayroong dalawang uri ng template:

- *Mga template implementation*: Mga espesyalisadong halimbawa ng code na nagpapatupad ng mga advanced na teknik mula sa mga totoong eksperimento sa mga larangan tulad ng chemistry at physics.
- *Mga base template*: Mga sanggunian para sa pagbuo ng iyong sariling orihinal na Qiskit Function. Itinatampok ng mga template na ito ang mga pinakamahusay na gawi sa interface development, pag-format ng code, unit testing, at iba pa.

## Mga template implementation {#template-implementations}

Ang mga Qiskit Function template implementation ay inayos ayon sa application area. Kasalukuyang kasama sa koleksyon ang isang physics template para sa Hamiltonian simulation gamit ang [AQC-Tensor Qiskit addon](./qiskit-addons-aqc) at isang chemistry template para sa electronic structure na may implicit solvent model gamit ang [SQD Qiskit addon](./qiskit-addons-sqd).
Ang mga mapagkukunan para makapagsimula sa dalawang template na ito ay makikita sa mga sumusunod na link:

- Electronic structure simulation na may implicit solvent model: [mga source file ng template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/chemistry/sqd_pcm) at [gabay](./function-template-chemistry-workflow)
- Hamiltonian simulation: [mga source file ng template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/physics/hamiltonian_simulation) at [gabay](./function-template-hamiltonian-simulation)

## Mga base template {#base-templates}

Ang mga *base template* ay mga sanggunian para sa pagbuo ng iyong sariling orihinal na Qiskit Function. Idinisenyo ang mga ito para tulungan kang simulan ang proseso ng development sa pamamagitan ng pagpapakita ng mga pinakamahusay na gawi sa interface development, pag-format ng code, paghawak ng I/O, mga mensahe ng error, mga checkpoint, unit testing, at iba pa. Ang repository ay inilalantad bilang isang [GitHub template](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-template-repository) para madaling mapalawak ang CI/CD implementation.

Kasalukuyang mayroong dalawang template: isang [circuit function template](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/circuit_function_template.py) at isang [application function template](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/application_function_template.py). Ang circuit function template ay katulad ng pagkakagawa ng [IBM Circuit Function](./ibm-circuit-function) at ang application function template ay nagpapakita kung paano istruktura ang isang karaniwang application workflow.
