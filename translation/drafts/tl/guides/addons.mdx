# Mga Qiskit addon

Ang mga Qiskit addon ay koleksyon ng mga research capabilities para sa pagtuklas ng mga algorithm sa utility scale. Ang mga kakayahang ito ay nagtitiwala sa matibay na pundasyon ng Qiskit — mga tools para sa paglikha at pagpapatakbo ng mga quantum algorithm. Ibinibigay ang mga ito bilang mga modular na software component na maaaring isaksak sa isang [workflow](/guides/intro-to-patterns) para mapalawak o magdisenyo ng mga bagong quantum algorithm.

## Mga addon para sa mapping {#addons-for-mapping}

### Approximate quantum compilation gamit ang tensor networks {#approximate-quantum-compilation-with-tensor-networks}

Ang Approximate quantum compilation with tensor networks (AQC-Tensor) ay nagbibigay-daan sa pagbuo ng mga high-fidelity na circuit na may mas mababang depth.

- Bisitahin ang [GitHub repository.](https://github.com/Qiskit/qiskit-addon-aqc-tensor)
- Basahin ang [dokumentasyon.](/guides/qiskit-addons-aqc)
- Basahin ang tutorial sa paggamit ng AQC para [mapabuti ang Trotterized time evolution.](/tutorials/approximate-quantum-compilation-for-time-evolution)

### Multi-product formulas {#multi-product-formulas}

Ang Multi-product formulas (MPF) ay nagbabawas ng Trotter error ng Hamiltonian dynamics sa pamamagitan ng weighted na kumbinasyon ng ilang circuit execution.

- Bisitahin ang [GitHub repository.](https://github.com/Qiskit/qiskit-addon-mpf)
- Basahin ang [dokumentasyon.](https://qiskit.github.io/qiskit-addon-mpf)

### Optimization mapper {#optimization-mapper}

Ang Optimization Mapper addon ay naglalaman ng functionality para i-model ang mga optimization problem sa pamamagitan ng pagbabalangkas ng mga ito sa abstract na modelo at pagkatapos ay pag-convert sa mga representasyong naiintindihan ng quantum computer.

- Basahin ang [dokumentasyon](https://qiskit.github.io/qiskit-addon-opt-mapper/) para sa mga how-to guide at API reference.
- Bisitahin ang [GitHub](https://github.com/qiskit/qiskit-addon-opt-mapper) repository.

## Mga addon para sa pag-optimize {#addons-for-optimizing}

### Operator backpropagation {#operator-backpropagation}

Ang Operator backpropagation (OBP) ay nagbabawas ng circuit depth sa pamamagitan ng pag-trim ng mga operasyon mula sa dulo, kapalit ng mas maraming operator measurement.

- Bisitahin ang [GitHub repository.](https://github.com/Qiskit/qiskit-addon-obp)
- Basahin ang [dokumentasyon.](/guides/qiskit-addons-obp)
- Basahin ang tutorial sa paggamit ng [OBP para mapabuti ang mga expectation value.](/tutorials/operator-back-propagation)

### Circuit cutting {#circuit-cutting}

Ang Circuit cutting ay nagbabawas ng depth ng mga na-transpile na circuit sa pamamagitan ng pag-decompose ng mga entangling gate sa pagitan ng mga hindi magkaratig na qubit.

- Bisitahin ang [GitHub repository.](https://github.com/Qiskit/qiskit-addon-cutting)
- Basahin ang [dokumentasyon.](https://qiskit.github.io/qiskit-addon-cutting)

## Mga addon para sa post-processing {#addons-for-post-processing}

### Sample-based quantum diagonalization {#sample-based-quantum-diagonalization}

Ang Sample-based quantum diagonalization (SQD) ay klasikal na nagpo-post-process ng mga maingay na quantum sample para makakuha ng mas tumpak na eigenvalue estimation ng mga quantum system Hamiltonian — halimbawa sa mga chemistry application.

- Bisitahin ang [GitHub repository.](https://github.com/Qiskit/qiskit-addon-sqd)
- Basahin ang [dokumentasyon.](/guides/qiskit-addons-sqd)
- Basahin ang tutorial sa [Sample-based quantum diagonalization ng isang chemistry Hamiltonian.](/tutorials/sample-based-quantum-diagonalization)

### SQD para sa HPC {#sqd-for-hpc}

Ang addon na ito ay isang HPC-ready na implementasyon ng SQD addon. Nakasulat ito sa modernong C++17 standards at dinisenyo para makagawa ng iisang compiled binary para gamitin sa MPI.

- Bisitahin ang [API reference](https://qiskit.github.io/qiskit-addon-sqd-hpc/) na dokumentasyon.
- Tingnan ang [GitHub](https://github.com/qiskit/qiskit-addon-sqd-hpc) repository.

## Mga addon para sa error mitigation {#addons-for-error-mitigation}

### M3 (mthree) {#m3-mthree}

Ang Matrix-free Measurement Mitigation (M3) ay isang package para sa scalable na quantum measurement error mitigation na maaaring i-compute nang sabay-sabay.

- Bisitahin ang [dokumentasyon](https://qiskit.github.io/qiskit-addon-mthree/) para sa mga how-to guide, tutorial, at API reference.
- Bisitahin ang [GitHub](https://github.com/Qiskit/qiskit-addon-mthree) page ng package.

### Shaded lightcones {#shaded-lightcones}

Ang shaded lightcones addon ay gumagamit ng Pauli propagation para mabawasan ang bilang ng mga error term sa noise model na kailangang i-mitigate, batay sa mga detalye ng target observable. Ang epekto nito ay pagbabawas ng sampling overhead para sa mga probabilistic error cancellation (PEC) workflow.

- Bisitahin ang [dokumentasyon](https://qiskit.github.io/qiskit-addon-slc/) para sa mga how-to guide, tutorial, at API reference.

### Propagated noise absorption {#propagated-noise-absorption}

Ang Propagated noise absorption (PNA) ay gumagamit ng Pauli propagation para ilipat ang impormasyon mula sa noise model patungo sa target observable. Ang pagsukat sa binagong observable na ito ay nagmimitigate ng noise ayon sa representasyon ng modelo.

- Bisitahin ang [dokumentasyon](https://qiskit.github.io/qiskit-addon-pna/apidocs/qiskit_addon_pna.html) para sa mga how-to guide, tutorial, at API reference.
