### I-box ang circuit
Para sa kaginhawahan sa implementasyon, gagamitin natin ang transpilation pass na `generate_boxing_pass_manager`, na naglalagay ng mga instruction ng circuit sa mga annotated box. Malinaw na ipinapakita ng mga box na ito kung saan, sa kaso ng PEC, dapat i-inject ang antinoise sa circuit. Para sa mga detalye ng mga setting, tingnan ang [dokumentasyon ng Samplomatic.](https://qiskit.github.io/samplomatic/)

Tandaan na ang SLC workflow ay gumagamit ng `inject_noise_strategy="individual_modification"` sa ibang bahagi ng proseso dahil nagbibigay ito sa atin ng kakayahang natatanging makilala ang bawat `BoxOp` sa circuit.

Ang function na `find_unique_box_instructions` ay umuulit sa ibinigay na boxed circuit at tinutukoy ang mga may natatanging 2Q layer o mga pagsukat, para sa layunin ng noise learning at noise injection.

```python
# Box circuit with Twirl and InjectNoise annotations
boxes_pm = generate_boxing_pass_manager(
    twirling_strategy="active",
    inject_noise_strategy="individual_modification",
    inject_noise_targets="gates",
    measure_annotations="all",
)

boxed_circuit = boxes_pm.run(isa_circuit)

# Find the unique instructions (layers) from boxed circuit
unique_2q_instructions = find_unique_box_instructions(
    boxed_circuit, normalize_annotations=None, undress_boxes=True
)
```

### Ihanda ang mga canonical bases measurement
Dahil sa paraan ng paglalagay ng label sa mga qubit kapag tinutukoy ang mga natatanging 2Q layer, kailangan nating mag-ingat sa pagsubaybay ng pagkakasunud-sunod ng qubit. Sa ibaba, ipinakilala natin ang konsepto ng `canonical_qubits` bilang paraan para maayos na i-update ang pagkakasunud-sunod ng qubit kapag ibinibigay ito sa executor, bilang resulta ng paraan ng pagkuha ng pagkakasunud-sunod ng qubit kapag nibo-box ang mga circuit at nahahanap ang mga natatanging instruction. Tingnan ang dokumentasyon ng [Qubit ordering convention](https://qiskit.github.io/samplomatic/guides/samplex_io.html#qubit-ordering-convention) para sa mga detalye.

```python
# Determine the canonical qubits order
meas_box = boxed_circuit.data[-1]
canonical_qubits = [
    idx for idx, qubit in enumerate(boxed_circuit.qubits) if qubit in meas_box.qubits
]

# map canonical qubit to physical (isa) qubit
c_2_p = {c: p for c, p in enumerate(canonical_qubits)}
# map physical (isa) qubit to virtual qubit (index in original circuit)
p_2_v = {p: v for v, p in enumerate(layout)}
# compute map between virtual and canonical qubit indices.
c_2_v = {c: p_2_v[p] for c, p in c_2_p.items()}

assert len(c_2_v) == num_qubits

bases_canon = [
    np.array([base_i[c_2_v[c]] for c in range(num_qubits)], dtype=np.uint8) for base_i in bases_virt
]
```
