# Qiskit Addons

Ang [Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) ay mga opisyal na extension package na nagbibigay ng mga espesyalisadong quantum computing algorithm at teknik. Ang bawat addon ay nakatuon sa isang partikular na larangan at may kasamang mga tutorial na may executable na code.

## Mga Available na Addon

| Addon | Package | Paglalarawan |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Bawasan ang lapad at lalim ng circuit sa pamamagitan ng pag-cut ng mga gate at wire |
| **SQD** | `qiskit-addon-sqd` | Sample-based Quantum Diagonalization para sa ground state estimation |
| **OBP** | `qiskit-addon-obp` | Operator Back-Propagation para mabawasan ang lalim ng circuit |
| **MPF** | `qiskit-addon-mpf` | Multi-Product Formulas para sa pinahusay na Hamiltonian simulation |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximate Quantum Compilation gamit ang tensor network |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption para sa error mitigation |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones para mabawasan ang PEC sampling overhead |

Lahat ng tutorial ay maaaring i-execute nang direkta sa browser gamit ang Binder, o sa sarili mong Jupyter server.

Para sa API documentation, bisitahin ang mga indibidwal na addon repo sa [GitHub](https://github.com/Qiskit?q=qiskit-addon).
