Bago magsimula, mangyaring sagutan ang maikling [pre-course survey](https://your.feedback.ibm.com/jfe/form/SV_cZ20ssflsjutfcG) na ito, na mahalaga para makatulong sa pagpapabuti ng aming mga nilalaman at karanasan ng mga gumagamit.

Sa unang leksyon ng kursong ito, mag-aaral tayo ng isang simpleng algorithmic framework — kilala bilang ang *query model* — at titingnan natin ang mga kalamangan na inaalok ng mga quantum computer sa loob ng framework na ito.

Ang query model ng computation ay parang isang petri dish para sa mga quantum algorithmic na ideya.
Ito ay mahigpit at hindi natural sa kahulugang hindi nito tumpak na kinakatawan ang mga uri ng computational na problema na karaniwang pinapahalagahan natin sa praktis, ngunit napatunayan pa rin itong napaka-kapaki-pakinabang bilang isang kasangkapan para sa pagbuo ng mga quantum algorithmic na teknik.
Kabilang dito ang mga nagpapalakas sa pinakasikat na mga quantum algorithm, tulad ng algorithm ni Shor para sa integer factorization.
Ang query model ay nagkataong isang napaka-kapaki-pakinabang na framework din para sa *pagpapaliwanag* ng mga quantum algorithmic na teknik.

Pagkatapos ipakilala ang query model mismo, tatalakayin natin ang pinakaunang quantum algorithm na natuklasan, na siyang *Deutsch's algorithm,* kasama ang isang extension ng Deutsch's algorithm na kilala bilang ang *Deutsch-Jozsa algorithm*.
Ipinapakita ng mga algorithm na ito ang mga nasusukat na kalamangan ng quantum kumpara sa mga klasikal na computer sa konteksto ng query model.
Tatalakayin din natin ang isang quantum algorithm na kilala bilang *Simon's algorithm,* na nag-aalok ng mas matibay at mas kasiya-siyang kalamangan ng quantum kumpara sa mga klasikal na computation, sa mga dahilang ipapaliwanag natin kapag nakarating na tayo doon.
