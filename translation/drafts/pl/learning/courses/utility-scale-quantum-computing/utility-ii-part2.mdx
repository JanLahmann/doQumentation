## 2. Definiowanie hamiltonianu Isinga w poprzecznym polu {#defining-the-transverse-field-ising-hamiltonian}

Rozważamy tu jednowymiarowy model Isinga w poprzecznym polu.

Najpierw utworzymy funkcję, która przyjmuje parametry układu $N$, $J$ oraz $h$ i zwraca nasz Hamiltonian jako `SparsePauliOp`. [SparsePauliOp](https://docs.quantum.ibm.com/api/qiskit/qiskit.quantum_info.SparsePauliOp) to rzadka reprezentacja operatora w postaci ważonych wyrazów [Pauli](https://docs.quantum.ibm.com/api/qiskit/qiskit.quantum_info.Pauli).
### 2.1 Ćwiczenie 1 {#activity-1}
Skonstruuj funkcję budującą Hamiltonian Isinga w poprzecznym polu (zobacz równanie powyżej), przyjmującą argumenty: „liczba kubitów”, „parametr J” oraz „parametr h”. Spróbuj wykonać to samodzielnie, korzystając z wcześniejszych przykładów. Przewiń w dół, aby zobaczyć rozwiązanie.

__Rozwiązanie:__

```python
def get_hamiltonian(nqubits, J, h):
    # List of Hamiltonian terms as 3-tuples containing
    # (1) the Pauli string,
    # (2) the qubit indices corresponding to the Pauli string,
    # (3) the coefficient.
    ZZ_tuples = [("ZZ", [i, i + 1], -J) for i in range(0, nqubits - 1)]
    X_tuples = [("X", [i], -h) for i in range(0, nqubits)]

    # We create the Hamiltonian as a SparsePauliOp, via the method
    # `from_sparse_list`, and multiply by the interaction term.
    hamiltonian = SparsePauliOp.from_sparse_list(
        [*ZZ_tuples, *X_tuples], num_qubits=nqubits
    )
    return hamiltonian.simplify()
```

$$
e^{H_1+H_2} \approx e^{H_1} e^{H_2}
$$

#### Wykonanie symulacji ewolucji czasowej {#perform-time-evolution-simulation}

```python
# Step 1. Map the problem
# Initiate the circuit
evolved_state = QuantumCircuit(initial_circuit.num_qubits)
# Start from the initial spin configuration

# Define backend (simulator)
# MPS
backend_mps = AerSimulator(method="matrix_product_state")
