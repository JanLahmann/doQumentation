Potrzebujemy również sposobu na wprowadzanie błędów.

Tutaj $\chi$ reprezentuje błędy, które chcemy wprowadzić — używamy dyskretnego rozkładu Gaussa na $\mathbb{Z}_{q}$ charakteryzującego się średnią 0 i odchyleniem standardowym $\sigma$.

Ta definicja błędów do wprowadzenia jest również publiczna.

Zatem w Pythonie potrzebujemy prostej funkcji, która przybliża losowe wkłady błędów/szumu $\varepsilon$ pobierane z dyskretnego rozkładu Gaussa $\chi$ o odchyleniu standardowym $\sigma$.

Wypisane są przykładowe wartości, abyś mógł zobaczyć rozkład wokół 0.

```python
def chi(stdev, modulus):
    return round((np.random.randn() * stdev**2)) % modulus

# wypisz kilka przykładów
sd = 2
m = 1000
for x in range(10):
    print("chi = ", chi(sd, m))
```

Następnie Alice musi wygenerować swoją parę kluczy.

Ustawia swój **private key**, wybierając $n$ wartości między 0 a $q$.

```python
# private key Alice
alice_private_key = np.random.randint(0, high=q, size=n)
print(f"Alice's private key: {alice_private_key}")
```

Alice teraz tworzy swój **public key**, wybierając losowe wektory, które następnie są łączone z wygenerowanymi błędami.

<details>
<summary>Matematyka</summary>

Dokładniej, musi wygenerować próbkę $A_{s,\chi}$.

W związku z tym losowo wybiera wektory $\mathbf{a} \in \mathbb{Z}_{q}^{n}$ oraz błędy $\varepsilon$ z $\chi$, aby skonstruować $N$ próbek $(\mathbf{a}, b)$.

</details>

```python
# Public Key Alice
alice_public_key = []

# N to liczba wartości, jakie chcemy mieć w kluczu
for i in range(N):
    # Pobierz n losowych wartości między 0 a <q
    a = np.random.randint(0, high=q, size=n)
    # pobierz błąd do wprowadzenia
    epsilon = chi(sigma, q)
    # oblicz iloczyn skalarny (czyli jak mnożenie tablic)
    b = (np.dot(a, alice_private_key) + epsilon) % q
    # wartość do dodania do klucza -
    sample = (a, b)
    alice_public_key.append(sample)

print(f"Alice's public key: {alice_public_key}")
```

Alice może teraz udostępnić swój public key.

Bob może teraz użyć go, aby wysłać Alice zaszyfrowaną wiadomość.

Wiadomość Boba to pojedynczy bit.

```python
# Szyfrowanie
bob_message_bit = 1
print(f"Bob's message bit={bob_message_bit}")
```

Aby zaszyfrować wiadomość, Bob musi wybrać losowo dowolną liczbę próbek z public key Alice, aby utworzyć szyfrogram.

W tym celu tworzy maskę, losowy wektor binarny $r$ o długości $N$.

```python
# lista N wartości między 0 a <2 - czyli 0 lub 1
r = np.random.randint(0, 2, N)
print(r)
```

Teraz bierzemy tę maskę i stosujemy ją do odpowiedniego wpisu w public key Alice, obliczając sumę znalezionych wartości.

```python
sum_ai = np.zeros(n, dtype=int)
sum_bi = 0

for i in range(N):
    sum_ai = sum_ai + r[i] * alice_public_key[i][0]
    sum_bi = sum_bi + r[i] * alice_public_key[i][1]
sum_ai = [x % q for x in sum_ai]
# sum_bi = sum_bi
ciphertext = (sum_ai, (bob_message_bit * int(np.floor(q / 2)) + sum_bi) % q)
print(f"ciphertext is: {ciphertext}")
```

Ostatecznie Bob nadaje szyfrogram, który Alice może odszyfrować przy użyciu swojego private key.

```python
# Odszyfrowanie
adots = np.dot(ciphertext[0], alice_private_key) % q
b_adots = (ciphertext[1] - adots) % q

decrypted_message_bit = round((2 * b_adots) / q) % 2

print(
    f"original message bit={bob_message_bit}, decrypted message bit={decrypted_message_bit}"
)

assert bob_message_bit == decrypted_message_bit
```

Ponieważ ten protokół działa bit po bicie przy szyfrowaniu dłuższych ciągów bitów, po prostu powtarzamy operacje w pętli. Poniżej przedstawiono scenariusz, w którym Bob chce przesłać 16 zaszyfrowanych bitów.

```python
bob_message_bits = np.random.randint(0, 2, 16)
print(f"Bob's message bits are : {bob_message_bits}")
decrypted_bits = []

for ib in range(len(bob_message_bits)):
    bob_message_bit = bob_message_bits[ib]

    r = np.random.randint(0, 2, N)

    sum_ai = np.zeros(n, dtype=int)
    sum_bi = 0
    for i in range(N):
        sum_ai = sum_ai + r[i] * alice_public_key[i][0]
        sum_bi = sum_bi + r[i] * alice_public_key[i][1]
    sum_ai = [x % q for x in sum_ai]

    ciphertext = (sum_ai, (bob_message_bit * int(np.floor(q / 2)) + sum_bi) % q)

    adots = np.dot(ciphertext[0], alice_private_key) % q
    b_adots = (ciphertext[1] - adots) % q

    decrypted_message_bit = round((2 * b_adots) / q) % 2
    assert decrypted_message_bit == bob_message_bit

    decrypted_bits.append(decrypted_message_bit)

print(f"Decrypted message bits = {np.array(decrypted_bits)}")
```
