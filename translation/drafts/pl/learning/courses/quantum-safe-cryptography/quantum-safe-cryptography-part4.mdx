
Potrzebujemy również sposobu wprowadzania błędów

Tutaj $\chi$ reprezentuje błędy, które chcemy wprowadzić — używamy dyskretnego rozkładu Gaussa na $\mathbb{Z}_{q}$ scharakteryzowanego średnią 0 i odchyleniem standardowym $\sigma$.

Ta definicja błędów do wprowadzenia jest również publiczna.

Tak więc w Pythonie potrzebujemy prostej funkcji, która przybliża losowe wkłady błędu/szumu $\varepsilon$ pobrane z dyskretnego rozkładu Gaussa $\chi$ o odchyleniu standardowym $\sigma$.

Wypisanych jest kilka przykładowych wartości, abyś mógł zobaczyć rozkład wokół 0.

```python
def chi(stdev, modulus):
    return round((np.random.randn() * stdev**2)) % modulus

# print some examples
sd = 2
m = 1000
for x in range(10):
    print("chi = ", chi(sd, m))
```

Następnie Alice musi wygenerować swoją parę kluczy.

Ustawia swój **klucz prywatny**, wybierając $n$ wartości pomiędzy 0 a $q$.

```python
# Alice's private key
alice_private_key = np.random.randint(0, high=q, size=n)
print(f"Alice's private key: {alice_private_key}")
```

Alice ustawia teraz swój **klucz publiczny**, wybierając losowe wektory, które są następnie łączone z wygenerowanymi błędami.

<details>
<summary>Matematyka</summary>

Mówiąc precyzyjniej, musi ona wygenerować próbkę $A_{s,\chi}$.

Odpowiednio, losowo wybiera wektory $\mathbf{a} \in \mathbb{Z}_{q}^{n}$ oraz błędy $\varepsilon$ z $\chi$, aby skonstruować $N$ próbek $(\mathbf{a}, b)$.

</details>

```python
# Alice's Public Key
alice_public_key = []

# N is the number of values we want in the key
for i in range(N):
    # Get n random values between 0 and <q
    a = np.random.randint(0, high=q, size=n)
    # get an error to introduce
    epsilon = chi(sigma, q)
    #  calculate dot product (ie like array multiplication)
    b = (np.dot(a, alice_private_key) + epsilon) % q
    # value to be added to the key -
    sample = (a, b)
    alice_public_key.append(sample)

print(f"Alice's public key: {alice_public_key}")
```

W tym celu tworzy maskę, losowy wektor binarny $r$ o długości $N$.

