## Testowanie relacji nieoznaczoności {#testing-uncertainty-relations}

Powyższy test pokazał jedynie słuszność relacji nieoznaczoności dla jednego konkretnego wyboru wektora stanu $|\psi\rangle = |+\rangle_x$. Aby przekonać się, że jest to ogólnie zgodne z eksperymentem, powinniśmy przeprowadzić podobne obliczenia przy użyciu Estimator dla wielu wyborów wektora stanu. Zacznijmy od obrotu naszego wektora stanu z dala od osi $z$, używając Gate `RY` do generowania różnych stanów początkowych za pomocą parametru $\theta$.

```python
# Poniższe obliczenia zajmują około 3-4 minut czasu QPU.
# Krok 1: Odwzorowanie problemu na Circuit kwantowy

from qiskit.circuit import Parameter
import numpy as np

# Określenie obserwabli
obs1 = SparsePauliOp("X")
obs2 = SparsePauliOp("Y")
obs3 = SparsePauliOp("Z")

# Definiowanie rejestrów
qr = QuantumRegister(1, "q")
cr = ClassicalRegister(1, "c")
qc = QuantumCircuit(qr, cr)

# Obrót z dala od |0>
theta = Parameter("θ")
qc.ry(theta, 0)

params = np.linspace(0, 2, num=21)

# Krok 2: Transpilacja Circuit

pm = generate_preset_pass_manager(target=target, optimization_level=3)
qc_isa = pm.run(qc)
obs1_isa = obs1.apply_layout(layout=qc_isa.layout)
obs2_isa = obs2.apply_layout(layout=qc_isa.layout)
obs3_isa = obs3.apply_layout(layout=qc_isa.layout)

# Krok 3: Uruchomienie Circuit na prawdziwym komputerze kwantowym

with Batch(backend=backend) as batch:
    estimator = Estimator(mode=batch)
    pubs = [(qc_isa, [[obs1_isa], [obs2_isa], [obs3_isa]], [params])]
    job = estimator.run(pubs, precision=0.01)
    res = job.result()
```

```python
xs = res[0].data.evs[0]

# Obliczanie niepewności

for i in range(len(xs)):

    delz.append(abs((1 - zs[i] * zs[i])) ** 0.5)
```

Zauważ, że czerwona krzywa $(\Delta X \Delta Z)$ jest zawsze większa niż pomarańczowa krzywa $\langle Y \rangle$. Czasami iloczyn niepewności maleje i jest nieco bliski granicy, a innym razem rośnie i oddala się od niej — jednak zawsze spełnia relację nieoznaczoności.

Oczywiście, może to nie być najlepszy test relacji nieoznaczoności, ponieważ nasza granica $\langle Y \rangle$ jest zawsze bardzo bliska zeru. Użyjmy stanu kwantowego, który ma większą rzutownię na własne stany $Y$. Konkretnie: nadal będziemy obracać $|0\rangle$ w dół z osi $z$ o różne kąty, ale teraz dodatkowo obrócimy otrzymany stan wokół $z$ o pewien kąt, być może $\pi/4$, i zobaczymy, co się stanie.
