### Krok 1: Mapowanie klasycznych danych wejściowych na problem kwantowy {#step-1-map-classical-inputs-to-a-quantum-problem}

Tutaj klasycznymi danymi wejściowymi są orientacje spinu przed pomiarem w urządzeniu Sterna-Gerlacha. Nie martw się zbytnio o dokładną naturę stanu kwantowego przed pomiarem. To jest temat innego modułu Qiskit Classrooms, poświęconego twierdzeniu Bella.

Zauważ, że komputery kwantowe IBM mierzą stany wzdłuż osi $z$. Dlatego ten pierwszy eksperyment będzie bardzo podobny do eksperymentu Sterna-Gerlacha z gradientem pola magnetycznego wzdłuż $z$. Zobaczymy później, jak zmienić układ, żeby mierzyć wzdłuż innych kierunków.

Zacznijmy od stworzenia analogu stanu spinowego, czyli pewnej mieszaniny $|\uparrow\rangle$ i $|\downarrow\rangle$, albo równoważnie $|0\rangle$ i $|1\rangle$. Zaproponowaliśmy pewne wartości początkowe, ale możesz swobodnie bawić się innymi wartościami lub nawet losowymi.

```python
import random
from numpy import pi
import numpy as np

# Use these lines to choose your own arbitrary state vector and normalize it.
# a = 2
# b = (1+1j)
# norm = np.sqrt(a*np.conjugate(a)+b*np.conjugate(b))
# a = a/norm
# b = b/norm
# print(a,b)

# Use these lines if you would rather look at at random spin orientations.
a = random.random()
b = random.random()
norm = np.sqrt(a * np.conjugate(a) + b * np.conjugate(b))
a = a / norm
b = b / norm
print(a, b)
```


Teraz użyjemy powyższych kątów jako parametrów w Circuit kwantowym. Rozważamy tylko jedną cząstkę naraz, więc użyjemy tylko jednego Qubit w naszym Circuit i będziemy potrzebować tylko jednego klasycznego rejestru.

```python
from qiskit.circuit import QuantumRegister, ClassicalRegister, QuantumCircuit, Parameter

# Define registers
qr = QuantumRegister(1, "q")
cr = ClassicalRegister(1, "c")
qc = QuantumCircuit(qr, cr)

# Initialize the quantum state
qc.initialize([a, b])
qc.measure(0, 0)
qc.draw("mpl")
```

Poniżej znajduje się kod do zapisania swoich danych uwierzytelniających przy pierwszym użyciu. Teraz musimy przeprowadzić transpilację Circuit, co oznacza, że musimy odwzorować nasz Circuit na podstawowe Gate dostępne w wybranym komputerze kwantowym, i chcemy zoptymalizować nasz Circuit pod kątem działania na tym komputerze kwantowym.

```python
# Transpile the circuit and optimize for running on the quantum computer selected
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

pm = generate_preset_pass_manager(optimization_level=3, backend=backend)
qc_ibm = pm.run(qc)
```

### Krok 3: Wykonanie przy użyciu Qiskit Runtime Primitives {#step-3-execute-using-qiskit-runtime-primitives}

Teraz chcemy uruchomić to na prawdziwym komputerze kwantowym. Cała niezbędna składnia do tego znajduje się w bloku kodu poniżej. Jeśli wyczerpałeś przydzielony czas na prawdziwych komputerach kwantowych lub nie masz połączenia z internetem, możesz odkomentować następny blok kodu, który uruchomi kod na lokalnym symulatorze.

```python
# Specify that we want to use only a single shot, to represent a single measurement of a spin in a SG device.
num_shots = 1

# Evaluate the problem using a QPU via Qiskit IBM Runtime
# The best practice is to use a session as shown below. This is available to Premium Plan, Flex Plan, and On-Prem (IBM Quantum Platform API) Plan users.
with Session(backend=backend) as session:
    sampler = Sampler(mode=session)
    dist = sampler.run([qc_ibm], shots=num_shots).result()
session.close()
counts = dist[0].data.c.get_counts()
```

Wykonaliśmy jeden pomiar i uzyskaliśmy „0". To nie jest dla nas żadnym zaskoczeniem. Wiemy, że używamy komputera kwantowego zbudowanego z Qubit i jesteśmy bardzo przyzwyczajeni do tego, że klasyczne bity zwracają 0 lub 1. Pamiętaj jednak: to jest kwantowo-obliczeniowy odpowiednik eksperymentów przeprowadzanych z cząstkami posiadającymi momenty magnetyczne. Gdybyśmy oczekiwali równomiernego rozkładu między 0 a +1, mogłoby nas rzeczywiście zaskoczyć, że przy pierwszym pomiarze uzyskaliśmy jedno ekstremum. To właśnie zaskakujące skwantowanie wyników w eksperymencie Sterna-Gerlacha doprowadziło nas do lepszego zrozumienia natury, co z kolei pomogło nam ostatecznie zbudować komputery kwantowe.
