## Algorytm Deutscha-Jozsy

Algorytm Deutscha był ważnym pierwszym krokiem w demonstrowaniu, jak komputer kwantowy może być wydajniejszy od klasycznego, ale stanowił jedynie skromną poprawę: wymagał tylko jednego zapytania, podczas gdy w podejściu klasycznym potrzebne były dwa. W 1992 roku Deutsch i jego współpracownik, Richard Jozsa, rozszerzyli pierwotny algorytm dwuqubitowy na większą liczbę qubitów. Problem pozostał ten sam: ustalić, czy funkcja jest *wyważona* czy *stała*. Tym razem jednak funkcja odwzorowuje $n$ bitów na jeden bit. Albo funkcja zwraca 0 i 1 w równej liczbie przypadków (jest *wyważona*), albo zawsze zwraca 1 lub zawsze 0 (jest *stała*).

Oto schemat obwodu algorytmu:

![DJ_algo.png](/learning/images/modules/computer-science/deutsch-jozsa/DJ_algo.avif)

Algorytm działa tak samo jak algorytm Deutscha: mechanizm phase-kickback pozwala odczytać stan qubitu 0 i ustalić, czy funkcja jest stała czy wyważona. Jest to nieco trudniejsze do zobaczenia niż w przypadku dwuqubitowego algorytmu Deutscha, ponieważ stany będą zawierać sumy po $n$ qubitach — dlatego wyznaczenie tych stanów pozostawiamy jako opcjonalne ćwiczenie na końcu modułu. Algorytm zwróci ciąg bitów złożony z samych zer, jeśli funkcja jest stała, lub ciąg zawierający co najmniej jedną jedynkę, jeśli funkcja jest wyważona.

Aby zobaczyć, jak algorytm działa w Qiskit, musimy najpierw wygenerować naszą wyrocznię: losową funkcję, która z gwarancją jest albo stała, albo wyważona. Poniższy kod będzie generował funkcję wyważoną w 50% przypadków i funkcję stałą w 50% przypadków. Nie przejmuj się, jeśli nie rozumiesz całego kodu — jest skomplikowany i nie jest konieczny do zrozumienia algorytmu kwantowego.

```python
from qiskit import QuantumCircuit
import numpy as np

def dj_function(num_qubits):
    """
    Create a random Deutsch-Jozsa function.
    """

    qc_dj = QuantumCircuit(num_qubits + 1)
    if np.random.randint(0, 2):
        # Flip output qubits with 50% chance
        qc_dj.x(num_qubits)
    if np.random.randint(0, 2):
        # return constant circuit with 50% chance.
        return qc_dj

    # If the "if" statement above was "TRUE" then we've returned the constant
    # function and the function is complete. If not, we proceed in creating our
    # balanced function. Everything below is to produce the balanced function:

    # select half of all possible states at random:
    on_states = np.random.choice(
        range(2**num_qubits),  # numbers to sample from
        2**num_qubits // 2,  # number of samples
        replace=False,  # makes sure states are only sampled once
    )

    def add_cx(qc_dj, bit_string):
        for qubit, bit in enumerate(reversed(bit_string)):
            if bit == "1":
                qc_dj.x(qubit)
        return qc_dj

    for state in on_states:
        # qc_dj.barrier()  # Barriers are added to help visualize how the functions are created. They can safely be removed.
        qc_dj = add_cx(qc_dj, f"{state:0b}")
        qc_dj.mcx(list(range(num_qubits)), num_qubits)
        qc_dj = add_cx(qc_dj, f"{state:0b}")

    # qc_dj.barrier()

    return qc_dj

n = 3  # number of input qubits

oracle = dj_function(n)

display(oracle.draw("mpl"))
```

Jeśli wyjście dla ostatniego qubitu zależy od pierwszych $n$ qubitów, czy możesz stwierdzić, czy to zależne wyjście jest wyważone?

Możemy określić, czy funkcja jest wyważona czy stała, patrząc na powyższy obwód, ale pamiętaj — na potrzeby tego problemu traktujemy tę funkcję jako „czarną skrzynkę". Nie możemy zaglądać do środka, żeby obejrzeć schemat obwodu. Zamiast tego musimy odpytywać skrzynkę.

### Problem Bernsteina-Vazirania

W 1997 roku Ethan Bernstein i Umesh Vazirani użyli algorytmu Deutscha-Jozsy do rozwiązania bardziej szczegółowego, ograniczonego problemu w porównaniu z problemem Deutscha-Jozsy. Zamiast jedynie rozróżniać między dwiema różnymi klasami funkcji, jak w przypadku D-J, Bernstein i Vazirani użyli algorytmu Deutscha-Jozsy do faktycznego odczytania ciągu zakodowanego w funkcji. Oto problem:

Funkcja $f:\{0,1\}^n \rightarrow \{0,1\}$ nadal przyjmuje $n$-bitowy ciąg i zwraca jeden bit. Ale teraz, zamiast obiecywać, że funkcja jest wyważona lub stała, obiecujemy, że funkcja jest iloczynem skalarnym ciągu wejściowego $x$ i pewnego tajnego $n$-bitowego ciągu $s$, modulo 2. (Ten iloczyn skalarny modulo 2 nazywa się „binarnym iloczynem skalarnym".) Problem polega na ustaleniu, czym jest ten tajny $n$-bitowy ciąg.

Inaczej mówiąc, dano nam funkcję czarnej skrzynki $f: {0,1}^n \rightarrow {0,1}$ spełniającą $f(x) = s \cdot x$ dla pewnego ciągu $s$, i chcemy dowiedzieć się, czym jest ciąg $s$.

Przyjrzyjmy się, jak algorytm D-J rozwiązuje ten problem:

1. Najpierw Gate Hadamarda jest stosowany do $n$ qubitów wejściowych, a Gate NOT plus Hadamard do qubitu wyjściowego, tworząc stan:

$$
|\Psi\rangle = |-\rangle_{n} \otimes |+\rangle_{n-1} \otimes |+\rangle_{n-2} \otimes ... \otimes |+\rangle_0
$$

   Stan qubitów od 1 do $n$ można zapisać prościej jako sumę po wszystkich $2^n$ stanach bazowych $n$-qubitowych $|00...00\rangle, |00...01\rangle, |000...11\rangle, ..., |111...11\rangle$. Zbiór tych stanów bazowych oznaczamy $\Sigma^n$. (Szczegóły znajdziesz w kursie [Fundamentals of Quantum Algorithms](/learning/courses/fundamentals-of-quantum-algorithms/quantum-query-algorithms/deutsch-jozsa-algorithm).)

$$
|\Psi\rangle = |-\rangle \otimes \frac{1}{\sqrt{2^n}}\sum\limits_{x \in \Sigma^n}{|x\rangle}
$$

2. Następnie Gate $U_f$ jest stosowany do qubitów. Ten Gate przyjmuje pierwsze n qubitów jako wejście (które są teraz w równej superpozycji wszystkich możliwych n-bitowych ciągów) i stosuje funkcję $f(x)=s \cdot x$ do qubitu wyjściowego, tak że ten qubit jest teraz w stanie: $ |- \oplus f(x)\rangle$. Dzięki mechanizmowi phase kickback stan tego qubitu pozostaje niezmieniony, ale niektóre człony w stanie qubitu wejściowego przyjmują znak minus:

$$
|\Psi\rangle = |-\rangle \otimes \frac{1}{\sqrt{2^n}}\sum\limits_{x \in \Sigma^n}{(-1)^{f(x)}|x\rangle}
$$

3. Teraz kolejny zestaw Hadamardów jest stosowany do qubitów od 0 do $n-1$. Śledzenie znaków minus w tym przypadku może być trudne. Pomocne jest wiedza o tym, że stosowanie warstwy Hadamardów do $n$ qubitów w standardowym stanie bazowym $|x\rangle$ można zapisać jako:

$$
H^{\otimes n} |x\rangle = \frac{1}{\sqrt{2^n}}\sum\limits_{y \in \Sigma^n}{(-1)^{x \cdot y}|y\rangle}
$$

Więc stan staje się:

$$
|\Psi\rangle = |-\rangle \otimes \frac{1}{2^n}\sum\limits_{x \in \Sigma^n}\sum\limits_{y \in \Sigma^n}{(-1)^{(s \cdot x) + (x \cdot y)}|y\rangle}
$$

4. Kolejnym krokiem jest zmierzenie pierwszych $n$ bitów. Ale co zmierzymy? Okazuje się, że powyższy stan upraszcza się do: $|\Psi\rangle = |-\rangle \otimes |s\rangle$, ale to wcale nie jest oczywiste. Jeśli chcesz prześledzić matematykę, zajrzyj do kursu Johna Watrouse'a [Fundamentals of Quantum Algorithms](/learning/courses/fundamentals-of-quantum-algorithms/quantum-query-algorithms/deutsch-jozsa-algorithm#the-bernstein-vazirani-problem). Chodzi jednak o to, że mechanizm phase kickback sprawia, że qubity wejściowe znajdują się w stanie $|s\rangle$. Aby więc dowiedzieć się, czym był tajny ciąg $s$, wystarczy zmierzyć qubity!

__Odpowiedź:__

Gdy wprost wypiszesz dwie sumy, powinieneś/powinnaś otrzymać stan z czterema członami (pomińmy stan wyjściowy $|-\rangle$):

$$
|\Psi\rangle = \frac{1}{2}[|0\rangle + (-1)^s |0\rangle + |1\rangle + (-1)^{(s+1)} |1\rangle]
$$

Jeśli $s=0$, to pierwsze dwa człony dodają się konstruktywnie, a ostatnie dwa się znoszą, pozostawiając $|\Psi\rangle = |0\rangle$. Jeśli $s=1$, to ostatnie dwa człony dodają się konstruktywnie, a pierwsze dwa się znoszą, pozostawiając $|\Psi\rangle = |1\rangle$. Tak więc w obu przypadkach $|\Psi\rangle = |s\rangle$. Mam nadzieję, że ten najprostszy przypadek daje ci poczucie, jak działa ogólny przypadek z $n$ qubitami: wszystkie człony inne niż $|s\rangle$ interferują i zanikają, pozostawiając tylko stan $|s\rangle$.

    """
    Create a Bernstein-Vazirani function from a string of 1s and 0s.
    """
    qc = QuantumCircuit(len(s) + 1)
    for index, bit in enumerate(reversed(s)):
        if bit == "1":
            qc.cx(index, len(s))
    return qc
