## Algorytm Deutscha-Jozsy

Algorytm Deutscha był ważnym pierwszym krokiem w pokazaniu, jak komputer kwantowy może być wydajniejszy od komputera klasycznego, ale stanowił jedynie skromne ulepszenie: wymagał tylko jednego zapytania, w porównaniu z dwoma w przypadku klasycznym. W 1992 roku Deutsch wraz ze swoim współpracownikiem, Richardem Jozsą, rozszerzyli oryginalny dwukubitowy algorytm na większą liczbę kubitów. Problem pozostał ten sam: ustalić, czy funkcja jest *zrównoważona* czy *stała*. Tym razem jednak funkcja przekształca $n$ bitów w pojedynczy bit. Funkcja albo zwraca 0 i 1 taką samą liczbę razy (jest *zrównoważona*), albo zawsze zwraca 1 lub zawsze zwraca 0 (jest *stała*).

Oto schemat obwodu tego algorytmu:

![DJ_algo.png](/learning/images/modules/computer-science/deutsch-jozsa/DJ_algo.avif)

Algorytm ten działa w taki sam sposób jak algorytm Deutscha: phase-kickback pozwala odczytać stan kubitu 0, aby ustalić, czy funkcja jest stała czy zrównoważona. W tym przypadku jest to nieco trudniejsze do zauważenia niż w wersji dwukubitowej algorytmu Deutscha, ponieważ stany będą zawierały sumy po $n$ kubitach, więc rozpisanie tych stanów pozostanie dla ciebie opcjonalnym ćwiczeniem na końcu modułu. Algorytm zwróci ciąg bitów składający się z samych zer, jeśli funkcja jest stała, oraz ciąg zawierający przynajmniej jedną jedynkę, jeśli funkcja jest zrównoważona.

Aby zobaczyć, jak ten algorytm działa w Qiskicie, najpierw musimy wygenerować wyrocznię: losową funkcję, która z gwarancją jest albo stała, albo zrównoważona. Poniższy kod w 50% przypadków wygeneruje funkcję zrównoważoną, a w 50% przypadków funkcję stałą. Nie martw się, jeśli nie do końca rozumiesz ten kod — jest skomplikowany i nie jest niezbędny do zrozumienia algorytmu kwantowego.

```python
from qiskit import QuantumCircuit
import numpy as np

def dj_function(num_qubits):
    """
    Create a random Deutsch-Jozsa function.
    """

    qc_dj = QuantumCircuit(num_qubits + 1)
    if np.random.randint(0, 2):
        # Flip output qubits with 50% chance
        qc_dj.x(num_qubits)
    if np.random.randint(0, 2):
        # return constant circuit with 50% chance.
        return qc_dj

    # If the "if" statement above was "TRUE" then we've returned the constant
    # function and the function is complete. If not, we proceed in creating our
    # balanced function. Everything below is to produce the balanced function:

    # select half of all possible states at random:
    on_states = np.random.choice(
        range(2**num_qubits),  # numbers to sample from
        2**num_qubits // 2,  # number of samples
        replace=False,  # makes sure states are only sampled once
    )

    def add_cx(qc_dj, bit_string):
        for qubit, bit in enumerate(reversed(bit_string)):
            if bit == "1":
                qc_dj.x(qubit)
        return qc_dj

    for state in on_states:
        # qc_dj.barrier()  # Barriers are added to help visualize how the functions are created. They can safely be removed.
        qc_dj = add_cx(qc_dj, f"{state:0b}")
        qc_dj.mcx(list(range(num_qubits)), num_qubits)
        qc_dj = add_cx(qc_dj, f"{state:0b}")

    # qc_dj.barrier()

    return qc_dj

n = 3  # number of input qubits

oracle = dj_function(n)

display(oracle.draw("mpl"))
```

To jest funkcja wyroczni, która jest albo zrównoważona, albo stała. Czy patrząc na nią potrafisz stwierdzić, czy wyjście na ostatnim kubicie zależy od wartości podanych na pierwszych $n$ kubitach? Jeśli wyjście na ostatnim kubicie zależy od pierwszych $n$ kubitów, czy potrafisz powiedzieć, czy ta zależna wartość wyjściowa jest zrównoważona, czy nie?

Możemy stwierdzić, czy funkcja jest zrównoważona czy stała, patrząc na powyższy obwód, ale pamiętaj, że na potrzeby tego problemu traktujemy tę funkcję jako "czarną skrzynkę". Nie możemy zajrzeć do środka, żeby zobaczyć schemat obwodu. Zamiast tego musimy odpytywać tę skrzynkę.

Aby odpytać skrzynkę, używamy algorytmu Deutscha-Jozsy i ustalamy, czy funkcja jest stała czy zrównoważona:

blackbox = oracle.to_gate()
blackbox.label = "$U_f$"

qc_dj = QuantumCircuit(n + 1, n)
qc_dj.x(n)
qc_dj.h(range(n + 1))
qc_dj.barrier()
qc_dj.compose(blackbox, inplace=True)
qc_dj.barrier()
qc_dj.h(range(n))
qc_dj.measure(range(n), range(n))

qc_dj.decompose().decompose()
