## Eksperyment 2: QKD z podsłuchiwaczem {#experiment-2-qkd-with-an-eavesdropper}

Zaimplementujemy dokładnie ten sam protokół co poprzednio. Tym razem wstawimy dodatkowy zestaw pomiarów — wykonywanych przez Eve — między Alicją a Bobem.

```python
from qiskit import ClassicalRegister, QuantumCircuit, QuantumRegister

# Qiskit patterns krok 1: Mapowanie problemu na Circuit kwantowy
# QKD krok 1: Losowe bity i bazy dla Alicji

bit_num = 20
qr = QuantumRegister(bit_num, "q")
cr = ClassicalRegister(bit_num, "c")
qc = QuantumCircuit(qr, cr)

# Losowe bity i bazy Alicji, jak poprzednio

abits = np.round(rng.random(bit_num))
abase = np.round(rng.random(bit_num))

# Przygotowanie stanu przez Alicję, jak poprzednio

for n in range(bit_num):
    if abits[n] == 0:
        if abase[n] == 1:
            qc.h(n)
    if abits[n] == 1:
        if abase[n] == 0:
            qc.x(n)
        if abase[n] == 1:
            qc.x(n)
            qc.h(n)

qc.barrier()

# Tutaj następuje podsłuchiwanie!
# Generowanie losowych baz pomiarowych Eve

ebase = np.round(rng.random(bit_num))

for m in range(bit_num):
    if ebase[m] == 1:
        qc.h(m)
    qc.measure(qr[m], cr[m])
```
