# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) to oficjalne pakiety rozszerzające, które dostarczają wyspecjalizowane algorytmy i techniki obliczeń kwantowych. Każdy addon koncentruje się na określonym obszarze i zawiera samouczki z wykonywalnym kodem.

## Dostępne Addons {#available-addons}

| Addon | Pakiet | Opis |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Zmniejsz szerokość i głębokość obwodu przez cięcie bramek i przewodów |
| **SQD** | `qiskit-addon-sqd` | Sample-based Quantum Diagonalization do szacowania stanu podstawowego |
| **OBP** | `qiskit-addon-obp` | Operator Back-Propagation do redukcji głębokości obwodu |
| **MPF** | `qiskit-addon-mpf` | Multi-Product Formulas dla ulepszonej symulacji Hamiltonianu |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximate Quantum Compilation z sieciami tensorowymi |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption do mitygacji błędów |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones do redukcji narzutu próbkowania PEC |

Wszystkie samouczki można uruchomić bezpośrednio w przeglądarce za pomocą Bindera lub na własnym serwerze Jupyter.

Dokumentację API znajdziesz w poszczególnych repozytoriach addonów na [GitHub](https://github.com/Qiskit?q=qiskit-addon).
