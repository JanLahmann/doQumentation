W mechanice kwantowej obserwowalne odpowiadają właściwościom fizycznym, które można zmierzyć.
Rozważając układ spinów, można na przykład chcieć zmierzyć energię układu lub uzyskać informacje o wyrównaniu spinów, takie jak magnetyzacja lub korelacje między spinami.

Aby zmierzyć $n$-qubitową obserwowaną $O$ na komputerze kwantowym, należy przedstawić ją jako sumę iloczynów tensorowych operatorów Pauliego, czyli

$$
O = \sum_{k=1}^K \alpha_k P_k,~~ P_k \in \{I, X, Y, Z\}^{\otimes n},~~ \alpha_k \in \mathbb{R},
$$

gdzie

$$
I = \begin{pmatrix}
1 & 0 \\ 0 & 1
\end{pmatrix}
~~
X = \begin{pmatrix}
0 & 1 \\ 1 & 0
\end{pmatrix}
~~
Y = \begin{pmatrix}
0 & -i \\ i & 0
\end{pmatrix}
~~
Z = \begin{pmatrix}
1 & 0 \\ 0 & -1
\end{pmatrix}
$$

i korzysta się z faktu, że obserwowalna jest hermitowska, tj. $O^\dagger = O$. Jeśli $O$ nie jest hermitowska, nadal można ją rozłożyć jako sumę operatorów Pauliego, ale współczynnik $\alpha_k$ staje się zespolony.

W wielu przypadkach obserwowalna jest naturalnie określona w tej reprezentacji po odwzorowaniu rozważanego układu na Qubit-y.
Na przykład układ spin-1/2 można odwzorować na hamiltonian Isinga

$$
H = \sum_{\langle i, j\rangle} Z_i Z_j - \sum_{i=1}^n X_i,
$$

gdzie indeksy $\langle i, j\rangle$ biegną po oddziałujących spinach, a spiny podlegają poprzecznemu polu w $X$.
Indeks dolny wskazuje, na który Qubit działa operator Pauliego, tzn. $X_i$ stosuje operator $X$ na Qubit-cie $i$, pozostawiając resztę bez zmian.

W Qiskit SDK ten hamiltonian można zbudować za pomocą poniższego kodu.


```python
from qiskit.quantum_info import SparsePauliOp

# define the number of qubits
n = 12

# define the single Pauli terms as ("Paulis", [indices], coefficient)
interactions = [
    ("ZZ", [i, i + 1], 1) for i in range(n - 1)
]  # we assume spins on a 1D line
field = [("X", [i], -1) for i in range(n)]

# build the operator
hamiltonian = SparsePauliOp.from_sparse_list(
    interactions + field, num_qubits=n
)
print(hamiltonian)
```

Jeśli chcemy zmierzyć energię, obserwowaną jest sam hamiltonian. Alternatywnie możemy być zainteresowani pomiarem właściwości układu, takich jak średnia magnetyzacja, zliczając liczbę spinów wyrównanych w kierunku $Z$ z obserwowaną

$$
O = \frac{1}{n} \sum_{i=1} Z_i
$$

W przypadku obserwowanych, które nie są podane w postaci operatorów Pauliego, lecz w postaci macierzowej, należy najpierw przeformułować je w bazie Pauliego, aby móc je oszacować na komputerze kwantowym.
Zawsze można znaleźć taką reprezentację, ponieważ macierze Pauliego tworzą bazę dla hermitowskich macierzy $2^n \times 2^n$.
Rozwijamy obserwowaną $O$ jako

$$
O = \sum_{P \in \{I, X, Y, Z\}^{\otimes n}} \mathrm{Tr}(O P) P,
$$

gdzie suma przebiega po wszystkich możliwych $n$-qubitowych wyrazach Pauliego, a $\mathrm{Tr}(\cdot)$ jest śladem macierzy, który działa jak iloczyn skalarny.
Tę dekompozycję z macierzy na wyrazy Pauliego można zaimplementować za pomocą metody `SparsePauliOp.from_operator`, w następujący sposób:

```python
import numpy as np
from qiskit.quantum_info import SparsePauliOp

matrix = np.array(
    [[-1, 0, 0.5, -1], [0, 1, 1, 0.5], [0.5, 1, -1, 0], [-1, 0.5, 0, 1]]
)

observable = SparsePauliOp.from_operator(matrix)
print(observable)
```

```text
SparsePauliOp(['IZ', 'XI', 'YY'],
              coeffs=[-1. +0.j,  0.5+0.j,  1. -0.j])
```

Oznacza to, że macierz można zapisać w postaci wyrazów Pauliego jako $O = -Z_1 + 0.5 X_2 + Y_2 Y_1$.

<Admonition type="note">
  Pamiętaj, że kolejność iloczynu tensorowego odwzorowuje się na Qubit-y jako $q_n \otimes q_{n-1} \otimes \cdots \otimes q_1$.
</Admonition>

<Admonition type="note">
  Jeśli obserwowalna jest hermitowska (tzn. $O^\dagger = O$), współczynniki Pauliego są liczbami rzeczywistymi.
  Możemy jednak również rozłożyć dowolną inną macierz zespoloną w postaci operatorów Pauliego, jeśli dopuścimy zespolone współczynniki.
</Admonition>

## Pomiar w bazach Pauliego {#measure-in-pauli-bases}

Pomiar rzutuje stan Qubit-a na bazę obliczeniową $\{|0\rangle, |1\rangle\}$. Oznacza to, że można mierzyć tylko obserwowalne diagonalne w tej bazie, takie jak operatory Pauliego składające się wyłącznie z wyrazów $I$ i $Z$.
Pomiar dowolnych wyrazów Pauliego wymaga zatem zmiany bazy w celu ich diagonalizacji. W tym celu należy wykonać następujące transformacje:

$$
\begin{aligned}
    X &\rightarrow Z = H X H \\
    Y &\rightarrow Z = H S^\dagger Y S H,
\end{aligned}
$$

gdzie $H$ to Gate Hadamarda, a $S = \sqrt{Z}$ jest czasem nazywany Gate'em fazy.
Jeśli używasz [Estimator-a](https://docs.quantum.ibm.com/api/qiskit/1.4/qiskit.primitives.Estimator) do obliczania wartości oczekiwanych, transformacje bazy są wykonywane automatycznie.

Poniżej znajduje się przykład pokazujący, jak przygotować obwód kwantowy i ręcznie zmierzyć Qubit 0 w bazie X, Qubit 1 w bazie Y i Qubit 2 w bazie Z.
Stosujemy transformacje pokazane w poprzednim równaniu i otrzymujemy następujący Circuit:

```python
from qiskit.circuit import QuantumCircuit

# create a circuit, where we would like to measure
# q0 in the X basis, q1 in the Y basis and q2 in the Z basis
circuit = QuantumCircuit(3)
circuit.ry(0.8, 0)
circuit.cx(0, 1)
circuit.cx(1, 2)
circuit.barrier()

# diagonalize X with the Hadamard gate
circuit.h(0)

# diagonalize Y with Hadamard as S^\dagger
circuit.sdg(1)
circuit.h(1)

# the Z basis is the default, no action required here

# measure all qubits
circuit.measure_all()
circuit.draw("mpl")
```
