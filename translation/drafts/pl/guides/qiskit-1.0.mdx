# Przewodnik migracji do Qiskit 1.0

Qiskit 1.0 wprowadza nowe gwarancje stabilności i usprawnienia wydajności, jednak
nie jest kompatybilny z żadną poprzednią wersją Qiskit. Zapoznaj się z poniższymi
przewodnikami, aby zaktualizować środowisko i kod do pracy z Qiskit v1.0.

- [Zainstaluj nowy pakiet Qiskit v1.0](/guides/qiskit-1.0-installation), aby uaktualnić środowiska Python i zależności projektu.
- [Zmiany funkcji w Qiskit v1.0](/guides/qiskit-1.0-features) – jak zaktualizować kod.
- [Przełomowe zmiany w pakietach Qiskit v1.0](/guides/metapackage-migration) – poznaj nową i starą strukturę pakietów.

Przeczytaj więcej o korzyściach płynących z Qiskit v1.0 na [blogu IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
