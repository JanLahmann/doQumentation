<summary>
### `resilience_level`
</summary>

Poziom odporności na błędy. Wyższe poziomy generują dokładniejsze wyniki kosztem dłuższego czasu przetwarzania.

**Dopuszczalne wartości**: `0`, `1`, `2`

**Domyślna wartość**: `1`

[Dokumentacja API `resilience_level`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/options-estimator-options#resilience_level)
</details>
