Formuły wieloproduktowe (MPF) umożliwiają dokładniejszą symulację dynamiki układu kwantowego kosztem zwiększonej liczby wykonań Circuit. Jest to technika post-processingowa, która łagodzi błąd wartości oczekiwanych dla stanów poddanych ewolucji czasowej.

Obliczenia klasyczne są używane do rozwiązania układu równań liniowych, który dostarcza współczynników do ważonej kombinacji kilku wykonań Circuit. Użycie tej ważonej kombinacji może zmniejszyć błąd związany z symulacją ewolucji czasowej, pod warunkiem dobrego doboru kroków Trottera. Narzędzie MPF przyjmuje zestaw danych — w tym liczbę kroków Trottera i rząd przybliżenia Trottera — aby przygotować i rozwiązać (lub przybliżyć rozwiązanie) powiązanego układu równań liniowych, który następnie możesz użyć do post-processingu pomiarów wartości oczekiwanych stanu poddanego ewolucji czasowej.
## Instalacja pakietu MPF {#install-the-mpf-package}

Pakiet MPF można zainstalować na dwa sposoby: przez PyPI lub budując ze źródła. Zaleca się instalację w [środowisku wirtualnym](https://docs.python.org/3.10/tutorial/venv.html), aby zapewnić separację między zależnościami pakietów.

### Instalacja z PyPI {#install-from-pypi}

Najprostszym sposobem instalacji pakietu `qiskit-addon-mpf` jest użycie PyPI.
```bash
pip install qiskit-addon-mpf
```

### Budowanie ze źródła {#build-from-source}
Użytkownicy, którzy chcą rozwijać kod w repozytorium lub zainstalować go ręcznie, mogą to zrobić, najpierw klonując repozytorium:
```bash
git clone git@github.com:Qiskit/qiskit-addon-mpf.git
```

i instalując pakiet przez `pip`. Repozytorium zawiera również szereg opcjonalnych zależności, które włączają określone funkcje.

Dostosuj opcje do swoich potrzeb.
```bash
pip install tox notebook -e '.[notebook-dependencies,dev]'
```
## Podstawy teoretyczne {#theoretical-background}

MPF może zmniejszyć błąd przybliżenia Trottera związany z symulacją dynamiki układów kwantowych poprzez ważoną kombinację kilku wykonań Circuit. Ta ważona suma jest zdefiniowana jako:

$$ \mu(t) = \sum_j x_j\rho_j^{k_j}\left(\frac{t}{k_j}\right) + \text{some remaining Trotter error}, $$

gdzie $x_j$ to współczynniki wagowe, $\rho_j^{k_j}$ to macierz gęstości odpowiadająca czystemu stanowi uzyskanemu przez ewolucję stanu początkowego za pomocą formuły produktowej $S^{k_j}$ przybliżającej operator ewolucji czasowej przy użyciu $k_j$ kroków Trottera, a $j$ indeksuje każdą formułę produktową użytą w sumie.

![Several circuits with a varying number of Trotter steps are used to compute the target observable](/docs/images/guides/qiskit-addons/mpf-diagram.avif)

Ogólnie jednak celem symulacji dynamiki kwantowej jest pomiar pewnej obserwowalnej $\mathcal{O}(t)$, która jest funkcją czasu. Przy użyciu MPF wykonywanych jest wiele Circuit — każdy używający $k_j$ kroków Trottera — aby uzyskać kilka pomiarów docelowej obserwowalnej $\mathcal{O}_{k_j}(t)$. Pomiar docelowej obserwowalnej jest następnie uzyskiwany przez obliczenie:

$$ \langle \mathcal{O}(t) \rangle = \sum_j x_j(t) \langle \mathcal{O}_{k_j}(t) \rangle. $$

W istocie, można zmniejszyć całkowity błąd Trottera, przybliżając operator ewolucji czasowej za pomocą kilku formuł produktowych ze zmienną liczbą kroków Trottera zamiast jednej formuły produktowej. Konstruujesz Circuit dla każdego członu w ważonej sumie, który ewoluuje układ zgodnie z każdą z $k_j$ liczb kroków Trottera. Każdy Circuit jest następnie wykonywany osobno na QPU, aby zrekonstruować wyniki w kroku post-processingu. Użyteczność tej techniki można postrzegać z dwóch perspektyw:
1. Dla ustalonej liczby kroków Trottera, które wykonujesz, możesz uzyskać wyniki z błędem Trottera, który jest mniejszy łącznie.
2. Dla liczby kroków Trottera, które skutkują głębokimi Circuit, możesz użyć MPF, aby znaleźć kilka Circuit o mniejszej głębokości do uruchomienia, co skutkuje podobnym błędem przybliżenia Trottera.

## Wyznaczanie współczynników MPF {#determine-mpf-coefficients}

Podstawowa funkcjonalność pakietu `qiskit-addon-mpf` polega na wyznaczaniu współczynników MPF $x_j(t)$ (które mogą być zależne od czasu). Proces uzyskiwania każdego $x_j(t)$ polega na rozwiązaniu układu równań liniowych $Ax=b$, gdzie $x$ jest wektorem współczynników do wyznaczenia, $A$ jest macierzą zależną od każdego z $k_j$ i użytej formuły produktowej $S$ (tj. rzędu przybliżenia i liczby kroków Trottera), a $b$ jest wektorem ograniczeń. Ten układ równań można rozwiązać albo dokładnie, albo za pomocą modelu przybliżonego minimalizującego normę 1 współczynników. Ponadto wybór każdego $k_j$ jest procesem heurystycznym, ale może być ograniczony przez następujące warunki:

1. Wartość **największego** $k_j$ jest ograniczona przez najwyższą głębokość Circuit, która może być niezawodnie wykonana
2. **Najmniejszy** $k_j$ powinien spełniać $dt = t/k_j < 1$, ponieważ tam błąd Trottera zachowuje się najlepiej
3. Żaden ze współczynników $x_j$ nie powinien być bliski $0$, gdyż oznaczałoby to, że nie wnoszą dużego wkładu do MPF
4. Podobnie, współczynnik związany z największą wartością $k_j$ nie powinien dominować, ponieważ oznaczałoby to, że używasz jednej formuły produktowej
5. Wreszcie, norma uzyskanych współczynników $x_j$ powinna być mała, co wskazuje na dobrze uwarunkowane MPF [1](#references)
