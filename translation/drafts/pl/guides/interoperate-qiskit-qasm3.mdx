## Importowanie programu OpenQASM 3 do Qiskit {#import-an-openqasm-3-program-into-qiskit}

Aby skorzystać z tej funkcji, musisz zainstalować pakiet `qiskit_qasm3_import `. Zainstaluj go za pomocą poniższego polecenia.

```bash
pip install qiskit-qasm3-import
```

Dostępne są obecnie dwie wysokopoziomowe funkcje służące do importowania z OpenQASM 3 do Qiskit. Są to `load()`, która przyjmuje nazwę pliku, oraz `loads()`, która przyjmuje sam program w postaci ciągu znaków:

```python
import qiskit.qasm3
qiskit.qasm3.load(file_name)
qiskit.qasm3.loads(program_string)
```
W tym przykładzie definiujemy program kwantowy przy użyciu OpenQASM 3 i używamy `loads()` do bezpośredniej konwersji go na QuantumCircuit:

```python
import qiskit.qasm3

program = """
    OPENQASM 3.0;
    include "stdgates.inc";

    input float[64] a;
    qubit[3] q;
    bit[2] mid;
    bit[3] out;

    let aliased = q[0:1];

    gate my_gate(a) c, t {
      gphase(a / 2);
      ry(a) c;
      cx c, t;
    }
    gate my_phase(a) c {
      ctrl @ inv @ gphase(a) c;
    }

    my_gate(a * 2) aliased[0], q[{1, 2}][0];
    measure q[0] -> mid[0];
    measure q[1] -> mid[1];

    while (mid == "00") {
      reset q[0];
      reset q[1];
      my_gate(a) q[0], q[1];
      my_phase(a - pi/2) q[1];
      mid[0] = measure q[0];
      mid[1] = measure q[1];
    }

    if (mid[0]) {
      let inner_alias = q[{0, 1}];
      reset inner_alias;
    }

    out = measure q;
"""
circuit = qiskit.qasm3.loads(program)
circuit.draw("mpl")
```

## Eksportowanie do OpenQASM 3 {#export-to-openqasm-3}

Kod Qiskit możesz wyeksportować do OpenQASM 3 za pomocą `dumps()`, które eksportuje do ciągu znaków, lub `dump()`, które eksportuje do pliku.

### Przykład z `dumps()` {#example-with-dumps}

```python
from qiskit import QuantumCircuit
from qiskit.qasm3 import dumps

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

dumps(qc)
```

```text
'OPENQASM 3.0;\ninclude "stdgates.inc";\nbit[2] meas;\nqubit[2] q;\nh q[0];\ncx q[0], q[1];\nbarrier q[0], q[1];\nmeas[0] = measure q[0];\nmeas[1] = measure q[1];\n'
```

### Przykład z `dump()` {#example-with-dump}

```python
from qiskit import QuantumCircuit
from qiskit.qasm3 import dump

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

f = open("my_file.txt", "w")
dump(qc, f)
f.close()
```

Więcej informacji znajdziesz w sekcji [Eksportowanie do OpenQASM 3](https://docs.quantum.ibm.com/api/qiskit/qasm3#exporting-to-openqasm-3) dokumentacji API.
