## 5. 초고밀도 부호화 {#5-superdense-coding}

거의 동일한 설정을 전혀 다른 목적에 활용할 수 있습니다. Alice가 Bob에게 두 비트의 고전 정보를 보내고 싶지만, Bob과 고전적인 통신 수단이 없는 상황을 가정해 봅시다. 그러나 Alice와 Bob은 얽힘 쌍을 공유하고 있고, Alice는 자신의 Qubit을 Bob의 위치로 전송할 수 있습니다. 이 점이 양자 텔레포테이션 프로토콜과 대비됩니다. 텔레포테이션에서는 고전적 통신이 가능하고 목표가 양자 상태를 전송하는 것이었다면, 여기서는 고전적 통신이 불가능하고 Qubit의 전송을 이용해 두 비트의 고전 정보를 공유합니다.

### 5.1 초고밀도 부호화 프로토콜 {#51-the-protocol-of-superdense-coding}

**가정**: Alice는 두 비트의 정보, 즉 $a_1a_2 \in \{00, 01, 10, 11\}$를 가지고 있습니다. Alice와 Bob은 얽힘 쌍(e-bit)을 공유하고 있지만, 고전적 통신은 할 수 없습니다.

1. Alice는 자신이 가진 e-bit 부분에 다음 중 하나의 연산을 수행합니다.
    - $a_1a_2 = 00$이면, 아무것도 하지 않습니다
    - $a_1a_2 = 01$이면, Z Gate를 적용합니다
    - $a_1a_2 = 10$이면, X Gate를 적용합니다
    - $a_1a_2 = 11$이면, Z Gate와 X Gate를 적용합니다.
2. Alice는 자신의 e-bit 부분을 Bob의 위치로 전송합니다.
3. Bob은 Alice로부터 받은 Qubit을 제어 Qubit으로, 자신의 Qubit을 타깃으로 하여 CNOT Gate를 적용한 뒤, Alice로부터 받은 Qubit에 H Gate를 적용하고 두 Qubit을 측정합니다. 가능한 초기 상태와 Bob의 연산 결과는 다음과 같습니다:
$$
\frac {|00\rangle + |11\rangle}{\sqrt 2} \rightarrow CX_{01}\otimes H_0 \rightarrow |00\rangle
$$
$$
\frac {|00\rangle - |11\rangle}{\sqrt 2} \rightarrow CX_{01}\otimes H_0 \rightarrow |01\rangle
$$
$$
\frac {|10\rangle + |01\rangle}{\sqrt 2} \rightarrow CX_{01}\otimes H_0 \rightarrow |10\rangle
$$
$$
\frac {|10\rangle - |01\rangle}{\sqrt 2} \rightarrow CX_{01}\otimes H_0 \rightarrow -|11\rangle
$$

$-|11\rangle$의 음수 부호는 전역 위상(global phase)이므로 측정 불가능합니다.

### 5.2 초고밀도 부호화를 시뮬레이션하는 양자 Circuit {#52-quantum-circuit-simulating-the-superdense-coding}

초고밀도 부호화 프로토콜을 바탕으로, 아래와 같이 초고밀도 부호화 Circuit을 구성할 수 있습니다. Alice가 Bob에게 전달하려는 메시지인 `msg`를 변경해 보세요.

```python
from qiskit import QuantumCircuit
```

Qiskit 패턴의 각 단계는 코드 주석에 표시되어 있습니다.

```python
# 단계 1: 문제를 양자 Circuit과 연산자로 매핑
# 2-Qubit Circuit 생성
qc = QuantumCircuit(2, 2)

# Eve가 EPR 쌍을 생성하고 q0을 Alice에게, q1을 Bob에게 전송
qc.h(0)
qc.cx(0, 1)
qc.barrier()

# Alice가 Bob에게 전달하려는 메시지 설정
msg = "11"  # 메시지를 변경할 수 있습니다

if msg == "00":
    pass
elif msg == "10":
    qc.x(0)
elif msg == "01":
    qc.z(0)
elif msg == "11":
    qc.z(0)
    qc.x(0)

qc.barrier()
# Bob이 Alice로부터 EPR Qubit을 받아 유니터리 연산 수행
qc.cx(0, 1)
qc.h(0)
qc.barrier()

# Bob이 q0과 q1을 측정
qc.measure(0, 0)
qc.measure(1, 1)

qc.draw(output="mpl")
```

![이전 코드 셀의 출력](/learning/images/courses/utility-scale-quantum-computing/teleportation/extracted-outputs/041f79eb-c516-4b54-ae23-b06bffd36336-0.avif)

```python
# 먼저 시뮬레이터에서 실행합니다
from qiskit_aer import AerSimulator
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager
from qiskit_ibm_runtime import Sampler

# Backend 정의
backend = AerSimulator()
shots = 1000

# Backend에 맞게 Transpile
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

# 작업 실행
sampler = Sampler(mode=backend)
job_sim = sampler.run([isa_qc], shots=shots)
result_sim = job_sim.result()

# 카운트 데이터 추출
counts = result_sim[0].data.c.get_counts()
print(counts)
```

## 6. 요약 {#6-summary}

이번 세션에서는 두 가지 양자 프로토콜을 구현했습니다. 두 프로토콜 모두 먼 거리의 친구 사이에서 통신하는 시나리오를 다루고 있어 단일 QPU에서의 양자 컴퓨팅과는 다소 거리가 있어 보이지만, 양자 컴퓨팅에서도 실제 응용 분야가 있으며 양자 정보의 전송 원리를 이해하는 데 도움이 됩니다.

- **양자 텔레포테이션**: 양자 상태를 복사할 수는 없지만, 얽힘을 공유함으로써 미지의 양자 상태를 텔레포트할 수 있습니다.
- **양자 초고밀도 부호화**: 얽힘 쌍을 공유하고 Qubit 하나를 전송함으로써 두 비트의 고전 정보를 전달할 수 있습니다.
