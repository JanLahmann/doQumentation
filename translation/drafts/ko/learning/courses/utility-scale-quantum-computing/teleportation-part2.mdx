### Exercise 1: $r_x$ 값 추정 {#exercise-1-estimate-r_x-value}
IBM&reg; 양자 컴퓨터는 $z$축을 따라 측정합니다(때로는 "$z$ 기저에서" 또는 "계산 기저에서" 측정한다고 표현합니다). 그러나 측정 전에 회전을 적용하면 양자 상태의 x축 방향 성분도 측정할 수 있습니다. 좀 더 정확히 말하면, $x$ 방향을 가리키던 것이 $z$ 방향을 가리키도록 시스템을 회전시키면, 하드웨어의 $z$ 방향 측정 장치는 그대로 유지하면서 방금 전까지 $x$ 방향을 따르던 상태에 대한 정보를 얻을 수 있습니다. 대부분의 양자 컴퓨터(IBM 양자 컴퓨터 포함)가 여러 축 방향으로 측정을 수행하는 방식이 바로 이것입니다.

이 원리를 이해했다면, 양자 상태 토모그래피를 사용해 $r_x$ 값을 추정하는 코드를 직접 작성해 보세요.

__풀이__:

```python
# create a 1-qubit quantum state psi from theta, varphi parameters
qc = QuantumCircuit(1, 1)
qc.u(theta, varphi, 0.0, 0)

qc.h(0)
qc.measure(0, 0)

qc.draw(output="mpl")
```

![Output of the previous code cell](/learning/images/courses/utility-scale-quantum-computing/teleportation/extracted-outputs/fcecbe31-9196-4804-8afa-e23e313880df-0.avif)

```python
# Define backend
backend = AerSimulator()
nshots = 10000

# Transpile to backend
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

# Run the job
sampler = Sampler(mode=backend)
job = sampler.run([isa_qc], shots=nshots)
result = job.result()

# Extract counts data
counts = result[0].data.c.get_counts()
print(counts)

# Plot the counts in a histogram
plot_histogram(counts)
```

```text
{'1': 5925, '0': 4075}
```

```python
rx_approx = (counts["0"] - counts["1"]) / nshots

print("rx = ", rx, " and approx of rx = ", rx_approx)
```


### Exercise 2: $r_y$ 값 추정 {#exercise-2-estimate-r_y-value}
앞서와 동일한 논리적 추론을 사용하면, 측정 전에 시스템을 회전시켜 $r_y$에 대한 정보를 얻을 수 있습니다.
양자 상태 토모그래피를 사용해 $r_y$ 값을 추정하는 코드를 직접 작성해 보세요. 앞선 예제를 출발점으로 삼되, 다른 회전을 적용하면 됩니다. (```sdg``` 등 사용되는 다양한 게이트에 대한 자세한 내용은 [API 레퍼런스](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.SdgGate)를 참고하세요.)

__풀이__:

```python
# create a 1-qubit quantum state psi from theta, varphi parameters
qc = QuantumCircuit(1, 1)
qc.u(theta, varphi, 0.0, 0)

qc.sdg(0)
qc.h(0)
qc.measure(0, 0)

qc.draw(output="mpl")
```

```python
# Define backend
backend = AerSimulator()
nshots = 10000

# Transpile to backend
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

# Run the job
sampler = Sampler(mode=backend)
job = sampler.run([isa_qc], shots=nshots)
result = job.result()

# Extract counts data
counts = result[0].data.c.get_counts()

# Plot the counts in a histogram
```
