## 4. 3-Qubit Grover 탐색

이번에는 3-Qubit Grover 탐색 예제를 시도해 보겠습니다.

```python
n = 3
qr = QuantumRegister(n, "q")
anc = QuantumRegister(1, "ancilla")
grover_circuit = QuantumCircuit(qr, anc)
```

```python
# the number of iterations
num_iterations = 2
```

```python
def oracle(circuit):
    circuit.mcx([0, 1, 2], 3)
    circuit.barrier()
```

이번에는 $|111\rangle$이 "정답" 상태입니다.

```python
# Let's do Grover search
initialization(grover_circuit)

for i in range(0, num_iterations):
    oracle(grover_circuit)
    diffusion(grover_circuit)

# Clear the ancilla bit
grover_circuit.h(n)
grover_circuit.x(n)

grover_circuit.draw(output="mpl", idle_wires=False)
```

```python

# Define backend
```

예상대로 $|0111\rangle$이 가장 높은 확률로 관측됩니다. 이 경우 반복 횟수 2회가 최적입니다. 다만, 정답을 얻을 확률이 100%가 아닌 것은 Grover 탐색에서 일반적인 현상입니다.
