### Python으로 보는 LWE 암호화 예시 {#illustration-of-lwe-encryption-in-python}

다음의 간단한 예시는 암호화 및 복호화에 <DefinitionTooltip definition="Learning With Errors. 방정식에 오류를 도입하여 비밀 정보를 숨기는 수학적 기법으로, 방정식을 풀기 어렵게 만듭니다.">LWE</DefinitionTooltip>를 활용하는 방법을 보여줍니다. Bob이 Alice에게 암호화된 메시지를 전송합니다.

먼저 Alice와 Bob은 **문제 매개변수**에 합의합니다. 이 매개변수들은 위의 수학 섹션에서 자세히 설명되어 있지만, 요약하면 $n, q, N, \chi$가 필요합니다.

기본 매개변수부터 시작합니다.

- $N$은 샘플 수를 나타냅니다.
- $q$는 모듈러스입니다.
- n은 보안 매개변수 또는 벡터 차원이라고 합니다.

이 매개변수들은 모두 원칙적으로 공개 정보입니다.

```python
import numpy as np

n = 8
q = 127
N = int(1.1 * n * np.log(q))
sigma = 1.0
print(f"n={n},q={q},N={N},sigma={sigma}")
```

오류를 도입하는 방법도 필요합니다.

여기서 $\chi$는 도입하려는 오류를 나타냅니다. 평균 0, 표준편차 $\sigma$를 가지는 $\mathbb{Z}_{q}$ 위의 이산 가우시안 분포를 사용합니다.

이 오류 도입 방법의 정의 역시 공개 정보입니다.

따라서 Python에서는 표준편차 $\sigma$를 가지는 이산 가우시안 분포 $\chi$에서 추출한 임의의 오류/노이즈 기여값 $\varepsilon$을 근사하는 간단한 함수가 필요합니다.

0을 중심으로 한 분포를 확인할 수 있도록 몇 가지 예시 값을 출력합니다.

```python
def chi(stdev, modulus):
    return round((np.random.randn() * stdev**2)) % modulus

# print some examples
sd = 2
m = 1000
for x in range(10):
    print("chi = ", chi(sd, m))
```

다음으로 Alice는 키 쌍을 생성해야 합니다.

Alice는 0과 $q$ 사이의 $n$개의 값을 선택하여 **개인 키**를 설정합니다.

```python
# Alice's private key
alice_private_key = np.random.randint(0, high=q, size=n)
print(f"Alice's private key: {alice_private_key}")
```

Alice는 임의의 벡터를 선택한 뒤 생성된 오류와 결합하여 **공개 키**를 설정합니다.

<details>
<summary>수학적 설명</summary>

보다 정확하게는, 샘플 $A_{s,\chi}$를 생성해야 합니다.

이를 위해 Alice는 $\mathbb{Z}_{q}^{n}$에서 벡터 $\mathbf{a}$를 임의로 선택하고, $\chi$에서 오류 $\varepsilon$을 뽑아 $N$개의 샘플 $(\mathbf{a}, b)$를 구성합니다.

</details>

```python
# Alice's Public Key
alice_public_key = []

# N is the number of values we want in the key
for i in range(N):
    # Get n random values between 0 and <q
    a = np.random.randint(0, high=q, size=n)
    # get an error to introduce
    epsilon = chi(sigma, q)
    #  calculate dot product (ie like array multiplication)
    b = (np.dot(a, alice_private_key) + epsilon) % q
    # value to be added to the key -
    sample = (a, b)
    alice_public_key.append(sample)

print(f"Alice's public key: {alice_public_key}")
```

Alice는 이제 공개 키를 공유할 수 있습니다.

Bob은 이 공개 키를 사용하여 Alice에게 암호화된 메시지를 보낼 수 있습니다.

Bob의 메시지는 단일 비트입니다.

```python
# Encryption
bob_message_bit = 1
print(f"Bob's message bit={bob_message_bit}")
```

메시지를 암호화하기 위해 Bob은 Alice의 공개 키에서 임의의 수의 샘플을 무작위로 선택하여 암호문을 구성해야 합니다.

이를 위해 Bob은 길이 $N$의 임의의 이진 벡터 $r$, 즉 마스크를 생성합니다.

```python
# a list of N values between 0 and <2 - ie 0 or 1
r = np.random.randint(0, 2, N)
print(r)
```

이제 이 마스크를 Alice의 공개 키의 해당 항목에 적용하여 찾은 값들의 합을 계산합니다.

```python
sum_ai = np.zeros(n, dtype=int)
sum_bi = 0

for i in range(N):
    sum_ai = sum_ai + r[i] * alice_public_key[i][0]
    sum_bi = sum_bi + r[i] * alice_public_key[i][1]
sum_ai = [x % q for x in sum_ai]
# sum_bi = sum_bi
ciphertext = (sum_ai, (bob_message_bit * int(np.floor(q / 2)) + sum_bi) % q)
print(f"ciphertext is: {ciphertext}")
```

마지막으로 Bob은 암호문을 전송하며, Alice는 자신의 개인 키를 사용하여 복호화합니다.

```python
# Decryption
adots = np.dot(ciphertext[0], alice_private_key) % q
b_adots = (ciphertext[1] - adots) % q

decrypted_message_bit = round((2 * b_adots) / q) % 2

print(
    f"original message bit={bob_message_bit}, decrypted message bit={decrypted_message_bit}"
)

assert bob_message_bit == decrypted_message_bit
```

이 프로토콜은 긴 비트 문자열을 암호화할 때 비트 단위로 동작하므로, 루프 안에서 동일한 연산을 반복하기만 하면 됩니다. 다음은 Bob이 16개의 암호화된 비트를 전송하는 시나리오를 보여줍니다.

```python
bob_message_bits = np.random.randint(0, 2, 16)
print(f"Bob's message bits are : {bob_message_bits}")
decrypted_bits = []

for ib in range(len(bob_message_bits)):
    bob_message_bit = bob_message_bits[ib]

    r = np.random.randint(0, 2, N)

    sum_ai = np.zeros(n, dtype=int)
    sum_bi = 0
    for i in range(N):
        sum_ai = sum_ai + r[i] * alice_public_key[i][0]
        sum_bi = sum_bi + r[i] * alice_public_key[i][1]
    sum_ai = [x % q for x in sum_ai]

    ciphertext = (sum_ai, (bob_message_bit * int(np.floor(q / 2)) + sum_bi) % q)

    adots = np.dot(ciphertext[0], alice_private_key) % q
    b_adots = (ciphertext[1] - adots) % q

    decrypted_message_bit = round((2 * b_adots) / q) % 2
    assert decrypted_message_bit == bob_message_bit

    decrypted_bits.append(decrypted_message_bit)

print(f"Decrypted message bits = {np.array(decrypted_bits)}")
```
