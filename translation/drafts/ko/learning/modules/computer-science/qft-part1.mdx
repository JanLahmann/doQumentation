## 소개 {#introduction}

푸리에 변환은 수학, 물리학, 신호 처리, 데이터 압축 등 수많은 분야에서 폭넓게 활용되는 도구입니다. 양자 버전의 푸리에 변환인 양자 푸리에 변환은 가장 중요한 양자 알고리즘의 근간을 이룹니다.

오늘은 고전 푸리에 변환을 간략히 복습한 뒤, 양자 컴퓨터에서 양자 푸리에 변환을 구현하는 방법을 살펴보겠습니다. 그런 다음 양자 푸리에 변환이 위상 추정 알고리즘이라는 알고리즘에 어떻게 응용되는지 알아봅니다. 양자 위상 추정은 양자 컴퓨팅의 "왕관의 보석"으로 불리는 Shor의 유명한 인수분해 알고리즘의 서브루틴입니다. 이 모듈은 Shor의 알고리즘을 다루는 다른 모듈로 이어지지만, 독립적으로도 학습할 수 있도록 구성되어 있습니다. 양자 푸리에 변환 자체가 매우 흥미롭고 유용한 알고리즘이기 때문입니다!

## 고전 푸리에 변환 {#the-classical-fourier-transform}

양자 푸리에 변환으로 넘어가기 전에, 먼저 고전 버전을 간략히 복습해 보겠습니다. 푸리에 변환은 하나의 이른바 "기저(basis)"에서 다른 기저로 변환하는 방법입니다. 두 기저는 동일한 문제를 서로 다른 관점에서 바라보는 것으로 생각할 수 있습니다. 둘 다 함수를 표현하는 유효한 방법이지만, 다루는 문제에 따라 어느 쪽이 더 유용할 수 있습니다. 푸리에 변환으로 연결되는 기저 쌍의 예로는 위치와 운동량, 시간과 주파수가 있습니다.

푸리에 변환이 음파 파형을 바탕으로 악기가 연주하는 음을 파악하는 데 어떻게 도움이 되는지 예시를 통해 살펴보겠습니다. 일반적으로 파형은 시간 기저로 표현됩니다. 즉, 파동의 진폭이 시간의 함수로 나타납니다.

![단일 사인파 신호를 시간의 함수로 나타낸 그래프.](/learning/images/modules/computer-science/qft/Cnote.avif)

이 파형에 푸리에 변환을 적용하면 시간 기저에서 주파수 기저로 이동할 수 있습니다:

![음파 파형의 주파수 스펙트럼. 260 Hz에서 뚜렷하고 날카로운 피크가 하나 나타납니다.](/learning/images/modules/computer-science/qft/Cnotefreq.avif)

주파수 기저에서는 약 260 Hz에서 명확한 피크를 쉽게 확인할 수 있습니다. 이는 가운데 C(미들 C)입니다!

푸리에 변환 없이도 가운데 C가 연주되고 있음을 파악할 수 있었겠지만, 여러 음이 동시에 연주된다면 어떨까요? 그러면 시간 기저에서 파형이 더 복잡해집니다:

![여러 사인파가 동시에 겹쳐 더 복잡한 주기적 패턴을 이루는 변위 대 시간 그래프.](/learning/images/modules/computer-science/qft/Cchord.avif)

하지만 주파수 스펙트럼은 세 개의 피크를 명확하게 보여줍니다:

![위 음파 파형의 주파수 스펙트럼. 약 260 Hz, 330 Hz, 392 Hz에서 피크가 세 개 나타납니다. 마지막 피크는 매우 약하지만 확인할 수 있습니다.](/learning/images/modules/computer-science/qft/Cchordfreq.avif)

이는 C, E, G 음을 연주하는 다장조(C-major) 화음이었습니다.

이러한 푸리에 분석은 복잡한 신호의 주파수 성분을 추출하는 데 도움이 됩니다.

### 이산 푸리에 변환 {#discrete-fourier-transform}

푸리에 변환은 다양한 신호 처리 응용에 유용합니다. 그러나 위의 음악 예시를 포함한 대부분의 실제 응용에서는 연속 함수가 아닌 이산적인 $N$개의 데이터 포인트를 변환해야 합니다. 이 경우 *이산* 푸리에 변환을 사용합니다. 이산 푸리에 변환(DFT)은 벡터 $(x_0, ..., x_{N-1})$에 작용하여 다음 공식에 따라 벡터 $(y_0, ..., y_{N-1})$로 변환합니다:

$$y_k = \frac{1}{\sqrt{N}}\sum_{j=0}^{N-1}x_j\omega_N^{jk}$$

여기서 $\omega_N^{jk} = e^{2\pi i \frac{jk}{N}}$으로 정의합니다. (지수에 음수 부호가 붙는 다른 관례도 있으므로, 실제로 DFT를 접할 때 주의하세요.) $e^{2\pi i \frac{jk}{N}}$는 주기 $\frac{N}{k}$를 갖는 주기 함수임을 기억하세요. 따라서 이 함수를 곱하는 푸리에 변환은 본질적으로 (이산) 함수 $\{x_{j}\}$를 각각 주기 $\frac{N}{k}$를 갖는 구성 주기 함수들의 선형 결합으로 분해하는 방법입니다.

## 양자 푸리에 변환 {#the-quantum-fourier-transform}

지금까지 푸리에 변환이 함수를 새로운 "기저 함수" 집합의 선형 결합으로 표현하는 데 사용됨을 살펴봤습니다. 기저 변환은 Qubit 상태에서도 자주 수행됩니다. 예를 들어, 단일 Qubit $|\psi\rangle$의 상태는 계산 기저(computational basis) $|\psi\rangle = c_0 |0\rangle + c_1 |1\rangle$ (기저 상태 $|0\rangle$과 $|1\rangle$ 사용), 또는 $X$ 기저 $|\psi\rangle = c_+ |+\rangle + c_- |-\rangle$ (기저 상태 $|+\rangle = \frac{1}{\sqrt{2}} (|0\rangle + |1\rangle)$과 $|-\rangle = \frac{1}{\sqrt{2}} (|0\rangle - |1\rangle)$ 사용)로 표현할 수 있습니다. 둘 다 유효하지만, 풀려는 문제에 따라 어느 쪽이 더 자연스러울 수 있습니다.

Qubit 상태는 푸리에 기저로도 표현할 수 있습니다. 이 경우 상태가 일반적인 계산 기저 상태 $|x\rangle$ 대신 푸리에 기저 상태 $|\phi_y\rangle$의 선형 결합으로 표현됩니다. 이를 위해 양자 푸리에 변환(QFT)을 적용합니다:

$$ | \phi_y \rangle =  \frac{1}{\sqrt{N}}\sum_{x=0}^{N-1}\omega_N^{y x} \vert x \rangle$$

위와 같이 $\omega_N^{yx} = e^{\frac{2\pi i y x}{N}}$이고, $N$은 양자 시스템의 기저 상태 수입니다. Qubit을 다루므로 $m$개의 Qubit은 $2^m$개의 기저 상태를 가져 $N=2^m$임을 유의하세요. 여기서 기저 상태는 $x$가 $0$에서 $N-1$까지 변하는 단일 숫자 $|x\rangle$로 표기되지만, 보통은 $|00...00\rangle$, $|00...01\rangle$, $|00...11\rangle$, ..., $|11...11\rangle$과 같이 각 이진수가 오른쪽에서 왼쪽으로 Qubit 0부터 $m-1$까지의 상태를 나타내는 형태로 표현됩니다. 이 이진 상태를 단일 숫자로 변환하는 방법은 간단합니다: 이진수로 취급하면 됩니다! 따라서 $|00...00\rangle = |0\rangle$, $|00...01\rangle = |1\rangle$, $|00...10\rangle = |2\rangle$, $|00...11\rangle = |3\rangle$, ..., $|11...11\rangle = |2^m -1\rangle = |N-1\rangle$입니다.

### 푸리에 기저 상태에 대한 직관 쌓기 {#build-intuition-for-the-fourier-basis-states}

지금까지 계산 기저 상태가 무엇인지, 그리고 어떻게 순서가 매겨지는지 살펴봤습니다. 계산 기저 상태는 각 Qubit이 $0$ 또는 $1$인 상태들의 집합이며, 모든 Qubit이 $0$인 상태 $|00...00\rangle$부터 모두 $1$인 상태 $|11...11\rangle$까지 순서가 정해집니다.

그렇다면 *푸리에* 기저 상태는 어떻게 이해할 수 있을까요? 모든 푸리에 기저 상태는 계산 기저 상태들의 동등한 중첩이지만, 각 상태는 성분들의 *위상*이 주기적으로 변하는 방식에서 서로 다릅니다. 이를 더 구체적으로 이해하기 위해 두 Qubit 시스템의 네 가지 푸리에 기저 상태를 살펴보겠습니다. 가장 낮은 푸리에 상태는 위상이 전혀 변하지 않는 상태입니다:

$$|\phi_0\rangle = \frac{1}{2} (|00\rangle + |01\rangle + |10\rangle + |11\rangle)$$

각 항의 복소 진폭을 그래프로 시각화해 보겠습니다. 빨간 선은 계산 기저 상태를 따라가면서 이 진폭의 위상이 복소 평면 위에서 어떻게 변하는지 보여줍니다. $|\phi_0\rangle$에서는 위상이 일정하게 유지됩니다:

![계산 기저 상태(z축)별 $phi_0$의 복소 진폭(x-y 평면)을 나타낸 막대 그래프. 모두 실수이므로 막대가 x축의 +1 방향을 가리킵니다.](/learning/images/modules/computer-science/qft/phi0.avif)

다음 푸리에 기저 상태는 성분들의 위상이 $0$에서 $2\pi$까지 정확히 한 번 회전하는 상태입니다:

$$|\phi_1\rangle = \frac{1}{2} (|00\rangle + e^{i\pi/2}|01\rangle + e^{i\pi}|10\rangle + e^{3i\pi/2}|11\rangle) = \frac{1}{2}(|00\rangle + i|01\rangle - |10\rangle - i|11\rangle)$$

계산 기저 상태 대비 복소 진폭 그래프에서 이 위상 회전을 확인할 수 있습니다:

![계산 기저 상태(z축)별 $phi_1$의 복소 진폭(x-y 평면)을 나타낸 막대 그래프. 빨간 선은 모든 계산 기저 상태를 순서대로 거치면서 복소 위상이 $2\pi$를 한 번 회전하도록 누적됨을 보여줍니다.](/learning/images/modules/computer-science/qft/phi1.avif)

이 예시에서는 기저 상태가 네 개($N=4$)이므로, 각 상태는 표준 순서로 정렬했을 때 이전 상태보다 $2\pi/4$ 라디안만큼 위상이 높습니다. 다음 기저 상태는 $0$에서 $2\pi$까지 두 번 회전합니다:

$$|\phi_2\rangle = \frac{1}{2} (|00\rangle + e^{i\pi}|01\rangle + e^{2i\pi}|10\rangle + e^{3i\pi}|11\rangle) = \frac{1}{2} (|00\rangle - |01\rangle + |10\rangle - |11\rangle)$$

![계산 기저 상태(z축)별 $phi_2$의 복소 진폭(x-y 평면)을 나타낸 막대 그래프. 빨간 선은 모든 계산 기저 상태를 순서대로 거치면서 복소 위상이 $2\pi$를 두 번 회전하도록 누적됨을 보여줍니다.](/learning/images/modules/computer-science/qft/phi2.avif)

마지막으로, 가장 높은 푸리에 성분은 위상 변화가 가장 빠른 상태입니다. 두 Qubit 예시에서는 위상이 $0$에서 $2\pi$까지 세 번 회전하는 상태입니다:

$$|\phi_3\rangle = \frac{1}{2} (|00\rangle + e^{3i\pi/2}|01\rangle + e^{6i\pi/2}|10\rangle + e^{9i\pi/2}|11\rangle) = \frac{1}{2} (|00\rangle - i|01\rangle - |10\rangle + i|11\rangle)$$

![계산 기저 상태(z축)별 $phi_3$의 복소 진폭(x-y 평면)을 나타낸 막대 그래프. 빨간 선은 모든 계산 기저 상태를 순서대로 거치면서 복소 위상이 $2\pi$를 세 번 회전하도록 누적됨을 보여줍니다.](/learning/images/modules/computer-science/qft/phi3.avif)

일반적으로 $m$-Qubit 상태에는 $2^m$개의 푸리에 기저 상태가 있으며, 위상 변화의 주파수는 $|\phi_0\rangle$의 상수(일정)부터 $|\phi_{2^m-1}\rangle$의 빠른 변화까지 이어지고, 상태들의 중첩에서 $2\pi$ 주위를 $2^m-1$번 감습니다. 따라서 양자 상태에 QFT를 적용하면, 음악 파형에서 수행한 분석과 본질적으로 동일한 작업을 하는 것입니다. 즉, 관심 있는 양자 상태를 구성하는 푸리에 주파수 성분을 파악하는 것입니다.
