## 첫 번째 실험: 단일 측정 {#background}

이 첫 번째 실험과 이후 모듈 전반에 걸쳐, "Qiskit 패턴(Qiskit patterns)"이라 불리는 양자 컴퓨팅 프레임워크를 사용합니다. 이 프레임워크는 작업 흐름을 다음 단계로 구분합니다.

- 1단계: 고전적 입력을 양자 문제로 변환
- 2단계: 양자 실행을 위한 문제 최적화
- 3단계: Qiskit Runtime Primitives를 사용한 실행
- 4단계: 후처리 및 고전적 분석

이 단계들을 대체로 따르되, 항상 명시적으로 표시하지는 않을 수 있습니다.

### 1단계: 고전적 입력을 양자 문제로 변환 {#what-should-we-expect-classically}

여기서 고전적 입력은 Stern-Gerlach 장치에서 측정 전 스핀의 방향입니다. 측정 전 양자 상태의 정확한 성질에 대해서는 너무 깊이 걱정하지 않아도 됩니다. 그 내용은 벨 정리(Bell's theorem)를 다루는 별도의 Qiskit Classrooms 모듈의 주제입니다.

IBM 양자 컴퓨터는 $z$ 축을 따라 상태를 측정한다는 점에 유의하세요. 따라서 이 첫 번째 실험은 자기장 기울기가 $z$ 방향인 Stern-Gerlach 실험과 매우 유사합니다. 다른 방향으로 측정하는 방법은 이후에 살펴보겠습니다.

먼저 스핀 상태의 유사체, 즉 $|\uparrow\rangle$와 $|\downarrow\rangle$, 또는 동등하게 $|0\rangle$와 $|1\rangle$의 어떤 혼합을 만들어 보겠습니다. 아래에 초기 값을 제안해 두었습니다. 다른 값이나 무작위 값을 자유롭게 사용해 보세요.

```python
import random
from numpy import pi
import numpy as np

# Use these lines to choose your own arbitrary state vector and normalize it.
# a = 2
# b = (1+1j)
# norm = np.sqrt(a*np.conjugate(a)+b*np.conjugate(b))
# a = a/norm
# b = b/norm
# print(a,b)

# Use these lines if you would rather look at at random spin orientations.
a = random.random()
b = random.random()
norm = np.sqrt(a * np.conjugate(a) + b * np.conjugate(b))
a = a / norm
b = b / norm
print(a, b)
```


이제 위의 각도를 양자 Circuit의 파라미터로 사용합니다. 한 번에 하나의 입자만 고려하므로 Circuit에는 Qubit 하나만 사용하고, 고전 레지스터도 하나만 필요합니다.

```python
from qiskit.circuit import QuantumRegister, ClassicalRegister, QuantumCircuit, Parameter

# Define registers
qr = QuantumRegister(1, "q")
cr = ClassicalRegister(1, "c")
qc = QuantumCircuit(qr, cr)

# Initialize the quantum state
qc.initialize([a, b])
qc.measure(0, 0)
qc.draw("mpl")
```

## 세 번째 실험: 오븐 속의 무작위 스핀 {#third-experiment-random-spins-in-an-oven}

슈테른-게를라흐 실험에서 연구자들은 오븐에서 나오는 각운동량 벡터의 방향을 직접 지정할 수 없었습니다. 방향은 무작위였습니다 (또는 그보다 더 신비로운 무언가였습니다! Bell 정리에 관한 Qiskit Classroom 모듈을 참고하세요). 이 실험의 합리적인 유사 실험은 Qubit의 상태를 무작위로 초기화하고 여러 번 측정하는 것입니다.

### 1단계: 고전 입력을 양자 문제로 변환하기 {#how-will-qubits-help-us-test-this}

구성하려는 Circuit은 이전과 동일합니다. 다른 점은 이번에는 자유 매개변수 $\theta$와 $\phi$를 사용하는 게이트로 Circuit을 구성한다는 것입니다. 이 매개변수의 수치는 각각의 새로운 실행에서 할당됩니다.

```python
# from qiskit.circuit import QuantumCircuit, Parameter

theta = Parameter("θ")
phi = Parameter("$\phi$")

# Define registers
qr = QuantumRegister(1, "q")
cr = ClassicalRegister(1, "c")
qc = QuantumCircuit(qr, cr)

# Add rotation gates for rotating the state of qubit 0 to random orientations
qc.rx(theta, 0)
qc.rz(phi, 0)
qc.measure(0, 0)

qc.draw("mpl")
```

```python
# Import an estimator, this time from qiskit (we import from Runtime for real hardware)
from qiskit.primitives import BackendSamplerV2

# This generates a simulator that mimics the real quantum system
backend_sim = AerSimulator.from_backend(backend)

# from qiskit.primitives import BackendSamplerV2
```

```python
# A list to store the accumulated probabilities of the two possible measurement outcomes.
probslist = {"0": 0.0, "1": 0.0}

# Choose how many "particles"/measurements
measurements = 100
num_shots = 1

for i in range(measurements):
    # Assign a random orientation for each measurement
    phi = random.random() * 2 * pi
    theta = random.random() * 2 * pi

    angles = [phi, theta]
    circuit = qc.assign_parameters(angles)
    qc_ibm = pm.run(circuit)

    # Run the circuit
    # job = sampler.run([circuit],num_shots = 1)
    dist = sampler_sim.run([qc_ibm], shots=num_shots).result()

    # Update the list of probabilities
    zeroterm = dist[0].data.c.get_counts().get("0") or 0
    oneterm = dist[0].data.c.get_counts().get("1") or 0
    probslist.update({"0": probslist.get("0") + zeroterm})
    probslist.update({"1": probslist.get("1") + oneterm})

probslist.update({"0": probslist.get("0") / measurements})
probslist.update({"1": probslist.get("1") / measurements})
# print(probslist)
```

## 네 번째 실험: 반복 측정 {#first-experiment-a-single-measurement}

Qubit이 무작위 상태에서 시작할 때, 두 극값 중 하나가 측정될 확률은 대략 50대50임을 확인했습니다. 그렇다면 측정 이후 Qubit의 상태(또는 입자의 각운동량)는 어떻게 될까요? 이를 알아보기 위해 동일한 Qubit에 대해 여러 번 측정을 수행할 수 있는 Circuit을 정의해 보겠습니다. 이 실험에 사용할 Circuit을 구성해 봅시다. $|0\rangle$과 $|1\rangle$ 두 상태 모두를 측정할 수 있어야 하므로, Qubit의 초기 상태를 기본값인 $|0\rangle$에서 회전시킬 무언가가 필요합니다. 여기서는 아다마르(Hadamard) Gate $H$를 사용하겠습니다. $H|0\rangle = \frac{1}{\sqrt{2}}(|0\rangle+|1\rangle)$이기 때문입니다. 기본적으로 이 두 측정은 모두 $z$ 방향을 따라 수행됩니다.

```python
from qiskit import QuantumCircuit

# Define registers
qr = QuantumRegister(1, "q")
cr = ClassicalRegister(2, "c")
qc = QuantumCircuit(qr, cr)

# Initialize the qubit to be a mixture of 0 and 1 states.
qc.h(0)

# Add a first measurement
qc.measure(0, 0)
qc.barrier()

# Add a second measurement
qc.measure(0, 1)

qc.draw("mpl")

qc_ibm = pm.run(qc)
```

```python
# Step 3: Run the job

num_shots = 1000

```

위 그림에서 막대는 "00", "01" 등으로 표시되어 있습니다. 여기서 이 숫자들은 "두 번째 결과, 첫 번째 결과"를 나타냅니다. 따라서 "00"은 두 번의 측정 모두 $|0\rangle$ 상태가 나왔음을 의미하고, "01"은 첫 번째 측정에서 $|1\rangle$이, 두 번째 측정에서 $|0\rangle$이 나왔음을 의미합니다. 대부분의 Qubit은 $|0\rangle$으로 두 번 측정되거나 $|1\rangle$로 두 번 측정되었습니다. Qubit이 한 상태에서 측정된 후 다른 상태로 측정되는 경우는 매우 드물었으며, 그런 경우(약 1%)는 노이즈 때문입니다. 이 경우 해당 노이즈는 실제 양자 시스템의 동작에 부합하는 방식으로 시뮬레이션된 것입니다. 연속적인 측정 간의 이러한 상관관계는 스핀 각운동량을 가진 입자에 대한 슈테른-게를라흐(Stern-Gerlach) 방식의 측정에서도 동일하게 관찰됩니다. 입자가 "스핀-업(spin-up)"으로 측정되면, 그 직후에 이루어지는 측정에서도 다시 "스핀-업"이 나타납니다(노이즈로 인한 약간의 변동은 있을 수 있습니다).

이것이 사소하게 보일 수도 있습니다. 결국 Qubit이 어떤 상태에서 측정된 후 바로 다시 측정하면, 여전히 같은 상태에 있지 않겠냐고 생각할 수도 있습니다. 하지만 이 미묘한 점을 제대로 파악한다면, 이 현상을 설명하는 수학적 도구를 선택하는 데 도움이 될 수 있습니다.
## 지금까지의 실험 분석 {#analysis-of-experiments-thus-far}

지금까지의 실험에서 얻은 몇 가지 관찰 결과를 정리해 보겠습니다.
- 양자 역학적 시스템을 측정하면 "허용된 값" 집합 중 하나만 얻을 수 있습니다. Qubit 또는 스핀-1/2 입자와 같은 이준위 시스템의 경우, 측정은 두 가지 이진 결과 중 하나만 산출합니다.
- 이진 시스템의 상태를 무작위로 초기화하면(예: 오븐에서 나오는 스핀-1/2 입자), 두 가지 이진 측정 결과 중 어느 것도 나올 수 있습니다.
- 측정이 이루어져 시스템의 상태가 알려지면, 동일한 물리적 관측량을 반복 측정해도 상태가 변하지 않습니다! 즉, 0 상태를 한 번 얻었다면, 다시 측정해도 여전히 0 상태입니다(시스템 잡음 ~0.1%~1%의 영향은 제외).

아직 양자 역학의 확률적 특성을 다루지 않았고, 상태가 고유 상태로 "붕괴"한다는 내용도 언급하지 않았다는 점을 명심하세요. 위의 관찰만을 바탕으로 하면, $z$ 방향으로 두 번 측정해도 같은 결과가 나오므로, 특별한 양자 역학적 상태 $|\psi\rangle$를 어떤 상수만큼 변화시키지 않고 그대로 두는 수학적 연산 $O$, 즉 $O|\psi\rangle = c|\psi \rangle$를 찾고 싶어질 수 있습니다. 결국 이 탐색만으로는 우리가 앞으로 보게 될 모든 현상을 기술하지 못할 것입니다. 하지만 일부 현상은 기술할 수 있으므로, 조금 더 추구해 보겠습니다.

그런 연산은 실제로 존재합니다. 행렬 연산은 어떤 벡터에는 변화를 주지만, 다른 벡터(고유 벡터)에는 상수 배만큼의 변화만 줍니다. 예를 들어, 행렬 $M$과 벡터 $|v\rangle$를 다음과 같이 정의해 봅시다.
$$
M= \begin{pmatrix} 1 & 2 \\ 2 & 1\end{pmatrix}
$$
그리고
$$
|v\rangle = \begin{pmatrix} 1\\1 \end{pmatrix}
$$
그러면
$$
M|v\rangle= \begin{pmatrix} 1 & 2 \\ 2 & 1\end{pmatrix}\begin{pmatrix} 1\\1 \end{pmatrix} = \begin{pmatrix} 1*1+2*1\\2*1+1*1 \end{pmatrix} = \begin{pmatrix} 3\\3 \end{pmatrix} = 3|v\rangle.
$$
임을 알 수 있습니다. 그러나 다른 벡터, 예를 들어 $|v'\rangle = \begin{pmatrix} 1\\2\end{pmatrix}$에 대해서는
$$
M|v'\rangle= \begin{pmatrix} 1 & 2 \\ 2 & 1\end{pmatrix}\begin{pmatrix} 1\\2 \end{pmatrix} = \begin{pmatrix} 1*1+2*2\\2*1+1*2 \end{pmatrix} = \begin{pmatrix} 5\\4 \end{pmatrix} \neq c|v'\rangle.
$$
입니다.

입자의 스핀을 행렬로 기술하고, 측정 후 입자의 상태를 "상태 벡터"라 불리는 벡터로 기술하고자 할 수 있습니다. 이러한 행렬이나 상태 벡터에 어떤 값이 들어가야 하는지는 전혀 자명하지 않지만, 지금까지의 측정에서 상태를 레이블링하는 데 사용할 수 있는 유일한 속성은 0 또는 1을 측정할 확률(스핀-1/2 입자의 맥락에서 "스핀 업" 또는 "스핀 다운")입니다. 상태 벡터의 성분들이 이 확률과 관련되어야 한다고 생각해야 합니다(정확한 확률 자체일 수도 있고, "확률 진폭"—즉, 성분을 제곱하면 확률이 되는 값—일 수도 있습니다). 그러나 이 시점에서는 이러한 행렬의 성분이 엄밀히 실수여야 하는지, 복소수여야 하는지 확실하지 않습니다. 안전하게, 벡터와 행렬이 다음 조건을 만족하는 프레임워크를 개발해 보겠습니다.
- 행렬 연산자는 가능한 한 실험과 연결되어야 합니다. 예를 들어, 스핀 행렬의 고유값을 실험에서 관측된 스핀 투영값과 연결지을 수 있습니다.
- 상태 벡터는 확률과 다음과 같은 방식으로 연결되어야 합니다: 입자가 상태 벡터 $|A\rangle$에 있을 때, 이후 측정에서 입자가 상태 $|B\rangle$에 있을 확률은 $P_{AB}\equiv |\langle A| B \rangle|^2$입니다.

이는 첫 번째 행렬을 구성하는 데 많은 자유도를 줍니다. 예를 들어, 단순하게 다음을 시도해 볼 수 있습니다.

$$
|0\rangle \sim |\uparrow\rangle \sim \begin{pmatrix} 1 \\0 \end{pmatrix}.
$$

여기서 $|0\rangle \sim |\uparrow\rangle$는 양자 컴퓨터의 상태와 스핀-1/2 시스템의 상태가 매우 유사하며, 종종 서로 대응시킨다는 의미입니다. 이들은 서로 다른 시스템을 지칭하므로 완전히 동일한 것은 아닙니다. 하지만 이 두 이준위 시스템을 기술하는 대수는 동일한 규칙을 따를 수 있습니다(결론을 먼저 말씀드리면, 실제로 그렇습니다!). 이 선택은 이미 하나의 좋은 성질을 갖고 있다는 점에 주목하세요.

$$
P_{00} = |\langle 0| 0 \rangle|^2 = \vert \begin{pmatrix} 1 & 0\end{pmatrix} \begin{pmatrix} 1 \\ 0\end{pmatrix}\vert ^2 = |1|^2 = 1.
$$

즉, 입자가 이미 $|0\rangle$ 상태에 있다면, 이후 측정에서도 $|0\rangle$이 나올 확률은 1입니다(잡음 효과 제외). 이는 매우 좋은 결과입니다. 왜냐하면 우리는 이미 상태가 0 또는 "스핀 업" 상태에 있으면, 이후 측정에서도 그 상태를 유지한다는 것을 확인했기 때문입니다. 위의 확률은 실제로 100%여야 합니다.

위의 실험에서 처음에 $|0\rangle$ 상태에 있던 Qubit는 이후 측정에서도 그 상태를 유지했다는 사실을 기억하세요. 같은 내용이 $|1\rangle$에도 적용됩니다. 이는 $|0\rangle$ 상태에 있는 Qubit가 $|1\rangle$ 상태로 측정될 확률이 0임을 의미합니다(잡음 효과 제외). 내적과 측정 확률의 연결 관계에 따르면

$$
P_{01} = P_{10}=0.
$$

일반성을 잃지 않고, 상태 벡터 $|1\rangle = \begin{pmatrix} a \\ b \end{pmatrix}$로 쓸 수 있습니다. 그러면

$$
P_{01} = |\langle 0| 1 \rangle|^2 = \vert \begin{pmatrix} 1 & 0\end{pmatrix} \begin{pmatrix} a \\ b\end{pmatrix}\vert ^2 = |a|^2 = 0.
$$

$P_{11} = 1$이라는 요건, 즉 "정규화 조건"으로부터 $|b|^2=1$이 됩니다. 이것만으로는 $b=e^{i\phi}$($\phi \in \mathbb{R}$)로 제한될 뿐입니다. 사실 이 주제의 입문을 넘어서는 다른 이유들로 인해 $b=1$을 선택합니다. 지금은 $b=1$이 하나의 허용 가능한 해라는 점만 알아두세요.

분석을 꽤 많이 진행했습니다. 상태 벡터의 형태를 선택함으로써, 여기서 작용하는 물리 현상을 기술하는 행렬을 구성할 수 있습니다. 특히, 원래의 슈테른-게를라흐 실험은 $z$축을 따른 스핀 각운동량 성분에 따른 궤적 분리를 측정했으므로, 정확히 그것을 기술하는 연산자 $S_z$를 원합니다. 실험과의 또 다른 핵심 연결은, 편향량, 비행 시간, 알려진 자기장 세기로부터 스핀의 $z$ 성분 크기를 결정할 수 있다는 점입니다. 이를 위해서는 실험 장치의 정밀도에 대한 많은 가정이 필요하지만, 여기서는 측정된 $z$ 성분 스핀 각운동량이 $\pm \hbar/2$임을 그대로 인용하겠습니다.

따라서 우리는 (에르미트 행렬에 의해 만족되는) 실수 고유값을 가지며, 고유값이 실험에서 관측된 스핀 성분에 해당하는 행렬을 구하고자 합니다. 일반성을 잃지 않고 $S_z = \begin{pmatrix} s_{11} & s_{12} \\ s_{21} & s_{22}\end{pmatrix}$로 쓰고, 다음을 요구합니다.

$$
S_z |\uparrow\rangle = \begin{pmatrix} s_{11} & s_{12} \\ s_{21} & s_{22}\end{pmatrix}\begin{pmatrix}1 \\0\end{pmatrix} = \begin{pmatrix} s_{11} \\ s_{21} \end{pmatrix} \overset{!}{=} \frac{\hbar}{2} \begin{pmatrix}1 \\0\end{pmatrix} \rightarrow s_{11} = \hbar/2,s_{21}=0
$$
$$
S_z |\downarrow\rangle = \begin{pmatrix} s_{11} & s_{12} \\ s_{21} & s_{22}\end{pmatrix}\begin{pmatrix}0 \\1\end{pmatrix} = \begin{pmatrix} s_{12} \\ s_{22} \end{pmatrix} \overset{!}{=} -\frac{\hbar}{2} \begin{pmatrix}0 \\1\end{pmatrix} \rightarrow s_{12} = 0, s_{22}=-\hbar/2
$$

이를 합치고 전체 인수 $\hbar/2$를 밖으로 빼면

$$
S_z = \frac{\hbar}{2} \begin{pmatrix} 1 & 0 \\ 0 & -1\end{pmatrix}.
$$
이것은 모든 양자 역학 교과서에 등장하는 잘 알려진 스핀-z 연산자입니다. 종종 $\hbar/2$를 제거한 형태로 나타나는데, 이때를 "파울리-z" 연산자라 하며 보통 $\sigma_z$로 표기합니다.

$$
\sigma_z = \begin{pmatrix} 1 & 0 \\ 0 & -1\end{pmatrix}.
$$

이는 유용합니다. 이 파울리 연산자(및 관련 행렬)는 양자 컴퓨터의 트랜스몬 Qubit 상태를 포함하여, 두 준위 또는 두 가지 측정 결과를 갖는 많은 물리 시스템을 기술할 수 있기 때문입니다.

다른 연산자와 이와 관련된 상태들로 넘어가기 전에, 많은 사람들이 잘못 이해하는 한 가지 사실을 짚고 넘어가야 합니다. __연산자의 작용은 측정과 다릅니다.__ 어떤 의미에서는 "물론 다르죠! 하나는 종이 위에서 수학적 계산의 일부로 이루어지고, 다른 하나는 실험실에서 물리 시스템에 실제로 이루어집니다."라고 말할 수 있습니다. 네, 맞습니다. 하지만 그 이상입니다. 스핀의 $z$ 성분을 측정하면, 시스템의 초기 상태에 관계없이 항상 "스핀 업" 또는 "스핀 다운" 상태가 나옵니다. 우리는 이것을 $|0\rangle$과 $|1\rangle$이라는 양자 컴퓨팅 유사체로 확인했습니다. 수백 개의 무작위 방향으로 상태를 초기화했을 때, 측정은 항상 $|0\rangle$ 또는 $|1\rangle$을 산출했습니다. 이를 측정이 상태를 고유 상태로 "붕괴"시킨다고 합니다. 이는 행렬을 상태에 적용할 때는 발생하지 않습니다. 아래 문제를 풀어보며 이를 탐구해 보세요.

<summary>

스핀 상태

$$
|\psi\rangle = \begin{pmatrix}\sqrt{3/5} \\ \sqrt{2/5}\end{pmatrix}
$$

에 있는 입자가 있다고 가정하겠습니다.

(a) 이 상태 벡터에 $S_z$ 연산자를 적용하면 어떤 결과가 나오나요? (b) 이 입자의 스핀 $z$ 성분을 한 번 측정하면 어떤 값을 얻게 되나요? (c) 이 상태와 동일하게 준비된 입자를 여러 개 만들어 스핀의 $z$ 성분을 수천 번 측정하면 어떤 결과를 얻게 되나요?

</summary>

__정답:__

(a) 다음과 같은 결과를 얻습니다.

$$
S_z |\psi\rangle = \frac{\hbar}{2} \begin{pmatrix} 1 & 0 \\ 0 & -1\end{pmatrix} \begin{pmatrix}\sqrt{3/5} \\ \sqrt{2/5}\end{pmatrix}
$$
$$
= \frac{\hbar}{2} \begin{pmatrix}\sqrt{3/5} \\ -\sqrt{2/5}\end{pmatrix}.
$$

이것이 전부입니다. 실험적인 결과는 존재하지 않습니다. 행렬을 벡터에 적용하기만 하면, 새로운 "−" 부호가 생긴 약간 다른 벡터가 나오며, 이 벡터는 앞에 $\hbar/2$라는 인수가 곱해져 있으므로 더 이상 크기가 1이 아닙니다.

(b) $|\uparrow\rangle$ 또는 $|\downarrow\rangle$를 얻게 됩니다. 이는 $z$ 축 위의 스핀 투영값으로 각각 $\hbar/2$ 또는 $-\hbar/2$를 관측하는 것에 해당합니다. 각 결과의 확률도 다음과 같이 구할 수 있습니다.
$$
P_{\uparrow\psi}=|\langle \uparrow|\psi\rangle|^2 = \vert \begin{pmatrix}1 & 0\end{pmatrix} \begin{pmatrix}\sqrt{3/5} \\ \sqrt{2/5}\end{pmatrix}\vert^2 = |\sqrt{3/5}|^2 = \frac{3}{5}
$$
$$
P_{\downarrow\psi}=|\langle \downarrow|\psi\rangle|^2 = \vert \begin{pmatrix}0 & 1\end{pmatrix} \begin{pmatrix}\sqrt{3/5} \\ \sqrt{2/5}\end{pmatrix}\vert^2 = |\sqrt{2/5}|^2 = \frac{2}{5}
$$

(c) (b)에서 계산한 측정 확률에 따르면, 측정의 약 60%는 스핀-업 입자를, 약 40%는 스핀-다운 입자를 반환합니다.

## 다섯 번째 실험: 다른 관측량 측정하기 {#fifth-experiment-measuring-different-observables}

지금까지 우리는 z축 방향으로만 측정을 수행했습니다. Stern-Gerlach 실험에서 예를 들어 x축 방향으로 측정하고 싶다면, 불균일 자기장의 방향을 $x$축으로 맞추고 스크린에서 $x$축 방향의 편향을 관찰하기만 하면 됩니다. 하지만 IBM 양자 컴퓨터는 하나의 축($z$)으로만 측정을 수행하도록 설계되어 있습니다. $x$ 방향으로 상태를 측정하려면 "기저 변환(change of basis)"을 수행해야 합니다. 이는 Bloch 구면에서 $x$축 방향의 상태를 $z$ 방향으로, 또 그 반대로 변환하는 연산을 수행하는 것을 의미합니다. 이를 구현하는 방법은 여러 가지가 있지만, 가장 많이 쓰이는 방법은 Hadamard Gate입니다:
$$
H=\frac{1}{\sqrt{2}}\begin{pmatrix}1 & 1 \\ 1 & -1\end{pmatrix}
$$

### 이해도 확인 {#check-your-understanding}
아래 질문을 읽고 답을 생각해본 뒤, 삼각형을 클릭하여 답을 확인하세요.

<details>
<summary>

$H|0\rangle = |+\rangle_x$ 이고 $H|+\rangle_x = |0\rangle$ 임을 보이세요.

</summary>

__답:__

$$
H|0\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix}1 & 1 \\ 1 & -1\end{pmatrix}\begin{pmatrix}1 \\ 0\end{pmatrix} = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ 1\end{pmatrix} = |+\rangle_x
$$
$$
H|+\rangle_x = \frac{1}{\sqrt(2)}\begin{pmatrix}1 & 1 \\ 1 & -1\end{pmatrix}\frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ 1\end{pmatrix} = \frac{1}{2}\begin{pmatrix}2 \\ 0\end{pmatrix} = \begin{pmatrix}1 \\ 0\end{pmatrix} = |0\rangle
$$

</details>

<details>
<summary>

$H|1\rangle = |-\rangle_x$ 이고 $H|-\rangle_x = |1\rangle$ 임을 보이세요.

</summary>

__답:__

$$
H|1\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix}1 & 1 \\ 1 & -1\end{pmatrix}\begin{pmatrix}0 \\ 1\end{pmatrix} = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ -1\end{pmatrix} = |-\rangle_x
$$
$$
H|-\rangle_x = \frac{1}{\sqrt(2)}\begin{pmatrix}1 & 1 \\ 1 & -1\end{pmatrix}\frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ -1\end{pmatrix} = \frac{1}{2}\begin{pmatrix}0 \\ 2\end{pmatrix} = \begin{pmatrix}0 \\ 1\end{pmatrix} = |1\rangle
$$

</details>

예상대로, 투영값이 $+x$ 방향이었다가 다시 $+x$로 측정되는 결과와, 처음에 $-x$ 방향이었다가 다시 $-x$로 측정되는 결과가 나타납니다. 투영값이 $+x$에서 $-x$로 또는 그 반대로 바뀌는 경우는 관측되지 않습니다. 이 관측 결과를 정리하여 행렬 형태의 추가 연산자와 상태 벡터를 개발하는 데 활용할 수 있습니다.

우리가 알고 있는 것:
- $+z$ 또는 $-z$ 방향으로 확실한 투영값을 가진 상태는 $+x$ 방향 투영값으로 측정될 확률이 50%, $-x$ 방향 투영값으로 측정될 확률이 50%입니다.
- $+x$ 방향으로 확실한 투영값을 가진 상태는 이후 $-x$ 방향 투영값으로 측정될 확률이 0이며, 그 반대도 마찬가지입니다.

이 결과를 활용해 $x$ 방향으로 확실히 양수인 투영값을 가진 상태($|+x\rangle$)와 확실히 음수인 투영값을 가진 상태($|-x\rangle$)를 구성할 수 있습니다. 이 상태들로부터 $S_z$에 대해 했던 것과 똑같은 방법으로 $S_x$에 해당하는 행렬을 구성할 수 있습니다. 이는 학습자의 연습 문제로 남겨둡니다. 마찬가지로, $y$축을 따라 측정하는 실험을 설계하고, $|+y\rangle$과 $|-y\rangle$ 벡터를 구한 뒤, 최종적으로 $S_y$의 식을 얻을 수 있습니다.

이 모든 벡터와 행렬을 모으면 다음과 같습니다.

$$
\begin{aligned}
|+x\rangle &= \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ 1\end{pmatrix} & \:  & |-x\rangle &=& \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ -1\end{pmatrix} & \: &S_x &=& \frac{\hbar}{2} \begin{pmatrix} 0 & 1 \\ 1 & 0\end{pmatrix}\\
|+y\rangle &= \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ i\end{pmatrix} & \:  & |-y\rangle &=& \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ -i\end{pmatrix} & \: &S_y &=& \frac{\hbar}{2} \begin{pmatrix} 0 & -i \\ i & 0\end{pmatrix}\\
|+z\rangle &= \begin{pmatrix}1 \\ 0\end{pmatrix} & \:  &|-z\rangle &=& \begin{pmatrix}0 \\ 1\end{pmatrix} & \: &S_z &=& \frac{\hbar}{2} \begin{pmatrix} 1 & 0 \\ 0 & 1\end{pmatrix}\\
\end{aligned}
$$

### 서술형 문제: {#free-response-problems}

1. 실험적 관찰과 제안된 표기법에 따라:

$$P_{\uparrow+x}=\vert \begin{pmatrix}1 & 0\end{pmatrix}\begin{pmatrix}a \\ b\end{pmatrix}\vert^2 = |a|^2 =\frac{1}{2} $$
$$P_{\downarrow+x}=\vert \begin{pmatrix}0 & 1\end{pmatrix}\begin{pmatrix}a \\ b\end{pmatrix}\vert^2 = |b|^2 =\frac{1}{2} $$

전체 위상까지

$$|+\rangle_x = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ e^{i\alpha}\end{pmatrix}$$

임을 보이시오. $\alpha = 0$으로 선택하여 표준 결과를 얻으시오.

$$P_{\downarrow-x}=\vert \begin{pmatrix}0 & 1\end{pmatrix}\begin{pmatrix}a \\ b\end{pmatrix}\vert^2 = |b|^2 =\frac{1}{2} $$

$$P_{+x-x}=\vert \frac{1}{\sqrt{2}}\begin{pmatrix}1 & 1\end{pmatrix}\begin{pmatrix}c \\ d\end{pmatrix}\vert^2 = \frac{1}{2}|c+d|^2 =0 $$

전체 위상까지

$$|+\rangle_x = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ -1\end{pmatrix}$$

임을 보이시오.
