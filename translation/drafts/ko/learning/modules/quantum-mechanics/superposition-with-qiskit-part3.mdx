## 양자의 본모습: 3차원 실험 {#the-quantum-revealed-an-experiment-in-three-dimensions}

사고 실험을 해봅시다. 동전을 공중에 던지고, 바닥에 떨어지게 두는 대신, 두 손 사이를 지나갈 때 박수를 쳐서 동전을 손바닥 사이에 끼웁니다. 이제 동전은 앞면이 위나 아래를 향하는 것이 아니라, 왼쪽이나 오른쪽을 향하게 됩니다.

__정답:__

확률은 여전히 50 대 50입니다. 동전 던지기 결과를 어느 방향으로 측정하든 확률에는 영향을 미치지 않습니다.

</details>
앞면이 왼쪽이나 오른쪽을 향할 확률이 여전히 50 대 50이라고 답했기를 바랍니다. 동전 던지기를 측정하는 방향은 결과의 확률에 영향을 미치지 않아야 합니다.

그런데 양자 동전에서는 어떻게 다를까요? 확인해봅시다.

Hadamard Gate를 적용했던 것과 동일한 방법으로 양자 중첩 상태를 만들 수 있습니다. 양자 동전에서 "앞면이 왼쪽이나 오른쪽을 향하는지" 측정하려면, 고전 동전에서 했던 것처럼 다른 축을 따라 측정하면 됩니다. 양자 컴퓨터에서의 표준 측정은 수직 축을 따라 이루어지는데, 이는 고전 동전의 일반적인 "앞면이 위 또는 아래를 향하는지" 측정과 같습니다. 하지만 양자 동전이 앞면이 왼쪽 또는 오른쪽을 향하는지, 즉 $x$축을 따라 $|+\rangle$ 또는 $|-\rangle$ 상태인지 물어볼 수도 있습니다. `Sampler`는 Z 측정 기저에서만 샘플링하지만, `Estimator`를 사용하면 X의 기댓값을 구할 수 있습니다. X의 값은 $|+\rangle$ 상태에서 +1, $|-\rangle$ 상태에서 -1입니다.

### 이해도 확인

아래 질문을 읽고 답을 생각한 뒤, 삼각형을 클릭하여 정답을 확인하세요.

<details>
<summary>

양자 동전이 이 경우 고전 동전처럼 동작한다면, $|+\rangle$와 $|-\rangle$ 상태를 측정할 확률이 50 대 50이 될 것입니다. 이 경우 `Estimator`가 반환할 X의 기댓값은 얼마일까요?

</summary>

__정답:__

X를 $|+\rangle$ 상태에 적용하면 +1 값을 얻고, $|-\rangle$ 상태에 적용하면 -1 값을 얻습니다. 따라서 50 대 50 분포라면 기댓값은 0이 됩니다.

</details>

```python
# Step 1: map problem

qcoin_lr = QuantumCircuit(1)
qcoin_lr.h(0)

obs = Pauli("X")

# Step 2: Transpile the circuit

pm = generate_preset_pass_manager(target=target, optimization_level=3)
qc_isa = pm.run(qcoin_lr)
obs_isa = obs.apply_layout(layout=qc_isa.layout)

# Step 3: Run the circuit on a real quantum computer

estimator = Estimator(mode=backend)
pubs = [(qc_isa, obs_isa)]
job = estimator.run([[qc_isa, obs_isa]])
res = job.result()

# Run the job on the Aer simulator with noise model from real backend
# job = estimator_sim.run([[qc_isa,obs_isa]])
# res=job.result()

# Step 4: Return the result in classical form, and analyze.

print(res[0].data.evs)
```

```text
0.9985207100591716
```

이 상태에서 X의 기댓값은 1입니다. 즉, $|+\rangle$와 $|-\rangle$를 측정할 확률이 50 대 50이 **아닙니다**.
