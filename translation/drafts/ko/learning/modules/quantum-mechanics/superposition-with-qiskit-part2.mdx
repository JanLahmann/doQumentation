위 Circuit을 1000번 샘플링하면, 약간의 통계적 변동을 제외하고 고전적인 동전 히스토그램과 거의 동일한 결과가 나타납니다.

양자 동전의 통계를 샘플링하는 것 외에도, `Estimator`라고 불리는 또 다른 Qiskit 프리미티브를 사용해 상태의 *기댓값(expectation value)*을 측정할 수 있습니다. 이 기댓값이 무엇인지 설명하기 위해 고전적인 동전을 예시로 들어 보겠습니다. 동전으로 도박을 한다고 상상해 보세요: 동전을 던져 "앞면"이 나오면 1달러를 얻고, "뒷면"이 나오면 1달러를 잃습니다. 한 번 던질 때 예상되는 금액(관측량 "돈"의 기댓값)을 계산하면 다음과 같습니다:

$$
E(money) = \frac{1}{2} (+ \text{1 dollar}) +  \frac{1}{2} (- \text{1 dollar}) = \text{0 dollars}
$$

1달러를 얻을 확률과 잃을 확률이 같으므로, 기댓값은 $0입니다.

마찬가지로, 양자 상태에서는 관측량 "Z"의 기댓값을 계산할 수 있습니다. 여기서 Z는 $|0\rangle$ 상태에 +1, $|1\rangle$ 상태에 -1을 대응시키는 파울리 행렬입니다.

$$
\langle \psi|Z|\psi \rangle = \frac{1}{2} \langle 0 | Z | 0 \rangle +  \frac{1}{2} \langle 1 | Z | 1 \rangle = \frac{1}{2} (+1) +  \frac{1}{2}(-1) = 0
$$

```python
from qiskit.quantum_info import Pauli

qcoin = QuantumCircuit(1)
qcoin.h(0)

# for Estimator, we do not apply the measurement to the circuit
```

```python
## Transpile
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

target = backend.target
pm = generate_preset_pass_manager(target=target, optimization_level=3)

obs = Pauli("Z")

qc_isa = pm.run(qcoin)
obs_isa = obs.apply_layout(layout=qc_isa.layout)
```

```python
## Execute

# On real hardware:
estimator = Estimator(mode=backend)
pubs = [(qc_isa, obs_isa)]
job = estimator.run([[qc_isa, obs_isa]])
res = job.result()

# On a simulator:
# job = estimator_sim.run([[qc_isa, obs_isa]])
# res=job.result()

print(res[0].data.evs)
```

```text
-0.014799284701239441
```

기댓값으로 0이 나왔습니다. 예상한 대로입니다(하하). 이는 0과 1을 측정할 확률이 동일하다는 것을 확인해 주는 또 다른 방법으로, 동전 던지기처럼 동작하는 것처럼 보입니다.

이 시점에서 "양자 동전"은 고전적인 동전과 완전히 동일해 보입니다. 하지만 다음 섹션에서는 두 가지의 근본적인 차이를 드러내는 실험을 진행합니다.
