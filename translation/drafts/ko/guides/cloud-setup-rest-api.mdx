# REST API로 IBM Quantum Platform 사용 설정하기 {#set-up-to-use-ibm-quantum-platform-with-rest-api}

REST API를 통해 양자 프로세서에 접근할 수 있으며, 이를 통해 어떤 프로그래밍 언어나 프레임워크에서도 QPU를 사용할 수 있습니다.

### 1. 접근 권한 얻기 {#1-get-access}

1. 사용자 계정이 없다면 [IBM Quantum 로그인 페이지](https://quantum.cloud.ibm.com/login)에서 계정을 만드세요.
1. [대시보드](https://quantum.cloud.ibm.com)에서 API 키(토큰이라고도 함)를 생성하세요. 동일한 API 키를 두 지역 모두에서 사용할 수 있습니다.
1. IBM Cloud Identity and Access Management (IAM) 베어러 토큰을 생성하세요. 이 토큰은 REST API 요청을 인증하는 데 사용되는 단기 토큰입니다. 생성하려면 다음 샘플 요청과 같이 [IAM Identity Services API](https://cloud.ibm.com/apidocs/iam-identity-token-api#create-api-key)를 호출하세요:
<Tabs>

   <TabItem value="Curl" label="Curl">
   ```bash
   curl -X POST 'https://iam.cloud.ibm.com/identity/token' \
   -H 'Content-Type: application/x-www-form-urlencoded' \
   -d 'grant_type=urn:ibm:params:oauth:grant-type:apikey&apikey=MY_APIKEY'
   ```
   예상 응답
   ```text
   {
      "access_token": "eyJhbGciOiJIUz......sgrKIi8hdFs",
      "refresh_token": "SPrXw5tBE3......KBQ+luWQVY=",
      "token_type": "Bearer",
      "expires_in": 3600,
      "expiration": 1473188353
   }
   ```
   </TabItem>

   <TabItem value="Python" label="Python">
   ```python
   # Use 'service' to invoke operations.
   import requests
   import json

   url = 'https://iam.cloud.ibm.com/identity/token'
   api_key = 'MY_APIKEY'
   headers = {
    'Content-Type': 'application/x-www-form-urlendcoded',
   }
   data = f'grant_type=urn:ibm:params:oauth:grant-type:apikey&apikey={api_key}'
   response = requests.post(url, headers=headers, data=data)

   # Bearer token to authorize requests to the REST API
   bearer_token = response.json()['access_token']
   ```
   </TabItem>
</Tabs>

### 2. 인증 방법 선택하기 {#2-choose-an-authentication-method}

작업 환경에 따라 적절한 인증 방법을 선택하세요:

* [API 키에 대한 환경 변수 만들기 (신뢰할 수 있는 Python 환경)](#environment_variable)
* [API 키 직접 사용 (신뢰할 수 없는 환경)](/guides/cloud-setup-untrusted#rest-untrusted)

<span id="environment_variable"></span>
#### 환경 변수 만들기 (신뢰할 수 있는 환경) {#create-an-environment-variable-trusted-environment}

1. 시스템에서 IQP_API_TOKEN 환경 변수를 설정하려면, 쉘 프로파일(예: .bashrc 또는 .zshrc)에 다음 줄을 추가하거나 터미널에서 직접 설정할 수 있습니다:

    ```shell
    export IQP_API_TOKEN=<your-API_KEY> # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard
    ```
    코드에서 환경 변수를 사용할 때는 다음 예시와 같이 `import os`를 포함하세요:

    ```python
    import os
    api_token = os.environ['IQP_API_TOKEN']
    ```
    환경 변수를 생성할 때 API 키는 여전히 로컬에 일반 텍스트로 저장되므로 안전하게 관리해야 합니다.

1. 요청의 헤더에 CRN과 베어러 토큰을 포함하여 Qiskit Runtime REST API에 대한 요청을 인증하세요.
```bash
curl -X 'GET' \
   'https://quantum.cloud.ibm.com/api/v1/usage' \
   '-H accept: application/json' \
   '-H authorization: Bearer <BEARER_TOKEN>' \
   '-H Service-CRN: <INSTANCE_CRN>'
```
