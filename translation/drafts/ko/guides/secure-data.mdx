# 데이터 보안 {#securing-your-data}

IBM Quantum&reg; Platform을 사용할 때 데이터를 안전하게 관리할 수 있도록, 어떤 데이터가 저장 및 암호화되는지, 그리고 저장된 데이터를 어떻게 삭제할 수 있는지 정확히 파악하는 것이 중요합니다.

## 민감한 데이터 보호 {#protecting-your-sensitive-data}

IBM Cloud&reg;에 저장하는 데이터는 무작위로 생성된 키를 사용하여 저장 시 암호화됩니다.

## 데이터 삭제 {#deleting-your-data}

서비스 인스턴스를 삭제하면 작업, 결과, 매개변수, 프로그램 등 해당 인스턴스와 관련된 모든 콘텐츠가 제거됩니다. 인스턴스를 삭제하려면 [인스턴스 페이지](https://quantum.cloud.ibm.com/instances)에서 제거하려는 인스턴스를 찾아 오버플로 메뉴를 클릭한 다음 **삭제**를 클릭하세요. 삭제 확인을 요청하는 메시지가 표시됩니다.

Manager 역할이 있는 경우 개별 작업 및 해당 데이터도 삭제할 수 있습니다. 액세스 정책, 그룹, 역할에 대한 자세한 내용은 [설명서](/guides/access-groups)를 참조하세요.

## 데이터 보존 정책 {#data-retention-policy}

IBM은 귀하가 서비스를 현재 사용하는 동안 (삭제되지 않은) 데이터를 3년간 보존합니다. 서비스 사용을 중단한 후 IBM이 데이터를 보존하는 기간에 대한 정보는 IBM Terms 사이트에서 제공되는 [Qiskit Runtime 데이터 시트](https://www.ibm.com/support/customer/csol/terms/?id=BB6EE1D06BDE11EBA199CCFE56FA6D9D)를 참조하세요.
