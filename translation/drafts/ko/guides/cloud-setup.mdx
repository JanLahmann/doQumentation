# IBM Cloud 계정 설정 {#set-up-your-ibm-cloud-account}

계정에 참여하라는 초대 이메일을 받으셨다면, 대신 [IBM Cloud 계정 설정 - 초대된 사용자](/guides/cloud-setup-invited)의 단계를 따르세요.

IBM Cloud&reg;에 계정을 설정하면 IBM&reg; 양자 처리 장치(QPU)에서 워크로드를 실행할 수 있습니다. 또한 [Qiskit 및 Qiskit Runtime을 설치하고 설정](/guides/install-qiskit#local)해야 합니다.

사용자 계정은 IBM Quantum 서비스에 대한 액세스를 제공하는 _클라우드 리소스 이름_(CRN)으로 식별되는 하나 이상의 [인스턴스](./instances)와 연결됩니다. 또한 각 계정에는 고유한 API 토큰(키라고도 함)이 할당되어 Qiskit에서 IBM Quantum에 접근할 수 있습니다. 특정 인스턴스를 선택하는 방법은 [인스턴스에 연결](./instances#connect-instance)을 참조하세요.

<span id="cloud"></span>
## IBM Cloud 사용 설정 {#set-up-to-use-ibm-cloud}

### 시작하기 전에 {#before-you-begin}

1. IBM Cloud 계정이 없다면 [IBM Cloud 계정](https://quantum.cloud.ibm.com/registration)을 만드세요.
   <Admonition type="note">
   여러 IBM Cloud 계정에 속할 수 있습니다. [IBM Quantum Platform](https://quantum.cloud.ibm.com) 인터페이스 헤더의 계정 전환기에서 언제든지 IBM Cloud 계정 중 하나에 액세스할 수 있습니다.
   </Admonition>
1. IBMid 또는 Google 계정으로 [IBM Quantum Platform](https://quantum.cloud.ibm.com)에 로그인하세요. 계정이 없는 경우 생성 과정이 안내됩니다. (이 로그인 자격 증명은 IBM Cloud 계정 자격 증명과 다릅니다.)
1. 다음 이미지와 같이 헤더의 계정 전환기에서 올바른 계정과 리전이 선택되어 있는지 확인하세요.

    리전은 작업이 실행되는 위치와 작업 데이터가 저장되는 위치를 제어합니다. 동일한 API 키를 사용하여 두 리전 모두에 액세스할 수 있지만, 현재 로그인한 리전에서 생성된 인스턴스만 확인하고 액세스할 수 있습니다.

    ![IBM Quantum Platform 헤더가 표시됩니다. 계정 전환기는 검색 창 오른쪽에 있습니다. 리전 전환기는 계정 전환기 오른쪽에 있습니다.](/docs/images/guides/cloud-setup/IQP-Header.svg 'IBM Quantum Platform 헤더')

### 1. 인스턴스 생성 {#1-create-an-instance}

IBM Quantum Platform [인스턴스](https://quantum.cloud.ibm.com/instances) 페이지에서 인스턴스를 생성할 수 있습니다. 자세한 지침은 [인스턴스 생성](/guides/instances#add-instance)을 참조하세요.

### 2. 선택 사항: 액세스 자격 증명 저장 {#2-optional-save-your-access-credentials}

개인 노트북과 같이 신뢰할 수 있는 Python 환경을 사용하는 경우, `save_account()` 메서드를 사용하여 [자격 증명을 로컬에 저장](/guides/save-credentials)하는 것을 권장합니다.

<span id="cloud-save"></span>
### 3. Qiskit을 서비스 인스턴스에 연결 {#3-connect-qiskit-with-your-service-instance}

- **신뢰할 수 있는 Python 환경**에서 [서비스 초기화](/guides/initialize-account)
- [**신뢰할 수 없는 환경**에서 서비스 초기화](/guides/cloud-setup-untrusted) (예: 공용 컴퓨터)

<Admonition type="note" title="참고 사항">
   * Qiskit 대신 REST API를 사용하여 연결하려면 [이 지침](/guides/cloud-setup-rest-api)을 따르세요.
   * 필요한 경우 [이 정보](/guides/quickstart-steps-org#firewall)를 사용하여 IBM Quantum API 엔드포인트에 대한 액세스를 허용하도록 방화벽을 구성하세요.
</Admonition>
