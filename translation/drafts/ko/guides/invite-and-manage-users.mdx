# 사용자 초대 및 액세스 관리 {#invite-users-and-manage-their-access}

<Admonition type="tip" title="시작하기 전에">
사용자를 초대하기 전에 계정의 인스턴스를 먼저 생성하세요. IBM Quantum Platform을 통해 생성하는 모든 인스턴스에 대해 연결된 [액세스 그룹](/guides/access-groups)이 자동으로 생성됩니다. 인스턴스에 사용자를 초대할 때, 초대 과정의 일부로 특정 인스턴스의 액세스 그룹에 사용자를 할당할 수 있습니다.
</Admonition>

<span id="invite"></span>
## 조직 계정에 사용자 초대하기 {#invite-users-to-your-organizations-account}

<Admonition note="tip" title="IBM Cloud 계정을 보유한 사용자">
사용자가 기존 IBM Cloud 계정을 보유하고 있는 경우, [IBM Cloud 사용자 관리](/guides/manage-cloud-users) 섹션의 안내를 따르세요.
</Admonition>

1. 먼저, 헤더의 계정 및 지역 전환기에서 올바른 계정과 지역이 선택되어 있는지 확인하세요.
1. 메인 메뉴에서 [*액세스 관리*](https://quantum.cloud.ibm.com/users)를 선택하세요.
1. 사용자 탭에서 "사용자 초대"를 클릭하세요.
1. 사용자의 이메일 주소를 입력하고, 선택적으로 초대된 사용자를 액세스 그룹에 할당하세요.
1. 사용자는 IBM Cloud 계정을 생성하고 인스턴스에 참여하라는 초대를 받게 됩니다. 초대를 수락하면 IBM Quantum Platform 페이지 상단의 계정 전환기에 새 계정이 표시됩니다.

## 사용자의 액세스 그룹 관리하기 {#manage-a-users-access-groups}

사용자의 액세스 그룹을 변경하려면, 메인 메뉴에서 [*액세스 관리*](https://quantum.cloud.ibm.com/users)로 이동하세요. 사용자 행 끝에 있는 오버플로 메뉴를 클릭하고 *액세스 관리*를 선택하세요. 열리는 모달에서 사용자가 속해야 할 액세스 그룹을 선택하세요.

## 사용자 제거하기 {#remove-a-user}

인스턴스에서 사용자를 제거하려면, 메인 메뉴에서 [*액세스 관리*](https://quantum.cloud.ibm.com/users)로 이동하세요. 사용자 행 끝에 있는 오버플로 메뉴를 클릭하고 *삭제*를 선택하세요. 삭제를 확인하라는 메시지가 표시됩니다.

<Admonition type="note">
사용자가 인스턴스에서 제거되면, 해당 사용자의 워크로드와 관련 결과물은 더 이상 해당 사용자에게 표시되지 않습니다. 동일한 인스턴스의 다른 사용자는 워크로드 데이터와 결과물에 계속 액세스할 수 있습니다.
</Admonition>
