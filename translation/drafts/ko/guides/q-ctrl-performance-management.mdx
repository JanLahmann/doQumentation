# Performance Management: Q-CTRL Fire Opalмқҳ Qiskit Function {#performance-management-a-qiskit-function-by-q-ctrl-fire-opal}

<Admonition type="note" title="м°ёкі ">
Qiskit FunctionsлҠ” IBM Quantum&reg; Premium Plan, Flex Plan, On-Prem(IBM Quantum Platform API кІҪмң ) Plan мӮ¬мҡ©мһҗм—җкІҢл§Ң м ңкіөлҗҳлҠ” мӢӨн—ҳм Ғ кё°лҠҘмһ…лӢҲлӢӨ. нҳ„мһ¬ лҜёлҰ¬ліҙкё° лҰҙлҰ¬мҠӨ мғҒнғңмқҙл©° ліҖкІҪлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

## к°ңмҡ” {#overview}

Fire Opal Performance ManagementлҘј мӮ¬мҡ©н•ҳл©ҙ м–‘мһҗ н•ҳл“ңмӣЁм–ҙ м „л¬ёк°Җк°Җ м•„лӢҲлҚ”лқјлҸ„ лҲ„кө¬лӮҳ лҢҖк·ңлӘЁ м–‘мһҗ м»ҙн“Ён„°м—җм„ң мқҳлҜё мһҲлҠ” кІ°кіјлҘј мүҪкІҢ м–»мқ„ мҲҳ мһҲмҠөлӢҲлӢӨ. Fire Opal Performance ManagementлЎң Circuitмқ„ мӢӨн–үн•ҳл©ҙ AI кё°л°ҳ мҳӨлҘҳ м–өм ң кё°мҲ мқҙ мһҗлҸҷмңјлЎң м Ғмҡ©лҗҳм–ҙ, лҚ” л§ҺмқҖ GateмҷҖ Qubitмқ„ мӮ¬мҡ©н•ҳлҠ” лҚ” нҒ° л¬ём ңлЎң нҷ•мһҘн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ л°©мӢқмқҖ м¶”к°Җ мҳӨлІ„н—Өл“ң м—Ҷмқҙ м •нҷ•н•ң лӢөмқ„ м–»лҠ” лҚ° н•„мҡ”н•ң мғ· мҲҳлҘј мӨ„м—¬мЈјм–ҙ, м»ҙн“ЁнҢ… мӢңк°„кіј л№„мҡ©мқ„ нҒ¬кІҢ м Ҳк°җн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

Performance ManagementлҠ” мҳӨлҘҳлҘј м–өм ңн•ҳм—¬ мһЎмқҢмқҙ л§ҺмқҖ н•ҳл“ңмӣЁм–ҙм—җм„ң м •нҷ•н•ң лӢөмқ„ м–»мқ„ нҷ•лҘ мқ„ лҶ’м—¬мӨҚлӢҲлӢӨ. мҰү, мӢ нҳё лҢҖ мһЎмқҢл№„лҘј н–ҘмғҒмӢңнӮөлӢҲлӢӨ. м•„лһҳ мқҙлҜём§ҖлҠ” Performance ManagementлЎң н–ҘмғҒлҗң м •нҷ•лҸ„к°Җ 10-Qubit м–‘мһҗ н‘ёлҰ¬м—җ ліҖнҷҳ(Quantum Fourier Transform) м•Ңкі лҰ¬мҰҳм—җм„ң м¶”к°Җ мғ·мқҳ н•„мҡ”м„ұмқ„ м–ҙл–»кІҢ мӨ„мқҙлҠ”м§Җ ліҙм—¬мӨҚлӢҲлӢӨ. Q-CTRLмқҖ лӢЁ 30мғ·мңјлЎң 99% мӢ лў° мһ„кі„к°’м—җ лҸ„лӢ¬н•ҳлҠ” л°ҳл©ҙ, кё°ліёк°’(`QiskitRuntime` Sampler, `optimization_level`=3, `resilience_level`=1, `ibm_sherbrooke`)мқҖ 170,000мғ·мқҙ н•„мҡ”н•©лӢҲлӢӨ. лҚ” л№ лҘҙкІҢ м •нҷ•н•ң лӢөмқ„ м–»мқҢмңјлЎңмҚЁ мғҒлӢ№н•ң м»ҙн“ЁнҢ… лҹ°нғҖмһ„мқ„ м Ҳм•Ҫн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

![н–ҘмғҒлҗң лҹ°нғҖмһ„ мӢңк°Ғнҷ”](/docs/images/guides/qctrl-performance-management/achieve_more.svg)

Performance Management н•ЁмҲҳлҠ” лӘЁл“  м•Ңкі лҰ¬мҰҳм—җ мӮ¬мҡ©н•  мҲҳ мһҲмңјл©°, н‘ңмӨҖ [Qiskit Runtime primitives](./primitives) лҢҖмӢ  мүҪкІҢ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. лӮҙл¶Җм ҒмңјлЎң м—¬лҹ¬ мҳӨлҘҳ м–өм ң кё°мҲ мқҙ н•Ёк»ҳ мһ‘лҸҷн•ҳм—¬ лҹ°нғҖмһ„м—җ мҳӨлҘҳк°Җ л°ңмғқн•ҳлҠ” кІғмқ„ л°©м§Җн•©лӢҲлӢӨ. лӘЁл“  Fire Opal нҢҢмқҙн”„лқјмқё л©”м„ңл“ңлҠ” мӮ¬м „ кө¬м„ұлҗҳм–ҙ мһҲмңјл©° м•Ңкі лҰ¬мҰҳм—җ кө¬м• л°ӣм§Җ м•ҠмңјлҜҖлЎң, лі„лҸ„мқҳ м„Өм • м—Ҷмқҙ н•ӯмғҒ мөңмғҒмқҳ м„ұлҠҘмқ„ л°ңнңҳн•©лӢҲлӢӨ.

Performance Managementм—җ лҢҖн•ң м•Ўм„ёмҠӨ к¶Ңн•ңмқ„ м–»мңјл Өл©ҙ [Q-CTRLм—җ л¬ёмқҳ](https://form.typeform.com/to/uOAVDnGg?typeform-source=q-ctrl.com)н•ҳм„ёмҡ”.
## м„ӨлӘ… {#description}

Fire Opal Performance ManagementлҠ” Qiskit Runtime primitivesмҷҖ мң мӮ¬н•ң л‘җ к°Җм§Җ мӢӨн–ү мҳөм…ҳмқ„ м ңкіөн•ҳлҜҖлЎң, Q-CTRL SamplerмҷҖ EstimatorлЎң мҶҗмүҪкІҢ көҗмІҙн•ҳм—¬ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. Performance Management н•ЁмҲҳ мӮ¬мҡ©мқҳ мқјл°ҳм Ғмқё мӣҢнҒ¬н”ҢлЎңлҠ” лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ:
1. Circuitмқ„ м •мқҳн•©лӢҲлӢӨ(Estimatorмқҳ кІҪмҡ° м—°мӮ°мһҗлҸ„ н•Ёк»ҳ м •мқҳ).
2. Circuitмқ„ мӢӨн–үн•©лӢҲлӢӨ.
3. кІ°кіјлҘј мЎ°нҡҢн•©лӢҲлӢӨ.

н•ҳл“ңмӣЁм–ҙ мһЎмқҢмқ„ мӨ„мқҙкё° мң„н•ҙ Fire OpalмқҖ лӢӨмқҢ мқҙлҜём§Җм—җ лӮҳнғҖлӮң лӢӨм–‘н•ң AI кё°л°ҳ мҳӨлҘҳ м–өм ң кё°мҲ мқ„ нҷңмҡ©н•©лӢҲлӢӨ. Fire Opalмқ„ мӮ¬мҡ©н•ҳл©ҙ м „мІҙ нҢҢмқҙн”„лқјмқёмқҙ мҷ„м „нһҲ мһҗлҸҷнҷ”лҗҳм–ҙ лі„лҸ„мқҳ кө¬м„ұмқҙ н•„мҡ” м—ҶмҠөлӢҲлӢӨ.

Fire Opalмқҳ нҢҢмқҙн”„лқјмқёмқҖ м–‘мһҗ лҹ°нғҖмһ„ мҰқк°ҖлӮҳ м¶”к°Җ л¬јлҰ¬ Qubitкіј к°ҷмқҖ мҳӨлІ„н—Өл“ңмқҳ н•„мҡ”м„ұмқ„ м—Ҷм• мӨҚлӢҲлӢӨ. лӢЁ, нҒҙлһҳмӢқ мІҳлҰ¬ мӢңк°„мқҖ м—¬м „нһҲ кі л Өн•ҙм•ј н•©лӢҲлӢӨ([лІӨм№ҳл§ҲнҒ¬](#benchmarks) м„№м…ҳм—җм„ң м¶”м •м№ҳлҘј м°ёкі н•ҳм„ёмҡ”. "мҙқ мӢңк°„"мқҖ нҒҙлһҳмӢқ л°Ҹ м–‘мһҗ мІҳлҰ¬ мӢңк°„ лӘЁл‘җлҘј л°ҳмҳҒн•©лӢҲлӢӨ). мғҳн”Ңл§Ғ нҳ•нғңмқҳ мҳӨлІ„н—Өл“ңк°Җ н•„мҡ”н•ң мҳӨлҘҳ мҷ„нҷ”(error mitigation)мҷҖ лӢ¬лҰ¬, Fire Opalмқҳ мҳӨлҘҳ м–өм ңлҠ” Gate л°Ҹ нҺ„мҠӨ мҲҳмӨҖ лӘЁл‘җм—җм„ң мһ‘лҸҷн•ҳм—¬ лӢӨм–‘н•ң мһЎмқҢ мӣҗмқёмқ„ н•ҙкІ°н•ҳкі  мҳӨлҘҳ л°ңмғқ к°ҖлҠҘм„ұмқ„ л°©м§Җн•©лӢҲлӢӨ. мҳӨлҘҳлҘј мӮ¬м „м—җ л°©м§Җн•ЁмңјлЎңмҚЁ л№„мҡ©мқҙ нҒ° мӮ¬нӣ„ мІҳлҰ¬мқҳ н•„мҡ”м„ұмқ„ м—Ҷм• мӨҚлӢҲлӢӨ.

м•„лһҳ мқҙлҜём§ҖлҠ” Fire Opal Performance Managementк°Җ мһҗлҸҷнҷ”н•ҳлҠ” мҳӨлҘҳ м–өм ң л°©лІ•мқ„ ліҙм—¬мӨҚлӢҲлӢӨ.

![мҳӨлҘҳ м–өм ң нҢҢмқҙн”„лқјмқё мӢңк°Ғнҷ”](/docs/images/guides/qctrl-performance-management/error_suppression.svg)

мқҙ н•ЁмҲҳлҠ” SamplerмҷҖ Estimator л‘җ к°Җм§Җ primitiveлҘј м ңкіөн•ҳл©°, л‘җ primitiveмқҳ мһ…л Ҙ л°Ҹ м¶ңл ҘмқҖ лӘЁл‘җ [Qiskit Runtime V2 primitives](/guides/primitive-input-output)мқҳ кө¬нҳ„ мӮ¬м–‘мқ„ нҷ•мһҘн•©лӢҲлӢӨ.
## лІӨм№ҳл§ҲнҒ¬ {#benchmarks}

[л°ңн‘ңлҗң м•Ңкі лҰ¬мҰҳ лІӨм№ҳл§ҲнӮ№](https://journals.aps.org/prapplied/abstract/10.1103/PhysRevApplied.20.024034) кІ°кіјлҠ” Bernstein-Vazirani, м–‘мһҗ н‘ёлҰ¬м—җ ліҖнҷҳ(Quantum Fourier Transform), Grover нғҗмғү, м–‘мһҗ к·јмӮ¬ мөңм Ғнҷ” м•Ңкі лҰ¬мҰҳ(QAOA), ліҖл¶„ м–‘мһҗ кі мң к°’ кі„мӮ°(VQE) л“ұ лӢӨм–‘н•ң м•Ңкі лҰ¬мҰҳм—җм„ң м„ұлҠҘмқҙ нҒ¬кІҢ н–ҘмғҒлҗҳм—ҲмқҢмқ„ ліҙм—¬мӨҚлӢҲлӢӨ. мқҙ м„№м…ҳмқҳ лӮҳлЁём§Җ л¶Җл¶„м—җм„ңлҠ” мӢӨн–ү к°ҖлҠҘн•ң м•Ңкі лҰ¬мҰҳ мң нҳ•кіј мҳҲмғҒ м„ұлҠҘ л°Ҹ лҹ°нғҖмһ„м—җ лҢҖн•ң мһҗм„ён•ң м •ліҙлҘј м ңкіөн•©лӢҲлӢӨ.

лӢӨмқҢмқҳ лҸ…лҰҪм Ғмқё м—°кө¬л“ӨмқҖ Q-CTRLмқҳ Performance Managementк°Җ кё°лЎқм Ғмқё к·ңлӘЁмқҳ м•Ңкі лҰ¬мҰҳ м—°кө¬лҘј к°ҖлҠҘн•ҳкІҢ н•ңлӢӨлҠ” кІғмқ„ ліҙм—¬мӨҚлӢҲлӢӨ:
- [Parametrized Energy-Efficient Quantum Kernels for Network Service Fault Diagnosis](https://arxiv.org/abs/2405.09724v1) - мөңлҢҖ 50-Qubit м–‘мһҗ м»Өл„җ н•ҷмҠө
- [Tensor-based quantum phase difference estimation for large-scale demonstration](https://arxiv.org/abs/2408.04946) - мөңлҢҖ 33-Qubit м–‘мһҗ мң„мғҒ м¶”м •
- [Hierarchical Learning for Quantum ML: Novel Training Technique for Large-Scale Variational Quantum Circuits](https://arxiv.org/abs/2311.12929) - мөңлҢҖ 21-Qubit м–‘мһҗ лҚ°мқҙн„° лЎңл”©

лӢӨмқҢ н‘ңлҠ” `ibm_fez`м—җм„ң мҲҳн–үлҗң мқҙм „ лІӨм№ҳл§ҲнӮ№ мӢӨн–үмқҳ м •нҷ•лҸ„ л°Ҹ лҹ°нғҖмһ„м—җ лҢҖн•ң лҢҖлһөм Ғмқё к°Җмқҙл“ңлҘј м ңкіөн•©лӢҲлӢӨ. лӢӨлҘё л””л°”мқҙмҠӨм—җм„ңмқҳ м„ұлҠҘмқҖ лӢӨлҘј мҲҳ мһҲмҠөлӢҲлӢӨ. мӮ¬мҡ© мӢңк°„мқҖ CircuitлӢ№ 10,000мғ·мқ„ кё°мӨҖмңјлЎң н•©лӢҲлӢӨ. "Qubit мҲҳ"лҠ” м—„кІ©н•ң м ңн•ңмқҙ м•„лӢҲлқј к·№лҸ„лЎң мқјкҙҖлҗң мҶ”лЈЁм…ҳ м •нҷ•лҸ„лҘј кё°лҢҖн•  мҲҳ мһҲлҠ” лҢҖлһөм Ғмқё мһ„кі„к°’мқ„ лӮҳнғҖлғ…лӢҲлӢӨ. лҚ” нҒ° л¬ём ң к·ңлӘЁлҸ„ м„ұкіөм ҒмңјлЎң н•ҙкІ°лҗң л°” мһҲмңјл©°, мқҙлҹ¬н•ң н•ңкі„лҘј л„ҳм–ҙ н…ҢмҠӨнҠён•ҙ ліҙлҠ” кІғмқ„ к¶ҢмһҘн•©лӢҲлӢӨ.

кё°лҢ“к°’ мёЎм •мқҳ м •нҷ•лҸ„ м •мқҳ - мёЎм • кё°мӨҖ $A$лҠ” лӢӨмқҢкіј к°ҷмқҙ м •мқҳлҗ©лӢҲлӢӨ:
$$
A = 1 - \frac{|\epsilon^{ideal} - \epsilon^{meas}|}{\epsilon^{ideal}_{max} - \epsilon^{ideal}_{min}},
$$
м—¬кё°м„ң $ \epsilon^{ideal} $ = мқҙмғҒм Ғмқё кё°лҢ“к°’, $ \epsilon^{meas} $ = мёЎм •лҗң кё°лҢ“к°’, $\epsilon^{ideal}_{max} $ = мқҙмғҒм Ғмқё мөңлҢҖк°’, $\epsilon^{ideal}_{min}$ = мқҙмғҒм Ғмқё мөңмҶҢк°’мһ…лӢҲлӢӨ. $A_{mean}$мқҖ м—¬лҹ¬ мёЎм •м—җ кұём№ң $A$ к°’мқҳ нҸүк· мһ…лӢҲлӢӨ.

мқҙ мёЎм • кё°мӨҖмқҖ лӢ¬м„ұ к°ҖлҠҘн•ң к°’мқҳ лІ”мң„м—җм„ң м „м—ӯм Ғ мқҙлҸҷ л°Ҹ мҠӨмјҖмқјл§Ғм—җ л¶ҲліҖмқҙкё° л•Ңл¬ём—җ мӮ¬мҡ©лҗ©лӢҲлӢӨ. мҰү, к°ҖлҠҘн•ң кё°лҢ“к°’мқҳ лІ”мң„лҘј лҶ’мқҙкұ°лӮҳ лӮ®м¶”кұ°лӮҳ л¶„мӮ°мқ„ лҠҳлҰ¬лҚ”лқјлҸ„ $A$мқҳ к°’мқҖ мқјкҙҖлҗҳкІҢ мң м§Җлҗ©лӢҲлӢӨ.

**1. Circuit мғқм„ұ**

мҳҲмӢңлЎң, Performance Management н•ЁмҲҳм—җ мһ…л Ҙн•  л¬ҙмһ‘мң„ м—җлҘҙлҜёнҠё м—°мӮ°мһҗмҷҖ кҙҖмёЎлҹүмқ„ мғқм„ұн•©лӢҲлӢӨ.

```python
import numpy as np

from qiskit.quantum_info import random_hermitian, SparsePauliOp

# Generate a random circuit

circuit = iqp(mat)
circuit.measure_all()

# Define observables as a string
observable = SparsePauliOp("Z" * n_qubits)
```

```python
# Create PUB tuple
estimator_pubs = [(circuit, observable)]
```

### Estimator мһ…л Ҙ {#estimator-inputs}
<Admonition type="caution">Fire Opal Performance ManagementлҠ” м¶”мғҒ Circuitмқ„ н—Ҳмҡ©н•©лӢҲлӢӨ. мқҙлҠ” лҢҖмғҒ Backendмқҳ лӘ…л №м–ҙ м§‘н•© м•„нӮӨн…ҚмІҳ(ISA)лЎң мһ‘м„ұлҗң Circuitл§Ң н—Ҳмҡ©н•ҳлҠ” кё°ліё Qiskit Runtime primitivesмҷҖ лӢӨлҰ…лӢҲлӢӨ. мөңмғҒмқҳ кІ°кіјлҘј мң„н•ҙ Performance Management н•ЁмҲҳлҘј нҶөн•ҙ м ңм¶ңн•ҳкё° м „м—җ Circuitмқ„ TranspilerлЎң ліҖнҷҳн•ҳм§Җ л§Ҳм„ёмҡ”. </Admonition>

| мқҙлҰ„  | мң нҳ•  | м„ӨлӘ… | н•„мҲҳ м—¬л¶Җ | кё°ліёк°’  | мҳҲмӢң |
|------|------|----------|----------|-----------|-----------|
| pubs       | `QctrlEstimatorPubLike` лҳҗлҠ” `list[QctrlEstimatorPubLike]` | `EstimatorPubLike` кө¬м„ұ мҡ”мҶҢм—җ лӮҳм—ҙлҗң мһ…л Ҙмқ„ нҸ¬н•Ён•ҳлҠ” н•ҳлӮҳ мқҙмғҒмқҳ нҠңн”Ң   | мҳҲ   | н•ҙлӢ№ м—ҶмқҢ   | `(circuit, observables, parameter_values)`   |
| backend_name| `str`  | Backendмқҳ мқҙлҰ„ | м•„лӢҲмҡ”  |  мқёмҠӨн„ҙмҠӨк°Җ м•Ўм„ёмҠӨн•  мҲҳ мһҲлҠ” к°ҖмһҘ лҚң л°”мҒң Backend | `"ibm_fez"` |

**`QctrlEstimatorPubLike` кө¬м„ұ мҡ”мҶҢ([Qiskit Runtime PUB м •мқҳ](/guides/primitive-input-output#estimator-pub)м—җм„ң нҢҢмғқ):**
- `QuantumCircuit` лҳҗлҠ” OpenQASM 2.0 лҳҗлҠ” 3.0 л¬ёмһҗм—ҙ нҳ•мӢқмңјлЎң м •мқҳлҗң лӢЁмқј Circuit.
- "м§ҖмӣҗлҗҳлҠ” кҙҖмёЎлҹү нҳ•мӢқ" лӘ©лЎқм—җ лӘ…мӢңлҗң нҳ•мӢқ мӨ‘ н•ҳлӮҳлЎң, м¶”м •н•  кё°лҢ“к°’мқ„ м§Җм •н•ҳлҠ” н•ҳлӮҳ мқҙмғҒмқҳ кҙҖмёЎлҹү.
- (м„ нғқ мӮ¬н•ӯ) `QiskitRuntime` primitivesмҷҖ лҸҷмқјн•ң [л°°м—ҙ лёҢлЎңл“ңмәҗмҠӨнҢ… к·ңм№ҷ](./primitive-input-output#broadcasting-rules)мқ„ л”°лҘҙлҠ”, Circuitм—җ л°”мқёл”©н•  нҢҢлқјлҜён„° к°’мқҳ лӘЁмқҢ.
- (м„ нғқ мӮ¬н•ӯ) м¶”м •н•  кё°лҢ“к°’мқҳ лӘ©н‘ң м •л°ҖлҸ„.
- (м„ нғқ мӮ¬н•ӯ) м •л°ҖлҸ„лҘј лӮҳнғҖлӮҙлҠ” мӢӨмҲҳ, лҳҗлҠ” мғ· мҲҳлҘј нҸ¬н•Ён•ҳлҠ” мӢӨн–ү мҳөм…ҳ л”•м…”л„ҲлҰ¬. мҳҲ: `{"shots": <int>}`.

**м§ҖмӣҗлҗҳлҠ” кҙҖмёЎлҹү нҳ•мӢқ:**
- `Pauli`, `SparsePauliOp`, `PauliList`, `str` л“ұ `ObservablesArrayLike` нҳ•мӢқ мӨ‘ н•ҳлӮҳ
- Pauli л¬ёмһҗм—ҙ: `"XY"`
- л”•м…”л„ҲлҰ¬ - кі„мҲҳк°Җ мһҲлҠ” Pauli л¬ёмһҗм—ҙ: `{"XY": 0.5, "YZ": 0.3}`
- Pauli л¬ёмһҗм—ҙ лӘ©лЎқ: `["XY", "YZ", "ZX"]`
- кі„мҲҳк°Җ мһҲлҠ” Pauli л¬ёмһҗм—ҙ лӘ©лЎқ: `[("XY", 0.5), ("YZ", 0.3)]`
- мӨ‘мІ©лҗң Pauli л¬ёмһҗм—ҙ лӘ©лЎқ: `[["XY", "YZ"], ["ZX", "XX"]]`
- кі„мҲҳк°Җ мһҲлҠ” мӨ‘мІ©лҗң Pauli л¬ёмһҗм—ҙ лӘ©лЎқ: `[[("XY", 0.1), ("YZ", 0.2)], [("ZX", 0.3), ("XX", 0.4)]]`

**м§ҖмӣҗлҗҳлҠ” Backend:**

## Sampler primitive {#sampler-primitive}
### Sampler мҳҲм ң {#sampler-example}

Fire Opal Performance Managementмқҳ Sampler primitiveлҘј мӮ¬мҡ©н•ҳм—¬ BernsteinвҖ“Vazirani Circuitмқ„ мӢӨн–үн•©лӢҲлӢӨ. мқҙ м•Ңкі лҰ¬мҰҳмқҖ лё”лһҷл°•мҠӨ н•ЁмҲҳмқҳ м¶ңл ҘмңјлЎңл¶Җн„° мҲЁкІЁм§„ л¬ёмһҗм—ҙмқ„ м°ҫлҠ” лҚ° мӮ¬мҡ©лҗҳл©°, м •лӢөмқҙ н•ҳлӮҳлҝҗмқҙкё° л•Ңл¬ём—җ мқјл°ҳм Ғмқё лІӨм№ҳл§ҲнӮ№ м•Ңкі лҰ¬мҰҳмңјлЎң нҷңмҡ©лҗ©лӢҲлӢӨ.

**1. Circuit мғқм„ұ**

м•Ңкі лҰ¬мҰҳмқҳ м •лӢөмқё мҲЁкІЁм§„ л№„нҠёмҠӨнҠёл§Ғкіј BernsteinвҖ“Vazirani Circuitмқ„ м •мқҳн•©лӢҲлӢӨ. `circuit_width`лҘј ліҖкІҪн•ҳлҠ” кІғл§ҢмңјлЎң Circuitмқҳ л„Ҳл№„лҘј мЎ°м •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

```python
import qiskit

circuit_width = 35
hidden_bitstring = "1" * circuit_width

# Create circuit, reserving one qubit for BV oracle
bv_circuit = qiskit.QuantumCircuit(circuit_width + 1, circuit_width)
bv_circuit.x(circuit_width)
bv_circuit.h(range(circuit_width + 1))
for input_qubit, bit in enumerate(reversed(hidden_bitstring)):
    if bit == "1":
        bv_circuit.cx(input_qubit, circuit_width)
bv_circuit.barrier()
bv_circuit.h(range(circuit_width + 1))
bv_circuit.barrier()
for input_qubit in range(circuit_width):
    bv_circuit.measure(input_qubit, input_qubit)

# Create PUB tuple
sampler_pubs = [(bv_circuit,)]
```

**2. Circuit мӢӨн–ү**

Circuitмқ„ мӢӨн–үн•ҳкі , м„ нғқм ҒмңјлЎң BackendмҷҖ мғ· мҲҳлҘј м •мқҳн•©лӢҲлӢӨ.

```python

    pubs=sampler_pubs,
    backend_name=backend_name,
)
```

**3. мғҒмң„ л№„нҠёмҠӨнҠёл§Ғ н”ҢлЎҜ**

```python
import matplotlib.pyplot as plt

def plot_top_bitstrings(counts_dict, hidden_bitstring=None):
    # Sort and take the top 100 bitstrings
    top_100 = sorted(counts_dict.items(), key=lambda x: x[1], reverse=True)[
        :100
    ]
    if not top_100:
        print("No bitstrings found in the input dictionary.")
        return

    # Unzip the bitstrings and their counts
    bitstrings, counts = zip(*top_100)

    # Assign colors: purple if the bitstring matches hidden_bitstring, otherwise gray
    colors = [
        "#680CE9" if bit == hidden_bitstring else "gray" for bit in bitstrings
    ]

    # Create the bar plot
    plt.figure(figsize=(15, 8))
    plt.bar(
        range(len(bitstrings)), counts, tick_label=bitstrings, color=colors
    )

    # Rotate the bitstrings for better readability
    plt.xticks(rotation=90, fontsize=8)
    plt.xlabel("Bitstrings")
    plt.ylabel("Counts")
    plt.title("Top 100 Bitstrings by Counts")

    # Show the plot
    plt.tight_layout()
    plt.show()
```
