## qiskit.converters {#qiskitconverters}

Funkce `qiskit.converters.ast_to_dag` byla z Qiskitu odstraněna. Převáděla
abstraktní syntaktický strom generovaný starým parserem OpenQASM 2 na
[`DAGCircuit`](https://docs.quantum.ibm.com/api/qiskit/qiskit.dagcircuit.DAGCircuit). Protože starý
parser OpenQASM 2 byl odstraněn (viz [qiskit.qasm](#qiskit.qasm)), tato
funkce již nemá opodstatnění. Místo ní načti soubory OpenQASM 2 do
`QuantumCircuit` pomocí konstruktorových metod
[`QuantumCircuit.from_qasm_file`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#from_qasm_file)
nebo
[`QuantumCircuit.from_qasm_str`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#from_qasm_str)
(nebo modulu [`qiskit.qasm2`](https://docs.quantum.ibm.com/api/qiskit/qasm2)) a poté převeď
výsledný `QuantumCircuit` na
[`DAGCircuit`](https://docs.quantum.ibm.com/api/qiskit/qiskit.dagcircuit.DAGCircuit) pomocí
[`circuit_to_dag`](https://docs.quantum.ibm.com/api/qiskit/converters#circuit_to_dag).

```python
# Previous
from qiskit.converters import ast_to_dag
from qiskit.qasm import Qasm

dag = ast_to_dag(Qasm(filename="myfile.qasm").parse())

# Current alternative
import qiskit.qasm2
from qiskit.converters import circuit_to_dag

dag = circuit_to_dag(qiskit.qasm2.load("myfile.qasm"))
```

## qiskit.extensions {#qiskitextensions}

Modul `qiskit.extensions` již není dostupný. Většina jeho objektů byla
začleněna do knihovny Circuit
([`qiskit.circuit.library`](https://docs.quantum.ibm.com/api/qiskit/circuit_library)). Při migraci
na nové umístění jednoduše nahraď `qiskit.extensions` za `qiskit.circuit.library`
v cestě importu objektu. Jde o náhradu beze změny rozhraní.

```python
# Previous
from qiskit.extensions import DiagonalGate

# Current alternative
from qiskit.circuit.library import DiagonalGate
```

Třídy přesunuté do [`qiskit.circuit.library`](https://docs.quantum.ibm.com/api/qiskit/circuit_library) jsou:

* [`DiagonalGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.DiagonalGate)
* [`HamiltonianGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.HamiltonianGate)
* [`Initialize`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.Initialize)
* [`Isometry`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.Isometry)
* `qiskit.circuit.library.generalized_gates.mcg_up_diag.MCGupDiag`
* [`UCGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.UCGate)
* [`UCPauliRotGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.UCPauliRotGate)
* [`UCRXGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.UCRXGate)
* [`UCRYGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.UCRYGate)
* [`UCRZGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.UCRZGate)
* [`UnitaryGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.UnitaryGate)

Následující třídy byly z kódu odstraněny, protože jejich funkce byly buď
nadbytečné, nebo svázané s modulem `extensions`:

Odstraněno                       |   Alternativa
---------------------------------|--------------------------------------------
`SingleQubitUnitary`    |  [`qiskit.circuit.library.UnitaryGate`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.UnitaryGate)
`Snapshot`             |  Použij [instrukce pro ukládání](https://qiskit.org/ecosystem/aer/apidocs/circuit.html#saving-simulator-data) z `qiskit-aer`
`ExtensionError` | Příslušná třída chyby

## qiskit.primitives {#qiskitprimitives}

Nejvýznamnější změnou v modulu [`qiskit.primitives`](https://docs.quantum.ibm.com/api/qiskit/primitives) je
zavedení nového rozhraní **primitives V2**. Tato sekce ukazuje, jak migrovat svůj
pracovní postup z primitives V1 na primitives V2, a také popisuje dílčí změny
ve vstupech přijímaných rozhraním V1.

<Admonition type="note">
Od verze 1.0 budeme označovat rozhraní primitives z doby před verzí 1.0 jako „primitives V1".
</Admonition>
