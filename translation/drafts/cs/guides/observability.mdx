# Události sledování aktivity pro Qiskit Runtime {#activity-tracking-events-for-qiskit-runtime}

Služby IBM Cloud&reg;, jako je Qiskit Runtime, generují události sledování aktivity.

Události sledování aktivity zaznamenávají činnosti, které mění stav služby v IBM Cloud. Události můžeš využít k vyšetřování neobvyklé aktivity a kritických akcí a ke splnění požadavků na regulatorní audit.

Pomocí služby IBM Cloud Activity Tracker Event Routing, což je platformová služba, můžeš směrovat auditní události ve svém účtu do cílů dle svého výběru konfigurací cílů a tras, které určují, kam se události sledování aktivity odesílají. Více informací najdeš v části [O směrování událostí IBM Cloud Activity Tracker.](https://cloud.ibm.com/docs/atracker?topic=atracker-about)

Pomocí IBM Cloud Logs můžeš vizualizovat události generované ve svém účtu a směrované službou IBM Cloud Activity Tracker Event Routing do instance IBM Cloud Logs a nastavovat na ně upozornění.

## Umístění, kde se generují události sledování aktivity {#umístění-kde-se-generují-události-sledování-aktivity}

Qiskit Runtime odesílá události sledování aktivity v následujících regionech:

- Washington ("us-east")
- Frankfurt ("eu-de")

## Zobrazení událostí sledování aktivity pro Qiskit Runtime {#zobrazení-událostí-sledování-aktivity-pro-qiskit-runtime}

Pomocí IBM Cloud Logs můžeš vizualizovat události generované ve svém účtu a směrované službou IBM Cloud Activity Tracker Event Routing do instance IBM Cloud Logs a nastavovat na ně upozornění.
