# Začínáme s diagonalizací kvantových vzorků (SQD) {#getting-started-with-sample-based-quantum-diagonalization-sqd}
<details>
<summary><b>Verze balíčků</b></summary>

Kód na této stránce byl vyvinut s použitím níže uvedených požadavků.
Doporučujeme používat tyto verze nebo novější.

```
qiskit-addon-sqd~=0.10.0
pyscf~=2.8.0; sys.platform != 'win32'
```
</details>
<Admonition type="caution" title="Poznámka pro uživatele Windows">

Tento příklad využívá balíček `pyscf`, knihovnu pro simulaci chemie v Pythonu, která není nativně podporována na Windows; níže uvedené příklady proto nebudou fungovat.

Aby uživatelé Windows mohli spouštět níže uvedené ukázky kódu, musí si vytvořit prostředí Python pomocí Windows Subsystem for Linux, jak je doporučeno v [průvodci instalací `pyscf`.](https://pyscf.org/user/install.html)

</Admonition>

Tento průvodce ukazuje jednoduchý funkční příklad, který ti pomůže začít s balíčkem `qiskit-addon-sqd`. V tomto příkladu použiješ SQD k získání aproximace základního stavu molekuly $N_2$ v rovnovážné poloze.

Pro začátek si připomeň, že obecný Hamiltonián interagujících elektronů má tvar:

$$ \hat{H} = \sum_{\substack{pr\\ \sigma}}h_{pr} \hat{a}_{p\sigma}^\dagger \hat{a}_{r\sigma} + \sum_{\substack{prqs\\ \sigma\tau}} \frac{1}{2}\left(pr|qs\right)\hat{a}_{p\sigma}^\dagger \hat{a}_{q\tau}^\dagger \hat{a}_{s\tau}\hat{a}_{r\sigma} $$

kde $\hat{a}_{p\sigma}^\dagger$ a $\hat{a}_{p\sigma}$ jsou fermionské operátory tvorby a zániku asociované s $p$-tým orbitalem se spinem $\sigma$ a $h_{pr}$ a $\left(pr|qs\right)$ jsou jednoelektronové a dvouelectronové elektronové integrály.
## Příprava informací o molekule {#prepare-molecule-information}

Pro začátek zadej molekulu a její vlastnosti pomocí `pyscf` a elektronových integrálů uložených v souboru `n2_fci.txt`.
<Admonition type="note">
Tento příklad načítá předem vygenerovaná data ze souboru `n2_fci.txt`, který obsahuje jednoelectronové a dvouelectronové elektronové integrály. Chceš-li spustit níže uvedené buňky kódu na místním počítači, zkopíruj data do souboru se stejným názvem. Tento příklad také vyžaduje nainstalovaný balíček `pyscf`.
</Admonition>
