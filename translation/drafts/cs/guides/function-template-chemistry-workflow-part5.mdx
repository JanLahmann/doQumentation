
Zkontroluj podrobný stav úlohy:

```python
import time

t0 = time.time()
status = job.status()
if status == "QUEUED":
    print(f"time = {time.time()-t0:.2f}, status = QUEUED")
while True:
    status = job.status()
    if status == "QUEUED":
        continue
    print(f"time = {time.time()-t0:.2f}, status = {status}")
    if status == "DONE" or status == "ERROR":
        break
```

```text
time = 2.35, status = DONE
```

Zatímco úloha běží, můžeš načítat logy vytvořené z výstupů `logger.info`. Ty mohou poskytovat užitečné informace o průběhu pracovního postupu SQD IEF-PCM. Například stejná spinová orbitální propojení nebo hloubka dvou-qubitových bran finálního ISA Circuit určeného ke spuštění na hardwaru.

```python
print(job.logs())
```

Volání pro získání výsledku úlohy blokuje zbytek programu, dokud výsledek není dostupný. Jakmile je úloha hotová, můžeš výsledky načíst. Zahrnují volnou energii solvace, jakož i informace o dávce s nejnižší energií, hodnotě nejnižší energie a další užitečné informace, například celkovou dobu běhu řešiče.
