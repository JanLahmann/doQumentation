### Autentizace {#authentication}

Pomocí `qiskit-ibm-catalog` se autentizuj v `QiskitServerless` se svým API klíčem (tokenem), který najdeš na dashboardu platformy [IBM Quantum Platform](https://quantum.cloud.ibm.com). To ti umožní vytvořit instanci serverless klienta pro nahrání nebo spuštění vybrané funkce:

```python
from qiskit_ibm_catalog import QiskitServerless

serverless = QiskitServerless(
    channel="ibm_quantum_platform",
    instance="INSTANCE_CRN",
    token="YOUR_API_KEY" # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard
)
```

Volitelně můžeš použít `save_account()` k uložení svých přihlašovacích údajů v lokálním prostředí (viz průvodce [Nastavení účtu IBM Cloud](/guides/cloud-setup#cloud-save)). Tato funkce zapisuje tvé přihlašovací údaje do stejného souboru jako [`QiskitRuntimeService.save_account()`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#save_account):

```python
QiskitServerless.save_account(token="YOUR_API_KEY", channel="ibm_quantum_platform", instance="INSTANCE_CRN")
```

Pokud máš [účet uložený,](/guides/save-credentials) není třeba při autentizaci token zadávat:

```python
from qiskit_ibm_catalog import QiskitServerless

serverless = QiskitServerless()
```

### Nahrání šablony {#upload-the-template}

Chceš-li nahrát vlastní Qiskit Function, musíš nejprve vytvořit instanci objektu `QiskitFunction`, který definuje zdrojový kód funkce. Název ti umožní funkci identifikovat ve vzdáleném clusteru. Hlavní vstupní bod je soubor obsahující `if __name__ == "__main__"`. Pokud tvůj pracovní postup vyžaduje další zdrojové soubory, můžeš definovat pracovní adresář, který bude nahrán spolu se vstupním bodem.

```python
from qiskit_ibm_catalog import QiskitFunction

template = QiskitFunction(
    title="sqd_pcm_template",
    entrypoint="sqd_pcm_entrypoint.py",
    working_dir="./source_files/",  # all files in this directory will be uploaded
    dependencies=[
        "ffsim==0.0.54",
        "pyscf==2.9.0",
        "qiskit_addon_sqd==0.10.0",
    ],
)
print(template)
```

```text
QiskitFunction(sqd_pcm_template)
```

Jakmile je instance připravena, nahraj ji do serverless:

```python
serverless.upload(template)
```

```text
QiskitFunction(sqd_pcm_template)
```

Chceš-li ověřit, zda byl program úspěšně nahrán, použij `serverless.list()`:

```python
serverless.list()
```
