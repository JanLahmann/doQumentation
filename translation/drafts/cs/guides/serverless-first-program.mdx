## Příklad: vzdálená transpilace s Qiskit Serverless {#example-remote-transpilation-with-qiskit-serverless}

Začni s následujícím příkladem, který transpiluje `circuit` pro zadaný `backend` a cílový `optimization_level`, a postupně přidávej další prvky pro nasazení své zátěže do Qiskit Serverless.

Vlož následující buňku kódu do souboru `./source_files/transpile_remote.py`. Tento soubor je program, který nahraješ do Qiskit Serverless.

```python
%%writefile ./source_files/transpile_remote.py

from qiskit.transpiler import generate_preset_pass_manager

def transpile_remote(circuit, optimization_level, backend):
    """Transpiles an abstract circuit into an ISA circuit for a given backend."""
    pass_manager = generate_preset_pass_manager(
        optimization_level=optimization_level,
		backend=backend
    )
    isa_circuit = pass_manager.run(circuit)
    return isa_circuit
```

## Nastavení souborů {#set-up-your-files}

Qiskit Serverless vyžaduje uspořádání `.py` souborů tvé zátěže do vyhrazeného adresáře. Následující struktura je příkladem dobré praxe:

```text
serverless_program
├── program_uploader.ipynb
└── source_files
    ├── transpile_remote.py
    └── *.py
```

Serverless nahraje obsah adresáře `source_files` ke vzdálenému spuštění. Jakmile jsou tyto soubory připraveny, můžeš upravit `transpile_remote.py` tak, aby načítal vstupní hodnoty a vracel výstupy.

### Získání argumentů programu {#get-program-arguments}

Tvůj počáteční soubor `transpile_remote.py` má tři vstupy: `circuits`, `backend_name` a `optimization_level`. Serverless je v současnosti omezen pouze na serializovatelné vstupy a výstupy. Z tohoto důvodu nelze předat `backend` přímo, takže místo toho použij `backend_name` jako řetězec.

```python
%%writefile --append ./source_files/transpile_remote.py

from qiskit_serverless import get_arguments, save_result, distribute_task, get

# Get program arguments
arguments = get_arguments()
circuits = arguments.get("circuits")
backend_name = arguments.get("backend_name")
optimization_level = arguments.get("optimization_level")
```

```text
Appending to ./source_files/transpile_remote.py
```

V tuto chvíli můžeš získat svůj Backend pomocí `QiskitRuntimeService` a přidat svůj stávající program pomocí následujícího kódu. Tento kód vyžaduje, aby sis již [uložil přihlašovací údaje](/guides/cloud-setup).

```python
%%writefile --append ./source_files/transpile_remote.py

from qiskit_ibm_runtime import QiskitRuntimeService

service = QiskitRuntimeService()
backend = service.backend(backend_name)
```

```text
Appending to ./source_files/transpile_remote.py
```

Nakonec můžeš spustit `transpile_remote()` pro všechny předané `circuits` a vrátit `transpiled_circuits` jako výsledek:

```python
%%writefile --append ./source_files/transpile_remote.py

results = [
    transpile_remote(circuit, 1, backend)
    for circuit in circuits
]

save_result({
    "transpiled_circuits": results
})
```


## Nasazení na IBM Quantum Platform {#deploy-to-ibm-quantum-platform}

Předchozí část vytvořila program určený ke vzdálenému spuštění. Buňky kódu v této části nahrají tento program do Qiskit Serverless.

Použij `qiskit-ibm-catalog` pro ověření v `QiskitServerless` pomocí svého API klíče, který najdeš na [IBM Quantum dashboardu](https://quantum.cloud.ibm.com), a nahraj program.

Pomocí `save_account()` můžeš uložit své přihlašovací údaje (viz část [Nastavení pro použití IBM Cloud](/guides/cloud-setup#cloud-save)). Poznamenej, že tím zapíšeš své přihlašovací údaje do stejného souboru jako [`QiskitRuntimeService.save_account()`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#save_account).

```python
from qiskit_ibm_catalog import QiskitServerless, QiskitFunction

# Authenticate to the remote cluster and submit the pattern for remote execution
serverless = QiskitServerless()
```

Qiskit Serverless zkomprimuje obsah `working_dir` (v tomto případě `source_files`) do souboru `tar`, který se nahraje a po skončení vyčistí. Parametr `entrypoint` identifikuje hlavní spustitelný soubor programu, který má Qiskit Serverless spustit. Pokud má tvůj program vlastní závislosti `pip`, můžeš je přidat do pole `dependencies`:

```python
transpile_remote_demo = QiskitFunction(
    title="transpile_remote_serverless",
    entrypoint="transpile_remote.py",
    working_dir="./source_files/",
)
```

```python
serverless.upload(transpile_remote_demo)
```

```text
QiskitFunction(transpile_remote_serverless)
```

```python
# This cell is hidden from users, it checks the program executes correctly
from time import sleep
from qiskit import QuantumCircuit

qc = QuantumCircuit(2)
transpile_remote_serverless = serverless.load("transpile_remote_serverless")
job = transpile_remote_serverless.run(
    circuits=[qc],
    backend="ibm_sherbrooke",
    optimization_level=1,
)
while True:
    sleep(5)
    status = job.status()
    if status not in ["QUEUED", "INITIALIZING", "RUNNING", "DONE"]:
        raise Exception(
            f"Unexpected job status: '{status}'\n"
            + "Here are the logs:\n"
            + job.logs()
        )
    if status == "DONE":
        break
```
