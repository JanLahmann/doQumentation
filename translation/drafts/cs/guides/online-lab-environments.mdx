# Online laboratorní prostředí

Kromě stažení a instalace Qiskitu na svůj lokální počítač existuje několik možností pro ty, kteří preferují práci v online vývojovém prostředí.

Několik poskytovatelů nabízí online Jupyter prostředí s předinstalovaným Qiskitem pro rychlý a pohodlný onboarding. Tito doporučení poskytovatelé zahrnují poskytovatele cloudových služeb OVHcloud a cloudovou platformu pro vývoj kvantového softwaru qBraid.

## Doporučená notebooková prostředí s předinstalovaným Qiskitem

Další online Jupyter notebooková prostředí, jako jsou IBM Watson&reg; Studio, Google Colab a Microsoft Azure Machine Learning Studio, lze použít k práci s Qiskitem podle příslušných pokynů.

### OVHcloud AI notebooky

Uživatelé OVHcloud Public Cloud a OVHcloud Startup mohou používat své OVHcloud kvantové notebooky pro přístup k Qiskitu a odesílání úloh na IBM&reg; QPU pomocí svého účtu IBM Quantum&reg; nebo IBM Cloud&reg;. Chceš-li se dozvědět více, navštiv stránku [Dokumentace a tutoriály OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
OVHcloud AI notebooky nejsou dostupné v některých regionech, například ve Spojených státech. Uživatelská rozhraní se navíc mohou v různých regionech lišit.
</Admonition>

Začni podle těchto kroků:

1. [Vytvoř si účet IBM Cloud.](./cloud-setup)
1. [Nastav si účet OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Vytvoř projekt OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Postupuj podle kroků v části [Spusť svůj první AI Notebook,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) přičemž jako framework vyber **Qiskit**.

### qBraid Lab

Uživatelé Qiskitu mohou používat předkonfigurovaná Python prostředí od qBraid podle těchto pokynů. Více informací najdeš v [dokumentaci qBraid.](https://docs.qbraid.com/home/introduction)

1. [Vytvoř si bezplatný účet qBraid.](https://docs.qbraid.com/home/account)
3. [Vytvoř Lab prostředí.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Spusť qBraid Lab, klikni na ikonu **ENVS** vpravo, klikni na **+ ADD** v horní části nového postranního panelu a vyhledej Qiskit. Klikni na verzi, kterou chceš použít, a poté klikni na **Install**.
4. Vytvoř Jupyter notebook kliknutím na **Python 3 [Qiskit]** v sekci Notebook.
4. V buňce notebooku můžeš nainstalovat libovolné další Python balíčky (například Qiskit v1.2) spuštěním příkazu `%pip install [package_name].` Více informací najdeš v tématu [Pip (magic) commands](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Další notebooková prostředí pro uživatele Qiskitu

U platforem bez předkonfigurovaného Qiskitu jsou pokyny k nastavení podobné, jen musíš Qiskit nainstalovat ručně. Vytvoříš si účet, spustíš nový projekt (nebo notebook či pracovní prostor, podle platformy) a nainstaluješ Qiskit.

### IBM Watson Studio

1. [Nastav si bezplatnou zkušební verzi Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Vytvoř projekt Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Chceš-li nainstalovat potřebné balíčky a závislosti Qiskitu, otevři nový notebook a spusť následující příkaz:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab

1. [Vytvoř si Google účet.](https://support.google.com/accounts/answer/27441?hl)
2. Přejdi na [úvodní stránku Google Colab](https://colab.research.google.com/) a přihlas se pomocí svého Google účtu.
3. [Vytvoř Colab notebook](https://colab.research.google.com/#create=true) a nainstaluj Qiskit spuštěním následujícího příkazu:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio

1. [Vytvoř si Azure účet.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Vytvoř pracovní prostor.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Vytvoř Jupyter notebook ve svém pracovním prostoru.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Z notebooku spusť tento příkaz pro instalaci Qiskitu:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
