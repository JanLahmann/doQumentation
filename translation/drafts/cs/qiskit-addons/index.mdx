# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) jsou oficiální rozšiřující balíčky, které poskytují specializované algoritmy a techniky pro kvantové výpočty. Každý addon se zaměřuje na konkrétní oblast a obsahuje tutoriály se spustitelným kódem.

## Dostupné Addons {#dostupne-addons}

| Addon | Balíček | Popis |
|-------|---------|-------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Zmenši šířku a hloubku Circuit rozdělením Gate a drátů |
| **SQD** | `qiskit-addon-sqd` | Diagonalizace založená na vzorcích (Sample-based Quantum Diagonalization) pro odhadování základního stavu |
| **OBP** | `qiskit-addon-obp` | Zpětná propagace operátorů (Operator Back-Propagation) pro snížení hloubky Circuit |
| **MPF** | `qiskit-addon-mpf` | Víceproduktové vzorce (Multi-Product Formulas) pro vylepšenou simulaci hamiltoniánů |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Aproximovaná kvantová kompilace (Approximate Quantum Compilation) s tenzorními sítěmi |
| **PNA** | `qiskit-addon-pna` | Absorpce propagovaného šumu (Propagated Noise Absorption) pro zmírnění chyb |
| **SLC** | `qiskit-addon-slc` | Stínované světelné kužely (Shaded Lightcones) pro snížení režijních nákladů na vzorkování PEC |

Všechny tutoriály lze spustit přímo v prohlížeči pomocí Binder, nebo na vlastním Jupyter serveru.

Dokumentaci API najdeš v jednotlivých repozitářích addonů na [GitHubu](https://github.com/Qiskit?q=qiskit-addon).
