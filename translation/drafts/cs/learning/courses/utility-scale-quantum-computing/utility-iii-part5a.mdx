## 3. Strategie 2. Vyvážený strom qubitů {#3-strategy-2-balanced-tree-of-qubits}

Další myšlenkou je najít vyvážený strom qubitů. Použitím stromu místo řetězce by se měla hloubka Circuit snížit. Nejprve odstraníme ze grafu propojení uzly se „špatnými" chybami čtení a hrany se „špatnými" chybami Gate.

```python
BAD_READOUT_ERROR_THRESHOLD = 0.1
BAD_ECRGATE_ERROR_THRESHOLD = 0.1
bad_readout_qubits = [
    q
    for q in range(backend.num_qubits)
    if backend.target["measure"][(q,)].error > BAD_READOUT_ERROR_THRESHOLD
]
bad_ecrgate_edges = [
    qpair
    for qpair in backend.target["ecr"]
    if backend.target["ecr"][qpair].error > BAD_ECRGATE_ERROR_THRESHOLD
]
print("Bad readout qubits:", bad_readout_qubits)
print("Bad ECR gates:", bad_ecrgate_edges)
```


```python
g = backend.coupling_map.graph.copy().to_undirected()
g.remove_edges_from(
    bad_ecrgate_edges
)  # remove edge first (otherwise might fail with a NoEdgeBetweenNodes error)
g.remove_nodes_from(bad_readout_qubits)
```

```python
N = 16
```

Zavoláme funkci `betweenness_centrality`, abychom našli Qubit pro kořenový uzel. Uzel s nejvyšší hodnotou centrality prostřednictví leží ve středu grafu. Odkaz: https://www.rustworkx.org/tutorial/betweenness_centrality.html

Nebo ho můžeš vybrat ručně.

```python
# central = 65 #Select the center node manually
c_degree = dict(rx.betweenness_centrality(g))
central = max(c_degree, key=c_degree.get)
central
```

Počínaje kořenovým uzlem vygenerujeme strom prohledáváním do šířky (BFS). Odkaz: https://qiskit.org/ecosystem/rustworkx/apiref/rustworkx.bfs_search.html#rustworkx-bfs-search

```python
class TreeEdgesRecorder(rx.visit.BFSVisitor):
    def __init__(self, N):
        self.edges = []
        self.N = N

    def tree_edge(self, edge):
        self.edges.append(edge)
        if len(self.edges) >= self.N - 1:
            raise rx.visit.StopSearch()

vis = TreeEdgesRecorder(N)
rx.bfs_search(g, [central], vis)
best_qubits = sorted(list(set(q for e in vis.edges for q in (e[0], e[1]))))
# print('Tree edges:', vis.edges)
```
