## 2. Matice hustoty {#2-density-matrices}

Kvantový stav $|\psi \rangle$ lze zapsat také ve formě matice hustoty. Tato forma je užitečná pro vyjádření pravděpodobnostní směsi čistých kvantových stavů. V případě jednoho Qubitu lze psát

$$
|\psi \rangle \langle \psi| \equiv  \rho = \left(
\begin{pmatrix}
\cos\frac{\theta}{2}\\
e^{i\varphi}\sin\frac{\theta}{2}
\end{pmatrix}
\right)
\left(
\begin{pmatrix}
\cos\frac{\theta}{2} & e^{-i\varphi}\sin\frac{\theta}{2}
\end{pmatrix}
\right)
=\frac{1}{2}\left(\begin{pmatrix}
1+\cos\theta & e^{-i\varphi}\sin\theta\\
e^{-i\varphi}\sin\theta & 1-\cos\theta
\end{pmatrix}\right)
$$

Všimni si, že matice hustoty $\rho$ je lineární kombinací Pauliho matic, jak je uvedeno níže:

$$
\rho = \frac{1}{2}\bigl( \textbf{I} + (\sin{\theta}\cos{\varphi})\textbf{X}+  (\sin{\theta}\sin{\varphi})\textbf{Y} +  (\cos{\theta})\textbf{Z} \bigr)
$$

Nebo obecně:

$$
\rho = \frac{1}{2}(\textbf{I} + r_{x}\textbf{X}+ r_{y}\textbf{Y} + r_{z}\textbf{Z})
$$

kde
$r_{x}^2+r_{y}^2+r_{z}^2=1$.

Blochův vektor je $\textbf{r} = (r_{x}, r_{y}, r_{z})$.

Nyní vytvoříme libovolný kvantový stav pomocí náhodných čísel.

```python
import numpy as np

# create a random 1-qubit state from a random (theta, varphi) to define r vector
np.random.seed(1)  # fixing seed for repeatibility

theta = np.random.uniform(0.0, 1.0) * np.pi  # from 0 to pi
varphi = np.random.uniform(0.0, 2.0) * np.pi  # from 0 to 2*pi

def get_r_vec(theta, varphi):
    rx = np.sin(theta) * np.cos(varphi)
    ry = np.sin(theta) * np.sin(varphi)
    rz = np.cos(theta)
    return (rx, ry, rz)

# get r vector
rx, ry, rz = get_r_vec(theta, varphi)

print("theta=" + str(theta), ",varphi=" + str(varphi))
print("(rx, ry, rz) = (" + str(rx) + ", " + str(ry) + ", " + str(rz) + ")")
```

```text
(rx, ry, rz) = (-0.1791150283307452, -0.9494670044331133, 0.2577405946274022)
```

Tento Blochův vektor lze zobrazit na Blochově sféře.

## 3. Kvantová stavová tomografie {#3-quantum-state-tomography}

Pokud měříš kvantový stav pouze ve výpočetní bázi ($|0 \rangle$ a $|1 \rangle$), informace o fázi (informace o komplexních číslech) se ztratí. Ale pokud máme mnoho kopií $|\psi \rangle$ opakováním přípravného procesu (stavy nelze klonovat, ale přípravné procesy lze opakovat), můžeme odhadnout hodnoty $r_{x}, r_{y}, r_{z}$ provedením *kvantové stavové tomografie* pro matici hustoty $\rho$. Vzhledem k tvaru:

$$
\rho = \frac{1}{2}(\textbf{I} + r_{x}\textbf{X}+ r_{y}\textbf{Y} + r_{z}\textbf{Z})
$$

platí

$$
Tr(\textbf{X} \rho) =  r_{x}, \quad Tr(\textbf{Y} \rho) =  r_{y}, \quad Tr(\textbf{Z} \rho) =  r_{z}
$$

V případě $r_{z}$:

$$
Tr(\textbf{Z} \rho) =  \langle 0|\textbf{Z} \rho|0 \rangle +  \langle 1|\textbf{Z} \rho|1 \rangle
$$
$$
= \langle 0|(|0 \rangle\langle 0|-|1 \rangle\langle 1|) \rho|0 \rangle +\langle 1|(|0 \rangle\langle 0|-|1 \rangle\langle 1|) \rho|1 \rangle
$$
$$
=\langle 0|\rho|0 \rangle- \langle 1| \rho|1 \rangle
$$
$$
=\langle 0|\psi\rangle\langle \psi|0 \rangle -  \langle 1| \psi\rangle\langle \psi|1 \rangle
$$
$$
=|\alpha|^2-|\beta|^2
$$

Poslední transformace rovnice platí pro $|\psi \rangle  =\alpha|0\rangle+\beta|1\rangle$. Hodnotu $r_{z}$ tedy lze získat jako pravděpodobnost $|0 \rangle$ mínus pravděpodobnost $|1 \rangle$.

### Odhad hodnoty $r_z$ {#estimate-r-z-value}

Abychom odhadli $r_z$, vytvoříme kvantový stav a změříme ho. Přípravu a měření pak mnohokrát zopakujeme. Nakonec využijeme statistiku měření k odhadu výše uvedených pravděpodobností, a tím odhadneme $r_z$.

Pro vytvoření náhodného kvantového stavu použijeme obecný unitární Gate $U$ s parametry $\theta, \varphi$. (Další informace najdeš v dokumentaci [U-gate](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.UGate).)

```python
from qiskit import QuantumCircuit

# create a 1-qubit quantum state psi from theta, varphi parameters
qc = QuantumCircuit(1, 1)
qc.u(theta, varphi, 0.0, 0)

# measure in computational basis
qc.measure(0, 0)

qc.draw(output="mpl")
```

![Output of the previous code cell](/learning/images/courses/utility-scale-quantum-computing/teleportation/extracted-outputs/b1327267-f2fe-4a61-91d9-328c672b557e-0.avif)

Pomocí `AerSimulator` provedeme měření ve výpočetní bázi a odhadneme $r_z$.

```python
# see if the expected value of measuring in the computational basis
# approaches the limit of rz
from qiskit_aer import AerSimulator
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager
from qiskit_ibm_runtime import Sampler
from qiskit.visualization import plot_histogram

# Define backend
backend = AerSimulator()
nshots = 1000  # or 10000
# nshots = 10000

# Transpile to backend
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

# Run the job
sampler = Sampler(mode=backend)
job = sampler.run([isa_qc], shots=nshots)
result = job.result()

# Extract counts data
counts = result[0].data.c.get_counts()
print(counts)

# Plot the counts in a histogram

plot_histogram(counts)
```

```text
{'1': 375, '0': 625}
```

```python
rz_approx = (counts["0"] - counts["1"]) / nshots

```


Pomocí metody kvantové stavové tomografie jsme odhadli hodnotu $r_z$. V tomto případě jsme pro „náhodný" stav zvolili parametry, takže hodnotu $r_z$ známe a můžeme si výsledek ověřit. Práce v oblasti utility-scale však ze své podstaty není vždy tak snadné ověřit. O kontrole kvantových výsledků si povíme více později v tomto kurzu. Pro tuto chvíli si pouze poznamenej, že náš odhad byl přiměřeně přesný.

### Cvičení 1: Odhad hodnoty $r_x$ {#exercise-1-estimate-r-x-value}

Pamatuj si, že kvantové počítače IBM&reg; měří podél osy $z$ (někdy se říká „v bázi $z$" nebo „ve výpočetní bázi"). Pomocí rotací před měřením však lze měřit i projekci kvantového stavu na osu x. Přesněji řečeno, pokud náš systém otočíme tak, aby to, co mířilo podél $x$, nyní mířilo podél $z$, můžeme zachovat stejný měřicí hardware podél $z$, ale zjistit informace o stavu, který chvíli před tím mířil podél $x$. Takto většina kvantových počítačů (a všechny kvantové počítače IBM) provádí měření podél více os.

S tímto pochopením zkus napsat kód pro odhad hodnoty $r_x$ pomocí kvantové stavové tomografie.
