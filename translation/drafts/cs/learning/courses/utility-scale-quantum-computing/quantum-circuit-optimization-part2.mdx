## 2. Na optimalizaci Circuit záleží {#circuit-optimization-matters}

Nejprve porovnáme výsledky spuštění přípravných Circuit pro 5-qubitový stav GHZ ($\frac{1}{\sqrt{2}} \left( |00000\rangle + |11111\rangle \right)$) s optimalizací a bez ní.

```python
from qiskit.circuit import QuantumCircuit
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager
from qiskit.primitives import BackendSamplerV2 as Sampler
```

```python
from qiskit_ibm_runtime.fake_provider import FakeBrisbane

backend = FakeBrisbane()
```

Nejprve použijeme Circuit pro stav GHZ naivně syntetizovaný takto.

```python
num_qubits = 5

ghz_circ = QuantumCircuit(num_qubits)
ghz_circ.h(0)
[ghz_circ.cx(0, i) for i in range(1, num_qubits)]
ghz_circ.measure_all()
ghz_circ.draw("mpl")
```

### 2.1 Úroveň optimalizace {#optimization-level}
K dispozici jsou 4 úrovně `optimization_level` v rozsahu 0–3. Čím vyšší je úroveň optimalizace, tím více výpočetního úsilí je vynaloženo na optimalizaci Circuit. Úroveň 0 neprovádí žádnou optimalizaci a pouze vykoná minimální množství práce potřebné ke spuštění Circuit na zvoleném Backend. Úroveň 3 vynakládá největší úsilí (a zpravidla i čas) na pokus o optimalizaci Circuit. Úroveň 1 je výchozí úrovní optimalizace.
Circuit transpilujeme bez optimalizace (`optimization_level=0`) a s optimalizací (`optimization_level=2`).
Uvidíme velký rozdíl v délce transpilovaných Circuit.

```python
pm0 = generate_preset_pass_manager(
    optimization_level=0, backend=backend, seed_transpiler=777
)
pm2 = generate_preset_pass_manager(
    optimization_level=2, backend=backend, seed_transpiler=777
)
circ0 = pm0.run(ghz_circ)
circ2 = pm2.run(ghz_circ)
print("optimization_level=0:")
display(circ0.draw("mpl", idle_wires=False, fold=-1))
print("optimization_level=2:")
display(circ2.draw("mpl", idle_wires=False, fold=-1))
```

Spusť na fiktivním Backend (simulace se šumem). Viz Příloha 1 pro spuštění na reálném Backend.

```python
# run the circuits on the fake backend (noisy simulator)
sampler = Sampler(backend=backend)
job = sampler.run([circ0, circ2], shots=10000)
print(f"Job ID: {job.job_id()}")
```

```text
Job ID: 93a4ac70-e3ea-44ad-aea9-5045840c9076
```

```python
# get results
result = job.result()
unoptimized_result = result[0].data.meas.get_counts()
optimized_result = result[1].data.meas.get_counts()
```

```python
from qiskit.visualization import plot_histogram

# plot
sim_result = {"0" * 5: 0.5, "1" * 5: 0.5}
plot_histogram(
    [result for result in [sim_result, unoptimized_result, optimized_result]],
    bar_labels=False,
    legend=[
        "ideal",
        "no optimization",
        "with optimization",
    ],
)
```
