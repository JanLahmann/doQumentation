Stručně si zopakujeme GHZ stavy a to, jaké rozložení lze očekávat při aplikaci ```Sampler``` na jeden z nich. Poté jasně stanovíme cíl této lekce.

###  1.1 GHZ stav

GHZ stav (stav Greenbergera-Hornea-Zeilingera) pro $n$ qubitů je definován jako

$$
\frac{1}{\sqrt 2}(|0\rangle ^ {\otimes n}+ |1\rangle^ {\otimes n})
$$

Přirozeně ho lze pro 6 qubitů vytvořit pomocí následujícího kvantového Circuit.

```python
N = 6
qc = QuantumCircuit(N, N)

qc.h(0)
for i in range(N - 1):
    qc.cx(0, i + 1)

# qc.measure_all()
qc.barrier()
qc.measure(list(range(N)), list(range(N)))

qc.draw(output="mpl", idle_wires=False, scale=0.5)
```

Hloubka není příliš velká, i když z předchozích lekcí víš, že lze dosáhnout lepšího výsledku. Vyberme Backend a transpilujme tento Circuit.

```python
service = QiskitRuntimeService()
backend = service.least_busy(operational=True, simulator=False)
backend.name
```

```python
pm = generate_preset_pass_manager(3, backend=backend)
qc_transpiled = pm.run(qc)
qc_transpiled.draw(output="mpl", idle_wires=False, fold=-1)
```

Transpilovaná hloubka dvou-qubitových operací opět není příliš velká. Abychom ale mohli pracovat s GHZ stavem na více qubitech, bude zřejmě nutné přemýšlet o optimalizaci Circuit. Spusťme ho pomocí ```Sampler``` a podívejme se, co vrátí skutečný kvantový počítač.

```python
sampler = Sampler(mode=backend)
shots = 40000
job = sampler.run([qc_transpiled], shots=shots)
job_id = job.job_id()
print(job_id)
```
