```python
import numpy as np

n = 8
q = 127
N = int(1.1 * n * np.log(q))
sigma = 1.0
print(f"n={n},q={q},N={N},sigma={sigma}")
```

Také potřebujeme způsob, jak zavádět chyby.

Zde $\chi$ představuje chyby, které chceme zavést – používáme diskrétní Gaussovo rozdělení na $\mathbb{Z}_{q}$ charakterizované střední hodnotou 0 a směrodatnou odchylkou $\sigma$.

Tato definice chyb, které se mají zavést, je rovněž veřejná.

V Pythonu tedy potřebujeme jednoduchou funkci, která aproximuje náhodné příspěvky chyb/šumu $\varepsilon$ vzorkované z diskrétního Gaussova rozdělení $\chi$ se směrodatnou odchylkou $\sigma$.

Vypíšeme několik příkladů, abyste viděli rozdělení kolem 0.

```python
def chi(stdev, modulus):
    return round((np.random.randn() * stdev**2)) % modulus

# print some examples
sd = 2
m = 1000
for x in range(10):
    print("chi = ", chi(sd, m))
```

Dále musí Alice vygenerovat svůj pár klíčů.

Svůj **soukromý klíč** vytvoří tak, že zvolí $n$ hodnot mezi 0 a $q$.

```python
# Alice's private key
alice_private_key = np.random.randint(0, high=q, size=n)
print(f"Alice's private key: {alice_private_key}")
```

Alice nyní vytvoří svůj **veřejný klíč** tím, že zvolí náhodné vektory, které se pak zkombinují s vygenerovanými chybami.

<details>
<summary>Matematika</summary>

Přesněji řečeno, potřebuje vygenerovat vzorek $A_{s,\chi}$.

Tedy náhodně zvolí vektory $\mathbf{a} \in \mathbb{Z}_{q}^{n}$ a chyby $\varepsilon$ z $\chi$, aby sestrojila $N$ vzorků $(\mathbf{a}, b)$.

</details>

```python
# Alice's Public Key
alice_public_key = []

# N is the number of values we want in the key
for i in range(N):
    # Get n random values between 0 and <q
    a = np.random.randint(0, high=q, size=n)
    # get an error to introduce
    epsilon = chi(sigma, q)
    #  calculate dot product (ie like array multiplication)
    b = (np.dot(a, alice_private_key) + epsilon) % q
    # value to be added to the key -
    sample = (a, b)
    alice_public_key.append(sample)

print(f"Alice's public key: {alice_public_key}")
```

Alice teď může svůj veřejný klíč sdílet.

Bob jej nyní může použít k odeslání šifrované zprávy Alici.

Bobova zpráva je jediný bit.

```python
# Encryption
bob_message_bit = 1
print(f"Bob's message bit={bob_message_bit}")
```

Aby zprávu zašifroval, musí Bob z Aliciného veřejného klíče náhodně vybrat libovolný počet vzorků, ze kterých sestaví šifrový text.

K tomu si vytvoří masku – náhodný binární vektor $r$ délky $N$.

```python
# a list of N values between 0 and <2 - ie 0 or 1
r = np.random.randint(0, 2, N)
print(r)
```

Tuto masku nyní aplikujeme na odpovídající položku v Alicině veřejném klíči a spočteme součet získaných hodnot.

```python
sum_ai = np.zeros(n, dtype=int)
sum_bi = 0

for i in range(N):
    sum_ai = sum_ai + r[i] * alice_public_key[i][0]
    sum_bi = sum_bi + r[i] * alice_public_key[i][1]
sum_ai = [x % q for x in sum_ai]
# sum_bi = sum_bi
ciphertext = (sum_ai, (bob_message_bit * int(np.floor(q / 2)) + sum_bi) % q)
print(f"ciphertext is: {ciphertext}")
```

Nakonec Bob odvysílá šifrový text, který Alice dokáže dešifrovat pomocí svého soukromého klíče.

```python
# Decryption
adots = np.dot(ciphertext[0], alice_private_key) % q
b_adots = (ciphertext[1] - adots) % q

decrypted_message_bit = round((2 * b_adots) / q) % 2

print(
    f"original message bit={bob_message_bit}, decrypted message bit={decrypted_message_bit}"
)

assert bob_message_bit == decrypted_message_bit
```

Protože tento protokol funguje bit po bitu, pro šifrování delších bitových řetězců jednoduše operace opakujeme ve smyčce. Následující ukázka zachycuje situaci, kdy chce Bob přenést 16 šifrovaných bitů.

```python
bob_message_bits = np.random.randint(0, 2, 16)
print(f"Bob's message bits are : {bob_message_bits}")
decrypted_bits = []

for ib in range(len(bob_message_bits)):
    bob_message_bit = bob_message_bits[ib]

    r = np.random.randint(0, 2, N)

    sum_ai = np.zeros(n, dtype=int)
    sum_bi = 0
    for i in range(N):
        sum_ai = sum_ai + r[i] * alice_public_key[i][0]
        sum_bi = sum_bi + r[i] * alice_public_key[i][1]
    sum_ai = [x % q for x in sum_ai]

    ciphertext = (sum_ai, (bob_message_bit * int(np.floor(q / 2)) + sum_bi) % q)

    adots = np.dot(ciphertext[0], alice_private_key) % q
    b_adots = (ciphertext[1] - adots) % q

    decrypted_message_bit = round((2 * b_adots) / q) % 2
    assert decrypted_message_bit == bob_message_bit

    decrypted_bits.append(decrypted_message_bit)

print(f"Decrypted message bits = {np.array(decrypted_bits)}")
```
