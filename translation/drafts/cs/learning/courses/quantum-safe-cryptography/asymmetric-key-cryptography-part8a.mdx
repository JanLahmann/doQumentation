### Ilustrace DSA v Pythonu {#illustration-of-the-dsa-in-python}
V tomto příkladu použijeme malá prvočísla, abychom ilustrovali proces DSA ve scénáři, kde Alice posílá podepsanou zprávu Bobovi. Použijeme malá celá čísla. V reálném světě by se používala mnohem větší prvočísla v řádu 2048–3072 bitů.

<Admonition type="note">
Můžeš zkusit spustit tento příklad znovu s jinými hodnotami, abys viděl/a, jak se algoritmus chová, ale ujisti se, že začínáš spouštět od první buňky v této části.
</Admonition>

Začneme importem potřebných knihoven a volbou parametrů.
Celočíselné parametry $p$, $q$, $g$ jsou veřejné informace a musí splňovat následující pravidla:

- $p$, $q$ jsou obě prvočísla
- $(p-1) \mod\ q = 0$
- $g \ge 2$
- $g \le (p-2)$
- $g^{(p-1)/q} \mod\ p \neq 1$

```python
from random import randint

# parameter generation: select the primes q, p and generator g:
# EXPERIMENT with the values, they must meet certain rules
# this example code does not verify p,q are prime

q = 11
p = 23
g = 4

assert (p - 1) % q == 0
assert g >= 2
assert g <= (p - 2)
assert (pow(g, (p - 1) / q) % p) != 1

print(f"Public information is good: q={p}, p={q}, g={g}")
```

Dále podepisující strana, Alice, vygeneruje svůj soukromý klíč.

Soukromý klíč k (`alice-private-key` v kódu Pythonu) musí splňovat:
 - $k \ge 2$
 - $k \le (q-1)$

```python
# Alice chooses an integer randomly from {2..q-1}
# EXPERIMENT with the values

alice_private_key = randint(2, q - 1)
# alice_private_key =

assert alice_private_key >= 2
assert alice_private_key <= (q - 1)

print(f"Alice's private key is {alice_private_key}")
```

Alice si svůj soukromý klíč uchovává v tajnosti.

Dále Alice vypočítá svůj veřejný klíč pomocí modulárního umocňování. Invertovat tento krok za účelem získání soukromého klíče zpět je instancí DLP, a proto je na klasických počítačích obtížné jej vypočítat, pokud jsou použita velká prvočísla.

Z výše uvedeného matematického vysvětlení víme, že toto lze vypočítat pomocí vzorce:

$y = g^{x} \mod\ p$

```python
alice_public_key = pow(g, alice_private_key, p)
# Alternatively can use (g ** alice_private_key) % p

print(f"Alice's public key is {alice_public_key}")
```

Alice jako obvykle zpřístupní svůj veřejný klíč přes kanál, který nemusí být tajný.

Nyní může Alice podepsat zprávu.

Pro schéma digitálního podpisu potřebujeme vygenerovat hash $H(m)$ podepisované zprávy $m$.

Předpokládejme, že se Alice a Bob dohodnou na použití konkrétního hashovacího algoritmu s délkou hashe $N$ rovnou počtu bitů v $q$. V tomto jednoduchém příkladu omezíme výstupy naší simulované hashovací funkce hodnotou $q$. Hashovací funkce je zde triviální a jednoduše generuje náhodné celé číslo.

```python
hash_dict = {}

def mock_hash_func(input_message):
    print(input_message)
    if input_message not in hash_dict:
        hash_dict[input_message] = randint(1, q)
    return hash_dict[input_message]

alice_message = "Inspection tomorrow!"
alice_hash = mock_hash_func(alice_message)  # In reality, you'd use a hash function
print(f"Alice's message hash is: {alice_hash}")
```

Dále Alice potřebuje vygenerovat náhodné tajné číslo $k$ pro každou zprávu, jakož i jeho multiplikativní inverzi modulo $q$, aby mohla vypočítat podpis.

K výpočtu modulární inverze lze použít jednoduchý algoritmus hrubé síly. Je však lepší použít jednu z vestavěných funkcí Pythonu `pow`, jak je ukázáno níže
