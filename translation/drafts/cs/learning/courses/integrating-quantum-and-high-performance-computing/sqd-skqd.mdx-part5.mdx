```

### Post-processing (`postprocessing.py` a `postprocessing.sh`) {#post-processing-postprocessingpy-and-postprocessingsh}

V posledním kroku se vracíme ke klasickému počítači, abychom analyzovali data z kvantového experimentu a vypočítali konečný výsledek: energii základního stavu našeho cílového systému.

Program `postprocessing.py` nejprve načte soubor `counts.json` obsahující výsledky měření. Poté rekonstruuje hamiltonián Andersonova modelu (se stejnými parametry jako v `mapping.py`). Naměřené bitové řetězce a definici hamiltoniánu pak předá funkci `diagonalize_fermionic_hamiltonian`. Tato funkce provádí jádro logiky SKQD: pomocí bitových řetězců sestaví projektovaný hamiltonián $\tilde{H}$ a diagonalizuje ho, aby našla energii základního stavu.

```python

def callback(results: list[SCIResult]):

        print(f"\t\tEnergy: {result.energy}")
