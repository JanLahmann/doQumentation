### Fázové kódování {#phase-encoding}

Fázové kódování je velmi podobné výše popsanému úhlovému kódování. Fázový úhel qubitu je reálně hodnotný úhel $\phi$ kolem osy $z$ od osy +$x$. Data jsou mapována pomocí fázové rotace $P(\phi) = e^{i\phi/2}R_Z(\phi)$, kde $\phi \in (0,2\pi]$ (více informací viz [Qiskit PhaseGate](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.library.PhaseGate)). Doporučuje se přeškálovat data tak, aby $\vec{x}^{(j)}_k \in (0,2\pi]$. Předejdeš tím ztrátě informací a dalším potenciálně nežádoucím efektům[\[1,2\]](#references).

Qubit je často inicializován ve stavu $|0\rangle$, který je vlastním stavem operátoru fázové rotace, což znamená, že stav qubitu je nejprve potřeba otočit, aby bylo možné fázové kódování implementovat. Proto dává smysl inicializovat stav Hadamardovým Gate: $H|0\rangle = |+\rangle = \textstyle\frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)$. Fázové kódování na jednom qubitu znamená udělení relativní fáze úměrné hodnotě dat:

$$
|\vec{x}^{(j)}_k\rangle = P(\phi=\vec{x}^{(j)}_k)|+\rangle = \textstyle\frac{1}{\sqrt{2}}\big(|0\rangle + e^{i\vec{x}^{(j)}_k}|1\rangle\big).
$$

Postup fázového kódování mapuje každou hodnotu příznaku na fázi odpovídajícího qubitu, $\vec{x}^{(j)}_k \rightarrow Q_k$. Celkově má fázové kódování hloubku Circuit 2, včetně Hadamardovy vrstvy, což z něj činí efektivní schéma kódování. Fázově zakódovaný vícequbitový stav ($n$ Qubitů pro $N=n$ příznaků) je produktový stav:

$$
|\vec{x}^{(j)}\rangle = \bigotimes_{k=1}^{N} P_k(\phi = \vec{x}^{(j)}_k)|+\rangle^{\otimes N} = {\textstyle\frac{1}{\sqrt{2^N}}} \bigotimes_{k=1}^{N}\big(|0\rangle + e^{i\vec{x}^{(j)}_k}|1\rangle\big).
$$

Následující kód v Qiskit nejprve připraví počáteční stav jednoho qubitu jeho otočením Hadamardovým Gate, a poté ho znovu otočí pomocí phase gate, aby zakódoval datový příznak $\vec{x}^{(j)}_k=\frac{1}{2}\pi$.

```python
qc = QuantumCircuit(1)
qc.h(0)  # Hadamard gate rotates state down to Bloch equator
state1 = Statevector.from_instruction(qc)

qc.p(pi / 2, 0)  # Phase gate rotates by an angle pi/2
state2 = Statevector.from_instruction(qc)

states = state1, state2

qc.draw("mpl", scale=1)
```

Fázové kódování se používá v mnoha kvantových příznakovýkch mapách, zejména ve $Z$ a $ZZ$ příznakových mapách a obecných Pauliho příznakových mapách a dalších.

```
phase_data = [4, 8, 5, 9, 8, 6, 2, 9, 2, 5, 7, 0]
qc = QuantumCircuit(len(phase_data))
for i in range(0, len(phase_data)):
    qc.h(i)
    qc.rz(phase_data[i] * 2 * math.pi / float(max(phase_data)), i)
qc.draw(output="mpl")
```
