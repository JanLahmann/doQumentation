</summary>

__Odpověď:__

Přepišme působení $M_2$ na stavy ve dvojkové soustavě:

$$
\begin{aligned}
M_2|0001\rangle &= |0010\rangle \\
M_2|0010\rangle &= |0100\rangle \\
M_2|0100\rangle &= |1000\rangle \\
M_2|1000\rangle &= |0001\rangle \\
\end{aligned}
$$

Každé z těchto zobrazení lze provést jednoduchým hradlem SWAP. $M_2|0001\rangle$ získáme prohozením stavů qubitu $0$ a $1$. $M_2|0010\rangle$ získáme prohozením stavů qubitu $1$ a $2$. A tak dále. Matici $M_2$ tedy lze rozložit na následující posloupnost hradel SWAP:

$$
M_2 = SWAP(0,1)SWAP(1,2)SWAP(2,3)
$$

S vědomím, že operátory působí zprava doleva, ověřme, že to má požadovaný efekt na každý ze stavů:

$$
\begin{aligned}
M_2|0001\rangle &= SWAP(0,1)SWAP(1,2)SWAP(2,3)|0001\rangle \\
&= SWAP(0,1)SWAP(1,2)|0001\rangle \\
&= SWAP(0,1)|0001\rangle \\
&=|0010\rangle  \checkmark \\
M_2|0010\rangle &= SWAP(0,1)SWAP(1,2)SWAP(2,3)|0010\rangle \\
&= SWAP(0,1)SWAP(1,2)|0010\rangle \\
&= SWAP(0,1)|0100\rangle \\
&=|0100\rangle  \checkmark \\
M_2|0100\rangle &= SWAP(0,1)SWAP(1,2)SWAP(2,3)|0100\rangle \\
&= SWAP(0,1)SWAP(1,2)|1000\rangle \\
&= SWAP(0,1)|1000\rangle \\
&=|1000\rangle  \checkmark \\
M_2|1000\rangle &= SWAP(0,1)SWAP(1,2)SWAP(2,3)|1000\rangle \\
&= SWAP(0,1)SWAP(1,2)|0100\rangle \\
&= SWAP(0,1)|0010\rangle \\
&=|0001\rangle  \checkmark \\
\end{aligned}
$$
</details>

```python
def M2mod15():
    """
    M2 (mod 15)
    """
    b = 2
    U = QuantumCircuit(4)

    U.swap(2, 3)
    U.swap(1, 2)
    U.swap(0, 1)

    U = U.to_gate()
    U.name = f"M_{b}"

    return U
```

```python
# Get the M2 operator

```

Algoritmus QPE používá řízené hradlo $U$ (controlled-$U$). Teď, když máme Circuit pro $M_2$, musíme z něj udělat *řízený*-$M_2$ Circuit:

```python
def controlled_M2mod15():
    """
    Controlled M2 (mod 15)
    """
    b = 2
    U = QuantumCircuit(4)

    U.swap(2, 3)
    U.swap(1, 2)
    U.swap(0, 1)

    U = U.to_gate()
    U.name = f"M_{b}"
    c_U = U.control()

    return c_U
```

```python
# Get the controlled-M2 operator
controlled_M2 = controlled_M2mod15()

# Add it to a circuit and plot
circ = QuantumCircuit(5)
circ.compose(controlled_M2, inplace=True)
circ.decompose(reps=1).draw(output="mpl", fold=-1)
```

Nyní máme naše řízené hradlo $U$. Abychom ale mohli spustit algoritmus Quantum Phase Estimation, budeme potřebovat řízené $U^2$, řízené $U^4$, až po řízené $U^{2^{m-1}}$, kde $m$ je počet qubitů použitých k odhadu fáze. Čím více qubitů, tím přesnější odhad fáze. Pro náš postup odhadu fáze použijeme $m=8$ řídicích qubitů. Potřebujeme tedy:

$$
M_{a^{2^k}}|y\rangle \equiv |a^{2^k} y \bmod N \rangle
$$

kde index $k$, přičemž $0 \le k \le m-1 = 7$, odpovídá řídicímu qubitu. Vypočítejme nyní $a^{2^k}\bmod N$ pro každou hodnotu $k$:

```python
def a2kmodN(a, k, N):
    """Compute a^{2^k} (mod N) by repeated squaring"""
    for _ in range(k):
        a = int(np.mod(a**2, N))
    return a
```

```python
k_list = range(8)
b_list = [a2kmodN(2, k, 15) for k in k_list]

print(b_list)
```

```text
[2, 4, 1, 1, 1, 1, 1, 1]
```

Protože $a^{2^k} \bmod N = 1$ pro $k \ge 2$, jsou všechny odpovídající operátory ($M_8$ a vyšší) ekvivalentní identitě. Potřebujeme tedy zkonstruovat jen jednu další matici: $M_4.$

**Poznámka:** Toto zjednodušení funguje zde pouze proto, že řád $2 \bmod 15$ je $4$. Jakmile $k=2$ (tedy $2^k = 4$), každá další mocnina operátoru je identita. Obecně platí, že pro větší čísla $N$ nebo jiné volby $a$ nelze přeskočit konstrukci vyšších mocnin. To je jeden z důvodů, proč se jedná o *hračkový příklad*: malá čísla umožňují zkratky, které by pro větší případy nefungovaly.

def M4mod15():

    M4 (mod 15)

Nyní vše složíme dohromady, abychom s pomocí odhadu fáze nalezli řád $2\bmod 15$ pomocí kvantového Circuit:

```python
# Order finding problem for N = 15 with a = 2
N = 15
a = 2

# Number of qubits
num_target = floor(log(N - 1, 2)) + 1  # for modular exponentiation operators
num_control = 2 * num_target  # for enough precision of estimation

# List of M_b operators in order
k_list = range(num_control)
b_list = [a2kmodN(2, k, 15) for k in k_list]

# Initialize the circuit
control = QuantumRegister(num_control, name="C")
target = QuantumRegister(num_target, name="T")
output = ClassicalRegister(num_control, name="out")
circuit = QuantumCircuit(control, target, output)

# Initialize the target register to the state |1>
circuit.x(num_control)

# Add the Hadamard gates and controlled versions of the
# multiplication gates
for k, qubit in enumerate(control):
    circuit.h(k)
    b = b_list[k]
    if b == 2:
        circuit.compose(
            M2mod15().control(), qubits=[qubit] + list(target), inplace=True
        )
    elif b == 4:
        circuit.compose(
            M4mod15().control(), qubits=[qubit] + list(target), inplace=True
        )
    else:
        continue  # M1 is the identity operator

# Apply the inverse QFT to the control register
circuit.compose(QFTGate(num_control).inverse(), qubits=control, inplace=True)

# Measure the control register
circuit.measure(control, output)

circuit.draw("mpl", fold=-1)
```
