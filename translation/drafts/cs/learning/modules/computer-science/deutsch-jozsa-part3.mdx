<details>
<summary>

Ověř, že stav z kroku 3 výše je skutečně stav $|s\rangle$ pro speciální případ $n=1$.

</summary>

__Odpověď:__

Když explicitně rozepíšeš obě sumy, dostaneš stav se čtyřmi členy (vynecháme výstupní stav $|-\rangle$):

$$
|\Psi\rangle = \frac{1}{2}[|0\rangle + (-1)^s |0\rangle + |1\rangle + (-1)^{(s+1)} |1\rangle]
$$

Pokud $s=0$, první dva členy se sčítají konstruktivně a poslední dva se vyruší, takže $|\Psi\rangle = |0\rangle$. Pokud $s=1$, poslední dva členy se sčítají konstruktivně a první dva se vyruší, takže $|\Psi\rangle = |1\rangle$. V obou případech tedy $|\Psi\rangle = |s\rangle$. Snad ti tento nejjednodušší případ dá intuici pro obecný případ s $n$ Qubity: všechny členy, které nejsou $|s\rangle$, se vzájemně vyruší interferencí a zbyde jen stav $|s\rangle$.

</details>

<details>
<summary>
Jak může tentýž algoritmus řešit jak Bernstein-Vaziraniho, tak Deutsch-Jozsův problém? Aby to dávalo smysl, zamysli se nad Bernstein-Vaziraniho funkcemi ve tvaru $f(x) = s \cdot x$. Jsou tyto funkce zároveň Deutsch-Jozsovými funkcemi? Tedy urči, zda funkce tohoto tvaru splňují příslib Deutsch-Jozsova problému: že jsou buď *konstantní*, nebo *vyvážené*. Jak nám to pomáhá pochopit, že tentýž algoritmus řeší dva různé problémy?

</summary>

__Odpověď:__

Každá Bernstein-Vaziraniho funkce ve tvaru $f(x) = s \cdot x$ zároveň splňuje příslib Deutsch-Jozsova problému: pokud s=00...00, je funkce konstantní (pro každý řetězec x vrací vždy 0). Pokud je s jakýkoli jiný řetězec, je funkce vyvážená. Aplikace Deutsch-Jozsova algoritmu na jednu z těchto funkcí tedy současně řeší oba problémy! Vrátí řetězec, a pokud je tento řetězec 00...00, víme, že je konstantní; pokud obsahuje alespoň jednu „1", víme, že je vyvážený.

</details>

Můžeme také ověřit, že tento algoritmus Bernstein-Vaziraniho problém úspěšně řeší, a to experimentálně. Nejprve vytvoříme B-V funkci, která se skrývá uvnitř černé skříňky:

```python
# Step 1: Map the problem

def bv_function(s):
    """
    Create a Bernstein-Vazirani function from a string of 1s and 0s.
    """
    qc = QuantumCircuit(len(s) + 1)
    for index, bit in enumerate(reversed(s)):
        if bit == "1":
            qc.cx(index, len(s))
    return qc

display(bv_function("1000").draw("mpl"))
```

## Závěr

Doufáme, že prozkoumáním těchto jednoduchých příkladů jsme ti poskytli lepší intuici pro to, jak jsou kvantové počítače schopny využít superpozici, provázanost a interferenci k dosažení svého výkonu oproti klasickým počítačům.

Deutsch-Jozsův algoritmus má obrovský historický význam, protože byl prvním, který prokázal jakékoli zrychlení oproti klasickému algoritmu – šlo však pouze o polynomiální zrychlení. Deutsch-Jozsův algoritmus je jen začátkem příběhu.

Poté, co Bernstein a Vazirani použili algoritmus k vyřešení svého problému, ho využili jako základ pro složitější, rekurzivní problém nazvaný *problém rekurzivního Fourierova vzorkování*. Jejich řešení nabídlo super-polynomiální zrychlení oproti klasickým algoritmům. A ještě před Bernsteinem a Vazirani přišel Peter Shor se svým slavným algoritmem, který umožnil kvantovým počítačům faktorizovat velká čísla exponenciálně rychleji, než by dokázal jakýkoli klasický algoritmus. Tyto výsledky společně ukázaly vzrušující příslib budoucích kvantových počítačů a podnítily fyziky a inženýry, aby tuto budoucnost proměnili ve skutečnost.
