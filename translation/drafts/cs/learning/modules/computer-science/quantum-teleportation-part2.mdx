### Krok 1: Namapuj svůj problém na kvantový Circuit {#step-1-map-your-problem-to-a-quantum-circuit}

Veškerá matematika, kterou jsme udělali výše, popisovala krok 1. Nyní ji implementujeme a sestavíme náš kvantový Circuit v Qiskitu! Začneme vytvořením kvantového Circuit se třemi Qubitmi a propojíme dva Qubitmy Alice a Boba. Označíme je jako Qubitmy 1 a 2 a Qubit 0 si rezervujeme pro tajný stav.

```python
# Step 1: Map your problem to a quantum circuit

# Import some general packages
from qiskit import ClassicalRegister, QuantumCircuit, QuantumRegister
import numpy as np

# Define registers
secret = QuantumRegister(1, "Q")
Alice = QuantumRegister(1, "A")
Bob = QuantumRegister(1, "B")

cr = ClassicalRegister(3, "c")

qc = QuantumCircuit(secret, Alice, Bob, cr)

# We entangle Alice's and Bob's qubits as in our work above. We apply a Hadamard gate and then a CNOT gate.
# Note that the second argument in the CNOT gate is the target.
qc.h(Alice)
qc.cx(Alice, Bob)

# Inserting a barrier changes nothing about the logic. It just allows us to force gates to be positioned in "layers".
qc.barrier()

# Now we will use random variables to create the secret state. Don't worry about the "u" gate and the details.
np.random.seed(42)  # fixing seed for repeatability
theta = np.random.uniform(0.0, 1.0) * np.pi  # from 0 to pi
varphi = np.random.uniform(0.0, 2.0) * np.pi  # from 0 to 2*pi

# Assign the secret state to the qubit on the other side of Alice's (qubit 0), labeled Q
qc.u(theta, varphi, 0.0, secret)
qc.barrier()

# Now entangle Q and Alice's qubits as in the discussion above.
qc.cx(secret, Alice)
qc.h(secret)
qc.barrier()

# Now Alice measures her qubits, and stores the outcomes in the "classical registers" cr[]
qc.measure(Alice, cr[1])
qc.measure(secret, cr[0])

# Now we insert some conditional logic. If Alice measures Q in a "1" we need a Z gate, and if Alice measures A in a "1" we need an X gate (see the table).
with qc.if_test((cr[1], 1)):
    qc.x(Bob)
with qc.if_test((cr[0], 1)):
    qc.z(Bob)

qc.draw(output="mpl")
```

To je vše, co potřebujeme udělat, aby byl Alicin stav teleportován k Bobovi. Vzpomeň si ale, že když změříme kvantový stav $\alpha_0 |0\rangle+\alpha_1|1\rangle$, dostaneme buď $|0\rangle$, nebo $|1\rangle.$ Na konci celého procesu Bob Alicin tajný stav skutečně má, ale snadno to měřením ověřit nemůžeme. Aby nám měření řeklo, že jsme to udělali správně, musíme použít trik. Měli jsme operátor označený „U" pro „unitární", kterým jsme připravili Alicin tajný stav. Na konec Circuit můžeme aplikovat inverzi U. Pokud U zobrazilo Alicin stav $|0\rangle$ na $\alpha_0 |0\rangle+\alpha_1|1\rangle$, pak inverze U zobrazí Bobův stav $\alpha_0 |0\rangle+\alpha_1|1\rangle$ zpět na $|0\rangle.$ Tato závěrečná část by se normálně neprováděla, kdyby byl cíl pouze přenést kvantovou informaci. Děláme to jen proto, abychom si mohli ověřit výsledek.

```python
# Add the inverse of U and measure Bob's qubit.
qc.barrier()

qc.u(theta, varphi, 0.0, Bob).inverse()  # inverse of u(theta,varphi,0.0)
qc.measure(Bob, cr[2])  # add measurement gate

qc.draw(output="mpl")
```

Pokud jsme vše udělali správně, mělo by měření Bobova Qubitu dát stav $|0\rangle$. Tato měření jsou samozřejmě pravděpodobnostní. Takže i kdyby existovala jen malá šance naměřit Bobův Qubit ve stavu $|1\rangle$, jediné měření by mohlo skončit právě $|1\rangle.$ Abychom se ujistili, že pravděpodobnost $|0\rangle$ je skutečně vysoká, potřebujeme provést mnoho měření.

Nyní musíme kvantový Circuit „transpilovat". Tento proces zahrnuje mnoho dílčích kroků a je to fascinující téma. Jako příklad dílčího kroku: ne všechny kvantové počítače umí přímo implementovat všechny logické Gate z Qiskitu. Gate z našeho Circuit musíme přepsat pomocí Gate, které daný kvantový počítač umí provést. Tento i další procesy lze provést pomocí přednastavené pass manager. Nastavení ```optimization = 3``` (nejvyšší úroveň optimalizace) zajišťuje, že mapování z abstraktního kvantového Circuit na instrukce předávané kvantovému počítači je tak efektivní, jak jen naše předzpracování dokáže.

```python
# Step 2: Transpile
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

target = backend.target
pm = generate_preset_pass_manager(target=target, optimization_level=3)
qc_isa = pm.run(qc)
```

Sampler je primitiv navržený k vzorkování možných stavů výstupních z kvantového Circuit, sbírání statistik o tom, jaké stavy mohou být naměřeny a s jakou pravděpodobností. Zde importujeme Qiskit Runtime Sampler:

```python
# Load the Runtime primitive and session
from qiskit_ibm_runtime import SamplerV2 as Sampler

sampler = Sampler(mode=backend)
```

Ne všechny výpočty na kvantovém počítači lze rozumně simulovat na klasickém počítači. __Odpověď:__

Stavy $|000\rangle,$ $|001\rangle,$ $|010\rangle,$ $|011\rangle$ jsou všechny konzistentní s úspěšnou teleportací. Přidali jsme totiž Gate, která vrátí zpět počáteční přípravu tajného stavu. Pokud byl tajný stav úspěšně teleportován do Bobova Qubitu, tato dodatečná Gate by měla vrátit Bobův Qubit do stavu $|0\rangle$. Každý stav výše, ve kterém je Bobův Qubit (Qubit 0, naměřený do 0. složky klasického registru, tedy nejvyšší/nejpravější) ve stavu $|0\rangle$, tedy značí úspěch.

</details>

Tento graf zobrazuje všechny výsledky měření pro tři Qubitmy za 5 000 pokusů neboli „shots". Dříve jsme poukázali na to, že Alice bude měřit všechny možné stavy pro Qubitmy A a Q se stejnou pravděpodobností. Qubitmy 0–2 v Circuit jsme přiřadili Q, A a B v tomto pořadí. V little-endian zápisu je Bobův Qubit úplně vlevo/nejníže. Čtyři sloupce vlevo tedy odpovídají Bobovu Qubitu ve stavu $|0\rangle$, přičemž ostatní dva Qubitmy jsou ve všech možných kombinacích zhruba se stejnou pravděpodobností. Všimni si, že téměř všechna (obvykle \~95 %) měření dávají Bobův Qubit ve stavu $|0\rangle$, což znamená, že naše nastavení bylo úspěšné! Existuje jen hrstka shots (\~5 %), které daly Bobův Qubit ve stavu $|1\rangle$. Logicky by to být nemělo. Všechny moderní kvantové počítače však trpí šumem a chybami v mnohem větší míře než klasické počítače. A kvantová oprava chyb je stále rozvíjejícím se oborem.
