### Vyzkoušej příklady QFT {#try-some-example-qfts}

Pojďme si dál budovat intuici pro kvantovou Fourierovu transformaci tím, že vytvoříme stav ve výpočetní bázi a pak uvidíme, co se stane, když na něj aplikujeme QFT. Prozatím budeme QFT považovat za černou skříňku, kterou aplikujeme pomocí `QFTGate` z [knihovny obvodů Qiskit.](/guides/circuit-library) Později nahlédneme pod pokličku a uvidíme, jak je implementována.

Začneme načtením potřebných balíčků a výběrem zařízení, na kterém spustíme náš Circuit:

```python
import numpy as np
from qiskit import QuantumCircuit
from qiskit.visualization import plot_histogram
from qiskit.circuit.library import QFTGate
```

```python
# Load the Qiskit Runtime service
from qiskit_ibm_runtime import QiskitRuntimeService

# Load the Runtime primitive and session
from qiskit_ibm_runtime import SamplerV2 as Sampler

service = QiskitRuntimeService()

# Use the least busy backend
# backend = service.least_busy(operational=True, simulator=False, min_num_qubits = 127)
backend = service.backend("ibm_pinguino2")

print(backend.name)
```

```text
ibm_pinguino2
```

Pokud nemáš na svém účtu dostupný čas nebo chceš z jakéhokoli důvodu použít simulátor, můžeš spustit níže uvedenou buňku a nastavit simulátor, který bude napodobovat kvantové zařízení, které jsme vybrali výše:

```python
# Load the backend sampler
from qiskit.primitives import BackendSamplerV2

# Load the Aer simulator and generate a noise model based on the currently-selected backend.
from qiskit_aer import AerSimulator
from qiskit_aer.noise import NoiseModel

noise_model = NoiseModel.from_backend(backend)

# Define a simulator using Aer, and use it in Sampler.
backend_sim = AerSimulator(noise_model=noise_model)
sampler_sim = BackendSamplerV2(backend=backend_sim)
```

```python
# Alternatively, load a fake backend with generic properties and define a simulator.
from qiskit.providers.fake_provider import GenericBackendV2

backend_gen = GenericBackendV2(num_qubits=18)
sampler_gen = BackendSamplerV2(backend=backend_gen)
```

#### Jeden stav výpočetní báze {#single-computational-basis-state}

Nejprve zkusme transformovat jeden stav výpočetní báze. Začneme vytvořením náhodného výpočetního stavu:

```python
# Step 1: Map

qubits = 4
N = 2**qubits

qc = QuantumCircuit(qubits)

# flip state of random qubits to put in a random single computational basis state
for i in range(1, qubits):
    if np.random.randint(0, 2):
        qc.x(i)

# make a copy of the above circuit. (to be used when we apply the QFT in next part)
qc_qft = qc.copy()

qc.measure_all()
qc.draw("mpl")
```

Teď pojďme tento stav Fourierovsky transformovat pomocí `QFTGate`:

```python

qc_qft.measure_all()
qc_qft.draw("mpl")
```

Jak vidíš, obsazenosti jednotlivých stavů měříme víceméně rovnoměrně, s výjimkou experimentálního a statistického šumu. Pokud tedy vezmete QFT jednoho stavu výpočetní báze, výsledkem je rovnoměrná superpozice všech stavů. Pokud znáš Fourierovy transformace, tě to asi nepřekvapí. Jeden základní princip, který nám může pomoci vybudovat intuitivní spojení mezi funkcí a její Fourierovou transformací, je ten, že šířka funkce je nepřímo úměrná šířce její Fourierovy transformace. Něco, co je tedy velmi lokalizované v čase – například velmi krátký impuls – bude k vytvoření tohoto impulsu potřebovat široké spektrum frekvencí. Takový signál bude ve Fourierově prostoru velmi široký.

Tento fakt je ve skutečnosti spojený s kvantovou neurčitostí! Heisenbergův princip neurčitosti se typicky vyjadřuje jako $\Delta x \Delta p \ge \hbar / 2 $. Pokud je tedy neurčitost v $x$ ($\Delta x$) malá, neurčitost hybnosti ($\Delta p$) musí být velká, a naopak. Ukazuje se, že transformace z poziční báze $x$ do hybnostní báze $p$ se provádí právě Fourierovou transformací.

Poznámka: Měj na paměti, že měříme obsazenosti jednotlivých stavů báze, takže ztrácíme informaci o relativních fázích mezi různými částmi superpozice. Ačkoli tedy QFT libovolného jednoho stavu výpočetní báze vždy dá stejné rovnoměrné rozložení obsazeností přes všechny stavy báze, *fáze* nutně stejné být nemusí.
