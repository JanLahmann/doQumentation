#### Dva stavy výpočetní báze

Podívejme se teď, co se stane, když připravíme superpozici stavů výpočetní báze. Jak myslíš, že bude Fourierova transformace v tomto případě vypadat?

Zvolme superpozici:

$$|\psi\rangle = \frac{1}{\sqrt{2}} (|0\rangle + |N/2\rangle) = \frac{1}{\sqrt{2}} (|000...0\rangle + |100...0\rangle)$$

```python
# Step 1: Map
qubits = 4
N = 2**qubits

qc = QuantumCircuit(qubits)

# To make this state, we just need to apply a Hadamard to the last qubit

qc.h(qubits - 1)

qc_qft = qc.copy()

qc.measure_all()

qc.draw("mpl")
```

Tento výsledek tě možná trochu překvapí. Zdá se, že QFT stavu $|\psi\rangle = \frac{1}{\sqrt{2}} (|0\rangle + |N/2\rangle)$ je superpozicí všech sudých stavů báze. Pokud si ale vzpomeneme na naši vizualizaci každého bázového stavu $|\phi_y\rangle$ a na to, jak se fáze každé složky otáčí $2\pi$ $y$-krát kolem dokola, může nám důvod tohoto výsledku přijít jasný.

Pomocí výše uvedené nápovědy vysvětli, proč je výsledek, který jsme dostali pro QFT stavu $|\psi\rangle = \frac{1}{\sqrt{2}} (|0\rangle + |N/2\rangle)$, očekávaný.

__Odpověď:__

Původní stav má relativní fázi 0 (nebo celočíselný násobek $2\pi$) mezi oběma částmi superpozice. Víme tedy, že tento stav má Fourierovy složky, jejichž fáze se shodují stejným způsobem: ty, které mají nulový fázový posun mezi členem |0000> a členem |1000>. Každý Fourierův bázový stav $|\phi_y\rangle$ se skládá z členů, jejichž fáze se akumuluje rychlostí $2\pi y/N$, což znamená, že v obvyklém pořadí má každý člen superpozice o $2\pi y/N$ větší fázi než člen předchozí. Takže v polovině $N/2$ chceme, aby fáze $2\pi y/N * N/2$ byla celočíselným násobkem $2\pi$. To nastane, když je $y$ sudé.

</details>
