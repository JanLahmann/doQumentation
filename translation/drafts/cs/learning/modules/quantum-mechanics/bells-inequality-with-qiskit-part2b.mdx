### Krok 1: Převod našich výsledků do kvantového Circuit {#step-1-mapping-our-results-to-a-quantum-circuit}

```python
# Začneme importem qiskitu a vizualizačního modulu, abychom mohli vykreslit histogram výsledků.

from qiskit.visualization import plot_histogram
```

Vzpomeň si, že 1/3 času bude Rihannina zvolená osa $2\pi/3$ radiánů od Lucasovy, 1/3 času bude $4 \pi/3$ radiánů od Lucasovy a 1/3 času si vyberou stejnou osu. Potřebujeme tedy sestavit 3 kvantové Circuit pro tyto 3 případy a výsledky sečíst. První podrobně vysvětlíme, zbývající dva pouze uvedeme.

```python
# Začneme deklarací prvního kvantového Circuit a přidáme mu dva Qubity (první "2") a dva klasické bity pro uložení výstupů (druhé "2")
# Definice registrů
from qiskit import ClassicalRegister, QuantumRegister

qr = QuantumRegister(2, "q")
cr = ClassicalRegister(2, "c")
qc1 = QuantumCircuit(qr, cr)

# Z naší analýzy výše víme, že potřebujeme Gate X působící na každý z Qubitů (L a R)
qc1.x([0, 1])
# Potřebujeme Hadamardův Gate působící na Lucasův Qubit, který označujeme jako 0. Qubit.
qc1.h(0)
# Gate controlled-NOT používá 0. Qubit (Lucasův) jako řídící a 1. Qubit (Rihannin) jako cíl.
qc1.cx(0, 1)
# Rotační Gate působí na 1. Qubit (Rihannin) s argumentem -2 pi/3
qc1.ry(-2 * pi / 3, 1)
# Nakonec chceme změřit všechny Qubity v Circuit, abychom získali pravděpodobnosti měření, a výsledky uložit do klasických bitů.
qc1.measure([0, 1], [0, 1])
# Nyní můžeme nakreslit první ze tří Circuit, které nám ověří Bellovu nerovnost.
qc1.draw(output="mpl")
```

Níže uvedený kód rychle sestaví všechny tři Circuit přehlednějším způsobem. Všimni si, že jediný rozdíl mezi třemi Circuit je to, o kolik otočíme dva Qubity kolem osy $y$.

```python
qcs = [QuantumCircuit(2, 2), QuantumCircuit(2, 2), QuantumCircuit(2, 2)]
for i in range(0, len(qcs)):
    qcs[i].x([0, 1])
    qcs[i].h(0)
    qcs[i].cx(0, 1)

qcs[0].ry(-2 * pi / 3, 1)
qcs[1].ry(-4 * pi / 3, 1)
qcs[2].ry(-2 * pi / 3, 1)
qcs[2].ry(-4 * pi / 3, 1)

for i in range(0, len(qcs)):
    qcs[i].barrier()
    qcs[i].measure([0, 1], [0, 1])

counts_list = [None] * len(qcs)
```

```python
qcs[0].draw(output="mpl")
```

Nyní použijeme qiskitový primitiv zvaný StatevectorSampler. Sampler je primitiv navržený tak, aby vzorkoval všechny možné stavy systému a vracel pravděpodobnosti (nebo v některých případech kvazipravděpodobnosti) získání každého stavu. Můžeme zadat počet „shots" a prohlédnout si „counts" pro každý stav.

```python
from qiskit.primitives import StatevectorSampler

sampler = StatevectorSampler()

# Spustíme job, který vrátí shots pro všechny sady 100 hodnot parametrů.
for i in range(0, len(qcs)):
    pub = qcs[i]
    job = sampler.run([pub], shots=10000)
    # Extrahujeme výsledek pro 0. PUB (tento příklad má pouze jeden PUB).
    result = job.result()
    data_pub = result[0].data
    counts = data_pub.c.get_counts()
    counts_list[i] = counts
#    plot_histogram(counts)
```

Podíváme-li se na counts z každého Circuit, vidíme, že dva z nich byly v podstatě identické a třetí se výrazně lišil.

```python
plot_histogram(counts_list)
```

Sestavme seznam možných výsledků a sečteme všechny counts každého stavu ze všech tří Circuit, abychom získali celkové pravděpodobnosti.

```python
outcomes = ("00", "01", "10", "11")

# Zde převedeme hodnoty "None" na 0, abychom mohli sčítat.

for i in range(0, len(qcs)):
    for j in range(0, len(outcomes)):
        if counts_list[i].get(outcomes[j]) is None:
            counts_list[i].update({outcomes[j]: 0})

# Zde vytvoříme slovník, který obsahuje všechny výsledky a sčítá jejich výskyty v každém z Circuit.

total_counts = {}
for i in range(0, len(outcomes)):
    total_counts[outcomes[i]] = sum(
        counts_list[j].get(outcomes[i]) for j in range(0, len(qcs))
    )
```

Nyní můžeme vypsat celkové counts pro každý výsledek a vykreslit histogram.

```python
print(total_counts)
plot_histogram(total_counts)
```
