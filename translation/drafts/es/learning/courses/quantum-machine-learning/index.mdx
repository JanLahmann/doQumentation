# Descripción general

!["Imagen de portada"](/learning/images/courses/quantum-machine-learning/hero.svg)

Este curso explora el emocionante papel que la computación cuántica puede desempeñar en el campo del Machine Learning. El curso ofrece un repaso de los enfoques clásicos de Machine Learning, aunque asume cierta familiaridad con estos métodos. El enfoque principal es cómo la computación cuántica puede integrarse en los flujos de trabajo existentes de Machine Learning. Los temas incluyen codificación de datos, kernels cuánticos y circuitos cuánticos variacionales.
