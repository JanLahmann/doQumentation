# Descripción general

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Este curso está dirigido a personas que ya tienen experiencia en computación cuántica y desean aprender a escribir algoritmos variacionales con Qiskit Runtime Primitives. También es adecuado para profesionales de la computación cuántica que deseen ampliar sus conocimientos y habilidades en el área de algoritmos variacionales.

## Examen {#exam}

Para obtener tu insignia de Variational algorithm design, realiza el examen en IBM&reg; Training. Este examen está diseñado para realizarse después de leer las lecciones de este curso. Tras aprobar el examen, Credly te notificará que has obtenido una insignia.

[Examen](/learning/courses/variational-algorithm-design/exam)
