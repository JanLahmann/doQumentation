En lecciones anteriores de este curso, hemos conocido algunos ejemplos de códigos correctores de errores cuánticos, incluidos el código de Shor de 9 qubits, el código de Steane de 7 qubits y el código de 5 qubits.
Estos códigos son sin duda interesantes y representan un punto de partida natural para explorar la corrección de errores cuánticos, pero un problema con ellos es que solo pueden tolerar una tasa de errores muy baja.
Corregir un error en cinco, siete o nueve qubits no está mal, pero según todas las previsiones necesitaremos tolerar muchos más errores para hacer realidad la computación cuántica a gran escala.

En esta lección, daremos un primer vistazo a algunas construcciones más sofisticadas de códigos correctores de errores cuánticos, incluidos códigos que pueden tolerar una tasa de errores significativamente mayor que los vistos hasta ahora y que se consideran candidatos prometedores para la corrección de errores cuánticos práctica.

Comenzamos con una clase de códigos correctores de errores cuánticos conocida como *códigos CSS*, nombrados en honor a Robert Calderbank, Peter Shor y Andrew Steane, quienes los descubrieron por primera vez.
La construcción de códigos CSS permite combinar ciertos *pares* de códigos correctores de errores clásicos en un único código corrector de errores cuánticos.

La segunda parte de la lección trata sobre un código conocido como el *código tórico*.
Este es un ejemplo fundamental (y realmente hermoso) de un código corrector de errores cuánticos que puede tolerar tasas de error relativamente altas.
De hecho, el código tórico no es un solo ejemplo, sino una familia infinita de códigos — uno para cada entero positivo mayor que uno.

Finalmente, en la última parte de la lección discutiremos brevemente otras familias de códigos cuánticos, incluyendo los *códigos de superficie* (que están estrechamente relacionados con el código tórico) y los *códigos de color*.
