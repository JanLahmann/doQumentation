# Guía de migración a Qiskit 1.0 {#qiskit-10-migration-guide}

Qiskit 1.0 introduce nuevas garantías de estabilidad y mejoras de rendimiento, pero
rompe la compatibilidad con todas las versiones anteriores de Qiskit. Consulta las
siguientes guías para actualizar tu entorno y tu código para que funcionen con Qiskit v1.0.

- [Instala el nuevo paquete Qiskit v1.0](/guides/qiskit-1.0-installation) para actualizar entornos de Python y dependencias de proyectos.
- [Cambios en las funcionalidades de Qiskit v1.0](/guides/qiskit-1.0-features) para actualizar tu código.
- [Cambios de paquete incompatibles de Qiskit v1.0](/guides/metapackage-migration) para entender la nueva y antigua estructura de paquetes.

Lee más sobre los beneficios de Qiskit v1.0 en el [blog de IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
