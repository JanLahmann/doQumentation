# Entornos de laboratorio en línea {#online-lab-environments}

Además de descargar e instalar Qiskit para trabajar en tu máquina local, existen varias opciones disponibles para quienes prefieren trabajar en un entorno de desarrollo en línea.

Varios proveedores ofrecen entornos Jupyter en línea con Qiskit preinstalado para un inicio rápido y conveniente. Entre los proveedores recomendados se encuentran el proveedor de servicios en la nube OVHcloud y la plataforma de desarrollo de software cuántico en la nube qBraid.

## Entornos de notebooks recomendados con Qiskit preinstalado {#recommended-notebook-environments-with-qiskit-preinstalled}

Otros entornos de Jupyter notebooks en línea, como IBM Watson&reg; Studio, Google Colab y Microsoft Azure Machine Learning Studio, también pueden usarse con Qiskit siguiendo las instrucciones correspondientes.

### Notebooks de IA de OVHcloud {#ovhcloud-ai-notebooks}

Los usuarios de OVHcloud Public Cloud y OVHcloud Startup pueden utilizar sus notebooks cuánticos de OVHcloud para acceder a Qiskit y enviar trabajos a las QPUs de IBM&reg; con su cuenta de IBM Quantum&reg; o IBM Cloud&reg;. Para obtener más información, visita la página de [documentación y tutoriales de OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Los notebooks de IA de OVHcloud no están disponibles en algunas regiones, como los Estados Unidos. Además, las interfaces de usuario pueden verse diferentes según la región.
</Admonition>

Sigue estos pasos para comenzar:

1. [Crea una cuenta de IBM Cloud.](./cloud-setup)
1. [Configura una cuenta de OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Crea un proyecto de OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Sigue los pasos en [Lanza tu primer AI Notebook,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) seleccionando **Qiskit** como framework.

### qBraid Lab {#qbraid-lab}

Los usuarios de Qiskit pueden usar los entornos Python preconfigurados de qBraid siguiendo estas instrucciones. Para más información, consulta la [documentación de qBraid.](https://docs.qbraid.com/home/introduction)

1. [Crea una cuenta gratuita en qBraid.](https://docs.qbraid.com/home/account)
3. [Crea un entorno Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Inicia qBraid Lab, haz clic en el ícono **ENVS** a la derecha, haz clic en **+ ADD** en la parte superior de la nueva barra lateral y busca Qiskit. Haz clic en la versión que deseas usar y luego en **Install**.
4. Crea un Jupyter notebook haciendo clic en **Python 3 [Qiskit]** en la sección Notebook.
4. Puedes instalar cualquier paquete adicional de Python (como Qiskit v1.2) en una celda del notebook ejecutando `%pip install [package_name].` Para más información, consulta el tema [Pip (magic) commands](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Entornos de notebooks adicionales para usuarios de Qiskit {#additional-notebook-environments-for-qiskit-users}

Para plataformas sin Qiskit preconfigurado, las instrucciones de configuración son similares, salvo que debes instalar Qiskit manualmente. Tendrás que crear una cuenta, iniciar un nuevo proyecto (o notebook o espacio de trabajo, según la plataforma) e instalar Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Configura una prueba gratuita de Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Crea un proyecto de Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Para instalar los paquetes de Qiskit y las dependencias necesarias, abre un nuevo notebook y ejecuta el siguiente comando:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Crea una cuenta de Google.](https://support.google.com/accounts/answer/27441?hl)
2. Ve a la [página de inicio de Google Colab](https://colab.research.google.com/) e inicia sesión con tu cuenta de Google.
3. [Crea un notebook de Colab](https://colab.research.google.com/#create=true) e instala Qiskit ejecutando el siguiente comando:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Crea una cuenta de Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Crea un espacio de trabajo.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Crea un Jupyter notebook en tu espacio de trabajo.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Desde el notebook, ejecuta este comando para instalar Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
