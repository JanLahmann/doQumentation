# Servidores MCP de Qiskit {#qiskit-mcp-servers}

Los servidores MCP de Qiskit son una colección de servidores del [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) que integran capacidades de computación cuántica en sistemas de IA. Estos servidores permiten que asistentes de IA, modelos de lenguaje grande (LLMs) y agentes accedan a los servicios de IBM Quantum&reg; y a las bibliotecas de Qiskit.

<Admonition type="note">
    Este es un proyecto de código abierto. Para obtener la información más actualizada, cobertura detallada de la API y configuraciones avanzadas, consulta el [repositorio de GitHub de los servidores MCP de Qiskit.](https://github.com/Qiskit/mcp-servers)
</Admonition>

## ¿Cuándo debo usar los servidores MCP de Qiskit? {#when-should-i-use-qiskit-mcp-servers}

Los servidores MCP de Qiskit permiten que los asistentes de IA te ayuden con tareas de computación cuántica, como las siguientes:

- Crear, manipular y transpilar circuitos cuánticos
- Responder preguntas de computación cuántica directamente desde fuentes confiables
- Generar código cuántico con conciencia contextual
- Optimizar circuitos mediante transpilación potenciada por IA
- Conectarse automáticamente a hardware cuántico real
- Analizar y seleccionar los backends cuánticos óptimos
- Ejecutar circuitos cuánticos y hacer seguimiento del estado de los trabajos

## ¿Qué está incluido? {#what-is-included}

Los siguientes servidores MCP están disponibles:

### Servidor MCP de Qiskit {#qiskit-mcp-server}

Proporciona capacidades de computación cuántica fundamentales para el desarrollo local. Este servidor permite que los asistentes de IA creen, manipulen y transpilen circuitos cuánticos, así como los serialicen en formatos como OpenQASM 3 y QPY.

### Servidor MCP del Asistente de Código de Qiskit {#qiskit-code-assistant-mcp-server}

Proporciona autocompletado inteligente de código cuántico y asistencia a través del [Asistente de Código de Qiskit](/guides/qiskit-code-assistant). Este servidor integra las capacidades del Asistente de Código de Qiskit directamente en herramientas de IA compatibles con MCP.

Al igual que con cualquier otra función del Asistente de Código de Qiskit, este servidor MCP solo funciona para usuarios del Plan Premium de IBM Quantum registrados en [IBM Quantum Platform.](https://quantum.cloud.ibm.com/)

### Servidor MCP de Qiskit Runtime {#qiskit-runtime-mcp-server}

Proporciona acceso a los servicios en la nube de IBM Quantum a través de Qiskit Runtime. Esto permite que los asistentes de IA interactúen con backends cuánticos, envíen trabajos y recuperen resultados.

### Servidor MCP del Transpilador IBM de Qiskit {#qiskit-ibm-transpiler-mcp-server}

Proporciona optimización de circuitos potenciada por IA a través de los [pases de transpilador potenciados por IA](/guides/ai-transpiler-passes). Este servidor permite que los asistentes de IA optimicen circuitos cuánticos usando algoritmos avanzados de enrutamiento y optimización.

### Servidores de la comunidad {#community-servers}

Los siguientes servidores MCP aportados por la comunidad también están disponibles:

#### Servidor MCP de Qiskit Gym {#qiskit-gym-mcp-server}

Proporciona capacidades de aprendizaje por refuerzo para la síntesis de circuitos cuánticos usando la biblioteca [qiskit-gym](https://github.com/AI4quantum/qiskit-gym). Este servidor permite que los asistentes de IA utilicen técnicas de optimización basadas en RL para la transpilación y síntesis de circuitos cuánticos.

## Instalar los servidores MCP de Qiskit {#install-qiskit-mcp-servers}

Para usar los servidores MCP de Qiskit, necesitas:

- Python 3.10 o posterior (se recomienda 3.11+)
- Una cuenta de IBM Quantum y un token de API (consulta [Configura tu cuenta de IBM Cloud&reg;](/guides/cloud-setup) para ver las instrucciones)

Instala todos los servidores MCP de Qiskit disponibles ejecutando el siguiente comando desde una terminal:

```bash
pip install qiskit-mcp-servers[all]
```

También puedes instalar servidores MCP individuales:

```bash
pip install qiskit-mcp-server
pip install qiskit-code-assistant-mcp-server
pip install qiskit-ibm-runtime-mcp-server
pip install qiskit-ibm-transpiler-mcp-server
pip install qiskit-gym-mcp-server
```

## Configurar los servidores MCP de Qiskit {#configure-qiskit-mcp-servers}

### Establecer variables de entorno {#set-environment-variables}

Configura las variables de entorno necesarias para la autenticación. El servidor MCP de Qiskit no requiere autenticación para uso local.

Para los servidores MCP de Qiskit Runtime, Asistente de Código de Qiskit o Transpilador IBM de Qiskit:

```bash
export QISKIT_IBM_TOKEN="<your_token>"
```

Puedes encontrar tu token de API en la [configuración de tu cuenta de IBM Quantum.](https://quantum.cloud.ibm.com/)

### Configurar el cliente MCP {#configure-the-mcp-client}

Puedes configurar un cliente compatible con MCP (como Claude Desktop, Cursor u otras herramientas de IA) para usar los servidores MCP de Qiskit. La configuración exacta depende de tu cliente. Por ejemplo, un archivo de configuración de cliente MCP podría verse así:

```json
{
  "mcpServers": {
    "qiskit": {
      "command": "qiskit-mcp-server"
    },
    "qiskit-ibm-runtime": {
      "command": "qiskit-ibm-runtime-mcp-server"
    },
    "qiskit-code-assistant": {
      "command": "qiskit-code-assistant-mcp-server"
    },
    "qiskit-ibm-transpiler": {
      "command": "qiskit-ibm-transpiler-mcp-server"
    },
    "qiskit-gym": {
      "command": "qiskit-gym-mcp-server"
    }
  }
}
```

Consulta la documentación de tu cliente MCP para obtener instrucciones de configuración específicas.

## Probar los servidores MCP {#test-the-mcp-servers}

Puedes probar los servidores MCP de forma interactiva usando el MCP Inspector:

```bash
# Probar el servidor MCP de Qiskit
npx @modelcontextprotocol/inspector qiskit-mcp-server

# Probar el servidor MCP de Qiskit Runtime
npx @modelcontextprotocol/inspector qiskit-ibm-runtime-mcp-server

# Probar el servidor MCP del Asistente de Código de Qiskit
npx @modelcontextprotocol/inspector qiskit-code-assistant-mcp-server

# Probar el servidor MCP del Transpilador IBM de Qiskit
npx @modelcontextprotocol/inspector qiskit-ibm-transpiler-mcp-server

# Probar el servidor MCP de Qiskit Gym
npx @modelcontextprotocol/inspector qiskit-gym-mcp-server
```
