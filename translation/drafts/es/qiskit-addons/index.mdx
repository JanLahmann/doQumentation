# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) son paquetes de extensión oficiales que proporcionan algoritmos y técnicas especializadas de computación cuántica. Cada addon se centra en un área específica e incluye tutoriales con código ejecutable.

## Addons Disponibles {#available-addons}

| Addon | Paquete | Descripción |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Reduce el ancho y la profundidad del Circuit cortando Gates y cables |
| **SQD** | `qiskit-addon-sqd` | Diagonalización Cuántica Basada en Muestras para la estimación del estado fundamental |
| **OBP** | `qiskit-addon-obp` | Retropropagación de Operadores para reducir la profundidad del Circuit |
| **MPF** | `qiskit-addon-mpf` | Fórmulas Multi-Producto para simulación mejorada del Hamiltoniano |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Compilación Cuántica Aproximada con redes tensoriales |
| **PNA** | `qiskit-addon-pna` | Absorción de Ruido Propagado para la mitigación de errores |
| **SLC** | `qiskit-addon-slc` | Conos de Luz Sombreados para reducir la sobrecarga de muestreo PEC |

Todos los tutoriales pueden ejecutarse directamente en tu navegador usando Binder, o en tu propio servidor Jupyter.

Para la documentación de la API, visita los repositorios individuales de cada addon en [GitHub](https://github.com/Qiskit?q=qiskit-addon).
