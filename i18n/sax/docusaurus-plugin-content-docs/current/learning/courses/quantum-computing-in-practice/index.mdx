# Ieberblick {#overview}

!["Hero image"](/learning/images/courses/quantum-computing-in-practice/hero.avif)

Dees Gurs dreht sich um de heidichen Quantencomputer un wie mer se vollständich nutzen gann. Er behandelt realistiche Anwendungsfälle fier's Quantencomputing wie ach de besten Meedoden fier's Betreeben un Experimentier'n mit Quantenprozessorn, de 100 oder mehr Qubits hab'm.

## Empfohlne Vorkenntnisse {#recommended-background}

Damitt de meeschd aus'm Gurs rausholng, empfehl'n mer Grundkenndnisse von de Quanteninformatschjoon un Quantum-Circuits — auf'm Niveau, wie's im Gurs [Grundlagen von de Quanteninformatschjoon](/learning/courses/basics-of-quantum-information) erglert werd.

Weitere empfohln Quellen sin de folch'nd'n:

- Chris Bernhardt. [Quantum Computing for Everyone.](https://mitpress.mit.edu/9780262539531/quantum-computing-for-everyone/)
- Thomas Wong. [Introduction to Classical and Quantum Computing.](https://www.thomaswong.net/introduction-to-classical-and-quantum-computing-1e4p.pdf)
