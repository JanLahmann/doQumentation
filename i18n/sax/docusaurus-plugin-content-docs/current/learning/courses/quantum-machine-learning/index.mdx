# Ieberblick {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Dieser Gurs unnersucht, wie Quantencomputing e spannende Rolle beim Maschinellernen spieln gann. De Gurs wird e bißl die klassischn Maschinenlernen-Ansätz wiederholn, aber setzt aa e gewisse Vertraadheit mit diesen Methoden voraus. De Hauptaugenmärg liegt draff, wie Quantencomputing in bestehende Maschinenlernen-Arbeitsabläufe eengebundn werd'n gann. Themen sin Datenkodierung, Quantekerne un varjationale Quantenschaltkrees.

## Priefung {#exam}

Um eier Abzeichen fier Quantenmaschinellernen zu verdien'n, macht de Priefung bei IBM&reg; Training. Diese Priefung is gedacht, nachdem ihr de Lektschjoon'n in diesem Gurs durchgelesen habbt. Nachdem ihr bestanden habbt, werd ihr von Credly benachrichticht, dass ihr e Abzeichen verdient habbt.
