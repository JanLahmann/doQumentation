# Ieberbligg {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Desser Gurs is fier Leut jedacht, de schon e bissl Erfahrung mit'm Quantencomputing hamm un lernm woll'n, wie se varjatschjoonelle Algorithm'n mit de Qiskit-Runtime-Primitiven schreib'n gennt. Er eignd sich aa fier Quantencomputing-Fachleut, de ihr Wiss'n un ihre Fähichkeet'n uff em Gebiet de varjatschjoonell'n Algorithm'n erweitern wolln.

## Priefung {#exam}

Damitt ihr eiern Badzsch fier de varjatschjoonelle Algorithmusentwicklung griechd, leggt de Priefung beim IBM&reg; Training ab. De Priefung is so jedacht, dass se nach'm Durchles'n de Lektsjoonen in desem Gurs abgelecht werd. Wenn ihr de Priefung bestand'n habbt, wird eich von Credly bescheid gesacht, dass ihr en Badzsch verdient habbt.
