# Qiskit Addons {#qiskit-addons}

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon)는 특화된 양자 컴퓨팅 알고리즘과 기법을 제공하는 공식 확장 패키지입니다. 각 애드온은 특정 분야에 초점을 맞추며 실행 가능한 코드가 포함된 튜토리얼을 제공합니다.

## 사용 가능한 Addons {#available-addons}

| Addon | 패키지 | 설명 |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | 게이트와 와이어를 커팅하여 회로 폭과 깊이를 줄입니다 |
| **SQD** | `qiskit-addon-sqd` | 기저 상태 추정을 위한 샘플 기반 양자 대각화 |
| **OBP** | `qiskit-addon-obp` | 회로 깊이를 줄이기 위한 연산자 역전파 |
| **MPF** | `qiskit-addon-mpf` | 향상된 해밀토니안 시뮬레이션을 위한 다중 곱 공식 |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | 텐서 네트워크를 이용한 근사 양자 컴파일 |
| **PNA** | `qiskit-addon-pna` | 오류 완화를 위한 전파 잡음 흡수 |
| **SLC** | `qiskit-addon-slc` | PEC 샘플링 오버헤드를 줄이기 위한 Shaded Lightcones |

모든 튜토리얼은 Binder를 사용하여 브라우저에서 직접 실행하거나 자체 Jupyter 서버에서 실행할 수 있습니다.

API 문서는 [GitHub](https://github.com/Qiskit?q=qiskit-addon)의 개별 애드온 저장소를 방문하세요.
