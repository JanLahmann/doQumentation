# Qiskit 1.0 마이그레이션 가이드 {#qiskit-10-migration-guide}

Qiskit 1.0은 새로운 안정성 보장과 성능 개선을 제공하지만, 이전 모든 버전의 Qiskit과의 호환성을 지원하지 않습니다. 환경과 코드를 Qiskit v1.0에서 작동하도록 업데이트하는 방법은 다음 가이드를 참조해 주세요.

- [새 Qiskit v1.0 패키지 설치](/guides/qiskit-1.0-installation): Python 환경 및 프로젝트 의존성 업그레이드 방법입니다.
- [Qiskit v1.0 기능 변경 사항](/guides/qiskit-1.0-features): 코드 업데이트 방법입니다.
- [Qiskit v1.0의 패키지 주요 변경 사항](/guides/metapackage-migration): 새로운 패키지 구조와 기존 패키지 구조를 이해하는 방법입니다.

Qiskit v1.0의 이점에 대한 자세한 내용은 [IBM&reg; 블로그](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary)에서 확인하실 수 있습니다.
