# Open Plan에서 업그레이드하기 {#upgrade-from-the-open-plan}

이 가이드는 IBM Quantum&reg; Open Plan에서 IBM Cloud&reg; Pay-As-You-Go Plan으로 업그레이드하려는 사용자를 위한 안내입니다. 클라우드 계정 설정, 서비스 인스턴스 생성, 사용자 관리에 대한 지침은 아래 단계를 따르세요.

처음으로 서비스를 설정하는 개인 사용자이거나 관리자로부터 인스턴스에 초대받은 경우에는 대신 [Qiskit 설치](./install-qiskit)의 단계를 따르세요.

## 시작하기 전에 {#before-you-begin}

필요한 경우, [IBM Cloud 사용 설정](./cloud-setup#cloud) 섹션의 단계를 따라 다음 작업을 완료하세요:

- IBM Cloud 계정에 가입합니다.
- IBM Quantum 인스턴스를 생성합니다.

<span id="update-cloud-account"></span>
## IBM Cloud 계정 업그레이드하기 {#upgrade-your-ibm-cloud-account}

이미 IBM Cloud Pay-as-you-go 계정이 있다면 이 단계를 건너뛰세요. 새로 만든 모든 IBM Cloud 계정은 Pay-as-you-go로 설정됩니다.

다음의 경우에는 IBM Cloud 계정을 업그레이드해야 합니다:

- Trial 계정을 보유하고 있는 경우.
- 2021년 10월 25일 이전에 생성한 Lite 계정(사용 중단됨)을 보유하고 있는 경우.

업그레이드 후에도 Lite 계정으로 생성한 인스턴스를 계속 사용할 수 있습니다. Pay-As-You-Go 계정으로 업그레이드하려면 다음 단계를 완료하세요:

1. IBM Cloud 콘솔에서 Manage > [Account](https://cloud.ibm.com/account)로 이동합니다.
2. **Account settings**를 선택한 후 **Add credit card**를 클릭합니다.
3. 결제 정보를 입력하고 **Next**를 클릭하여 정보를 제출합니다.
4. 결제 정보가 처리되면 계정이 업그레이드됩니다.
   <Admonition type="note">
   Pay-As-You-Go Plan 인스턴스를 생성하고 이를 통해 QPU에 워크로드를 전송하지 않는 한 요금이 청구되지 않습니다.
   </Admonition>

## Pay-As-You-Go Plan 이용하기 {#access-your-pay-as-you-go-plan}

IBM Cloud Pay-As-You-Go Plan을 사용하려면 새 [인스턴스](/guides/instances)를 생성하고 Pay-As-You-Go Plan에 할당하세요.  <ins>이 인스턴스를 지정하여 QPU에 전송되는 모든 작업에는 비용이 발생합니다.</ins>  비용 없이 QPU를 사용하려면 Open Plan 또는 다른 무료 플랜에 할당된 인스턴스를 사용해야 합니다(보유한 경우).

Pay-As-You-Go Plan 사용 시 인스턴스에 [비용 한도](/guides/manage-cost#cost-limit)를 설정하고, 비용을 관리하는 다른 방법도 알아보는 것을 권장합니다.
