# Qiskit 애드온 {#qiskit-addons}

Qiskit 애드온은 유틸리티 규모에서 알고리즘 발견을 가능하게 하는 연구 기능의 모음입니다. 이 기능들은 양자 알고리즘을 생성하고 실행하기 위한 Qiskit의 고성능 도구 기반 위에 구축됩니다. 새로운 양자 알고리즘을 확장하거나 설계하기 위해 [워크플로우](/guides/intro-to-patterns)에 연결할 수 있는 모듈형 소프트웨어 구성 요소로 제공됩니다.

## 매핑을 위한 애드온 {#addons-for-mapping}

### 텐서 네트워크를 이용한 근사 양자 컴파일 {#approximate-quantum-compilation-with-tensor-networks}

텐서 네트워크를 이용한 근사 양자 컴파일(AQC-Tensor)은 깊이를 줄이면서 높은 충실도의 Circuit을 구성할 수 있게 합니다.

- [GitHub 저장소](https://github.com/Qiskit/qiskit-addon-aqc-tensor)를 방문하세요.
- [문서](/guides/qiskit-addons-aqc)를 읽어보세요.
- AQC를 사용하여 [Trotterized 시간 진화를 개선하는](/tutorials/approximate-quantum-compilation-for-time-evolution) 튜토리얼을 읽어보세요.

### 다중 곱 공식 {#multi-product-formulas}

다중 곱 공식(MPF)은 여러 Circuit 실행의 가중 조합을 통해 해밀토니안 동역학의 Trotter 오류를 줄입니다.

- [GitHub 저장소](https://github.com/Qiskit/qiskit-addon-mpf)를 방문하세요.
- [문서](https://qiskit.github.io/qiskit-addon-mpf)를 읽어보세요.

### 최적화 매퍼 {#optimization-mapper}

최적화 매퍼 애드온은 최적화 문제를 추상 모델로 공식화한 다음 양자 컴퓨터가 이해할 수 있는 표현으로 변환하여 모델링하는 기능을 포함합니다.

- 사용 방법 안내 및 API 참조는 [문서](https://qiskit.github.io/qiskit-addon-opt-mapper/)를 읽어보세요.
- [GitHub](https://github.com/qiskit/qiskit-addon-opt-mapper) 저장소를 방문하세요.

## 최적화를 위한 애드온 {#addons-for-optimizing}

### 연산자 역전파 {#operator-backpropagation}

연산자 역전파(OBP)는 더 많은 연산자 측정을 대가로 끝에서 연산을 잘라내어 Circuit 깊이를 줄입니다.

- [GitHub 저장소](https://github.com/Qiskit/qiskit-addon-obp)를 방문하세요.
- [문서](/guides/qiskit-addons-obp)를 읽어보세요.
- [OBP를 사용하여 기댓값을 개선하는](/tutorials/operator-back-propagation) 튜토리얼을 읽어보세요.

### Circuit 절단 {#circuit-cutting}

Circuit 절단은 인접하지 않은 Qubit 간의 얽힘 Gate를 분해하여 Transpiler된 Circuit의 깊이를 줄입니다.

- [GitHub 저장소](https://github.com/Qiskit/qiskit-addon-cutting)를 방문하세요.
- [문서](https://qiskit.github.io/qiskit-addon-cutting)를 읽어보세요.

## 후처리를 위한 애드온 {#addons-for-post-processing}

### 샘플 기반 양자 대각화 {#sample-based-quantum-diagonalization}

샘플 기반 양자 대각화(SQD)는 노이즈가 있는 양자 샘플을 고전적으로 후처리하여, 예를 들어 화학 응용 분야에서 양자 시스템 해밀토니안의 고유값 추정을 더 정확하게 산출합니다.

- [GitHub 저장소](https://github.com/Qiskit/qiskit-addon-sqd)를 방문하세요.
- [문서](/guides/qiskit-addons-sqd)를 읽어보세요.
- [화학 해밀토니안의 샘플 기반 양자 대각화](/tutorials/sample-based-quantum-diagonalization) 튜토리얼을 읽어보세요.

### HPC용 SQD {#sqd-for-hpc}

이 애드온은 SQD 애드온의 HPC 지원 구현입니다. 현대적인 C++17 표준으로 작성되었으며 MPI와 함께 사용할 단일 컴파일된 바이너리를 생성하도록 설계되었습니다.

- [API 참조](https://qiskit.github.io/qiskit-addon-sqd-hpc/) 문서를 방문하세요.
- [GitHub](https://github.com/qiskit/qiskit-addon-sqd-hpc) 저장소를 확인하세요.

## 오류 완화를 위한 애드온 {#addons-for-error-mitigation}

### M3 (mthree) {#m3-mthree}

행렬 없는 측정 완화(M3)는 병렬로 계산할 수 있는 확장 가능한 양자 측정 오류 완화 패키지입니다.

- 사용 방법 안내, 튜토리얼 및 API 참조는 [문서](https://qiskit.github.io/qiskit-addon-mthree/)를 방문하세요.
- 패키지의 [GitHub](https://github.com/Qiskit/qiskit-addon-mthree) 페이지를 방문하세요.

### Shaded lightcones {#shaded-lightcones}

Shaded lightcones 애드온은 Pauli 전파를 사용하여 대상 관측량의 특성에 따라 완화가 필요한 노이즈 모델의 오류 항 수를 줄입니다. 이는 확률적 오류 소거(PEC) 워크플로우의 샘플링 오버헤드를 줄이는 효과가 있습니다.

- 사용 방법 안내, 튜토리얼 및 API 참조는 [문서](https://qiskit.github.io/qiskit-addon-slc/)를 방문하세요.

### 전파된 노이즈 흡수 {#propagated-noise-absorption}

전파된 노이즈 흡수(PNA)는 Pauli 전파를 사용하여 노이즈 모델의 정보를 대상 관측량에 흡수합니다. 이 수정된 관측량을 측정하면 모델로 표현된 노이즈를 완화하는 효과가 있습니다.

- 사용 방법 안내, 튜토리얼 및 API 참조는 [문서](https://qiskit.github.io/qiskit-addon-pna/apidocs/qiskit_addon_pna.html)를 방문하세요.
