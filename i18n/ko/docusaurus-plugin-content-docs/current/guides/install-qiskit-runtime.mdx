# Qiskit Runtime 클라이언트 설치 {#install-the-qiskit-runtime-client}

Qiskit Runtime은 IBM Quantum&reg; 하드웨어에서 양자 계산을 실행하기 위한 클라우드 기반 서비스입니다. 이 가이드에서는 Qiskit Runtime 서비스에 접근하기 위한 클라이언트 패키지인 `qiskit-ibm-runtime`을 설치하는 방법을 안내합니다.

Qiskit Runtime 클라이언트를 설치하기 전에 다음 가이드의 지침을 따르세요:

1. [Qiskit 설치](/guides/install-qiskit).
2. [IBM Cloud 계정 설정](cloud-setup).

다음 명령어를 실행하여 Qiskit Runtime을 설치합니다:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="최신 버전으로 유지하기">
[Qiskit Runtime 릴리스 노트](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes)를 주기적으로 확인하여 새 릴리스를 확인하세요. `pip install -U qiskit-ibm-runtime`을 실행하여 `qiskit-ibm-runtime` 요구 사항을 자주 업데이트할 것을 권장합니다.
</Admonition>
