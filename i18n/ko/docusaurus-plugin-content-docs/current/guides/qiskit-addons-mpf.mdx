다중 곱 공식(MPF)은 Circuit 실행 횟수가 증가하는 대신 양자 시스템의 동역학을 더 정확하게 시뮬레이션하는 데 사용할 수 있습니다. 이것은 시간 진화된 상태에 대한 기댓값의 오류를 완화하는 후처리 기법입니다.

고전 컴퓨팅은 여러 Circuit 실행의 가중 조합에 계수를 제공하는 선형 방정식 시스템을 푸는 데 사용됩니다. 이 가중 조합을 사용하면 좋은 Trotter 단계 선택을 통해 시간 진화 시뮬레이션과 관련된 오류를 줄일 수 있습니다. MPF 도구는 Trotter 단계의 수와 Trotter 근사 차수를 포함한 선택된 데이터를 수집하여 관련 선형 방정식 시스템을 준비하고 풀거나 근사 해를 구합니다. 이를 통해 시간 진화된 상태의 기댓값 측정을 후처리하는 데 활용할 수 있습니다.
## MPF 패키지 설치 {#install-the-mpf-package}

MPF 패키지를 설치하는 방법은 두 가지가 있습니다. PyPI를 통한 설치와 소스에서 빌드하는 방법입니다. 패키지 의존성 간의 분리를 보장하기 위해 [가상 환경](https://docs.python.org/3.10/tutorial/venv.html)에 설치하는 것을 권장합니다.

### PyPI에서 설치 {#install-from-pypi}

`qiskit-addon-mpf` 패키지를 설치하는 가장 간단한 방법은 PyPI를 이용하는 것입니다.
```bash
pip install qiskit-addon-mpf
```

### 소스에서 빌드 {#build-from-source}
저장소에서 개발하거나 수동으로 설치하고자 하는 사용자는 먼저 저장소를 클론하여 진행할 수 있습니다:
```bash
git clone git@github.com:Qiskit/qiskit-addon-mpf.git
```

그런 다음 `pip`를 통해 패키지를 설치합니다. 저장소에는 특정 기능을 활성화하는 여러 선택적 의존성도 포함되어 있습니다.

필요에 맞게 옵션을 조정하세요.
```bash
pip install tox notebook -e '.[notebook-dependencies,dev]'
```
## 이론적 배경 {#theoretical-background}

MPF는 여러 Circuit 실행의 가중 조합을 통해 양자 시스템의 동역학 시뮬레이션과 관련된 Trotter 근사 오류를 줄일 수 있습니다. 이 가중합은 다음과 같이 정의됩니다:

$$ \mu(t) = \sum_j x_j\rho_j^{k_j}\left(\frac{t}{k_j}\right) + \text{some remaining Trotter error}, $$

여기서 $x_j$는 가중 계수이고, $\rho_j^{k_j}$는 $k_j$개의 Trotter 단계를 사용하여 시간 진화 연산자를 근사하는 곱 공식 $S^{k_j}$를 통해 초기 상태를 진화시켜 얻은 순수 상태에 해당하는 밀도 행렬이며, $j$는 합에서 사용되는 각 곱 공식의 인덱스입니다.

![다양한 수의 Trotter 단계를 가진 여러 Circuit이 목표 관측값을 계산하는 데 사용됩니다](/docs/images/guides/qiskit-addons/mpf-diagram.avif)

그러나 일반적으로 양자 동역학 시뮬레이션의 목표는 시간의 함수인 관측값 $\mathcal{O}(t)$를 측정하는 것입니다. MPF를 사용할 때는 각각 $k_j$개의 Trotter 단계를 사용하는 여러 Circuit을 실행하여 목표 관측값 $\mathcal{O}_{k_j}(t)$의 여러 측정값을 얻습니다. 그런 다음 목표 관측값의 측정값은 다음을 계산하여 구합니다:

$$ \langle \mathcal{O}(t) \rangle = \sum_j x_j(t) \langle \mathcal{O}_{k_j}(t) \rangle. $$

본질적으로, 단일 곱 공식 대신 가변적인 수의 Trotter 단계를 가진 여러 곱 공식을 사용하여 시간 진화 연산자를 근사함으로써 전체 Trotter 오류를 줄일 수 있습니다. 가중합의 각 항에 대해 Circuit을 구성하여 각각의 $k_j$ Trotter 단계 수에 따라 시스템을 진화시킵니다. 각 Circuit은 후처리 단계에서 결과를 재구성하기 위해 QPU에서 별도로 실행됩니다. 이 기법의 유용성은 두 가지 관점에서 볼 수 있습니다:
1. 실행하는 Trotter 단계의 수가 고정된 경우, 총 Trotter 오류가 더 작은 결과를 얻을 수 있습니다.
2. 깊은 Circuit을 초래하는 수의 Trotter 단계에 대해, MPF를 사용하여 실행할 여러 개의 더 짧은 깊이의 Circuit을 찾을 수 있으며, 이는 유사한 Trotter 근사 오류를 초래합니다.

## MPF 계수 결정 {#determine-mpf-coefficients}

`qiskit-addon-mpf` 패키지의 핵심 기능은 MPF 계수 $x_j(t)$ (시간 의존적일 수 있음)를 결정하는 것입니다. 각 $x_j(t)$를 구하는 과정은 선형 방정식 시스템 $Ax=b$를 푸는 것을 포함합니다. 여기서 $x$는 결정할 계수 벡터이고, $A$는 각 $k_j$와 사용된 곱 공식 $S$ (즉, 근사 차수와 Trotter 단계 수)에 의존하는 행렬이며, $b$는 제약 벡터입니다. 이 방정식 시스템은 정확하게 풀거나 계수의 1-norm을 최소화하는 근사 모델로 풀 수 있습니다. 또한 각 $k_j$의 선택은 휴리스틱 과정이지만 다음 제약 조건으로 제한될 수 있습니다:

1. **가장 큰** $k_j$ 값은 신뢰할 수 있게 실행 가능한 Circuit의 최대 깊이에 의해 제한됩니다.
2. **가장 작은** $k_j$는 $dt = t/k_j < 1$을 만족해야 합니다. 이 조건에서 Trotter 오류가 가장 잘 동작하기 때문입니다.
3. 계수 $x_j$ 중 어느 것도 $0$에 가깝지 않아야 합니다. 이는 MPF에 기여하지 않음을 의미하기 때문입니다.
4. 마찬가지로, 가장 큰 $k_j$ 값과 관련된 계수가 지배적이지 않아야 합니다. 이는 단일 곱 공식을 사용하고 있음을 의미하기 때문입니다.
5. 마지막으로, 구한 계수 $x_j$의 노름이 작아야 합니다. 이는 잘 조건화된 MPF를 나타내기 때문입니다 [1](#references).
