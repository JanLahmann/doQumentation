# Primitives 소개 {#introduction-to-primitives}

Primitives는 양자 컴퓨터의 가장 일반적인 작업인 양자 상태 샘플링과 기댓값 계산을 단순화하기 위해 만들어졌습니다. 첫 번째 Qiskit Runtime Primitives([`EstimatorV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2) 및 [`SamplerV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2))는 Qiskit [Primitives 기본 클래스](/guides/primitives)의 구현체입니다. 이들은 클라우드 기반 서비스로 더 정교한 구현(예: 오류 완화 포함)을 제공하며 IBM Quantum&reg; 하드웨어에 접근하는 데 사용됩니다.

가장 새로운 Qiskit Runtime Primitive인 [Executor](https://qiskit.github.io/qiskit-ibm-runtime/stubs/qiskit_ibm_runtime.Executor.html)(베타)는 성능을 희생하지 않으면서 더 많은 가시성과 제어를 제공하는 저수준 인터페이스를 제공합니다.
