# 비용 확인 {#view-the-cost}

언제든지 인스턴스와 연결된 워크로드에서 발생한 비용을 확인할 수 있습니다.

## 인스턴스 비용 확인 {#view-instance-cost}

인스턴스의 사용량을 확인하려면 [인스턴스 페이지](https://quantum.cloud.ibm.com/instances)에서 해당 인스턴스를 클릭하여 상세 페이지를 여세요. **청구 개요**에 연결된 링크를 클릭하면 전체 비용 및 청구 세부 정보를 확인할 수 있습니다.

**인스턴스 사용량**은 인스턴스가 연결된 플랜 유형에 따라 다른 값을 보고합니다:

- Premium 및 Open 플랜의 경우, 현재 날짜의 현재 시간까지 최근 28일(롤링) 기준의 실시간 사용량입니다.
- 종량제(Pay-As-You-Go) 플랜의 경우, 인스턴스가 생성된 시점부터의 전체 사용량입니다.
- Flex 플랜의 경우, 계약 시작 시점부터의 사용량입니다.

## 워크로드 사용량 확인 {#view-workload-usage}

특정 워크로드에서 발생한 사용량을 확인하려면 [워크로드 페이지](https://quantum.cloud.ibm.com/workloads)를 열고 해당 워크로드의 "사용량" 열을 확인하세요.
