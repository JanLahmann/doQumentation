# ID 제공자 사용자 관리 {#manage-id-provider-users}

외부 ID 제공자(IDP)와 통합하여 외부 사용자를 IBM Cloud 계정에 안전하게 인증할 수 있습니다. 외부 제공자를 사용하면 회사 사용자들이 단일 로그인(SSO)을 사용할 수 있는 방법을 제공할 수 있습니다. 다음 방법 중 하나를 통해 클라우드 계정을 외부 IDP에 연결할 수 있습니다:

- [IBM Cloud&reg; SAML 서비스 제공자(SP)](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-cloud-saml) 사용
- [App ID](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-appid) 사용 (비용이 발생할 수 있습니다)
- [IBMid를 통해 모든 IBM&reg; 제품에 사용자 페더레이션](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-ibmid)

<Admonition type="tip">
IDP를 연결한 후, 동적 규칙을 설정하면 사용자가 가입할 때마다 모든 그룹에 수동으로 추가할 필요가 없습니다. 지침은 [외부 ID 제공자의 인증 사용 설정](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#app-id-dynamic-rules)을 참조하십시오.
</Admonition>
