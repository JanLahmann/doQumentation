# 사용 가능한 플랜 개요 {#overview-of-available-plans}

IBM Quantum Platform에는 아래에 설명된 Open(무료 액세스) 플랜과 유료 플랜이 있습니다. 자세한 내용은 [IBM Quantum 액세스 플랜](https://www.ibm.com/quantum/pricing)을 참조하세요.

여러 플랜에 액세스하려면 여러 인스턴스를 생성하여 각각 다른 플랜을 할당할 수 있습니다. 예를 들어, Open Plan 인스턴스와 Pay-As-You-Go Plan 인스턴스를 동시에 가질 수 있습니다. `QiskitRuntimeService` 초기화 메서드 또는 `QiskitRuntimeService.save_account()` 메서드에서 기본 인스턴스를 설정할 수 있습니다. 예시:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Open Plan {#open-plan}

세계 최고의 QPU에서 양자 Circuit을 무료로 실행하세요(28일 롤링 기간당 최대 10분의 양자 시간). [플랫폼 대시보드](https://quantum.cloud.ibm.com/) 및 [워크로드](https://quantum.cloud.ibm.com/workloads) 페이지에서 사용량을 추적할 수 있습니다. Open Plan은 us-east 리전에서만 사용 가능합니다.

이 플랜은 양자 컴퓨팅을 학습하고 IBM 양자 기술을 탐색하는 분들에게 권장됩니다.

## Flex Plan {#flex-plan}

1년 이내에 사용할 양자 작업 실행을 위한 일정 분량의 시간(최소 400분)을 사전 구매하세요. 언제든지 더 많은 시간을 추가 구매할 수 있습니다.

시간을 구매한 후에는 해당 분량을 계정의 Flex Plan에 대한 기존 또는 새 인스턴스에 할당해야 합니다. [인스턴스 페이지](https://quantum.cloud.ibm.com/instances)에서 할당 한도를 설정하거나 업데이트하거나, [새 Flex Plan 인스턴스 생성 시](/guides/instances) 할당량을 설정할 수 있습니다. 모든 분량을 기존 인스턴스에 할당한 경우, 더 많은 시간을 구매하거나 다른 인스턴스의 할당량에서 일부를 제거하기 전까지는 Flex Plan에 대한 새 인스턴스를 생성할 수 없습니다. 자세한 내용은 [인스턴스 할당 한도 설정](allocation-limits)을 참조하세요.

10,000분 이상이 필요한 경우 Premium Plan 구독을 고려해야 합니다. 400분을 구매하지 않으려면 Pay-As-You-Go Plan을 고려해 보세요.

Flex Plan에 가입하려면 [IBM에 문의](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum)하세요.

## Pay-As-You-Go Plan {#pay-as-you-go-plan}

세계 최고의 QPU에서 양자 Circuit을 실행하고 사용한 양자 시간에 대해서만 비용을 지불하세요.

이 플랜은 양자 유틸리티 연구 프로젝트를 수행하고 유연한 액세스로 비즈니스 모델을 테스트하는 분들에게 권장됩니다. 내년에 최소 400분을 사용할 예정이라면 Flex Plan을 사용하는 것이 권장됩니다.

## Premium Plan {#premium-plan}

엔터프라이즈 양자 시간 구독을 사용하여 세계 최고의 QPU에서 양자 Circuit을 실행하세요. 또한 [Qiskit Functions](functions) 및 [Qiskit Transpiler as a Service](qiskit-transpiler-service)와 같은 일부 기능은 Premium Plan 사용자에게 제공됩니다.

이 플랜은 전략적 양자 로드맵을 실행하고 대규모로 양자 알고리즘과 애플리케이션을 개발하려는 분들에게 권장됩니다. 자세한 내용은 [IBM에 문의](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum)하세요.

## On-Prem Plan {#on-prem-plan}

IBM Quantum이 서비스하고 유지 관리하는 완전히 전용 온프레미스 양자 시스템에 액세스하세요.

이 서비스는 클래식-양자 알고리즘과 애플리케이션을 탐색하면서 리소스와 데이터에 대한 높은 제어권이 필요한 분들에게 권장됩니다. 자세한 내용은 [IBM에 문의](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum)하세요.
