# 온라인 랩 환경 {#online-lab-environments}

로컬 컴퓨터에 Qiskit을 다운로드하여 설치하는 방법 외에도, 온라인 개발 환경에서 작업하는 것을 선호하는 분들을 위한 여러 옵션이 있습니다.

여러 공급자가 Qiskit이 사전 설치된 온라인 Jupyter 환경을 제공하여 빠르고 편리한 온보딩을 지원합니다. 권장 공급자로는 클라우드 서비스 공급자인 OVHcloud와 양자 소프트웨어 개발 클라우드 플랫폼인 qBraid가 있습니다.

## Qiskit이 사전 설치된 권장 노트북 환경 {#recommended-notebook-environments-with-qiskit-preinstalled}

IBM Watson&reg; Studio, Google Colab, Microsoft Azure Machine Learning Studio와 같은 추가적인 온라인 Jupyter 노트북 환경도 적절한 지침에 따라 Qiskit과 함께 사용할 수 있습니다.

### OVHcloud AI 노트북 {#ovhcloud-ai-notebooks}

OVHcloud Public Cloud 및 OVHcloud Startup 사용자는 OVHcloud 양자 노트북을 사용하여 Qiskit에 접근하고, IBM Quantum&reg; 또는 IBM Cloud&reg; 계정을 통해 IBM&reg; QPU에 작업을 제출할 수 있습니다. 자세한 내용은 [OVHcloud 문서 및 튜토리얼](https://help.ovhcloud.com/csm/world-documentation?id=kb_home) 페이지를 방문하세요.

<Admonition type="note">
OVHcloud AI 노트북은 미국 등 일부 지역에서는 이용하지 못할 수 있습니다. 또한, 지역에 따라 사용자 인터페이스가 다르게 표시될 수 있습니다.
</Admonition>

시작하려면 다음 단계를 따르세요:

1. [IBM Cloud 계정을 만드세요.](./cloud-setup)
1. [OVHcloud 계정을 설정하세요.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [OVH Public Cloud 프로젝트를 만드세요.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. [첫 번째 AI 노트북 시작하기](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270)의 단계를 따르되, 프레임워크로 **Qiskit**을 선택하세요.

### qBraid Lab {#qbraid-lab}

Qiskit 사용자는 다음 지침에 따라 qBraid의 사전 구성된 Python 환경을 사용할 수 있습니다. 자세한 내용은 [qBraid 문서](https://docs.qbraid.com/home/introduction)를 참조하세요.

1. [qBraid 무료 계정을 설정하세요.](https://docs.qbraid.com/home/account)
3. [Lab 환경을 만드세요.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. qBraid Lab을 실행하고, 오른쪽의 **ENVS** 아이콘을 클릭한 다음, 새 사이드바 상단의 **+ ADD**를 클릭하고 Qiskit을 검색하세요. 사용하려는 버전을 클릭한 후 **Install**을 클릭하세요.
4. 노트북 섹션에서 **Python 3 [Qiskit]**을 클릭하여 Jupyter 노트북을 만드세요.
4. 노트북 셀에서 `%pip install [package_name].`을 실행하여 추가 Python 패키지(예: Qiskit v1.2)를 설치할 수 있습니다. 자세한 내용은 [Pip (magic) commands](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands) 항목을 참조하세요.

## Qiskit 사용자를 위한 추가 노트북 환경 {#additional-notebook-environments-for-qiskit-users}

Qiskit이 사전 구성되지 않은 플랫폼의 경우 설정 방법은 비슷하지만, Qiskit을 직접 설치해야 합니다. 계정을 만들고 새 프로젝트(또는 플랫폼에 따라 노트북이나 워크스페이스)를 시작한 후 Qiskit을 설치하면 됩니다.

### IBM Watson Studio {#ibm-watson-studio}

1. [Watson Studio 무료 평가판을 설정하세요.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Watson Studio 프로젝트를 만드세요.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. 필요한 Qiskit 패키지 및 종속성을 설치하려면 새 노트북을 열고 다음 명령을 실행하세요:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Google 계정을 만드세요.](https://support.google.com/accounts/answer/27441?hl)
2. [Google Colab 랜딩 페이지](https://colab.research.google.com/)로 이동하여 Google 계정으로 로그인하세요.
3. [Colab 노트북을 만들고](https://colab.research.google.com/#create=true) 다음 명령을 실행하여 Qiskit을 설치하세요:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Azure 계정을 만드세요.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [워크스페이스를 만드세요.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [워크스페이스에서 Jupyter 노트북을 만드세요.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. 노트북에서 다음 명령을 실행하여 Qiskit을 설치하세요:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
