# 조직을 위한 IBM Quantum Platform 구성 {#configure-ibm-quantum-platform-for-an-organization}

다음 단계에 따라 IBM Quantum Platform을 설정하세요.

<span id="iam-org"></span>
## 1. IAM 설정 구성 {#configure-iam-settings}

먼저, 관리자의 IAM 계정에서 다음 설정을 구성하세요. 이 설정을 검토하고 구성하려면 [관리 → IAM → 설정](https://cloud.ibm.com/iam/settings)으로 이동하세요.

* **사용자 목록 가시성**은 프로젝트 할당 여부와 관계없이 사용자가 서로를 볼 수 있는지 여부를 결정합니다. `enabled` 설정은 사용자 가시성을 제한합니다. 즉, 계정의 사용자는 동일한 리소스에 액세스할 수 있더라도 서로를 볼 수 없습니다. 환경에 맞는 적절한 값을 선택하세요. 자세한 내용은 [사용자 가시성 제어](https://cloud.ibm.com/docs/account?topic=account-iam-user-setting)를 참고하세요.
* **API 키 생성**은 사용자가 API 키를 생성할 수 있는지 여부를 제어합니다. IBM Quantum Platform에서는 API 키를 사용하는 것이 일반적입니다. API 키를 사용하는 경우 `disabled`를 선택하세요. 또는 각 사용자에게 특정 권한을 부여할 수도 있습니다.

<span id="crt-rsc-grp-org"></span>
## 2. (선택 사항) 리소스 그룹 생성 {#create-resource-groups}

IBM Quantum 서비스 인스턴스를 직접 참조하는 액세스 그룹을 사용하는 경우 이 단계를 건너뛰세요.

리소스 그룹을 사용하기로 선택한 경우 [관리 → 계정 → 리소스 그룹(계정 리소스 내)](https://cloud.ibm.com/account/resource-groups)으로 이동하여 **만들기**를 클릭하세요.

<span id="create-instance-org"></span>
## 3. IBM Quantum 서비스 인스턴스 생성 {#create-ibm-quantum-service-instances}

이미 IBM Quantum 서비스 인스턴스를 생성한 경우 이 단계를 건너뛰세요.

* 리소스 그룹을 사용하는 경우 적절한 리소스 그룹에 서비스 인스턴스를 생성해야 합니다.
* `QR-ml`과 같은 서비스 인스턴스 이름은 액세스 그룹 참조에 필요합니다.

지침은 [IBM Cloud 사용 설정](/guides/cloud-setup#cloud)의 단계를 따르세요.

<span id="create-group-org"></span>
## 4. 프로젝트를 위한 액세스 그룹 생성 {#create-access-groups-for-projects}

먼저, 사용자가 IBM Quantum 서비스 인스턴스를 사용하여 작업을 수행할 수 있도록 허용하는 [사용자 지정 역할을 생성](/guides/custom-roles#custom)하세요.
그런 다음, 각 프로젝트에 대해 [액세스 그룹을 생성](/guides/access-groups)하고 프로젝트 리소스를 사용하는 데 필요한 최소 권한 집합을 해당 그룹에 부여하세요.

<span id="setup-idp-org"></span>

<span id="firewall"></span>
## 6. 방화벽 구성 {#configure-your-firewall}

IBM Quantum API 엔드포인트에 대한 액세스를 허용하도록 방화벽을 구성하려면 다음 URL을 허용 목록에 추가하세요.

- IBM Cloud 글로벌 서비스:
    ```text
    cloud.ibm.com
    iam.cloud.ibm.com
    api.global-search-tagging.cloud.ibm.com
    globalcatalog.cloud.ibm.com
    resource-controller.cloud.ibm.com
    ```
- IBM Quantum Qiskit Runtime 서비스 - us-east 지역:
    ```text
    quantum.cloud.ibm.com
    s3.us-east.cloud-object-storage.appdomain.cloud
    ```
- IBM Quantum Qiskit Runtime 서비스 - eu-de 지역:
    ```text
    eu-de.quantum.cloud.ibm.com
    s3.direct.eu-de.cloud-object-storage.appdomain.cloud
    ```
- IBM Quantum Qiskit Functions 미리 보기 서비스:
    ```text
    qiskit-serverless.quantum.ibm.com
    ```
