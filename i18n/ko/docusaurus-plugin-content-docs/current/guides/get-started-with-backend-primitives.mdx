# 백엔드 Primitives 시작하기 {#get-started-with-the-backend-primitives}

공급자별 Primitives와 달리, 백엔드 Primitives는 [`BackendV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.providers.BackendV2) 인터페이스를 구현하는 한 임의의 `backend` 객체와 함께 사용할 수 있는 일반적인 구현입니다. 일부 공급자는 Primitives를 기본적으로 구현합니다. 자세한 내용은 [Qiskit 생태계 페이지](https://www.ibm.com/quantum/ecosystem?tag=provider)를 참조하세요.

<span id="estimator"></span>
## Estimator 백엔드 Primitive 시작하기 {#get-started-with-the-estimator-backend-primitive}

Estimator Primitive는 [`qiskit.primitives.BackendEstimatorV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendEstimatorV2) 클래스를 사용하여 모든 공급자와 함께 실행할 수 있습니다. 그러나 백엔드 Primitives는 사용자의 로컬 머신에서 실행되도록 설계되었으므로 "기본 제공" 측정 또는 게이트 오류 완화 구현을 제공하지 않습니다.

### 예제: {#example}

```python
from qiskit.primitives import BackendEstimatorV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
estimator = BackendEstimatorV2(backend)
```

<span id="sampler"></span>
## Sampler 백엔드 Primitive 시작하기 {#get-started-with-the-sampler-backend-primitive}

Sampler Primitive는 [`qiskit.primitives.BackendSamplerV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendSamplerV2)를 사용하여 모든 공급자와 함께 실행할 수 있습니다. 단, `memory` 옵션을 지원하는 백엔드가 필요합니다.

### 예제: {#example}

```python
from qiskit.primitives import BackendSamplerV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
sampler = BackendSamplerV2(backend)
```
