# 개요 {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

이 과정에서는 양자 컴퓨팅이 머신 러닝에서 어떻게 흥미로운 역할을 할 수 있는지 살펴봅니다. 고전적인 머신 러닝 접근 방식에 대한 일부 복습을 제공하지만, 이러한 방법에 대한 어느 정도의 친숙함도 가정합니다. 주된 초점은 양자 컴퓨팅이 기존 머신 러닝 워크플로에 어떻게 적합할 수 있는지입니다. 주제에는 데이터 인코딩, 양자 커널, 변분 양자 회로가 포함됩니다.

## 시험 {#exam}

양자 머신 러닝 배지를 얻으려면 IBM&reg; Training에서 시험을 치르세요. 이 시험은 이 과정의 수업을 읽은 후에 치르도록 설계되었습니다. 시험에 합격하면 Credly로부터 배지를 획득했다는 알림을 받게 됩니다.
