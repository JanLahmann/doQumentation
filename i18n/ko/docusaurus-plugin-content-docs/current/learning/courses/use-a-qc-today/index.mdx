# 개요 {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

*양자 컴퓨터를 오늘 바로 사용해 보세요*에 오신 것을 환영합니다! 이 강좌에서는 즉시 양자 컴퓨터를 사용해 보면서 양자 컴퓨팅의 기본 원리를 함께 배우게 됩니다.

이 강좌는 호기심 많고 바쁜 분들을 위해 설계되었습니다. 물리학 학위나 수년간의 프로그래밍 경험은 필요하지 않습니다. 노트북을 따라가며 "실행" 버튼을 클릭할 수 있다면 준비가 된 것입니다.
