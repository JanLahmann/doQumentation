# 개요 {#overview}

!["Hero image"](/learning/images/courses/fundamentals-of-quantum-algorithms/hero.svg)

*양자 알고리즘의 기본 원리*에 오신 것을 환영합니다. 이 과정은 다음 과정들로 구성된 *양자 정보와 계산 이해하기* 시리즈의 두 번째 과정입니다.

- [양자 정보의 기초](/learning/courses/basics-of-quantum-information)
- 양자 알고리즘의 기본 원리 (이 과정)
- [양자 정보의 일반적 정식화](/learning/courses/general-formulation-of-quantum-information)
- [양자 오류 정정의 기초](/learning/courses/foundations-of-quantum-error-correction)

이 과정에서는 양자 컴퓨터로 무엇을 할 수 있는지와 고전 컴퓨터에 대비한 양자 컴퓨터의 장점을 포함하여 양자 정보의 계산적 이점을 탐구합니다.
단순한 개념 증명을 제공하는 양자 질의(query) 알고리즘으로 시작하여, 정수 소인수분해와 비정형 검색을 포함한
문제들에 대한 양자 알고리즘으로 넘어갑니다.

이 과정은 양자 정보와 계산의 이론적 기초에 대한 지식을 얻고자 하는 컴퓨터 과학, 물리학, 공학, 수학 등 분야의 학생, 전문가, 그리고 취미로 배우는 사람들을 대상으로 합니다.
