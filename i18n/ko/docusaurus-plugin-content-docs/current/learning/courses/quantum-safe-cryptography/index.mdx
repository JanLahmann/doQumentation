# 개요 {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

양자 안전 암호에 대한 실용적인 소개는 IBM Quantum&reg; Learning의 무료 온라인 과정으로, 양자 안전 암호의 기초 개념에 대한 입문서 역할을 합니다.

이 과정은 애플리케이션 보안의 현대화에 관심이 있는 개발자를 위해 만들어졌으며, 네 가지 핵심 암호 분야(암호 해시 함수, 대칭 키 암호, 비대칭 키 암호, 양자 안전 암호)를 다루는 멀티모달 수업과 대화형 라이브 코드 예제를 특징으로 합니다.

이 과정을 통해 개발자는 사이버 보안 위험 환경이 어떻게 진화하고 있는지 배우고, 양자 시대를 형성할 현대적인 접근 방식을 탐구할 수 있습니다.
