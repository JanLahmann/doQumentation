# 개요 {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

이 과정은 양자 컴퓨팅에 대한 어느 정도의 경험이 있고 Qiskit Runtime 프리미티브를 사용하여 변분 알고리즘을 작성하는 방법을 배우고자 하는 개인을 대상으로 합니다. 또한 변분 알고리즘 분야에서 지식과 기술을 확장하려는 양자 컴퓨팅 전문가에게도 적합합니다.

## 시험 {#exam}

변분 알고리즘 설계 배지를 얻으려면 IBM&reg; Training에서 시험을 치르세요. 이 시험은 이 과정의 수업을 읽은 후에 치르도록 설계되었습니다. 시험에 합격하면 Credly로부터 배지를 획득했다는 알림을 받게 됩니다.
