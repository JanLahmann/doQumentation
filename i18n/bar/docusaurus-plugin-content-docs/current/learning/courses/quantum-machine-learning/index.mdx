# Überblick {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Der Kurs erforscht, wie Quantum Computing a spannende Rolle im Machine Learning spielen kann. Der Kurs bietet a Wiederholung vo klassischen Machine Learning Ansätze, geht aber a vo a gwissen Vertrautheit mit dene Methoden aus. Der Hauptfokus liegt drauf, wie Quantum Computing in bestehende Machine Learning Workflows passt. Themen san Data Encoding, Quantum Kernels und variational quantum circuits.
