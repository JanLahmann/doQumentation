# Resumen {#overview}

!["Hero image"](/learning/images/courses/designing-and-leading-quantum-projects/hero.svg)

Este curso está diseñado para ayudar a equipos y organizaciones a desarrollar eficazmente las capacidades necesarias para poner en marcha iniciativas cuánticas con éxito.

Recomendamos dedicar aproximadamente 4 horas a completar este contenido, según la experiencia previa de cada persona.

### Público objetivo {#intended-audience}

Recomendamos este curso para líderes y arquitectos de iniciativas cuánticas, incluyendo profesorado universitario, líderes de proyectos en grandes empresas, equipos directivos de startups y más. Será de particular utilidad para organizaciones que se inician en la computación cuántica.

## Conocimientos previos recomendados {#recommended-background}

No hay requisitos técnicos previos para este curso, pero recomendamos que los participantes tomen primero el curso [Quantum Business Foundations](/learning/courses/quantum-business-foundations), que es una buena introducción al ecosistema cuántico.

Si tienes preguntas o inquietudes sobre este curso, por favor contacta al equipo de IBM Quantum Learning&reg;.
