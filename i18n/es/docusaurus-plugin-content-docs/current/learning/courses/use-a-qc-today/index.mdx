# Visión general {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

¡Bienvenido/a a *Usa una computadora cuántica hoy*! En este curso, empezarás a usar una computadora cuántica de inmediato y aprenderás los principios básicos de la computación cuántica a lo largo del camino.

Este curso está diseñado para las personas curiosas y ocupadas. No necesitas un título en física ni años de experiencia en programación. Si puedes seguir un notebook y hacer clic en "Run", estás listo/a.
