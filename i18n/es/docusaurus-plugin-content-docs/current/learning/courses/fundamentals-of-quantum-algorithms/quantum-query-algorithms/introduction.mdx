En esta primera lección del curso, formulamos un marco algorítmico simple — el llamado *modelo de consulta* — y examinamos las ventajas que ofrecen las computadoras cuánticas dentro de este marco.

El modelo de consulta de la computación es como una placa de Petri para las ideas sobre algoritmos cuánticos.
Es rígido y artificial en el sentido de que no refleja fielmente los tipos de problemas computacionales que normalmente nos interesan en la práctica — sin embargo, ha demostrado ser una herramienta increíblemente útil para el desarrollo de técnicas para algoritmos cuánticos.
Esto incluye las técnicas subyacentes a los algoritmos cuánticos más conocidos, como el algoritmo de Shor para la factorización de enteros.
El modelo de consulta es además un marco muy útil para *explicar* las técnicas de los algoritmos cuánticos.

Después de introducir el modelo de consulta en sí, discutimos el primer algoritmo cuántico jamás descubierto: el *algoritmo de Deutsch*, así como una extensión llamada *algoritmo de Deutsch-Jozsa*.
Estos algoritmos demuestran ventajas medibles de las computadoras cuánticas sobre las computadoras clásicas en el marco del modelo de consulta.
A continuación, discutimos un algoritmo cuántico llamado *algoritmo de Simon*, que ofrece una ventaja más robusta y convincente de la computación cuántica sobre la clásica — por razones que se explicarán cuando lleguemos a ello.
