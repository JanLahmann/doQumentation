## Pon a prueba tus conocimientos {#test-your-knowledge}

Después de completar las lecturas y actividades de este curso, haz clic en el botón de abajo para navegar a IBM&reg; Training, donde podrás realizar el examen de insignia para Variational algorithm design.

El examen consta de 20 preguntas, de las cuales debes responder correctamente al menos 16 para aprobar. Tras obtener un resultado aprobatorio, tu insignia se enviará automáticamente a la dirección de correo electrónico con la que te registraste en IBM&reg; Training.

Asegúrate de estar preparado para realizar el examen. Si no obtienes un resultado suficiente en dos intentos, el examen se bloqueará temporalmente.

Nota: Estos exámenes en el sitio web de IBM Training pueden no estar disponibles temporalmente en los próximos días debido a tareas de mantenimiento.
