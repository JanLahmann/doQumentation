Este curso ofrece una introducción enfocada al solucionador variacional de valores propios cuánticos (VQE), incluyendo la aplicación de VQE en química. Está dirigido a profesionales técnicos en el campo de la computación cuántica que desean aplicar la computación cuántica a problemas químicos, bioquímicos o biofísicos. El curso cubre componentes importantes de un cálculo VQE, entre ellos:

* Hamiltoniano
* Ansatz
* Optimizador clásico

Todos estos elementos se integran para construir cálculos VQE con Qiskit Runtime Primitives, principalmente usando Estimator. Puedes encontrar información sobre Primitives en la [documentación](/guides/primitives). Al construir un cálculo VQE, seguimos el framework estándar de Qiskit Patterns:

* Mapear entradas clásicas a un problema cuántico
* Optimizar el problema para la ejecución cuántica
* Ejecutar con Qiskit Runtime Primitives
* Posprocesar y devolver el resultado en formato clásico

Una explicación del framework aparece en varios lugares de este curso, pero para una explicación completa con ejemplos de Qiskit Patterns, lee [esta documentación](/guides/intro-to-patterns).

Recomendamos planificar aproximadamente de 6 a 10 horas para completar este curso. Se estima que los videos y las lecturas requerirán dos horas. El tiempo restante está destinado a ejecutar y adaptar celdas de código, responder preguntas de cuestionarios y leer la documentación enlazada.

IBM Quantum&reg; ofrece otros recursos de formación adicionales sobre VQE. Si ya estás familiarizado/a con los componentes de VQE y simplemente quieres aplicarlos en la práctica, consulta nuestro tutorial: [Estimación de la energía del estado fundamental de la cadena de Heisenberg con VQE](/tutorials/spin-chain-vqe).
