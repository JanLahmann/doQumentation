## Pon a prueba tus conocimientos {#test-your-knowledge}

Después de completar las lecturas y actividades de este curso, haz clic en el botón de abajo para dirigirte a IBM&reg; Training, donde podrás realizar el examen para la insignia de Quantum Machine Learning.

El examen consta de 20 preguntas y debes responder correctamente al menos 16 de ellas para aprobar. Si alcanzas una puntuación aprobatoria, tu insignia se enviará automáticamente a la dirección de correo electrónico que utilizaste para registrarte en IBM Training.

Asegúrate de estar preparado para realizar el examen. Si no logras una puntuación aprobatoria en dos intentos, el examen se bloqueará temporalmente.

Nota: Estos exámenes en el sitio web de IBM Training pueden no estar disponibles temporalmente en los próximos días debido a trabajos de mantenimiento.
