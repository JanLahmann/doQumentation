# Instalar el cliente de Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime es el servicio basado en la nube para ejecutar cálculos cuánticos en hardware de IBM Quantum&reg;. Esta guía te muestra cómo instalar `qiskit-ibm-runtime`, que es el paquete cliente para acceder al servicio Qiskit Runtime.

Antes de instalar el cliente de Qiskit Runtime, sigue las instrucciones de estas guías:

1. [Instalar Qiskit](/guides/install-qiskit).
2. [Configurar tu cuenta de IBM Cloud](cloud-setup).

Ejecuta este comando para instalar Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Mantente al día con las últimas versiones">
Consulta periódicamente las [notas de versión de Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) para ver las nuevas versiones. Recomendamos actualizar frecuentemente los requisitos de `qiskit-ibm-runtime` ejecutando `pip install -U qiskit-ibm-runtime`.
</Admonition>
