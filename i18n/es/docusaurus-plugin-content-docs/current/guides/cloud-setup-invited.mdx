# Configura tu cuenta de IBM Cloud - usuarios invitados {#set-up-your-ibm-cloud-account-invited-users}

Si no tienes una invitación de un administrador para unirte a una cuenta que un administrador configuró, ponte en contacto con tu administrador. Si necesitas configurar tu propia cuenta, sigue los pasos de [Configurar tu cuenta de IBM Cloud](/guides/cloud-setup).

Puedes ejecutar cargas de trabajo en unidades de procesamiento cuántico (QPUs) de IBM&reg; configurando una cuenta en IBM Cloud&reg;. También necesitarás [instalar y configurar Qiskit y Qiskit Runtime](/guides/install-qiskit#local).

Tu cuenta de usuario está asociada con una o más [instancias](./instances), identificadas por _Nombres de Recursos en la Nube (Cloud Resource Names, CRNs)_ que dan acceso a los servicios de IBM Quantum. Adicionalmente, se asigna un token de API único (también llamado clave o key) a cada cuenta, lo que permite el acceso a IBM Quantum desde Qiskit. Para obtener instrucciones sobre cómo elegir una instancia específica, consulta [Conectarse a una instancia](./instances#connect-instance).

    <Admonition type="note">
      El [panel principal (dashboard) de IBM Quantum Platform](https://quantum.cloud.ibm.com) y la [página de Instancias (Instances)](https://quantum.cloud.ibm.com/instances) enumeran las instancias a las que puedes acceder.
    </Admonition>

## Configurar para usar IBM Cloud {#set-up-to-use-ibm-cloud}

### Antes de comenzar {#before-you-begin}

1. Acepta la invitación recibida por correo electrónico.
2. Inicia sesión con un IBMid o una cuenta de Google. Ten en cuenta que el perfil que usas para iniciar sesión en IBM Cloud o IBM Quantum Platform en este paso no es tu cuenta de IBM Cloud. Después de iniciar sesión, confirma tu información y acepta la invitación para unirte a la cuenta.
3. Navega de regreso a [IBM Quantum Platform.](https://quantum.cloud.ibm.com)
4. Asegúrate de que estén seleccionadas la cuenta y la región correctas en el selector de cuenta en el encabezado, como se muestra en la siguiente imagen.
    * Elige la cuenta que fue invitada en el correo electrónico.
    * La región controla dónde se ejecutan tus trabajos y dónde se guardan los datos de los mismos. Puedes acceder a cualquiera de las regiones usando la misma clave de API, pero solo puedes ver y acceder a las instancias que fueron creadas en la región en la que has iniciado sesión.

    ![El encabezado de IBM Quantum Platform. El selector de cuenta está a la derecha de la barra de búsqueda. El selector de región está a la derecha del selector de cuenta.](/docs/images/guides/cloud-setup/IQP-Header.svg 'IBM Quantum Platform header')

### 2. Opcional: Guarda tus credenciales de acceso {#optional-save-your-access-credentials}

Si estás usando un entorno de Python de confianza, como una computadora portátil personal, se recomienda que uses el método `save_account()` para [guardar tus credenciales localmente](/guides/save-credentials).
