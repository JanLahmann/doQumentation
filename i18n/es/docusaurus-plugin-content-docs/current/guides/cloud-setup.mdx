# Configura tu cuenta de IBM Cloud {#set-up-your-ibm-cloud-account}

Si se te envió por correo electrónico una invitación para unirte a una cuenta, sigue los pasos en [Configura tu cuenta de IBM Cloud - usuarios invitados](/guides/cloud-setup-invited) en su lugar.

Puedes ejecutar cargas de trabajo en unidades de procesamiento cuántico (QPUs) de IBM&reg; configurando una cuenta en IBM Cloud&reg;. También requerirás [instalar y configurar Qiskit y Qiskit Runtime](/guides/install-qiskit#local).

Tu cuenta de usuario está asociada con una o más [instancias,](./instances) identificadas por _Nombres de Recursos en la Nube (Cloud Resource Names, CRNs)_ que dan acceso a los servicios de IBM Quantum. Además, a cada cuenta se le asigna un token de API único (también llamado clave o key), lo que permite el acceso a IBM Quantum desde Qiskit. Para obtener instrucciones sobre cómo elegir una instancia específica, consulta [Conectarse a una instancia](./instances#connect-instance).

<span id="cloud"></span>
## Configurar para usar IBM Cloud {#set-up-to-use-ibm-cloud}

### Antes de comenzar {#before-you-begin}

1. Si aún no tienes una, configura una [cuenta de IBM Cloud](https://quantum.cloud.ibm.com/registration).
   <Admonition type="note">
   Puedes formar parte de múltiples cuentas de IBM Cloud. Puedes acceder a cualquiera de tus cuentas de IBM Cloud en cualquier momento desde el selector de cuenta en el encabezado de la interfaz de [IBM Quantum Platform](https://quantum.cloud.ibm.com).
   </Admonition>
2. Inicia sesión en [IBM Quantum Platform](https://quantum.cloud.ibm.com) con un IBMid o una cuenta de Google. Si no tienes una cuenta, se te guiará para crear una. (Estas credenciales de inicio de sesión no son las credenciales de tu cuenta de IBM Cloud).
3. Asegúrate de que estén seleccionadas la cuenta y la región correctas en el selector de la cuenta en el encabezado, como se muestra en la siguiente imagen.

    La región controla dónde se ejecutan tus trabajos y dónde se guardan los datos de los mismos. Puedes acceder a cualquiera de la regiones usando la misma clave de API, pero solo puedes ver y acceder a las instancias que fueron creadas en la región en la que has iniciado sesión.

    ![El encabezado de IBM Quantum Platform. El selector de cuenta está a la derecha de la barra de búsqueda. El selector de región está a la derecha del selector de cuenta.](/docs/images/guides/cloud-setup/IQP-Header.svg 'IBM Quantum Platform header')

### 1. Crear una instancia {#create-an-instance}

Puedes crear una instancia desde la página de [Instancias (Instances)](https://quantum.cloud.ibm.com/instances) de IBM Quantum Platform. Consulta [Crear una instancia](/guides/instances#add-instance) para obtener instrucciones detalladas.

### 2. Opcional: Guarda tus credenciales de acceso {#optional-save-your-access-credentials}

Si estás utilizando un entorno de Python de confianza, como una computadora portátil personal, se recomienda que uses el método `save_account()` para [guardar tus credenciales localmente](/guides/save-credentials).

<span id="cloud-save"></span>
### 3. Conecta Qiskit con tu instancia de servicio {#connect-qiskit-with-your-service-instance}

- [Inicializa el servicio](/guides/initialize-account) en un **entorno de Python de confianza**
- [Inicializa el servicio en un **entorno no confiable**](/guides/cloud-setup-untrusted) (como una computadora pública)

<Admonition type="note" title="Notas">
   * Sigue [estas instrucciones](/guides/cloud-setup-rest-api) si deseas conectarte usando la API REST en lugar de usar Qiskit.
   * Si es necesario, usa [esta información](/guides/quickstart-steps-org#firewall) para configurar tu firewall para permitir el acceso a los puntos de enlace (endpoints) de la API de IBM Quantum.
</Admonition>
