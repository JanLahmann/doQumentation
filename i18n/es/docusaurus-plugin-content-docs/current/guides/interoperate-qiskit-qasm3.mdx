## Importar un programa OpenQASM 3 en Qiskit {#import-an-openqasm-3-program-into-qiskit}

Para usar esta función, debes instalar el paquete `qiskit_qasm3_import`. Instálalo con el siguiente comando.

```bash
pip install qiskit-qasm3-import
```

Actualmente hay dos funciones de alto nivel disponibles para importar desde OpenQASM 3 a Qiskit. Estas funciones son `load()`, que recibe un nombre de archivo, y `loads()`, que recibe el programa directamente como cadena de texto:

```python
import qiskit.qasm3
qiskit.qasm3.load(file_name)
qiskit.qasm3.loads(program_string)
```
En este ejemplo, definimos un programa cuántico usando OpenQASM 3 y usamos `loads()` para convertirlo directamente en un QuantumCircuit:

```python
import qiskit.qasm3

program = """
    OPENQASM 3.0;
    include "stdgates.inc";

    input float[64] a;
    qubit[3] q;
    bit[2] mid;
    bit[3] out;

    let aliased = q[0:1];

    gate my_gate(a) c, t {
      gphase(a / 2);
      ry(a) c;
      cx c, t;
    }
    gate my_phase(a) c {
      ctrl @ inv @ gphase(a) c;
    }

    my_gate(a * 2) aliased[0], q[{1, 2}][0];
    measure q[0] -> mid[0];
    measure q[1] -> mid[1];

    while (mid == "00") {
      reset q[0];
      reset q[1];
      my_gate(a) q[0], q[1];
      my_phase(a - pi/2) q[1];
      mid[0] = measure q[0];
      mid[1] = measure q[1];
    }

    if (mid[0]) {
      let inner_alias = q[{0, 1}];
      reset inner_alias;
    }

    out = measure q;
"""
circuit = qiskit.qasm3.loads(program)
circuit.draw("mpl")
```

## Exportar a OpenQASM 3 {#export-to-openqasm-3}

Puedes exportar código de Qiskit a OpenQASM 3 con `dumps()`, que exporta a una cadena de texto, o `dump()`, que exporta a un archivo.

### Ejemplo con `dumps()` {#example-with-dumps}

```python
from qiskit import QuantumCircuit
from qiskit.qasm3 import dumps

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

dumps(qc)
```

```text
'OPENQASM 3.0;\ninclude "stdgates.inc";\nbit[2] meas;\nqubit[2] q;\nh q[0];\ncx q[0], q[1];\nbarrier q[0], q[1];\nmeas[0] = measure q[0];\nmeas[1] = measure q[1];\n'
```

### Ejemplo con `dump()` {#example-with-dump}

```python
from qiskit import QuantumCircuit
from qiskit.qasm3 import dump

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

f = open("my_file.txt", "w")
dump(qc, f)
f.close()
```

Para más información, consulta la sección [Exportar a OpenQASM 3](https://docs.quantum.ibm.com/api/qiskit/qasm3#exporting-to-openqasm-3) de la referencia de la API.
