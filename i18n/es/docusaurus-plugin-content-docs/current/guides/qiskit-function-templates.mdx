# Introducción a las plantillas de Qiskit Function {#introduction-to-qiskit-function-templates}

Las [plantillas de Qiskit Function](https://github.com/qiskit-community/qiskit-function-templates) son una colección de código abierto de ejemplos de código realistas que te dan acceso a algunos de los flujos de trabajo de investigación más recientes, aprovechando herramientas como el SDK de Qiskit y los [complementos de Qiskit](/guides/addons). Estos flujos de trabajo predefinidos toman información de experimentos prácticos y los empaquetan de forma que puedas personalizarlos y reutilizarlos en tu propia investigación sin necesidad de construirlos desde cero. Todas las plantillas se pueden desplegar fácilmente en [Qiskit Serverless](./serverless), donde puedes aprovechar el cómputo clásico como parte de tu flujo de trabajo.

Hay dos tipos de plantillas:

- *Implementaciones de plantillas*: Ejemplos de código especializados que implementan técnicas avanzadas de experimentos del mundo real en áreas como química y física.
- *Plantillas base*: Referencias para desarrollar tu propia Qiskit Function original. Estas plantillas destacan las mejores prácticas en desarrollo de interfaces, formato de código, pruebas unitarias y más.

## Implementaciones de plantillas {#template-implementations}

Las implementaciones de plantillas de Qiskit Function están organizadas por área de aplicación. Actualmente, la colección incluye una plantilla de física para la simulación hamiltoniana usando el [complemento AQC-Tensor de Qiskit](./qiskit-addons-aqc) y una plantilla de química para la estructura electrónica con el modelo de solvente implícito usando el [complemento SQD de Qiskit](./qiskit-addons-sqd).
Los recursos para comenzar con estas dos plantillas están disponibles en los siguientes enlaces:

- Simulación de estructura electrónica con modelo de solvente implícito: [archivos fuente de la plantilla](https://github.com/qiskit-community/qiskit-function-templates/tree/main/chemistry/sqd_pcm) y [guía](./function-template-chemistry-workflow)
- Simulación hamiltoniana: [archivos fuente de la plantilla](https://github.com/qiskit-community/qiskit-function-templates/tree/main/physics/hamiltonian_simulation) y [guía](./function-template-hamiltonian-simulation)

## Plantillas base {#base-templates}

Las *plantillas base* son referencias para desarrollar tu propia Qiskit Function original. Están diseñadas para ayudarte a iniciar el proceso de desarrollo ilustrando las mejores prácticas en desarrollo de interfaces, formato de código, manejo de entrada/salida, mensajes de error, puntos de control, pruebas unitarias y más. El repositorio está expuesto como una [plantilla de GitHub](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-template-repository) para que la implementación de CI/CD pueda extenderse fácilmente.

Actualmente hay dos plantillas: una [plantilla de función de circuito](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/circuit_function_template.py) y una [plantilla de función de aplicación](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/application_function_template.py). La plantilla de función de circuito está construida de forma similar a la [IBM Circuit Function](./ibm-circuit-function) y la plantilla de función de aplicación ilustra cómo estructurar un flujo de trabajo de aplicación típico.
