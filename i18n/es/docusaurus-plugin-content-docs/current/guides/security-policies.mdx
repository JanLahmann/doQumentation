# Políticas de seguridad del repositorio {#repository-security-policies}

Es de vital importancia que mantengas tus sistemas tan seguros como sea posible. Por ello, se recomienda encarecidamente que uses herramientas de seguridad como [Dependabot](https://github.com/dependabot), que te notifican automáticamente cuando las dependencias están desactualizadas.
