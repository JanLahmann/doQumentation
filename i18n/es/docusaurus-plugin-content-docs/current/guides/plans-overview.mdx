# Descripción general de planes {#plans-overview}

IBM Quantum Platform tiene planes Open (acceso gratuito) y planes de pago, como se describe a continuación. Consulta [IBM Quantum access plans](https://www.ibm.com/quantum/pricing) para todos los detalles.

Si deseas acceder a varios planes, puedes crear múltiples instancias y asignarles distintos planes. Por ejemplo, puedes tener una instancia del Plan Open y una instancia del Plan de Pago por uso. Puedes establecer las instancias preferidas en el inicializador de `QiskitRuntimeService` o en el método `QiskitRuntimeService.save_account()`. Ejemplo:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Plan Open {#open-plan}

Ejecuta tus circuitos cuánticos en las mejores QPUs del mundo de forma gratuita (hasta 10 minutos por ventana móvil de 28 días). Puedes hacer seguimiento de tu uso en las páginas [Panel de la plataforma](https://quantum.cloud.ibm.com/) y [Cargas de trabajo](https://quantum.cloud.ibm.com/workloads). Las instancias del Plan Open solo pueden crearse en la región us-east (puedes cambiar de región desde el menú desplegable en la parte superior del Panel de la plataforma). [Haz clic aquí para ir a la página de registro](https://quantum.cloud.ibm.com/registration).

Este plan es recomendado para quienes están aprendiendo computación cuántica y explorando la tecnología cuántica de IBM.

<Admonition type="note" title="Promoción por tiempo limitado">
A partir del 16 de marzo de 2026, los usuarios activos del Plan Open pueden optar por recibir 180 minutos adicionales durante los próximos 12 meses.
Lee el blog [Doubling down on open-access quantum computing](https://www.ibm.com/quantum/blog/open-plan-updates) para obtener todos los detalles.
</Admonition>

## Plan Flex {#flex-plan}

Compra previamente una cantidad determinada de minutos (al menos 400) para ejecutar trabajos cuánticos, que deben utilizarse dentro del año posterior a la compra. Puedes adquirir más minutos en cualquier momento.

Tras comprar tiempo, debes asignar los minutos a instancias existentes o nuevas del Plan Flex para tu cuenta. Establece o actualiza los límites de asignación en la [página de Instancias](https://quantum.cloud.ibm.com/instances) o configura la asignación cuando [crees una nueva instancia del Plan Flex](/guides/instances). Si asignas todos los minutos a las instancias existentes, no podrás crear una nueva instancia para el Plan Flex hasta que compres más minutos o retires algunos minutos de la asignación de otra instancia. Consulta [Establecer límites de asignación de instancias](allocation-limits) para obtener más información.

Si necesitas más de 10 000 minutos, deberías considerar una suscripción al Plan Premium. Si no deseas comprar 400 minutos, considera el Plan de Pago por uso.

[Contacta con IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) para suscribirte a un Plan Flex.

## Plan de Pago por uso {#pay-as-you-go-plan}

Ejecuta circuitos cuánticos en las mejores QPUs del mundo y paga solo por el consumo de QPU que utilices.

Este plan es recomendado para quienes realizan proyectos de investigación de utilidad cuántica y prueban modelos de negocio con acceso flexible. Si vas a usar al menos 400 minutos en el próximo año, se recomienda que utilices el Plan Flex.

## Plan Premium {#premium-plan}

Ejecuta circuitos cuánticos en las mejores QPUs del mundo con una suscripción empresarial. Además, algunas funcionalidades, como [Qiskit Functions](functions) y [Qiskit Transpiler as a Service](qiskit-transpiler-service), están disponibles para los usuarios del Plan Premium.

Este plan es recomendado para quienes desean ejecutar una hoja de ruta cuántica estratégica y desarrollar algoritmos y aplicaciones cuánticas a escala. [Contacta con IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) para obtener más información.

## Plan On-Prem {#on-prem-plan}

Accede a un sistema cuántico dedicado completamente en las instalaciones del cliente, con mantenimiento y servicio a cargo de IBM Quantum.

Este servicio es recomendado para quienes exploran algoritmos y aplicaciones cuánticas clásicas y necesitan un alto control sobre sus recursos y datos. [Contacta con IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) para obtener más información.
