# Instalar el cliente de Qiskit Runtime desde el código fuente {#install-the-qiskit-runtime-client-from-source}

Cuando instalas Qiskit Runtime desde el código fuente, puedes acceder a la versión de desarrollo actual, en lugar de usar la versión del repositorio Python Package Index (PyPI). Por lo tanto, puedes inspeccionar y ampliar la última versión del código de manera más eficiente.

## Crear y activar un nuevo entorno virtual {#create-and-activate-a-new-virtual-environment}

1. Navega al directorio de tu proyecto y crea un entorno mínimo con solo Python instalado.

        <OperatingSystemTabs>
          <TabItem value="mac" label="macOS">
            ```shell
            python3 -m venv .venv
            ```
          </TabItem>

          <TabItem value="linux" label="Linux">

          <TabItem value="win" label="Windows">

         </OperatingSystemTabs>

1. Activa tu nuevo entorno.

         <OperatingSystemTabs>
           <TabItem value="mac" label="macOS">

## Instalar Qiskit Runtime {#install-qiskit-runtime}

Sigue estos pasos para instalar Qiskit Runtime:

1. Clona el repositorio de Qiskit Runtime.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. Cambia al directorio `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. Instala `qiskit-runtime`. Recomendamos usar un [entorno virtual](https://docs.python.org/3/library/venv.html) para evitar contaminar tu instalación global de Python.

  * **Instalación estándar**:

    ```bash
    pip install .
    ```

  * **Modo editable**: En este modo, no necesitas reinstalar Qiskit cuando haya cambios de código en el proyecto.

    ```bash
    pip install -e .
    ```

    En modo editable, las extensiones compiladas se construyen en _modo de depuración_ sin optimizaciones.

4. (Opcional) Si quieres ejecutar pruebas o verificaciones de linting, instala los requisitos de desarrollador. Recomendamos usar un [entorno virtual](https://docs.python.org/3/library/venv.html) para evitar contaminar tu instalación global de Python.

```bash
pip install -e ".[dev]"
```
