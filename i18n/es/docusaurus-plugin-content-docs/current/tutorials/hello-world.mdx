Construye un [estado de Bell](https://en.wikipedia.org/wiki/Bell_state) (dos qubits entrelazados) y ejecútalo de tres maneras:

1. **Simulación ideal** — resultados perfectos, sin cuenta necesaria
2. **Simulación con ruido** — simula un dispositivo real, sin cuenta necesaria
3. **Hardware cuántico real** — requiere una cuenta de IBM Quantum gratuita (pasos de configuración a continuación)
## Construir el circuito {#build-the-circuit}

```python
# Added by doQumentation — required packages for this notebook
!pip install -q qiskit qiskit-ibm-runtime
```

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

qc.draw(output="mpl")
```

## Opción 1: Simulación ideal (sin cuenta necesaria) {#option-1-ideal-simulation-no-account-needed}

Usa `StatevectorSampler` — un simulador local con resultados perfectos, sin ruido.

```python
from qiskit.primitives import StatevectorSampler

result = StatevectorSampler().run([qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
from qiskit.visualization import plot_histogram
plot_histogram(counts)
```

## Opción 2: Simulación con ruido (sin cuenta necesaria) {#option-2-noisy-simulation-no-account-needed}

Usa `FakeManilaV2` — un simulador local que imita un dispositivo cuántico real de IBM, incluyendo sus características de ruido. El circuito debe transpilarse primero (adaptarse) al conjunto de puertas y la conectividad de qubits del dispositivo.

```python
from qiskit_ibm_runtime import SamplerV2
from qiskit_ibm_runtime.fake_provider import FakeManilaV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

backend = FakeManilaV2()
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```

## Opción 3: Hardware cuántico real {#option-3-real-quantum-hardware}

Requiere una cuenta de IBM Quantum gratuita. Para configurar una:

1. Regístrate en [quantum.cloud.ibm.com/registration](https://quantum.cloud.ibm.com/registration) — no se requiere tarjeta de crédito durante los primeros 30 días
2. Inicia sesión en [quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) y selecciona la región **us-east** (requerida para el Plan Abierto gratuito)
3. Crea una instancia (Plan Abierto gratuito) en [Instances](https://quantum.cloud.ibm.com/instances) si aún no tienes una
4. Crea una clave de API en [quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) (o en [cloud.ibm.com/iam/apikeys](https://cloud.ibm.com/iam/apikeys))
5. Copia tu **CRN** (Cloud Resource Name) desde tu página de [Instances](https://quantum.cloud.ibm.com/instances)

Luego, si aún no has guardado tus credenciales en esta sesión de Binder, ejecuta la siguiente celda. Reemplaza `<your-api-key>` con la clave de API del paso 4, y `<your-crn>` con el CRN del paso 5.

```python
from qiskit_ibm_runtime import QiskitRuntimeService

QiskitRuntimeService.save_account(
    channel="ibm_quantum_platform",
    token="<your-api-key>",
    instance="<your-crn>",
    set_as_default=True,
    overwrite=True,
)
```

**Nota:** Los trabajos en hardware real pueden tardar algún tiempo dependiendo de los tiempos de cola. Si la celda aún se está ejecutando, puedes verificar el estado de tu trabajo y ver los resultados en [quantum.cloud.ibm.com/workloads](https://quantum.cloud.ibm.com/workloads?user=me).

```python
from qiskit_ibm_runtime import QiskitRuntimeService, SamplerV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

service = QiskitRuntimeService()
backend = service.least_busy(operational=True, simulator=False)
print(f"Running on {backend.name}")

pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```
