# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) adalah paket ekstensi resmi yang menyediakan algoritma dan teknik komputasi kuantum yang terspesialisasi. Setiap addon berfokus pada area tertentu dan dilengkapi tutorial dengan kode yang bisa dijalankan.

## Addon yang Tersedia {#available-addons}

| Addon | Paket | Deskripsi |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Kurangi lebar dan kedalaman Circuit dengan memotong gate dan wire |
| **SQD** | `qiskit-addon-sqd` | Sample-based Quantum Diagonalization untuk estimasi ground state |
| **OBP** | `qiskit-addon-obp` | Operator Back-Propagation untuk mengurangi kedalaman Circuit |
| **MPF** | `qiskit-addon-mpf` | Multi-Product Formulas untuk simulasi Hamiltonian yang lebih baik |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximate Quantum Compilation dengan tensor network |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption untuk mitigasi error |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones untuk mengurangi overhead sampling PEC |

Semua tutorial bisa dijalankan langsung di browser menggunakan Binder, atau di server Jupyter milikmu sendiri.

Untuk dokumentasi API, kunjungi repo addon masing-masing di [GitHub](https://github.com/Qiskit?q=qiskit-addon).
