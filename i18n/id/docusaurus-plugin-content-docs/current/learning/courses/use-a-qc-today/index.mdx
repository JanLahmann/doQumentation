# Ikhtisar {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

Selamat datang di *Gunakan komputer kuantum hari ini*! Di kursus ini, kamu langsung mulai menggunakan komputer kuantum dan mempelajari prinsip-prinsip dasar komputasi kuantum sepanjang perjalanan.

Kursus ini dirancang untuk orang yang penasaran dan yang sibuk. Kamu tidak perlu gelar fisika atau pengalaman pemrograman bertahun-tahun. Kalau kamu bisa mengikuti notebook dan klik "Run," kamu sudah siap.
