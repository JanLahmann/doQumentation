# Gambaran Umum {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Kursus ini mengeksplorasi bagaimana komputasi kuantum dapat memainkan peran menarik dalam machine learning. Kursus ini akan menawarkan beberapa ulasan pendekatan machine learning klasik, tapi juga mengasumsikan kamu sudah familiar dengan metode-metode ini. Fokus utamanya adalah bagaimana komputasi kuantum bisa cocok dengan alur kerja machine learning yang sudah ada. Topik mencakup pengkodean data, quantum kernels, dan variational quantum circuits.

## Ujian {#exam}
