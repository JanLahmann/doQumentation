## Uji pengetahuanmu {#test-your-knowledge}

Setelah menyelesaikan bacaan dan aktivitas dalam kursus ini, klik tombol di bawah untuk navigasi ke IBM&reg; Training, di mana kamu bisa mengikuti ujian lencana untuk Quantum machine learning.

Ujian terdiri dari 20 pertanyaan, dan kamu harus menjawab minimal 16 dengan benar untuk lulus. Setelah mendapatkan nilai lulus, lencana kamu akan otomatis dikirimkan ke email yang kamu gunakan untuk mendaftar di IBM Training.

Pastikan kamu siap untuk mengikuti ujian. Jika kamu tidak mendapat nilai lulus dalam dua percobaan, ujian akan terkunci sementara.

Pemberitahuan: Ujian-ujian di situs IBM Training mungkin tidak tersedia sementara dalam beberapa hari ke depan, karena pemeliharaan.
