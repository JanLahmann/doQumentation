# Gambaran Umum {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Kursus ini ditujukan untuk individu yang memiliki beberapa pengalaman dengan komputasi kuantum dan ingin belajar cara menulis algoritma variasional menggunakan Qiskit Runtime primitives. Ini juga cocok untuk para profesional komputasi kuantum yang ingin memperluas pengetahuan dan keterampilan mereka di bidang algoritma variasional.

## Ujian {#exam}

Untuk mendapatkan lencana Desain algoritma variasional, ikuti ujian di IBM&reg; Training. Ujian ini dimaksudkan untuk diikuti setelah membaca pelajaran-pelajaran dalam kursus ini. Setelah kamu lulus ujian, kamu akan diberitahu oleh Credly bahwa kamu telah mendapatkan lencana.
