# Ujian dan lencana {#exam-and-badge}

![A square badge stating the name of this course, advanced level, and issued by IBM Quantum&reg;.](/learning/images/courses/foundations-of-quantum-error-correction/exam/badge.avif)

## Survei pasca-kursus {#post-course-survey}

Selamat telah menyelesaikan kursus "Fondasi koreksi kesalahan kuantum" dari IBM! Tolong luangkan waktu sebentar untuk membantu kami meningkatkan kursus ini dengan mengisi [survei singkat](https://your.feedback.ibm.com/jfe/form/SV_9WuCWHSXP8rXo7Y) berikut. Masukan kamu akan digunakan untuk meningkatkan konten dan pengalaman pengguna kami. Terima kasih!

## Uji pengetahuanmu {#test-your-knowledge}

Setelah menyelesaikan bacaan dan aktivitas dalam kursus ini, klik tombol di bawah untuk menuju ke IBM&reg; Training, tempat kamu bisa mengikuti ujian lencana untuk Fondasi koreksi kesalahan kuantum.

Ujian terdiri dari 20 pertanyaan, dan kamu harus menjawab setidaknya 16 dengan benar untuk lulus. Setelah mendapat nilai lulus, lencana akan otomatis dikirimkan ke email yang kamu gunakan untuk mendaftar di IBM&reg; Training.

Pastikan kamu siap untuk mengikuti ujian. Jika kamu tidak mendapat nilai lulus dalam dua percobaan, ujian akan terkunci sementara.
