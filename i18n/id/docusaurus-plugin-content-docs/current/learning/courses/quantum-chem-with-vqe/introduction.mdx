Kursus ini adalah pengantar terfokus untuk variational quantum eigensolver (VQE), termasuk penerapan VQE pada kimia. Kursus ini ditujukan bagi praktisi teknis komputasi kuantum yang tertarik menerapkan komputasi kuantum pada masalah kimia, biokimia, atau biofisika. Kursus ini mencakup elemen-elemen penting dari perhitungan VQE, termasuk berikut ini:

* Hamiltonian
* Ansatz
* Optimizer klasik

Semua elemen ini digabungkan untuk membangun perhitungan VQE pada Qiskit Runtime Primitives, terutama Estimator. Untuk informasi tentang primitif, lihat [dokumentasi](/guides/primitives). Dalam menyiapkan perhitungan VQE, kita akan mengikuti kerangka Qiskit patterns standar:

* Memetakan input klasik ke masalah kuantum
* Mengoptimalkan masalah untuk eksekusi kuantum
* Mengeksekusi menggunakan Qiskit Runtime primitives
* Pasca-pemrosesan, mengembalikan hasil dalam format klasik

Beberapa penjelasan tentang kerangka ini akan ditemukan di seluruh kursus ini, tetapi untuk penjelasan lengkap dengan contoh Qiskit patterns, lihat [dokumentasi ini](/guides/intro-to-patterns).

Kami menyarankan untuk merencanakan sekitar 6-10 jam untuk menyelesaikan kursus ini. Video dan bacaan kemungkinan membutuhkan dua jam untuk diselesaikan. Waktu yang tersisa untuk mengeksekusi dan memodifikasi sel kode, menjawab pertanyaan kuis, dan membaca dokumentasi yang ditautkan.
