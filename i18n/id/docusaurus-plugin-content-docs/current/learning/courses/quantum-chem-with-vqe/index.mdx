# Ikhtisar {#overview}

!["Hero image"](/learning/images/courses/quantum-chem-with-vqe/hero.svg)

Selamat datang di **Kimia kuantum dengan VQE**. Variational quantum eigensolver (VQE) adalah algoritma kuantum dengan aplikasi dalam kimia kuantum dan simulasi. Dalam kursus pengantar ini, Anda akan mempelajari blok-blok bangunan dasar VQE, pilihan dalam implementasinya, dan beberapa aplikasi. Konten disampaikan melalui kombinasi teks, video, dan kode.

Topik-topik utama meliputi:

    - Hamiltonian
    - Ansaetze
    - Optimizer klasik
    - Menemukan energi keadaan dasar menggunakan pola Qiskit
    - Menemukan panjang ikatan menggunakan pola Qiskit

Kursus ini berfokus pada aplikasi kimia kuantum, meskipun banyak blok bangunan yang sama (Circuit ansatz, optimizer klasik, dan kerangka pola Qiskit) dapat diterapkan pada berbagai bidang di luar kimia kuantum.

## Sumber daya berguna {#helpful-resources}

Pengantar VQE ini dirancang untuk peserta tingkat menengah dalam teknik komputasi kuantum. Tidak diperlukan latar belakang dalam VQE. Anda dianggap sudah familier dengan Qiskit, serta setidaknya mengenal mekanika kuantum dan matematika terkait (seperti operator). Kami menyarankan untuk meluangkan sekitar 6-10 jam guna menyelesaikan konten ini, tergantung pengalaman Anda sebelumnya.

    - Kursus tentang [Basics of Quantum Information](/learning/courses/basics-of-quantum-information/index) mungkin berguna untuk memberikan latar belakang dan dasar-dasar informasi dan komputasi kuantum.
