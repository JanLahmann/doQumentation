# Pengantar primitif {#introduction-to-primitives}

Primitif dibuat untuk menyederhanakan tugas paling umum pada komputer kuantum: yaitu, sampling state kuantum dan menghitung nilai ekspektasi. Primitif Qiskit Runtime pertama ([`EstimatorV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2) dan [`SamplerV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2)) adalah implementasi dari [kelas dasar primitif](/guides/primitives) Qiskit. Keduanya menyediakan implementasi yang lebih canggih (misalnya, dengan menyertakan mitigasi error) sebagai layanan berbasis cloud dan digunakan untuk mengakses hardware IBM Quantum&reg;.

Primitif Qiskit Runtime terbaru, [Executor](https://qiskit.github.io/qiskit-ibm-runtime/stubs/qiskit_ibm_runtime.Executor.html) (yang masih dalam beta), menyediakan antarmuka tingkat lebih rendah yang memberikan visibilitas dan kontrol lebih tanpa mengorbankan performa.
