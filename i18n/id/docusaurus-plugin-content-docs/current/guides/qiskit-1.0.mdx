# Panduan migrasi Qiskit 1.0 {#qiskit-10-migration-guide}

Qiskit 1.0 hadir dengan jaminan stabilitas baru dan peningkatan performa, tapi sayangnya memutus kompatibilitas dengan semua versi Qiskit sebelumnya. Lihat panduan-panduan berikut untuk memperbarui environment dan kode kamu agar bisa bekerja dengan Qiskit v1.0.

- [Instal paket Qiskit v1.0 yang baru](/guides/qiskit-1.0-installation) untuk upgrade environment Python dan dependensi proyek.
- [Perubahan fitur Qiskit v1.0](/guides/qiskit-1.0-features) untuk memperbarui kode kamu.
- [Perubahan paket yang memutus kompatibilitas di Qiskit v1.0](/guides/metapackage-migration) untuk memahami struktur paket baru dan lama.

Baca lebih lanjut tentang manfaat Qiskit v1.0 di [blog IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
