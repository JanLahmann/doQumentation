# Instal Qiskit Runtime client {#install-the-qiskit-runtime-client}

Qiskit Runtime adalah layanan berbasis cloud untuk menjalankan komputasi kuantum di hardware IBM Quantum&reg;. Panduan ini menunjukkan cara menginstal `qiskit-ibm-runtime`, yaitu paket client untuk mengakses layanan Qiskit Runtime.

Sebelum menginstal Qiskit Runtime client, ikuti instruksi di panduan-panduan berikut:

1. [Instal Qiskit](/guides/install-qiskit).
2. [Atur akun IBM Cloud-mu](cloud-setup).

Jalankan perintah ini untuk menginstal Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Tetap update dengan versi terbaru">
Cek secara berkala [catatan rilis Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) untuk melihat rilis baru. Kami menyarankan untuk sering memperbarui kebutuhanmu untuk `qiskit-ibm-runtime` dengan menjalankan `pip install -U qiskit-ibm-runtime`.
</Admonition>
