# Mulai dengan primitif backend {#get-started-with-the-backend-primitives}

Tidak seperti primitif khusus provider, primitif backend adalah implementasi generik yang bisa digunakan dengan objek `backend` apa pun, selama mengimplementasikan antarmuka [`BackendV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.providers.BackendV2). Beberapa provider mengimplementasikan primitif secara native. Lihat [halaman Ekosistem Qiskit](https://www.ibm.com/quantum/ecosystem?tag=provider) untuk detailnya.

<span id="estimator"></span>
## Mulai dengan primitif backend Estimator {#get-started-with-the-estimator-backend-primitive}

Primitif Estimator bisa dijalankan dengan provider mana pun menggunakan kelas [`qiskit.primitives.BackendEstimatorV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendEstimatorV2). Namun, primitif ini tidak menyediakan implementasi mitigasi error pengukuran atau gate "out-of-the-box", karena primitif backend dirancang untuk berjalan secara lokal di mesin pengguna.

### Contoh: {#example}

```python
from qiskit.primitives import BackendEstimatorV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
estimator = BackendEstimatorV2(backend)
```

<span id="sampler"></span>
## Mulai dengan primitif backend Sampler {#get-started-with-the-sampler-backend-primitive}

Primitif Sampler bisa dijalankan dengan provider mana pun menggunakan [`qiskit.primitives.BackendSamplerV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendSamplerV2). Namun, primitif ini membutuhkan backend yang mendukung opsi `memory`.

### Contoh: {#example-1}

```python
from qiskit.primitives import BackendSamplerV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
sampler = BackendSamplerV2(backend)
```
