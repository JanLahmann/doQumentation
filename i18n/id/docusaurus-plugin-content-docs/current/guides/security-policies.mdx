# Kebijakan keamanan repositori {#repository-security-policies}

Sangat penting untuk menjaga sistem kamu seaman mungkin. Oleh karena itu, sangat disarankan agar kamu menggunakan alat keamanan seperti [Dependabot](https://github.com/dependabot) yang secara otomatis memberi tahu kamu ketika dependensi sudah usang. Lihat kebijakan keamanan untuk repositori GitHub berikut yang terkait dengan IBM Quantum&reg;:

- [Qiskit](https://github.com/Qiskit/qiskit/security)
- [Qiskit IBM Runtime](https://github.com/Qiskit/qiskit-ibm-runtime/security)
- [Addon Qiskit pemotong Circuit](https://github.com/Qiskit/qiskit-addon-cutting/security)
- [Addon Qiskit pemetaan fermionic](https://github.com/Qiskit/qiskit-fermions/blob/main/SECURITY.md)
- [Addon Qiskit formula multi-produk (MPF)](https://github.com/Qiskit/qiskit-addon-mpf/security)
- [Addon Qiskit backpropagasi operator (OBP)](https://github.com/Qiskit/qiskit-addon-obp/security)
- [Addon Qiskit pemetaan optimasi](https://github.com/Qiskit/qiskit-addon-opt-mapper/blob/main/SECURITY.md)
- [Addon Qiskit propagasi Pauli](https://github.com/Qiskit/pauli-prop/blob/main/SECURITY.md)
- [Addon Qiskit penyerapan noise yang dipropagasikan (PNA)](https://github.com/Qiskit/qiskit-addon-pna/blob/main/SECURITY.md)
- [Utilitas addon Qiskit](https://github.com/Qiskit/qiskit-addon-utils/security)
- [Addon Qiskit diagonalisasi kuantum berbasis sampel (SQD)](https://github.com/Qiskit/qiskit-addon-sqd/security)
- [Addon Qiskit shaded lightcones](https://github.com/Qiskit/qiskit-addon-slc/blob/main/SECURITY.md)
