# Instal Qiskit Runtime client dari sumber {#install-the-qiskit-runtime-client-from-source}

Saat kamu menginstal Qiskit Runtime dari sumber, kamu bisa mengakses versi pengembangan saat ini, bukan menggunakan versi yang ada di repositori Python Package Index (PyPI). Dengan demikian kamu bisa memeriksa dan memperluas versi terbaru kode dengan lebih efisien.

## Buat dan aktifkan lingkungan virtual baru {#create-and-activate-a-new-virtual-environment}

1. Navigasikan ke direktori proyekmu dan buat lingkungan minimal yang hanya berisi Python.

        <OperatingSystemTabs>
          <TabItem value="mac" label="macOS">

          </TabItem>

          <TabItem value="linux" label="Linux">

          </TabItem>

          <TabItem value="win" label="Windows">

         </OperatingSystemTabs>

1. Aktifkan lingkungan barumu.

         <OperatingSystemTabs>
           <TabItem value="mac" label="macOS">

## Instal Qiskit Runtime {#install-qiskit-runtime}

Ikuti langkah-langkah berikut untuk menginstal Qiskit Runtime:

1. Clone repositori Qiskit Runtime.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. Pindah ke direktori `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. Instal `qiskit-runtime`. Kami menyarankan menggunakan [lingkungan virtual](https://docs.python.org/3/library/venv.html) untuk menghindari polusi pada instalasi Python global-mu.

  * **Instalasi standar**:

    ```bash
    pip install .
    ```

  * **Mode editable**: Dalam mode ini, kamu tidak perlu menginstal ulang Qiskit saat ada perubahan kode pada proyek.

    ```bash
    pip install -e .
    ```

    Dalam mode editable, ekstensi yang dikompilasi dibuat dalam _debug mode_ tanpa optimasi.

4. (Opsional) Jika kamu ingin menjalankan tes atau pemeriksaan linting, instal persyaratan developer. Kami menyarankan menggunakan [lingkungan virtual](https://docs.python.org/3/library/venv.html) untuk menghindari polusi pada instalasi Python global-mu.

```bash
pip install -e ".[dev]"
```
