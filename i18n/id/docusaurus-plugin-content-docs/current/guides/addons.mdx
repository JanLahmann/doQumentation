# Teknik lanjutan - Addon Qiskit {#advanced-techniques---qiskit-addons}

Addon Qiskit adalah kumpulan kemampuan riset untuk mendukung penemuan algoritma pada skala utilitas. Komponen perangkat lunak modular ini dibangun di atas fondasi performa Qiskit dan bisa dipasang ke dalam sebuah [alur kerja](/guides/intro-to-patterns) untuk mengembangkan atau merancang algoritma kuantum baru. Halaman ini menyoroti alat yang tersedia di berbagai kategori fungsional utama untuk membantu kamu memilih kemampuan yang relevan saat membangun alur kerja.
