# Pengantar template Qiskit Function {#introduction-to-qiskit-function-templates}

[Template Qiskit Function](https://github.com/qiskit-community/qiskit-function-templates) adalah kumpulan contoh kode realistis yang bersifat open-source, yang memberi Anda akses ke beberapa workflow riset terbaru yang memanfaatkan alat seperti Qiskit SDK dan [Qiskit addons](/guides/addons). Workflow yang sudah dibangun sebelumnya ini mengambil informasi dari eksperimen praktis dan mengemasnya dengan cara yang bisa Anda kustomisasi dan gunakan kembali dalam risetmu sendiri, sehingga Anda tidak perlu membangun dari nol. Semua template siap digunakan di [Qiskit Serverless](/guides/serverless) tempat Anda bisa dengan mudah memanfaatkan komputasi klasik sebagai bagian dari workflow-mu.

Ada dua jenis template:

- *Implementasi template*: Contoh kode khusus yang mengimplementasikan teknik-teknik canggih dari eksperimen dunia nyata di bidang seperti kimia dan fisika.
- *Template dasar*: Referensi untuk mengembangkan Qiskit Function originalmu sendiri. Template ini menonjolkan praktik terbaik dalam pengembangan antarmuka, pemformatan kode, pengujian unit, dan lainnya.

## Implementasi template {#template-implementations}

Implementasi template Qiskit Function diorganisir berdasarkan area aplikasi. Saat ini yang termasuk dalam koleksi adalah template fisika untuk simulasi Hamiltonian menggunakan [Qiskit addon AQC-Tensor](https://qiskit.github.io/qiskit-addon-aqc-tensor/) dan template kimia untuk struktur elektronik dengan model pelarut implisit menggunakan [Qiskit addon SQD](/guides/qiskit-addons-sqd).
Sumber daya untuk memulai dengan dua template ini tersedia di tautan berikut:

- Simulasi struktur elektronik dengan model pelarut implisit: [file sumber template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/chemistry/sqd_pcm) dan [panduan](/guides/function-template-chemistry-workflow)
- Simulasi Hamiltonian: [file sumber template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/physics/hamiltonian_simulation) dan [panduan](/guides/function-template-hamiltonian-simulation)

## Template dasar {#base-templates}

*Template dasar* adalah referensi untuk mengembangkan Qiskit Function originalmu sendiri. Template ini dirancang untuk membantumu memulai proses pengembangan dengan mengilustrasikan praktik terbaik dalam pengembangan antarmuka, pemformatan kode, penanganan I/O, pesan error, checkpoint, pengujian unit, dan lainnya. Repositori ini diekspos sebagai [template GitHub](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-template-repository) sehingga implementasi CI/CD bisa diperluas dengan mudah.

Saat ini ada dua template: [template circuit function](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/circuit_function_template.py) dan [template application function](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/application_function_template.py).
