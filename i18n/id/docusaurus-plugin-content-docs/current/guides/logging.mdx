# Logging untuk Qiskit Runtime {#logging-for-qiskit-runtime}

Layanan IBM Cloud&reg;, seperti Qiskit Runtime, menghasilkan log platform yang bisa kamu gunakan untuk menyelidiki aktivitas tidak biasa dan tindakan kritis di akunmu, serta memecahkan masalah.

Kamu bisa menggunakan [IBM Cloud Logs Routing,](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about) yang merupakan layanan platform, untuk merutekan log platform di akunmu ke tujuan yang kamu pilih dengan mengonfigurasi tenant yang menentukan ke mana log platform dikirim. Untuk informasi lebih lanjut, lihat [Tentang Logs Routing.](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about)

Kamu bisa menggunakan [IBM Cloud Logs](https://cloud.ibm.com/docs/cloud-logs) untuk memvisualisasikan dan memberi peringatan pada log platform yang dihasilkan di akunmu dan dirutekan oleh IBM Cloud Logs Routing ke instans IBM Cloud Logs.

## Lokasi tempat log platform dihasilkan {#locations-where-platform-logs-are-generated}

IBM Cloud Container Registry menghasilkan log platform yang ditampilkan di instans logging-mu. Tabel berikut mencantumkan lokasi di mana pengumpulan otomatis log layanan Container Registry diaktifkan.

| Lokasi                     | Apakah log layanan platform tersedia? |
|----------------------------|---------------------------------------|
| Washington, D.C. (us-east) |                  Ya                   |
| Frankfurt (eu-de)          |                  Ya                   |

## Log platform yang dihasilkan {#platform-logs-that-are-generated}

IBM Qiskit Runtime menghasilkan log layanan platform yang ditampilkan di instans logging-mu untuk kasus-kasus berikut:

- Saat status sebuah job berubah.

## Melihat log dari halaman Observability {#viewing-logs-from-the-observability-page}

Untuk informasi tentang cara membuka antarmuka IBM Cloud Logs, lihat **Launching the UI** di [dokumentasi IBM Cloud Logs.](https://cloud.ibm.com/docs/cloud-logs?topic=cloud-logs-instance-launch)

## Kolom berdasarkan tipe log {#fields-by-log-type}

Untuk informasi tentang kolom yang ada di setiap log platform, lihat [Fields that are included in platform logs.](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about-platform-logs#about-platform-logs-2)

Kolom-kolom berikut ada di dalam rekaman log.

| Kolom           | Tipe     | Deskripsi                                                                      |
|-----------------|----------|--------------------------------------------------------------------------------|
| logSourceCRN    | Required | Mendefinisikan akun dan instans flow log tempat log dipublikasikan.            |
| saveServiceCopy | Required | Mendefinisikan apakah IBM menyimpan salinan rekaman untuk keperluan operasional. |
| message         | Required | Deskripsi dari log yang dihasilkan.                                            |
