# Ikhtisar paket layanan yang tersedia {#overview-of-available-plans}

IBM Quantum Platform memiliki paket Open (akses gratis) dan paket berbayar, sebagaimana dijelaskan di bawah ini. Lihat [Paket akses IBM Quantum](https://www.ibm.com/quantum/pricing) untuk detail lengkapnya.

Jika kamu ingin akses ke beberapa paket, kamu bisa membuat beberapa instance dan menetapkan paket yang berbeda untuk masing-masing. Misalnya, kamu bisa punya instance Open Plan dan instance Pay-As-You-Go Plan. Kamu bisa mengatur instance yang diutamakan di initializer `QiskitRuntimeService` atau metode `QiskitRuntimeService.save_account()`. Contoh:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Open Plan {#open-plan}

Jalankan Circuit kuantummu di QPU terbaik di dunia secara gratis (hingga 10 menit waktu kuantum per jendela bergulir 28 hari). Kamu bisa memantau penggunaanmu di halaman [Platform dashboard](https://quantum.cloud.ibm.com/) dan [Workloads](https://quantum.cloud.ibm.com/workloads). Open Plan hanya tersedia di wilayah us-east.

Paket ini direkomendasikan bagi mereka yang sedang belajar komputasi kuantum dan menjelajahi teknologi kuantum IBM.

## Flex Plan {#flex-plan}

Beli terlebih dahulu sejumlah menit tertentu (minimal 400) untuk menjalankan job kuantum, yang harus digunakan dalam satu tahun sejak pembelian. Kamu bisa membeli lebih banyak menit kapan saja.

Setelah membeli waktu, kamu harus mengalokasikan menit-menit tersebut ke instance yang sudah ada atau baru pada Flex Plan untuk akunmu. Atur atau perbarui batas alokasi di [halaman Instances](https://quantum.cloud.ibm.com/instances) atau tetapkan alokasi saat kamu [membuat instance Flex Plan baru.](/guides/instances) Jika kamu mengalokasikan semua menit ke instance yang ada, kamu tidak bisa membuat instance baru untuk Flex Plan hingga membeli lebih banyak menit atau mengurangi menit dari alokasi instance lain. Lihat [Atur batas alokasi instance](allocation-limits) untuk mempelajari lebih lanjut.

Jika kamu butuh lebih dari 10.000 menit, sebaiknya pertimbangkan langganan Premium Plan. Jika kamu tidak ingin membeli 400 menit, pertimbangkan Pay-As-You-Go Plan sebagai alternatif.

[Hubungi IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) untuk mendaftar Flex Plan.

## Pay-As-You-Go Plan {#pay-as-you-go-plan}

Jalankan Circuit kuantum di QPU terbaik di dunia dan bayar hanya untuk waktu kuantum yang kamu gunakan.

Paket ini direkomendasikan bagi mereka yang menjalankan proyek riset utilitas kuantum dan menguji model bisnis dengan akses fleksibel. Jika kamu akan menggunakan setidaknya 400 menit dalam setahun ke depan, disarankan untuk menggunakan Flex Plan.

## Premium Plan {#premium-plan}

Jalankan Circuit kuantum di QPU terbaik di dunia menggunakan langganan waktu kuantum enterprise. Selain itu, beberapa fungsionalitas, seperti [Qiskit Functions](functions) dan [Qiskit Transpiler as a Service](qiskit-transpiler-service), tersedia untuk pengguna Premium Plan.

Paket ini direkomendasikan bagi mereka yang ingin menjalankan peta jalan kuantum strategis dan mengembangkan algoritma serta aplikasi kuantum dalam skala besar. [Hubungi IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) untuk informasi lebih lanjut.

## On-Prem Plan {#on-prem-plan}

Akses sistem kuantum on-premises yang sepenuhnya didedikasikan, dirawat dan dipelihara oleh IBM Quantum.

Layanan ini direkomendasikan bagi mereka yang mengeksplorasi algoritma dan aplikasi kuantum klasik serta membutuhkan kendali tinggi atas sumber daya dan data mereka. [Hubungi IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) untuk informasi lebih lanjut.
