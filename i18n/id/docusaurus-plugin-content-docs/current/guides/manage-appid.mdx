# Kelola pengguna ID provider {#manage-id-provider-users}

Kamu bisa mengintegrasikan dengan identity provider (IDP) eksternal untuk mengautentikasi pengguna eksternal ke akun IBM Cloud secara aman. Dengan menggunakan provider eksternal, kamu bisa menyediakan cara bagi pengguna di perusahaanmu untuk menggunakan single sign-on (SSO). Kamu bisa menghubungkan akun cloud ke IDP eksternal dengan salah satu cara berikut:

- Gunakan [IBM Cloud&reg; SAML service provider (SP)](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-cloud-saml)
- Gunakan [App ID](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-appid) (mungkin dikenakan biaya)
- [Federasikan penggunamu ke semua produk IBM&reg; dengan IBMid](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-ibmid)

<Admonition type="tip">
Setelah menghubungkan IDP, kamu bisa menyiapkan aturan dinamis sehingga tidak perlu menambahkan setiap pengguna ke setiap grup secara manual saat mereka bergabung. Lihat [Enabling authentication from an external identity provider](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#app-id-dynamic-rules) untuk instruksinya.
</Admonition>
