# Upgrade dari Open Plan {#upgrade-from-the-open-plan}

Panduan ini ditujukan bagi organisasi atau individu yang ingin upgrade dari IBM Quantum&reg; Open Plan ke IBM Cloud&reg; Pay-As-You-Go Plan. Untuk instruksi menyiapkan akun cloud, membuat service instance, dan mengelola pengguna, ikuti langkah-langkah berikut.

Kalau kamu adalah individu yang baru pertama kali menyiapkan layanan ini, atau sudah diundang ke instance oleh administrator, ikuti langkah-langkah di [Install Qiskit](./install-qiskit) saja.

## Sebelum mulai {#before-you-begin}

Kalau perlu, ikuti langkah-langkah di bagian [Set up to use IBM Cloud](./cloud-setup#cloud) untuk menyelesaikan hal berikut:

- Daftar akun IBM Cloud.
- Buat IBM Quantum instance.

<span id="update-cloud-account"></span>
## Upgrade akun IBM Cloud-mu {#upgrade-your-ibm-cloud-account}

Lewati bagian ini kalau kamu sudah punya akun IBM Cloud Pay-as-you-go. Semua akun IBM Cloud baru sudah otomatis dibuat sebagai Pay-as-you-go.

Kamu perlu upgrade akun IBM Cloud dalam kondisi berikut:

- Kamu punya akun Trial.
- Kamu punya akun Lite (sudah dihentikan) yang dibuat sebelum 25 Oktober 2021.

Setelah upgrade, kamu tetap bisa menggunakan instance yang sudah dibuat dengan akun Lite-mu. Untuk upgrade ke akun Pay-As-You-Go, ikuti langkah-langkah berikut:

1. Pergi ke Manage > [Account](https://cloud.ibm.com/account) di IBM Cloud console.
2. Pilih **Account settings**, lalu klik **Add credit card**.
3. Masukkan informasi pembayaranmu dan klik **Next** untuk mengirimkan informasi.
4. Setelah informasi pembayaranmu diproses, akun kamu akan di-upgrade.
   <Admonition type="note">
   Kamu tidak akan dikenakan biaya kecuali kalau kamu membuat instance Pay-As-You-Go Plan dan menggunakannya untuk mengirim workload ke QPU.
   </Admonition>

## Akses Pay-As-You-Go Plan-mu {#access-your-pay-as-you-go-plan}

Untuk menggunakan IBM Cloud Pay-As-You-Go Plan, buat [instance](/guides/instances) baru dan tetapkan ke Pay-As-You-Go Plan.  <ins>Semua job yang dikirim ke QPU dengan menggunakan instance ini akan dikenakan biaya.</ins>  Untuk menggunakan QPU tanpa biaya, kamu harus menggunakan instance yang ditetapkan ke Open plan atau paket gratis lainnya, kalau kamu punya.

Disarankan agar kamu menyiapkan [cost limit](/guides/manage-cost#cost-limit) pada instance tersebut dan mempelajari cara lain untuk mengelola biaya saat menggunakan Pay-As-You-Go Plan.
