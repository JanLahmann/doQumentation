# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon)は、専門的な量子コンピューティング・アルゴリズムと技術を提供する公式拡張パッケージです。各アドオンは特定の分野に焦点を当て、実行可能なコードを含むチュートリアルを提供しています。

## 利用可能なアドオン {#available-addons}

| アドオン | パッケージ | 説明 |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | ゲートとワイヤをカットすることで回路の幅と深さを削減 |
| **SQD** | `qiskit-addon-sqd` | 基底状態推定のためのサンプルベース量子対角化 |
| **OBP** | `qiskit-addon-obp` | 回路の深さを削減するための演算子逆伝播 |
| **MPF** | `qiskit-addon-mpf` | ハミルトニアン・シミュレーションを改善するマルチプロダクト・フォーミュラ |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | テンソルネットワークを用いた近似量子コンパイル |
| **PNA** | `qiskit-addon-pna` | エラー緩和のための伝播ノイズ吸収 |
| **SLC** | `qiskit-addon-slc` | PECサンプリング・オーバーヘッドを削減するシェーデッド・ライトコーン |

すべてのチュートリアルはBinderを使ってブラウザ上で直接実行するか、自分のJupyterサーバーで実行できます。

APIドキュメントについては、[GitHub](https://github.com/Qiskit?q=qiskit-addon)の各アドオン・リポジトリをご覧ください。
