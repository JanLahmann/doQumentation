# 概要 {#overview}

!["Hero image"](/learning/images/modules/quantum-mechanics/hero.avif)

この「Qiskit in classrooms」モジュールのコレクションは、学部生向けの量子力学コースに組み込まれるよう設計されています。目標は、教育者や学生が量子コンピューターを活用して、不確定性や重ね合わせなど、量子力学コースの核心をなすトピックを実験的に探求できるよう支援することです。
各モジュールは、クラスでグループ活動や教員のサポートを受けながら開始することを想定していますが、個人または少人数グループで完結させることも可能です。すべてのアクティビティは、全ユーザーに提供されているオープンプランの無料利用時間の範囲内で十分に完了できます。学生が誤って無料の量子コンピューティング時間を使い切ってしまった場合に備え、シミュレーターのオプションも組み込まれています。

教員の方は、ノートブックの使用状況に関するこちらの[簡単なアンケート](https://ibm.biz/classrooms_instructor_key_request)に回答することで、解答例や一般的なカリキュラムへの配置に関するガイダンスが含まれたバージョンのノートブックをリクエストできます。

## 推奨される事前知識 {#recommended-background}

各モジュールには、カリキュラムのどの段階でそのモジュールが最も効果的かについての推奨事項が記載されています。一般的に、以下の知識を持つ学生がこれらのモジュールを最大限に活用できます。

- 線形代数の基礎知識（$2 \times 2$ 行列の知識があれば十分です）
- Pythonの基本的な知識（ほとんどのノートブックはそのまま実行できるため、必須ではありません）
