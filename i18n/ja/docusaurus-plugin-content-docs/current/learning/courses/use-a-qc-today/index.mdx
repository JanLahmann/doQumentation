# 概要 {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

*今日から量子コンピューターを使う* へようこそ！このコースでは、すぐに量子コンピューターを使い始めながら、量子コンピューティングの基本原理を学んでいきます。

このコースは好奇心旺盛な方や忙しい方のために設計されています。物理学の学位や長年のプログラミング経験は必要ありません。ノートブックを読み進めて「Run」をクリックできれば、準備完了です。
