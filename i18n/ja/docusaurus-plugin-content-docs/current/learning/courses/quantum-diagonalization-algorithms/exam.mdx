## 知識を試しましょう {#test-your-knowledge}

このコースの学習とアクティビティを完了したら、以下のボタンをクリックしてIBM&reg; Trainingに移動し、量子対角化アルゴリズムのバッジ試験を受けてください。

試験は20問で構成されており、合格するには少なくとも16問に正解する必要があります。合格点を取得すると、IBM&reg; Trainingへの登録に使用したメールアドレスにバッジが自動的に送信されます。

試験を受ける準備が整っていることを確認してください。2回の試験で合格点を取得できなかった場合、試験は一時的にロックされます。

注意：IBM TrainingサイトのこれらのExamは、メンテナンスのため今後数日間、一時的に利用できない場合があります。
