# 概要 {#overview}

!["Hero image"](/learning/images/courses/utility-scale-quantum-computing/hero.svg)

このイベント収録コースは、IBM Quantum&reg; が東京大学と協力して開発・実施した14のレッスンとラボで構成されています。これらのレッスンは量子コンピューティングにおける幅広い重要トピックを扱っており、ユーティリティスケールの計算の実現に向けて構成されています。その成果は、2023年6月にNature誌の表紙を飾った論文と非常に類似したものとなっています。
