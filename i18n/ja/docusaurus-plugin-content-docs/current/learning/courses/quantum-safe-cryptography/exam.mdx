## 知識を試してみましょう {#test-your-knowledge}

このコースの読み物とアクティビティを完了したら、下のボタンをクリックしてIBM&reg; Trainingに移動し、量子安全暗号のバッジ試験を受けてください。

試験は20問で構成されており、合格するには少なくとも16問に正解する必要があります。合格点を取得すると、IBM&reg; Trainingへの登録に使用したメールアドレスに自動的にバッジが送付されます。

試験を受ける前に十分な準備が整っていることを確認してください。2回の受験で合格点に達しない場合、試験は一時的にロックされます。

注意：IBM TrainingサイトのこれらのExamは、メンテナンスのため、今後数日間一時的に利用できない場合があります。
