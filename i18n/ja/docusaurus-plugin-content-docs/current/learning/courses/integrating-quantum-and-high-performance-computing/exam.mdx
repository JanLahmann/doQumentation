# 試験とバッジ {#exam-and-badge}

![このコースの名称、中級レベル、IBM Quantum&reg;発行と記されている正方形のバッジ。](/learning/images/courses/integrating-quantum-and-high-performance-computing/exam/badge.svg)

## コース修了後のアンケート {#post-course-survey}

IBM&reg;の「量子コンピューティングとハイパフォーマンスコンピューティングの統合」コースを修了おめでとうございます！

## 知識の確認 {#test-your-knowledge}

このコースの読み物とアクティビティを完了したら、以下のボタンをクリックしてIBM Trainingに移動し、量子コンピューティングとハイパフォーマンスコンピューティングの統合のバッジ取得試験を受験してください。

試験は20問で構成されており、合格するには16問以上の正解が必要です。合格点を取得すると、IBM Trainingへの登録時に使用したメールアドレスにバッジが自動的に送付されます。

試験を受ける前に十分な準備を整えてください。2回の受験で合格点を取得できない場合、試験は一時的にロックされます。
