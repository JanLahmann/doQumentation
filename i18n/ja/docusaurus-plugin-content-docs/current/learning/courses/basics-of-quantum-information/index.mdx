# 概要 {#overview}

!["Hero image"](/learning/images/courses/basics-of-quantum-information/hero.svg)

*量子情報の基礎*へようこそ。このコースは、以下のコースで構成される*量子情報と計算の理解*シリーズの最初のコースです:

- 量子情報の基礎（このコース）
- [量子アルゴリズムの基礎](/learning/courses/fundamentals-of-quantum-algorithms)
- [量子情報の一般的な定式化](/learning/courses/general-formulation-of-quantum-information)
- [量子エラー訂正の基礎](/learning/courses/foundations-of-quantum-error-correction)

このコースは、単一システムと複数システムの両方に対する量子情報の記述を含む、量子情報の数学への導入から始まります。次に、量子計算を記述する標準的な方法を提供する量子回路に移ります。最後に、量子エンタングルメントの現象に関連する3つの基本的に重要な例、すなわち量子テレポーテーション、超密度符号化、CHSHゲーム（CHSH不等式としても知られています）について説明します。

このコースは、量子情報と計算の理論的基礎に関する知識を熱望するコンピュータサイエンス、物理学、工学、数学などの分野の学生、専門家、愛好家を対象としています。

## 推奨される予備知識 {#recommended-background}

このコースを最大限に活用するには、基本的な線形代数、複素数、および集合や関数を含む初歩的な数学的概念に精通していることをお勧めします。以下のリソースは、この内容をカバーする多くのリソースの中のほんの一部です:

- [Khan Academy: Linear algebra](https://www.khanacademy.org/math/linear-algebra)

   このビデオシリーズでは、Sal Khanが私たちが依拠する線形代数の主要概念を紹介しています。

- Stephen Friedberg, Arnold Insel, and Lawrence Spence. [Linear Algebra](https://www.pearson.com/en-us/subject-catalog/p/linear-algebra/P200000006185/9780137515424)

   この線形代数の教科書は、私たちが必要とする内容をカバーし、集合、関数、複素数に関する付録も含まれています。

- Sheldon Axler. [Linear Algebra Done Right](https://link.springer.com/book/10.1007/978-3-319-11080-6)

   上級学部生以上のレベルに適した線形代数の古典的なテキストです。

- Ricky Shadrach and Rod Pierce. [Introduction to Sets](https://www.mathsisfun.com/sets/sets-introduction.html)

   一部の読者が理解を深めるのに役立つ、集合に関する初級レベルのWebページです。

- John K. Hunter. [An Introduction to Real Analysis: Chapter 1](https://www.math.ucdavis.edu/~hunter/intro_analysis_pdf/ch1.pdf)

   これらの講義ノートの第1章には、集合と関数に関するより形式的で詳細な導入が含まれています。
