このレッスンでは、*複数の*システムという文脈における量子情報の基礎を扱います。
この文脈は、古典・量子を問わず、情報処理において一般的かつ自然に生じるものです。
情報を担うシステムは、ビットや量子ビットのような小さなシステムの集合から構成されるのが一般的です。

このレッスンに入るにあたって念頭に置いておくべき、シンプルながらも非常に重要なアイデアがあります。それは、複数のシステムを常に*まとめて*一つの複合システムとして捉えることができる、という点です。前のレッスンの議論はそのまま適用できます。
実際、このアイデアは、複数システムにおける量子状態・測定・演算がどのように機能するかの説明へと、非常に直接的につながっています。

しかし、複数の量子システムを理解するには、単にそれらを一つのシステムとして集合的に見なすことができると認識するだけでは十分ではありません。
例えば、複数の量子システムが集合的に特定の量子状態にある場合に、その中の一部のシステムだけを測定することを選択するかもしれません。
一般に、これは測定されなかったシステムの状態に影響を与えます。量子アルゴリズムやプロトコルを分析する際には、その影響がどのようなものかを正確に理解することが重要です。
また、複数のシステム間の*相関*の種類、特に*エンタングルメント*と呼ばれる相関の種類を理解することも、量子情報と量子計算において重要です。
