# 概要 {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

このコースは、量子コンピューティングの経験がある方で、Qiskit Runtime プリミティブを使用した変分アルゴリズムの書き方を学びたい方を対象としています。また、変分アルゴリズムの分野における知識とスキルを広げたい量子コンピューティングの専門家にも適しています。

## 試験 {#exam}

変分アルゴリズム設計のバッジを取得するには、IBM&reg; Training で試験を受けてください。この試験は、本コースのレッスンを読み終えた後に受験することを想定しています。試験に合格すると、Credly からバッジを取得したことが通知されます。

[試験](/learning/courses/variational-algorithm-design/exam)
