# 概要 {#overview}

!["Hero image"](/learning/images/courses/designing-and-leading-quantum-projects/hero.svg)

このコースは、チームや組織が量子イニシアティブを効果的に立ち上げるために必要な能力を構築できるよう支援することを目的としています。

コンテンツの修了には、事前知識の有無によって異なりますが、約4時間を目安にしてください。

### 対象者 {#intended-audience}

このコースは、量子イニシアティブのリーダーやアーキテクト、たとえば大学教員、大企業のプロジェクト・リーダー、スタートアップの経営者などを対象としています。量子コンピューティングを初めて導入する組織に特に役立つ内容です。

## 推奨する事前知識 {#recommended-background}

このコースに技術的な前提知識はありませんが、まず[量子ビジネス基礎](/learning/courses/quantum-business-foundations)コースを受講することをお勧めします。このコースは量子エコシステムへの良い入門となっています。
