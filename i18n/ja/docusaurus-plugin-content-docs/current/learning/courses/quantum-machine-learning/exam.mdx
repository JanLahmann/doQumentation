## 知識を確認する {#test-your-knowledge}

このコースの読み物とアクティビティを完了したら、下のボタンをクリックしてIBM&reg; トレーニングに移動し、量子機械学習のバッジ試験を受けてください。

試験は20問で構成されており、合格するには少なくとも16問に正解する必要があります。合格点を取得すると、IBM トレーニングの登録に使用したメールアドレスにバッジが自動的に送付されます。

試験を受ける準備ができているか確認してください。2回の試行で合格点を取れなかった場合、試験は一時的にロックされます。

注意：IBM トレーニングサイト上のこれらの試験は、メンテナンスのため今後数日間、一時的に利用できない場合があります。
