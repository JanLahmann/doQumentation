# 概要 {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

このコースでは、量子コンピューティングが機械学習においてどのような重要な役割を果たせるかを探求します。古典的な機械学習のアプローチについても一部復習しますが、これらの手法についてある程度の知識があることを前提としています。主な焦点は、量子コンピューティングが既存の機械学習ワークフローにどのように組み込めるかという点です。トピックには、データエンコーディング、量子カーネル、変分量子回路が含まれます。
