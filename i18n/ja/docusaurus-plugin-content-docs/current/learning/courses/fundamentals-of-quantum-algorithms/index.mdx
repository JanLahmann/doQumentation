# 概要 {#overview}

!["Hero image"](/learning/images/courses/fundamentals-of-quantum-algorithms/hero.svg)

*量子アルゴリズムの基礎*へようこそ。このコースは、以下のコースで構成される*量子情報と計算の理解*シリーズの第2コースです。

- [量子情報の基礎](/learning/courses/basics-of-quantum-information)
- 量子アルゴリズムの基礎（本コース）
- [量子情報の一般的な定式化](/learning/courses/general-formulation-of-quantum-information)
- [量子誤り訂正の基礎](/learning/courses/foundations-of-quantum-error-correction)

このコースでは、量子コンピューターで何ができるか、また古典コンピューターに対するその優位性など、量子情報の計算上の利点を探求します。
まず、量子アルゴリズムの概念実証として分かりやすい量子クエリアルゴリズムから始め、続いて整数の素因数分解や非構造化探索などの問題に対する量子アルゴリズムへと進みます。

このコースは、量子情報と計算の理論的基礎に関する知識を習得したいと考えている、コンピューターサイエンス、物理学、工学、数学などの分野の学生、専門家、愛好家を対象としています。
