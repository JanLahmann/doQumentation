量子ワークフローは完了までに時間がかかることが多く、複数のセッションにわたって実行される場合があります。Python カーネルを再起動すると、メモリに保存されていた結果はすべて失われてしまいます。データの損失を防ぐために、結果をファイルに保存したり、IBM Quantum&reg; から過去のジョブの結果を取得したりすることができます。これにより、次のセッションで前回の続きから作業を再開できます。
## IBM Quantum からジョブ結果を取得する {#retrieve-job-results-from-ibm-quantum}

IBM Quantum は、すべてのジョブの結果を自動的に保存しており、後日取得することができます。この機能を活用することで、カーネルの再起動をまたいで量子プログラムを継続したり、過去の結果を確認したりすることができます。ジョブの ID はプログラムから `job_id` メソッドを使用して取得できます。また、[Workloads ページ](https://quantum.cloud.ibm.com/workloads)では、送信済みのすべてのジョブとその ID を確認することができます。

プログラムからジョブを検索するには、[`QiskitRuntimeService.jobs`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#jobs) メソッドを使用します。デフォルトでは最近送信されたジョブが返されますが、バックエンド名や作成日などでフィルタリングすることも可能です。以下のセルでは、過去3か月以内に送信されたジョブを検索しています。`created_after` 引数には [`datetime.datetime`](https://docs.python.org/3.8/library/datetime.html#datetime.datetime) オブジェクトを指定する必要があります。

```python
# Added by doQumentation — required packages for this notebook
!pip install -q qiskit-ibm-runtime
```

```python
import datetime
from qiskit_ibm_runtime import QiskitRuntimeService

three_months_ago = datetime.datetime.now() - datetime.timedelta(days=90)

service = QiskitRuntimeService()
jobs_in_last_three_months = service.jobs(created_after=three_months_ago)
jobs_in_last_three_months[:3]  # show first three jobs
```

バックエンド、ジョブの状態、セッションなどによって選択することもできます。詳細については、API ドキュメントの [`QiskitRuntimeService.jobs`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#jobs) を参照してください。

ジョブ ID を取得したら、[`QiskitRuntimeService.job`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#job) メソッドを使用して取得します。

```python
# Get ID of most recent successful job for demonstration.
# This will not work if you've never successfully run a job.
successful_job = next(
    j for j in service.jobs(limit=1000) if j.status() == "DONE"
)
job_id = successful_job.job_id()
print(job_id)
```

## 結果をディスクに保存する {#save-results-to-disk}

結果をディスクに保存したい場合もあるでしょう。そのためには、Qiskit Runtime のエンコーダーとともに Python 組み込みの JSON ライブラリを使用します。

```python
import json
from qiskit_ibm_runtime import RuntimeEncoder

with open("result.json", "w") as file:
    json.dump(retrieved_job.result(), file, cls=RuntimeEncoder)
```

その後、別のカーネルでこの配列をディスクから読み込むことができます。

```python
from qiskit_ibm_runtime import RuntimeDecoder

with open("result.json", "r") as file:
    result = json.load(file, cls=RuntimeDecoder)

result
```
