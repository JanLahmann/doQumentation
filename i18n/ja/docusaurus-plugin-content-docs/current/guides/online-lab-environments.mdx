# オンラインラボ環境 {#online-lab-environments}

Qiskitをローカルマシンにダウンロードしてインストールする方法に加え、オンライン開発環境での作業を希望される方向けにいくつかの選択肢がございます。

複数のプロバイダーが、QiskitをプリインストールしたオンラインのJupyter環境を提供しており、迅速かつ便利なオンボーディングが可能です。推奨プロバイダーには、クラウドサービスプロバイダーのOVHcloudと、量子ソフトウェア開発クラウドプラットフォームのqBraidが含まれます。

## Qiskitプリインストール済みの推奨ノートブック環境 {#recommended-notebook-environments-with-qiskit-preinstalled}

IBM Watson&reg; Studio、Google Colab、Microsoft Azure Machine Learning Studioなどの追加オンラインJupyterノートブック環境も、適切な手順に従ってQiskitとともに使用できます。

### OVHcloud AIノートブック {#ovhcloud-ai-notebooks}

OVHcloud Public CloudおよびOVHcloud Startupユーザーは、OVHcloudの量子ノートブックを使用して、IBM&reg; QPUへのジョブ送信をIBM Quantum&reg;またはIBM Cloud&reg;アカウントでアクセスできます。詳細については、[OVHcloudドキュメントとチュートリアル](https://help.ovhcloud.com/csm/world-documentation?id=kb_home)のページをご覧ください。

<Admonition type="note">
OVHcloud AIノートブックは、米国など一部の地域ではご利用いただけません。また、地域によってユーザーインターフェースの表示が異なる場合があります。
</Admonition>

始めるには、以下の手順に従ってください。

1. [IBM Cloudアカウントを作成します。](./cloud-setup)
1. [OVHcloudアカウントをセットアップします。](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [OVH Public Cloudプロジェクトを作成します。](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. [最初のAIノートブックを起動する](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270)の手順に従い、フレームワークとして**Qiskit**を選択してください。

### qBraid Lab {#qbraid-lab}

Qiskitユーザーは、以下の手順に従ってqBraidの事前構成済みPython環境を利用できます。詳細については、[qBraidドキュメント](https://docs.qbraid.com/home/introduction)をご参照ください。

1. [qBraidの無料アカウントをセットアップします。](https://docs.qbraid.com/home/account)
3. [Lab環境を作成します。](https://docs.qbraid.com/lab/user-guide/getting-started)
3. qBraid Labを起動し、右側の**ENVS**アイコンをクリックして、新しいサイドバーの上部にある**+ ADD**をクリックし、Qiskitを検索します。使用するバージョンをクリックし、**Install**をクリックしてください。
4. ノートブックセクションの**Python 3 [Qiskit]**をクリックしてJupyterノートブックを作成します。
4. ノートブックセルで`%pip install [package_name].`を実行することで、追加のPythonパッケージ（Qiskit v1.2など）をインストールできます。詳細については、[Pip（マジック）コマンド](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands)のトピックをご参照ください。

## Qiskitユーザー向けの追加ノートブック環境 {#additional-notebook-environments-for-qiskit-users}

Qiskitが事前構成されていないプラットフォームでは、セットアップ手順は概ね同様ですが、Qiskitを手動でインストールする必要があります。アカウントを作成し、新しいプロジェクト（またはプラットフォームに応じてノートブックやワークスペース）を開始して、Qiskitをインストールしてください。

### IBM Watson Studio {#ibm-watson-studio}

1. [Watson Studioの無料トライアルをセットアップします。](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Watson Studioプロジェクトを作成します。](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. 必要なQiskitパッケージと依存関係をインストールするには、新しいノートブックを開いて以下のコマンドを実行してください。
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Googleアカウントを作成します。](https://support.google.com/accounts/answer/27441?hl)
2. [Google Colabのランディングページ](https://colab.research.google.com/)にアクセスし、Googleアカウントでサインインします。
3. [Colabノートブックを作成し](https://colab.research.google.com/#create=true)、以下のコマンドを実行してQiskitをインストールします。
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Azureアカウントを作成します。](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [ワークスペースを作成します。](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [ワークスペースにJupyterノートブックを作成します。](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. ノートブックから以下のコマンドを実行してQiskitをインストールします。
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
