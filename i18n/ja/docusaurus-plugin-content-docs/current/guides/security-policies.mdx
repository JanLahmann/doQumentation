# リポジトリのセキュリティ・ポリシー {#repository-security-policies}

システムをできる限り安全に保つことは非常に重要です。そのため、依存関係が古くなったときに自動的に通知してくれる [Dependabot](https://github.com/dependabot) などのセキュリティ・ツールの使用を強くお勧めします。IBM Quantum&reg; に関連する以下の GitHub リポジトリのセキュリティ・ポリシーをご確認ください。
