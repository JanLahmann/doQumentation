## IBM Quantum PlatformとQiskitのサポートを受ける {#get-help-with-ibm-quantum-platform-and-qiskit}

**Premiumプラン、On-Premプラン、Direct Access、Flexプラン、Pay-As-You-Goプラン、特別プログラム、およびOpenプラン（Trialアカウントを除く）:**

すべてのサポートに関するお問い合わせは、[IBM Cloud&reg; Support Center](https://cloud.ibm.com/unifiedsupport/cases/form)を通じて送信してください。アカウントの種類に応じて、Cloud Case（Topic: Qiskit Runtime）またはTechnical Issue（Topic: Report-an-Issue::Qiskit Runtime）を開くことができます。アカウント切り替えツールで適切なIBM Cloudアカウント（機関のクラウドアカウント、またはTrialアカウント以外のアカウント）が選択されていることを確認してください。

オープンなIBM Cloudケースをすべて確認したり、新しいケースを開いたりするには、[IBM Cloud Support Center](https://cloud.ibm.com/unifiedsupport/cases)をご利用ください。

プラットフォームの問題、ジョブエラー、Qiskitのガイダンス、または量子コンピューティングに関するより広範な質問など、技術的な事項については何でもお問い合わせください。私たちのチームには、これらすべてのトピックにわたってサポートできる技術専門家が揃っています。

**コミュニティサポート:**

Trialアカウントをお持ちの方、またはその他のサポートオプションをお探しの方は、[Qiskit Slackコミュニティに参加](https://qisk.it/join-slack)して、多くのユーザーや専門家のコミュニティからサポートを受けてください。Slackでは、#qiskit-runtime、#ibm-quantum-platform、#qiskit-transpiler、#qiskit-ecosystemなど、さまざまなトピックチャンネルを使用できます。

> **Note:** This survey is provided by IBM Quantum and relates to the original English content. To give feedback on doQumentation's website, translations, or code execution, please [open a GitHub issue](https://github.com/JanLahmann/doQumentation/issues).

量子コンピューティングやプログラミング関連の質問については、[Stack Exchange](https://quantumcomputing.stackexchange.com/?tags=qiskit)を通じて幅広いコミュニティからサポートを受けることもできます。

## アカウントの作成やログインでお困りですか？ {#having-trouble-creating-an-account-or-logging-in}

**IBM Cloud&reg;アカウントの作成時にエラーが発生した場合は、**まずIBM Cloudの[Support Center](https://cloud.ibm.com/unifiedsupport/cases/form)にアクセスできるかどうかを確認してください。

- 「Create Case」ページにアクセスできる場合は、カテゴリとしてAccount（Managing your account and resources）を選択し、次にSubtopicとしてAccount verificationを選択してください。
- アクセスできない場合は、[アカウント作成・ログイン・請求リクエスト](https://watson.service-now.com/x_ibmwc_open_case_app.do#!/create)のページからケースを開いてください。

アカウントの作成中にエラーが発生し、**お住まいの国が[個人使用の利用可能地域](https://cloud.ibm.com/docs/account?topic=account-account-getting-started#signup-personalaccts)のリストに含まれている場合は、**以下のオプションをご検討ください。

1. アカウントなしで[ドキュメント](/docs)および[学習](/learning)リソースを使用できます。
2. 学位授与の認定を受けた高等教育機関の学生または教育者の方は、[SkillsBuild Technology Accessアカウント](https://github.com/academic-initiative/documentation/blob/main/academic-initiative/how-to/How-to-register-with-the-IBM-Academic-Initiative/readme.md)を作成できます（詳細については[SkillsBuild Technology Access FAQ](https://academic.ibm.com/a2mt/faq#/)をご参照ください）。
3. 30日間の無料アクセスのために[Trialアカウントを作成](https://quantum.cloud.ibm.com/registration)してください（Trialの終了後はアクセスを失う可能性があります）。

**「このアカウントは無効です」というメッセージが表示された場合は、**[IBM Cloud Docs](https://cloud.ibm.com/docs/account?topic=account-ts_accnt_inactive)でその原因と解決方法をご確認いただくか、[IBM Cloud](https://cloud.ibm.com)にアクセスしてアカウントで完了が必要な保留中のアクションがないかご確認ください。

<Admonition type="caution" title="注意">
Pay-As-You-GoまたはSubscriptionアカウントの**所有者**の場合は、IBM Cloudアカウントを作成し、確認のための請求情報を提供する必要があります。

アカウントへの**招待を受けた場合**は、メールの招待を承認し、[案内の手順](/guides/cloud-setup-invited)に従うだけで、請求情報を提供せずにアクセスできます。

**Trialアカウントの所有者**は、請求情報を提供せずに30日間、Open Planインスタンスを作成できます。Trialの終了後、ユーザーはアカウントを確認して引き続き使用するために請求情報を提供する必要があります。無料のOpen Planを引き続き使用する場合は*課金されません*が、その時点で他の有料プランへのアップグレードを選択することもできます。

IBM Cloudアカウントの詳細については、[IBM CloudアカウントFAQ](https://cloud.ibm.com/docs/account?topic=account-accountfaqs)ページを参照してください。
</Admonition>
