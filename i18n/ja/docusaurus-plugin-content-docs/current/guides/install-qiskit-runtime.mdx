# Qiskit Runtime クライアントのインストール {#install-the-qiskit-runtime-client}

Qiskit Runtime は、IBM Quantum&reg; ハードウェア上で量子計算を実行するためのクラウドベースのサービスです。このガイドでは、Qiskit Runtime サービスにアクセスするためのクライアント・パッケージである `qiskit-ibm-runtime` のインストール方法を説明します。

Qiskit Runtime クライアントをインストールする前に、以下のガイドの手順に従ってください。

1. [Qiskit のインストール](/guides/install-qiskit)。
2. [IBM Cloud アカウントのセットアップ](cloud-setup)。

次のコマンドを実行して Qiskit Runtime をインストールします。

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="最新バージョンの維持">
[Qiskit Runtime リリースノート](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes)を定期的に確認して、新しいリリースを把握してください。`pip install -U qiskit-ibm-runtime` を実行して、`qiskit-ibm-runtime` の要件を頻繁に更新することをお勧めします。
</Admonition>
