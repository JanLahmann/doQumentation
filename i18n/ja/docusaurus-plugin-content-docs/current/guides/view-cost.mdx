# コストの確認 {#view-the-cost}

インスタンスに関連付けられたワークロードで発生したコストは、いつでも確認できます。

## インスタンスのコストを確認する {#view-instance-cost}

インスタンスの使用量を把握するには、[インスタンスページ](https://quantum.cloud.ibm.com/instances)からインスタンスをクリックして詳細ページを開きます。**請求の概要**に関連付けられたリンクをクリックすると、完全なコストおよび請求の詳細を確認できます。

**インスタンスの使用量**は、インスタンスに関連付けられたプランの種類によって、報告される値が異なります。

- Premium プランおよび Open プランの場合、過去 28 日間（ローリング）のリアルタイム使用量（当日の現在時刻まで）が表示されます。
- Pay-As-You-Go プランの場合、インスタンスが作成された時点からのすべての使用量が表示されます。
- Flex プランの場合、契約開始時点からの使用量が表示されます。

## ワークロードの使用量を確認する {#view-workload-usage}

特定のワークロードで発生した使用量を把握するには、[ワークロードページ](https://quantum.cloud.ibm.com/workloads)を開き、対象のワークロードの「Usage」列を確認してください。
