# はじめに {#introduction}

## QiskitとIBM Quantumの紹介 {#introduction-to-qiskit-and-ibm-quantum}

Qiskit、その関連パッケージ、およびIBM Quantum&reg; Platformのドキュメントへようこそ。このドキュメントには、ツールの使用を開始するためのハウツーガイド、エンドツーエンドの例を含む特定のユースケースのチュートリアル、およびAPIリファレンスのコレクションが含まれています。

Qiskitは、アルゴリズム、ハイパフォーマンスコンピューティング、および量子情報科学全体にわたる量子研究および開発のためのモジュール式で拡張可能なフレームワークを提供します。これにより、研究者は専門的なアドオン、ソフトウェアツール、および豊富なリソースを使用して、量子ワークフローを構築、最適化、実行できます。IBM Quantum Platformを通じて、ユーザーはQiskit RuntimeやQiskit Functions Catalogなどの量子コンピューティングサービスにアクセスし、IBM&reg;の量子コンピューターの群でワークロードを効率的に実行できます。

![IBM Quantum PlatformとQiskitのさまざまなコンポーネントを示すグラフィック。](/docs/images/guides/index/overview-image.svg)

Qiskitとその関連パッケージを超えて、[Qiskitエコシステム](https://www.ibm.com/quantum/ecosystem)があります。これは、Qiskitと連携してその機能を拡張するオープンソースプロジェクトのカタログです。
