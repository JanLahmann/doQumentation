# Gerenciar usuários do provedor de identidade {#manage-id-provider-users}

Você pode integrar com seu provedor de identidade externo (IDP) para autenticar usuários externos na sua conta IBM Cloud de forma segura. Ao usar um provedor externo, você oferece uma forma para os usuários da sua empresa utilizarem o login único (SSO). Você pode conectar sua conta na nuvem a um IDP externo de uma das seguintes maneiras:

- Use o [IBM Cloud&reg; provedor de serviço SAML (SP)](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-cloud-saml)
- Use o [App ID](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-appid) (pode gerar custos)
- [Federe seus usuários a todos os produtos IBM&reg; com o IBMid](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-ibmid)

<Admonition type="tip">
Após conectar seu IDP, você pode configurar regras dinâmicas para não precisar adicionar manualmente cada usuário a cada grupo quando eles entrarem. Consulte [Habilitando autenticação de um provedor de identidade externo](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#app-id-dynamic-rules) para instruções.
</Admonition>
