# Instalar o cliente Qiskit Runtime {#install-the-qiskit-runtime-client}

O Qiskit Runtime é o serviço baseado em nuvem para executar computações quânticas no hardware IBM Quantum&reg;. Este guia mostra como instalar o `qiskit-ibm-runtime`, que é o pacote cliente para acessar o serviço Qiskit Runtime.

Antes de instalar o cliente Qiskit Runtime, siga as instruções nestes guias:

1. [Instalar o Qiskit](/guides/install-qiskit).
2. [Configurar sua conta IBM Cloud](cloud-setup).

Execute este comando para instalar o Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Mantenha-se atualizado com as versões mais recentes">
Verifique periodicamente as [notas de lançamento do Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) para ver novos lançamentos. Recomendamos atualizar frequentemente seus requisitos para `qiskit-ibm-runtime` executando `pip install -U qiskit-ibm-runtime`.
</Admonition>
