# Políticas de segurança do repositório {#repository-security-policies}

É extremamente importante manter seus sistemas o mais seguros possível. Por isso, é altamente recomendado que você use ferramentas de segurança como o [Dependabot](https://github.com/dependabot), que notifica automaticamente quando as dependências estão desatualizadas. Consulte as políticas de segurança dos seguintes repositórios do GitHub relacionados ao IBM Quantum&reg;:

- [Qiskit](https://github.com/Qiskit/qiskit/security)
- [Qiskit IBM Runtime](https://github.com/Qiskit/qiskit-ibm-runtime/security)
- [Complemento Qiskit de corte de circuitos](https://github.com/Qiskit/qiskit-addon-cutting/security)
- [Complemento Qiskit de mapeamento fermiônico](https://github.com/Qiskit/qiskit-fermions/blob/main/SECURITY.md)
- [Complemento Qiskit de fórmulas multi-produto (MPF)](https://github.com/Qiskit/qiskit-addon-mpf/security)
- [Complemento Qiskit de retropropagação de operador (OBP)](https://github.com/Qiskit/qiskit-addon-obp/security)
- [Complemento Qiskit de mapeamento de otimização](https://github.com/Qiskit/qiskit-addon-opt-mapper/blob/main/SECURITY.md)
- [Complemento Qiskit de propagação de Pauli](https://github.com/Qiskit/pauli-prop/blob/main/SECURITY.md)
- [Complemento Qiskit de absorção de ruído propagado (PNA)](https://github.com/Qiskit/qiskit-addon-pna/blob/main/SECURITY.md)
