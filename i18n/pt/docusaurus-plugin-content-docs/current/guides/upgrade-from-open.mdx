# Fazer upgrade do Plano Open {#upgrade-from-the-open-plan}

Este guia é para entidades que desejam fazer upgrade do IBM Quantum&reg; Open Plan para um IBM Cloud&reg; Pay-As-You-Go Plan. Para instruções sobre como configurar uma conta na cloud, configurar uma instância de serviço e trabalhar com utilizadores, segue estes passos.

Se és um indivíduo a configurar o serviço pela primeira vez ou se foste convidado para uma instância por um administrador, segue os passos em [Instalar o Qiskit](./install-qiskit).

## Antes de começar {#before-you-begin}

Se necessário, segue os passos na secção [Configurar para usar o IBM Cloud](./cloud-setup#cloud) para completar o seguinte:

- Criar uma conta IBM Cloud.
- Criar uma instância IBM Quantum.

<span id="update-cloud-account"></span>
## Fazer upgrade da tua conta IBM Cloud {#upgrade-your-ibm-cloud-account}

Salta este passo se já tens uma conta IBM Cloud Pay-as-you-go. Todas as novas contas IBM Cloud são configuradas como Pay-as-you-go.

Precisas de fazer upgrade da tua conta IBM Cloud nas seguintes circunstâncias:

- Tens uma conta Trial.
- Tens uma conta Lite (descontinuada) que criaste antes de 25 de outubro de 2021.

Depois de fazeres o upgrade, podes continuar a usar quaisquer instâncias que criaste com a tua conta Lite. Para fazer upgrade para uma conta Pay-As-You-Go, completa os seguintes passos:

1. Vai a Manage > [Account](https://cloud.ibm.com/account) na consola IBM Cloud.
2. Seleciona **Account settings** e clica em **Add credit card**.
3. Introduz as tuas informações de pagamento e clica em **Next** para submeter as tuas informações.
4. Depois de as tuas informações de pagamento serem processadas, a tua conta é atualizada.
   <Admonition type="note">
   Não serás cobrado a menos que cries uma instância Pay-As-You-Go Plan e a uses para enviar cargas de trabalho para QPUs.
   </Admonition>

## Aceder ao teu Pay-As-You-Go Plan {#access-your-pay-as-you-go-plan}

Para usar o teu IBM Cloud Pay-As-You-Go Plan, cria uma nova [instância](/guides/instances) e atribui-a ao Pay-As-You-Go Plan.  <ins>Todos os jobs enviados para QPUs ao especificar esta instância terão um custo associado.</ins>  Para usar QPUs sem incorrer em custos, precisas de usar uma instância atribuída ao plano Open ou outro plano gratuito, se tiveres um.

É recomendado que configures um [limite de custo](/guides/manage-cost#cost-limit) na instância e aprendas sobre outras formas de gerir os teus custos ao usar o Pay-As-You-Go Plan.
