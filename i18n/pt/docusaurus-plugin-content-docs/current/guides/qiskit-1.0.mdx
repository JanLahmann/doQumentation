# Guia de migração para o Qiskit 1.0 {#qiskit-10-migration-guide}

O Qiskit 1.0 traz novas garantias de estabilidade e melhorias de desempenho, mas quebra a compatibilidade com todas as versões anteriores do Qiskit. Confira os guias a seguir para atualizar seu ambiente e código para funcionar com o Qiskit v1.0.

- [Instale o novo pacote Qiskit v1.0](/guides/qiskit-1.0-installation) para atualizar ambientes Python e dependências do projeto.
- [Mudanças de funcionalidades do Qiskit v1.0](/guides/qiskit-1.0-features) para atualizar seu código.
- [Mudanças de pacotes incompatíveis do Qiskit v1.0](/guides/metapackage-migration) para entender a estrutura nova e antiga dos pacotes.

Leia mais sobre os benefícios do Qiskit v1.0 no [blog da IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
