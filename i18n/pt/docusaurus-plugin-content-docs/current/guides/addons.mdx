# Addons do Qiskit {#qiskit-addons}

Os addons do Qiskit são uma coleção de capacidades de pesquisa para possibilitar a descoberta de algoritmos em escala utilitária. Essas capacidades se constroem sobre a base de ferramentas de alto desempenho do Qiskit para criar e executar algoritmos quânticos. Elas são fornecidas como componentes de software modulares que podem ser integrados a um [fluxo de trabalho](/guides/intro-to-patterns) para escalar ou projetar novos algoritmos quânticos.

## Addons para mapeamento {#addons-for-mapping}

### Compilação quântica aproximada com redes tensoriais {#approximate-quantum-compilation-with-tensor-networks}

A compilação quântica aproximada com redes tensoriais (AQC-Tensor) permite a construção de circuitos de alta fidelidade com profundidade reduzida.

- Acesse o [repositório GitHub.](https://github.com/Qiskit/qiskit-addon-aqc-tensor)
- Leia a [documentação.](/guides/qiskit-addons-aqc)
- Leia o tutorial sobre como usar o AQC para [melhorar a evolução temporal Trotterizada.](/tutorials/approximate-quantum-compilation-for-time-evolution)

### Fórmulas multi-produto {#multi-product-formulas}

As fórmulas multi-produto (MPF) reduzem o erro de Trotter na dinâmica hamiltoniana por meio de uma combinação ponderada de várias execuções de circuitos.

- Acesse o [repositório GitHub.](https://github.com/Qiskit/qiskit-addon-mpf)
- Leia a [documentação.](https://qiskit.github.io/qiskit-addon-mpf)

### Mapeador de otimização {#optimization-mapper}

O addon Optimization Mapper contém funcionalidades para modelar problemas de otimização, formulando-os em modelos abstratos e convertendo-os em representações que um computador quântico consegue entender.

- Leia a [documentação](https://qiskit.github.io/qiskit-addon-opt-mapper/) para guias práticos e referências de API.
- Acesse o repositório no [GitHub](https://github.com/qiskit/qiskit-addon-opt-mapper).

## Addons para otimização {#addons-for-optimizing}

### Retropropagação de operadores {#operator-backpropagation}

A retropropagação de operadores (OBP) reduz a profundidade do circuito removendo operações do final, ao custo de mais medições de operadores.

- Acesse o [repositório GitHub.](https://github.com/Qiskit/qiskit-addon-obp)
- Leia a [documentação.](/guides/qiskit-addons-obp)
- Leia o tutorial sobre como usar o [OBP para melhorar valores de expectativa.](/tutorials/operator-back-propagation)

### Corte de circuitos {#circuit-cutting}

O corte de circuitos reduz a profundidade dos circuitos transpilados decompondo portas de entrelaçamento entre qubits não adjacentes.

- Acesse o [repositório GitHub.](https://github.com/Qiskit/qiskit-addon-cutting)
- Leia a [documentação.](https://qiskit.github.io/qiskit-addon-cutting)

## Addons para pós-processamento {#addons-for-post-processing}

### Diagonalização quântica baseada em amostras {#sample-based-quantum-diagonalization}

A diagonalização quântica baseada em amostras (SQD) realiza o pós-processamento clássico de amostras quânticas ruidosas para obter estimativas mais precisas de autovalores de hamiltonianos de sistemas quânticos, como em aplicações de química.

- Acesse o [repositório GitHub.](https://github.com/Qiskit/qiskit-addon-sqd)
- Leia a [documentação.](/guides/qiskit-addons-sqd)
- Leia o tutorial sobre [diagonalização quântica baseada em amostras de um hamiltoniano de química.](/tutorials/sample-based-quantum-diagonalization)

### SQD para HPC {#sqd-for-hpc}

Este addon é uma implementação do addon SQD preparada para HPC. Ele é escrito seguindo os padrões modernos do C++17 e foi projetado para criar um único binário compilado para uso com MPI.

- Acesse a documentação de [referência da API](https://qiskit.github.io/qiskit-addon-sqd-hpc/).
- Veja o repositório no [GitHub](https://github.com/qiskit/qiskit-addon-sqd-hpc).

## Addons para mitigação de erros {#addons-for-error-mitigation}

### M3 (mthree) {#m3-mthree}

A Mitigação de Erros de Medição sem Matriz (M3) é um pacote para mitigação escalável de erros de medição quântica que pode ser computado em paralelo.

- Acesse a [documentação](https://qiskit.github.io/qiskit-addon-mthree/) para guias práticos, tutoriais e referências de API.
- Acesse a página do pacote no [GitHub](https://github.com/Qiskit/qiskit-addon-mthree).

### Cones de luz sombreados {#shaded-lightcones}

O addon de cones de luz sombreados utiliza a propagação de Pauli para reduzir o número de termos de erro em um modelo de ruído que precisam ser mitigados, de acordo com as especificidades do observável alvo. Isso tem o efeito de reduzir a sobrecarga de amostragem em fluxos de trabalho de cancelamento probabilístico de erros (PEC).

- Acesse a [documentação](https://qiskit.github.io/qiskit-addon-slc/) para guias práticos, tutoriais e referências de API.

### Absorção de ruído propagado {#propagated-noise-absorption}

A absorção de ruído propagado (PNA) utiliza a propagação de Pauli para absorver informações de um modelo de ruído em um observável alvo. Medir esse observável modificado tem o efeito de mitigar o ruído conforme representado pelo modelo.

- Acesse a [documentação](https://qiskit.github.io/qiskit-addon-pna/apidocs/qiskit_addon_pna.html) para guias práticos, tutoriais e referências de API.
