# Técnicas avançadas - Addons do Qiskit {#advanced-techniques-qiskit-addons}

Os addons do Qiskit são uma coleção de capacidades de pesquisa para possibilitar a descoberta de algoritmos em escala utilitária. Esses componentes de software modulares se constroem sobre a base de alto desempenho do Qiskit e podem ser integrados a um [fluxo de trabalho](/guides/intro-to-patterns) para escalar ou projetar novos algoritmos quânticos. Esta página destaca as ferramentas disponíveis nas principais categorias funcionais para ajudá-lo a escolher as capacidades relevantes ao construir seus fluxos de trabalho.
