# Configure sua conta IBM Cloud {#set-up-your-ibm-cloud-account}

Se você recebeu um convite por e-mail para entrar em uma conta, siga as etapas em [Configure sua conta IBM Cloud - usuários convidados](/guides/cloud-setup-invited).

Você pode executar cargas de trabalho em unidades de processamento quântico (QPUs) da IBM&reg; configurando uma conta no IBM Cloud&reg;. Você também precisará [instalar e configurar o Qiskit e o Qiskit Runtime](/guides/install-qiskit#local).

Sua conta de usuário está associada a uma ou mais [instâncias,](./instances) identificadas por _Cloud Resource Names_ (CRNs) que fornecem acesso aos serviços IBM Quantum. Além disso, um token de API exclusivo (também chamado de chave) é atribuído a cada conta, permitindo o acesso ao IBM Quantum pelo Qiskit. Para instruções sobre como escolher uma instância específica, consulte [Conectar-se a uma instância](./instances#connect-instance).

<span id="cloud"></span>
## Configurar para usar o IBM Cloud {#set-up-to-use-ibm-cloud}

### Antes de começar {#before-you-begin}

1. Se você ainda não tiver uma, crie uma [conta IBM Cloud.](https://quantum.cloud.ibm.com/registration)
   <Admonition type="note">
   Você pode fazer parte de várias contas IBM Cloud. Você pode acessar qualquer uma das suas contas IBM Cloud a qualquer momento pelo seletor de contas no cabeçalho da interface do [IBM Quantum Platform](https://quantum.cloud.ibm.com).
   </Admonition>
1. Faça login no [IBM Quantum Platform](https://quantum.cloud.ibm.com) com um IBMid ou conta Google. Se você não tiver um, será guiado pelo processo de criação. (Essas credenciais de login não são as credenciais da sua conta IBM Cloud.)
1. Certifique-se de que a conta e a região corretas estão selecionadas no seletor de contas no cabeçalho, conforme mostrado na imagem a seguir.

    A região controla onde seus jobs são executados e onde os dados dos jobs são armazenados. Você pode acessar qualquer região usando a mesma chave de API, mas só pode ver e acessar as instâncias criadas na região em que você está conectado.

    ![O cabeçalho do IBM Quantum Platform é exibido. O seletor de contas está à direita da barra de pesquisa. O seletor de região está à direita do seletor de contas.](/docs/images/guides/cloud-setup/IQP-Header.svg 'IBM Quantum Platform header')

### 1. Criar uma instância {#1-create-an-instance}

Você pode criar uma instância na página [Instâncias](https://quantum.cloud.ibm.com/instances) do IBM Quantum Platform. Consulte [Criar uma instância](/guides/instances#add-instance) para obter instruções detalhadas.

### 2. Opcional: Salvar suas credenciais de acesso {#2-optional-save-your-access-credentials}

Se você estiver usando um ambiente Python confiável, como um laptop pessoal, é recomendável usar o método `save_account()` para [salvar suas credenciais localmente.](/guides/save-credentials)

<span id="cloud-save"></span>
### 3. Conectar o Qiskit à sua instância de serviço {#3-connect-qiskit-with-your-service-instance}

- [Inicialize o serviço](/guides/initialize-account) em um **ambiente Python confiável**
- [Inicialize o serviço em um **ambiente não confiável**](/guides/cloud-setup-untrusted) (como um computador público)

<Admonition type="note" title="Notas">
   * Siga [estas instruções](/guides/cloud-setup-rest-api) se quiser conectar usando a REST API em vez do Qiskit.
   * Se necessário, use [estas informações](/guides/quickstart-steps-org#firewall) para configurar seu firewall e habilitar o acesso aos endpoints da API IBM Quantum.
</Admonition>
