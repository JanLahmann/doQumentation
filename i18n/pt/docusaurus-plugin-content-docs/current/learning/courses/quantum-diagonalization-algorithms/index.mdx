# Visão Geral {#overview}

!["Imagem de destaque"](/learning/images/courses/quantum-diagonalization-algorithms/hero.avif)

Este curso oferece uma visão geral de várias abordagens para diagonalização quântica. São fornecidos conceitos básicos sobre os métodos clássicos utilizados, ou que motivaram os algoritmos quânticos. O conteúdo então percorre a implementação dos algoritmos quânticos em computadores quânticos reais. Há uma discussão substancial sobre quais fatores determinam o escalonamento das abordagens que usam algoritmos clássicos e quânticos.

## Conhecimentos recomendados {#recommended-background}

Este curso é destinado a pessoas que têm alguma experiência com computação quântica e querem aproveitar seu poder na diagonalização de matrizes.

Recomendamos que os alunos completem os seguintes pré-requisitos antes deste curso:

- Configure seu ambiente local seguindo o guia [Primeiros passos](/guides/) na nossa página de documentação.
- Para uma compreensão mais aprofundada da informação quântica, experimente [Fundamentos de Informação Quântica](/learning/courses/basics-of-quantum-information) com John Watrous.
