# Visão geral {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

Bem-vindo ao *Use um computador quântico hoje*! Neste curso, você começará a usar um computador quântico imediatamente e aprenderá os princípios básicos da computação quântica ao longo do caminho.

Este curso foi criado para os curiosos e para quem tem pouco tempo. Você não precisa de um diploma em física nem de anos de experiência em programação. Se você consegue acompanhar um notebook e clicar em "Run", está pronto.
