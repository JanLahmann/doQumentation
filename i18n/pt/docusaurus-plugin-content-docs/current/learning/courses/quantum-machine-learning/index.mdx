# Visão geral {#overview}

!["Imagem de destaque"](/learning/images/courses/quantum-machine-learning/hero.svg)

Este curso explora como a computação quântica pode ter um papel empolgante no aprendizado de máquina. O curso oferece uma revisão das abordagens clássicas de aprendizado de máquina, mas também pressupõe alguma familiaridade com esses métodos. O foco principal será em como a computação quântica pode se integrar aos fluxos de trabalho de aprendizado de máquina já existentes. Os tópicos incluem codificação de dados, kernels quânticos e circuitos quânticos variacionais.

## Exame {#exam}

Para conquistar o seu badge de Aprendizado de máquina quântico, faça o exame no IBM&reg; Training. Este exame deve ser realizado após a leitura das lições deste curso. Depois que você passar no exame, a Credly vai te notificar que você ganhou um badge.
