# Exame e badge {#exam-and-badge}

![Um badge quadrado com o nome deste curso, nível fundamental, emitido pelo IBM Quantum&reg;.](/learning/images/courses/quantum-machine-learning/exam/badge.avif)

## Pesquisa pós-curso {#post-course-survey}

Parabéns por concluir este curso! Por favor, tire um momento para nos ajudar a melhorá-lo preenchendo esta [pesquisa rápida](https://your.feedback.ibm.com/jfe/form/SV_a68ils4bH0vX0O2). Seu feedback será usado para aprimorar nosso conteúdo e a experiência dos usuários. Obrigado!

## Teste seus conhecimentos {#test-your-knowledge}

Após concluir as leituras e atividades deste curso, clique no botão abaixo para acessar o IBM&reg; Training, onde você pode fazer o exame de badging para Aprendizado de máquina quântico.

O exame consiste em 20 questões, e você precisa responder corretamente pelo menos 16 delas para passar. Ao atingir uma pontuação aprovada, seu badge será enviado automaticamente para o e-mail que você usou para se registrar no IBM Training.

Certifique-se de que está pronto para fazer o exame. Se você não atingir a pontuação mínima em duas tentativas, o exame ficará temporariamente bloqueado.

Aviso: Esses exames no site do IBM Training podem estar temporariamente indisponíveis nos próximos dias, devido a manutenção.
