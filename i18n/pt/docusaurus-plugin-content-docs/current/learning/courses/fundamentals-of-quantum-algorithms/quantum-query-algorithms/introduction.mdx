Nesta primeira lição do curso, vamos formular um framework algorítmico simples — conhecido como o *modelo de consulta* — e explorar as vantagens que os computadores quânticos oferecem dentro desse framework.

O modelo de consulta de computação é como uma placa de Petri para ideias algorítmicas quânticas.
Ele é rígido e artificial no sentido de que não representa com precisão os tipos de problemas computacionais com os quais geralmente nos preocupamos na prática, mas ainda assim se mostrou incrivelmente útil como ferramenta para o desenvolvimento de técnicas algorítmicas quânticas.
Isso inclui as técnicas que impulsionam os algoritmos quânticos mais conhecidos, como o algoritmo de Shor para fatoração de inteiros.
O modelo de consulta também se mostra um framework muito útil para *explicar* técnicas algorítmicas quânticas.

Após introduzir o próprio modelo de consulta, vamos discutir o primeiro algoritmo quântico que foi descoberto, o *algoritmo de Deutsch,* juntamente com uma extensão do algoritmo de Deutsch conhecida como o *algoritmo de Deutsch-Jozsa*.
Esses algoritmos demonstram vantagens quantificáveis dos computadores quânticos em relação aos clássicos no contexto do modelo de consulta.
Em seguida, vamos discutir um algoritmo quântico conhecido como *algoritmo de Simon,* que oferece uma vantagem mais robusta e satisfatória dos computadores quânticos sobre as computações clássicas, por razões que serão explicadas quando chegarmos a esse ponto.
