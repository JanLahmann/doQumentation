## Teste seus conhecimentos {#test-your-knowledge}

Após concluir as leituras e atividades deste curso, clique no botão abaixo para acessar o IBM&reg; Training, onde você pode fazer o exame de certificação de Fundamentos de algoritmos quânticos.

O exame consiste em 20 questões, e você deve responder pelo menos 16 corretamente para ser aprovado. Ao obter uma pontuação aprovada, seu badge será enviado automaticamente para o e-mail que você usou para se registrar no IBM Training.

Certifique-se de que está pronto para fazer o exame. Se você não obtiver uma pontuação aprovada em duas tentativas, o exame ficará temporariamente bloqueado.

Aviso: Esses exames no site IBM Training podem estar temporariamente indisponíveis nos próximos dias devido a manutenção.
