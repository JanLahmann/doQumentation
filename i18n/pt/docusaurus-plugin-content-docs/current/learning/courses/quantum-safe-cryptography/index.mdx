# Visão geral {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

Uma introdução prática à criptografia quântica segura é um curso online gratuito no IBM Quantum&reg; Learning e serve como base para os conceitos fundamentais de criptografia quântica segura.

Este curso foi criado para desenvolvedores interessados em modernizar a segurança de suas aplicações e traz lições multimodais e exemplos de código ao vivo interativos em quatro áreas centrais de criptografia: funções de hash criptográficas, criptografia de chave simétrica, criptografia de chave assimétrica e criptografia quântica segura.

Ao fazer este curso, os desenvolvedores vão aprender como o cenário de risco em segurança cibernética está evoluindo, além de explorar abordagens contemporâneas que prometem moldar a era quântica.
