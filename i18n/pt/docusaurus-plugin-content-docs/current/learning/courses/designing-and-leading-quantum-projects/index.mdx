# Visão geral {#overview}

!["Hero image"](/learning/images/courses/designing-and-leading-quantum-projects/hero.svg)

Este curso foi desenvolvido para ajudar equipes e organizações a construir efetivamente as capacidades necessárias para iniciar com sucesso iniciativas quânticas.

Recomendamos dedicar aproximadamente 4 horas para concluir este conteúdo, dependendo da experiência prévia de cada um.

### Público-alvo {#intended-audience}

Recomendamos este curso para líderes e arquitetos de iniciativas quânticas, incluindo professores acadêmicos, líderes de projetos em grandes empresas, lideranças de startups e outros. Será especialmente útil para organizações que estão começando no computação quântica.

## Conhecimento recomendado {#recommended-background}

Não há pré-requisito técnico para este curso, mas recomendamos que os estudantes façam primeiro o curso [Quantum Business Foundations](/learning/courses/quantum-business-foundations), que é uma boa introdução ao ecossistema quântico.

Se você tiver dúvidas ou preocupações sobre este curso, entre em contato com a equipe IBM Quantum Learning&reg;.
