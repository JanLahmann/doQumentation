# Exame e badge {#exam-and-badge}

![A square badge stating the name of this course, intermediate level, and issued by IBM Quantum&reg;.](/learning/images/courses/integrating-quantum-and-high-performance-computing/exam/badge.svg)

## Pesquisa pós-curso {#post-course-survey}

Parabéns por concluir o curso "Integrando computação quântica e de alto desempenho" da IBM&reg;!

## Teste seus conhecimentos {#test-your-knowledge}

Após concluir as leituras e atividades deste curso, clique no botão abaixo para navegar até o IBM Training, onde você pode fazer o exame de badge para Integrando computação quântica e de alto desempenho.

O exame é composto por 20 questões, e você deve responder pelo menos 16 corretamente para passar. Ao receber uma pontuação aprovada, sua badge será enviada automaticamente para o e-mail que você usou para se registrar no IBM Training.

Certifique-se de estar pronto(a) para fazer o exame. Se você não receber uma pontuação aprovada em duas tentativas, o exame ficará temporariamente bloqueado.
