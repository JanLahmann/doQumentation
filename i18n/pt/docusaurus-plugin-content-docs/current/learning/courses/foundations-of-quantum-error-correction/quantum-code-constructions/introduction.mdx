Vimos alguns exemplos de códigos de correção de erros quânticos em lições anteriores deste curso, incluindo o código de Shor de 9 qubits, o código de Steane de 7 qubits e o código de 5 qubits.
Esses códigos são inegavelmente interessantes e representam um ponto de partida natural para explorar a correção de erros quânticos, mas um problema com eles é que conseguem tolerar apenas uma taxa de erros muito baixa.
Corrigir um erro em um qubit de cinco, sete ou nove não é ruim, mas muito provavelmente precisaremos ser capazes de tolerar muito mais erros do que isso para tornar a computação quântica em larga escala uma realidade.

Nesta lição, veremos pela primeira vez algumas construções de códigos de correção de erros quânticos mais sofisticadas, incluindo códigos capazes de tolerar uma taxa de erros muito mais alta do que os que vimos até agora, e que são considerados candidatos promissores para a correção prática de erros quânticos.

Começaremos com uma classe de códigos de correção de erros quânticos conhecida como *códigos CSS*, nomeada em homenagem a Robert Calderbank, Peter Shor e Andrew Steane, que os descobriram pela primeira vez.
A construção do código CSS permite pegar certos *pares* de códigos de correção de erros clássicos e combiná-los em um único código de correção de erros quânticos.

A segunda parte da lição trata de um código conhecido como *código tórico*.
Este é um exemplo fundamental (e verdadeiramente belo) de um código de correção de erros quânticos capaz de tolerar taxas de erros relativamente altas.
Na verdade, o código tórico não é um único exemplo de código de correção de erros quânticos, mas sim uma família infinita de códigos, um para cada inteiro positivo maior que um.

Por fim, na última parte da lição, discutiremos brevemente outras famílias de códigos quânticos, incluindo os *surface codes* (que estão intimamente relacionados ao código tórico) e os *color codes*.
