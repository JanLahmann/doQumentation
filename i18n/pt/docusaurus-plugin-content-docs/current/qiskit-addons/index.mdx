# Qiskit Addons

Os [Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) são pacotes oficiais de extensão que fornecem algoritmos e técnicas especializados de computação quântica. Cada addon foca em uma área específica e inclui tutoriais com código executável.

## Addons disponíveis {#available-addons}

| Addon | Pacote | Descrição |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Reduz a largura e a profundidade do circuito cortando portas e fios |
| **SQD** | `qiskit-addon-sqd` | Diagonalização quântica baseada em amostras para estimativa do estado fundamental |
| **OBP** | `qiskit-addon-obp` | Retropropagação de operadores para reduzir a profundidade do circuito |
| **MPF** | `qiskit-addon-mpf` | Fórmulas de múltiplos produtos para simulação aprimorada de Hamiltonianos |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Compilação Quântica Aproximada com redes de tensores |
| **PNA** | `qiskit-addon-pna` | Absorção de ruído propagado para mitigação de erros |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones para reduzir o overhead de amostragem do PEC |

Todos os tutoriais podem ser executados diretamente no seu navegador usando o Binder, ou no seu próprio servidor Jupyter.

Para a documentação da API, visite os repositórios individuais de cada addon no [GitHub](https://github.com/Qiskit?q=qiskit-addon).
