# Prezentare generală

!["Hero image"](/learning/images/courses/utility-scale-quantum-computing/hero.svg)

Acest curs de reluare a evenimentului constă din 14 lecții și laboratoare dezvoltate și conduse de IBM Quantum&reg; în colaborare cu Universitatea din Tokyo. Aceste lecții acoperă o varietate largă de subiecte importante în calculul cuantic, dar sunt axate pe construirea unui calcul la scară utilitară. Rezultatul este foarte similar cu lucrarea publicată pe coperta revistei Nature în iunie 2023.
