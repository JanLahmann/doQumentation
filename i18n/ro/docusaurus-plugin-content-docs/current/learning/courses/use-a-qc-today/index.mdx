# Prezentare generală {#overview}

Bun venit la *Folosește un calculator cuantic astăzi*! În cadrul acestui curs, vei începe să folosești un calculator cuantic imediat și vei învăța principiile de bază ale calculului cuantic pe parcurs.

Acest curs este conceput pentru cei curioși și cei ocupați. Nu ai nevoie de o diplomă în fizică sau de ani de experiență în programare. Dacă poți urmări un notebook și apăsa „Run", ești pregătit.
