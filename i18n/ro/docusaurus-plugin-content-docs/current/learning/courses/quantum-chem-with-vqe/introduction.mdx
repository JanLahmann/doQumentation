Acest curs este o introducere concentrată în variational quantum eigensolver (VQE), incluzând aplicarea VQE în chimie. Cursul se adresează practicienilor tehnici ai calculului cuantic care sunt interesați să aplice calculul cuantic la probleme chimice, biochimice sau biofizice. Cursul acoperă elemente importante ale unui calcul VQE, inclusiv acestea:

* Hamiltonian
* Ansatz
* Optimizer clasic

Toate aceste elemente sunt reunite pentru a construi calcule VQE pe Qiskit Runtime Primitives, în principal Estimator. Pentru informații despre primitive, consultă [documentația](/guides/primitives). La configurarea unui calcul VQE, vom urma cadrul standard Qiskit patterns:

* Mapează intrările clasice la o problemă cuantică
* Optimizează problema pentru execuție cuantică
* Execută folosind Qiskit Runtime primitives
* Post-procesează, returnează rezultatul în format clasic

O parte din explicația cadrului va fi găsită pe parcursul acestui curs, dar pentru o explicație completă cu exemple de Qiskit patterns, consultă [această documentație](/guides/intro-to-patterns).
