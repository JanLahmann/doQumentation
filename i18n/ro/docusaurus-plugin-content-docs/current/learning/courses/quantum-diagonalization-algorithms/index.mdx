# Prezentare generală

!["Hero image"](/learning/images/courses/quantum-diagonalization-algorithms/hero.avif)

Acest curs oferă o prezentare generală a mai multor abordări ale diagonalizării cuantice. Este furnizat un context despre metodele clasice care sunt utilizate sau care au motivat algoritmii cuantici. Conținutul parcurge apoi implementarea algoritmilor cuantici pe calculatoare cuantice reale. Există o discuție substanțială despre factorii care determină scalarea abordărilor folosind algoritmi clasici și cuantici.

## Cunoștințe recomandate

Acest curs este destinat persoanelor care au ceva experiență cu calculul cuantic și doresc să valorifice puterea sa în diagonalizarea matricială.

Recomandăm cursanților să finalizeze următoarele cunoștințe preliminare înainte de acest curs:

- Configurează-ți mediul local urmând ghidul [Primii pași](/guides/) din pagina noastră de documentație.
- Pentru o înțelegere mai profundă a informației cuantice, încearcă [Noțiuni de bază ale informației cuantice](/learning/courses/basics-of-quantum-information) cu John Watrous.
