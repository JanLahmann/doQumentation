# Prezentare generală {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

O introducere practică în criptografia sigură cuantic este un curs online gratuit în IBM Quantum&reg; Learning și servește ca o introducere în conceptele fundamentale ale criptografiei sigure cuantic.

Acest curs a fost creat pentru dezvoltatori care doresc să modernizeze securitatea aplicațiilor lor și include lecții multimodale și exemple de cod live interactive în patru domenii principale ale criptografiei: funcții hash criptografice, criptografie cu cheie simetrică, criptografie cu cheie asimetrică și criptografie sigură cuantic.

Parcurgând acest curs, vei învăța cum evoluează peisajul riscurilor de securitate cibernetică și vei explora abordări contemporane pregătite să modeleze era cuantică.
