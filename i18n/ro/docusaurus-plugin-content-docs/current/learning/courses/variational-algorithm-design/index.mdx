# Prezentare generală

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Acest curs este destinat persoanelor care au o anumită experiență cu calculul cuantic și doresc să învețe cum să scrie algoritmi variaționale folosind primitivele Qiskit Runtime. Este, de asemenea, potrivit pentru profesioniștii în calcul cuantic care doresc să-și extindă cunoștințele și abilitățile în domeniul algoritmilor variaționale.

## Examen

Pentru a obține insigna pentru Variational algorithm design, susține examenul la IBM&reg; Training. Acest examen este destinat să fie susținut după parcurgerea lecțiilor din acest curs. După ce promovezi examenul, vei fi notificat de Credly că ai obținut o insignă.
