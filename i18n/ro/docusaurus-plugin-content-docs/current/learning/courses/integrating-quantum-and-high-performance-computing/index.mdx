# Prezentare generală

!["Hero image"](/learning/images/courses/integrating-quantum-and-high-performance-computing/index/hero.svg)

Acest curs oferă o prezentare generală a resurselor clasice și cuantice, precum și a fluxurilor de lucru hibride cu exemple.

## Background recomandat

Acest curs este destinat a două grupuri distincte de persoane: celor cu experiență în computație cuantică, dar nu în HPC, și celor cu experiență în HPC, dar nu în computație cuantică. Aceste tehnologii se completează reciproc, iar cei cu expertiză în una sunt din ce în ce mai conștienți de importanța celeilalte. Participanții pot sări peste secțiunile din propriul domeniu de expertiză, după cum este cazul.

Recomandăm ca participanții să finalizeze următoarea condiție prealabilă înainte de acest curs:

- Configurează-ți mediul local urmând ghidul [Începe](/guides/) din pagina noastră de documentație.

Pentru o înțelegere mai profundă a informației cuantice, finalizează acest curs opțional:
- [Bazele informației cuantice](/learning/courses/basics-of-quantum-information) cu John Watrous.
