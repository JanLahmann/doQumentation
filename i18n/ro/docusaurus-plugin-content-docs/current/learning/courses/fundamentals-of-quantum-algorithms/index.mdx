# Prezentare generală

!["Hero image"](/learning/images/courses/fundamentals-of-quantum-algorithms/hero.svg)

Bine ai venit la *Fundamentele algoritmilor cuantici,* al doilea curs din seria *Înțelegerea informației și a calculului cuantic*, care cuprinde următoarele cursuri:

- [Bazele informației cuantice](/learning/courses/basics-of-quantum-information)
- Fundamentele algoritmilor cuantici (acest curs)
- [Formularea generală a informației cuantice](/learning/courses/general-formulation-of-quantum-information)
- [Fundamentele corecției erorilor cuantice](/learning/courses/foundations-of-quantum-error-correction)

Acest curs explorează avantajele computaționale ale informației cuantice, inclusiv ce putem face cu calculatoarele cuantice și avantajele lor față de calculatoarele clasice.
Începe cu algoritmi cuantici de interogare, care oferă demonstrații simple de principiu pentru algoritmii cuantici, și continuă cu algoritmi cuantici pentru probleme
precum factorizarea numerelor întregi și căutarea nestructurată.

Acest curs este destinat studenților, profesioniștilor și pasionaților din domenii precum informatică, fizică, inginerie și matematică, care doresc să dobândească cunoștințe despre fundamentele teoretice ale informației și calculului cuantic.
