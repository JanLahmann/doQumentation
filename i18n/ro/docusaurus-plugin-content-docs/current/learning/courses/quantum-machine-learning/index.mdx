# Prezentare generală {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Acest curs explorează modul în care calculul cuantic poate juca un rol interesant în învățarea automată. Cursul va oferi o trecere în revistă a abordărilor clasice de învățare automată, dar va presupune și o anumită familiaritate cu aceste metode. Accentul principal va fi pus pe modul în care calculul cuantic se poate integra în fluxurile de lucru existente de învățare automată. Subiectele includ codificarea datelor, kerneluri cuantice și circuite cuantice variationale.

## Examen {#exam}

Pentru a obține insigna pentru Quantum machine learning, susține examenul la IBM&reg; Training. Acest examen este destinat a fi susținut după parcurgerea lecțiilor din acest curs. După ce treci examenul, vei fi notificat de Credly că ai câștigat o insignă.
