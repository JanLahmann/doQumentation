# Prezentare generală

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Acest curs explorează modul în care calculul cuantic poate juca un rol captivant în învățarea automată. Cursul va oferi o recapitulare a abordărilor clasice de învățare automată, dar va presupune și o anumită familiaritate cu aceste metode. Focusul principal va fi modul în care calculul cuantic se poate integra în fluxurile de lucru existente de învățare automată. Subiectele includ codificarea datelor, kernel-uri cuantice și circuite cuantice variationale.

## Examen

Pentru a obține insigna pentru Quantum machine learning, susține examenul la IBM&reg; Training. Acest examen este destinat să fie susținut după parcurgerea lecțiilor din acest curs. După ce promovezi examenul, vei fi notificat de Credly că ai obținut o insignă.
