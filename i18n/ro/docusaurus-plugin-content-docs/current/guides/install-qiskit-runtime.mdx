# Instalarea clientului Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime este serviciul bazat pe cloud pentru executarea calculelor cuantice pe hardware IBM Quantum&reg;. Acest ghid îți arată cum să instalezi `qiskit-ibm-runtime`, care este pachetul client pentru accesarea serviciului Qiskit Runtime.

Înainte de a instala clientul Qiskit Runtime, urmează instrucțiunile din aceste ghiduri:

1. [Instalare Qiskit](/guides/install-qiskit).
2. [Configurare cont IBM Cloud](cloud-setup).

Rulează această comandă pentru a instala Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Rămâi la curent cu cele mai recente versiuni">
Verifică periodic [notele de lansare Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) pentru a vedea noile versiuni. Recomandăm actualizarea frecventă a cerințelor pentru `qiskit-ibm-runtime` prin rularea `pip install -U qiskit-ibm-runtime`.
</Admonition>
