# Ghid de migrare Qiskit 1.0 {#qiskit-10-migration-guide}

Qiskit 1.0 aduce noi garanții de stabilitate și îmbunătățiri de performanță, dar
nu este compatibil cu nicio versiune anterioară a Qiskit. Consultă
următoarele ghiduri pentru a-ți actualiza mediul și codul pentru a funcționa cu Qiskit v1.0.

- [Instalează noul pachet Qiskit v1.0](/guides/qiskit-1.0-installation) pentru actualizarea mediilor Python și a dependențelor de proiect.
- [Modificări de funcționalități Qiskit v1.0](/guides/qiskit-1.0-features) pentru actualizarea codului tău.
- [Modificări de pachete cu impact major în Qiskit v1.0](/guides/metapackage-migration) pentru a înțelege structura noilor și vechilor pachete.

Citește mai multe despre beneficiile Qiskit v1.0 pe [blogul IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
