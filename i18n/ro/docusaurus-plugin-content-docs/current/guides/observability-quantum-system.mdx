# Evenimente de urmărire a activității pentru serviciul IBM Quantum System {#activity-tracking-events-for-the-ibm-quantum-system-service}

Serviciile IBM Cloud&reg;, cum ar fi serviciul IBM&reg; Quantum System, generează evenimente de urmărire a activității.

Evenimentele de urmărire a activității raportează activitățile care modifică starea unui serviciu în IBM Cloud. Poți folosi aceste evenimente pentru a investiga activități anormale și acțiuni critice și pentru a respecta cerințele de audit reglementar.

Poți utiliza IBM Cloud Activity Tracker Event Routing, un serviciu de platformă, pentru a direcționa evenimentele de audit din contul tău către destinații la alegere, configurând ținte și rute care definesc unde sunt trimise evenimentele de urmărire a activității. Pentru mai multe informații, consultă [Despre IBM Cloud Activity Tracker Events Routing](https://cloud.ibm.com/docs/atracker?topic=atracker-about).

Poți utiliza IBM Cloud Logs pentru a vizualiza și a primi alerte cu privire la evenimentele generate în contul tău și direcționate de IBM Cloud Activity Tracker Event Routing către o instanță IBM Cloud Logs.

## Vizualizarea evenimentelor de urmărire a activității pentru serviciul IBM Quantum System {#viewing-activity-tracking-events-for-the-ibm-quantum-system-service}
