# Tehnici avansate - Addon-uri Qiskit {#advanced-techniques-qiskit-addons}

Addon-urile Qiskit sunt o colecție de capabilități de cercetare pentru a permite descoperirea algoritmilor la scară de utilitate. Aceste componente software modulare se construiesc pe fundația performantă a Qiskit și pot fi integrate într-un [flux de lucru](/guides/intro-to-patterns) pentru a scala sau a proiecta noi algoritmi cuantici. Această pagină evidențiază instrumentele disponibile în categorii funcționale cheie pentru a te ajuta să alegi capabilitățile relevante atunci când îți construiești fluxurile de lucru.
