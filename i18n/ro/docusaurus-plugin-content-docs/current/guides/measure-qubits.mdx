Pentru a obține informații despre starea unui Qubit, îl poți _măsura_ pe un [bit clasic](https://docs.quantum.ibm.com/api/qiskit/circuit#qiskit.circuit.Clbit). În Qiskit, măsurătorile se efectuează în baza computațională, adică baza Pauli-$Z$ pentru un singur Qubit. Prin urmare, o măsurătoare produce 0 sau 1, în funcție de suprapunerea cu stările proprii Pauli-$Z$ $|0\rangle$ și $|1\rangle$:

$$
|q\rangle \xrightarrow{measure}\begin{cases}
      0 (\text{outcome}+1), \text{with probability } p_0=|\langle q|0\rangle|^{2}\text{,} \\
      1 (\text{outcome}-1), \text{with probability } p_1=|\langle q|1\rangle|^{2}\text{.}
    \end{cases}
$$

## Aplicarea unei măsurători într-un Circuit {#apply-a-measurement-to-a-circuit}

Există mai multe modalități de a aplica măsurători într-un Circuit:

### Metoda `QuantumCircuit.measure` {#quantumcircuitmeasure-method}

Folosește metoda [`measure`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure) pentru a măsura un [`QuantumCircuit`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#quantumcircuit-class).

Exemple:

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(5, 5)
qc.x(0)
qc.x(1)
qc.x(4)
qc.measure(
    range(5), range(5)
)  #  Measures all qubits into the corresponding clbit.
```

### Clasa `Measure` {#measure-class}

Clasa Qiskit [Measure](https://docs.quantum.ibm.com/api/qiskit/circuit#qiskit.circuit.Measure) măsoară qubiții specificați.

```python
from qiskit.circuit import Measure

qc = QuantumCircuit(3, 1)
qc.x([0, 1])
qc.append(Measure(), [0], [0])  # measure qubit 0 into clbit 0
```

### Metoda `QuantumCircuit.measure_all` {#quantumcircuitmeasure_all-method}

Pentru a măsura toți qubiții în biții clasici corespunzători, folosește metoda [`measure_all`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure_all). În mod implicit, această metodă adaugă noi biți clasici într-un `ClassicalRegister` pentru a stoca aceste măsurători.

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(3, 1)
qc.x([0, 2])
qc.measure_all()  # Measure all qubits.
```

### Metoda `QuantumCircuit.measure_active` {#quantumcircuitmeasure_active-method}

Pentru a măsura toți qubiții care nu sunt inactivi, folosește metoda [`measure_active`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure_active). Această metodă creează un nou `ClassicalRegister` cu o dimensiune egală cu numărul de qubiți non-inactivi care sunt măsurați.

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(3, 1)
qc.x([0, 2])
qc.measure_active()  # Measure qubits that are not idle, that is, qubits 0 and 2.
```
