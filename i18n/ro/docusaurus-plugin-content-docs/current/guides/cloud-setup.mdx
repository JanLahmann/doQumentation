# Configurează-ți contul IBM Cloud {#set-up-your-ibm-cloud-account}

Dacă ai primit o invitație prin e-mail pentru a te alătura unui cont, urmează în schimb pașii din [Configurează-ți contul IBM Cloud - utilizatori invitați](/guides/cloud-setup-invited).

Poți rula sarcini de lucru pe unitățile de procesare cuantică (QPU) IBM&reg; configurând un cont pe IBM Cloud&reg;. Va trebui, de asemenea, să [instalezi și să configurezi Qiskit și Qiskit Runtime](/guides/install-qiskit#local).

Contul tău de utilizator este asociat cu una sau mai multe [instanțe,](./instances) identificate prin _Cloud Resource Names_ (CRN-uri) care oferă acces la serviciile IBM Quantum. În plus, fiecărui cont îi este atribuit un token API unic (numit și cheie), permițând accesul la IBM Quantum din Qiskit. Pentru instrucțiuni de conectare la o instanță specifică, consultă [Conectează-te la o instanță](./instances#connect-instance).

<span id="cloud"></span>
## Configurare pentru utilizarea IBM Cloud {#set-up-to-use-ibm-cloud}

### Înainte de a începe {#before-you-begin}

1. Dacă nu ai deja unul, creează un [cont IBM Cloud.](https://quantum.cloud.ibm.com/registration)
   <Admonition type="note">
   Poți face parte din mai multe conturi IBM Cloud. Poți accesa oricare dintre conturile tale IBM Cloud oricând din selectorul de conturi din antetul interfeței [IBM Quantum Platform](https://quantum.cloud.ibm.com).
   </Admonition>
1. Autentifică-te pe [IBM Quantum Platform](https://quantum.cloud.ibm.com) cu un IBMid sau un cont Google. Dacă nu ai unul, vei fi ghidat prin procesul de creare. (Aceste credențiale de autentificare nu sunt credențialele contului tău IBM Cloud.)
1. Asigură-te că în selectorul de conturi din antet sunt selectate contul și regiunea corecte, așa cum este arătat în imaginea următoare.

    Regiunea controlează unde sunt rulate joburile tale și unde sunt păstrate datele joburilor. Poți accesa oricare regiune folosind aceeași cheie API, dar poți vedea și accesa doar instanțele care au fost create în regiunea în care ești autentificat.

    ![Antetul IBM Quantum Platform este afișat. Selectorul de conturi se află în dreapta barei de căutare. Selectorul de regiune se află în dreapta selectorului de conturi.](/docs/images/guides/cloud-setup/IQP-Header.svg 'IBM Quantum Platform header')

### 1. Creează o instanță {#1-create-an-instance}

Poți crea o instanță din pagina [Instances](https://quantum.cloud.ibm.com/instances) a IBM Quantum Platform. Consultă [Creează o instanță](/guides/instances#add-instance) pentru instrucțiuni detaliate.

### 2. Opțional: Salvează-ți credențialele de acces {#2-optional-save-your-access-credentials}

Dacă folosești un mediu Python de încredere, cum ar fi un laptop personal, este recomandat să folosești metoda `save_account()` pentru a-ți [salva credențialele local.](/guides/save-credentials)

<span id="cloud-save"></span>
### 3. Conectează Qiskit cu instanța ta de serviciu {#3-connect-qiskit-with-your-service-instance}

- [Inițializează serviciul](/guides/initialize-account) într-un **mediu Python de încredere**
- [Inițializează serviciul într-un **mediu neîncrezut**](/guides/cloud-setup-untrusted) (cum ar fi un computer public)

<Admonition type="note" title="Note">
   * Urmează [aceste instrucțiuni](/guides/cloud-setup-rest-api) dacă dorești să te conectezi folosind REST API în loc de Qiskit.
   * Dacă este necesar, folosește [aceste informații](/guides/quickstart-steps-org#firewall) pentru a configura firewall-ul tău și a permite accesul la endpoint-urile API IBM Quantum.
</Admonition>
