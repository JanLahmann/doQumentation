# Începe cu primitivele backend {#get-started-with-the-backend-primitives}

Spre deosebire de primitivele specifice unui furnizor, primitivele backend sunt implementări generice care pot fi utilizate cu orice obiect `backend`, atâta timp cât implementează interfața [`BackendV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.providers.BackendV2). Unii furnizori implementează primitive în mod nativ. Consultă [pagina Qiskit Ecosystem](https://www.ibm.com/quantum/ecosystem?tag=provider) pentru detalii.

<span id="estimator"></span>
## Începe cu primitiva backend Estimator {#get-started-with-the-estimator-backend-primitive}

Primitiva Estimator poate fi rulată cu orice furnizor folosind clasa [`qiskit.primitives.BackendEstimatorV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendEstimatorV2). Cu toate acestea, nu oferă implementări de atenuare a erorilor de măsurare sau de poartă „out-of-the-box", deoarece primitivele backend sunt concepute să ruleze local pe mașina utilizatorului.

### Exemplu: {#example}

```python
from qiskit.primitives import BackendEstimatorV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
estimator = BackendEstimatorV2(backend)
```

<span id="sampler"></span>
## Începe cu primitiva backend Sampler {#get-started-with-the-sampler-backend-primitive}

Primitiva Sampler poate fi rulată cu orice furnizor folosind [`qiskit.primitives.BackendSamplerV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendSamplerV2). Cu toate acestea, necesită un backend care suportă opțiunea `memory`.

### Exemplu: {#example}

```python
from qiskit.primitives import BackendSamplerV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
sampler = BackendSamplerV2(backend)
```
