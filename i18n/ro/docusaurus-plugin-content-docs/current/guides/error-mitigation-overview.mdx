# Prezentare generală a tehnicilor de gestionare a zgomotului {#overview-of-noise-management-techniques}

La executarea sarcinilor de lucru cuantice, există mai multe modalități de a reduce impactul zgomotului. Qiskit addons open-source oferă tehnici de atenuare și suprimare a erorilor care se integrează direct în fluxul de lucru de dezvoltare, în timp ce Qiskit Runtime aplică strategii avansate de atenuare a erorilor în mod automat atunci când joburile sunt trimise pentru execuție. Această pagină indexează toate instrumentele și caracteristicile disponibile în ambele opțiuni pentru a te ajuta să alegi abordarea potrivită pentru a gestiona zgomotul atunci când construiești sarcini de lucru cuantice.
