# Open source

Din 2016, IBM Quantum&reg; a valorificat puterea inovației bazate pe comunitate, punând la dispoziția tuturor resursele de calcul cuantic prin intermediul cloud-ului.
Această inițiativă revoluționară include deschiderea codului-sursă pentru componente-cheie ale stivei de software cuantic și introducerea Qiskit, un framework versatil de calcul cuantic.
Astăzi, Qiskit este cel mai popular framework cuantic, datorat în parte comunității sale vibrate.
Comunitatea Qiskit este un grup dinamic și primitor de dezvoltatori, oameni de știință și entuziaști care colaborează pentru a avansa software-ul, cercetarea, educația și afacerile prin intermediul calculului cuantic.
Făcând această tehnologie transformatoare deschisă și accesibilă, accelerăm inovația și descoperim noi aplicații.

Pe măsură ce tehnologia cuantică evoluează, contribuțiile din proiectele și ale dezvoltatorilor din ecosistemul open-source [Qiskit ecosystem](https://www.ibm.com/quantum/ecosystem) sunt esențiale în modelarea viitorului său.

Alăturându-te comunității Qiskit, poți participa la dezvoltarea tehnologiilor cuantice.
Fie că ești un programator cuantic experimentat sau faci primii pași în călătoria ta cuantică, Qiskit îți oferă suportul și inspirația necesare pentru a depăși limitele.
Primim cu bucurie toate tipurile de contribuții, de la cod în pull request-uri până la suport comunitar pe [Slack](https://qisk.it/join-slack), asigurându-ne că toată lumea poate juca un rol în avansarea tehnologiei cuantice.
