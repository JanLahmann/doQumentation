# Instalarea clientului Qiskit Runtime din sursă {#install-the-qiskit-runtime-client-from-source}

Când instalezi Qiskit Runtime din sursă, poți accesa versiunea curentă de dezvoltare, în loc să folosești versiunea din depozitul Python Package Index (PyPI). Poți astfel inspecta și extinde mai eficient cea mai recentă versiune a codului.

## Crearea și activarea unui nou mediu virtual {#create-and-activate-a-new-virtual-environment}

1. Navighează la directorul proiectului și creează un mediu minimal cu doar Python instalat în el.

        <OperatingSystemTabs>
          <TabItem value="mac" label="macOS">
            ```shell
            python3 -m venv .venv
            ```
          </TabItem>

          <TabItem value="linux" label="Linux">
            ```shell
            python3 -m venv .venv
            ```
          </TabItem>

          <TabItem value="win" label="Windows">
            ```text
            python -m venv .venv
            ```
          </TabItem>
         </OperatingSystemTabs>

1. Activează noul mediu.

         <OperatingSystemTabs>
           <TabItem value="mac" label="macOS">
             ```shell
             source .venv/bin/activate
             ```
           </TabItem>

           <TabItem value="linux" label="Linux">
             ```shell
             source .venv/bin/activate
             ```
           </TabItem>

           <TabItem value="win" label="Windows">
             ```text
             .venv\Scripts\Activate.ps1
             ```
           </TabItem>
         </OperatingSystemTabs>

## Instalarea Qiskit Runtime {#install-qiskit-runtime}

Urmează acești pași pentru a instala Qiskit Runtime:

1. Clonează depozitul Qiskit Runtime.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. Schimbă directorul la `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. Instalează `qiskit-runtime`. Recomandăm utilizarea unui [mediu virtual](https://docs.python.org/3/library/venv.html) pentru a evita poluarea instalării globale Python.

  * **Instalare standard**:

    ```bash
    pip install .
    ```

  * **Mod editabil**: În acest mod, nu trebuie să reinstalezi Qiskit când există modificări de cod în proiect.

    ```bash
    pip install -e .
    ```

    În modul editabil, extensiile compilate sunt construite în _modul de depanare_ fără optimizări.

4. (Opțional) Dacă vrei să rulezi teste sau verificări de linting, instalează cerințele pentru dezvoltatori. Recomandăm utilizarea unui [mediu virtual](https://docs.python.org/3/library/venv.html) pentru a evita poluarea instalării globale Python.

```bash
pip install -e ".[dev]"
```
