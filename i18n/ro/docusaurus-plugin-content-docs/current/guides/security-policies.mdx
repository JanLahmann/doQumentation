# Politici de securitate ale depozitelor {#repository-security-policies}

Este extrem de important să îți menții sistemele cât mai sigure posibil. Prin urmare, se recomandă cu tărie să folosești instrumente de securitate precum [Dependabot](https://github.com/dependabot), care te notifică automat când dependențele sunt depășite.
