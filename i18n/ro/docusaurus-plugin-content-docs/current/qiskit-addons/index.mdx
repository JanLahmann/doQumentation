# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) sunt pachete de extensie oficiale care oferă algoritmi și tehnici specializate de calcul cuantic. Fiecare addon se concentrează pe un domeniu specific și include tutoriale cu cod executabil.

## Addons disponibile {#available-addons}

| Addon | Pachet | Descriere |
|-------|--------|-----------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Reduce lățimea și adâncimea Circuit-ului prin tăierea Gate-urilor și firelor |
| **SQD** | `qiskit-addon-sqd` | Diagonalizare Cuantică Bazată pe Eșantionare pentru estimarea stării fundamentale |
| **OBP** | `qiskit-addon-obp` | Backpropagare de Operatori pentru reducerea adâncimii Circuit-ului |
| **MPF** | `qiskit-addon-mpf` | Formule Multi-Produs pentru simulare Hamiltoniană îmbunătățită |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Compilare Cuantică Aproximativă cu rețele de tensori |
| **PNA** | `qiskit-addon-pna` | Absorbție de Zgomot Propagat pentru atenuarea erorilor |
| **SLC** | `qiskit-addon-slc` | Conuri de Lumină Umbrite pentru reducerea suprasolicitării de eșantionare PEC |

Toate tutorialele pot fi executate direct în browser folosind Binder sau pe propriul tău server Jupyter.

Pentru documentația API, vizitează repo-urile individuale ale addon-urilor pe [GitHub](https://github.com/Qiskit?q=qiskit-addon).
