# Instalace klienta Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime je cloudová služba pro provádění kvantových výpočtů na hardwaru IBM Quantum&reg;. Tento průvodce ti ukáže, jak nainstalovat `qiskit-ibm-runtime`, což je klientský balíček pro přístup ke službě Qiskit Runtime.

Před instalací klienta Qiskit Runtime postupuj podle pokynů v těchto průvodcích:

1. [Instalace Qiskit](/guides/install-qiskit).
2. [Nastavení účtu IBM Cloud](cloud-setup).

Spusť tento příkaz pro instalaci Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Udržuj se aktuální s nejnovějšími verzemi">
Pravidelně kontroluj [poznámky k vydání Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes), abys viděl nová vydání. Doporučujeme často aktualizovat své požadavky pro `qiskit-ibm-runtime` spuštěním `pip install -U qiskit-ibm-runtime`.
</Admonition>
