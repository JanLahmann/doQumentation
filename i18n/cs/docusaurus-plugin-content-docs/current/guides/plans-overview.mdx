# Přehled plánů {#plans-overview}

IBM Quantum Platform nabízí plán Open (bezplatný přístup) a placené plány, jak je popsáno níže. Úplné podrobnosti najdeš na stránce [IBM Quantum access plans](https://www.ibm.com/quantum/pricing).

Pokud chceš mít přístup k více plánům, můžeš vytvořit více instancí a přiřadit jim různé plány. Můžeš mít například instanci s plánem Open a instanci s plánem Pay-As-You-Go. Preferované instance k použití nastavíš v inicializátoru `QiskitRuntimeService` nebo metodou `QiskitRuntimeService.save_account()`. Příklad:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    # For `token`, use the 44-character API_KEY you created 
    # and saved from the IBM Quantum Platform Home dashboard
    # Your token is confidential. Do not share your key in public code.
    token="<your-API_KEY>", 
    # Optionally specify the instance to use.
    instance="<IBM Cloud CRN or instance name>", 
    # Optionally set the types of plans to prioritize.  
    # This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded 
    # if the plan name is not specified.
    plans_preference="['open', 'premium']", 
    # Optionally set the region to prioritize. 
    # Accepted values are 'us-east' or 'eu-de'. 
    # This is ignored if the instance is specified.
    region="us-east", 
    # Optionally name this set of account credentials.
    name="<account-name>", 
    # Optionally set these as your default credentials.
    set_as_default=True, 
  )
```

## Plán Open {#open-plan}

Spouštěj své kvantové Circuit na nejlepších QPU světa zdarma (až 10 minut za pohyblivé 28denní okno). Využití můžeš sledovat na stránkách [Platform dashboard](https://quantum.cloud.ibm.com/) a [Workloads](https://quantum.cloud.ibm.com/workloads). Instance plánu Open lze vytvářet pouze v oblasti us-east (oblast lze přepnout v rozbalovací nabídce v horní části Platform dashboard). [Kliknutím přejdeš na stránku registrace](https://quantum.cloud.ibm.com/registration).

Tento plán je doporučen těm, kteří se učí kvantové výpočty a prozkoumávají technologii IBM Quantum.

<Admonition type="note" title="Časově omezená akce">
Od 16. března 2026 si aktivní uživatelé plánu Open mohou zvolit dodatečných 180 minut na dalších 12 měsíců.
Přečti si blog [Doubling down on open-access quantum computing](https://www.ibm.com/quantum/blog/open-plan-updates), kde najdeš všechny podrobnosti.
</Admonition>

## Plán Flex {#flex-plan}

Předem si zakup určitý počet minut (minimálně 400) pro spouštění kvantových úloh, které musíš využít do jednoho roku od zakoupení. Další minuty si můžeš dokoupit kdykoli.

Po zakoupení času musíš minuty přidělit stávajícím nebo novým instancím s plánem Flex pro svůj účet. Limity přidělení nastavíš nebo aktualizuješ na [stránce Instances](https://quantum.cloud.ibm.com/instances) nebo je nastavíš při [vytváření nové instance plánu Flex](/guides/instances). Pokud přidělíš všechny minuty stávajícím instancím, nemůžeš pro plán Flex vytvořit novou instanci, dokud nedokoupíš další minuty nebo neodebereš část minut z přidělení jiné instance. Více informací najdeš v části [Nastavení limitů přidělení instance](allocation-limits).

Pokud potřebuješ více než 10 000 minut, zvaž raději předplatné plánu Premium. Pokud nechceš kupovat 400 minut, zvaž plán Pay-As-You-Go.

Pro přihlášení k plánu Flex [kontaktuj IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum).

## Plán Pay-As-You-Go {#pay-as-you-go-plan}

Spouštěj kvantové Circuit na nejlepších QPU světa a plať pouze za spotřebu QPU, kterou skutečně využiješ.

Tento plán je doporučen těm, kteří provádějí výzkumné projekty kvantové utility a testují obchodní modely s flexibilním přístupem. Pokud plánuješ v příštím roce využít alespoň 400 minut, doporučujeme místo toho použít plán Flex.

## Plán Premium {#premium-plan}

Spouštěj kvantové Circuit na nejlepších QPU světa s podnikovým předplatným. Navíc jsou uživatelům plánu Premium dostupné některé funkce, jako například [Qiskit Functions](/guides/functions).

Tento plán je doporučen těm, kteří chtějí realizovat strategický kvantový plán a vyvíjet kvantové algoritmy a aplikace ve velkém měřítku. [Kontaktuj IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) a zjisti více.

## Plán On-Prem {#on-prem-plan}

Získej přístup k plně dedikovanému kvantovému systému on-premises, který je spravován a udržován společností IBM Quantum.

Tato služba je doporučena těm, kteří prozkoumávají klasicko-kvantové algoritmy a aplikace a potřebují vysokou míru kontroly nad svými zdroji a daty. [Kontaktuj IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) a zjisti více.
