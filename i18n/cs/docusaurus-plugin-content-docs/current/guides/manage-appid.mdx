# Správa uživatelů poskytovatele identity {#manage-id-provider-users}

Svůj účet IBM Cloud můžeš integrovat s externím poskytovatelem identity (IDP), aby se externí uživatelé mohli bezpečně ověřovat. Pomocí externího poskytovatele umožníš uživatelům ve své firmě využívat jednotné přihlašování (SSO). Svůj cloudový účet s externím IDP propojíš jedním z následujících způsobů:

- Použij [poskytovatele služeb SAML IBM Cloud&reg; (SP)](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-cloud-saml)
- Použij [App ID](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-appid) (může být zpoplatněno)
- [Propoj své uživatele se všemi produkty IBM&reg; prostřednictvím IBMid](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-ibmid)

<Admonition type="tip">
Po propojení svého IDP můžeš nastavit dynamická pravidla, díky nimž nebudeš muset ručně přidávat každého uživatele do každé skupiny při jejich příchodu. Pokyny najdeš v článku [Povolení ověřování z externího poskytovatele identity](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#app-id-dynamic-rules).
</Admonition>
