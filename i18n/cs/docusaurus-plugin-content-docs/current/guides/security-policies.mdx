# Zásady zabezpečení repozitářů {#repository-security-policies}

Je nanejvýš důležité, aby sis udržoval své systémy co nejbezpečnější. Proto se důrazně doporučuje používat bezpečnostní nástroje jako [Dependabot](https://github.com/dependabot), které tě automaticky upozorní, když jsou závislosti zastaralé. Prohlédni si zásady zabezpečení následujících repozitářů na GitHubu, které se týkají IBM Quantum&reg;:

- [Qiskit](https://github.com/Qiskit/qiskit/security)
- [Qiskit IBM Runtime](https://github.com/Qiskit/qiskit-ibm-runtime/security)
- [Doplněk Qiskit pro dělení obvodů (Circuit cutting)](https://github.com/Qiskit/qiskit-addon-cutting/security)
- [Doplněk Qiskit pro fermionické mapování (Fermionic mapper)](https://github.com/Qiskit/qiskit-fermions/blob/main/SECURITY.md)
- [Doplněk Qiskit pro vzorce s více produkty (MPF)](https://github.com/Qiskit/qiskit-addon-mpf/security)
- [Doplněk Qiskit pro zpětné šíření operátorů (OBP)](https://github.com/Qiskit/qiskit-addon-obp/security)
- [Doplněk Qiskit pro optimalizační mapování (Optimization mapper)](https://github.com/Qiskit/qiskit-addon-opt-mapper/blob/main/SECURITY.md)
- [Doplněk Qiskit pro šíření Pauliho operátorů (Pauli propagation)](https://github.com/Qiskit/pauli-prop/blob/main/SECURITY.md)
- [Doplněk Qiskit pro absorbci šířeného šumu (PNA)](https://github.com/Qiskit/qiskit-addon-pna/blob/main/SECURITY.md)
- [Nástroje pro doplňky Qiskit](https://github.com/Qiskit/qiskit-addon-utils/security)
- [Doplněk Qiskit pro kvantovou diagonalizaci na základě vzorků (SQD)](https://github.com/Qiskit/qiskit-addon-sqd/security)
- [Doplněk Qiskit pro stínované světelné kužele (Shaded lightcones)](https://github.com/Qiskit/qiskit-addon-slc/blob/main/SECURITY.md)
