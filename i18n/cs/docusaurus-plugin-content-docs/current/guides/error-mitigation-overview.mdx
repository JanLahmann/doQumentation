# Přehled technik správy šumu {#overview-of-noise-management-techniques}

Při provádění kvantových pracovních zátěží existuje několik způsobů, jak snížit dopad šumu. Open-source doplňky Qiskit poskytují techniky zmírnění a potlačení chyb, které se přímo integrují do tvého vývojového pracovního postupu, zatímco Qiskit Runtime automaticky aplikuje pokročilé strategie zmírnění chyb při odesílání úloh ke spuštění. Tato stránka indexuje všechny dostupné nástroje a funkce napříč oběma možnostmi, aby ti pomohla vybrat správný přístup ke správě šumu při budování kvantových pracovních zátěží.
