# Nastavení účtu IBM Cloud {#set-up-your-ibm-cloud-account}

Pokud ti přišla e-mailem pozvánka k připojení k účtu, postupuj místo toho podle kroků v části [Nastavení účtu IBM Cloud – pozvaní uživatelé](/guides/cloud-setup-invited).

Na IBM&reg; kvantových procesorech (QPU) můžeš spouštět úlohy tak, že si vytvoříš účet na IBM Cloud&reg;. Budeš také muset [nainstalovat a nastavit Qiskit a Qiskit Runtime](/guides/install-qiskit#local).

Tvůj uživatelský účet je přidružen k jedné nebo více [instancím,](./instances) identifikovaným pomocí _Cloud Resource Names_ (CRN), které poskytují přístup ke službám IBM Quantum. Každému účtu je navíc přiřazen jedinečný API token (nazývaný také klíč), který umožňuje přístup k IBM Quantum z Qiskitu. Pokyny k výběru konkrétní instance najdeš v části [Připojení k instanci](./instances#connect-instance).

<span id="cloud"></span>
## Nastavení pro použití IBM Cloud {#set-up-to-use-ibm-cloud}

### Než začneš {#before-you-begin}

1. Pokud ještě nemáš účet, vytvoř si [účet IBM Cloud.](https://quantum.cloud.ibm.com/registration)
   <Admonition type="note">
   Můžeš být součástí více účtů IBM Cloud. K libovolnému ze svých účtů IBM Cloud se kdykoli dostaneš pomocí přepínače účtů v záhlaví rozhraní [IBM Quantum Platform](https://quantum.cloud.ibm.com).
   </Admonition>
1. Přihlas se na [IBM Quantum Platform](https://quantum.cloud.ibm.com) pomocí IBMid nebo účtu Google. Pokud žádný nemáš, systém tě provede jeho vytvořením. (Tyto přihlašovací údaje nejsou stejné jako přihlašovací údaje k tvému účtu IBM Cloud.)
1. Zkontroluj, zda jsou v přepínači účtů v záhlaví vybrány správný účet a region, jak je znázorněno na následujícím obrázku.

    Region určuje, kde jsou tvé úlohy spouštěny a kde jsou uchovávána data úloh. K oběma regionům se dostaneš pomocí stejného API klíče, ale vidíš a máš přístup pouze k instancím vytvořeným v regionu, do kterého jsi přihlášen/a.

    ![Je zobrazeno záhlaví IBM Quantum Platform. Přepínač účtů se nachází napravo od vyhledávacího pole. Přepínač regionů se nachází napravo od přepínače účtů.](/docs/images/guides/cloud-setup/IQP-Header.svg 'IBM Quantum Platform header')

### 1. Vytvoření instance {#1-create-an-instance}

Instanci můžeš vytvořit na stránce [Instances](https://quantum.cloud.ibm.com/instances) v IBM Quantum Platform. Podrobné pokyny najdeš v části [Vytvoření instance](/guides/instances#add-instance).

### 2. Volitelné: Uložení přihlašovacích údajů {#2-optional-save-your-access-credentials}

Pokud používáš důvěryhodné prostředí Python, například osobní laptop, doporučuje se použít metodu `save_account()` k [místnímu uložení přihlašovacích údajů.](/guides/save-credentials)
