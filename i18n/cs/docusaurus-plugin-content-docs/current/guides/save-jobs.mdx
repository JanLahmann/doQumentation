Kvantové pracovní postupy často trvají delší dobu a mohou probíhat v rámci mnoha session. Restartování Python kernelu znamená, že přijdeš o veškeré výsledky uložené v paměti. Abys předešel ztrátě dat, můžeš výsledky uložit do souboru a načíst výsledky dřívějších úloh z IBM Quantum&reg;, aby tvá příští session mohla navázat tam, kde jsi přestal.
## Načtení výsledků úloh z IBM Quantum {#retrieve-job-results-from-ibm-quantum}

IBM Quantum automaticky ukládá výsledky každé úlohy, abys je mohl načíst kdykoli později. Tuto funkci využij k pokračování kvantových programů po restartu kernelu a k prohlížení dřívějších výsledků. ID úlohy lze získat programově pomocí metody `job_id`, nebo si můžeš zobrazit všechny odeslané úlohy a jejich ID na [stránce Workloads](https://quantum.cloud.ibm.com/workloads).

Chceš-li úlohu vyhledat programově, použij metodu [`QiskitRuntimeService.jobs`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#jobs). Ve výchozím nastavení vrací naposledy odeslané úlohy, ale můžeš je také filtrovat podle názvu backendu, data vytvoření a dalších kritérií. Následující buňka vyhledá všechny úlohy odeslané za poslední tři měsíce. Argument `created_after` musí být objekt [`datetime.datetime`](https://docs.python.org/3.8/library/datetime.html#datetime.datetime).


```python
import datetime
from qiskit_ibm_runtime import QiskitRuntimeService

three_months_ago = datetime.datetime.now() - datetime.timedelta(days=90)

service = QiskitRuntimeService()
jobs_in_last_three_months = service.jobs(created_after=three_months_ago)
jobs_in_last_three_months[:3]  # show first three jobs
```

Můžeš také vybírat podle backendu, stavu úlohy, session a dalšího. Více informací najdeš v dokumentaci API u [`QiskitRuntimeService.jobs`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#jobs).

Jakmile máš ID úlohy, načti ji pomocí metody [`QiskitRuntimeService.job`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/qiskit-runtime-service#job).

```python
# Get ID of most recent successful job for demonstration.
# This will not work if you've never successfully run a job.
successful_job = next(
    j for j in service.jobs(limit=1000) if j.status() == "DONE"
)
job_id = successful_job.job_id()
print(job_id)
```

```python
retrieved_job = service.job(job_id)
```

## Uložení výsledků na disk {#save-results-to-disk}

Výsledky si možná budeš chtít uložit také na disk. K tomu použij vestavěnou knihovnu JSON v Pythonu spolu s enkodéry z Qiskit Runtime.

```python
import json
from qiskit_ibm_runtime import RuntimeEncoder

with open("result.json", "w") as file:
    json.dump(retrieved_job.result(), file, cls=RuntimeEncoder)
```

Toto pole pak můžeš načíst z disku v samostatném kernelu.

```python
from qiskit_ibm_runtime import RuntimeDecoder

with open("result.json", "r") as file:
    result = json.load(file, cls=RuntimeDecoder)

result
```
