# Instalace klienta Qiskit Runtime ze zdrojového kódu {#install-the-qiskit-runtime-client-from-source}

Když nainstaluješ Qiskit Runtime ze zdrojového kódu, získáš přístup k aktuální vývojové verzi místo verze v repozitáři Python Package Index (PyPI). Díky tomu můžeš efektivněji prozkoumat a rozšířit nejnovější verzi kódu.

## Vytvoření a aktivace nového virtuálního prostředí {#create-and-activate-a-new-virtual-environment}

1. Přejdi do adresáře svého projektu a vytvoř minimální prostředí pouze s nainstalovaným Pythonem.

## Instalace Qiskit Runtime {#install-qiskit-runtime}

Postupuj podle těchto kroků pro instalaci Qiskit Runtime:

1. Naklonuj repozitář Qiskit Runtime.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. Přejdi do adresáře `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. Nainstaluj `qiskit-runtime`. Doporučujeme použít [virtuální prostředí](https://docs.python.org/3/library/venv.html), aby nedošlo ke znečištění globální instalace Pythonu.

  * **Standardní instalace**:

    ```bash
    pip install .
    ```

  * **Upravitelný režim**: V tomto režimu nemusíš reinstalovat Qiskit při změnách kódu projektu.

    ```bash
    pip install -e .
    ```

    V upravitelném režimu jsou zkompilovaná rozšíření sestavena v _režimu ladění_ bez optimalizací.

4. (Volitelné) Chceš-li spouštět testy nebo kontroly lintingu, nainstaluj vývojářské požadavky. Doporučujeme použít [virtuální prostředí](https://docs.python.org/3/library/venv.html), aby nedošlo ke znečištění globální instalace Pythonu.

```bash
pip install -e ".[dev]"
```
