Chceš-li získat informace o stavu qubitu, můžeš ho _změřit_ na [klasický bit](https://docs.quantum.ibm.com/api/qiskit/circuit#qiskit.circuit.Clbit). V Qiskitu se měření provádějí ve výpočetní bázi, tj. v jednoqubitové Pauliho-$Z$ bázi. Měření tedy vrátí hodnotu 0 nebo 1 podle překryvu s vlastními stavy Pauliho-$Z$ operátoru $|0\rangle$ a $|1\rangle$:

$$
|q\rangle \xrightarrow{measure}\begin{cases}
      0 (\text{outcome}+1), \text{with probability } p_0=|\langle q|0\rangle|^{2}\text{,} \\
      1 (\text{outcome}-1), \text{with probability } p_1=|\langle q|1\rangle|^{2}\text{.}
    \end{cases}
$$

## Přidání měření do obvodu {#apply-a-measurement-to-a-circuit}

Existuje několik způsobů, jak přidat měření do obvodu:

### Metoda `QuantumCircuit.measure` {#quantumcircuitmeasure-method}

Metodu [`measure`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure) použij k měření [`QuantumCircuit`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#quantumcircuit-class).

Příklady:

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(5, 5)
qc.x(0)
qc.x(1)
qc.x(4)
qc.measure(
    range(5), range(5)
)  #  Measures all qubits into the corresponding clbit.
```

### Třída `Measure` {#measure-class}

Třída [Measure](https://docs.quantum.ibm.com/api/qiskit/circuit#qiskit.circuit.Measure) v Qiskitu změří zadané qubity.

```python
from qiskit.circuit import Measure

qc = QuantumCircuit(3, 1)
qc.x([0, 1])
qc.append(Measure(), [0], [0])  # measure qubit 0 into clbit 0
```

### Metoda `QuantumCircuit.measure_all` {#quantumcircuitmeasure_all-method}

Chceš-li změřit všechny qubity do příslušných klasických bitů, použij metodu [`measure_all`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure_all). Ve výchozím nastavení tato metoda přidá nové klasické bity v `ClassicalRegister` pro uložení těchto měření.

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(3, 1)
qc.x([0, 2])
qc.measure_all()  # Measure all qubits.
```

### Metoda `QuantumCircuit.measure_active` {#quantumcircuitmeasure_active-method}

Chceš-li změřit všechny qubity, které nejsou nečinné, použij metodu [`measure_active`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure_active). Tato metoda vytvoří nový `ClassicalRegister` o velikosti odpovídající počtu měřených aktivních qubitů.

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(3, 1)
qc.x([0, 2])
qc.measure_active()  # Measure qubits that are not idle, that is, qubits 0 and 2.
```
