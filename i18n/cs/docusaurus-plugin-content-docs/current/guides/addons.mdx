# Doplňky Qiskit {#qiskit-addons}

Doplňky Qiskit jsou kolekcí výzkumných možností, které umožňují objevování algoritmů v měřítku utility. Tyto možnosti staví na výkonném základu nástrojů Qiskit pro vytváření a spouštění kvantových algoritmů. Jsou poskytovány jako modulární softwarové komponenty, které lze zapojit do [pracovního postupu](/guides/intro-to-patterns) pro škálování nebo navrhování nových kvantových algoritmů.

## Doplňky pro mapování {#addons-for-mapping}

### Přibližná kvantová kompilace s tenzorovou sítí {#approximate-quantum-compilation-with-tensor-networks}

Přibližná kvantová kompilace s tenzorovou sítí (AQC-Tensor) umožňuje konstrukci obvodů s vysokou věrností a sníženou hloubkou.

- Navštiv [repozitář na GitHubu.](https://github.com/Qiskit/qiskit-addon-aqc-tensor)
- Přečti si [dokumentaci.](/guides/qiskit-addons-aqc)
- Přečti si tutoriál o použití AQC pro [zlepšení Trotterizované časové evoluce.](/tutorials/approximate-quantum-compilation-for-time-evolution)

### Multi-product formulas

Multi-product formulas (MPF) snižují Trotterovu chybu hamiltonovské dynamiky pomocí váženého součtu několika spuštění obvodu.

- Navštiv [repozitář na GitHubu.](https://github.com/Qiskit/qiskit-addon-mpf)
- Přečti si [dokumentaci.](https://qiskit.github.io/qiskit-addon-mpf)

### Optimalizační mapper {#optimization-mapper}

Doplněk Optimization Mapper obsahuje funkcionalitu pro modelování optimalizačních problémů tak, že je formuluje v abstraktních modelech a poté je převádí do reprezentací, kterým kvantový počítač rozumí.

- Přečti si [dokumentaci](https://qiskit.github.io/qiskit-addon-opt-mapper/) pro návody a API reference.
- Navštiv repozitář na [GitHubu.](https://github.com/qiskit/qiskit-addon-opt-mapper)

## Doplňky pro optimalizaci {#addons-for-optimizing}

### Zpětná propagace operátorů {#operator-backpropagation}

Zpětná propagace operátorů (OBP) snižuje hloubku obvodu odstraňováním operací z konce za cenu většího počtu měření operátorů.

- Navštiv [repozitář na GitHubu.](https://github.com/Qiskit/qiskit-addon-obp)
- Přečti si [dokumentaci.](/guides/qiskit-addons-obp)
- Přečti si tutoriál o použití [OBP pro zlepšení očekávaných hodnot.](/tutorials/operator-back-propagation)

### Řezání obvodu {#circuit-cutting}

Řezání obvodu (Circuit cutting) snižuje hloubku transpilovaných obvodů rozkladem propojovacích Gate mezi nesousedními Qubity.

- Navštiv [repozitář na GitHubu.](https://github.com/Qiskit/qiskit-addon-cutting)
- Přečti si [dokumentaci.](https://qiskit.github.io/qiskit-addon-cutting)

## Doplňky pro post-processing {#addons-for-post-processing}

### Diagonalizace kvanta na základě vzorků {#sample-based-quantum-diagonalization}

Diagonalizace kvanta na základě vzorků (SQD) klasicky post-processuje zašuměné kvantové vzorky, aby získala přesnější odhady vlastních hodnot hamiltonianů kvantových systémů, například v chemických aplikacích.

- Navštiv [repozitář na GitHubu.](https://github.com/Qiskit/qiskit-addon-sqd)
- Přečti si [dokumentaci.](/guides/qiskit-addons-sqd)
- Přečti si tutoriál o [diagonalizaci kvanta na základě vzorků pro chemický Hamiltonian.](/tutorials/sample-based-quantum-diagonalization)

### SQD pro HPC {#sqd-for-hpc}

Tento doplněk je implementací doplňku SQD připravenou pro HPC. Je napsán v moderním standardu C++17 a je navržen tak, aby vytvořil jediný zkompilovaný binární soubor pro použití s MPI.

- Navštiv dokumentaci [API reference.](https://qiskit.github.io/qiskit-addon-sqd-hpc/)
- Prohlédni si repozitář na [GitHubu.](https://github.com/qiskit/qiskit-addon-sqd-hpc)

## Doplňky pro mitigaci chyb {#addons-for-error-mitigation}

### M3 (mthree)

Matrix-free Measurement Mitigation (M3) je balíček pro škálovatelnou mitigaci chyb kvantových měření, která lze vypočítat paralelně.

- Navštiv [dokumentaci](https://qiskit.github.io/qiskit-addon-mthree/) pro návody, tutoriály a API reference.
- Navštiv [GitHub](https://github.com/Qiskit/qiskit-addon-mthree) stránku balíčku.

### Stínované světelné kužely {#shaded-lightcones}

Doplněk stínovaných světelných kuželů (shaded lightcones) používá Pauliho propagaci ke snížení počtu chybových členů v modelu šumu, které je třeba mitigovat, v závislosti na specifikách cílové observovatelné veličiny. To má za efekt snížení overhead vzorkování pro pracovní postupy pravděpodobnostního rušení chyb (PEC).

- Navštiv [dokumentaci](https://qiskit.github.io/qiskit-addon-slc/) pro návody, tutoriály a API reference.

### Absorpce propagovaného šumu {#propagated-noise-absorption}

Absorpce propagovaného šumu (PNA) používá Pauliho propagaci k absorpci informací z modelu šumu do cílové observovatelné veličiny. Měření této upravené observovatelné veličiny má za efekt mitigaci šumu reprezentovaného daným modelem.

- Navštiv [dokumentaci](https://qiskit.github.io/qiskit-addon-pna/apidocs/qiskit_addon_pna.html) pro návody, tutoriály a API reference.
