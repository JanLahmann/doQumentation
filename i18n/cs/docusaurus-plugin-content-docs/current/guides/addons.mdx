# Pokročilé techniky – doplňky Qiskit {#advanced-techniques-qiskit-addons}

Doplňky Qiskit jsou kolekcí výzkumných možností, které umožňují objevování algoritmů v měřítku utility. Tyto modulární softwarové komponenty staví na výkonném základu Qiskitu a lze je zapojit do [pracovního postupu](/guides/intro-to-patterns) pro škálování nebo navrhování nových kvantových algoritmů. Tato stránka přehledně uvádí dostupné nástroje napříč klíčovými funkčními kategoriemi, aby ti pomohla vybrat relevantní možnosti při sestavování pracovních postupů.
