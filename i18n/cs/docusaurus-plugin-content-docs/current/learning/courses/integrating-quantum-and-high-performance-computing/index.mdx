# Přehled {#overview}

!["Hero image"](/learning/images/courses/integrating-quantum-and-high-performance-computing/index/hero.svg)

Tento kurz poskytuje přehled klasických a kvantových zdrojů a také hybridních pracovních postupů s příklady.

## Doporučené předchozí znalosti {#recommended-background}

Tento kurz je určen pro dvě odlišné skupiny lidí: ty, kteří mají zkušenosti s kvantovými výpočty, ale ne s HPC, a ty, kteří mají zkušenosti s HPC, ale ne s kvantovými výpočty. Tyto technologie se vzájemně doplňují a ti, kteří se vyznají v jedné z nich, si stále více uvědomují důležitost té druhé. Studenti mohou podle potřeby přeskočit části ze svého vlastního oboru.

Doporučujeme, aby studenti před tímto kurzem splnili následující předpoklad:

- Nastav si své lokální prostředí podle průvodce [Začínáme](/guides/) na naší stránce s dokumentací.

Pro hlubší pochopení kvantových informací absolvuj tento volitelný kurz:
- [Základy kvantových informací](/learning/courses/basics-of-quantum-information) s Johnem Watrousem.
