# Přehled {#overview}

!["Hero image"](/learning/images/courses/quantum-diagonalization-algorithms/hero.avif)

Tento kurz poskytuje přehled několika přístupů ke kvantové diagonalizaci. Jsou zde uvedeny základní informace o klasických metodách, které se používají nebo které motivovaly kvantové algoritmy. Obsah pak provází implementací kvantových algoritmů na reálných kvantových počítačích. Podrobně se rozebírají faktory, které určují škálování přístupů používajících klasické a kvantové algoritmy.

## Doporučené předpoklady {#recommended-background}

Tento kurz je určen pro jednotlivce, kteří mají určité zkušenosti s kvantovými výpočty a chtějí využít jejich sílu při diagonalizaci matic.

Doporučujeme, aby studenti před tímto kurzem absolvovali následující předpoklady:

- Nastav si místní prostředí podle průvodce [Začínáme](/guides/) v naší dokumentaci.
- Pro hlubší pochopení kvantových informací vyzkoušej [Základy kvantových informací](/learning/courses/basics-of-quantum-information) s Johnem Watrousem.
