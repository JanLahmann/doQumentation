# Přehled {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Tento kurz je určen pro ty, kteří mají určité zkušenosti s kvantovým počítáním a chtějí se naučit psát variační algoritmy pomocí primitiv Qiskit Runtime. Je také vhodný pro profesionály v oblasti kvantového počítání, kteří chtějí rozšířit své znalosti a dovednosti v oblasti variačních algoritmů.
