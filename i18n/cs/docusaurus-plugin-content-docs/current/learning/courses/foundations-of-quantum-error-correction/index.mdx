# Přehled {#overview}

!["Hero image"](/learning/images/courses/foundations-of-quantum-error-correction/hero.svg)

Vítej v kurzu *Základy kvantové korekce chyb*, čtvrtém kurzu v sérii *Porozumění kvantovým informacím a výpočtům* sestávající z následujících kurzů:

- [Základy kvantových informací](/learning/courses/basics-of-quantum-information)
- [Základy kvantových algoritmů](/learning/courses/fundamentals-of-quantum-algorithms)
- [Obecná formulace kvantových informací](/learning/courses/general-formulation-of-quantum-information)
- Základy kvantové korekce chyb (tento kurz)

Tento kurz se věnuje kvantové korekci chyb se zaměřením na základní koncepty.
Začíná vysvětlením toho, co jsou kvantové kódy pro korekci chyb a jak fungují.
Poté přechází k fenoménu diskretizace chyb, stabilizátorovému formalismu pro
popis kvantových kódů pro korekci chyb, Calderbank–Shor–Steaneovým (CSS) kódům a několika
klíčovým příkladům kvantových kódů pro korekci chyb, včetně torického kódu a povrchových kódů.
Kurz také pojednává o kvantovém výpočtu odolném vůči chybám, při kterém se kvantové výpočty
provádějí na kvantových informacích s korekcí chyb.

Tento kurz je určen studentům, profesionálům a nadšencům v oborech jako informatika, fyzika,
inženýrství a matematika, kteří se chtějí dozvědět více o teoretických základech kvantových informací
a výpočtů.
