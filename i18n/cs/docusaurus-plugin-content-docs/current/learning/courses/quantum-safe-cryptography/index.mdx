# Přehled {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

Praktický úvod do kvantově bezpečné kryptografie je bezplatný online kurz v IBM Quantum&reg; Learning a slouží jako primer k základním konceptům kvantově bezpečné kryptografie.

Tento kurz byl vytvořen pro vývojáře, kteří mají zájem o modernizaci zabezpečení svých aplikací. Obsahuje multimodální lekce a interaktivní živé příklady kódu ve čtyřech klíčových oblastech kryptografie: kryptografické hašovací funkce, symetrická kryptografie, asymetrická kryptografie a kvantově bezpečná kryptografie.

Absolvováním tohoto kurzu se vývojáři dozvědí, jak se vyvíjí prostředí kybernetických rizik, a prozkoumají současné přístupy, které mají formovat kvantovou éru.
