# Přehled {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Tento kurz zkoumá, jak může kvantové počítání hrát vzrušující roli ve strojovém učení. Kurz nabídne stručné opakování klasických přístupů ke strojovému učení, ale předpokládá také určitou znalost těchto metod. Hlavní důraz je kladen na to, jak může kvantové počítání zapadnout do stávajících pracovních postupů strojového učení. Témata zahrnují kódování dat, kvantová jádra a variační kvantové Circuit.
