Nakita na natin ang ilang halimbawa ng quantum error correcting codes sa mga nakaraang aralin ng kursong ito, kabilang ang 9-qubit Shor code, ang 7-qubit Steane code, at ang 5-qubit code.
Walang alinlangan na kawili-wili ang mga code na ito at natural silang panimulang punto sa pag-explore ng quantum error correction, ngunit ang problema sa kanila ay napakababa lang ng error rate na kaya nilang tiisin.
Hindi naman masama ang mag-correct ng error sa isang qubit sa lima, pito, o siyam, pero malaki ang posibilidad na kailangan nating kayanin ang mas maraming error pa kayun para maging katotohanan ang large-scale quantum computing.

Sa araling ito, unang titingnan natin ang ilang mas sopistikadong konstruksyon ng quantum error correcting code, kabilang ang mga code na kaya nilang tiisin ang mas mataas na error rate kaysa sa mga nakita natin noon, at mga itinuturing na promising na kandidato para sa praktikal na quantum error correction.

Magsisimula tayo sa isang klase ng quantum error correcting codes na kilala bilang *CSS codes*, na pinangalan kay Robert Calderbank, Peter Shor, at Andrew Steane, ang mga unang natuklasan nito.
Pinapayagan ng CSS code construction ang isa na kumuha ng ilang tiyak na *pares* ng classical error correcting codes at pagsasamahin ang mga ito sa iisang quantum error correcting code.

Ang pangalawang bahagi ng aralin ay tungkol sa isang code na kilala bilang *toric code*.
Ito ay isang pangunahin (at tunay na magandang) halimbawa ng isang quantum error correcting code na kayang tiisin ang medyo mataas na error rate.
Sa katunayan, ang toric code ay hindi isang solong halimbawa ng quantum error correcting code, kundi isang walang katapusang pamilya ng mga code, isa para sa bawat positibong integer na mas malaki sa isa.

Sa huli, sa huling bahagi ng aralin, maikling tatalakayin natin ang ilang iba pang pamilya ng quantum codes, kabilang ang *surface codes* (na malapit na konektado sa toric code) at *color codes*.
