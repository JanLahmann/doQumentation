# Pangkalahatang-tanaw {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Tinutuklas ng kursong ito kung paano maaaring gumanap ng kapana-panabik na papel ang quantum computing sa machine learning. Mag-aalok ang kurso ng ilang pagbabalik-aral sa mga klasikal na diskarte sa machine learning, ngunit inaasahan ding may kaunting kaalaman ka na sa mga pamamaraang ito. Ang pangunahing pokus ay kung paano maaaring isama ang quantum computing sa mga kasalukuyang workflow ng machine learning. Kasama sa mga paksa ang pag-encode ng data, quantum kernels, at variational quantum circuits.

## Pagsusulit {#exam}

Para makuha ang iyong badge para sa Quantum machine learning, kumuha ng pagsusulit sa IBM&reg; Training. Ang pagsusulit na ito ay inilaan para gawin pagkatapos basahin ang mga aralin sa kursong ito. Pagkatapos mong pumasa sa pagsusulit, aabisuhan ka ng Credly na nakakuha ka ng badge.
