# Pangkalahatang-tanaw {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Ang kursong ito ay para sa mga taong may kaunting karanasan sa quantum computing at gustong matuto kung paano magsulat ng mga variational algorithm gamit ang Qiskit Runtime primitives. Angkop din ito para sa mga propesyonal sa quantum computing na gustong palawakin ang kanilang kaalaman at kasanayan sa larangan ng mga variational algorithm.

## Pagsusulit {#exam}

Para makuha ang iyong badge para sa Variational algorithm design, kumuha ng pagsusulit sa IBM&reg; Training. Ang pagsusulit na ito ay inilaan para gawin pagkatapos basahin ang mga aralin sa kursong ito. Pagkatapos mong maipasa ang pagsusulit, aabisuhan ka ng Credly na nakakuha ka na ng badge.
