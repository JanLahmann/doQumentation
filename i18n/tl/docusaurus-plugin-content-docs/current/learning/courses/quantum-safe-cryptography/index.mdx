# Pangkalahatang-tanaw {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

Ang "A practical introduction to quantum-safe cryptography" ay isang libreng online na kurso sa IBM Quantum&reg; Learning na nagsisilbing panimula sa mga pundamental na konsepto ng quantum-safe cryptography.

Ang kursong ito ay ginawa para sa mga developer na interesado sa pag-modernize ng kanilang application security. Nagtatampok ito ng multimodal na mga aralin at interactive na live code examples sa apat na pangunahing larangan ng cryptography: cryptographic hash functions, symmetric key cryptography, asymmetric key cryptography, at quantum-safe cryptography.

Sa pamamagitan ng kursong ito, matututunan ng mga developer kung paano nagbabago ang landscape ng cybersecurity risk, at maaari ring tuklasin ang mga makabagong pamamaraan na huhubog sa quantum era.
