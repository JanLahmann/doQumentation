# Ano ang susunod? {#whats-next}

Ngayong natutunan mo na ang mga pangunahing kaalaman ng quantum-safe cryptography, tuklasin ang mga teknolohiyang inaalok ng IBM Quantum&reg; para i-modernize ang iyong cryptographic landscape at buuin ang quantum cyber resilience. Ang [IBM Quantum Safe&trade;](https://www.ibm.com/quantum/quantum-safe) ay isang komprehensibong hanay ng mga tool, kakayahan, at diskarte na sinasamahan ng malalim na kaalaman para tulungan kang planuhin at isagawa ang paglipat ng iyong organisasyon sa quantum-safe cryptography. Tuklasin, obserbahan, at baguhin ang iyong cryptography gamit ang tatlong pangunahing teknikal na kakayahan:

**1. IBM Quantum Safe Explorer**

Pinasimple ng IBM Quantum Safe Explorer ang pagtuklas at pamamahala ng cryptography sa pamamagitan ng pag-scan ng source code at object code para matukoy ang lahat ng cryptographically relevant na artifact, matuklasan ang mga dependency, at maipakita ang mga kahinaan. Kinakatawan ng Explorer ang data na ito sa isang [Cryptography Bill of Materials](https://github.com/IBM/CBOM) (CBOM), isang karaniwang extension ng Software Bill of Materials (SBOM) na nagpapatakbo ng paglikha at pagbabahagi ng metadata na naglalarawan ng mga cryptographic artifact.

**2. IBM Quantum Safe Advisor**

Pinapalaki ng IBM Quantum Safe Advisor ang static code analysis mula sa IBM Quantum Safe Explorer gamit ang dynamic, runtime na pananaw ng iyong cryptographic posture at compliance. Iniuugnay ng Advisor ang mga [CBOM][CBOM] at pinagsasama ang data mula sa mga network at endpoint scan para makabuo ng komprehensibong cryptographic inventory. Inilalarawan ng inventory na ito ang:

- Ang mga uri at lokasyon ng iyong mga cryptographic instance
- Mga kaugnay na sertipiko, cipher suite, at CBOM
- Ang mga relasyon sa pagitan ng mga asset at data flow
- Mga potensyal na kahinaan sa quantum technology

Sa tulong ng policy-based enrichment nito, inihahanda ka ng Advisor na suriin ang iyong cryptography at data compliance at makabuo ng prioritized na listahan ng mga kahinaan para makapagtatag ka ng quantum-safe remediation strategy.

**3. IBM Quantum Safe Remediator**

Binibigyan ka ng IBM Quantum Safe Remediator ng kakayahang ilapat ang mga pinakamahusay na gawi para sa mga quantum-safe remediation pattern na nakatuon sa VPN communication, TLS communication, at cluster access communication. Bumuo ng crypto-agility sa pamamagitan ng pag-explore ng mga solusyon para baguhin ang iyong cryptography nang hindi nagagambala ang mga operasyon ng negosyo.

Basahin ang [technology guide](https://www.ibm.com/quantum/assets/quantum-safe/IBM_Quantum-Safe-Technology-Brochure.pdf), at matuto pa sa [IBM Quantum Safe site.](https://www.ibm.com/quantum/quantum-safe)

[CBOM]: #definition-tooltip "A schema extension of the CycloneDX Software Bill of Materials (SBOM) standard that provides an object model for describing cryptographic artifacts and their dependencies."
