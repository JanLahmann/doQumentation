# Qiskit Addons

Ang [Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) ay mga opisyal na extension package na nagbibigay ng mga espesyal na quantum computing algorithm at teknik. Bawat addon ay nakatuon sa isang partikular na area at may kasamang mga tutorial na may executable code.

## Mga Available na Addon {#available-addons}

| Addon | Package | Paglalarawan |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Bawasan ang circuit width at depth sa pamamagitan ng pagputol ng mga gate at wire |
| **SQD** | `qiskit-addon-sqd` | Sample-based Quantum Diagonalization para sa pagtataya ng ground state |
| **OBP** | `qiskit-addon-obp` | Operator Back-Propagation para bawasan ang circuit depth |
| **MPF** | `qiskit-addon-mpf` | Multi-Product Formulas para sa pinahusay na Hamiltonian simulation |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximate Quantum Compilation gamit ang tensor network |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption para sa error mitigation |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones para sa pagbawas ng PEC sampling overhead |

Lahat ng tutorial ay maaaring direktang isagawa sa iyong browser gamit ang Binder, o sa sarili mong Jupyter server.

Para sa API documentation, bisitahin ang mga indibidwal na addon repo sa [GitHub](https://github.com/Qiskit?q=qiskit-addon).
