# I-install ang Qiskit Runtime client mula sa pinagmulan {#install-the-qiskit-runtime-client-from-source}

Kapag nag-install ka ng Qiskit Runtime mula sa pinagmulan, maaari kang mag-access ng kasalukuyang development version, sa halip na gamitin ang bersyon sa Python Package Index (PyPI) repository. Maaari kang mag-inspect at mag-extend ng pinakabagong bersyon ng code nang mas mahusay.

## Lumikha at i-activate ang bagong virtual environment {#create-and-activate-a-new-virtual-environment}

1. Pumunta sa iyong project directory at lumikha ng minimal na kapaligiran na may Python lamang na naka-install dito.

## I-install ang Qiskit Runtime {#install-qiskit-runtime}

Sundin ang mga hakbang na ito para i-install ang Qiskit Runtime:

1. I-clone ang Qiskit Runtime repository.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. Pumunta sa direktoryo ng `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. I-install ang `qiskit-runtime`. Inirerekumenda naming gumamit ng [virtual environment](https://docs.python.org/3/library/venv.html) para maiwasan ang polusyon ng iyong global na Python installation.

  * **Karaniwang install**:

    ```bash
    pip install .
    ```

  * **Editable mode**: Sa mode na ito, hindi mo kailangang muling i-install ang Qiskit kapag may mga pagbabago sa code sa proyekto.

    ```bash
    pip install -e .
    ```

    Sa editable mode, ang mga compiled extension ay binubuo sa _debug mode_ nang walang mga optimization.

4. (Opsyonal) Kung gusto mong magpatakbo ng mga test o linting check, i-install ang mga developer requirement. Inirerekumenda naming gumamit ng [virtual environment](https://docs.python.org/3/library/venv.html) para maiwasan ang polusyon ng iyong global na Python installation.

```bash
pip install -e ".[dev]"
```
