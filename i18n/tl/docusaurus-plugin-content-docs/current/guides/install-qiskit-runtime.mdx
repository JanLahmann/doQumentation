# I-install ang Qiskit Runtime client {#install-the-qiskit-runtime-client}

Ang Qiskit Runtime ay ang cloud-based na serbisyo para sa pagpapatupad ng mga quantum computation sa IBM Quantum&reg; hardware. Ipinapakita ng gabay na ito kung paano i-install ang `qiskit-ibm-runtime`, na siyang client package para sa pag-access sa serbisyo ng Qiskit Runtime.

Bago i-install ang Qiskit Runtime client, sundin ang mga tagubilin sa mga gabay na ito:

1. [I-install ang Qiskit](/guides/install-qiskit).
2. [I-set up ang iyong IBM Cloud account](cloud-setup).

Patakbuhin ang command na ito para i-install ang Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Manatiling updated sa pinakabagong mga bersyon">
Pana-panahong tingnan ang [Qiskit Runtime release notes](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) para makita ang mga bagong release. Inirerekumenda naming madalas na i-update ang iyong mga kinakailangan para sa `qiskit-ibm-runtime` sa pamamagitan ng pagpapatakbo ng `pip install -U qiskit-ibm-runtime`.
</Admonition>
