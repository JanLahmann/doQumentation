# Tingnan ang gastos {#view-the-cost}

Anumang oras, makikita mo kung magkano na ang nagastos ng mga workload na nauugnay sa isang instance.

## Tingnan ang gastos ng instance {#view-instance-cost}

Para malaman kung gaano na kagamit ang isang instance, mula sa [pahina ng Instances](https://quantum.cloud.ibm.com/instances), i-click ang instance para buksan ang detalye nito. Maaari mong i-click ang link na nauugnay sa **Billing overview** para sa buong detalye ng gastos at billing.

Nag-uulat ang **Instance usage** ng iba't ibang halaga, depende sa uri ng plano na nauugnay sa instance:

- Para sa Premium at Open na plano, ito ang real-time na paggamit sa nakalipas na 28 araw (rolling), hanggang sa kasalukuyang oras sa kasalukuyang araw.
- Para sa Pay-As-You-Go na plano, ito ang lahat ng paggamit mula nang likhain ang instance.
- Para sa Flex na plano, ito ang paggamit mula sa simula ng kontrata.

## Tingnan ang paggamit ng workload {#view-workload-usage}

Para malaman kung gaano karaming paggamit ang nagastos ng isang partikular na workload, buksan ang [pahina ng Workloads](https://quantum.cloud.ibm.com/workloads) at tingnan ang kolum na "Usage" para sa workload.
