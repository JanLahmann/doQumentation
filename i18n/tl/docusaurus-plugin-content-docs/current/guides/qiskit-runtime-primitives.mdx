# Panimula sa mga primitive {#introduction-to-primitives}

Ang mga primitive ay nilikha para gawing mas simple ang mga pinakakaraniwang gawain para sa mga quantum computer: ang pag-sample ng mga quantum state at pagkalkula ng mga expectation value. Ang unang Qiskit Runtime primitive ([`EstimatorV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2) at [`SamplerV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2)) ay mga implementasyon ng Qiskit [primitives base classes](/guides/primitives). Nagbibigay sila ng mas sopistikadong implementasyon (halimbawa, sa pamamagitan ng pagsasama ng error mitigation) bilang isang cloud-based na serbisyo at ginagamit para ma-access ang IBM Quantum&reg; hardware.

Ang pinakabagong Qiskit Runtime primitive, [Executor](https://qiskit.github.io/qiskit-ibm-runtime/stubs/qiskit_ibm_runtime.Executor.html) (na nasa beta pa), ay nagbibigay ng mas mababang antas ng interface na nagbibigay ng mas maraming visibility at kontrol nang hindi sinisasakripisyo ang performance.
