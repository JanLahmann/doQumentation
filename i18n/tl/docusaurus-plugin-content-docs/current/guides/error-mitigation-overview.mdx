# Pangkalahatang-ideya ng mga teknik sa noise management {#overview-of-noise-management-techniques}

Kapag nagpapatupad ng mga quantum workload, may maraming paraan para bawasan ang epekto ng noise. Ang mga open-source na Qiskit addon ay nagbibigay ng mga teknik ng error mitigation at suppression na direktang nag-iintegrate sa iyong development workflow, habang ang Qiskit Runtime ay awtomatikong nag-aaplay ng mga advanced na estratehiya ng error mitigation kapag nagsusumite ng mga job para sa pagpapatupad. Ang pahinang ito ay nagtatala ng lahat ng available na tool at feature sa parehong opsyon para tulungan kang pumili ng tamang diskarte para pamahalaan ang noise kapag nagtatayo ng mga quantum workload.
