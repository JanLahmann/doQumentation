# Gabay sa migration ng Qiskit 1.0 {#qiskit-10-migration-guide}

Ang Qiskit 1.0 ay nagdadala ng mga bagong garantiya sa katatagan at mga pagpapabuti sa performance, ngunit hindi ito compatible sa lahat ng nakaraang bersyon ng Qiskit. Tingnan ang mga sumusunod na gabay para ma-update ang iyong environment at code para gumana sa Qiskit v1.0.

- [I-install ang bagong Qiskit v1.0 package](/guides/qiskit-1.0-installation) para sa pag-upgrade ng mga Python environment at project dependencies.
- [Mga pagbabago sa features ng Qiskit v1.0](/guides/qiskit-1.0-features) para sa pag-update ng iyong code.
- [Mga breaking package changes ng Qiskit v1.0](/guides/metapackage-migration) para maunawaan ang bago at lumang istraktura ng package.

Basahin ang higit pa tungkol sa mga benepisyo ng Qiskit v1.0 sa [IBM&reg; blog](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
