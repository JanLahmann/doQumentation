# Advanced techniques - Mga Qiskit addon {#advanced-techniques---qiskit-addons}

Ang mga Qiskit addon ay koleksyon ng mga research capabilities para sa pagtuklas ng mga algorithm sa utility scale. Ang mga modular na software component na ito ay nagtitiwala sa matibay na pundasyon ng Qiskit at maaaring isaksak sa isang [workflow](/guides/intro-to-patterns) para mapalawak o magdisenyo ng mga bagong quantum algorithm. Itinatampok ng page na ito ang mga available na tools sa mga pangunahing functional na kategorya para matulungan kang pumili ng mga kaugnay na kakayahan habang binubuo ang iyong mga workflow.
