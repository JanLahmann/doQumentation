# Przegląd {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

Praktyczne wprowadzenie do kryptografii kwantowo-bezpiecznej to bezpłatny kurs online dostępny w IBM Quantum&reg; Learning, stanowiący wprowadzenie do podstawowych pojęć z zakresu kryptografii kwantowo-bezpiecznej.

Kurs został stworzony dla programistów zainteresowanych modernizacją bezpieczeństwa swoich aplikacji i zawiera wielomodalne lekcje oraz interaktywne przykłady kodu live w czterech kluczowych obszarach kryptografii: kryptograficzne funkcje skrótu, kryptografia symetryczna, kryptografia asymetryczna i kryptografia kwantowo-bezpieczna.

Biorąc udział w tym kursie, programiści dowiedzą się, jak ewoluuje krajobraz zagrożeń w cyberbezpieczeństwie, oraz poznają współczesne podejścia gotowe kształtować erę kwantową.
