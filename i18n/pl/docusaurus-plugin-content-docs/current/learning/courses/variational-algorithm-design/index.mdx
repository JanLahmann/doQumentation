# Przegląd {#overview}

!["Obraz główny"](/learning/images/courses/variational-algorithm-design/hero.svg)

Ten kurs jest przeznaczony dla osób, które mają pewne doświadczenie z obliczeniami kwantowymi i chcą nauczyć się pisać algorytmy wariacyjne przy użyciu prymitywów Qiskit Runtime. Jest również odpowiedni dla specjalistów od obliczeń kwantowych, którzy chcą poszerzyć swoją wiedzę i umiejętności w dziedzinie algorytmów wariacyjnych.
