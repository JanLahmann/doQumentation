# Przegląd {#overview}

!["Obraz główny"](/learning/images/courses/quantum-machine-learning/hero.svg)

Ten kurs bada, jaką ekscytującą rolę obliczenia kwantowe mogą odegrać w uczeniu maszynowym. Kurs zawiera pewne omówienie klasycznych podejść do uczenia maszynowego, ale zakłada również pewną znajomość tych metod. Główny nacisk kładziony jest na to, jak obliczenia kwantowe mogą wpisać się w istniejące procesy uczenia maszynowego. Tematy obejmują kodowanie danych, jądra kwantowe i wariacyjne Circuit kwantowe.
