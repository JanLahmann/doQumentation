# Przegląd {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

Witaj w kursie *Użyj komputera kwantowego już dziś*! W tym kursie od razu zaczniesz używać komputera kwantowego i po drodze poznasz podstawowe zasady obliczeń kwantowych.

Ten kurs jest stworzony dla ciekawskich i zapracowanych. Nie potrzebujesz dyplomu z fizyki ani lat doświadczenia w programowaniu. Jeśli potrafisz śledzić notebook i kliknąć „Run", jesteś gotowy.
