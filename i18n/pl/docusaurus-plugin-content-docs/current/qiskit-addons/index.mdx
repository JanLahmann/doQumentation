# Qiskit Addons {#qiskit-addons}

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) to oficjalne pakiety rozszerzeń oferujące wyspecjalizowane algorytmy i techniki obliczeń kwantowych. Każdy addon skupia się na konkretnym obszarze i zawiera samouczki z wykonywalnym kodem.

## Dostępne Addons {#available-addons}

| Addon | Pakiet | Opis |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Zmniejsz szerokość i głębokość Circuit przez cięcie Gate i połączeń |
| **SQD** | `qiskit-addon-sqd` | Sample-based Quantum Diagonalization do szacowania stanu podstawowego |
| **OBP** | `qiskit-addon-obp` | Operator Back-Propagation w celu redukcji głębokości Circuit |
| **MPF** | `qiskit-addon-mpf` | Multi-Product Formulas dla ulepszonej symulacji Hamiltonian |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximate Quantum Compilation z sieciami tensorowymi |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption do łagodzenia błędów |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones do redukcji narzutu próbkowania PEC |

Wszystkie samouczki możesz uruchomić bezpośrednio w przeglądarce za pomocą Binder lub na własnym serwerze Jupyter.

Dokumentację API znajdziesz w poszczególnych repozytoriach addons na [GitHubie](https://github.com/Qiskit?q=qiskit-addon).
