Aby uzyskać informacje o stanie Qubitu, możesz go _zmierzyć_ na [klasyczny bit](https://docs.quantum.ibm.com/api/qiskit/circuit#qiskit.circuit.Clbit). W Qiskit pomiary wykonywane są w bazie obliczeniowej, czyli jednoQubitowej bazie Pauliego-$Z$. W związku z tym pomiar daje wynik 0 lub 1, zależnie od pokrycia z stanami własnymi Pauliego-$Z$, $|0\rangle$ i $|1\rangle$:

$$
|q\rangle \xrightarrow{measure}\begin{cases}
      0 (\text{outcome}+1), \text{with probability } p_0=|\langle q|0\rangle|^{2}\text{,} \\
      1 (\text{outcome}-1), \text{with probability } p_1=|\langle q|1\rangle|^{2}\text{.}
    \end{cases}
$$

## Zastosowanie pomiaru do Circuit {#apply-a-measurement-to-a-circuit}

Istnieje kilka sposobów na zastosowanie pomiarów do Circuit:

### Metoda `QuantumCircuit.measure` {#quantumcircuitmeasure-method}

Użyj metody [`measure`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure), aby zmierzyć [`QuantumCircuit`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#quantumcircuit-class).

Przykłady:

```python
# Added by doQumentation — required packages for this notebook

```

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(5, 5)
qc.x(0)
qc.x(1)
qc.x(4)
qc.measure(
    range(5), range(5)
)  #  Measures all qubits into the corresponding clbit.
```

```python
qc = QuantumCircuit(3, 1)
```

### Klasa `Measure` {#measure-class}

Klasa Qiskit [Measure](https://docs.quantum.ibm.com/api/qiskit/circuit#qiskit.circuit.Measure) mierzy wskazane Qubity.

```python
from qiskit.circuit import Measure

qc = QuantumCircuit(3, 1)
qc.x([0, 1])
qc.append(Measure(), [0], [0])  # measure qubit 0 into clbit 0
```


### Metoda `QuantumCircuit.measure_all` {#quantumcircuitmeasure_all-method}

Aby zmierzyć wszystkie Qubity na odpowiadające im klasyczne bity, użyj metody [`measure_all`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure_all). Domyślnie ta metoda dodaje nowe klasyczne bity w `ClassicalRegister`, które przechowują te pomiary.

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(3, 1)
qc.x([0, 2])
qc.measure_all()  # Measure all qubits.
```

### Metoda `QuantumCircuit.measure_active` {#quantumcircuitmeasure_active-method}

Aby zmierzyć wszystkie Qubity, które nie są bezczynne, użyj metody [`measure_active`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure_active). Ta metoda tworzy nowy `ClassicalRegister` o rozmiarze równym liczbie mierzonych niebezczynnych Qubitów.

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(3, 1)
qc.x([0, 2])
qc.measure_active()  # Measure qubits that are not idle, that is, qubits 0 and 2.
```
