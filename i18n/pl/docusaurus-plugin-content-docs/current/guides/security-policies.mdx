# Polityki bezpieczeństwa repozytoriów {#repository-security-policies}

Niezwykle ważne jest, abyś utrzymywał swoje systemy jak najbezpieczniejszymi. Dlatego zdecydowanie zalecamy korzystanie z narzędzi bezpieczeństwa, takich jak [Dependabot](https://github.com/dependabot), które automatycznie powiadamiają cię, gdy zależności są nieaktualne. Zapoznaj się z politykami bezpieczeństwa następujących repozytoriów GitHub powiązanych z IBM Quantum&reg;:

- [Qiskit](https://github.com/Qiskit/qiskit/security)
- [Qiskit IBM Runtime](https://github.com/Qiskit/qiskit-ibm-runtime/security)
- [Dodatek Qiskit do cięcia obwodów (Circuit cutting)](https://github.com/Qiskit/qiskit-addon-cutting/security)
- [Dodatek Qiskit do mapowania fermionowego (Fermionic mapper)](https://github.com/Qiskit/qiskit-fermions/blob/main/SECURITY.md)
- [Dodatek Qiskit do formuł wieloproduktowych (MPF)](https://github.com/Qiskit/qiskit-addon-mpf/security)
- [Dodatek Qiskit do wstecznej propagacji operatorów (OBP)](https://github.com/Qiskit/qiskit-addon-obp/security)
- [Dodatek Qiskit do mapowania optymalizacji (Optimization mapper)](https://github.com/Qiskit/qiskit-addon-opt-mapper/blob/main/SECURITY.md)
- [Dodatek Qiskit do propagacji Pauliego (Pauli propagation)](https://github.com/Qiskit/pauli-prop/blob/main/SECURITY.md)
- [Dodatek Qiskit do propagowanej absorpcji szumu (PNA)](https://github.com/Qiskit/qiskit-addon-pna/blob/main/SECURITY.md)
- [Narzędzia dodatkowe Qiskit (Qiskit addon utilities)](https://github.com/Qiskit/qiskit-addon-utils/security)
- [Dodatek Qiskit do diagonalizacji kwantowej opartej na próbkach (SQD)](https://github.com/Qiskit/qiskit-addon-sqd/security)
- [Dodatek Qiskit do zacienionych stożków świetlnych (Shaded lightcones)](https://github.com/Qiskit/qiskit-addon-slc/blob/main/SECURITY.md)
