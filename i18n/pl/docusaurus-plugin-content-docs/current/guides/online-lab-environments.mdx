# Środowiska laboratoryjne online {#online-lab-environments}

Oprócz pobierania i instalowania Qiskit na lokalnym komputerze, dostępnych jest kilka opcji dla tych, którzy wolą pracować w środowisku deweloperskim online.

Kilku dostawców oferuje środowiska Jupyter online z preinstalowanym Qiskit, zapewniając szybkie i wygodne wdrożenie. Do polecanych dostawców należą: dostawca usług chmurowych OVHcloud oraz platforma chmurowa do tworzenia oprogramowania kwantowego qBraid.

## Polecane środowiska notebooków z preinstalowanym Qiskit {#recommended-notebook-environments-with-qiskit-preinstalled}

Dodatkowe środowiska Jupyter notebook online, takie jak IBM Watson&reg; Studio, Google Colab i Microsoft Azure Machine Learning Studio, mogą być używane do pracy z Qiskit po wykonaniu odpowiednich instrukcji.

### Notebooki AI OVHcloud {#ovhcloud-ai-notebooks}

Użytkownicy OVHcloud Public Cloud i OVHcloud Startup mogą korzystać ze swoich kwantowych notebooków OVHcloud, aby uzyskać dostęp do Qiskit i przesyłać zadania do QPU IBM&reg; za pomocą swojego konta IBM Quantum&reg; lub IBM Cloud&reg;. Aby dowiedzieć się więcej, odwiedź stronę [Dokumentacja i samouczki OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Notebooki AI OVHcloud nie są dostępne w niektórych regionach, na przykład w Stanach Zjednoczonych. Ponadto interfejsy użytkownika mogą wyglądać inaczej w różnych regionach.
</Admonition>

Wykonaj poniższe kroki, aby rozpocząć:

1. [Utwórz konto IBM Cloud.](./cloud-setup)
1. [Skonfiguruj konto OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Utwórz projekt OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Wykonaj kroki opisane w [Uruchom swój pierwszy Notebook AI,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) wybierając **Qiskit** jako framework.

### qBraid Lab {#qbraid-lab}

Użytkownicy Qiskit mogą korzystać ze wstępnie skonfigurowanych środowisk Python firmy qBraid, postępując zgodnie z tymi instrukcjami. Więcej informacji znajdziesz w [dokumentacji qBraid.](https://docs.qbraid.com/home/introduction)

1. [Załóż bezpłatne konto qBraid.](https://docs.qbraid.com/home/account)
3. [Utwórz środowisko Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Uruchom qBraid Lab, kliknij ikonę **ENVS** po prawej stronie, kliknij **+ ADD** u góry nowego paska bocznego i wyszukaj Qiskit. Kliknij wersję, której chcesz użyć, a następnie kliknij **Install**.
4. Utwórz notebook Jupyter, klikając **Python 3 [Qiskit]** w sekcji Notebook.
4. Możesz zainstalować dodatkowe pakiety Python (np. Qiskit v1.2) w komórce notebooka, uruchamiając `%pip install [package_name].` Więcej informacji znajdziesz w temacie [Polecenia Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Dodatkowe środowiska notebooków dla użytkowników Qiskit {#additional-notebook-environments-for-qiskit-users}

W przypadku platform bez wstępnie skonfigurowanego Qiskit instrukcje konfiguracji są podobne, z tą różnicą, że musisz zainstalować Qiskit ręcznie. Utworzysz konto, rozpoczniesz nowy projekt (lub notebook, lub obszar roboczy, w zależności od platformy) i zainstalujesz Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Skonfiguruj bezpłatną wersję próbną Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Utwórz projekt Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Aby zainstalować wymagane pakiety Qiskit i zależności, otwórz nowy notebook i uruchom następujące polecenie:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Utwórz konto Google.](https://support.google.com/accounts/answer/27441?hl)
2. Przejdź na [stronę główną Google Colab](https://colab.research.google.com/) i zaloguj się na swoje konto Google.
3. [Utwórz notebook Colab](https://colab.research.google.com/#create=true) i zainstaluj Qiskit, uruchamiając następujące polecenie:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Utwórz konto Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Utwórz obszar roboczy.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Utwórz notebook Jupyter w swoim obszarze roboczym.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Z poziomu notebooka uruchom to polecenie, aby zainstalować Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
