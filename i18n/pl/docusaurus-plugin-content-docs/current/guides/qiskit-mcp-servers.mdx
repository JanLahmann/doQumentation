# Serwery Qiskit MCP {#qiskit-mcp-servers}

Serwery Qiskit MCP to kolekcja serwerów [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) integrujących możliwości obliczeń kwantowych z systemami AI. Serwery te umożliwiają asystentom AI, dużym modelom językowym (LLM) i agentom dostęp do usług IBM Quantum&reg; i bibliotek Qiskit.

<Admonition type="note">
To jest projekt open-source. Najaktualniejsze informacje, szczegółowe pokrycie API i zaawansowane konfiguracje znajdziesz w [repozytorium GitHub Qiskit MCP Servers](https://github.com/Qiskit/mcp-servers).
</Admonition>

## Kiedy powinienem używać serwerów Qiskit MCP? {#when-should-i-use-qiskit-mcp-servers}

Serwery Qiskit MCP pozwalają asystentom AI pomagać ci w zadaniach związanych z obliczeniami kwantowymi, takich jak:

- Tworzenie, manipulowanie i transpilowanie Circuit kwantowych
- Pobieranie i przeszukiwanie dokumentacji Qiskit, przewodników i referencji API
- Odpowiadanie na pytania dotyczące obliczeń kwantowych bezpośrednio z wiarygodnych źródeł
- Generowanie kodu kwantowego ze świadomością kontekstu
- Optymalizacja Circuit przy użyciu transpilacji opartej na AI
- Automatyczne łączenie z rzeczywistym sprzętem kwantowym
- Analiza i wybór optymalnych Backend kwantowych
- Wykonywanie Circuit kwantowych i śledzenie statusu zadań

## Co jest dostępne? {#what-is-included}

Dostępne są następujące serwery MCP:

### Serwer Qiskit MCP {#qiskit-mcp-server}

Zapewnia podstawowe możliwości obliczeń kwantowych do lokalnego tworzenia oprogramowania. Ten serwer umożliwia asystentom AI tworzenie, manipulowanie i transpilowanie Circuit kwantowych, a także serializowanie ich do formatów takich jak OpenQASM 3 i QPY.

### Serwer Qiskit Runtime MCP {#qiskit-runtime-mcp-server}

Zapewnia dostęp do usług chmurowych IBM Quantum przez Qiskit Runtime. Umożliwia asystentom AI interakcję z Backend kwantowymi, przesyłanie zadań i pobieranie wyników.

### Serwer Qiskit Docs MCP {#qiskit-docs-mcp-server}

Zapewnia asystentom AI dostęp do pełnego ekosystemu dokumentacji Qiskit, w tym dokumentacji modułów SDK, przewodników implementacji i najlepszych praktyk. Ten serwer umożliwia inteligentne pobieranie i przeszukiwanie dokumentacji Qiskit bez konieczności uwierzytelniania.

### Serwer Qiskit IBM Transpiler MCP {#qiskit-ibm-transpiler-mcp-server}

Zapewnia optymalizację Circuit opartą na AI poprzez [przejścia Transpilatora oparte na AI](/guides/ai-transpiler-passes). Ten serwer umożliwia asystentom AI optymalizację Circuit kwantowych przy użyciu zaawansowanych algorytmów routingu i optymalizacji.

### Serwery społecznościowe {#community-servers}

Dostępne są również następujące serwery MCP stworzone przez społeczność:

#### Serwer Qiskit Gym MCP {#qiskit-gym-mcp-server}

Zapewnia możliwości uczenia przez wzmacnianie (RL) do syntezy Circuit kwantowych przy użyciu biblioteki [qiskit-gym](https://github.com/AI4quantum/qiskit-gym). Ten serwer umożliwia asystentom AI korzystanie z technik optymalizacji opartych na RL do transpilacji i syntezy Circuit kwantowych.

## Instalacja serwerów Qiskit MCP {#install-qiskit-mcp-servers}

Aby korzystać z serwerów Qiskit MCP, potrzebujesz:

- Python 3.10 lub nowszego (zalecany 3.11+)
- Konta IBM Quantum i tokenu API (instrukcje znajdziesz w [Konfiguracja konta IBM Cloud&reg;](/guides/cloud-setup))

Zainstaluj wszystkie dostępne serwery Qiskit MCP, uruchamiając następujące polecenie w terminalu:

```bash
pip install qiskit-mcp-servers[all]
```

Możesz też zainstalować poszczególne serwery MCP:

```bash
pip install qiskit-mcp-server
pip install qiskit-code-assistant-mcp-server
pip install qiskit-ibm-runtime-mcp-server
pip install qiskit-docs-mcp-server
pip install qiskit-ibm-transpiler-mcp-server
pip install qiskit-gym-mcp-server
```

## Konfiguracja serwerów Qiskit MCP {#configure-qiskit-mcp-servers}

### Ustawianie zmiennych środowiskowych {#set-environment-variables}

Skonfiguruj wymagane zmienne środowiskowe do uwierzytelniania. Serwer Qiskit MCP i Serwer Qiskit Docs MCP nie wymagają uwierzytelniania.

W przypadku serwerów MCP Qiskit Runtime lub Qiskit IBM Transpiler:

```bash
export QISKIT_IBM_TOKEN="<your_token>"
```

Token API znajdziesz w [ustawieniach konta IBM Quantum](https://quantum.cloud.ibm.com/).

### Konfiguracja klienta MCP {#configure-the-mcp-client}

Możesz skonfigurować klienta zgodnego z MCP (np. Claude Desktop, Cursor lub inne narzędzia AI) do używania serwerów Qiskit MCP. Dokładna konfiguracja zależy od twojego klienta. Na przykład plik konfiguracyjny klienta MCP może wyglądać tak:

```json
{
  "mcpServers": {
    "qiskit": {
      "command": "qiskit-mcp-server"
    },
    "qiskit-ibm-runtime": {
      "command": "qiskit-ibm-runtime-mcp-server"
    },
    "qiskit-code-assistant": {
      "command": "qiskit-code-assistant-mcp-server"
    },
    "qiskit-docs": {
      "command": "qiskit-docs-mcp-server"
    },
    "qiskit-ibm-transpiler": {
      "command": "qiskit-ibm-transpiler-mcp-server"
    },
    "qiskit-gym": {
      "command": "qiskit-gym-mcp-server"
    }
  }
}
```

Szczegółowe instrukcje konfiguracji znajdziesz w dokumentacji swojego klienta MCP.

## Testowanie serwerów MCP {#test-the-mcp-servers}

Serwery MCP możesz testować interaktywnie za pomocą MCP Inspector:

```bash
# Test the Qiskit MCP Server
npx @modelcontextprotocol/inspector qiskit-mcp-server

# Test the Qiskit Runtime MCP Server
npx @modelcontextprotocol/inspector qiskit-ibm-runtime-mcp-server

# Test the Qiskit Docs MCP Server
npx @modelcontextprotocol/inspector qiskit-docs-mcp-server

# Test the Qiskit IBM Transpiler MCP Server
npx @modelcontextprotocol/inspector qiskit-ibm-transpiler-mcp-server

# Test the Qiskit Gym MCP Server
npx @modelcontextprotocol/inspector qiskit-gym-mcp-server
```
