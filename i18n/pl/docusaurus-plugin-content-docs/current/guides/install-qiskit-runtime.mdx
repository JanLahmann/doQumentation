# Instalacja klienta Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime to usługa chmurowa do wykonywania obliczeń kwantowych na sprzęcie IBM Quantum&reg;. Ten przewodnik pokazuje, jak zainstalować `qiskit-ibm-runtime` — pakiet kliencki umożliwiający dostęp do usługi Qiskit Runtime.

Przed instalacją klienta Qiskit Runtime wykonaj instrukcje zawarte w tych przewodnikach:

1. [Zainstaluj Qiskit](/guides/install-qiskit).
2. [Skonfiguruj konto IBM Cloud](cloud-setup).

Uruchom to polecenie, aby zainstalować Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Bądź na bieżąco z najnowszymi wersjami">
Regularnie sprawdzaj [informacje o wydaniach Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes), aby śledzić nowe wersje. Zalecamy częste aktualizowanie wymagań dla `qiskit-ibm-runtime` poprzez uruchomienie `pip install -U qiskit-ibm-runtime`.
</Admonition>
