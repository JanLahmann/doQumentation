# Iberblick {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Dää Kurs erforsch, wie Quantum Computing ä spannendi Rolle im Machine Learning spiele kann. Der Kurs bietet ä Wiederholig vun klassische Machine Learning Asätz, goht aber au vun ere gwisse Vertrautheit mit däne Methode us. Der Hauptfokus litt druf, wie Quantum Computing in bestehendi Machine Learning Workflows passt. Thema sin Data Encoding, Quantum Kernels un variational quantum circuits.
