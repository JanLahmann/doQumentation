# Überblick {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

A practical introduction to quantum-safe cryptography isch e koschtaloser Online-Kurs bi IBM Quantum&reg; Learning un dient als Eiführung in di Grundlagekonzepte vo quantum-safe cryptography.

Dä Kurs isch für Entwickler gmacht worre, die interessiert sin an dr Modernisierung vo ihrer Awendungssicherheit, un er beinhaltet multimodali Lektione un interaktivi Live-Code-Beispiel in vier Kernbereich vo dr Kryptografie: kryptografischi Hash-Funktione, symmetrischi Schlüsselkryptografie, asymmetrischi Schlüsselkryptografie un quantum-safe cryptography.

Durch des Mache vo däm Kurs lerne Entwickler, wie sich di Cybersicherheit-Risikolandschaft entwickelt, un sie erforsche zitgenössischi Ansätz, die di Quantum-Ära präge werre.
