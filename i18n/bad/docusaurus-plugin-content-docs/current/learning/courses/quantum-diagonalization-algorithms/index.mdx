# Iberblick {#overview}

!["Hero image"](/learning/images/courses/quantum-diagonalization-algorithms/hero.avif)

Dää Kurs bietet ä Iberblick iber mehreri Asätz zur Quantum-Diagonalisierig. Es wird ä bissle Hintergrund zu de klassische Methode gliefert, wo verwändet werre oder wo d' Quantum-Algorithme motiviert hän. Der Inhalt goht dann durch d' Implementierig vun de Quantum-Algorithme uf echti Quantecomputer durch. Es git ä substantielli Diskussion driber, welchi Faktore d' Skalierig vun Asätz mit klassische un Quantum-Algorithme bestimme.

## Empfohlener Hintergrund {#recommended-background}

Dää Kurs isch fer Einzelpersone gedacht, wo scho ä bissle Erfahrig mit Quantum Computing hän un däss Kraft bi der Matrixdiagonalisierig nutze wän.

Mir empfehle, dass d' Lernende d' folgende Voraussetzunge vor däm Kurs erfülle:

- Richtet eieri lokali Umgebig i, indäm ihr däm [Get started](/guides/) Leitfade uf unserer Dokumentationssit folgt.
- Fer ä tieferes Verständnis vun Quanteinformation, probiert [Basics of Quantum Information](/learning/courses/basics-of-quantum-information) mit John Watrous.
