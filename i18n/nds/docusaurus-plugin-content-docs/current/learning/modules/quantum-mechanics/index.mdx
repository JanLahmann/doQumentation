# Överlick {#overview}

!["Hero image"](/learning/images/modules/quantum-mechanics/hero.avif)

Disse Sammlung vun Qiskit-in-de-School-Moduln is so utleggt, dat se in en Quantenmechanik-Kurs an de Universität past. Dat Ziel is, Lehrers un Studenten dorbi to ünnerstütten, Quantencomputer to bruken, üm Themen, de wieldat in en Quantenmechanikkurs wichtig sünd — as Unscherpheit un Superposizschoon — experimentell to utforschen. Jeden Modul schull in de Klass anbunnen warrn, mit wat Gruppenarbeit un Ünnerstüttung vun'n Lehrens, aver kann ok selfständig oder in lütte Gruppen afsloten warrn. All Aktivitäten köönt in veel weniger Tiet afsloten warrn, as de köstenlose Tiet, de as Deel vun'n Open Plan för all Bruukers to'r Verfögung steiht. Simulator-Optschoonen sünd ok inbuut, blooß för den Fall, dat Studenten ehr köstenlose Quantenrechenzeiten mal verfüttert hebbt.

Lehrers köönt Verschoonen vun disse Notizbücher mit Lösungsslötels un Hinwiesen op Platzeerung in gangbore Lehrpläne anfragen, indeem se dissen [kortten Fragebogen](https://ibm.biz/classrooms_instructor_key_request) dorto utföhrt, wo de Notizbücher innsett warrn.

## Anbefolen Vorkenntnisse {#recommended-background}

Jeden Modul hett Empfehlungen dorto, wo in en Lehrplan jeden Modul am meisten bringe deiht. In't Allgemeen kriegt Studenten dat meiste ut disse Moduln rut, wenn se:

- Wat Grundlagen in linearer Algebra hebbt ($2 \times 2$ Matrixen schullen töreeken)
- Wat Vertrautheit mit Python hebbt, obwol de meisten Notizbücher eenfach so utföhrt warrn köönt, as se schreeven sünd
