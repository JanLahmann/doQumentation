# Överschicht

!["Hero image"](/learning/images/courses/fundamentals-of-quantum-algorithms/hero.svg)

Willkamen bi *Fundamenten vun Quantenalgorithmen,* den tweeden Kurs in de Reih *Quanteninformatschoon un -rekenwerk verstohn*, de de nakumenden Kürse umfaat:

- [Grundlagen vun de Quanteninformatschoon](/learning/courses/basics-of-quantum-information)
- Fundamenten vun Quantenalgorithmen (disse Kurs)
- [Allgemene Formuleerung vun de Quanteninformatschoon](/learning/courses/general-formulation-of-quantum-information)
- [Grundlagen vun de Quantenfehlerkorrektur](/learning/courses/foundations-of-quantum-error-correction)

Disse Kurs utforscht de rekenwettmäßigen Vördelen vun Quanteninformatschoon, insl. wat wi mit Quantencomputers maken köönt un ehr Vördelen tegenöver klassische Computers. He fangt mit Quantenabfragealgorithmen an, de eenfache Machbarkeitstests för Quantenalgorithmen levert, un geiht dorna wieter to Quantenalgorithmen för Probleme insl. Ganztallenfaktoriseeren un unstruktureertes Söken.

Disse Kurs richtt sück an Studenten, Fackleute un Hobbyisten ut Beräken as Informatik, Physik, Ingenieurwetenschop un Mathematik, de segern Kenntnis över de theoretsche Grundlagen vun Quanteninformatschoon un -rekenwerk wüllt.
