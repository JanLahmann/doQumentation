# Överzicht {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Düsse Kurs is för Lüü, de al wat Erföhrung mit Quantencomputing hebbt un leren wüllt, woans man variationelle Algorithmen mit Qiskit Runtime Primitives schrieven deit. He is ok goot för Quantencomputing-Fachleute, de ehr Weten un Künnen op dat Gebeet vun variationellen Algorithmen utbuen wüllt.

## Prövung {#exam}

To den Badge för Variational Algorithm Design to kriegen, maak de Prövung bi IBM&reg; Training. Düsse Prövung is dordör dacht, dat se na dat Lesen vun de Lektschonen in düssen Kurs afleggt warrt. Wenn ji bestahn hebbt, kriggt ji vun Credly en Bescheed, dat ji den Badge verdeent hebbt.
