# Überblick

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Dieser Kurs richtet sich an Leute, de schon wat Erfahrung mitm Quantencomputing hamm und lernen wollen, wie man variationale Algorithmen mit den Qiskit Runtime Primitives schreibt. Er passt ooch für Quantencomputing-Profis, de ihr Wissen und ihre Fähigkeiten uffm Jebiet der variationalen Algorithmen erweitern wollen.

## Prüfung {#exam}

Um euer Badge für det Design variationaler Algorithmen zu kriegen, macht de Prüfung bei IBM&reg; Training. De Prüfung is dazu jedacht, nachm Durchlesen der Lektionen in diesem Kurs jemacht zu werden. Wenn ihr se bestanden habt, kriegt ihr von Credly Bescheid, det ihr een Badge verdient habt.
