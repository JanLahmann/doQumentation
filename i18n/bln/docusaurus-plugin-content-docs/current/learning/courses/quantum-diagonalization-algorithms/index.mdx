# Überblick

!["Hero image"](/learning/images/courses/quantum-diagonalization-algorithms/hero.avif)

Dieser Kurs jibt nen Überblick über verschiedene Ansätze zur Quantendiagonalisierung. Et wird nen bisken Hintergrund zu de klassischen Methoden jeliefert, de benutzt werden oder de de Quantenalgorithmen inspiriert haben. Danach jeht't Schritt für Schritt durch de Implementierung von de Quantenalgorithmen uff echten Quantencomputern. Det Skalierungsverhalten von klassischen und Quantenalgorithmen — wat dit bestimmt — wird janz ausführlich behandelt.

## Empfohlene Vorkenntnisse {#recommended-background}

Dieser Kurs is für Leute jedacht, de schon nen bisschen Erfahrung mit Quantencomputing haben und dit für de Matrixdiagonalisierung nutzen wollen.

Wir empfehlen, dat ihr vor diesem Kurs folgende Voraussetzungen erfüllt:

- Richtet euch eure lokale Umjebung ein, indem ihr de Anleitung [Erste Schritte](/guides/) in unserer Dokumentationsseite befolgt.
- Für ein tieferet Verständnis von de Quanteninformation, probiert mal [Jrundlagen von de Quanteninformation](/learning/courses/basics-of-quantum-information) mit John Watrous.
