บทเรียนนี้แนะนำโมเดลการคำนวณแบบ *Circuit ควอนตัม* ซึ่งเป็นวิธีมาตรฐานในการอธิบายการคำนวณเชิงควอนตัม

บทเรียนยังแนะนำแนวคิดทางคณิตศาสตร์สำคัญอีกไม่กี่อย่าง รวมถึง *inner product* ระหว่างเวกเตอร์ แนวคิดของ *orthogonality* และ *orthonormality* และ *projection* กับ *projective measurement* ซึ่งเป็นการวัดทั่วไปกว่าการวัดฐานมาตรฐาน
ผ่านแนวคิดเหล่านี้ เราจะสรุปข้อจำกัดพื้นฐานของข้อมูลเชิงควอนตัม รวมถึง *no-cloning theorem* และความเป็นไปไม่ได้ที่จะแยกแยะสถานะควอนตัมที่ไม่ orthogonal ได้อย่างสมบูรณ์
