# ภาพรวม {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

A practical introduction to quantum-safe cryptography คือคอร์สออนไลน์ฟรีใน IBM Quantum&reg; Learning และทำหน้าที่เป็น primer เกี่ยวกับแนวคิดพื้นฐานใน quantum-safe cryptography

คอร์สนี้สร้างขึ้นสำหรับนักพัฒนาที่สนใจการปรับปรุงความปลอดภัยของแอปพลิเคชันและมีบทเรียน multimodal และตัวอย่าง live code แบบ interactive ในสี่พื้นที่หลักของ cryptography ได้แก่ cryptographic hash functions, symmetric key cryptography, asymmetric key cryptography และ quantum-safe cryptography

โดยการเรียนคอร์สนี้ นักพัฒนาจะเรียนรู้ว่า cybersecurity risk landscape กำลังพัฒนาอย่างไร รวมถึงสำรวจแนวทางร่วมสมัยที่พร้อมกำหนดรูปแบบยุค quantum
