# ขั้นตอนต่อไปคืออะไร? {#whats-next}

ตอนนี้ที่คุณได้เรียนรู้พื้นฐานของ quantum-safe cryptography แล้ว ลองสำรวจเทคโนโลยีที่ IBM Quantum&reg; เสนอสำหรับการปรับปรุง cryptographic landscape ของคุณและสร้าง quantum cyber resilience [IBM Quantum Safe&trade;](https://www.ibm.com/quantum/quantum-safe) คือชุดเครื่องมือ ความสามารถ และแนวทางที่ครอบคลุมร่วมกับความเชี่ยวชาญเชิงลึกเพื่อช่วยคุณวางแผนและดำเนินการย้ายองค์กรของคุณไปสู่ quantum-safe cryptography ค้นพบ สังเกต และแปลง cryptography ของคุณด้วยความสามารถทางเทคนิคหลักสามประการ:

**1. IBM Quantum Safe Explorer**

IBM Quantum Safe Explorer ลดความซับซ้อนในการค้นพบและจัดการ cryptography โดยการสแกน source code และ object code เพื่อระบุ artifacts ที่เกี่ยวข้องกับ cryptography ทั้งหมด เปิดเผย dependencies และแสดงความเปราะบาง Explorer แสดงข้อมูลนี้ใน [Cryptography Bill of Materials](https://github.com/IBM/CBOM) (CBOM) ซึ่งเป็น extension ทั่วไปต่อ Software Bill of Materials (SBOM) ที่ทำให้การสร้างและแบ่งปัน metadata ที่อธิบาย cryptographic artifacts เป็นระบบ

**2. IBM Quantum Safe Advisor**

IBM Quantum Safe Advisor เสริม static code analysis จาก IBM Quantum Safe Explorer ด้วยมุมมอง dynamic, runtime ของ cryptographic posture และ compliance ของคุณ Advisor เชื่อมโยง [CBOMs][CBOM] และรวบรวมข้อมูลจาก network และ endpoint scans เพื่อสร้าง cryptographic inventory ที่ครอบคลุม inventory นี้ให้รายละเอียด:

- ประเภทและตำแหน่งของ cryptographic instances ของคุณ
- certificates, cipher suites และ CBOMs ที่เกี่ยวข้อง
- ความสัมพันธ์ระหว่าง assets และ data flows
- ความเปราะบางที่อาจเกิดขึ้นต่อ quantum technology

ด้วย policy-based enrichment Advisor ช่วยให้คุณประเมิน cryptography และ data compliance และสร้างรายการ vulnerabilities ที่มีลำดับความสำคัญ เพื่อที่คุณสามารถกำหนดกลยุทธ์การแก้ไข quantum-safe

**3. IBM Quantum Safe Remediator**

IBM Quantum Safe Remediator ช่วยให้คุณใช้ best practices สำหรับ quantum-safe remediation patterns ที่เน้นที่ VPN communication, TLS communication และ cluster access communication สร้าง crypto-agility โดยการสำรวจ solutions สำหรับการแปลง cryptography ของคุณโดยไม่รบกวน business operations

อ่าน [technology guide](https://www.ibm.com/quantum/assets/quantum-safe/IBM_Quantum-Safe-Technology-Brochure.pdf) และเรียนรู้เพิ่มเติมที่ [IBM Quantum Safe site](https://www.ibm.com/quantum/quantum-safe)

[CBOM]: #definition-tooltip "A schema extension of the CycloneDX Software Bill of Materials (SBOM) standard that provides an object model for describing cryptographic artifacts and their dependencies."
