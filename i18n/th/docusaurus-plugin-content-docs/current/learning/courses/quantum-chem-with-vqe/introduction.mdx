ก่อนเริ่ม กรุณากรอก[แบบสำรวจก่อนเรียนสั้น ๆ](https://your.feedback.ibm.com/jfe/form/SV_ePXDhFWwqj4Q0HY) นี้ ซึ่งมีความสำคัญในการปรับปรุงเนื้อหาและประสบการณ์ผู้ใช้ของเรา

คอร์สนี้เป็นบทนำเฉพาะด้านสำหรับ variational quantum eigensolver (VQE) รวมถึงการประยุกต์ใช้ VQE กับเคมี คอร์สนี้มีไว้สำหรับผู้ที่ปฏิบัติงานด้านเทคนิคของการคำนวณควอนตัมที่สนใจนำการคำนวณควอนตัมไปใช้กับปัญหาเคมี ชีวเคมี หรือ biophysics คอร์สนี้ครอบคลุมองค์ประกอบสำคัญของการคำนวณ VQE รวมถึงสิ่งเหล่านี้:

* Hamiltonian
* Ansatz
* Classical optimizer

องค์ประกอบทั้งหมดเหล่านี้ถูกนำมารวมกันเพื่อสร้างการคำนวณ VQE บน Qiskit Runtime Primitives โดยหลักคือ Estimator สำหรับข้อมูลเกี่ยวกับ primitives ดู [documentation](/guides/primitives) ในการตั้งค่าการคำนวณ VQE เราจะปฏิบัติตาม standard Qiskit patterns framework:

* แมป classical inputs ไปยังปัญหาควอนตัม
* Optimize ปัญหาสำหรับการรันบนควอนตัม
* รันโดยใช้ Qiskit Runtime primitives
* Post-process คืนผลลัพธ์ในรูปแบบ classical

คำอธิบาย framework บางส่วนจะพบตลอดคอร์สนี้ แต่สำหรับคำอธิบายครบถ้วนพร้อมตัวอย่างของ Qiskit patterns ดู [documentation นี้](/guides/intro-to-patterns)

เราแนะนำให้วางแผนสละเวลาประมาณ 6-10 ชั่วโมงในการทำคอร์สนี้ให้เสร็จสมบูรณ์ วิดีโอและเนื้อหาอ่านน่าจะใช้เวลาสองชั่วโมง เวลาที่เหลือสำหรับการรันและแก้ไข code cells ทำแบบทดสอบ และอ่านเอกสารที่ลิงก์ไว้
