การคำนวณเชิงควอนตัมมีศักยภาพในการแก้ปัญหาการคำนวณที่ยังไม่มีอัลกอริทึมแบบคลาสสิกที่มีประสิทธิภาพ และอาจไม่มีเลยด้วยซ้ำ
อย่างไรก็ตาม มีความท้าทายสำคัญมากที่ต้องเอาชนะก่อนที่เราจะสามารถดำเนินการคำนวณเชิงควอนตัมขนาดใหญ่ที่เชื่อถือได้อย่างที่เราหวัง

ประเด็นสำคัญคือสารสนเทศควอนตัมเปราะบางอย่างยิ่ง — แค่มองดูมันก็ทำให้เสียหายได้
ด้วยเหตุนี้ คอมพิวเตอร์ควอนตัมต้องแยกสารสนเทศควอนตัมที่จัดเก็บออกจากสภาพแวดล้อมรอบข้างในระดับสูงสุดเพื่อให้ทำงานได้ถูกต้อง
แต่ในขณะเดียวกัน คอมพิวเตอร์ควอนตัมต้องให้การควบคุมที่แม่นยำมากต่อสารสนเทศควอนตัมนี้ รวมถึงการ initialize ที่ถูกต้อง การดำเนินการยูนิทารีที่แม่นยำและเชื่อถือได้ และความสามารถในการวัดเพื่อให้ได้ผลลัพธ์ของการคำนวณ
มีความขัดแย้งชัดเจนระหว่างข้อกำหนดเหล่านี้ และในยุคแรก ๆ ของการคำนวณเชิงควอนตัม บางคนมองว่าความเปราะบางของสารสนเทศควอนตัมและความอ่อนไหวต่อทั้งความไม่แม่นยำและสัญญาณรบกวนจากสภาพแวดล้อมจะทำให้การคำนวณเชิงควอนตัมเป็นไปไม่ได้ในที่สุด

ทุกวันนี้ แทบไม่มีข้อสงสัยว่าการสร้างคอมพิวเตอร์ควอนตัมขนาดใหญ่ที่แม่นยำและเชื่อถือได้เป็นความท้าทายอย่างยิ่ง
แต่เรามีเครื่องมือสำคัญที่ช่วยในความพยายามนี้ — การแก้ไขข้อผิดพลาดเชิงควอนตัม — ซึ่งทำให้ผู้ที่มีความรู้ในสาขานี้ส่วนใหญ่มองโลกในแง่ดีว่าการคำนวณเชิงควอนตัมขนาดใหญ่จะกลายเป็นความจริงได้ในวันหนึ่ง

เราจะศึกษาการแก้ไขข้อผิดพลาดเชิงควอนตัมในคอร์สนี้ โดยเน้นที่พื้นฐาน
ในบทเรียนนี้ เราจะมองการแก้ไขข้อผิดพลาดเชิงควอนตัมเป็นครั้งแรก รวมถึง quantum error correcting code แรกที่ค้นพบ — *9-qubit Shor code* — และเราจะพูดถึงแนวคิดพื้นฐานใน quantum error correction ที่เรียกว่า *การแยกประเภทของข้อผิดพลาด*
