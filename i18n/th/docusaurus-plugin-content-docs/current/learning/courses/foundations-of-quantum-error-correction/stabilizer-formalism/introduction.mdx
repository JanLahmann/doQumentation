ในบทก่อนหน้า เราได้ดูการแก้ไขข้อผิดพลาดเชิงควอนตัมเป็นครั้งแรก โดยมุ่งเน้นที่ 9-qubit Shor code
ในบทนี้ เราจะแนะนำ *stabilizer formalism* ซึ่งเป็นกรอบคณิตศาสตร์ที่ช่วยระบุและวิเคราะห์โค้ดแก้ไขข้อผิดพลาดเชิงควอนตัมประเภทกว้างที่เรียกว่า *stabilizer codes*
ซึ่งรวมถึง 9-qubit Shor code พร้อมตัวอย่างอื่น ๆ อีกมาก รวมถึงโค้ดที่ดูเหมือนจะเหมาะกับอุปกรณ์ควอนตัมในโลกแห่งความเป็นจริง
ไม่ใช่ทุก quantum error correcting code ที่เป็น stabilizer code แต่หลายโค้ดก็เป็น รวมถึงทุกตัวอย่างที่เราจะเห็นในคอร์สนี้

บทเริ่มด้วยการพูดถึงสั้น ๆ เกี่ยวกับ Pauli matrices และ tensor products ของ Pauli matrices โดยทั่วไป ซึ่งสามารถแทน operations บน qubits ได้ รวมถึง measurements ของ qubits ด้วย — ในกรณีนั้นมักเรียกว่า *observables*
เราจะกลับไปดู repetition code อีกครั้งและดูว่ามันอธิบายได้อย่างไรในแง่ของ Pauli matrix observables
สิ่งนี้จะทั้งแจ้งให้ทราบและนำไปสู่การอภิปรายทั่วไปเกี่ยวกับ stabilizer codes รวมถึงตัวอย่างหลายอย่าง คุณสมบัติพื้นฐานของ stabilizer codes และวิธีปฏิบัติงานพื้นฐานของการ encoding การตรวจจับข้อผิดพลาด และการแก้ไขข้อผิดพลาดเหล่านั้น
