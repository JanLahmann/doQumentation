ยินดีต้อนรับสู่การประมวลผลเชิงควอนตัมระดับ utility-scale!

คอร์สนี้อิงจากคอร์สสดที่จัดขึ้นที่มหาวิทยาลัยโตเกียวในปี 2024 แต่ละบทเรียนประกอบด้วย PDF ที่ใช้ในการบรรยาย และกิจกรรมแล็บเชิงคำนวณ ในบางกรณี แล็บจะมีเนื้อหาเบาบางและเพียงแค่แสดงขั้นตอนการทำงาน โดยคำอธิบายจะอยู่ใน PDF ที่เกี่ยวข้อง ในกรณีอื่น แล็บจะมีคำอธิบายละเอียดและแรงจูงใจทางคณิตศาสตร์

**บทที่ 2-6** ครอบคลุมอัลกอริทึมที่มีชื่อเสียง เช่น Grover และการประมาณค่าเฟสเชิงควอนตัม อัลกอริทึมเหล่านี้มีความสำคัญอย่างยิ่งต่อความเข้าใจเกี่ยวกับศักยภาพของการประมวลผลเชิงควอนตัม แต่ส่วนใหญ่เป็นเนื้อหาที่อยู่นอกเหนือจากการคำนวณระดับ utility-scale ที่เรากำลังสร้าง หากคุณเพิ่งเริ่มต้นกับการประมวลผลเชิงควอนตัม ควรศึกษาบทเหล่านี้ให้ครบถ้วน หากคุณมีประสบการณ์มากพอแล้ว บทเหล่านี้ไม่มีเนื้อหาใหม่ที่เฉพาะเจาะจงสำหรับการคำนวณระดับ utility-scale ที่เรากำลังทำ

**บทที่ 7-8** ทบทวนการจำลองเชิงควอนตัมและแบบคลาสสิก บทเรียนเรื่องการจำลองแบบคลาสสิกเขียนโดย ศ. Yoshiaki Kawase แห่งมหาวิทยาลัยโตเกียว และเก็บอยู่บน GitHub

**บทที่ 9** มุ่งเน้นไปที่ฮาร์ดแวร์การประมวลผลเชิงควอนตัมสมัยใหม่

เริ่มต้นตั้งแต่**บทที่ 10** เป็นต้นไป เราจะครอบคลุมหัวข้อที่มีความสำคัญพิเศษในบริบทของการคำนวณระดับ utility-scale การเข้าใจการปรับแต่ง Circuit (**บทที่ 10**) และการลดความผิดพลาด (**บทที่ 11**) มีความสำคัญอย่างยิ่งต่อการได้ผลลัพธ์ที่ดีในระดับนี้ การคำนวณระดับ utility-scale เริ่มต้นใน**บทที่ 12** และสรุปใน**บทที่ 14**

มีการประมาณเวลา QPU ที่จำเป็นอย่างสุจริตไว้ที่ด้านบนของแต่ละบทเรียน แต่การคำนวณของคุณอาจแตกต่างออกไปเล็กน้อย ไม่มีแบบทดสอบหรือเครื่องหมายรับรองสำหรับคอร์สนี้

หวังว่าทรัพยากรการสอนเหล่านี้จะเป็นประโยชน์ต่อคุณและนักเรียนในการบรรลุเป้าหมายด้านการประมวลผลเชิงควอนตัมที่มีประโยชน์
