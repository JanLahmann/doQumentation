# ภาพรวม {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

คอร์สนี้เหมาะสำหรับผู้ที่มีประสบการณ์ด้านการประมวลผลเชิงควอนตัมบ้างแล้วและต้องการเรียนรู้วิธีเขียนอัลกอริทึม variational โดยใช้ Qiskit Runtime primitives นอกจากนี้ยังเหมาะสำหรับนักพัฒนาและนักวิจัยด้านการประมวลผลเชิงควอนตัมที่ต้องการขยายความรู้และทักษะในด้านอัลกอริทึม variational

## ข้อสอบ {#exam}

เพื่อรับ badge สำหรับ Variational algorithm design ให้ทำข้อสอบที่ IBM&reg; Training ข้อสอบนี้ควรทำหลังจากอ่านบทเรียนในคอร์สนี้แล้ว หลังจากผ่านข้อสอบ คุณจะได้รับแจ้งจาก Credly ว่าได้รับ badge แล้ว
