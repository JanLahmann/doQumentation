# ภาพรวม {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

ยินดีต้อนรับสู่ *ใช้คอมพิวเตอร์ควอนตัมได้วันนี้เลย*! ในคอร์สนี้ คุณจะเริ่มใช้คอมพิวเตอร์ควอนตัมได้ทันที และเรียนรู้หลักการพื้นฐานของการคำนวณควอนตัมไปพร้อมกัน

คอร์สนี้ออกแบบมาสำหรับคนที่อยากรู้และคนที่มีเวลาจำกัด ไม่ต้องมีปริญญาด้านฟิสิกส์หรือประสบการณ์การเขียนโปรแกรมมาหลายปี แค่ทำตาม Notebook และกด "Run" ได้ก็พร้อมแล้ว
