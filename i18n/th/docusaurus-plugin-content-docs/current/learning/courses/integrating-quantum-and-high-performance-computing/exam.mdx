# ข้อสอบและแบดจ์ {#exam-and-badge}

ยินดีด้วยที่เรียนจบคอร์สนี้!

ข้อสอบและแบดจ์สำหรับคอร์สนี้จะเปิดให้ใช้งานในเร็วๆ นี้ กรุณาตรวจสอบอีกครั้งในอีกไม่กี่วัน

หมายเหตุ: IBM ใช้บริการของ Credly ซึ่งเป็นผู้ประมวลผลข้อมูลบุคคลที่สามที่ได้รับอนุญาตจาก IBM และตั้งอยู่ในสหรัฐอเมริกา เพื่อช่วยในการบริหารจัดการโปรแกรม IBM Digital Badge ในการออกแบดจ์ดิจิทัล IBM ให้กับคุณ ข้อมูลส่วนตัว (ชื่อ ที่อยู่อีเมล และแบดจ์ที่ได้รับ) จะถูกแชร์กับ Credly คุณจะได้รับการแจ้งเตือนทางอีเมลจาก Credly พร้อมคำแนะนำในการรับแบดจ์ ข้อมูลส่วนตัวของคุณจะถูกใช้เพื่อออกแบดจ์และเพื่อการรายงานโปรแกรมและวัตถุประสงค์ในการดำเนินงาน IBM อาจแชร์ข้อมูลส่วนตัวที่เก็บรวบรวมกับบริษัทในเครือและบุคคลที่สามทั่วโลก ข้อมูลจะได้รับการจัดการในลักษณะที่สอดคล้องกับแนวปฏิบัติด้านความเป็นส่วนตัวของ IBM สามารถดูแถลงการณ์ความเป็นส่วนตัวของ IBM ได้ [ที่นี่](https://www.ibm.com/privacy/us/en/)
