# ภาพรวม {#overview}

!["Hero image"](/learning/images/courses/fundamentals-of-quantum-algorithms/hero.svg)

ยินดีต้อนรับสู่ *พื้นฐานของอัลกอริทึมควอนตัม* ซึ่งเป็นคอร์สที่สองในซีรีส์ *ทำความเข้าใจสารสนเทศและการคำนวณควอนตัม* ที่ประกอบด้วยคอร์สต่อไปนี้:

- [พื้นฐานของสารสนเทศควอนตัม](/learning/courses/basics-of-quantum-information)
- พื้นฐานของอัลกอริทึมควอนตัม (คอร์สนี้)
- [การกำหนดสูตรทั่วไปของสารสนเทศควอนตัม](/learning/courses/general-formulation-of-quantum-information)
- [รากฐานของการแก้ไขข้อผิดพลาดควอนตัม](/learning/courses/foundations-of-quantum-error-correction)

คอร์สนี้สำรวจข้อได้เปรียบเชิงคำนวณของสารสนเทศควอนตัม รวมถึงสิ่งที่เราทำได้ด้วยคอมพิวเตอร์ควอนตัมและข้อได้เปรียบเหนือคอมพิวเตอร์แบบคลาสสิก
เริ่มด้วย quantum query algorithms ซึ่งเป็นการสาธิต proof-of-concept ง่าย ๆ สำหรับอัลกอริทึมควอนตัม แล้วต่อด้วยอัลกอริทึมควอนตัมสำหรับปัญหาต่าง ๆ
รวมถึงการแยกตัวประกอบจำนวนเต็มและ unstructured search

คอร์สนี้เหมาะสำหรับนักศึกษา ผู้เชี่ยวชาญ และผู้ที่สนใจในสาขาวิชาต่าง ๆ เช่น วิทยาการคอมพิวเตอร์ ฟิสิกส์ วิศวกรรม และคณิตศาสตร์ ที่ต้องการเพิ่มพูนความรู้เกี่ยวกับรากฐานทางทฤษฎีของสารสนเทศและการคำนวณควอนตัม
