# ภาพรวม {#overview}

!["Hero image"](/learning/images/courses/designing-and-leading-quantum-projects/hero.svg)

หลักสูตรนี้ออกแบบมาเพื่อช่วยให้ทีมและองค์กรสามารถสร้างขีดความสามารถที่จำเป็นเพื่อเริ่มต้นโครงการริเริ่มด้านควอนตัมได้อย่างมีประสิทธิภาพ

แนะนำให้ใช้เวลาประมาณ 4 ชั่วโมงเพื่อเรียนรู้เนื้อหาทั้งหมด ขึ้นอยู่กับประสบการณ์ก่อนหน้าของแต่ละคน

### กลุ่มเป้าหมาย {#intended-audience}

แนะนำหลักสูตรนี้สำหรับผู้นำและสถาปนิกของโครงการริเริ่มด้านควอนตัม รวมถึงคณาจารย์ในสถาบันการศึกษา ผู้นำโครงการในองค์กรขนาดใหญ่ ผู้นำ startup และอื่นๆ หลักสูตรนี้จะเป็นประโยชน์อย่างยิ่งสำหรับองค์กรที่เพิ่งเริ่มต้นกับการประมวลผลเชิงควอนตัม

## ความรู้พื้นฐานที่แนะนำ {#recommended-background}

ไม่มีข้อกำหนดด้านเทคนิคสำหรับหลักสูตรนี้ แต่แนะนำให้ผู้เรียนศึกษาหลักสูตร [Quantum Business Foundations](/learning/courses/quantum-business-foundations) ก่อน ซึ่งเป็นบทนำที่ดีสู่ระบบนิเวศควอนตัม
