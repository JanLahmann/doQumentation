# ภาพรวม {#overview}

!["Hero image"](/learning/images/modules/quantum-mechanics/hero.avif)

คอลเลกชัน Qiskit in classrooms นี้ออกแบบมาเพื่อใช้ร่วมกับหลักสูตรกลศาสตร์ควอนตัมระดับปริญญาตรี เป้าหมายคือช่วยให้นักการศึกษาและนักเรียนใช้ประโยชน์จากคอมพิวเตอร์ควอนตัมเพื่อสำรวจหัวข้อที่สำคัญในหลักสูตรกลศาสตร์ควอนตัมเชิงทดลอง เช่น ความไม่แน่นอนและ superposition
โมดูลแต่ละตัวควรเริ่มในชั้นเรียนพร้อมการสนับสนุนจากกลุ่มและผู้สอน แต่สามารถทำให้เสร็จสมบูรณ์ได้อย่างอิสระหรือในกลุ่มเล็ก กิจกรรมทั้งหมดสามารถทำได้ในเวลาน้อยกว่าเวลาว่างที่ให้ไว้ในแผน Open Plan ที่พร้อมใช้งานสำหรับผู้ใช้ทุกคน ตัวเลือก Simulator ก็มีอยู่ด้วย เผื่อนักเรียนใช้เวลาคอมพิวเตอร์ควอนตัมฟรีหมดโดยบังเอิญ

นักการศึกษาสามารถขอรับเวอร์ชัน notebook ที่มีเฉลยและคำแนะนำเกี่ยวกับการวางในหลักสูตรทั่วไปได้โดยกรอก[แบบสำรวจสั้น ๆ](https://ibm.biz/classrooms_instructor_key_request) นี้เกี่ยวกับวิธีใช้ notebook

## พื้นฐานที่แนะนำ {#recommended-background}

โมดูลแต่ละตัวมีคำแนะนำเกี่ยวกับตำแหน่งในหลักสูตรที่โมดูลแต่ละตัวอาจมีประสิทธิผลสูงสุด โดยทั่วไป นักเรียนจะได้รับประโยชน์สูงสุดจากโมดูลเหล่านี้ถ้ามี:

- พื้นฐานพีชคณิตเชิงเส้นบ้าง (เมทริกซ์ $2 \times 2$ น่าจะเพียงพอ)
- ความคุ้นเคยกับ Python บ้าง แม้ว่า notebook ส่วนใหญ่สามารถรันตามที่เขียนไว้ได้โดยตรง
