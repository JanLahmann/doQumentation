# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) คือแพ็กเกจส่วนขยายอย่างเป็นทางการที่มอบอัลกอริทึมและเทคนิคการคำนวณเชิงควอนตัมเฉพาะทาง แต่ละ Addon มุ่งเน้นด้านใดด้านหนึ่งและมีบทเรียนพร้อมโค้ดที่รันได้จริง

## Addons ที่มีให้ใช้งาน {#available-addons}

| Addon | แพ็กเกจ | คำอธิบาย |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | ลดความกว้างและความลึกของ Circuit ด้วยการตัด Gate และสาย |
| **SQD** | `qiskit-addon-sqd` | Sample-based Quantum Diagonalization สำหรับประมาณค่า ground state |
| **OBP** | `qiskit-addon-obp` | Operator Back-Propagation เพื่อลดความลึกของ Circuit |
| **MPF** | `qiskit-addon-mpf` | Multi-Product Formulas สำหรับการจำลอง Hamiltonian ที่แม่นยำยิ่งขึ้น |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximate Quantum Compilation ด้วย tensor networks |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption สำหรับลดข้อผิดพลาด |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones สำหรับลด overhead การสุ่มตัวอย่าง PEC |

บทเรียนทั้งหมดสามารถรันได้โดยตรงในเบราว์เซอร์ผ่าน Binder หรือบน Jupyter server ของคุณเอง

สำหรับเอกสาร API ให้เยี่ยมชม repo ของแต่ละ Addon บน [GitHub](https://github.com/Qiskit?q=qiskit-addon)
