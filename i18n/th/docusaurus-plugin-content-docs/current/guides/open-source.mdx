# โอเพนซอร์ส {#open-source}

ตั้งแต่ปี 2016 IBM Quantum&reg; ได้ใช้พลังของนวัตกรรมที่ขับเคลื่อนโดยชุมชน โดยทำให้ทรัพยากรการคำนวณเชิงควอนตัมเข้าถึงได้สำหรับทุกคนผ่านระบบคลาวด์
โครงการบุกเบิกนี้ครอบคลุมการเปิดซอร์สโค้ดส่วนสำคัญของ quantum software stack และการเปิดตัว Qiskit ซึ่งเป็น framework การคำนวณเชิงควอนตัมที่ยืดหยุ่น
วันนี้ Qiskit เป็น quantum framework ที่ได้รับความนิยมมากที่สุด ส่วนหนึ่งต้องยกเครดิตให้กับชุมชนที่มีชีวิตชีวา
ชุมชน Qiskit คือกลุ่มนักพัฒนา นักวิทยาศาสตร์ และผู้ที่ชื่นชอบที่เปิดรับทุกคน โดยร่วมมือกันพัฒนาซอฟต์แวร์ งานวิจัย การศึกษา และธุรกิจผ่านการคำนวณเชิงควอนตัม
การทำให้เทคโนโลยีที่เปลี่ยนโลกนี้เปิดกว้างและเข้าถึงได้ ช่วยเร่งนวัตกรรมและค้นพบแอปพลิเคชันใหม่ ๆ

เมื่อเทคโนโลยีควอนตัมพัฒนาไปเรื่อย ๆ การมีส่วนร่วมจากโปรเจกต์และนักพัฒนาใน [Qiskit ecosystem](https://www.ibm.com/quantum/ecosystem) ที่เป็นโอเพนซอร์ส จึงมีความสำคัญอย่างยิ่งต่อการกำหนดทิศทางในอนาคต

การเข้าร่วมชุมชน Qiskit ทำให้เราสามารถมีส่วนร่วมในการพัฒนาเทคโนโลยีควอนตัมได้โดยตรง
ไม่ว่าจะเป็นนักเขียนโปรแกรมควอนตัมที่มีประสบการณ์ หรือเพิ่งเริ่มต้นก้าวแรกในการเดินทางสู่โลกควอนตัม Qiskit มีทั้งการสนับสนุนและแรงบันดาลใจที่จะช่วยผลักดันขีดจำกัดใหม่
เรายินดีรับทุกประเภทของการมีส่วนร่วม ตั้งแต่โค้ดใน pull request ไปจนถึงการสนับสนุนชุมชนบน [Slack](https://qisk.it/join-slack) เพื่อให้ทุกคนมีบทบาทในการพัฒนาเทคโนโลยีควอนตัม
