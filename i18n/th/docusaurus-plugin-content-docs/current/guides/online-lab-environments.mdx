# สภาพแวดล้อมแล็บออนไลน์ {#online-lab-environments}

นอกจากการดาวน์โหลดและติดตั้ง Qiskit เพื่อใช้งานบนเครื่องของคุณเอง ยังมีตัวเลือกอื่นสำหรับผู้ที่ต้องการทำงานในสภาพแวดล้อมการพัฒนาออนไลน์

ผู้ให้บริการหลายรายนำเสนอสภาพแวดล้อม Jupyter ออนไลน์ที่ติดตั้ง Qiskit ไว้ล่วงหน้าเพื่อการเริ่มต้นใช้งานที่รวดเร็วและสะดวก ผู้ให้บริการที่แนะนำได้แก่ผู้ให้บริการคลาวด์ OVHcloud และแพลตฟอร์มคลาวด์พัฒนาซอฟต์แวร์ควอนตัม qBraid

## สภาพแวดล้อม notebook ที่แนะนำพร้อมติดตั้ง Qiskit ไว้ล่วงหน้า {#recommended-notebook-environments-with-qiskit-preinstalled}

สภาพแวดล้อม Jupyter notebook ออนไลน์อื่นๆ อย่าง IBM Watson&reg; Studio, Google Colab และ Microsoft Azure Machine Learning Studio สามารถใช้งานกับ Qiskit ได้โดยทำตามคำแนะนำที่เหมาะสม

### OVHcloud AI notebooks {#ovhcloud-ai-notebooks}

ผู้ใช้ OVHcloud Public Cloud และ OVHcloud Startup สามารถใช้ OVHcloud quantum notebooks เพื่อเข้าถึง Qiskit และส่งงานไปยัง IBM&reg; QPU ด้วยบัญชี IBM Quantum&reg; หรือ IBM Cloud&reg; ของตน สำหรับข้อมูลเพิ่มเติม เยี่ยมชมหน้า [OVHcloud Documentation and tutorials](https://help.ovhcloud.com/csm/world-documentation?id=kb_home)

<Admonition type="note">
OVHcloud AI notebooks ไม่พร้อมใช้งานในบางภูมิภาค เช่น สหรัฐอเมริกา นอกจากนี้ อินเทอร์เฟซผู้ใช้อาจมีลักษณะต่างกันในแต่ละภูมิภาค
</Admonition>

ทำตามขั้นตอนเหล่านี้เพื่อเริ่มต้น:

1. [สร้างบัญชี IBM Cloud](./cloud-setup)
1. [ตั้งค่าบัญชี OVHcloud](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [สร้างโปรเจกต์ OVH Public Cloud](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. ทำตามขั้นตอนใน [Launch your first AI Notebook](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) โดยเลือก **Qiskit** สำหรับ framework

### qBraid Lab {#qbraid-lab}

ผู้ใช้ Qiskit สามารถใช้สภาพแวดล้อม Python ที่กำหนดค่าล่วงหน้าของ qBraid ได้โดยทำตามคำแนะนำเหล่านี้ สำหรับข้อมูลเพิ่มเติม ดูที่ [qBraid Docs](https://docs.qbraid.com/home/introduction)

1. [ตั้งค่าบัญชีฟรีของ qBraid](https://docs.qbraid.com/home/account)
3. [สร้างสภาพแวดล้อม Lab](https://docs.qbraid.com/lab/user-guide/getting-started)
3. เปิด qBraid Lab คลิกไอคอน **ENVS** ทางขวา คลิก **+ ADD** ที่ด้านบนของแถบด้านข้างใหม่ และค้นหา Qiskit แล้วคลิกเวอร์ชันที่ต้องการใช้ จากนั้นคลิก **Install**
4. สร้าง Jupyter notebook โดยคลิก **Python 3 [Qiskit]** ในส่วน Notebook
4. คุณสามารถติดตั้งแพ็กเกจ Python เพิ่มเติม (เช่น Qiskit v1.2) ใน notebook cell ได้โดยรัน `%pip install [package_name]` สำหรับข้อมูลเพิ่มเติม ดูหัวข้อ [Pip (magic) commands](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands)

## สภาพแวดล้อม notebook เพิ่มเติมสำหรับผู้ใช้ Qiskit {#additional-notebook-environments-for-qiskit-users}

สำหรับแพลตฟอร์มที่ไม่ได้กำหนดค่า Qiskit ไว้ล่วงหน้า ขั้นตอนการตั้งค่าจะคล้ายกัน ยกเว้นว่าคุณต้องติดตั้ง Qiskit เอง คุณจะสร้างบัญชี เริ่มโปรเจกต์ใหม่ (หรือ notebook หรือ workspace ขึ้นอยู่กับแพลตฟอร์ม) และติดตั้ง Qiskit

### IBM Watson Studio {#ibm-watson-studio}

1. [ตั้งค่า Watson Studio รุ่นทดลองฟรี](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [สร้างโปรเจกต์ Watson Studio](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. เพื่อติดตั้งแพ็กเกจ Qiskit และ dependencies ที่จำเป็น ให้เปิด notebook ใหม่และรันคำสั่งต่อไปนี้:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [สร้างบัญชี Google](https://support.google.com/accounts/answer/27441?hl)
2. ไปที่ [หน้า Google Colab](https://colab.research.google.com/) และลงชื่อเข้าใช้ด้วยบัญชี Google ของคุณ
3. [สร้าง Colab notebook](https://colab.research.google.com/#create=true) และติดตั้ง Qiskit โดยรันคำสั่งต่อไปนี้:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [สร้างบัญชี Azure](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [สร้าง workspace](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [สร้าง Jupyter notebook ใน workspace ของคุณ](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. จาก notebook รันคำสั่งนี้เพื่อติดตั้ง Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
