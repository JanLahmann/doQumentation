# อัปเกรดจาก Open Plan {#upgrade-from-the-open-plan}

คู่มือนี้สำหรับองค์กรที่ต้องการอัปเกรดจาก IBM Quantum&reg; Open Plan ไปยัง IBM Cloud&reg; Pay-As-You-Go Plan สำหรับขั้นตอนการตั้งค่าบัญชีคลาวด์ ตั้งค่า service instance และจัดการผู้ใช้ ทำตามขั้นตอนเหล่านี้

ถ้าคุณเป็นบุคคลที่ตั้งค่าบริการเป็นครั้งแรก หรือถ้าคุณได้รับเชิญเข้า instance โดย administrator ให้ทำตามขั้นตอนใน [Install Qiskit](./install-qiskit) แทน

## ก่อนเริ่มต้น {#before-you-begin}

หากจำเป็น ทำตามขั้นตอนในส่วน [Set up to use IBM Cloud](./cloud-setup#cloud) เพื่อดำเนินการดังต่อไปนี้:

- สมัครบัญชี IBM Cloud
- สร้าง IBM Quantum instance

<span id="update-cloud-account"></span>
## อัปเกรดบัญชี IBM Cloud ของคุณ {#upgrade-your-ibm-cloud-account}

ข้ามส่วนนี้ถ้าคุณมีบัญชี IBM Cloud Pay-as-you-go อยู่แล้ว บัญชี IBM Cloud ใหม่ทั้งหมดถูกตั้งค่าเป็น Pay-as-you-go

คุณต้องอัปเกรดบัญชี IBM Cloud ในกรณีต่อไปนี้:

- คุณมีบัญชีแบบ Trial
- คุณมีบัญชีแบบ Lite (เลิกใช้แล้ว) ที่สร้างก่อน 25 ตุลาคม 2021

หลังจากอัปเกรด คุณสามารถใช้ instance ที่สร้างไว้กับบัญชี Lite ต่อไปได้ หากต้องการอัปเกรดเป็นบัญชี Pay-As-You-Go ทำตามขั้นตอนต่อไปนี้:

1. ไปที่ Manage > [Account](https://cloud.ibm.com/account) ใน IBM Cloud console
2. เลือก **Account settings** และคลิก **Add credit card**
3. ใส่ข้อมูลการชำระเงินและคลิก **Next** เพื่อส่งข้อมูล
4. หลังจากข้อมูลการชำระเงินถูกประมวลผล บัญชีของคุณจะถูกอัปเกรด
   <Admonition type="note">
   คุณจะไม่ถูกเรียกเก็บเงินเว้นแต่คุณสร้าง Pay-As-You-Go Plan instance และใช้เพื่อส่ง workload ไปยัง QPU
   </Admonition>

## เข้าถึง Pay-As-You-Go Plan ของคุณ {#access-your-pay-as-you-go-plan}

หากต้องการใช้ IBM Cloud Pay-As-You-Go Plan สร้าง [instance](/guides/instances) ใหม่และกำหนดให้กับ Pay-As-You-Go Plan  <ins>งานทั้งหมดที่ส่งไปยัง QPU โดยระบุ instance นี้จะมีค่าใช้จ่าย</ins>  หากต้องการใช้ QPU โดยไม่มีค่าใช้จ่าย คุณต้องใช้ instance ที่กำหนดให้กับ Open plan หรือ plan ฟรีอื่น ถ้ามี

แนะนำให้ตั้งค่า [cost limit](/guides/manage-cost#cost-limit) บน instance และเรียนรู้วิธีอื่นในการจัดการค่าใช้จ่ายเมื่อใช้ Pay-As-You-Go Plan
