# เหตุการณ์ติดตามกิจกรรมสำหรับ Qiskit Runtime {#activity-tracking-events-for-qiskit-runtime}

บริการ IBM Cloud&reg; อย่าง Qiskit Runtime จะสร้างเหตุการณ์ติดตามกิจกรรม

เหตุการณ์ติดตามกิจกรรมจะรายงานกิจกรรมที่เปลี่ยนแปลงสถานะของบริการใน IBM Cloud คุณสามารถใช้เหตุการณ์เหล่านี้เพื่อตรวจสอบกิจกรรมผิดปกติ การดำเนินการที่สำคัญ และเพื่อให้สอดคล้องกับข้อกำหนดการตรวจสอบด้านกฎระเบียบ

คุณสามารถใช้ IBM Cloud Activity Tracker Event Routing ซึ่งเป็นบริการแพลตฟอร์ม เพื่อนำทางเหตุการณ์การตรวจสอบในบัญชีของคุณไปยังปลายทางที่เลือกได้ โดยการกำหนดค่า targets และ routes ที่ระบุว่าจะส่งเหตุการณ์ติดตามกิจกรรมไปที่ใด สำหรับข้อมูลเพิ่มเติม ดูที่ [About IBM Cloud Activity Tracker Events Routing](https://cloud.ibm.com/docs/atracker?topic=atracker-about)

คุณสามารถใช้ IBM Cloud Logs เพื่อแสดงภาพและแจ้งเตือนสำหรับเหตุการณ์ที่สร้างขึ้นในบัญชีของคุณและถูกนำทางโดย IBM Cloud Activity Tracker Event Routing ไปยัง IBM Cloud Logs instance

## ตำแหน่งที่สร้างเหตุการณ์ติดตามกิจกรรม {#locations-where-activity-tracking-events-are-generated}

Qiskit Runtime ส่งเหตุการณ์ติดตามกิจกรรมในภูมิภาคต่อไปนี้:

- Washington ("us-east")
- Frankfurt ("eu-de")

## การดูเหตุการณ์ติดตามกิจกรรมสำหรับ Qiskit Runtime {#viewing-activity-tracking-events-for-qiskit-runtime}

คุณสามารถใช้ IBM Cloud Logs เพื่อแสดงภาพและแจ้งเตือนสำหรับเหตุการณ์ที่สร้างขึ้นในบัญชีของคุณและถูกนำทางโดย IBM Cloud Activity Tracker Event Routing ไปยัง IBM Cloud Logs instance
