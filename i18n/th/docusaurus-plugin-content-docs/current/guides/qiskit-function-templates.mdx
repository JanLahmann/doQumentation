# บทนำสู่ Qiskit Function templates {#introduction-to-qiskit-function-templates}

[Qiskit Function templates](https://github.com/qiskit-community/qiskit-function-templates) คือชุดตัวอย่างโค้ดที่สมจริงแบบโอเพนซอร์ส ซึ่งให้เข้าถึงเวิร์กโฟลว์งานวิจัยล่าสุดที่ใช้เครื่องมืออย่าง Qiskit SDK และ [Qiskit addons](/guides/addons) เวิร์กโฟลว์สำเร็จรูปเหล่านี้นำข้อมูลจากการทดลองจริงมาบรรจุในรูปแบบที่คุณปรับแต่งและนำกลับมาใช้ในงานวิจัยของตัวเองได้ โดยไม่ต้องสร้างใหม่ตั้งแต่ต้น Template ทุกชิ้นพร้อมนำไปใช้งานบน [Qiskit Serverless](./serverless) ที่ช่วยให้ดึง classical compute มาใช้ในเวิร์กโฟลว์ได้อย่างง่ายดาย

Template มีสองประเภท:

- *Template implementations*: ตัวอย่างโค้ดเฉพาะทางที่ใช้เทคนิคขั้นสูงจากการทดลองจริงในสาขาต่าง ๆ เช่น เคมีและฟิสิกส์
- *Base templates*: เอกสารอ้างอิงสำหรับพัฒนา Qiskit Function ต้นฉบับของตัวเอง Template เหล่านี้เน้นแนวทางปฏิบัติที่ดีในด้านการออกแบบ interface, การจัดรูปแบบโค้ด, การทดสอบ unit และอื่น ๆ

## Template implementations {#template-implementations}

Qiskit Function template implementations จัดตามสาขาการประยุกต์ใช้งาน ปัจจุบันในชุดนี้มี physics template สำหรับ Hamiltonian simulation ด้วย [AQC-Tensor Qiskit addon](./qiskit-addons-aqc) และ chemistry template สำหรับโครงสร้างอิเล็กทรอนิกส์พร้อม implicit solvent model ด้วย [SQD Qiskit addon](./qiskit-addons-sqd)
แหล่งข้อมูลสำหรับเริ่มใช้งาน template ทั้งสองมีอยู่ที่ลิงก์ด้านล่าง:

- Electronic structure simulation พร้อม implicit solvent model: [ไฟล์ต้นฉบับ template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/chemistry/sqd_pcm) และ [คู่มือ](./function-template-chemistry-workflow)
- Hamiltonian simulation: [ไฟล์ต้นฉบับ template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/physics/hamiltonian_simulation) และ [คู่มือ](./function-template-hamiltonian-simulation)

## Base templates {#base-templates}

*Base templates* คือเอกสารอ้างอิงสำหรับพัฒนา Qiskit Function ต้นฉบับของตัวเอง ออกแบบมาเพื่อช่วยให้เริ่มกระบวนการพัฒนาได้เร็วขึ้น โดยแสดงแนวทางปฏิบัติที่ดีในด้านการออกแบบ interface, การจัดรูปแบบโค้ด, การจัดการ I/O, ข้อความแจ้งข้อผิดพลาด, checkpoints, การทดสอบ unit และอื่น ๆ repository เปิดเป็น [GitHub template](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-template-repository) เพื่อให้ขยายการใช้งาน CI/CD ได้อย่างสะดวก

ปัจจุบันมีสอง template: [circuit function template](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/circuit_function_template.py) และ [application function template](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/application_function_template.py) Circuit function template สร้างในลักษณะเดียวกับ [IBM Circuit Function](./ibm-circuit-function) ส่วน application function template แสดงวิธีจัดโครงสร้างเวิร์กโฟลว์ application ทั่วไป
