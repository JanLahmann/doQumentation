# ติดตั้ง Qiskit Runtime client จาก source {#install-the-qiskit-runtime-client-from-source}

เมื่อติดตั้ง Qiskit Runtime จาก source คุณสามารถเข้าถึงเวอร์ชัน development ปัจจุบันแทนที่จะใช้เวอร์ชันใน Python Package Index (PyPI) repository ดังนั้น คุณสามารถตรวจสอบและขยายโค้ดเวอร์ชันล่าสุดได้อย่างมีประสิทธิภาพมากขึ้น

1. ไปยัง directory โครงการของคุณและสร้างสภาพแวดล้อมขั้นต่ำที่มีเฉพาะ Python ติดตั้งอยู่

## ติดตั้ง Qiskit Runtime {#install-qiskit-runtime}

ทำตามขั้นตอนเหล่านี้เพื่อติดตั้ง Qiskit Runtime:

1. Clone Qiskit Runtime repository

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. เปลี่ยนไปยัง directory `qiskit-ibm-runtime`

```bash
cd qiskit-ibm-runtime
```

3. ติดตั้ง `qiskit-runtime` แนะนำให้ใช้ [virtual environment](https://docs.python.org/3/library/venv.html) เพื่อหลีกเลี่ยงการรบกวน Python installation ทั่วโลกของคุณ

  * **การติดตั้งแบบมาตรฐาน**:

    ```bash
    pip install .
    ```

  * **Editable mode**: ในโหมดนี้ คุณไม่ต้องติดตั้ง Qiskit ใหม่เมื่อมีการเปลี่ยนแปลงโค้ดในโครงการ

    ```bash
    pip install -e .
    ```

    ในโหมด editable, compiled extensions จะถูก build ใน _debug mode_ โดยไม่มีการ optimize

4. (ทางเลือก) หากต้องการรัน tests หรือ linting checks ให้ติดตั้ง developer requirements แนะนำให้ใช้ [virtual environment](https://docs.python.org/3/library/venv.html) เพื่อหลีกเลี่ยงการรบกวน Python installation ทั่วโลกของคุณ

```bash
pip install -e ".[dev]"
```
