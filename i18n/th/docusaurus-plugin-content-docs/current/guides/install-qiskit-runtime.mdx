# ติดตั้ง Qiskit Runtime client {#install-the-qiskit-runtime-client}

Qiskit Runtime คือบริการบนคลาวด์สำหรับการรันการคำนวณเชิงควอนตัมบนฮาร์ดแวร์ IBM Quantum&reg; คู่มือนี้แสดงวิธีติดตั้ง `qiskit-ibm-runtime` ซึ่งเป็น client package สำหรับเข้าถึงบริการ Qiskit Runtime

ก่อนติดตั้ง Qiskit Runtime client ให้ทำตามคำแนะนำในคู่มือเหล่านี้:

1. [ติดตั้ง Qiskit](/guides/install-qiskit)
2. [ตั้งค่าบัญชี IBM Cloud](cloud-setup)

รันคำสั่งนี้เพื่อติดตั้ง Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="อัปเดตให้ทันสมัยอยู่เสมอ">
ตรวจสอบ [Qiskit Runtime release notes](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) เป็นระยะเพื่อดูการเผยแพร่ใหม่ แนะนำให้อัปเดต requirements สำหรับ `qiskit-ibm-runtime` บ่อยๆ โดยรัน `pip install -U qiskit-ibm-runtime`
</Admonition>
