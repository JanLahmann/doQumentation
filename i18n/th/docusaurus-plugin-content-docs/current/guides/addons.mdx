# Advanced techniques - Qiskit addons {#advanced-techniques-qiskit-addons}

Qiskit addons คือชุดความสามารถด้านการวิจัยสำหรับการค้นพบ algorithm ในระดับ utility scale โดย modular software components เหล่านี้ต่อยอดจากรากฐานที่มีประสิทธิภาพของ Qiskit และเสียบเข้ากับ [workflow](/guides/intro-to-patterns) เพื่อขยายขนาดหรือออกแบบ quantum algorithms ใหม่ หน้านี้จะเน้นเครื่องมือที่มีให้ใช้งานในหมวดการทำงานหลักต่าง ๆ เพื่อช่วยให้คุณเลือกความสามารถที่เกี่ยวข้องเมื่อสร้าง workflows ของคุณ
