# Qiskit addons

Qiskit addons คือชุดความสามารถด้านการวิจัยสำหรับการค้นพบ algorithm ในระดับ utility scale ความสามารถเหล่านี้ต่อยอดจากรากฐานที่มีประสิทธิภาพของ Qiskit สำหรับการสร้างและรัน quantum algorithms โดยมีให้ใช้งานเป็น modular software components ที่เสียบเข้ากับ [workflow](/guides/intro-to-patterns) เพื่อขยายขนาดหรือออกแบบ quantum algorithms ใหม่

## Addons สำหรับการ mapping {#addons-for-mapping}

### การคอมไพล์ quantum แบบประมาณค่าด้วย tensor networks {#approximate-quantum-compilation-with-tensor-networks}

Approximate quantum compilation with tensor networks (AQC-Tensor) ช่วยให้สร้าง Circuit ที่มีความเที่ยงตรงสูงพร้อมลดความลึกลง

- ดู [GitHub repository](https://github.com/Qiskit/qiskit-addon-aqc-tensor)
- อ่าน [เอกสาร](/guides/qiskit-addons-aqc)
- อ่าน tutorial เกี่ยวกับการใช้ AQC เพื่อ [ปรับปรุง Trotterized time evolution](/tutorials/approximate-quantum-compilation-for-time-evolution)

### Multi-product formulas {#multi-product-formulas}

Multi-product formulas (MPF) ลด Trotter error ของ Hamiltonian dynamics ผ่านการรวมแบบถ่วงน้ำหนักของการรัน Circuit หลายครั้ง

- ดู [GitHub repository](https://github.com/Qiskit/qiskit-addon-mpf)
- อ่าน [เอกสาร](https://qiskit.github.io/qiskit-addon-mpf)

### Optimization mapper {#optimization-mapper}

Optimization Mapper addon มีฟังก์ชันสำหรับสร้างโมเดลปัญหา optimization โดยกำหนดสูตรในโมเดล abstract จากนั้นแปลงเป็นรูปแบบที่คอมพิวเตอร์ควอนตัมเข้าใจได้

- อ่าน [เอกสาร](https://qiskit.github.io/qiskit-addon-opt-mapper/) สำหรับ how-to guides และ API references
- ดู [GitHub](https://github.com/qiskit/qiskit-addon-opt-mapper) repository

## Addons สำหรับการ optimizing {#addons-for-optimizing}

### Operator backpropagation {#operator-backpropagation}

Operator backpropagation (OBP) ลดความลึกของ Circuit โดยตัดการดำเนินการจากปลายออก แลกกับการวัด operator มากขึ้น

- ดู [GitHub repository](https://github.com/Qiskit/qiskit-addon-obp)
- อ่าน [เอกสาร](/guides/qiskit-addons-obp)
- อ่าน tutorial เกี่ยวกับการใช้ [OBP เพื่อปรับปรุง expectation values](/tutorials/operator-back-propagation)

### Circuit cutting {#circuit-cutting}

Circuit cutting ลดความลึกของ Circuit ที่ผ่านการ transpile โดยแยก entangling gates ระหว่าง qubits ที่ไม่ติดกัน

- ดู [GitHub repository](https://github.com/Qiskit/qiskit-addon-cutting)
- อ่าน [เอกสาร](https://qiskit.github.io/qiskit-addon-cutting)

## Addons สำหรับ post-processing {#addons-for-post-processing}

### Sample-based quantum diagonalization {#sample-based-quantum-diagonalization}

Sample-based quantum diagonalization (SQD) ประมวลผลตัวอย่าง quantum ที่มี noise แบบ classical เพื่อให้ได้การประมาณค่า eigenvalue ของ Hamiltonians ของระบบ quantum ที่แม่นยำกว่า เช่น ในแอปพลิเคชันเคมี

- ดู [GitHub repository](https://github.com/Qiskit/qiskit-addon-sqd)
- อ่าน [เอกสาร](/guides/qiskit-addons-sqd)
- อ่าน tutorial เกี่ยวกับ [Sample-based quantum diagonalization ของ chemistry Hamiltonian](/tutorials/sample-based-quantum-diagonalization)

### SQD for HPC {#sqd-for-hpc}

addon นี้เป็นการใช้งาน SQD addon ที่พร้อมรองรับ HPC เขียนด้วยมาตรฐาน C++17 สมัยใหม่ และออกแบบมาเพื่อสร้าง compiled binary เดียวสำหรับใช้กับ MPI

- ดู [API reference](https://qiskit.github.io/qiskit-addon-sqd-hpc/) documentation
- ดู [GitHub](https://github.com/qiskit/qiskit-addon-sqd-hpc) repository

## Addons สำหรับ error mitigation {#addons-for-error-mitigation}

### M3 (mthree) {#m3-mthree}

Matrix-free Measurement Mitigation (M3) คือ package สำหรับ quantum measurement error mitigation ที่ขยายขนาดได้และคำนวณแบบขนานได้

- ดู [เอกสาร](https://qiskit.github.io/qiskit-addon-mthree/) สำหรับ how-to guides, tutorials และ API references
- ดู [GitHub](https://github.com/Qiskit/qiskit-addon-mthree) ของ package

### Shaded lightcones {#shaded-lightcones}

Shaded lightcones addon ใช้ Pauli propagation เพื่อลดจำนวน error terms ใน noise model ที่ต้องลดทอน ตามข้อมูลเฉพาะของ observable เป้าหมาย ซึ่งมีผลลดค่าใช้จ่ายในการ sampling สำหรับ probabilistic error cancellation (PEC) workflows

- ดู [เอกสาร](https://qiskit.github.io/qiskit-addon-slc/) สำหรับ how-to guides, tutorials และ API references

### Propagated noise absorption {#propagated-noise-absorption}

Propagated noise absorption (PNA) ใช้ Pauli propagation เพื่อดูดซับข้อมูลจาก noise model เข้าสู่ observable เป้าหมาย การวัด observable ที่ปรับแล้วนี้มีผลลดทอน noise ตามที่แสดงโดยโมเดล

- ดู [เอกสาร](https://qiskit.github.io/qiskit-addon-pna/apidocs/qiskit_addon_pna.html) สำหรับ how-to guides, tutorials และ API references
