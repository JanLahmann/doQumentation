# Activity tracking events สำหรับ IBM Quantum System service {#activity-tracking-events-for-the-ibm-quantum-system-service}

บริการ IBM Cloud&reg; เช่น IBM&reg; Quantum System service จะสร้าง activity tracking events

Activity tracking events รายงานกิจกรรมที่เปลี่ยนแปลงสถานะของบริการใน IBM Cloud คุณสามารถใช้ events เหล่านี้เพื่อตรวจสอบกิจกรรมที่ผิดปกติ การดำเนินการที่สำคัญ และเพื่อให้สอดคล้องกับข้อกำหนดการตรวจสอบด้านกฎระเบียบ

คุณสามารถใช้ IBM Cloud Activity Tracker Event Routing ซึ่งเป็นบริการแพลตฟอร์ม เพื่อกำหนดเส้นทาง auditing events ในบัญชีของคุณไปยังปลายทางที่เลือกได้โดยการตั้งค่า targets และ routes ที่กำหนดว่า activity tracking events จะถูกส่งไปที่ใด สำหรับข้อมูลเพิ่มเติม ดูที่ [About IBM Cloud Activity Tracker Events Routing](https://cloud.ibm.com/docs/atracker?topic=atracker-about)

คุณสามารถใช้ IBM Cloud Logs เพื่อแสดงภาพและแจ้งเตือนสำหรับ events ที่สร้างขึ้นในบัญชีของคุณและกำหนดเส้นทางโดย IBM Cloud Activity Tracker Event Routing ไปยัง IBM Cloud Logs instance

## การดู activity tracking events สำหรับ IBM Quantum System service {#viewing-activity-tracking-events-for-the-ibm-quantum-system-service}

คุณสามารถใช้ IBM Cloud Logs เพื่อแสดงภาพและแจ้งเตือนสำหรับ events ที่สร้างขึ้นในบัญชีของคุณและกำหนดเส้นทางโดย IBM Cloud Activity Tracker Event Routing ไปยัง IBM Cloud Logs instance
