# ตั้งค่าบัญชี IBM Cloud - ผู้ใช้ที่ได้รับคำเชิญ {#set-up-your-ibm-cloud-account-invited-users}

ถ้ายังไม่ได้รับคำเชิญจากผู้ดูแลระบบให้เข้าร่วมบัญชีที่ผู้ดูแลระบบตั้งค่าไว้ ให้ติดต่อผู้ดูแลระบบ  หากต้องการตั้งค่าบัญชีเอง ให้ทำตามขั้นตอนใน [ตั้งค่าบัญชี IBM Cloud](/guides/cloud-setup)

สามารถรัน workloads บน IBM&reg; quantum processing units (QPUs) ได้โดยการตั้งค่าบัญชีบน IBM Cloud&reg;  นอกจากนี้ยังต้อง [ติดตั้งและตั้งค่า Qiskit และ Qiskit Runtime](/guides/install-qiskit#local) ด้วย

บัญชีผู้ใช้ผูกกับ [instances](./instances) หนึ่งรายการหรือมากกว่า ซึ่งระบุด้วย _Cloud Resource Names_ (CRNs) ที่ให้สิทธิ์การเข้าถึง IBM Quantum services  นอกจากนี้ API token เฉพาะ (เรียกอีกชื่อว่า key) จะถูกกำหนดให้กับแต่ละบัญชี ซึ่งอนุญาตให้เข้าถึง IBM Quantum จาก Qiskit ได้  สำหรับคำแนะนำการเลือก instance เฉพาะ ดู [Connect to an instance](./instances#connect-instance)

    <Admonition type="note">
      [IBM Quantum Platform dashboard](https://quantum.cloud.ibm.com) และ [Instances page](https://quantum.cloud.ibm.com/instances) แสดงรายการ instances ที่สามารถเข้าถึงได้
    </Admonition>

<span id="cloud"></span>
## ตั้งค่าเพื่อใช้งาน IBM Cloud {#set-up-to-use-ibm-cloud}

### ก่อนเริ่มต้น {#before-you-begin}

1. ยอมรับคำเชิญทางอีเมล
1. เข้าสู่ระบบด้วย IBMid หรือบัญชี Google  โปรดทราบว่าโปรไฟล์ที่ใช้เข้าสู่ระบบ IBM Cloud หรือ IBM Quantum Platform ในขั้นตอนนี้ไม่ใช่บัญชี IBM Cloud  หลังจากเข้าสู่ระบบ ให้ยืนยันข้อมูลและยอมรับคำเชิญเข้าร่วมบัญชี
1. กลับไปที่ [IBM Quantum Platform](https://quantum.cloud.ibm.com)
1. ตรวจสอบให้แน่ใจว่าเลือกบัญชีและ region ที่ถูกต้องใน account switcher ในส่วนหัว ดังที่แสดงในภาพต่อไปนี้
    * เลือกบัญชีที่ได้รับคำเชิญในอีเมล
    * Region ควบคุมว่า jobs จะรันที่ไหนและข้อมูล job จะถูกเก็บไว้ที่ไหน  สามารถเข้าถึง region ใดก็ได้โดยใช้ API key เดียวกัน แต่จะมองเห็นและเข้าถึงได้เฉพาะ instances ที่สร้างใน region ที่กำลังเข้าสู่ระบบอยู่เท่านั้น

    ![แสดงส่วนหัวของ IBM Quantum Platform  account switcher อยู่ทางขวาของแถบค้นหา  region switcher อยู่ทางขวาของ account switcher](/docs/images/guides/cloud-setup/IQP-Header.svg 'ส่วนหัว IBM Quantum Platform')

### 2. ไม่บังคับ: บันทึก access credentials {#2-optional-save-your-access-credentials}

ถ้าใช้งานใน Python environment ที่เชื่อถือได้ เช่น แล็ปท็อปส่วนตัว แนะนำให้ใช้ method `save_account()` เพื่อ [บันทึก credentials ไว้ในเครื่อง](/guides/save-credentials)

<span id="cloud-save"></span>
### 3. เชื่อมต่อ Qiskit กับ service instance {#3-connect-qiskit-with-your-service-instance}

- [เริ่มต้น service](/guides/initialize-account) ใน **Python environment ที่เชื่อถือได้**
- [เริ่มต้น service ใน **environment ที่ไม่เชื่อถือ**](/guides/cloud-setup-untrusted) (เช่น คอมพิวเตอร์สาธารณะ)

<Admonition type="note" title="หมายเหตุ">
   * ทำตาม [คำแนะนำเหล่านี้](/guides/cloud-setup-rest-api) หากต้องการเชื่อมต่อโดยใช้ REST API แทน Qiskit
   * หากจำเป็น ใช้ [ข้อมูลนี้](/guides/quickstart-steps-org#firewall) เพื่อกำหนดค่า firewall ให้เปิดใช้งานการเข้าถึง IBM Quantum API endpoints
