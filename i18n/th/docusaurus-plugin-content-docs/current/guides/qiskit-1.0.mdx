# คู่มือการย้ายไปใช้ Qiskit 1.0 {#qiskit-10-migration-guide}

Qiskit 1.0 มาพร้อมการรับประกันความเสถียรและการปรับปรุงประสิทธิภาพใหม่ แต่ก็ไม่รองรับการทำงานร่วมกับ Qiskit เวอร์ชันก่อนหน้าทั้งหมด ดูคู่มือต่อไปนี้เพื่ออัปเดต environment และโค้ดของคุณให้ใช้งานกับ Qiskit v1.0 ได้

- [ติดตั้งแพ็กเกจ Qiskit v1.0 ใหม่](/guides/qiskit-1.0-installation) สำหรับการอัปเกรด Python environment และ dependencies ของโปรเจกต์
- [การเปลี่ยนแปลงฟีเจอร์ใน Qiskit v1.0](/guides/qiskit-1.0-features) สำหรับการอัปเดตโค้ดของคุณ
- [การเปลี่ยนแปลงแพ็กเกจที่ทำให้ใช้งานร่วมกันไม่ได้ใน Qiskit v1.0](/guides/metapackage-migration) เพื่อทำความเข้าใจโครงสร้างแพ็กเกจใหม่และเก่า

อ่านเพิ่มเติมเกี่ยวกับประโยชน์ของ Qiskit v1.0 ได้ที่ [IBM&reg; blog](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary)
