# นโยบายความปลอดภัยของ Repository {#repository-security-policies}

การรักษาความปลอดภัยของระบบให้อยู่ในระดับสูงสุดเป็นเรื่องสำคัญมาก ดังนั้น แนะนำอย่างยิ่งให้ใช้เครื่องมือด้านความปลอดภัยอย่าง [Dependabot](https://github.com/dependabot) ที่จะแจ้งเตือนโดยอัตโนมัติเมื่อ dependencies ล้าสมัย ดูนโยบายความปลอดภัยสำหรับ GitHub repositories ที่เกี่ยวข้องกับ IBM Quantum&reg; ได้จากลิงก์ต่อไปนี้:

- [Qiskit](https://github.com/Qiskit/qiskit/security)
- [Qiskit IBM Runtime](https://github.com/Qiskit/qiskit-ibm-runtime/security)
- [Circuit cutting Qiskit addon](https://github.com/Qiskit/qiskit-addon-cutting/security)
- [Fermionic mapper Qiskit addon](https://github.com/Qiskit/qiskit-fermions/blob/main/SECURITY.md)
- [Multi-product formulas (MPF) Qiskit addon](https://github.com/Qiskit/qiskit-addon-mpf/security)
- [Operator backpropagation (OBP) Qiskit addon](https://github.com/Qiskit/qiskit-addon-obp/security)
- [Optimization mapper Qiskit addon](https://github.com/Qiskit/qiskit-addon-opt-mapper/blob/main/SECURITY.md)
- [Pauli propagation Qiskit addon](https://github.com/Qiskit/pauli-prop/blob/main/SECURITY.md)
- [Propagated noise absorption (PNA) Qiskit addon](https://github.com/Qiskit/qiskit-addon-pna/blob/main/SECURITY.md)
- [Qiskit addon utilities](https://github.com/Qiskit/qiskit-addon-utils/security)
- [Sample-based quantum diagonalization (SQD) Qiskit addon](https://github.com/Qiskit/qiskit-addon-sqd/security)
- [Shaded lightcones Qiskit addon](https://github.com/Qiskit/qiskit-addon-slc/blob/main/SECURITY.md)
