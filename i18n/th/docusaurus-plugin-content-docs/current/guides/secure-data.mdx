# การรักษาความปลอดภัยของข้อมูล {#securing-your-data}

เพื่อให้มั่นใจว่าสามารถจัดการข้อมูลได้อย่างปลอดภัยเมื่อใช้งาน IBM Quantum&reg; Platform จึงเป็นเรื่องสำคัญที่จะต้องทราบว่าข้อมูลใดบ้างที่ถูกจัดเก็บและเข้ารหัส และจะลบข้อมูลที่จัดเก็บไว้ได้อย่างไร

## การปกป้องข้อมูลที่มีความอ่อนไหว {#protecting-your-sensitive-data}

ข้อมูลที่จัดเก็บไว้ใน IBM Cloud&reg; จะถูกเข้ารหัสขณะพักโดยใช้คีย์ที่สร้างขึ้นแบบสุ่ม

## การลบข้อมูล {#deleting-your-data}

การลบ service instance จะลบเนื้อหาทั้งหมดที่เกี่ยวข้องกับ instance นั้น เช่น Job ผลลัพธ์ พารามิเตอร์ และโปรแกรมต่าง ๆ หากต้องการลบ instance ให้ไปที่ [หน้า Instances](https://quantum.cloud.ibm.com/instances) ค้นหา instance ที่ต้องการลบ คลิกเมนู overflow แล้วคลิก **Delete** ระบบจะขอให้ยืนยันการลบ

หากมีบทบาท Manager ยังสามารถลบ Job แต่ละรายการพร้อมข้อมูลได้ด้วย ดูรายละเอียดเพิ่มเติมเกี่ยวกับนโยบายการเข้าถึง กลุ่ม และบทบาทได้ใน[เอกสาร](/guides/access-groups)

## นโยบายการเก็บรักษาข้อมูล {#data-retention-policy}

IBM จะเก็บข้อมูล (ที่ยังไม่ได้ลบ) ไว้เป็นเวลาสามปีในขณะที่เป็นผู้ใช้งานบริการอยู่ หากต้องการทราบข้อมูลว่า IBM จะเก็บข้อมูลไว้นานเท่าใดหลังจากหยุดใช้บริการ ให้ดูที่ [Qiskit Runtime data sheet](https://www.ibm.com/support/customer/csol/terms/?id=BB6EE1D06BDE11EBA199CCFE56FA6D9D) ซึ่งอยู่ใน IBM Terms site
