# ภาพรวมของแผนบริการที่มีให้เลือก {#overview-of-available-plans}

IBM Quantum Platform มีแผนบริการแบบ Open (เข้าถึงฟรี) และแผนแบบเสียค่าใช้จ่าย ตามที่อธิบายไว้ด้านล่าง ดูรายละเอียดทั้งหมดได้ที่ [IBM Quantum access plans](https://www.ibm.com/quantum/pricing)

ถ้าต้องการเข้าถึงหลายแผนพร้อมกัน สามารถสร้างหลาย instance และกำหนดแผนต่างกันให้แต่ละ instance ได้ ตัวอย่างเช่น มี Open Plan instance และ Pay-As-You-Go Plan instance ไว้พร้อมกัน สามารถตั้ง instance ที่ต้องการใช้เป็นค่าเริ่มต้นใน `QiskitRuntimeService` initializer หรือเมธอด `QiskitRuntimeService.save_account()` ได้ ตัวอย่าง:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Open Plan {#open-plan}

รันวงจรควอนตัมบน QPU ที่ดีที่สุดในโลกได้ฟรี (สูงสุด 10 นาทีต่อช่วง 28 วันแบบ rolling window) ดูการใช้งานได้ที่หน้า [Platform dashboard](https://quantum.cloud.ibm.com/) และหน้า [Workloads](https://quantum.cloud.ibm.com/workloads) Open Plan ให้บริการเฉพาะในภูมิภาค us-east เท่านั้น

แผนนี้แนะนำสำหรับผู้ที่กำลังเรียนรู้การคำนวณเชิงควอนตัมและต้องการสำรวจเทคโนโลยีควอนตัมของ IBM

## Flex Plan {#flex-plan}

ซื้อจำนวนนาทีล่วงหน้า (อย่างน้อย 400 นาที) สำหรับรัน quantum jobs โดยต้องใช้ภายในหนึ่งปีนับจากวันที่ซื้อ สามารถซื้อเพิ่มได้ทุกเมื่อ

หลังจากซื้อเวลาแล้ว ต้องจัดสรรนาทีให้กับ instance ที่มีอยู่หรือสร้างใหม่บน Flex Plan สำหรับบัญชีของตน ตั้งหรืออัปเดตขีดจำกัดการจัดสรรได้ที่ [หน้า Instances](https://quantum.cloud.ibm.com/instances) หรือตั้งค่าการจัดสรรตอน[สร้าง Flex Plan instance ใหม่](/guides/instances) ถ้าจัดสรรนาทีทั้งหมดให้ instance ที่มีอยู่แล้ว จะไม่สามารถสร้าง instance ใหม่สำหรับ Flex Plan ได้จนกว่าจะซื้อนาทีเพิ่มหรือนำนาทีออกจาก instance อื่น ดู [Set instance allocation limits](allocation-limits) เพื่อเรียนรู้เพิ่มเติม

ถ้าต้องการมากกว่า 10,000 นาที ควรพิจารณาใช้ Premium Plan subscription แทน ถ้าไม่ต้องการซื้อ 400 นาที ให้พิจารณา Pay-As-You-Go Plan แทน

[ติดต่อ IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) เพื่อสมัคร Flex Plan

## Pay-As-You-Go Plan {#pay-as-you-go-plan}

รัน quantum circuits บน QPU ที่ดีที่สุดในโลก และจ่ายเฉพาะเวลาควอนตัมที่ใช้จริง

แผนนี้แนะนำสำหรับผู้ที่ทำวิจัย quantum utility และทดสอบโมเดลธุรกิจที่ต้องการความยืดหยุ่น ถ้าคาดว่าจะใช้อย่างน้อย 400 นาทีในปีหน้า แนะนำให้ใช้ Flex Plan แทน

## Premium Plan {#premium-plan}

รัน quantum circuits บน QPU ที่ดีที่สุดในโลกด้วย enterprise quantum time subscription นอกจากนี้ ฟีเจอร์บางอย่าง เช่น [Qiskit Functions](functions) และ [Qiskit Transpiler as a Service](qiskit-transpiler-service) ก็ให้บริการแก่ผู้ใช้ Premium Plan ด้วย

แผนนี้แนะนำสำหรับผู้ที่ต้องการดำเนิน quantum roadmap เชิงกลยุทธ์และพัฒนาอัลกอริทึมและแอปพลิเคชันควอนตัมในระดับขนาดใหญ่ [ติดต่อ IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) เพื่อเรียนรู้เพิ่มเติม

## On-Prem Plan {#on-prem-plan}

เข้าถึงระบบควอนตัมแบบ on-premises ที่มอบให้โดยเฉพาะ ซึ่งได้รับการดูแลและบำรุงรักษาโดย IBM Quantum

บริการนี้แนะนำสำหรับผู้ที่สำรวจอัลกอริทึมและแอปพลิเคชันควอนตัมแบบ classical-quantum และต้องการควบคุมทรัพยากรและข้อมูลสูง [ติดต่อ IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) เพื่อเรียนรู้เพิ่มเติม
