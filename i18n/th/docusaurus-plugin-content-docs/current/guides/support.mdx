# การสนับสนุน {#support}

<span id="get-help-with-qiskit-and-ibm-quantum-platform"></span>
## ขอความช่วยเหลือเกี่ยวกับ IBM Quantum Platform และ Qiskit {#get-help-with-qiskit-and-ibm-quantum-platform}

**Premium Plan, On-Prem Plan, Direct Access, Flex Plan, Pay-As-You-Go Plan, โปรแกรมพิเศษ และ Open Plan (ยกเว้นบัญชีทดลองใช้):**

คำร้องขอการสนับสนุนทั้งหมดต้องส่งผ่าน [การสร้างเคสใน IBM Cloud&reg; Support Center](https://cloud.ibm.com/unifiedsupport/cases/form) และเลือก 'Qiskit Runtime' ในช่อง Topic ตรวจสอบให้แน่ใจว่าได้เลือกบัญชี IBM Cloud ที่ถูกต้อง (บัญชี cloud ขององค์กร หรือบัญชีที่ไม่ใช่บัญชีทดลองใช้) ในตัวสลับบัญชี

ติดต่อเราได้สำหรับทุกเรื่องทางเทคนิค ไม่ว่าจะเป็นปัญหาแพลตฟอร์ม ข้อผิดพลาดของ job คำแนะนำ Qiskit หรือคำถามเกี่ยวกับการคำนวณควอนตัมในวงกว้าง ทีมของเราประกอบด้วยผู้เชี่ยวชาญด้านเทคนิคที่พร้อมให้ความช่วยเหลือในทุกหัวข้อเหล่านี้

**การสนับสนุนจากชุมชน:**

มีบัญชีทดลองใช้หรือกำลังมองหาตัวเลือกการสนับสนุนอื่น ๆ? [เข้าร่วมชุมชน Qiskit Slack](https://qisk.it/join-slack) เพื่อรับความช่วยเหลือจากชุมชนผู้ใช้และผู้เชี่ยวชาญจำนวนมาก ใน Slack จะพบช่องหัวข้อต่าง ๆ เช่น #qiskit-runtime, #ibm-quantum-platform, #qiskit-transpiler, #qiskit-ecosystem และอื่น ๆ อีกมากมาย

นอกจากนี้ยังสามารถใช้ [Stack Exchange](https://quantumcomputing.stackexchange.com/?tags=qiskit) เพื่อรับการสนับสนุนจากชุมชนในวงกว้างเกี่ยวกับคำถามด้านการคำนวณควอนตัมและการเขียนโปรแกรม

## มีปัญหาในการสร้างบัญชีหรือเข้าสู่ระบบ? {#having-trouble-creating-an-account-or-logging-in}

ถ้าได้รับข้อผิดพลาดเมื่อสร้างบัญชี IBM Cloud&reg; ให้ตรวจสอบก่อนว่ามีสิทธิ์เข้าถึง [Support Center](https://cloud.ibm.com/unifiedsupport/cases/form) บน IBM Cloud หรือไม่

- ถ้ามีสิทธิ์เข้าถึง ให้คลิก "create case" จากนั้นเลือก *Account* เป็น Topic และ *Account verification* เป็น Subtopic
- ถ้าไม่มีสิทธิ์เข้าถึง ให้เปิดเคสในหน้า [Create an Account, Login or Billing Request](https://watson.service-now.com/x_ibmwc_open_case_app.do#!/create)

ถ้าได้รับข้อผิดพลาดขณะพยายามสร้างบัญชีและประเทศอยู่ในรายการ [Personal use availability](https://cloud.ibm.com/docs/account?topic=account-account-getting-started#signup-personalaccts) ให้ลองตัวเลือกต่อไปนี้:

1. สามารถใช้ทรัพยากร [Documentation](/docs) และ [Learning](/learning) ได้โดยไม่ต้องมีบัญชี
2. สร้าง [บัญชี SkillsBuild](https://www.ibm.com/academic/) ถ้าเป็นนักศึกษามหาวิทยาลัย
3. [สร้างบัญชีทดลองใช้](https://quantum.cloud.ibm.com/registration) เพื่อเข้าถึงฟรี 30 วัน (โปรดทราบว่าอาจสูญเสียสิทธิ์การเข้าถึงเมื่อสิ้นสุดช่วงทดลองใช้)

<Admonition type="caution" title="หมายเหตุ">
ถ้าเป็น **เจ้าของ** บัญชี Pay-As-You-Go หรือ Subscription จะต้องสร้างบัญชี IBM Cloud และให้ข้อมูลการชำระเงินเพื่อยืนยันตัวตน

ถ้า **ได้รับการเชิญ** ให้เข้าร่วมบัญชี เพียงยอมรับการเชิญทางอีเมลและทำตาม [ขั้นตอนที่กำหนด](/guides/cloud-setup-invited) เพื่อเข้าถึงโดยไม่ต้องให้ข้อมูลการชำระเงิน

**เจ้าของบัญชีทดลองใช้** สามารถสร้าง Open Plan instance ได้ฟรี 30 วันโดยไม่ต้องให้ข้อมูลการชำระเงิน เมื่อสิ้นสุดช่วงทดลองใช้ ผู้ใช้ต้องให้ข้อมูลการชำระเงินเพื่อยืนยันและใช้งานบัญชีต่อ ผู้ใช้ *จะไม่ถูกเรียกเก็บเงิน* ถ้าใช้งาน Open Plan ฟรีต่อไป แต่สามารถเลือกอัปเกรดเป็นแผนชำระเงินอื่น ๆ ได้ในขณะนั้น

ดูข้อมูลเพิ่มเติมเกี่ยวกับบัญชี IBM Cloud ได้ที่หน้า [IBM Cloud account FAQs](https://cloud.ibm.com/docs/account?topic=account-accountfaqs)
</Admonition>
