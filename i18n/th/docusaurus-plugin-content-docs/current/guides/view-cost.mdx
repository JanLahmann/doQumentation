# ดูค่าใช้จ่าย {#view-the-cost}

คุณสามารถดูค่าใช้จ่ายที่เกิดขึ้นจาก workload ที่เชื่อมกับ instance ได้ตลอดเวลา

## ดูค่าใช้จ่ายของ Instance {#view-instance-cost}

หากต้องการทราบว่า instance ถูกใช้งานไปเท่าไร จาก [หน้า Instances](https://quantum.cloud.ibm.com/instances) คลิก instance เพื่อเปิดหน้ารายละเอียด คุณสามารถคลิกลิงก์ที่เชื่อมกับ **Billing overview** สำหรับรายละเอียดค่าใช้จ่ายและการเรียกเก็บเงินเต็มรูปแบบ

**Instance usage** รายงานค่าที่แตกต่างกัน ขึ้นอยู่กับประเภทของ plan ที่ instance เชื่อมอยู่:

- สำหรับ Premium และ Open plan นี่คือการใช้งานแบบ real-time สำหรับ 28 วันล่าสุด (แบบ rolling) จนถึงเวลาปัจจุบันของวันนี้
- สำหรับ Pay-As-You-Go plan นี่คือการใช้งานทั้งหมดตั้งแต่เวลาที่ instance ถูกสร้าง
- สำหรับ Flex plan นี่คือการใช้งานตั้งแต่ต้นสัญญา

## ดูการใช้งาน Workload {#view-workload-usage}

หากต้องการทราบว่า workload เฉพาะเจาะจงใช้ไปเท่าไร เปิด [หน้า Workloads](https://quantum.cloud.ibm.com/workloads) และดูคอลัมน์ "Usage" สำหรับ workload นั้น
