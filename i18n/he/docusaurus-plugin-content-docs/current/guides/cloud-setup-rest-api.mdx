# הגדרת שימוש ב-IBM Quantum Platform עם REST API {#set-up-to-use-ibm-quantum-platform-with-rest-api}

ניתן לגשת למעבדים קוונטיים באמצעות REST APIs, מה שמאפשר לך לעבוד עם QPUs בכל שפת תכנות או מסגרת עבודה.

### 1. קבלת גישה {#1-get-access}

1. אם עדיין אין לך חשבון משתמש, צור אחד ב[דף ההתחברות של IBM Quantum.](https://quantum.cloud.ibm.com/login)
1. צור מפתח API (הנקרא גם טוקן) ב[לוח הבקרה.](https://quantum.cloud.ibm.com) שים לב שניתן להשתמש באותו מפתח API עבור כל אחד מהאזורים.
1. צור טוקן bearer של IBM Cloud Identity and Access Management‏ (IAM). זהו טוקן קצר-מועד המשמש לאימות בקשות ל-REST API. כדי ליצור אותו, קרא ל-[IAM Identity Services API](https://cloud.ibm.com/apidocs/iam-identity-token-api#create-api-key) כפי שמוצג בדוגמת הבקשה הבאה:
<Tabs>

   <TabItem value="Curl" label="Curl">
   ```bash
   curl -X POST 'https://iam.cloud.ibm.com/identity/token' \
   -H 'Content-Type: application/x-www-form-urlencoded' \
   -d 'grant_type=urn:ibm:params:oauth:grant-type:apikey&apikey=MY_APIKEY'
   ```
   תגובה צפויה
   ```text
   {
      "access_token": "eyJhbGciOiJIUz......sgrKIi8hdFs",
      "refresh_token": "SPrXw5tBE3......KBQ+luWQVY=",
      "token_type": "Bearer",
      "expires_in": 3600,
      "expiration": 1473188353
   }
   ```
   </TabItem>

   <TabItem value="Python" label="Python">
   ```python
   # Use 'service' to invoke operations.
   import requests
   import json

   url = 'https://iam.cloud.ibm.com/identity/token'
   api_key = 'MY_APIKEY'
   headers = {
    'Content-Type': 'application/x-www-form-urlendcoded',
   }
   data = f'grant_type=urn:ibm:params:oauth:grant-type:apikey&apikey={api_key}'
   response = requests.post(url, headers=headers, data=data)

   # Bearer token to authorize requests to the REST API
   bearer_token = response.json()['access_token']
   ```
   </TabItem>
</Tabs>

### 2. בחירת שיטת אימות {#2-choose-an-authentication-method}

בחר את שיטת האימות המתאימה, בהתאם לסביבת העבודה שלך:

* [צור משתנה סביבה עבור מפתח ה-API שלך (סביבות Python מהימנות)](#environment_variable)
* [השתמש במפתח ה-API שלך ישירות (סביבה לא מהימנה)](/guides/cloud-setup-untrusted#rest-untrusted)

<span id="environment_variable"></span>
#### יצירת משתנה סביבה (סביבה מהימנה) {#create-an-environment-variable-trusted-environment}

1. כדי להגדיר את משתנה הסביבה IQP_API_TOKEN במערכת שלך, תוכל להוסיף את השורה הבאה לפרופיל ה-shell שלך (לדוגמה, .bashrc או .zshrc) או להגדיר אותו ישירות בטרמינל:

    ```shell
    export IQP_API_TOKEN=<your-API_KEY> # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard
    ```
    כשאתה מפעיל את משתנה הסביבה בקוד שלך, כלול `import os`, כמו בדוגמה הזו:

    ```python
    import os
    api_token = os.environ['IQP_API_TOKEN']
    ```
    שים לב שבעת יצירת משתנה סביבה, מפתח ה-API שלך עדיין מאוחסן באופן מקומי כטקסט רגיל, ויש לשמור עליו בזהירות.

1. אמת בקשות ל-Qiskit Runtime REST API על ידי הוספת ה-CRN וטוקן ה-bearer לכותרות הבקשה.
```bash
curl -X 'GET' \
   'https://quantum.cloud.ibm.com/api/v1/usage' \
   '-H accept: application/json' \
   '-H authorization: Bearer <BEARER_TOKEN>' \
   '-H Service-CRN: <INSTANCE_CRN>'
```
