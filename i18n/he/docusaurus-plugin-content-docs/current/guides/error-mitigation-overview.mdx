# סקירה של טכניקות ניהול רעש {#overview-of-noise-management-techniques}

בעת הרצת עומסי עבודה קוונטיים, יש מספר דרכים להפחית את השפעת הרעש. תוספי Qiskit בקוד-פתוח מספקים טכניקות הפחתת שגיאות ודיכוי שגיאות שמשתלבות ישירות בתהליך העבודה שלך, בעוד Qiskit Runtime מיישם אסטרטגיות מתקדמות של הפחתת שגיאות באופן אוטומטי כשעבודות מוגשות להרצה. דף זה מאנדקס את כל הכלים והתכונות הזמינות בשני האפשרויות כדי לעזור לך לבחור את הגישה הנכונה לניהול רעש בבניית עומסי עבודה קוונטיים.
