# צפייה בעלות {#view-the-cost}

בכל עת אפשר לראות כמה עלות נצברה עבור עומסי העבודה המשויכים ל-instance.

## צפייה בעלות של instance {#view-instance-cost}

כדי לבדוק כמה שימוש נעשה ב-instance, היכנסו ל[דף ה-Instances](https://quantum.cloud.ibm.com/instances) ולחצו על ה-instance כדי לפתוח את דף הפרטים שלו. תוכלו ללחוץ על הקישור **Billing overview** כדי לראות את פרטי העלות והחיוב המלאים.

**Instance usage** מציג ערך שונה בהתאם לסוג התוכנית שה-instance משויך אליה:

- בתוכניות Premium ו-Open, זהו השימוש בזמן אמת ב-28 הימים האחרונים (חלון גלילה), עד לשעה הנוכחית ביום הנוכחי.
- בתוכניות Pay-As-You-Go, זהו כל השימוש מאז יצירת ה-instance.
- בתוכניות Flex, זהו השימוש מתחילת החוזה.

## צפייה בשימוש של workload {#view-workload-usage}

כדי לבדוק כמה שימוש גרם workload ספציפי, פתחו את [דף ה-Workloads](https://quantum.cloud.ibm.com/workloads) וצפו בעמודה "Usage" עבור ה-workload.
