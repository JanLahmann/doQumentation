# מבוא ל-Primitives {#introduction-to-primitives}

ה-Primitives נוצרו כדי לפשט את המשימות הנפוצות ביותר של מחשבים קוונטיים: דגימת מצבים קוונטיים וחישוב ערכי ציפייה. ה-primitives הראשונים של Qiskit Runtime ([`EstimatorV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2) ו-[`SamplerV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2)) הם מימושים של [מחלקות הבסיס של primitives](/guides/primitives) ב-Qiskit. הם מספקים מימוש מתוחכם יותר (לדוגמה, כולל הפחתת שגיאות) כשירות מבוסס ענן ומשמשים לגישה לחומרת IBM Quantum&reg;.

ה-primitive החדש ביותר של Qiskit Runtime, [Executor](https://qiskit.github.io/qiskit-ibm-runtime/stubs/qiskit_ibm_runtime.Executor.html) (שנמצא בבטא), מספק ממשק ברמה נמוכה יותר שנותן יותר נראות ושליטה מבלי לפגוע בביצועים.
