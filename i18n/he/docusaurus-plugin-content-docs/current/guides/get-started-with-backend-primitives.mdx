# התחלה עם primitives של Backend {#get-started-with-the-backend-primitives}

בניגוד ל-primitives ספציפיים לספק, primitives של Backend הם מימושים גנריים שניתן להשתמש בהם עם כל אובייקט `backend`, כל עוד הוא מממש את הממשק [`BackendV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.providers.BackendV2). חלק מהספקים מממשים primitives באופן מקורי. ראה את [דף המערכת האקולוגית של Qiskit](https://www.ibm.com/quantum/ecosystem?tag=provider) לפרטים.

<span id="estimator"></span>
## התחלה עם primitive Estimator של Backend {#get-started-with-the-estimator-backend-primitive}

ניתן להריץ את ה-primitive Estimator עם כל ספק באמצעות הכיתה [`qiskit.primitives.BackendEstimatorV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendEstimatorV2). עם זאת, הוא לא מציע מימושי הפחתת שגיאות מדידה או Gate "מהקופסה", מכיוון שה-primitives של Backend מתוכננים לפעול מקומית על מחשב המשתמש.

### דוגמה: {#example}

```python
from qiskit.primitives import BackendEstimatorV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
estimator = BackendEstimatorV2(backend)
```

<span id="sampler"></span>
## התחלה עם primitive Sampler של Backend {#get-started-with-the-sampler-backend-primitive}

ניתן להריץ את ה-primitive Sampler עם כל ספק באמצעות [`qiskit.primitives.BackendSamplerV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendSamplerV2). עם זאת, הוא דורש Backend שתומך באפשרות `memory`.

### דוגמה: {#example}

```python
from qiskit.primitives import BackendSamplerV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
sampler = BackendSamplerV2(backend)
```

    - עיין בתיעוד API של [`qiskit.primitives.BackendEstimatorV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendEstimatorV2).
</Admonition>
