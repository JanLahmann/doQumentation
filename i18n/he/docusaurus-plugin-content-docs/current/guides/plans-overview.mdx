# סקירת התוכניות הזמינות {#overview-of-available-plans}

ל-IBM Quantum Platform יש תוכנית Open (גישה חינמית) ותוכניות בתשלום, כמתואר להלן. ראה [תוכניות גישה של IBM Quantum](https://www.ibm.com/quantum/pricing) לפרטים מלאים.

אם אתה רוצה גישה למספר תוכניות, תוכל ליצור מספר מופעים ולהקצות להם תוכניות שונות. לדוגמה, יכול להיות לך מופע של Open Plan ומופע של Pay-As-You-Go Plan. אתה יכול להגדיר מופעים מועדפים לשימוש באתחול של `QiskitRuntimeService` או במתודה `QiskitRuntimeService.save_account()`. דוגמה:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Open Plan {#open-plan}

הרץ את ה-Circuits הקוונטיים שלך על ה-QPUs הטובים בעולם בחינם (עד 10 דקות זמן קוונטי בחלון גלגולי של 28 יום). אתה יכול לעקוב אחר השימוש שלך ב[לוח הבקרה של הפלטפורמה](https://quantum.cloud.ibm.com/) ובדפי [Workloads](https://quantum.cloud.ibm.com/workloads). ה-Open Plan זמין רק באזור us-east.

תוכנית זו מומלצת למי שלומד מחשוב קוונטי וחוקר את הטכנולוגיה הקוונטית של IBM.

## Flex Plan {#flex-plan}

רכוש מראש מספר מסוים של דקות (לפחות 400) להרצת עבודות קוונטיות, לשימוש תוך שנה מהרכישה. אתה יכול לרכוש דקות נוספות בכל עת.

לאחר רכישת הזמן, עליך להקצות את הדקות למופעים קיימים או חדשים ב-Flex Plan עבור החשבון שלך. הגדר או עדכן מגבלות הקצאה ב[דף Instances](https://quantum.cloud.ibm.com/instances) או הגדר את ההקצאה בעת [יצירת מופע Flex Plan חדש.](/guides/instances) אם אתה מקצה את כל הדקות למופעים קיימים, לא תוכל ליצור מופע חדש ל-Flex Plan עד שתרכוש דקות נוספות או תסיר דקות מהקצאת מופע אחר. ראה [הגדרת מגבלות הקצאה למופע](allocation-limits) למידע נוסף.

אם אתה זקוק ליותר מ-10,000 דקות, כדאי לשקול מנוי ל-Premium Plan. אם אינך רוצה לרכוש 400 דקות, שקול במקום זאת את Pay-As-You-Go Plan.

[צור קשר עם IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) להרשמה ל-Flex Plan.

## Pay-As-You-Go Plan {#pay-as-you-go-plan}

הרץ Circuits קוונטיים על ה-QPUs הטובים בעולם ושלם רק עבור הזמן הקוונטי שבו אתה משתמש.

תוכנית זו מומלצת למי שמבצע פרויקטי מחקר של שימושיות קוונטית ובוחן מודלים עסקיים עם גישה גמישה. אם אתה עומד להשתמש בלפחות 400 דקות בשנה הקרובה, מומלץ להשתמש במקום זאת ב-Flex Plan.

## Premium Plan {#premium-plan}

הרץ Circuits קוונטיים על ה-QPUs הטובים בעולם באמצעות מנוי ארגוני לזמן קוונטי. בנוסף, פונקציונליות מסוימת, כגון [Qiskit Functions](functions) ו-[Qiskit Transpiler as a Service](qiskit-transpiler-service), זמינה למשתמשי Premium Plan.

תוכנית זו מומלצת למי שרוצה לבצע מפת דרכים קוונטית אסטרטגית ולפתח אלגוריתמים ויישומים קוונטיים בקנה מידה. [צור קשר עם IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) למידע נוסף.

## On-Prem Plan {#on-prem-plan}

קבל גישה למערכת קוונטית מוקדשת לחלוטין על-גבי הציוד שלך, המשורתת ומתוחזקת על ידי IBM Quantum.

שירות זה מומלץ למי שחוקר אלגוריתמים קלאסיים-קוונטיים ויישומים וזקוק לשליטה גבוהה על המשאבים והנתונים שלו. [צור קשר עם IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum) למידע נוסף.
