# התקנת לקוח Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime הוא שירות מבוסס ענן להרצת חישובים קוונטיים על חומרת IBM Quantum&reg;. המדריך הזה מראה לך איך להתקין את `qiskit-ibm-runtime`, שהוא חבילת הלקוח לגישה לשירות Qiskit Runtime.

לפני שמתקינים את לקוח Qiskit Runtime, יש לבצע את ההוראות במדריכים הבאים:

1. [התקן את Qiskit](/guides/install-qiskit).
2. [הגדר את חשבון IBM Cloud שלך](cloud-setup).

הרץ את הפקודה הבאה כדי להתקין את Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="שמור על גרסאות עדכניות">
בדוק מדי פעם את [הערות הגרסה של Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) כדי לראות גרסאות חדשות. אנחנו ממליצים לעדכן את הדרישות של `qiskit-ibm-runtime` לעיתים קרובות על ידי הרצת `pip install -U qiskit-ibm-runtime`.
</Admonition>
