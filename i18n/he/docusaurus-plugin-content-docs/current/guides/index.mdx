# מבוא {#introduction}

## מבוא ל-Qiskit ול-IBM Quantum {#introduction-to-qiskit-and-ibm-quantum}

ברוכים הבאים לתיעוד של Qiskit, חבילותיו הנלוות ופלטפורמת IBM Quantum&reg;. התיעוד כולל מדריכי הוראות שיעזרו לכם להתחיל להשתמש בכלים שלנו, מדריכים מעשיים המציגים דוגמאות מקצה לקצה לתרחישי שימוש ספציפיים, וכן אוסף של חומרי עיון ל-API.

Qiskit מספקת מסגרת מודולרית וניתנת להרחבה למחקר ופיתוח קוונטי בתחומי אלגוריתמים, מחשוב ביצועים גבוהים ומדעי מידע קוונטי. בעזרתה, חוקרים יכולים לבנות, לאמן ולהריץ תהליכי עבודה קוונטיים עם תוספות מיוחדות, כלי תוכנה ומשאבים נרחבים. דרך IBM Quantum Platform, משתמשים יכולים לגשת לשירותי חישוב קוונטי, כגון Qiskit Runtime וקטלוג Qiskit Functions, כדי להריץ עומסי עבודה ביעילות על צי מחשבי הקוונטום של IBM&reg;.

![גרפיקה המציגה את הרכיבים השונים של IBM Quantum Platform ו-Qiskit.](/docs/images/guides/index/overview-image.svg)

מעבר ל-Qiskit וחבילותיו הנלוות קיים [מערכת האקולוגית של Qiskit](https://www.ibm.com/quantum/ecosystem) — קטלוג של פרויקטים בקוד פתוח המתממשקים עם Qiskit כדי להרחיב את הפונקציונליות שלו.
