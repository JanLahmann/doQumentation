# רישום יומן עבור Qiskit Runtime {#logging-for-qiskit-runtime}

שירותי IBM Cloud&reg;, כגון Qiskit Runtime, מייצרים יומני פלטפורמה שבהם אתה יכול להשתמש כדי לחקור פעילות חריגה ופעולות קריטיות בחשבון שלך, ולפתור בעיות.

אתה יכול להשתמש ב-[IBM Cloud Logs Routing](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about), שהוא שירות פלטפורמה, כדי לנתב יומני פלטפורמה בחשבונך ליעד לפי בחירתך על ידי הגדרת דייר המגדיר לאן נשלחים יומני הפלטפורמה. למידע נוסף, ראה [About Logs Routing.](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about)

אתה יכול להשתמש ב-[IBM Cloud Logs](https://cloud.ibm.com/docs/cloud-logs) כדי להמחיש ולקבל התראות על יומני פלטפורמה שנוצרים בחשבונך ומנותבים על ידי IBM Cloud Logs Routing לאינסטנס של IBM Cloud Logs.

## מיקומים שבהם נוצרים יומני פלטפורמה {#locations-where-platform-logs-are-generated}

IBM Cloud Container Registry מייצר יומני פלטפורמה המוצגים באינסטנסים של הרישום שלך. הטבלה הבאה מפרטת את המיקומים שבהם אוסף אוטומטי של יומני שירות Container Registry מופעל.

| מיקום                       | האם יומני שירות פלטפורמה זמינים? |
|-----------------------------|----------------------------------|
| וושינגטון די.סי. (us-east) |                כן                |
| פרנקפורט (eu-de)            |                כן                |

## יומני פלטפורמה שנוצרים {#platform-logs-that-are-generated}

IBM Qiskit Runtime מייצר יומני שירות פלטפורמה המוצגים באינסטנסים של הרישום שלך עבור המקרים הספציפיים הבאים:

- כאשר מצב המשימה משתנה.

## צפייה ביומנים מדף הניטור {#viewing-logs-from-the-observability-page}

למידע על הפעלת ממשק IBM Cloud Logs, ראה **Launching the UI** ב-[תיעוד IBM Cloud Logs.](https://cloud.ibm.com/docs/cloud-logs?topic=cloud-logs-instance-launch)

## שדות לפי סוג יומן {#fields-by-log-type}

למידע על שדות הכלולים בכל יומן פלטפורמה, ראה [Fields that are included in platform logs.](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about-platform-logs#about-platform-logs-2)

השדות הבאים כלולים ברשומת היומן.

| שדה             | סוג      | תיאור                                                                            |
|-----------------|----------|----------------------------------------------------------------------------------|
| logSourceCRN    | Required | מגדיר את החשבון ואת אינסטנס יומן הזרימה שבו היומן מתפרסם.                      |
| saveServiceCopy | Required | מגדיר אם IBM שומרת עותק של הרשומה למטרות תפעוליות.                              |
| message         | Required | תיאור היומן שנוצר.                                                               |
