# אירועי מעקב פעילות עבור שירות IBM Quantum System {#activity-tracking-events-for-the-ibm-quantum-system-service}

שירותי IBM Cloud&reg;, כמו שירות IBM&reg; Quantum System, מייצרים אירועי מעקב פעילות.

אירועי מעקב פעילות מדווחים על פעולות שמשנות את מצב השירות ב-IBM Cloud. אפשר להשתמש באירועים כדי לחקור פעילות חריגה ופעולות קריטיות, וכן כדי לעמוד בדרישות ביקורת רגולטוריות.

אפשר להשתמש ב-IBM Cloud Activity Tracker Event Routing, שירות פלטפורמה, כדי לנתב אירועי ביקורת בחשבון שלך ליעדים לפי בחירתך על ידי קביעת יעדים ומסלולים שמגדירים לאן נשלחים אירועי מעקב הפעילות. למידע נוסף, ראו [About IBM Cloud Activity Tracker Events Routing](https://cloud.ibm.com/docs/atracker?topic=atracker-about).

אפשר להשתמש ב-IBM Cloud Logs כדי להציג ולקבל התראות על אירועים שנוצרו בחשבון שלך ונותבו על ידי IBM Cloud Activity Tracker Event Routing למופע IBM Cloud Logs.

## צפייה באירועי מעקב פעילות עבור שירות IBM Quantum System {#viewing-activity-tracking-events-for-the-ibm-quantum-system-service}

אפשר להשתמש ב-IBM Cloud Logs כדי להציג ולקבל התראות על אירועים שנוצרו בחשבון שלך ונותבו על ידי IBM Cloud Activity Tracker Event Routing למופע IBM Cloud Logs.

### הפעלת IBM Cloud Logs מדף Observability {#launching-ibm-cloud-logs-from-the-observability-page}

למידע על הפעלת ממשק המשתמש של IBM Cloud Logs, ראו את [Launching the UI](https://cloud.ibm.com/docs/cloud-logs?topic=cloud-logs-instance-launch#instance-launch-cloud-ui) בתיעוד IBM Cloud Logs.

| פעולה | מופעל על ידי |
| -- | -- |
| `quantum-computing.direct-access-backend-configuration.read` | קבל תצורת Backend ‏(`GET /v1/backends/{backend-name}/configuration`) |
| `quantum-computing.direct-access-backend-properties.read` | קבל מאפייני Backend ‏(`GET /v1/backends/{backend-name}/properties`) |
| `quantum-computing.direct-access-backend.list` | קבל רשימת Backends ‏(`GET /v1/backends`) |
| `quantum-computing.direct-access-backend.read` | קבל פרטי Backend ‏(`GET /v1/backends/{backend-name}`) |
| `quantum-computing.direct-access-job.cancel` | בטל עבודה (`POST /v1/jobs/{job-id}/cancel`) |
| `quantum-computing.direct-access-job.create` | הרץ עבודה (`POST /v1/jobs`) |
| `quantum-computing.direct-access-job.delete` | מחק עבודה (`DELETE /v1/jobs/{job-id}`) |
| `quantum-computing.direct-access-job.list` | קבל עבודות (`GET /v1/jobs`) |
| `quantum-computing.direct-access-job.read` | קבל עבודה לפי ID ‏(`GET /v1/jobs/{job-id}`) |
| `quantum-computing.direct-access-lane-configuration.list` | קבל תצורת lanes של Backend ‏(`GET /v1/backends/{backend-name}/lanes`) |
