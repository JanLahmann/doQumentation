# סקירה כללית {#overview}

!["תמונת כותרת"](/learning/images/courses/quantum-diagonalization-algorithms/hero.avif)

קורס זה מספק סקירה של מספר גישות לאלכסון קוונטי. מובא רקע על השיטות הקלאסיות שנעשה בהן שימוש, או שהניעו את הפיתוח של האלגוריתמים הקוונטיים. לאחר מכן, התוכן מדריך צעד אחר צעד את המימוש של האלגוריתמים הקוונטיים על מחשבים קוונטיים אמיתיים. ישנה דיון מעמיק בגורמים הקובעים את יעילות הגישות תוך שימוש באלגוריתמים קלאסיים וקוונטיים.

## רקע מומלץ {#recommended-background}

קורס זה מיועד לאנשים שיש להם ניסיון מסוים בחישוב קוונטי ורוצים לנצל את כוחו לאלכסון מטריצות.

אנו ממליצים ללומדים להשלים את הדרישות המוקדמות הבאות לפני הקורס:

- הגדירו את הסביבה המקומית שלכם על ידי מעקב אחר מדריך [תחילת העבודה](/guides/) בדף התיעוד שלנו.
- להבנה מעמיקה יותר של מידע קוונטי, נסו את [יסודות המידע הקוונטי](/learning/courses/basics-of-quantum-information) עם John Watrous.

## מבחן {#exam}

כדי לקבל את התג שלכם באלגוריתמי אלכסון קוונטיים, גשו למבחן ב-IBM&reg; Training. מבחן זה נועד להיבחן לאחר קריאת השיעורים בקורס זה. לאחר שעוברים את המבחן, תקבלו הודעה מ-Credly שזכיתם בתג.
