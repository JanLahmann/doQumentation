בנה [מצב Bell](https://en.wikipedia.org/wiki/Bell_state) (שני קיוביטים שזורים) והפעל אותו בשלוש דרכים:

1. **סימולציה אידיאלית** — תוצאות מושלמות, ללא צורך בחשבון
2. **סימולציה עם רעש** — מדמה מכשיר אמיתי, ללא צורך בחשבון
3. **חומרה קוונטית אמיתית** — דורש חשבון IBM Quantum חינמי (שלבי ההגדרה למטה)
## בניית המעגל {#build-the-circuit}

```python
# Added by doQumentation — required packages for this notebook
!pip install -q qiskit qiskit-ibm-runtime
```

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

qc.draw(output="mpl")
```

## אפשרות 1: סימולציה אידיאלית (ללא צורך בחשבון) {#option-1-ideal-simulation-no-account-needed}

משתמש ב-`StatevectorSampler` — סימולטור מקומי עם תוצאות מושלמות, ללא רעש.

```python
from qiskit.primitives import StatevectorSampler

result = StatevectorSampler().run([qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
from qiskit.visualization import plot_histogram
plot_histogram(counts)
```

## אפשרות 2: סימולציה עם רעש (ללא צורך בחשבון) {#option-2-noisy-simulation-no-account-needed}

משתמש ב-`FakeManilaV2` — סימולטור מקומי המחקה מכשיר IBM Quantum אמיתי, כולל מאפייני הרעש שלו. המעגל חייב להיות מותאם (transpiled) תחילה לסט השערים וחיבוריות הקיוביטים של המכשיר.

```python
from qiskit_ibm_runtime import SamplerV2
from qiskit_ibm_runtime.fake_provider import FakeManilaV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

backend = FakeManilaV2()
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```

## אפשרות 3: חומרה קוונטית אמיתית {#option-3-real-quantum-hardware}

דורש חשבון IBM Quantum חינמי. כדי להגדיר חשבון:

1. הירשם ב-[quantum.cloud.ibm.com/registration](https://quantum.cloud.ibm.com/registration) — ללא צורך בכרטיס אשראי ב-30 הימים הראשונים
2. התחבר ב-[quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) ובחר את האזור **us-east** (נדרש עבור ה-Open Plan החינמי)
3. צור instance (Open Plan חינמי) ב-[Instances](https://quantum.cloud.ibm.com/instances) אם עדיין אין לך אחד
4. צור מפתח API ב-[quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) (או ב-[cloud.ibm.com/iam/apikeys](https://cloud.ibm.com/iam/apikeys))
5. העתק את ה-**CRN** שלך (Cloud Resource Name) מדף ה-[Instances](https://quantum.cloud.ibm.com/instances) שלך

לאחר מכן, אם עדיין לא שמרת את האישורים שלך בסשן Binder זה, הפעל את התא למטה. החלף את `<your-api-key>` במפתח ה-API משלב 4, ואת `<your-crn>` ב-CRN משלב 5.

```python
from qiskit_ibm_runtime import QiskitRuntimeService

QiskitRuntimeService.save_account(
    channel="ibm_quantum_platform",
    token="<your-api-key>",
    instance="<your-crn>",
    set_as_default=True,
    overwrite=True,
)
```

**הערה:** משימות על חומרה אמיתית עשויות לקחת זמן בהתאם לזמני התור. אם התא עדיין פועל, תוכל לבדוק את סטטוס המשימה שלך ולראות את התוצאות ב-[quantum.cloud.ibm.com/workloads](https://quantum.cloud.ibm.com/workloads?user=me).

```python
from qiskit_ibm_runtime import QiskitRuntimeService, SamplerV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

service = QiskitRuntimeService()
backend = service.least_busy(operational=True, simulator=False)
print(f"Running on {backend.name}")

pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```
