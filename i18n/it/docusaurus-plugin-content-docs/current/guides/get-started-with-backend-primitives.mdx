# Inizia con le primitive backend {#get-started-with-the-backend-primitives}

A differenza delle primitive specifiche per provider, le primitive backend sono implementazioni generiche che possono essere utilizzate con un oggetto `backend` arbitrario, purché implementi l'interfaccia [`BackendV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.providers.BackendV2). Alcuni provider implementano le primitive in modo nativo. Consulta la [pagina Qiskit Ecosystem](https://www.ibm.com/quantum/ecosystem?tag=provider) per i dettagli.

<span id="estimator"></span>
## Inizia con la primitiva backend Estimator {#get-started-with-the-estimator-backend-primitive}

La primitiva Estimator può essere eseguita con qualsiasi provider utilizzando la classe [`qiskit.primitives.BackendEstimatorV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendEstimatorV2). Tuttavia, non offre implementazioni di mitigazione degli errori di misura o di Gate "pronte all'uso", poiché le primitive backend sono progettate per essere eseguite localmente sulla macchina dell'utente.

### Esempio: {#example}

```python
from qiskit.primitives import BackendEstimatorV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
estimator = BackendEstimatorV2(backend)
```

<span id="sampler"></span>
## Inizia con la primitiva backend Sampler {#get-started-with-the-sampler-backend-primitive}

La primitiva Sampler può essere eseguita con qualsiasi provider utilizzando [`qiskit.primitives.BackendSamplerV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendSamplerV2). Tuttavia, richiede un backend che supporti l'opzione `memory`.

### Esempio: {#example}

```python
from qiskit.primitives import BackendSamplerV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
sampler = BackendSamplerV2(backend)
```
