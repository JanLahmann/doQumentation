# Installa il client Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime è il servizio cloud per l'esecuzione di calcoli quantistici sull'hardware IBM Quantum&reg;. Questa guida ti mostra come installare `qiskit-ibm-runtime`, il pacchetto client per accedere al servizio Qiskit Runtime.

Prima di installare il client Qiskit Runtime, segui le istruzioni in queste guide:

1. [Installa Qiskit](/guides/install-qiskit).
2. [Configura il tuo account IBM Cloud](cloud-setup).

Esegui questo comando per installare Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Resta aggiornato con le ultime versioni">
Controlla periodicamente le [note di rilascio di Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) per vedere le nuove versioni. Ti consigliamo di aggiornare frequentemente i requisiti per `qiskit-ibm-runtime` eseguendo `pip install -U qiskit-ibm-runtime`.
</Admonition>
