# Ambienti di laboratorio online {#online-lab-environments}

Oltre a scaricare e installare Qiskit per lavorare sulla tua macchina locale, sono disponibili diverse opzioni per chi preferisce lavorare in un ambiente di sviluppo online.

Diversi provider offrono ambienti Jupyter online con Qiskit preinstallato per un avvio rapido e conveniente. Tra i provider consigliati ci sono il provider di servizi cloud OVHcloud e la piattaforma cloud per lo sviluppo di software quantistico qBraid.

## Ambienti notebook consigliati con Qiskit preinstallato {#recommended-notebook-environments-with-qiskit-preinstalled}

Altri ambienti Jupyter notebook online come IBM Watson&reg; Studio, Google Colab e Microsoft Azure Machine Learning Studio possono essere usati con Qiskit seguendo le istruzioni appropriate.

### Notebook AI di OVHcloud {#ovhcloud-ai-notebooks}

Gli utenti di OVHcloud Public Cloud e OVHcloud Startup possono usare i propri notebook quantistici OVHcloud per accedere a Qiskit e inviare job alle QPU IBM&reg; con il proprio account IBM Quantum&reg; o IBM Cloud&reg;. Per saperne di più, visita la pagina [Documentazione e tutorial di OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
I notebook AI di OVHcloud non sono disponibili in alcune regioni, come gli Stati Uniti. Inoltre, le interfacce utente potrebbero avere un aspetto diverso a seconda della regione.
</Admonition>

Segui questi passaggi per iniziare:

1. [Crea un account IBM Cloud.](./cloud-setup)
1. [Configura un account OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Crea un progetto OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Segui i passaggi in [Avvia il tuo primo AI Notebook,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) selezionando **Qiskit** come framework.

### qBraid Lab {#qbraid-lab}

Gli utenti di Qiskit possono usare gli ambienti Python preconfigurati di qBraid seguendo queste istruzioni. Per ulteriori informazioni, consulta la [documentazione di qBraid.](https://docs.qbraid.com/home/introduction)

1. [Configura un account gratuito su qBraid.](https://docs.qbraid.com/home/account)
3. [Crea un ambiente Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Avvia qBraid Lab, clicca sull'icona **ENVS** a destra, clicca su **+ ADD** in cima alla nuova barra laterale e cerca Qiskit. Clicca sulla versione che vuoi usare, poi clicca su **Install**.
4. Crea un Jupyter notebook cliccando su **Python 3 [Qiskit]** nella sezione Notebook.
4. Puoi installare pacchetti Python aggiuntivi (come Qiskit v1.2) in una cella del notebook eseguendo `%pip install [package_name].` Per ulteriori informazioni, consulta l'argomento [Comandi Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Ambienti notebook aggiuntivi per gli utenti Qiskit {#additional-notebook-environments-for-qiskit-users}

Per le piattaforme senza Qiskit preconfigurato, le istruzioni di configurazione sono simili, tranne per il fatto che devi installare Qiskit manualmente. Dovrai creare un account, avviare un nuovo progetto (o notebook o workspace, a seconda della piattaforma) e installare Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Configura una prova gratuita di Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Crea un progetto Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Per installare i pacchetti Qiskit e le dipendenze necessarie, apri un nuovo notebook ed esegui il seguente comando:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Crea un account Google.](https://support.google.com/accounts/answer/27441?hl)
2. Vai alla [pagina di destinazione di Google Colab](https://colab.research.google.com/) e accedi con il tuo account Google.
3. [Crea un notebook Colab](https://colab.research.google.com/#create=true) e installa Qiskit eseguendo il seguente comando:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Crea un account Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Crea un workspace.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Crea un Jupyter notebook nel tuo workspace.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Dal notebook, esegui questo comando per installare Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
