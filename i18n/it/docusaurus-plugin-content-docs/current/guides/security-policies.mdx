# Politiche di sicurezza del repository {#repository-security-policies}

È di fondamentale importanza mantenere i tuoi sistemi il più sicuri possibile. Pertanto, è vivamente consigliato utilizzare strumenti di sicurezza come [Dependabot](https://github.com/dependabot) che ti notificano automaticamente quando le dipendenze non sono aggiornate.
