# Guida alla migrazione a Qiskit 1.0 {#qiskit-10-migration-guide}

Qiskit 1.0 introduce nuove garanzie di stabilità e miglioramenti delle prestazioni, ma non è compatibile con nessuna versione precedente di Qiskit. Consulta le guide seguenti per aggiornare il tuo ambiente e il tuo codice in modo che funzionino con Qiskit v1.0.

- [Installa il nuovo pacchetto Qiskit v1.0](/guides/qiskit-1.0-installation) per aggiornare gli ambienti Python e le dipendenze del progetto.
- [Modifiche alle funzionalità di Qiskit v1.0](/guides/qiskit-1.0-features) per aggiornare il tuo codice.
- [Modifiche ai pacchetti con incompatibilità in Qiskit v1.0](/guides/metapackage-migration) per comprendere la nuova e la vecchia struttura dei pacchetti.

Scopri di più sui vantaggi di Qiskit v1.0 sul [blog di IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
