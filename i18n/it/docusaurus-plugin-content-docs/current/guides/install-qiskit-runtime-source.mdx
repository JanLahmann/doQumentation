# Installa il client Qiskit Runtime dal codice sorgente {#install-the-qiskit-runtime-client-from-source}

Quando installi Qiskit Runtime dal codice sorgente, puoi accedere alla versione di sviluppo corrente, invece di usare la versione nel repository Python Package Index (PyPI). Puoi quindi ispezionare ed estendere l'ultima versione del codice in modo più efficiente.

## Crea e attiva un nuovo ambiente virtuale {#create-and-activate-a-new-virtual-environment}

1. Naviga nella directory del tuo progetto e crea un ambiente minimale con solo Python installato.

## Installa Qiskit Runtime {#install-qiskit-runtime}

Segui questi passaggi per installare Qiskit Runtime:

1. Clona il repository Qiskit Runtime.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. Passa alla directory `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. Installa `qiskit-runtime`. Ti consigliamo di utilizzare un [ambiente virtuale](https://docs.python.org/3/library/venv.html) per evitare di contaminare l'installazione globale di Python.

  * **Installazione standard**:

    ```bash
    pip install .
    ```

  * **Modalità modificabile**: In questa modalità, non è necessario reinstallare Qiskit quando ci sono modifiche al codice del progetto.

    ```bash
    pip install -e .
    ```

    In modalità modificabile, le estensioni compilate vengono costruite in _modalità debug_ senza ottimizzazioni.

4. (Opzionale) Se vuoi eseguire test o controlli di linting, installa i requisiti per sviluppatori. Ti consigliamo di utilizzare un [ambiente virtuale](https://docs.python.org/3/library/venv.html) per evitare di contaminare l'installazione globale di Python.

```bash
pip install -e ".[dev]"
```
