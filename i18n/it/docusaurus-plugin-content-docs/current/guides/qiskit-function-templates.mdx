# Introduzione ai template per le Qiskit Function {#introduction-to-qiskit-function-templates}

I [template per le Qiskit Function](https://github.com/qiskit-community/qiskit-function-templates) sono una raccolta open-source di esempi di codice realistici che ti danno accesso ad alcuni dei più recenti workflow di ricerca, sfruttando strumenti come il Qiskit SDK e i [Qiskit addon](/guides/addons). Questi workflow preconfigurati traggono spunto da esperimenti pratici e li organizzano in modo che tu possa personalizzarli e riutilizzarli nella tua ricerca, senza dover ricominciare da zero. Tutti i template sono immediatamente distribuibili su [Qiskit Serverless](./serverless), dove puoi integrare facilmente il calcolo classico nel tuo workflow.

Esistono due tipi di template:

- *Implementazioni di template*: esempi di codice specializzati che implementano tecniche avanzate provenienti da esperimenti reali in ambiti come la chimica e la fisica.
- *Template base*: riferimenti per sviluppare la tua Qiskit Function originale. Questi template mettono in evidenza le best practice nello sviluppo delle interfacce, nella formattazione del codice, nei test unitari e altro ancora.

## Implementazioni di template {#template-implementations}

Le implementazioni dei template per le Qiskit Function sono organizzate per area applicativa. Attualmente la raccolta include un template di fisica per la simulazione hamiltoniana con il [Qiskit addon AQC-Tensor](./qiskit-addons-aqc) e un template di chimica per la struttura elettronica con il modello di solvente implicito, usando il [Qiskit addon SQD](./qiskit-addons-sqd).
Le risorse per iniziare con questi due template sono disponibili ai seguenti link:

- Simulazione della struttura elettronica con il modello di solvente implicito: [file sorgente del template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/chemistry/sqd_pcm) e [guida](./function-template-chemistry-workflow)
- Simulazione hamiltoniana: [file sorgente del template](https://github.com/qiskit-community/qiskit-function-templates/tree/main/physics/hamiltonian_simulation) e [guida](./function-template-hamiltonian-simulation)

## Template base {#base-templates}

I *template base* sono riferimenti per sviluppare la tua Qiskit Function originale. Sono progettati per aiutarti ad avviare rapidamente il processo di sviluppo, illustrando le best practice nello sviluppo delle interfacce, nella formattazione del codice, nella gestione dell'I/O, nei messaggi di errore, nei checkpoint, nei test unitari e altro ancora. Il repository è esposto come [template GitHub](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-template-repository), così che l'implementazione CI/CD possa essere facilmente estesa.

Attualmente sono disponibili due template: un [template per circuit function](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/circuit_function_template.py) e un [template per application function](https://github.com/qiskit-community/qiskit-function-templates/blob/main/base_templates/application_function_template.py). Il template per circuit function è strutturato in modo simile alla [IBM Circuit Function](./ibm-circuit-function), mentre il template per application function illustra come organizzare un tipico workflow applicativo.
