# Panoramica {#overview}

!["Immagine hero"](/learning/images/courses/variational-algorithm-design/hero.svg)

Questo corso è destinato a chi ha già una certa esperienza con l'informatica quantistica e vuole imparare a scrivere algoritmi variazionali utilizzando le primitive di Qiskit Runtime. È adatto anche ai professionisti dell'informatica quantistica che desiderano ampliare le proprie conoscenze e competenze nel campo degli algoritmi variazionali.

## Esame {#exam}

Per ottenere il tuo badge per la progettazione di algoritmi variazionali, sostieni l'esame su IBM&reg; Training. Questo esame è pensato per essere affrontato dopo aver letto le lezioni di questo corso. Una volta superato l'esame, riceverai una notifica da Credly con la quale ti verrà comunicato che hai ottenuto il badge.
