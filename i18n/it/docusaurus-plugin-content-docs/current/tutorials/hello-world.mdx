Costruisci uno [stato di Bell](https://en.wikipedia.org/wiki/Bell_state) (due qubit entangled) ed eseguilo in tre modi:

1. **Simulazione ideale** — risultati perfetti, nessun account richiesto
2. **Simulazione con rumore** — simula un dispositivo reale, nessun account richiesto
3. **Hardware quantistico reale** — richiede un account IBM Quantum gratuito (i passaggi di configurazione sono descritti di seguito)
## Costruire il circuito {#build-the-circuit}

```python
# Added by doQumentation — required packages for this notebook
!pip install -q qiskit qiskit-ibm-runtime
```

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

qc.draw(output="mpl")
```

## Opzione 1: Simulazione ideale (nessun account richiesto) {#option-1-ideal-simulation-no-account-needed}

Utilizza `StatevectorSampler` — un simulatore locale con risultati perfetti e privi di rumore.

```python
from qiskit.primitives import StatevectorSampler

result = StatevectorSampler().run([qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
from qiskit.visualization import plot_histogram
plot_histogram(counts)
```

## Opzione 2: Simulazione con rumore (nessun account richiesto) {#option-2-noisy-simulation-no-account-needed}

Utilizza `FakeManilaV2` — un simulatore locale che imita un vero dispositivo quantistico IBM, incluse le sue caratteristiche di rumore. Il circuito deve prima essere transpilato (adattato) al set di gate e alla connettività dei qubit del dispositivo.

```python
from qiskit_ibm_runtime import SamplerV2
from qiskit_ibm_runtime.fake_provider import FakeManilaV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

backend = FakeManilaV2()
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```

## Opzione 3: Hardware quantistico reale {#option-3-real-quantum-hardware}

Richiede un account IBM Quantum gratuito. Per configurarne uno:

1. Registrati su [quantum.cloud.ibm.com/registration](https://quantum.cloud.ibm.com/registration) — nessuna carta di credito richiesta per i primi 30 giorni
2. Accedi su [quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) e seleziona la regione **us-east** (obbligatoria per il piano Open gratuito)
3. Crea un'istanza (piano Open gratuito) su [Instances](https://quantum.cloud.ibm.com/instances) se non ne hai ancora una
4. Crea una chiave API su [quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) (oppure su [cloud.ibm.com/iam/apikeys](https://cloud.ibm.com/iam/apikeys))
5. Copia il tuo **CRN** (Cloud Resource Name) dalla pagina [Instances](https://quantum.cloud.ibm.com/instances)

Quindi, se non hai ancora salvato le credenziali in questa sessione Binder, esegui la cella seguente. Sostituisci `<your-api-key>` con la chiave API del passaggio 4 e `<your-crn>` con il CRN del passaggio 5.

```python
from qiskit_ibm_runtime import QiskitRuntimeService

QiskitRuntimeService.save_account(
    channel="ibm_quantum_platform",
    token="<your-api-key>",
    instance="<your-crn>",
    set_as_default=True,
    overwrite=True,
)
```

**Nota:** I job su hardware reale possono richiedere del tempo a seconda dei tempi di attesa in coda. Se la cella è ancora in esecuzione, puoi controllare lo stato del tuo job e vedere i risultati su [quantum.cloud.ibm.com/workloads](https://quantum.cloud.ibm.com/workloads?user=me).

```python
from qiskit_ibm_runtime import QiskitRuntimeService, SamplerV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

service = QiskitRuntimeService()
backend = service.least_busy(operational=True, simulator=False)
print(f"Running on {backend.name}")

pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```
