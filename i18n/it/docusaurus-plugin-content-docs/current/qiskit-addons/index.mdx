# Qiskit Addons

I [Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) sono pacchetti di estensione ufficiali che forniscono algoritmi e tecniche di quantum computing specializzati. Ogni addon si concentra su un'area specifica e include tutorial con codice eseguibile.

## Addon Disponibili {#available-addons}

| Addon | Pacchetto | Descrizione |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Riduci la larghezza e la profondità del Circuit tagliando Gate e fili |
| **SQD** | `qiskit-addon-sqd` | Sample-based Quantum Diagonalization per la stima dello stato fondamentale |
| **OBP** | `qiskit-addon-obp` | Operator Back-Propagation per ridurre la profondità del Circuit |
| **MPF** | `qiskit-addon-mpf` | Multi-Product Formulas per una simulazione hamiltoniana migliorata |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximate Quantum Compilation con reti tensoriali |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption per la mitigazione degli errori |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones per ridurre l'overhead di campionamento PEC |

Tutti i tutorial possono essere eseguiti direttamente nel tuo browser usando Binder, oppure sul tuo server Jupyter personale.

Per la documentazione API, visita i singoli repository degli addon su [GitHub](https://github.com/Qiskit?q=qiskit-addon).
