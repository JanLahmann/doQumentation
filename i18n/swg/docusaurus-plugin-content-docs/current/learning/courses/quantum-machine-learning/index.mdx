# Überblick {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Dieser Kurs erforscht, wie Quantum Computing a spannende Rolle im Machine Learning spiele kann. Der Kurs bietet a Wiederholong vo klassischa Machine Learning Asätze, geht aber au vo a gewisse Vertrautheit mit dene Methoda aus. Der Hauptfokus liegt drauf, wie Quantum Computing in bestehende Machine Learning Workflows passt. Thema sind Data Encoding, Quantum Kernels ond variational quantum circuits.
