# Übersicht {#overview}

!["Hero image"](/learning/images/courses/foundations-of-quantum-error-correction/hero.svg)

Grieß Gott bei *Grundlaga vo dr Quantefehlerkorrektur*, dem vierte Kurs in dr Serie *Versteha vo Quanteinformation ond -berechnong*, wo folgende Kurs umfasst:

- [Basics of quantum information](/learning/courses/basics-of-quantum-information)
- [Fundamentals of quantum algorithms](/learning/courses/fundamentals-of-quantum-algorithms)
- [General formulation of quantum information](/learning/courses/general-formulation-of-quantum-information)
- Grundlaga vo dr Quantefehlerkorrektur (diedr Kurs)

Diedr Kurs handlet vo dr Quantefehlerkorrektur, mit oim Fokus auf grundlegenda Konzepte.
Er fangt a mit oinr Erkläriong, was Quantefehlerkorrekturcode send ond wia se funktioneret.
Danno geht's woiter mit dem Diskretisierongphänomen vo Fehler, dem Stabilisatorformalism zur
Beschreibong vo Quantefehlerkorrekturcode, Calderbank–Shor–Steane (CSS) Code ond mehrere
wichtige Beispiel vo Quantefehlerkorrekturcode, eischließlich dem toric Code ond Surface-Code.
Dr Kurs diskutiert au fehlertoleranti Quanteberechnong, bei dr Quanteberecnonga
auf fehlerkorrigierte Quanteinformation ausgeführt werdet.

Diedr Kurs richtet sich an Studenta, Fachleit ond Hobbyischte in Bereiche wia Informatik, Physik,
Ingenieurwesa ond Mathematik, dia Kenntniss über dia theoretischa Grundlaga vo dr Quanteinformation
ond -berechnong erlanga möchtet.
