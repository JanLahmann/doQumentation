# Überblick {#overview}

!["Hero image"](/learning/images/courses/quantum-chem-with-vqe/hero.svg)

Willkomma bei **Quantum chemistry with VQE**. Der variational quantum eigensolver (VQE) isch a Quantum-Algorithmus mit Awändonga in der Quantachemie ond Simulation. In dem Einführongskurs lernt se d' grundlegenda Bausteine vom VQE, Möglichkeita bei seiner Implementierong ond a paar Awändonga kenna. D' Inhalte werdet über a Kombination aus Text, Video ond Code vermittelt.

Hauptthema sind:

    - Hamiltonians
    - Ansaetze
    - Klassische Optimierer
    - Grundzustandsenergien finda mit Qiskit patterns
    - Bindongslänge finda mit Qiskit patterns

Dieser Kurs konzentriert sich auf Awändonga in der Quantachemie, obwohl viele vo de gleiche Bausteina (Ansatz Circuits, klassische Optimierer ond s' Qiskit patterns Framework) auf a breites Spektrum vo Bereiche außerhalb der Quantachemie awädt werda kennet.
