# Qiskit 1.0 Migrationsleitfaden {#qiskit-10-migration-guide}

Qiskit 1.0 bringt neue Stabilitätsgarantien und Leistungsverbesserungen, bricht
jedoch die Kompatibilität mit allen früheren Versionen von Qiskit. Sieh dir die
folgenden Leitfäden an, um deine Umgebung und deinen Code für Qiskit v1.0 zu aktualisieren.

- [Installiere das neue Qiskit v1.0-Paket](/guides/qiskit-1.0-installation) zum Aktualisieren von Python-Umgebungen und Projektabhängigkeiten.
- [Qiskit v1.0 Feature-Änderungen](/guides/qiskit-1.0-features) zum Aktualisieren deines Codes.
- [Grundlegende Paketänderungen von Qiskit v1.0](/guides/metapackage-migration) zum Verstehen der neuen und alten Paketstruktur.

Erfahre mehr über die Vorteile von Qiskit v1.0 im [IBM&reg;-Blog](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
