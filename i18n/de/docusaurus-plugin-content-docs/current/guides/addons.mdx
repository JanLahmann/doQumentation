# Fortgeschrittene Techniken - Qiskit-Erweiterungen {#advanced-techniques---qiskit-addons}

Qiskit-Erweiterungen (Addons) sind eine Sammlung von Forschungsfunktionen, die die Entdeckung neuer Algorithmen im Utility-Maßstab ermöglichen. Diese modularen Softwarekomponenten bauen auf Qiskits leistungsstarker Grundlage auf und können in einen [Workflow](/guides/intro-to-patterns) eingebunden werden, um neue Quantenalgorithmen zu skalieren oder zu entwickeln. Diese Seite stellt die verfügbaren Werkzeuge in wichtigen Funktionskategorien vor, damit du beim Erstellen deiner Workflows die passenden Fähigkeiten auswählen kannst.
