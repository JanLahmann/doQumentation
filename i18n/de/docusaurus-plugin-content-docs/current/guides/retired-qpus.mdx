## Stillgelegte Prozessorfamilien {#retired-processor-families}

- **Canary**: Die Canary-Familie umfasste kleine Designs mit 5 bis 16 Qubits.
- **Falcon**: Die Falcon-Gerätefamilie bot eine wertvolle Plattform für mittelgroße Circuits und diente zudem als Grundlage, um Leistungs- und Skalierbarkeitsverbesserungen zu demonstrieren, bevor diese auf größere Geräte übertragen wurden.
- **Egret**: Egret brachte die Innovationen abstimmbarer Koppler auf eine 33-Qubit-Plattform und ermöglichte damit schnellere Zwei-Qubit-Gates mit höherer Genauigkeit.
- **Hummingbird**: Mit einem heavy-hexagonalen Qubit-Layout erlaubte die Hummingbird-Familie bis zu 65 Qubits.
- **Osprey**: Osprey war mit 433 Qubits fast viermal so groß wie Eagle. Die größeren Chip-Abmessungen erforderten weitere Verbesserungen bei der Geräteverpackung sowie maßgefertigte Flexkabel im Kryostaten, um die höheren Ein-/Ausgabeanforderungen innerhalb desselben Verkabelungsbereichs zu erfüllen.
- **Eagle**: Die Eagle-Prozessorfamilie integrierte im Vergleich zu früheren Generationen skalierbarere Verpackungstechnologien. Insbesondere wurden Signale durch mehrere Chipschichten geführt, was eine hohe I/O-Dichte ohne Leistungseinbußen ermöglichte.

## Einen Job von einer stillgelegten QPU abrufen {#retrieve}

Verwende den folgenden Code, um einen Job von einer stillgelegten QPU abzurufen.

<Admonition type="note">
Die Angabe eines Cloud Resource Name (CRN) im folgenden Code ist optional. Wenn er angegeben wird, sind nur Backends und Jobs aus dieser Instanz verfügbar. Wird keine Instanz angegeben, sind alle Backends und Jobs aller Instanzen deines Kontos verfügbar. In diesem Fall wird bei Angabe eines Backends automatisch eine Instanz gefunden und verwendet, die Zugriff auf dieses Backend hat.
</Admonition>

```python
from qiskit_ibm_runtime import QiskitRuntimeService

# Load your IBM Quantum account. Replace "<IBM Cloud CRN>" with your desired instance's CRN.
service = QiskitRuntimeService(channel="ibm_quantum_platform", instance="<IBM Cloud CRN>")

# Retrieve a single job by id
job = service.job(<job_id>)

# Retrieve a batch of jobs. Filtering options can be found in the QiskitRuntimeService.jobs api reference
jobs = service.jobs(backend_name=<backend_name>)
```

<Admonition type="note" title="Veraltete Provider-Pakete">
  `service.jobs()` gibt Jobs zurück, die sowohl mit `qiskit-ibm-runtime` als auch mit dem veralteten Paket `qiskit-ibm-provider` ausgeführt wurden. Jobs, die mit dem noch älteren (ebenfalls veralteten) Paket `qiskit-ibmq-provider` eingereicht wurden, sind nicht mehr verfügbar.
</Admonition>
