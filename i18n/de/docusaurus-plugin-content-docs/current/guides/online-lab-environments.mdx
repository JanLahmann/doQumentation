# Online-Lab-Umgebungen {#online-lab-environments}

Neben dem Herunterladen und Installieren von Qiskit auf deinem lokalen Rechner gibt es mehrere Optionen für alle, die lieber in einer Online-Entwicklungsumgebung arbeiten möchten.

Verschiedene Anbieter bieten Online-Jupyter-Umgebungen mit vorinstalliertem Qiskit für einen schnellen und bequemen Einstieg an. Zu diesen empfohlenen Anbietern gehören der Cloud-Dienstleister OVHcloud und die Quantum-Software-Entwicklungsplattform qBraid.

## Empfohlene Notebook-Umgebungen mit vorinstalliertem Qiskit {#recommended-notebook-environments-with-qiskit-preinstalled}

Weitere Online-Jupyter-Notebook-Umgebungen wie IBM Watson&reg; Studio, Google Colab und Microsoft Azure Machine Learning Studio können für die Arbeit mit Qiskit genutzt werden, indem du die entsprechenden Anweisungen befolgst.

### OVHcloud AI Notebooks {#ovhcloud-ai-notebooks}

Nutzer von OVHcloud Public Cloud und OVHcloud Startup können ihre OVHcloud Quantum Notebooks verwenden, um auf Qiskit zuzugreifen und Jobs an IBM&reg; QPUs mit ihrem IBM Quantum&reg;- oder IBM Cloud&reg;-Konto zu übermitteln. Weitere Informationen findest du auf der Seite [OVHcloud Dokumentation und Tutorials](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
OVHcloud AI Notebooks sind in einigen Regionen nicht verfügbar, z. B. in den Vereinigten Staaten. Außerdem können Benutzeroberflächen in verschiedenen Regionen unterschiedlich aussehen.
</Admonition>

Befolge diese Schritte, um loszulegen:

1. [Erstelle ein IBM Cloud-Konto.](./cloud-setup)
1. [Richte ein OVHcloud-Konto ein.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Erstelle ein OVH Public Cloud-Projekt.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Befolge die Schritte unter [Launch your first AI Notebook,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) und wähle **Qiskit** als Framework.

### qBraid Lab {#qbraid-lab}

Qiskit-Nutzer können die vorkonfigurierten Python-Umgebungen von qBraid verwenden, indem sie diese Anweisungen befolgen. Weitere Informationen findest du in der [qBraid-Dokumentation.](https://docs.qbraid.com/home/introduction)

1. [Erstelle ein kostenloses qBraid-Konto.](https://docs.qbraid.com/home/account)
3. [Erstelle eine Lab-Umgebung.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Starte qBraid Lab, klicke auf das **ENVS**-Symbol auf der rechten Seite, klicke auf **+ ADD** oben in der neuen Seitenleiste und suche nach Qiskit. Klicke auf die gewünschte Version und dann auf **Install**.
4. Erstelle ein Jupyter Notebook, indem du im Abschnitt Notebook auf **Python 3 [Qiskit]** klickst.
4. Du kannst zusätzliche Python-Pakete (z. B. Qiskit v1.2) in einer Notebook-Zelle installieren, indem du `%pip install [package_name].` ausführst. Weitere Informationen findest du im Thema [Pip (magic) commands](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Weitere Notebook-Umgebungen für Qiskit-Nutzer {#additional-notebook-environments-for-qiskit-users}

Für Plattformen ohne vorkonfiguriertes Qiskit sind die Einrichtungsschritte ähnlich, allerdings musst du Qiskit manuell installieren. Du erstellst ein Konto, startest ein neues Projekt (oder Notebook oder Arbeitsbereich, je nach Plattform) und installierst Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Richte eine kostenlose Watson Studio-Testversion ein.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Erstelle ein Watson Studio-Projekt.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Um die benötigten Qiskit-Pakete und Abhängigkeiten zu installieren, öffne ein neues Notebook und führe den folgenden Befehl aus:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Erstelle ein Google-Konto.](https://support.google.com/accounts/answer/27441?hl)
2. Gehe zur [Google Colab-Startseite](https://colab.research.google.com/) und melde dich mit deinem Google-Konto an.
3. [Erstelle ein Colab-Notebook](https://colab.research.google.com/#create=true) und installiere Qiskit mit folgendem Befehl:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Erstelle ein Azure-Konto.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Erstelle einen Arbeitsbereich.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Erstelle ein Jupyter Notebook in deinem Arbeitsbereich.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Führe im Notebook diesen Befehl aus, um Qiskit zu installieren:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
