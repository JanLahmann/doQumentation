# Übersicht der verfügbaren Pläne {#overview-of-available-plans}

IBM Quantum Platform bietet einen Open-Plan (kostenloser Zugang) sowie kostenpflichtige Pläne, wie unten beschrieben. Vollständige Details findest du unter [IBM Quantum access plans](https://www.ibm.com/quantum/pricing).

Wenn du Zugang zu mehreren Plänen benötigst, kannst du mehrere Instanzen erstellen und ihnen verschiedene Pläne zuweisen. Du kannst zum Beispiel eine Open-Plan-Instanz und eine Pay-As-You-Go-Plan-Instanz haben. Du kannst bevorzugte Instanzen im `QiskitRuntimeService`-Initialisierer oder in der Methode `QiskitRuntimeService.save_account()` festlegen. Beispiel:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Open Plan {#open-plan}

Führe deine Quantum Circuits kostenlos auf den weltbesten QPUs aus (bis zu 10 Minuten Quantum-Zeit pro fortlaufendem 28-Tage-Fenster). Du kannst deine Nutzung auf dem [Platform-Dashboard](https://quantum.cloud.ibm.com/) und auf den [Workloads](https://quantum.cloud.ibm.com/workloads)-Seiten verfolgen. Der Open Plan ist nur in der Region us-east verfügbar.

Dieser Plan wird für alle empfohlen, die Quantencomputing erlernen und die IBM-Quantentechnologie erkunden möchten.

## Flex Plan {#flex-plan}

Kaufe vorab eine bestimmte Anzahl von Minuten (mindestens 400) für die Ausführung von Quantum-Jobs, die innerhalb eines Jahres nach dem Kauf genutzt werden müssen. Du kannst jederzeit weitere Minuten kaufen.

Nach dem Kauf musst du die Minuten vorhandenen oder neuen Instanzen im Flex Plan für dein Konto zuweisen. Lege Kontingentgrenzen auf der [Instanzen-Seite](https://quantum.cloud.ibm.com/instances) fest oder aktualisiere sie, oder setze die Zuweisung beim [Erstellen einer neuen Flex-Plan-Instanz.](/guides/instances) Wenn du alle Minuten vorhandenen Instanzen zugewiesen hast, kannst du keine neue Instanz für den Flex Plan erstellen, bis du weitere Minuten kaufst oder einige Minuten aus der Zuweisung einer anderen Instanz entfernst. Weitere Informationen findest du unter [Instanz-Kontingentgrenzen festlegen](allocation-limits).

Wenn du mehr als 10.000 Minuten benötigst, solltest du stattdessen ein Premium-Plan-Abonnement in Betracht ziehen. Wenn du keine 400 Minuten kaufen möchtest, ist der Pay-As-You-Go-Plan die Alternative.

[Kontaktiere IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum), um dich für einen Flex Plan anzumelden.

## Pay-As-You-Go Plan {#pay-as-you-go-plan}

Führe Quantum Circuits auf den weltbesten QPUs aus und zahle nur für die tatsächlich genutzte Quantum-Zeit.

Dieser Plan wird für alle empfohlen, die Quantum-Utility-Forschungsprojekte durchführen und Geschäftsmodelle mit flexiblem Zugang testen. Wenn du im nächsten Jahr mindestens 400 Minuten verwenden wirst, wird empfohlen, stattdessen den Flex Plan zu nutzen.

## Premium Plan {#premium-plan}

Führe Quantum Circuits auf den weltbesten QPUs mit einem Enterprise-Quantum-Zeit-Abonnement aus. Darüber hinaus stehen einige Funktionen, wie [Qiskit Functions](functions) und [Qiskit Transpiler as a Service](qiskit-transpiler-service), Premium-Plan-Nutzern zur Verfügung.

Dieser Plan wird für alle empfohlen, die eine strategische Quantum-Roadmap umsetzen und Quantenalgorithmen sowie -anwendungen im großen Maßstab entwickeln möchten. [Kontaktiere IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum), um mehr zu erfahren.

## On-Prem Plan {#on-prem-plan}

Erhalte Zugang zu einem vollständig dedizierten On-Premises-Quantensystem, das von IBM Quantum gewartet und betreut wird.

Dieser Service wird für alle empfohlen, die klassisch-quantenmechanische Algorithmen und Anwendungen erkunden und eine hohe Kontrolle über ihre Ressourcen und Daten benötigen. [Kontaktiere IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum), um mehr zu erfahren.
