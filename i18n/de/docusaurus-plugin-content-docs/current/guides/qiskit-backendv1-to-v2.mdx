# Von `BackendV1` zu `BackendV2` migrieren {#migrate-from-backendv1-to-backendv2}

Die Qiskit-Klasse [`BackendV1`](https://docs.quantum.ibm.com/api/qiskit/1.4/qiskit.providers.BackendV1) wurde als veraltet markiert und wird aus dem Dienst entfernt.
