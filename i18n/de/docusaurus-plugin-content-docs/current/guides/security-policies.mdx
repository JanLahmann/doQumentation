# Sicherheitsrichtlinien des Repositorys {#repository-security-policies}

Es ist äußerst wichtig, dass du deine Systeme so sicher wie möglich hältst. Daher wird dringend empfohlen, Sicherheitstools wie [Dependabot](https://github.com/dependabot) zu verwenden, die dich automatisch benachrichtigen, wenn Abhängigkeiten veraltet sind.
