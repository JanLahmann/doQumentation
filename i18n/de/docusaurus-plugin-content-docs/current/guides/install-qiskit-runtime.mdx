# Den Qiskit Runtime-Client installieren {#install-the-qiskit-runtime-client}

Qiskit Runtime ist der cloudbasierte Dienst zur Ausführung von Quantenberechnungen auf IBM Quantum&reg;-Hardware. Diese Anleitung zeigt dir, wie du `qiskit-ibm-runtime` installierst – das Client-Paket für den Zugriff auf den Qiskit Runtime-Dienst.

Bevor du den Qiskit Runtime-Client installierst, folge den Anweisungen in diesen Anleitungen:

1. [Qiskit installieren](/guides/install-qiskit). 
2. [Dein IBM Cloud-Konto einrichten](cloud-setup).

Führe diesen Befehl aus, um Qiskit Runtime zu installieren:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Bleib auf dem neuesten Stand">
Schau regelmäßig in die [Qiskit Runtime-Versionshinweise](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes), um neue Releases zu verfolgen. Wir empfehlen, deine Anforderungen für `qiskit-ibm-runtime` häufig zu aktualisieren, indem du `pip install -U qiskit-ibm-runtime` ausführst.
</Admonition>
