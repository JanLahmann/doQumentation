# Übersicht {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

Willkommen bei *Use a quantum computer today*! In diesem Kurs beginnst du sofort damit, einen Quantencomputer zu nutzen, und lernst dabei die Grundprinzipien des Quantencomputings.

Dieser Kurs richtet sich an Neugierige und Vielbeschäftigte. Du brauchst keinen Physikabschluss oder jahrelange Programmiererfahrung. Wenn du einem Notebook folgen und auf „Run" klicken kannst, bist du bereit.
