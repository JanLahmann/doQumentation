# Gambaran Keseluruhan {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Kursus ini ditujukan kepada individu yang mempunyai sedikit pengalaman dengan pengkomputeran kuantum dan ingin belajar cara menulis algoritma variasi menggunakan primitif Qiskit Runtime. Ia juga sesuai untuk profesional pengkomputeran kuantum yang ingin mengembangkan pengetahuan dan kemahiran mereka dalam bidang algoritma variasi.

## Peperiksaan {#exam}

Untuk mendapatkan lencana Variational algorithm design kamu, ambil peperiksaan di IBM&reg; Training. Peperiksaan ini bertujuan untuk diambil selepas membaca pelajaran dalam kursus ini. Selepas kamu lulus peperiksaan, kamu akan dimaklumkan oleh Credly bahawa kamu telah mendapat lencana.
