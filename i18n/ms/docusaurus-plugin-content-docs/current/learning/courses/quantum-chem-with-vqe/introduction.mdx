Kursus ini ialah pengenalan tertumpu kepada variational quantum eigensolver (VQE), termasuk aplikasi VQE dalam kimia. Kursus ini ditujukan untuk pengamal teknikal pengkomputeran kuantum yang berminat untuk mengaplikasikan pengkomputeran kuantum kepada masalah kimia, biokimia, atau biofizik. Kursus ini merangkumi elemen penting pengiraan VQE, termasuk yang berikut:

* Hamiltonian
* Ansatz
* Pengoptimum klasik

Semua elemen ini digabungkan untuk membina pengiraan VQE pada Qiskit Runtime Primitives, terutamanya Estimator. Untuk maklumat tentang primitif, lihat [dokumentasi](/guides/primitives). Dalam menetapkan pengiraan VQE, kita akan mengikuti rangka kerja corak Qiskit standard:

* Petakan input klasik kepada masalah kuantum
* Optimumkan masalah untuk pelaksanaan kuantum
* Laksanakan menggunakan primitif Qiskit Runtime
* Proses pasca, kembalikan hasil dalam format klasik

Beberapa penjelasan tentang rangka kerja akan ditemui sepanjang kursus ini, tetapi untuk penjelasan lengkap dengan contoh corak Qiskit, lihat [dokumentasi ini](/guides/intro-to-patterns).

Kami mengesyorkan merancang untuk meluangkan kira-kira 6-10 jam untuk menyelesaikan kursus ini. Video dan bacaan kemungkinan memerlukan masa dua jam untuk diselesaikan.
