# Gambaran Keseluruhan {#overview}

!["Hero image"](/learning/images/courses/quantum-safe-cryptography/hero.avif)

Pengenalan praktikal kepada kriptografi selamat-kuantum ialah kursus dalam talian percuma di IBM Quantum&reg; Learning dan berfungsi sebagai panduan asas kepada konsep-konsep asas dalam kriptografi selamat-kuantum.

Kursus ini dicipta untuk pembangun yang berminat untuk meningkatkan keselamatan aplikasi mereka dan menampilkan pelajaran pelbagai mod serta contoh kod interaktif dalam empat bidang teras kriptografi: fungsi cincang kriptografi, kriptografi kunci simetri, kriptografi kunci asimetri, dan kriptografi selamat-kuantum.

Dengan mengambil kursus ini, pembangun akan belajar bagaimana landskap risiko keselamatan siber sedang berkembang serta meneroka pendekatan semasa yang bakal membentuk era kuantum.
