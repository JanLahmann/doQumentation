## Uji pengetahuan kamu {#test-your-knowledge}

Selepas menyelesaikan bacaan dan aktiviti dalam kursus ini, klik butang di bawah untuk pergi ke IBM&reg; Training, di mana kamu boleh menduduki peperiksaan lencana untuk Algoritma penepatan kuantum.

Peperiksaan terdiri daripada 20 soalan, dan kamu mesti menjawab sekurang-kurangnya 16 soalan dengan betul untuk lulus. Setelah mendapat markah lulus, lencana kamu akan dihantar secara automatik ke e-mel yang kamu gunakan untuk mendaftar dengan IBM&reg; Training.

Pastikan kamu bersedia untuk menduduki peperiksaan. Jika kamu tidak mendapat markah lulus dalam dua percubaan, peperiksaan akan dikunci buat sementara waktu.

Notis: Peperiksaan di laman IBM Training ini mungkin tidak tersedia buat sementara waktu dalam beberapa hari akan datang, disebabkan penyelenggaraan.
