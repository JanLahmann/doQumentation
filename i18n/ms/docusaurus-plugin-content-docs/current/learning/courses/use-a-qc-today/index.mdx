# Gambaran Keseluruhan {#overview}

!["Imej hero"](/learning/images/courses/use-a-qc-today/hero.svg)

Selamat datang ke *Guna komputer kuantum hari ini*! Dalam kursus ini, kamu terus mula menggunakan komputer kuantum, sambil belajar prinsip-prinsip asas pengkomputeran kuantum dalam perjalanan.

Kursus ini direka untuk orang yang ingin tahu dan orang yang sibuk. Kamu tak perlu ijazah fizik atau bertahun-tahun pengalaman pengaturcaraan. Jika kamu boleh ikuti notebook dan klik "Run," kamu dah bersedia.
