# Gambaran keseluruhan {#overview}

!["Imej hero"](/learning/images/courses/quantum-machine-learning/hero.svg)

Kursus ini meneroka bagaimana pengkomputeran kuantum boleh memainkan peranan yang menarik dalam pembelajaran mesin. Kursus ini akan menawarkan beberapa ulasan pendekatan pembelajaran mesin klasik, tetapi juga mengandaikan kebiasaan dengan kaedah-kaedah ini. Fokus utama adalah bagaimana pengkomputeran kuantum boleh sesuai dengan aliran kerja pembelajaran mesin yang sedia ada. Topik-topik termasuk pengekodan data, kernel kuantum, dan Circuit kuantum variasi.

## Peperiksaan {#exam}

Untuk mendapatkan lencana anda untuk Pembelajaran mesin kuantum, duduki peperiksaan di IBM&reg; Training. Peperiksaan ini bertujuan untuk diduduki selepas membaca pelajaran dalam kursus ini. Selepas anda lulus peperiksaan, anda akan diberitahu oleh Credly bahawa anda telah mendapat lencana.
