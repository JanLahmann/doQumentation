## Uji pengetahuan anda {#test-your-knowledge}

Selepas menyelesaikan bacaan dan aktiviti dalam kursus ini, klik butang di bawah untuk navigasi ke IBM&reg; Training, di mana anda boleh menduduki peperiksaan lencana untuk Pembelajaran mesin kuantum.

Peperiksaan terdiri daripada 20 soalan, dan anda mesti menjawab sekurang-kurangnya 16 dengan betul untuk lulus. Setelah mendapat markah lulus, lencana anda akan dihantar secara automatik ke e-mel yang anda gunakan untuk mendaftar dengan IBM Training.

Pastikan anda bersedia untuk menduduki peperiksaan. Jika anda tidak mendapat markah lulus dalam dua percubaan, peperiksaan akan dikunci buat sementara waktu.

Notis: Peperiksaan-peperiksaan ini di laman IBM Training mungkin tidak tersedia buat sementara waktu dalam beberapa hari akan datang, disebabkan penyelenggaraan.
