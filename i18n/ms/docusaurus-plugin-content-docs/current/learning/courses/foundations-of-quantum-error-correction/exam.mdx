# Peperiksaan dan lencana {#exam-and-badge}

![Lencana berbentuk persegi yang menyatakan nama kursus ini, tahap lanjutan, dan diterbitkan oleh IBM Quantum&reg;.](/learning/images/courses/foundations-of-quantum-error-correction/exam/badge.avif)

## Tinjauan pasca-kursus {#post-course-survey}

Tahniah kerana telah menamatkan kursus "Asas pembetulan ralat kuantum" oleh IBM! Sila luangkan masa sebentar untuk membantu kami menambah baik kursus ini dengan mengisi [tinjauan ringkas](https://your.feedback.ibm.com/jfe/form/SV_9WuCWHSXP8rXo7Y) berikut. Maklum balas anda akan digunakan untuk meningkatkan tawaran kandungan dan pengalaman pengguna kami. Terima kasih!

## Uji pengetahuan anda {#test-your-knowledge}

Setelah selesai membaca dan melakukan aktiviti dalam kursus ini, klik butang di bawah untuk pergi ke IBM&reg; Training, di mana anda boleh menduduki peperiksaan lencana untuk Asas pembetulan ralat kuantum.

Peperiksaan terdiri daripada 20 soalan, dan anda mesti menjawab sekurang-kurangnya 16 dengan betul untuk lulus. Setelah mendapat markah lulus, lencana anda akan dihantar secara automatik ke e-mel yang anda gunakan untuk mendaftar dengan IBM&reg; Training.

Pastikan anda bersedia untuk menduduki peperiksaan. Jika anda tidak mendapat markah lulus dalam dua percubaan, peperiksaan akan dikunci buat sementara waktu.
