## Uji pengetahuan anda {#test-your-knowledge}

Setelah menamatkan bacaan dan aktiviti dalam kursus ini, klik butang di bawah untuk ke IBM&reg; Training, di mana anda boleh menduduki peperiksaan lencana untuk Asas maklumat kuantum.

Peperiksaan terdiri daripada 20 soalan, dan anda mesti menjawab sekurang-kurangnya 16 dengan betul untuk lulus. Setelah menerima markah lulus, lencana anda akan dihantar secara automatik ke e-mel yang anda gunakan untuk mendaftar dengan IBM&reg; Training.

Pastikan anda bersedia untuk menduduki peperiksaan. Jika anda tidak menerima markah lulus dalam dua percubaan, peperiksaan akan dikunci buat sementara waktu.

Notis: Peperiksaan-peperiksaan di laman IBM Training ini mungkin tidak tersedia buat sementara waktu dalam beberapa hari akan datang, disebabkan penyelenggaraan.
