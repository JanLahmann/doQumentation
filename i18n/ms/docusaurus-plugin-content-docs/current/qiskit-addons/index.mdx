# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) are official extension packages that provide specialized quantum computing algorithms and techniques. Each addon focuses on a specific area and includes tutorials with executable code.

## Available Addons

| Addon | Package | Description |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Reduce circuit width and depth by cutting gates and wires |
| **SQD** | `qiskit-addon-sqd` | Sample-based Quantum Diagonalization for ground state estimation |
| **OBP** | `qiskit-addon-obp` | Operator Back-Propagation to reduce circuit depth |
| **MPF** | `qiskit-addon-mpf` | Multi-Product Formulas for improved Hamiltonian simulation |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Approximate Quantum Compilation with tensor networks |
| **PNA** | `qiskit-addon-pna` | Propagated Noise Absorption for error mitigation |
| **SLC** | `qiskit-addon-slc` | Shaded Lightcones for reducing PEC sampling overhead |

All tutorials can be executed directly in your browser using Binder, or on your own Jupyter server.

For API documentation, visit the individual addon repos on [GitHub](https://github.com/Qiskit?q=qiskit-addon).
