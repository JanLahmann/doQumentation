# Teknik lanjutan - Qiskit addons {#advanced-techniques-qiskit-addons}

Qiskit addons adalah koleksi keupayaan penyelidikan untuk membolehkan penemuan algoritma pada skala utiliti. Komponen perisian modular ini dibina atas asas Qiskit yang berprestasi tinggi dan boleh disambungkan ke dalam [aliran kerja](/guides/intro-to-patterns) untuk menskalakan atau mereka bentuk algoritma kuantum baru. Halaman ini menonjolkan alat yang tersedia merentasi kategori fungsi utama untuk membantu anda memilih keupayaan yang relevan semasa membina aliran kerja anda.
