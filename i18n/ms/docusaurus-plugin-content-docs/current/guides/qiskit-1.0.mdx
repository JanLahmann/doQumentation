# Panduan migrasi Qiskit 1.0 {#qiskit-10-migration-guide}

Qiskit 1.0 membawa jaminan kestabilan baharu dan peningkatan prestasi, tetapi ia
memecahkan keserasian dengan semua versi Qiskit sebelumnya. Lihat
panduan berikut tentang mengemas kini persekitaran dan kod anda untuk berfungsi dengan Qiskit v1.0.

- [Pasang pakej Qiskit v1.0 baharu](/guides/qiskit-1.0-installation) untuk menaik taraf persekitaran Python dan kebergantungan projek.
- [Perubahan ciri Qiskit v1.0](/guides/qiskit-1.0-features) untuk mengemas kini kod anda.
- [Perubahan pakej yang memecah dalam Qiskit v1.0](/guides/metapackage-migration) untuk memahami struktur pakej baharu dan lama.

Baca lebih lanjut tentang manfaat Qiskit v1.0 di [blog IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
