# Pengenalan kepada primitif {#introduction-to-primitives}

Primitif dicipta untuk memudahkan tugas paling biasa dalam komputer kuantum: iaitu, pensampelan keadaan kuantum dan pengiraan nilai jangkaan. Primitif Qiskit Runtime pertama ([`EstimatorV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2) dan [`SamplerV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2)) adalah implementasi kelas asas [primitif](/guides/primitives) Qiskit. Ia menyediakan implementasi yang lebih canggih (contohnya, dengan memasukkan pengurangan ralat) sebagai perkhidmatan berasaskan awan dan digunakan untuk mengakses perkakasan IBM Quantum&reg;.

Primitif Qiskit Runtime terbaru, [Executor](https://qiskit.github.io/qiskit-ibm-runtime/stubs/qiskit_ibm_runtime.Executor.html) (yang masih dalam beta), menyediakan antara muka peringkat lebih rendah yang memberi lebih keterlihatan dan kawalan tanpa mengorbankan prestasi.

## Langkah seterusnya {#next-steps}

<Admonition type="tip" title="Cadangan">
- Ketahui tentang [primitif Qiskit](/guides/primitives) yang menjadi asas primitif Qiskit Runtime.
- Semak contoh terperinci [Estimator](/guides/estimator-examples), [Sampler](/guides/sampler-examples), atau [Executor](/guides/executor-examples).
- Lihat butiran input dan output untuk [Estimator](/guides/estimator-input-output), [Executor](/guides/executor-input-output), dan [Sampler](/guides/sampler-input-output).
- Amalkan penggunaan primitif melalui [pelajaran Fungsi kos](/learning/courses/variational-algorithm-design/cost-functions) dalam IBM Quantum Learning.
- Lihat [rujukan API EstimatorV2](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2), [rujukan API SamplerV2](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2), dan [rujukan API Executor](https://qiskit.github.io/qiskit-ibm-runtime/stubs/qiskit_ibm_runtime.Executor.html).
- Baca [Migrasi ke primitif V2](/guides/v2-primitives).
</Admonition>
