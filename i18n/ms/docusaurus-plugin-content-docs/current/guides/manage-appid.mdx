# Urus pengguna penyedia ID {#manage-id-provider-users}

Anda boleh berintegrasi dengan penyedia identiti luaran (IDP) anda untuk mengesahkan pengguna luaran ke akaun IBM Cloud anda secara selamat. Dengan menggunakan penyedia luaran, anda boleh menyediakan cara untuk pengguna dalam syarikat anda menggunakan log masuk tunggal (SSO). Anda boleh menghubungkan akaun awan anda ke IDP luaran dengan salah satu cara berikut:

- Gunakan [penyedia perkhidmatan (SP) SAML IBM Cloud&reg;](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-cloud-saml)
- Gunakan [App ID](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-appid) (mungkin dikenakan kos)
- [Federasi pengguna anda ke semua produk IBM&reg; dengan IBMid](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-ibmid)

<Admonition type="tip">
Selepas menghubungkan IDP anda, anda boleh menyediakan peraturan dinamik supaya anda tidak perlu menambah setiap pengguna secara manual ke setiap kumpulan apabila mereka menyertai. Lihat [Mendayakan pengesahan daripada penyedia identiti luaran](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#app-id-dynamic-rules) untuk arahan.
</Admonition>
