# Pengelogan untuk Qiskit Runtime {#logging-for-qiskit-runtime}

Perkhidmatan IBM Cloud&reg;, seperti Qiskit Runtime, menjana log platform yang boleh anda gunakan untuk menyiasat aktiviti luar biasa dan tindakan kritikal dalam akaun anda, serta menyelesaikan masalah.

Anda boleh menggunakan [IBM Cloud Logs Routing,](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about) iaitu perkhidmatan platform, untuk menghalakan log platform dalam akaun anda ke destinasi pilihan anda dengan mengkonfigurasi penyewa yang menentukan ke mana log platform dihantar. Untuk maklumat lanjut, lihat [Tentang Logs Routing.](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about)

Anda boleh menggunakan [IBM Cloud Logs](https://cloud.ibm.com/docs/cloud-logs) untuk menvisualisasikan dan memberi amaran tentang log platform yang dijana dalam akaun anda dan dihalakan oleh IBM Cloud Logs Routing ke instans IBM Cloud Logs.

## Lokasi di mana log platform dijana {#locations-where-platform-logs-are-generated}

IBM Cloud Container Registry menjana log platform yang dipaparkan dalam instans pengelogan anda. Jadual berikut menyenaraikan lokasi di mana pengumpulan automatik log perkhidmatan Container Registry didayakan.

| Lokasi                   | Adakah log perkhidmatan platform tersedia? |
|----------------------------|---------------------------------------|
| Washington, D.C. (us-east) |                  Ya                  |
| Frankfurt (eu-de)          |                  Ya                  |

## Log platform yang dijana {#platform-logs-that-are-generated}

IBM Qiskit Runtime menjana log perkhidmatan platform yang dipaparkan dalam instans pengelogan anda untuk kes-kes tertentu berikut:

- Apabila status kerja berubah.

## Melihat log daripada halaman Pemerhatian {#viewing-logs-from-the-observability-page}

Untuk maklumat tentang melancarkan antara muka IBM Cloud Logs, lihat **Melancarkan UI** dalam [dokumentasi IBM Cloud Logs.](https://cloud.ibm.com/docs/cloud-logs?topic=cloud-logs-instance-launch)

## Medan mengikut jenis log {#fields-by-log-type}

Untuk maklumat tentang medan yang disertakan dalam setiap log platform, lihat [Medan yang disertakan dalam log platform.](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about-platform-logs#about-platform-logs-2)

Medan-medan berikut disertakan dalam rekod log.

| Medan           | Jenis     | Penerangan                                                              |
|-----------------|----------|--------------------------------------------------------------------------|
| logSourceCRN    | Diperlukan | Menentukan akaun dan instans log aliran di mana log diterbitkan.    |
| saveServiceCopy | Diperlukan | Menentukan sama ada IBM menyimpan salinan rekod untuk tujuan operasi. |
| message         | Diperlukan | Penerangan log yang dijana.                                |
