# Dasar keselamatan repositori {#repository-security-policies}

Adalah sangat penting untuk anda memastikan sistem anda selamat seboleh mungkin. Oleh itu, adalah sangat disyorkan agar anda menggunakan alat keselamatan seperti [Dependabot](https://github.com/dependabot) yang secara automatik memberitahu anda apabila kebergantungan sudah lapuk. Lihat dasar keselamatan bagi repositori GitHub berikut yang berkaitan dengan IBM Quantum&reg;:

- [Qiskit](https://github.com/Qiskit/qiskit/security)
- [Qiskit IBM Runtime](https://github.com/Qiskit/qiskit-ibm-runtime/security)
- [Tambahan Qiskit pemotong Circuit](https://github.com/Qiskit/qiskit-addon-cutting/security)
- [Tambahan Qiskit pemetaan fermionik](https://github.com/Qiskit/qiskit-fermions/blob/main/SECURITY.md)
- [Tambahan Qiskit formula berbilang produk (MPF)](https://github.com/Qiskit/qiskit-addon-mpf/security)
- [Tambahan Qiskit perambatan balik pengoperasi (OBP)](https://github.com/Qiskit/qiskit-addon-obp/security)
- [Tambahan Qiskit pemetaan pengoptimuman](https://github.com/Qiskit/qiskit-addon-opt-mapper/blob/main/SECURITY.md)
- [Tambahan Qiskit perambatan Pauli](https://github.com/Qiskit/pauli-prop/blob/main/SECURITY.md)
- [Tambahan Qiskit penyerapan hingar terambat (PNA)](https://github.com/Qiskit/qiskit-addon-pna/blob/main/SECURITY.md)
- [Utiliti tambahan Qiskit](https://github.com/Qiskit/qiskit-addon-utils/security)
- [Tambahan Qiskit pepenjuru kuantum berasaskan sampel (SQD)](https://github.com/Qiskit/qiskit-addon-sqd/security)
- [Tambahan Qiskit kon cahaya berlorek](https://github.com/Qiskit/qiskit-addon-slc/blob/main/SECURITY.md)
