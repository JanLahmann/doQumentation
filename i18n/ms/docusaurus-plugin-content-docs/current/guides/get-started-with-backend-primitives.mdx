# Mulakan dengan primitif Backend {#get-started-with-the-backend-primitives}

Tidak seperti primitif khusus penyedia, primitif Backend adalah implementasi generik yang boleh digunakan dengan objek `backend` mana-mana, selagi ia melaksanakan antara muka [`BackendV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.providers.BackendV2). Sesetengah penyedia mengimplementasikan primitif secara asli. Lihat [halaman Ekosistem Qiskit](https://www.ibm.com/quantum/ecosystem?tag=provider) untuk butiran.

<span id="estimator"></span>
## Mulakan dengan primitif Backend Estimator {#get-started-with-the-estimator-backend-primitive}

Primitif Estimator boleh dijalankan dengan mana-mana penyedia menggunakan kelas [`qiskit.primitives.BackendEstimatorV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendEstimatorV2). Walau bagaimanapun, ia tidak menawarkan implementasi pengurangan ralat ukuran atau Gate "out-of-the-box", kerana primitif Backend direka untuk berjalan secara tempatan dalam mesin pengguna.

### Contoh: {#example}

```python
from qiskit.primitives import BackendEstimatorV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
estimator = BackendEstimatorV2(backend)
```

<span id="sampler"></span>
## Mulakan dengan primitif Backend Sampler {#get-started-with-the-sampler-backend-primitive}

Primitif Sampler boleh dijalankan dengan mana-mana penyedia menggunakan [`qiskit.primitives.BackendSamplerV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendSamplerV2). Walau bagaimanapun, ia memerlukan Backend yang menyokong pilihan `memory`.

### Contoh: {#example-1}

```python
from qiskit.primitives import BackendSamplerV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
sampler = BackendSamplerV2(backend)
```
