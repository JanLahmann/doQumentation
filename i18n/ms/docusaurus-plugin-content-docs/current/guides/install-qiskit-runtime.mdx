# Pasang klien Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime ialah perkhidmatan berasaskan awan untuk melaksanakan pengiraan kuantum pada perkakasan IBM Quantum&reg;. Panduan ini menunjukkan cara memasang `qiskit-ibm-runtime`, iaitu pakej klien untuk mengakses perkhidmatan Qiskit Runtime.

Sebelum memasang klien Qiskit Runtime, ikuti arahan dalam panduan berikut:

1. [Pasang Qiskit](/guides/install-qiskit).
2. [Sediakan akaun IBM Cloud kamu](cloud-setup).

Jalankan arahan ini untuk memasang Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Kekal terkini dengan versi terbaru">
Semak secara berkala [nota pelepasan Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) untuk melihat keluaran baru. Kami syorkan agar kamu selalu mengemaskini keperluan `qiskit-ibm-runtime` dengan menjalankan `pip install -U qiskit-ibm-runtime`.
</Admonition>
