# Persekitaran makmal dalam talian {#online-lab-environments}

Selain memuat turun dan memasang Qiskit untuk bekerja pada mesin tempatan anda, terdapat beberapa pilihan tersedia bagi mereka yang lebih suka bekerja dalam persekitaran pembangunan dalam talian.

Beberapa pembekal menawarkan persekitaran Jupyter dalam talian dengan Qiskit diprapasang untuk onboarding yang cepat dan mudah. Pembekal yang disyorkan ini termasuk pembekal perkhidmatan awan OVHcloud dan platform awan pembangunan perisian kuantum qBraid.

## Persekitaran notebook yang disyorkan dengan Qiskit diprapasang {#recommended-notebook-environments-with-qiskit-preinstalled}

Persekitaran notebook Jupyter dalam talian tambahan seperti IBM Watson&reg; Studio, Google Colab, dan Microsoft Azure Machine Learning Studio boleh digunakan untuk bekerja dengan Qiskit dengan mengikuti arahan yang sesuai.

### Notebook AI OVHcloud {#ovhcloud-ai-notebooks}

Pengguna OVHcloud Public Cloud dan OVHcloud Startup boleh menggunakan notebook kuantum OVHcloud mereka untuk mengakses Qiskit dan menyerahkan kerja ke QPU IBM&reg; dengan akaun IBM Quantum&reg; atau IBM Cloud&reg; mereka. Untuk mengetahui lebih lanjut, lawati halaman [Dokumentasi dan tutorial OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
Notebook AI OVHcloud tidak tersedia di sesetengah rantau, seperti Amerika Syarikat. Selain itu, antara muka pengguna mungkin kelihatan berbeza di rantau yang berbeza.
</Admonition>

Ikuti langkah-langkah ini untuk bermula:

1. [Buat akaun IBM Cloud.](./cloud-setup)
1. [Sediakan akaun OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [Buat projek OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. Ikuti langkah-langkah dalam [Lancarkan Notebook AI pertama anda,](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) pilih **Qiskit** untuk rangka kerja.

### qBraid Lab {#qbraid-lab}

Pengguna Qiskit boleh menggunakan persekitaran Python yang telah dikonfigurasi oleh qBraid dengan mengikuti arahan berikut. Untuk maklumat lanjut, lihat [Dokumentasi qBraid.](https://docs.qbraid.com/home/introduction)

1. [Sediakan akaun percuma qBraid.](https://docs.qbraid.com/home/account)
3. [Buat persekitaran Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. Lancarkan qBraid Lab, klik ikon **ENVS** di sebelah kanan, klik **+ ADD** di bahagian atas bar sisi baru, dan cari Qiskit. Klik versi yang ingin anda gunakan, kemudian klik **Install**.
4. Buat notebook Jupyter dengan mengklik **Python 3 [Qiskit]** dalam bahagian Notebook.
4. Anda boleh memasang sebarang pakej Python tambahan (seperti Qiskit v1.2) dalam sel notebook dengan menjalankan `%pip install [package_name].` Untuk maklumat lanjut, lihat topik [Arahan Pip (magic)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## Persekitaran notebook tambahan untuk pengguna Qiskit {#additional-notebook-environments-for-qiskit-users}

Untuk platform tanpa Qiskit dikonfigurasi terlebih dahulu, arahan persediaan adalah serupa, kecuali anda perlu memasang Qiskit secara manual. Anda akan mencipta akaun, memulakan projek baru (atau notebook atau ruang kerja, bergantung pada platform), dan memasang Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [Sediakan percubaan percuma Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [Buat Projek Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. Untuk memasang pakej Qiskit dan kebergantungan yang diperlukan, buka notebook baru dan jalankan arahan berikut:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [Buat akaun Google.](https://support.google.com/accounts/answer/27441?hl)
2. Pergi ke [halaman pendaratan Google Colab](https://colab.research.google.com/) dan log masuk dengan akaun Google anda.
3. [Buat notebook Colab](https://colab.research.google.com/#create=true) dan pasang Qiskit dengan menjalankan arahan berikut:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [Buat akaun Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [Buat ruang kerja.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [Buat notebook Jupyter dalam ruang kerja anda.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. Dari notebook, jalankan arahan ini untuk memasang Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
