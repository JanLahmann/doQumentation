# Pasang klien Qiskit Runtime dari sumber {#install-the-qiskit-runtime-client-from-source}

Apabila kamu memasang Qiskit Runtime dari sumber, kamu boleh mengakses versi pembangunan semasa, bukan menggunakan versi dalam repositori Python Package Index (PyPI). Oleh itu, kamu boleh memeriksa dan melanjutkan versi terbaru kod dengan lebih cekap.

## Cipta dan aktifkan persekitaran maya baru {#create-and-activate-a-new-virtual-environment}

1. Navigasi ke direktori projek kamu dan cipta persekitaran minimal dengan hanya Python yang dipasang di dalamnya.

        <OperatingSystemTabs>
          <TabItem value="mac" label="macOS">
            ```shell
            python3 -m venv .venv
            ```
          </TabItem>

          <TabItem value="linux" label="Linux">
            ```shell
            python3 -m venv .venv
            ```
          </TabItem>

          <TabItem value="win" label="Windows">
            ```text
            python -m venv .venv
            ```
          </TabItem>
         </OperatingSystemTabs>

1. Aktifkan persekitaran baru kamu.

         <OperatingSystemTabs>
           <TabItem value="mac" label="macOS">
             ```shell
             source .venv/bin/activate
             ```
           </TabItem>

           <TabItem value="linux" label="Linux">
             ```shell
             source .venv/bin/activate
             ```
           </TabItem>

           <TabItem value="win" label="Windows">
             ```text
             .venv\Scripts\Activate.ps1
             ```
           </TabItem>
         </OperatingSystemTabs>

## Pasang Qiskit Runtime {#install-qiskit-runtime}

Ikuti langkah-langkah ini untuk memasang Qiskit Runtime:

1. Klon repositori Qiskit Runtime.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. Tukar ke direktori `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. Pasang `qiskit-runtime`. Kami syorkan menggunakan [persekitaran maya](https://docs.python.org/3/library/venv.html) untuk mengelakkan pencemaran pemasangan Python global kamu.

  * **Pemasangan standard**:

    ```bash
    pip install .
    ```

  * **Mod boleh sunting**: Dalam mod ini, kamu tidak perlu memasang semula Qiskit apabila ada perubahan kod pada projek.

    ```bash
    pip install -e .
    ```

    Dalam mod boleh sunting, sambungan yang dikompil dibina dalam _mod nyahpepijat_ tanpa pengoptimuman.

4. (Pilihan) Jika kamu ingin menjalankan ujian atau semakan linting, pasang keperluan pembangun. Kami syorkan menggunakan [persekitaran maya](https://docs.python.org/3/library/venv.html) untuk mengelakkan pencemaran pemasangan Python global kamu.

```bash
pip install -e ".[dev]"
```
