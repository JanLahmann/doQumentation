Untuk mendapatkan maklumat tentang keadaan qubit, anda boleh _mengukurnya_ ke dalam [bit klasik](https://docs.quantum.ibm.com/api/qiskit/circuit#qiskit.circuit.Clbit). Dalam Qiskit, pengukuran dilakukan dalam asas pengiraan, iaitu asas Pauli-$Z$ satu-qubit. Oleh itu, pengukuran menghasilkan 0 atau 1, bergantung pada pertindihan dengan eigenstate Pauli-$Z$ $|0\rangle$ dan $|1\rangle$:

$$
|q\rangle \xrightarrow{measure}\begin{cases}
      0 (\text{outcome}+1), \text{with probability } p_0=|\langle q|0\rangle|^{2}\text{,} \\
      1 (\text{outcome}-1), \text{with probability } p_1=|\langle q|1\rangle|^{2}\text{.}
    \end{cases}
$$

## Gunakan pengukuran pada Circuit {#apply-a-measurement-to-a-circuit}

Terdapat beberapa cara untuk menerapkan pengukuran pada Circuit:

### Kaedah `QuantumCircuit.measure` {#quantumcircuitmeasure-method}

Gunakan kaedah [`measure`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure) untuk mengukur [`QuantumCircuit`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#quantumcircuit-class).

Contoh:

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(5, 5)
qc.x(0)
qc.x(1)
qc.x(4)
qc.measure(
    range(5), range(5)
)  #  Measures all qubits into the corresponding clbit.
```

### Kelas `Measure` {#measure-class}

Kelas [Measure](https://docs.quantum.ibm.com/api/qiskit/circuit#qiskit.circuit.Measure) Qiskit mengukur qubit yang dinyatakan.

```python
from qiskit.circuit import Measure

qc = QuantumCircuit(3, 1)
qc.x([0, 1])
qc.append(Measure(), [0], [0])  # measure qubit 0 into clbit 0
```

### Kaedah `QuantumCircuit.measure_all` {#quantumcircuitmeasure-all-method}

Untuk mengukur semua qubit ke dalam bit klasik yang sepadan, gunakan kaedah [`measure_all`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure_all). Secara lalai, kaedah ini menambah bit klasik baru dalam `ClassicalRegister` untuk menyimpan pengukuran ini.

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(3, 1)
qc.x([0, 2])
qc.measure_all()  # Measure all qubits.
```

### Kaedah `QuantumCircuit.measure_active` {#quantumcircuitmeasure-active-method}

Untuk mengukur semua qubit yang tidak melahu, gunakan kaedah [`measure_active`](https://docs.quantum.ibm.com/api/qiskit/qiskit.circuit.QuantumCircuit#measure_active). Kaedah ini mencipta `ClassicalRegister` baru dengan saiz yang sama dengan bilangan qubit bukan-melahu yang diukur.

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(3, 1)
qc.x([0, 2])
qc.measure_active()  # Measure qubits that are not idle, that is, qubits 0 and 2.
```
