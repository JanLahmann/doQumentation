```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

qc.draw(output="mpl")
```

## Pilihan 1: Simulasi ideal (tak perlu akaun) {#option-1-ideal-simulation-no-account-needed}

Menggunakan `StatevectorSampler` — simulator tempatan dengan hasil yang sempurna, tanpa bunyi.

```python
from qiskit.primitives import StatevectorSampler

result = StatevectorSampler().run([qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
from qiskit.visualization import plot_histogram
plot_histogram(counts)
```

## Pilihan 2: Simulasi berbising (tak perlu akaun) {#option-2-noisy-simulation-no-account-needed}

Menggunakan `FakeManilaV2` — simulator tempatan yang meniru peranti quantum IBM sebenar, termasuk ciri-ciri bunyinya. Circuit perlu di-transpile (disesuaikan) dahulu kepada set Gate dan sambungan Qubit peranti tersebut.

```python
from qiskit_ibm_runtime import SamplerV2
from qiskit_ibm_runtime.fake_provider import FakeManilaV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

backend = FakeManilaV2()
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```

## Pilihan 3: Perkakasan quantum sebenar {#option-3-real-quantum-hardware}

Perlukan akaun IBM Quantum. Lihat [Menyediakan akaun IBM Quantum kamu](https://janlahmann.github.io/Qiskit-documentation/#setting-up-your-ibm-quantum-account) untuk maklumat lanjut.

Kalau kamu belum simpan kelayakan dalam sesi Binder ini, jalankan kod ni dulu:

```python
from qiskit_ibm_runtime import QiskitRuntimeService

QiskitRuntimeService.save_account(
    channel="ibm_quantum_platform",
    token="<your-api-key>",
    instance="<your-crn>",
    set_as_default=True,
    overwrite=True,
)
```

**Nota:** Kerja pada perkakasan sebenar mungkin ambil masa bergantung pada masa giliran. Kalau sel masih berjalan, kamu boleh semak status kerja dan tengok keputusan di [quantum.cloud.ibm.com/workloads](https://quantum.cloud.ibm.com/workloads?user=me).

```python
from qiskit_ibm_runtime import QiskitRuntimeService, SamplerV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

service = QiskitRuntimeService()
backend = service.least_busy(operational=True, simulator=False)
print(f"Running on {backend.name}")

pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```
