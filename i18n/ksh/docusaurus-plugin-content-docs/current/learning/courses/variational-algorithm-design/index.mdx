# Övverbleck {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Dä Kurs rish sich aan Lück, die ald en bäsje Erfahrung met Quantencomputing han un liere wölle, wie mer variationelle Algorithme met de Qiskit Runtime Primitives schriev. Hä pass och jot för Quantencomputing-Profis, die ehr Wesse un ehr Fähichkeite em Beresch der variationellen Algorithme ußbaue wölle.
