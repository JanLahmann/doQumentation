# Überblick {#overview}

!["Hero image"](/learning/images/courses/quantum-diagonalization-algorithms/hero.avif)

Dieser Kurs gibt an Überblick üba mehrere Ansätze zur Quantendiagonalisierung. Es wird a bissl Hintergrundwissen zu de klassischen Methoden vermittelt, die verwendet werdn oda de Quantenalgorithmen motiviert habn. Donoch wird schrittweise d'Implementierung da Quantenalgorithmen auf echten Quantencomputern durchgegangen.
