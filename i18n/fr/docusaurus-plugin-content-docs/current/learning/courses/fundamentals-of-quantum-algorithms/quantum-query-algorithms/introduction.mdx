Dans cette première leçon du cours, nous allons formuler un cadre algorithmique simple — connu sous le nom de *modèle de requête* — et explorer les avantages qu'offrent les ordinateurs quantiques dans ce cadre.

Le modèle de requête de calcul est comme une boîte de Pétri pour les idées algorithmiques quantiques.
Il est rigide et artificiel dans le sens où il ne représente pas fidèlement les types de problèmes de calcul qui nous intéressent généralement en pratique, mais il s'est néanmoins révélé incroyablement utile comme outil de développement de techniques algorithmiques quantiques.
Cela inclut celles qui alimentent les algorithmes quantiques les plus connus, tels que l'algorithme de Shor pour la factorisation des entiers.
Le modèle de requête est également un cadre très utile pour *expliquer* les techniques algorithmiques quantiques.

Après avoir présenté le modèle de requête lui-même, nous aborderons le tout premier algorithme quantique découvert, à savoir *l'algorithme de Deutsch,* ainsi qu'une extension de l'algorithme de Deutsch connue sous le nom *d'algorithme de Deutsch-Jozsa*.
Ces algorithmes démontrent des avantages quantifiables du quantique sur le classique dans le contexte du modèle de requête.
Nous examinerons ensuite un algorithme quantique connu sous le nom *d'algorithme de Simon,* qui offre un avantage plus robuste et plus convaincant du quantique sur le classique, pour des raisons qui seront expliquées lorsque nous y arriverons.
