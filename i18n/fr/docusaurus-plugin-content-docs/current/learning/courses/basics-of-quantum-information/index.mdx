# Vue d'ensemble {#overview}

!["Hero image"](/learning/images/courses/basics-of-quantum-information/hero.svg)

Bienvenue dans *Bases de l'information quantique*, le premier cours de la série *Comprendre l'information et le calcul quantiques* qui comprend les cours suivants :

- Bases de l'information quantique (ce cours)
- [Fundamentals of quantum algorithms](/learning/courses/fundamentals-of-quantum-algorithms)
- [General formulation of quantum information](/learning/courses/general-formulation-of-quantum-information)
- [Foundations of quantum error correction](/learning/courses/foundations-of-quantum-error-correction)

Ce cours commence par une introduction aux mathématiques de l'information quantique, y compris une description de l'information quantique pour les systèmes simples et multiples. Il aborde ensuite les circuits quantiques, qui fournissent une méthode standard pour décrire les calculs quantiques. Enfin, trois exemples fondamentalement importants liés au phénomène d'intrication quantique sont expliqués : la téléportation quantique, le codage superdense et le jeu CHSH (également connu sous le nom d'inégalité CHSH).

Ce cours s'adresse aux étudiants, professionnels et passionnés dans des domaines tels que l'informatique, la physique, l'ingénierie et les mathématiques qui souhaitent acquérir des connaissances sur les fondements théoriques de l'information et du calcul quantiques.

## Prérequis recommandés {#recommended-background}

Pour tirer le meilleur parti de ce cours, nous recommandons une familiarité avec l'algèbre linéaire de base, les nombres complexes et les notions mathématiques élémentaires, y compris les ensembles et les fonctions. Les sources suivantes font partie des nombreuses ressources qui couvrent ce matériel :

- [Khan Academy: Linear algebra](https://www.khanacademy.org/math/linear-algebra)

   Dans cette série de vidéos, Sal Khan présente les concepts clés de l'algèbre linéaire sur lesquels nous nous appuierons.

- Stephen Friedberg, Arnold Insel, and Lawrence Spence. [Linear Algebra](https://www.pearson.com/en-us/subject-catalog/p/linear-algebra/P200000006185/9780137515424)

   Ce livre sur l'algèbre linéaire couvre le matériel dont nous avons besoin et comprend également des annexes sur les ensembles, les fonctions et les nombres complexes.

- Sheldon Axler. [Linear Algebra Done Right](https://link.springer.com/book/10.1007/978-3-319-11080-6)

   Un texte classique sur l'algèbre linéaire adapté à ceux qui se trouvent à un niveau de premier cycle avancé ou au-delà.

- Ricky Shadrach and Rod Pierce. [Introduction to Sets](https://www.mathsisfun.com/sets/sets-introduction.html)

   Une page web de niveau débutant sur les ensembles qui peut aider certains lecteurs à se mettre à niveau.

- John K. Hunter. [An Introduction to Real Analysis: Chapter 1](https://www.math.ucdavis.edu/~hunter/intro_analysis_pdf/ch1.pdf)

   Le premier chapitre de ces notes de cours comprend une introduction plus formelle et détaillée aux ensembles et aux fonctions.
