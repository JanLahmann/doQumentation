# Vue d'ensemble {#overview}

!["Image principale"](/learning/images/courses/foundations-of-quantum-error-correction/hero.svg)

Bienvenue dans *Fondements de la correction d'erreurs quantiques,* le quatrième cours de la série *Comprendre l'information et le calcul quantiques* comprenant les cours suivants :

- [Bases de l'information quantique](/learning/courses/basics-of-quantum-information)
- [Fondamentaux des algorithmes quantiques](/learning/courses/fundamentals-of-quantum-algorithms)
- [Formulation générale de l'information quantique](/learning/courses/general-formulation-of-quantum-information)
- Fondements de la correction d'erreurs quantiques (ce cours)

Ce cours porte sur la correction d'erreurs quantiques, avec un accent sur les concepts fondamentaux.
Il commence par une explication de ce que sont les codes correcteurs d'erreurs quantiques et de leur fonctionnement.
Il aborde ensuite le phénomène de discrétisation des erreurs, le formalisme des stabilisateurs pour
décrire les codes correcteurs d'erreurs quantiques, les codes de Calderbank–Shor–Steane (CSS), et plusieurs
exemples clés de codes correcteurs d'erreurs quantiques, notamment le code torique et les codes de surface.
Le cours aborde également le calcul quantique tolérant aux fautes, dans lequel les calculs quantiques
sont effectués sur de l'information quantique corrigée des erreurs.

Ce cours est destiné aux étudiants, aux professionnels et aux passionnés dans des domaines tels que l'informatique, la physique,
l'ingénierie et les mathématiques, qui souhaitent acquérir des connaissances sur les fondements théoriques de l'information quantique
et du calcul.
