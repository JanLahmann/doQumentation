# Vue d'ensemble {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Ce cours explore comment l'informatique quantique peut jouer un rôle passionnant dans l'apprentissage automatique. Le cours offrira une révision de certaines approches classiques de l'apprentissage automatique, mais supposera également une certaine familiarité avec ces méthodes. L'accent principal sera mis sur la façon dont l'informatique quantique peut s'intégrer dans les flux de travail d'apprentissage automatique existants. Les sujets abordés incluent l'encodage des données, les noyaux quantiques et les circuits quantiques variationnels.

## Examen {#exam}

Pour obtenir ton badge en apprentissage automatique quantique, passe l'examen sur IBM&reg; Training. Cet examen est conçu pour être passé après avoir lu les leçons de ce cours. Après avoir réussi l'examen, tu seras notifié par Credly que tu as obtenu un badge.
