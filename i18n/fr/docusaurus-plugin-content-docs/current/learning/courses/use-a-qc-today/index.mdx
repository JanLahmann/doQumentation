# Vue d'ensemble {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

Bienvenue dans *Utilise un ordinateur quantique aujourd'hui* ! Dans ce cours, tu vas commencer à utiliser un ordinateur quantique immédiatement, tout en apprenant les principes fondamentaux de l'informatique quantique au fil de ta progression.

Ce cours est conçu pour les curieux et les gens pressés. Tu n'as pas besoin d'un diplôme en physique ni de plusieurs années d'expérience en programmation. Si tu sais suivre un notebook et cliquer sur « Exécuter », tu es prêt.
