# Qiskit Addons

Les [Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) sont des packages d'extension officiels qui fournissent des algorithmes et des techniques de calcul quantique spécialisés. Chaque addon se concentre sur un domaine spécifique et inclut des tutoriels avec du code exécutable.

## Addons disponibles {#available-addons}

| Addon | Package | Description |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Réduis la largeur et la profondeur du Circuit en coupant des Gates et des fils |
| **SQD** | `qiskit-addon-sqd` | Diagonalisation Quantique par Échantillonnage pour l'estimation de l'état fondamental |
| **OBP** | `qiskit-addon-obp` | Rétropropagation d'Opérateur pour réduire la profondeur du Circuit |
| **MPF** | `qiskit-addon-mpf` | Formules Multi-Produits pour une simulation hamiltonienne améliorée |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Compilation Quantique Approximative avec des réseaux de tenseurs |
| **PNA** | `qiskit-addon-pna` | Absorption du Bruit Propagé pour l'atténuation des erreurs |
| **SLC** | `qiskit-addon-slc` | Cônes de lumière ombrés pour réduire la surcharge d'échantillonnage PEC |

Tous les tutoriels peuvent être exécutés directement dans ton navigateur via Binder, ou sur ton propre serveur Jupyter.

Pour la documentation de l'API, visite les dépôts individuels des addons sur [GitHub](https://github.com/Qiskit?q=qiskit-addon).
