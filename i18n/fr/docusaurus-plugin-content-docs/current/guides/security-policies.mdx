# Politiques de sécurité des dépôts {#repository-security-policies}

Il est absolument essentiel que tu gardes tes systèmes aussi sécurisés que possible. Il est donc fortement recommandé d'utiliser des outils de sécurité comme [Dependabot](https://github.com/dependabot) qui te notifient automatiquement lorsque des dépendances sont obsolètes. Consulte les politiques de sécurité des dépôts GitHub suivants liés à IBM Quantum&reg; :

- [Qiskit](https://github.com/Qiskit/qiskit/security)
- [Qiskit IBM Runtime](https://github.com/Qiskit/qiskit-ibm-runtime/security)

- [Extension Qiskit de correspondance fermionique](https://github.com/Qiskit/qiskit-fermions/blob/main/SECURITY.md)

- [Extension Qiskit de rétropropagation d'opérateurs (OBP)](https://github.com/Qiskit/qiskit-addon-obp/security)
- [Extension Qiskit de correspondance par optimisation](https://github.com/Qiskit/qiskit-addon-opt-mapper/blob/main/SECURITY.md)
- [Extension Qiskit de propagation de Pauli](https://github.com/Qiskit/pauli-prop/blob/main/SECURITY.md)
