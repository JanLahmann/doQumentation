# Installer le client Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime est le service basé sur le cloud pour l'exécution de calculs quantiques sur le matériel IBM Quantum&reg;. Ce guide t'explique comment installer `qiskit-ibm-runtime`, qui est le package client pour accéder au service Qiskit Runtime.

Avant d'installer le client Qiskit Runtime, suis les instructions de ces guides :

1. [Installer Qiskit](/guides/install-qiskit). 
2. [Configurer ton compte IBM Cloud](cloud-setup).

Exécute cette commande pour installer Qiskit Runtime :

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Reste à jour avec les dernières versions">
Vérifie régulièrement les [notes de version de Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) pour voir les nouvelles versions. Nous recommandons de mettre fréquemment à jour tes dépendances pour `qiskit-ibm-runtime` en exécutant `pip install -U qiskit-ibm-runtime`.
</Admonition>
