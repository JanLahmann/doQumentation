# Premiers pas avec la diagonalisation quantique basée sur l'échantillonnage (SQD) {#getting-started-with-sample-based-quantum-diagonalization-sqd}
<details>
<summary><b>Versions des paquets</b></summary>

Le code sur cette page a été développé en utilisant les exigences suivantes.
Nous te recommandons d'utiliser ces versions ou des versions plus récentes.

```
qiskit-addon-sqd~=0.10.0
pyscf~=2.8.0; sys.platform != 'win32'
```
</details>
<Admonition type="caution" title="Note pour les utilisateurs de Windows">

Cet exemple utilise le paquet `pyscf`, une bibliothèque de simulation chimique basée sur Python, qui n'est pas nativement prise en charge sur Windows. Par conséquent, les exemples ci-dessous ne fonctionneront pas.

Pour exécuter les fragments de code ci-dessous, les utilisateurs de Windows doivent créer un environnement Python en utilisant le Subsystème Windows pour Linux, comme suggéré dans le [guide d'installation de `pyscf`.](https://pyscf.org/user/install.html)

</Admonition>

Ce guide montre un exemple simple pour commencer avec le paquet `qiskit-addon-sqd`. Dans cet exemple, tu peux utiliser SQD pour obtenir une approximation de l'état fondamental de la molécule $N_2$ à l'équilibre.

Pour commencer, rappelle-toi que le Hamiltonien générique d'électrons en interaction a la forme :

$$ \hat{H} = \sum_{\substack{pr\\ \sigma}}h_{pr} \hat{a}_{p\sigma}^\dagger \hat{a}_{r\sigma} + \sum_{\substack{prqs\\ \sigma\tau}} \frac{1}{2}\left(pr|qs\right)\hat{a}_{p\sigma}^\dagger \hat{a}_{q\tau}^\dagger \hat{a}_{s\tau}\hat{a}_{r\sigma} $$

où $\hat{a}_{p\sigma}^\dagger$ et $\hat{a}_{p\sigma}$ sont les opérateurs de création et d'annihilation fermioniques associés à l'orbitale $p$-ième avec le spin $\sigma$, et $h_{pr}$ et $\left(pr|qs\right)$ sont les intégrales électroniques à un et deux corps.
## Préparer les informations sur la molécule {#prepare-molecule-information}

Pour commencer, spécifie la molécule et ses propriétés en utilisant `pyscf` et les intégrales électroniques stockées dans `n2_fci.txt`.
<Admonition type="note">
Cet exemple utilise des données pré-générées dans un fichier appelé `n2_fci.txt`, qui contient les intégrales électroniques à un et deux corps. Pour exécuter les cellules de code ci-dessous sur une machine locale, copie et colle les données dans un fichier portant le même nom. Cet exemple nécessite également que le paquet `pyscf` soit installé.
</Admonition>
