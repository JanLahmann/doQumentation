# Techniques avancées - Extensions Qiskit {#advanced-techniques---qiskit-addons}

Les extensions Qiskit sont un ensemble de fonctionnalités de recherche permettant la découverte d'algorithmes à l'échelle utilitaire. Ces composants logiciels modulaires s'appuient sur les bases performantes de Qiskit et peuvent s'intégrer dans un [workflow](/guides/intro-to-patterns) pour faire évoluer ou concevoir de nouveaux algorithmes quantiques. Cette page présente les outils disponibles dans les principales catégories fonctionnelles pour t'aider à choisir les fonctionnalités pertinentes lors de la construction de tes workflows.
