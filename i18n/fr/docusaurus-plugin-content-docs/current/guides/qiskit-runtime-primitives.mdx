# Introduction aux primitives {#introduction-to-primitives}

Les primitives ont été créées pour simplifier les tâches les plus courantes des ordinateurs quantiques : à savoir, l'échantillonnage des états quantiques et le calcul des valeurs d'espérance. Les premières primitives Qiskit Runtime ([`EstimatorV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2) et [`SamplerV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2)) sont des implémentations des classes de base des [primitives Qiskit](/guides/primitives). Elles fournissent une implémentation plus sophistiquée (par exemple, en incluant l'atténuation d'erreurs) en tant que service basé sur le cloud et sont utilisées pour accéder au matériel IBM Quantum&reg;.

La plus récente primitive Qiskit Runtime, [Executor](https://qiskit.github.io/qiskit-ibm-runtime/stubs/qiskit_ibm_runtime.Executor.html) (en version bêta), fournit une interface de plus bas niveau qui offre plus de visibilité et de contrôle sans sacrifier les performances.
