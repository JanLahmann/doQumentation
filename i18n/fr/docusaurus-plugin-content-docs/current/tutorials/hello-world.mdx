Construis un [état de Bell](https://en.wikipedia.org/wiki/Bell_state) (deux qubits intriqués) et exécute-le de trois manières :

1. **Simulation idéale** — résultats parfaits, aucun compte requis
2. **Simulation bruitée** — simule un véritable appareil, aucun compte requis
3. **Matériel quantique réel** — nécessite un compte IBM Quantum gratuit (étapes de configuration ci-dessous)
## Construire le circuit {#build-the-circuit}

```python
# Added by doQumentation — required packages for this notebook
!pip install -q qiskit qiskit-ibm-runtime
```

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

qc.draw(output="mpl")
```

## Option 1 : Simulation idéale (aucun compte requis) {#option-1-ideal-simulation-no-account-needed}

Utilise `StatevectorSampler` — un simulateur local avec des résultats parfaits, sans bruit.

```python
from qiskit.primitives import StatevectorSampler

result = StatevectorSampler().run([qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
from qiskit.visualization import plot_histogram
plot_histogram(counts)
```

## Option 2 : Simulation bruitée (aucun compte requis) {#option-2-noisy-simulation-no-account-needed}

Utilise `FakeManilaV2` — un simulateur local qui reproduit le comportement d'un véritable appareil quantique IBM, y compris ses caractéristiques de bruit. Le circuit doit d'abord être transpilé (adapté) au jeu de portes et à la connectivité des qubits de l'appareil.

```python
from qiskit_ibm_runtime import SamplerV2
from qiskit_ibm_runtime.fake_provider import FakeManilaV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

backend = FakeManilaV2()
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```

## Option 3 : Matériel quantique réel {#option-3-real-quantum-hardware}

Nécessite un compte IBM Quantum gratuit. Pour en créer un :

1. Inscris-toi sur [quantum.cloud.ibm.com/registration](https://quantum.cloud.ibm.com/registration) — aucune carte de crédit requise pendant les 30 premiers jours
2. Connecte-toi sur [quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) et sélectionne la région **us-east** (requise pour le forfait Open gratuit)
3. Crée une instance (forfait Open gratuit) sur [Instances](https://quantum.cloud.ibm.com/instances) si tu n'en as pas encore
4. Crée une clé API sur [quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) (ou sur [cloud.ibm.com/iam/apikeys](https://cloud.ibm.com/iam/apikeys))
5. Copie ton **CRN** (Cloud Resource Name) depuis ta page [Instances](https://quantum.cloud.ibm.com/instances)

Ensuite, si tu n'as pas encore enregistré tes identifiants dans cette session Binder, exécute la cellule ci-dessous. Remplace `<your-api-key>` par la clé API de l'étape 4, et `<your-crn>` par le CRN de l'étape 5.

```python
from qiskit_ibm_runtime import QiskitRuntimeService

QiskitRuntimeService.save_account(
    channel="ibm_quantum_platform",
    token="<your-api-key>",
    instance="<your-crn>",
    set_as_default=True,
    overwrite=True,
)
```

**Remarque :** Les tâches sur du matériel réel peuvent prendre un certain temps en fonction des délais d'attente. Si la cellule est toujours en cours d'exécution, tu peux vérifier l'état de ta tâche et consulter les résultats sur [quantum.cloud.ibm.com/workloads](https://quantum.cloud.ibm.com/workloads?user=me).

```python
from qiskit_ibm_runtime import QiskitRuntimeService, SamplerV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

service = QiskitRuntimeService()
backend = service.least_busy(operational=True, simulator=False)
print(f"Running on {backend.name}")

pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```
