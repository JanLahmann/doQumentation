# Überblick {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

Dä Kurs richtet sich aa Lüüt, wo scho chli Erfahrig mit Quantencomputing hei und lehrne wänd, wie me variationale Algorithme mit de Qiskit Runtime Primitives schribt. Er isch au guet geignet für Quantencomputing-Fachleute, wo ihri Kenntniss und Fähigkeite im Beriich vo de variationale Algorithme erwytere wänd.
