# Überblick {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Dää Kurs erforschet, wie Quantencomputing en aufregendi Rolle im Maschinelärne spile cha. De Kurs biet chli Review vo klassische Maschinelärne-Aasätz, setzt aber au e gwüssi Vertrauetheit mit dänne Methode voraus. De Hauptfokus liegt druf, wie Quantencomputing in beschtendi Maschinelärne-Workflows iigfüegt wärde cha. Themä umfassed Datekodierig, Quantekerne und variationelli Quanteschaltkreis.
