# بيئات المختبر الإلكترونية {#online-lab-environments}

إلى جانب تنزيل Qiskit وتثبيته على جهازك المحلي، تتوفر عدة خيارات لمن يفضلون العمل في بيئة تطوير إلكترونية.

يوفر عدد من المزودين بيئات Jupyter إلكترونية مع Qiskit مثبَّتاً مسبقاً لتهيئة سريعة ومريحة. تشمل هؤلاء المزودين الموصى بهم مزود الخدمات السحابية OVHcloud ومنصة تطوير البرمجيات الكمومية السحابية qBraid.

## بيئات دفاتر ملاحظات موصى بها مع Qiskit مثبَّت مسبقاً {#recommended-notebook-environments-with-qiskit-preinstalled}

يمكن استخدام بيئات دفاتر ملاحظات Jupyter الإلكترونية الأخرى مثل IBM Watson&reg; Studio وGoogle Colab وMicrosoft Azure Machine Learning Studio للعمل مع Qiskit باتباع التعليمات المناسبة.

### دفاتر ملاحظات OVHcloud الذكاء الاصطناعي {#ovhcloud-ai-notebooks}

يستطيع مستخدمو OVHcloud Public Cloud وOVHcloud Startup استخدام دفاتر ملاحظات OVHcloud الكمومية للوصول إلى Qiskit وإرسال المهام إلى وحدات المعالجة الكمومية IBM&reg; QPUs باستخدام حسابات IBM Quantum&reg; أو IBM Cloud&reg;. لمعرفة المزيد، زر صفحة [توثيق وبرامج تعليمية OVHcloud](https://help.ovhcloud.com/csm/world-documentation?id=kb_home).

<Admonition type="note">
دفاتر ملاحظات OVHcloud للذكاء الاصطناعي غير متاحة في بعض المناطق، كالولايات المتحدة الأمريكية. كذلك قد تبدو واجهات المستخدم مختلفة من منطقة إلى أخرى.
</Admonition>

اتبع الخطوات التالية للبدء:

1. [أنشئ حساب IBM Cloud.](./cloud-setup)
1. [أعِدَّ حساب OVHcloud.](https://help.ovhcloud.com/csm/en-account-create-ovhcloud-account?id=kb_article_view&sysparm_article=KB0043022)
2. [أنشئ مشروع OVH Public Cloud.](https://support.us.ovhcloud.com/hc/en-us/articles/360002245784-Creating-Your-First-Public-Cloud-Project)
6. اتبع الخطوات الواردة في [إطلاق أول دفتر ملاحظات ذكاء اصطناعي،](https://help.ovhcloud.com/csm/en-public-cloud-ai-notebooks-definition?id=kb_article_view&sysparm_article=KB0048270) مع اختيار **Qiskit** كإطار عمل.

### qBraid Lab {#qbraid-lab}

يمكن لمستخدمي Qiskit استخدام بيئات Python المُهيَّأة مسبقاً من qBraid باتباع هذه التعليمات. لمزيد من المعلومات، راجع [توثيق qBraid.](https://docs.qbraid.com/home/introduction)

1. [أنشئ حساباً مجانياً على qBraid.](https://docs.qbraid.com/home/account)
3. [أنشئ بيئة Lab.](https://docs.qbraid.com/lab/user-guide/getting-started)
3. شغِّل qBraid Lab، انقر على أيقونة **ENVS** على اليمين، ثم انقر **+ ADD** في أعلى الشريط الجانبي الجديد، وابحث عن Qiskit. انقر على الإصدار الذي تريد استخدامه، ثم انقر **Install**.
4. أنشئ دفتر ملاحظات Jupyter بالنقر على **Python 3 [Qiskit]** في قسم Notebook.
4. يمكنك تثبيت أي حزم Python إضافية (مثل Qiskit v1.2) داخل خلية دفتر الملاحظات بتشغيل `%pip install [package_name].` لمزيد من المعلومات، راجع موضوع [أوامر Pip (السحرية)](https://docs.qbraid.com/lab/user-guide/notebooks#pip-magic-commands).

## بيئات دفاتر ملاحظات إضافية لمستخدمي Qiskit {#additional-notebook-environments-for-qiskit-users}

بالنسبة للمنصات التي لا تأتي مع Qiskit مُهيَّأاً مسبقاً، تكون خطوات الإعداد متشابهة، غير أنك ستحتاج إلى تثبيت Qiskit يدوياً. ستُنشئ حساباً، وتبدأ مشروعاً جديداً (أو دفتر ملاحظات أو مساحة عمل، بحسب المنصة)، ثم تثبِّت Qiskit.

### IBM Watson Studio {#ibm-watson-studio}

1. [أعِدَّ نسخة تجريبية مجانية من Watson Studio.](https://cloud.ibm.com/docs/watson?topic=watson-about)
2. [أنشئ مشروع Watson Studio.](https://www.ibm.com/docs/db2-event-store/2.0.0?topic=data-projects#d275537e106)
3. لتثبيت حزم Qiskit والتبعيات اللازمة، افتح دفتر ملاحظات جديداً ونفِّذ الأمر التالي:
   ```python
   !pip install qiskit qiskit-ibm-runtime pylatexenc !yes | pip uninstall simplejson
   ```

### Google Colab {#google-colab}

1. [أنشئ حساب Google.](https://support.google.com/accounts/answer/27441?hl)
2. اذهب إلى [صفحة Google Colab الرئيسية](https://colab.research.google.com/) وسجِّل الدخول بحساب Google.
3. [أنشئ دفتر ملاحظات Colab](https://colab.research.google.com/#create=true) وثبِّت Qiskit بتشغيل الأمر التالي:
   ```python
   %pip install qiskit qiskit-ibm-runtime
   ```

### Microsoft Azure Machine Learning Studio {#microsoft-azure-machine-learning-studio}

1. [أنشئ حساب Azure.](https://learn.microsoft.com/en-us/dotnet/azure/create-azure-account)
2. [أنشئ مساحة عمل.](https://learn.microsoft.com/en-us/azure/machine-learning/concept-workspace?view=azureml-api-2#create-a-workspace)
3. [أنشئ دفتر ملاحظات Jupyter في مساحة العمل.](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2)
4. من داخل دفتر الملاحظات، نفِّذ هذا الأمر لتثبيت Qiskit:
   ```python
    %pip install qiskit qiskit-ibm-runtime
   ```
