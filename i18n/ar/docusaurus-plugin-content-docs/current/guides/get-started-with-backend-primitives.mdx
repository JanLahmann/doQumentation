# البداية مع Backend primitives {#get-started-with-the-backend-primitives}

على خلاف الـ Primitives الخاصة بالمزود، فإن Backend primitives هي تطبيقات عامة يمكن استخدامها مع أي كائن `backend` طالما أنه يُطبِّق واجهة [`BackendV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.providers.BackendV2). بعض المزودين يُطبِّقون الـ Primitives بصورة أصيلة. راجع [صفحة Qiskit Ecosystem](https://www.ibm.com/quantum/ecosystem?tag=provider) لمزيد من التفاصيل.

<span id="estimator"></span>
## البداية مع Estimator backend primitive {#get-started-with-the-estimator-backend-primitive}

يمكن تشغيل Estimator primitive مع أي مزود باستخدام الفئة [`qiskit.primitives.BackendEstimatorV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendEstimatorV2). غير أنها لا تقدم أي تطبيق لتخفيف أخطاء القياس أو البوابات "جاهزًا للاستخدام"، إذ صُمِّمت Backend primitives للتشغيل محليًا على جهاز المستخدم.

### مثال: {#example}

```python
from qiskit.primitives import BackendEstimatorV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
estimator = BackendEstimatorV2(backend)
```

<span id="sampler"></span>
## البداية مع Sampler backend primitive {#get-started-with-the-sampler-backend-primitive}

يمكن تشغيل Sampler primitive مع أي مزود باستخدام [`qiskit.primitives.BackendSamplerV2`](https://docs.quantum.ibm.com/api/qiskit/qiskit.primitives.BackendSamplerV2). غير أنه يتطلب Backend يدعم خيار `memory`.

### مثال: {#example}

```python
from qiskit.primitives import BackendSamplerV2
from <some_qiskit_provider> import QiskitProvider

provider = QiskitProvider()
backend = provider.get_backend('backend_name')
sampler = BackendSamplerV2(backend)
```
