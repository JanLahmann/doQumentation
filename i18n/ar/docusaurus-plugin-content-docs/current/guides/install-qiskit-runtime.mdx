# تثبيت عميل Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime هي الخدمة السحابية لتنفيذ الحسابات الكمية على أجهزة IBM Quantum&reg;. يوضح هذا الدليل كيفية تثبيت `qiskit-ibm-runtime`، وهي حزمة العميل للوصول إلى خدمة Qiskit Runtime.

قبل تثبيت عميل Qiskit Runtime، اتبع التعليمات الواردة في هذه الأدلة:

1. [تثبيت Qiskit](/guides/install-qiskit).
2. [إعداد حساب IBM Cloud الخاص بك](cloud-setup).

شغِّل هذا الأمر لتثبيت Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="ابقَ على اطلاع بأحدث الإصدارات">
تفقَّد دوريًا [ملاحظات إصدار Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes) للاطلاع على الإصدارات الجديدة. نوصي بتحديث متطلباتك لـ `qiskit-ibm-runtime` بشكل متكرر عبر تشغيل `pip install -U qiskit-ibm-runtime`.
</Admonition>
