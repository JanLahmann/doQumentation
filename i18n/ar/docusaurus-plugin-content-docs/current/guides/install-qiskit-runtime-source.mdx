# تثبيت عميل Qiskit Runtime من المصدر {#install-the-qiskit-runtime-client-from-source}

عند تثبيت Qiskit Runtime من المصدر، يمكنك الوصول إلى نسخة التطوير الحالية بدلاً من استخدام النسخة الموجودة في مستودع Python Package Index (PyPI). يمكنك بذلك فحص أحدث نسخة من الكود وتوسيعها بكفاءة أكبر.

## إنشاء بيئة افتراضية جديدة وتفعيلها {#create-and-activate-a-new-virtual-environment}

1. انتقل إلى مجلد مشروعك وأنشئ بيئة بسيطة تحتوي على Python فقط.

        <OperatingSystemTabs>
          <TabItem value="mac" label="macOS">

## تثبيت Qiskit Runtime {#install-qiskit-runtime}

اتبع هذه الخطوات لتثبيت Qiskit Runtime:

1. استنسِخ مستودع Qiskit Runtime.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. انتقل إلى مجلد `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. ثبِّت `qiskit-runtime`. نوصي باستخدام [بيئة افتراضية](https://docs.python.org/3/library/venv.html) لتجنب تلويث تثبيت Python العام.

  * **التثبيت المعياري**:

    ```bash
    pip install .
    ```

  * **وضع التعديل**: في هذا الوضع، لن تحتاج إلى إعادة تثبيت Qiskit عند إجراء تغييرات في الكود.

    ```bash
    pip install -e .
    ```

    في وضع التعديل، تُبنى الامتدادات المُصرَّفة في _وضع التصحيح_ دون تحسينات.

4. (اختياري) إن كنت تريد تشغيل الاختبارات أو فحوصات الجودة، ثبِّت متطلبات المطوِّر. نوصي باستخدام [بيئة افتراضية](https://docs.python.org/3/library/venv.html) لتجنب تلويث تثبيت Python العام.

```bash
pip install -e ".[dev]"
```
