# نظرة عامة على تقنيات إدارة الضوضاء {#overview-of-noise-management-techniques}

عند تنفيذ أحمال العمل الكمية، هناك طرق متعددة لتقليل تأثير الضوضاء. توفر إضافات Qiskit مفتوحة المصدر تقنيات لتخفيف الأخطاء وتثبيطها تتكامل مباشرةً مع سير عمل التطوير، بينما يطبِّق Qiskit Runtime استراتيجيات متقدمة لتخفيف الأخطاء تلقائيًا عند إرسال المهام للتنفيذ. تُفهرس هذه الصفحة جميع الأدوات والميزات المتاحة عبر كلا الخيارين لمساعدتك في اختيار النهج الصحيح لإدارة الضوضاء عند بناء أحمال العمل الكمية.
