# مقدمة إلى Primitives {#introduction-to-primitives}

صُمِّمت الـ Primitives لتبسيط أكثر المهام شيوعًا لأجهزة الكم: وهي أخذ عينات من الحالات الكمية وحساب قيم التوقع. أول Primitives في Qiskit Runtime ([`EstimatorV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2) و[`SamplerV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2)) هي تطبيقات للـ [فئات الأساسية للـ Primitives في Qiskit](/guides/primitives). توفر تطبيقًا أكثر تطورًا (مثلاً بتضمين تخفيف الأخطاء) كخدمة سحابية، وتُستخدم للوصول إلى أجهزة IBM Quantum&reg;.

أحدث Primitive في Qiskit Runtime، وهو [Executor](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/executor) (في مرحلة بيتا)، يوفر واجهة منخفضة المستوى تمنحك مزيدًا من الرؤية والتحكم دون التأثير على الأداء.
