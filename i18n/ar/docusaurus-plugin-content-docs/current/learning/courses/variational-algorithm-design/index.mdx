# نظرة عامة {#overview}

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

هذه الدورة مخصصة لمن لديهم خبرة مسبقة في الحوسبة الكمومية ويرغبون في تعلم كيفية كتابة الخوارزميات التغايرية باستخدام Qiskit Runtime primitives. كما تناسب المتخصصين في الحوسبة الكمومية الراغبين في توسيع معرفتهم ومهاراتهم في مجال الخوارزميات التغايرية.

## الاختبار {#exam}

لتحصل على شارة Variational Algorithm Design، أدِّ الاختبار عبر منصة IBM&reg; Training. هذا الاختبار مخصص للتقدم إليه بعد الانتهاء من قراءة دروس الدورة. بعد اجتيازك للاختبار، ستتلقى إشعارًا من Credly يُخبرك بأنك حصلت على الشارة.
