# نظرة عامة {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

مرحباً بك في *استخدام حاسوب كمومي اليوم*! في هذه الدورة، ستبدأ باستخدام حاسوب كمومي فوراً، وستتعلم المبادئ الأساسية للحوسبة الكمومية في الطريق.

هذه الدورة مصممة للفضوليين والمشغولين على حدٍّ سواء. لست بحاجة إلى درجة علمية في الفيزياء أو سنوات من خبرة البرمجة. إذا كنت تستطيع المتابعة مع مفكرة والنقر على «Run»، فأنت مستعد.
