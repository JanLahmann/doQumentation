# Огляд {#overview}

!["Hero image"](/learning/images/modules/computer-science/hero.avif)

Ця колекція модулів Qiskit для використання в аудиторії призначена для курсу бакалаврського рівня з інформатики, квантових обчислень або вступної квантової механіки. Мета полягає в тому, щоб допомогти викладачам та студентам використовувати квантові комп'ютери для експериментального дослідження фундаментальних квантових протоколів та алгоритмів, таких як квантова телепортація, квантовий розподіл ключів (QKD), а також алгоритми Дойча-Йожі та Бернштейна-Вазірані.

Кожен модуль слід розпочати в класі з підтримкою групи та викладача, але його можна завершити самостійно або в невеликих групах. Усі види діяльності можуть бути виконані за набагато менший час, ніж безкоштовний час, що надається як частина Open Plan, доступного для всіх користувачів. Варіанти симулятора також вбудовані на випадок, якщо студенти випадково вичерпають свій безкоштовний час квантових обчислень.

Викладачі можуть запросити версії цих notebook-ів з ключами відповідей та керівництвом щодо розміщення в загальних навчальних програмах, заповнивши це [коротке опитування](https://ibm.biz/classrooms_instructor_key_request) про те, як використовуються notebook-и.

## Рекомендована підготовка {#recommended-background}

Кожен модуль містить рекомендації щодо того, де в навчальній програмі кожен модуль може бути найбільш ефективним. Загалом, студенти отримають найбільше від цих модулів, якщо вони мають:

- Деяку підготовку з лінійної алгебри (матриць $2 \times 2$ має бути достатньо)
- Деяке знайомство з Python, хоча більшість notebook-ів можна просто запустити як написано
