# Огляд {#overview}

!["Hero image"](/learning/images/courses/fundamentals-of-quantum-algorithms/hero.svg)

Ласкаво просимо до курсу *«Основи квантових алгоритмів»* — другого курсу серії *«Розуміння квантової інформації та обчислень»*, що охоплює такі курси:

- [Основи квантової інформації](/learning/courses/basics-of-quantum-information)
- Основи квантових алгоритмів (цей курс)
- [Загальне формулювання квантової інформації](/learning/courses/general-formulation-of-quantum-information)
- [Основи квантового виправлення помилок](/learning/courses/foundations-of-quantum-error-correction)

Цей курс досліджує обчислювальні переваги квантової інформації: що ми можемо робити з квантовими комп'ютерами і в чому полягають їхні переваги перед класичними.
Він починається з квантових алгоритмів запитів — простих демонстрацій концепцій квантових алгоритмів, — а потім переходить до квантових алгоритмів для таких задач, як факторизація цілих чисел і неструктурований пошук.

Курс призначений для студентів, фахівців та ентузіастів у галузях комп'ютерних наук, фізики, інженерії та математики, які прагнуть отримати знання про теоретичні основи квантової інформації та обчислень.
