# Огляд {#overview}

!["Hero image"](/learning/images/courses/use-a-qc-today/hero.svg)

Ласкаво просимо до *Use a quantum computer today*! У цьому курсі ти одразу почнеш використовувати квантовий комп'ютер і по ходу вивчиш основні принципи квантових обчислень.

Цей курс створений для допитливих і зайнятих. Тобі не потрібен диплом фізика або багаторічний досвід програмування. Якщо ти можеш слідувати за ноутбуком і натискати "Run" — ти готовий.
