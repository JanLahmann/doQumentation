# Огляд {#overview}

!["Hero image"](/learning/images/courses/quantum-machine-learning/hero.svg)

Цей курс досліджує, яку захопливу роль квантові обчислення можуть відігравати у машинному навчанні. Курс включає огляд класичних підходів до машинного навчання, але також передбачає певну знайомість із цими методами. Основний фокус — на тому, як квантові обчислення можуть вписатися в наявні робочі процеси машинного навчання. Теми включають кодування даних, квантові ядра та варіаційні квантові схеми.
