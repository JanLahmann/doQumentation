# Іспит і значок {#exam-and-badge}

Вітаємо із завершенням курсу!

Іспит і значок для цього курсу незабаром будуть доступні. Будь ласка, поверніться через кілька днів.

ПРИМІТКА: IBM використовує послуги Credly, стороннього оператора обробки даних, авторизованого IBM та розташованого в США, для сприяння адмініструванню програми IBM Digital Badge. Щоб видати тобі IBM Digital Badge, твоя особиста інформація (ім'я, електронна адреса та отриманий значок) буде передана Credly. Ти отримаєш сповіщення електронною поштою від Credly з інструкціями щодо отримання значка. Твоя особиста інформація використовується для видачі значка, а також для звітності програми та операційних цілей. IBM може передавати зібрану особисту інформацію дочірнім компаніям IBM та третім сторонам по всьому світу. Вона оброблятиметься відповідно до практик IBM щодо конфіденційності. Заяву про конфіденційність IBM можна переглянути [тут.](https://www.ibm.com/privacy/us/en/)
