# Qiskit Addons

[Qiskit Addons](https://github.com/Qiskit?q=qiskit-addon) — це офіційні пакети розширень, що надають спеціалізовані алгоритми та методи квантових обчислень. Кожен аддон зосереджений на конкретній галузі та містить підручники з виконуваним кодом.

## Доступні Addons {#available-addons}

| Аддон | Пакет | Опис |
|-------|---------|-------------|
| **Circuit Cutting** | `qiskit-addon-cutting` | Зменшення ширини та глибини Circuit шляхом розрізання Gate і проводів |
| **SQD** | `qiskit-addon-sqd` | Квантова діагоналізація на основі вибірок для оцінки основного стану |
| **OBP** | `qiskit-addon-obp` | Зворотне поширення операторів для зменшення глибини Circuit |
| **MPF** | `qiskit-addon-mpf` | Формули мультипродуктів для покращеного моделювання гамільтоніана |
| **AQC-Tensor** | `qiskit-addon-aqc-tensor` | Наближена квантова компіляція з тензорними мережами |
| **PNA** | `qiskit-addon-pna` | Поглинання поширеного шуму для пом'якшення помилок |
| **SLC** | `qiskit-addon-slc` | Затінені світлові конуси для зменшення накладних витрат вибірки PEC |

Усі підручники можна виконати безпосередньо у браузері за допомогою Binder або на власному сервері Jupyter.

Для перегляду документації API відвідай окремі репозиторії аддонів на [GitHub](https://github.com/Qiskit?q=qiskit-addon).
