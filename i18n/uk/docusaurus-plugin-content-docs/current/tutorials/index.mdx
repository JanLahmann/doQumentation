# Посібники {#tutorials}

Використовуйте ці посібники, щоб навчитися застосовувати Qiskit до поширених задач квантових обчислень.

- Почніть із набору посібників у розділі [Початок роботи](#get-started), якщо Ви вперше запускаєте код на квантовому комп'ютері.
- Розділ [Робочі процеси на шляху до переваги](#explore-workflows-toward-advantage) містить наскрізні приклади використання квантового комп'ютера для розв'язання реальних задач. Ці посібники зосереджені на алгоритмах, які є перспективними кандидатами для досягнення обчислювальної переваги квантового комп'ютера над класичним.
- Розділ [Можливості Qiskit](#leverage-qiskit-capabilities) містить приклади, які використовують найновіші та найдосконаліші методи в екосистемі Qiskit для покращення частини або всього робочого процесу.

Ці посібники призначені для початківців, які готові досліджувати виконання квантових алгоритмів на квантовому комп'ютері.

## Дослідження робочих процесів на шляху до переваги {#explore-workflows-toward-advantage}

Посібники в цьому розділі охоплюють масштабні демонстрації квантових алгоритмів.

<details>
  <summary>**Алгоритми верифікованого семплювання**</summary>

Алгоритми цієї категорії зосереджені на квантових схемах, вихідні розподіли яких кодують розв'язки структурованих задач із верифікованим результатом. Верифікованість означає, що Ви можете перевірити узгодженість виміряних даних — або шляхом оцінки обраного бітового рядка, або завдяки тому, що хибнопозитивні результати не виникають.

Ці посібники висвітлюють методи, де повторне семплювання дає змогу оцінювати специфічні для задачі величини (наприклад, значення функції вартості або спектральні ваги). Ці методи є особливо актуальними для задач оптимізації та моделювання із симетріями.

* [Квантова діагоналізація хімічного гамільтоніана на основі семплювання](/tutorials/sample-based-quantum-diagonalization)

* [Крилівська квантова діагоналізація ферміонної ґраткової моделі на основі семплювання](/tutorials/sample-based-krylov-quantum-diagonalization)

* [Квантовий алгоритм наближеної оптимізації](/tutorials/quantum-approximate-optimization-algorithm)

* [Просунуті методи для QAOA](/tutorials/advanced-techniques-for-qaoa)

* [Кодування кореляцій Паулі для зменшення вимог Maxcut](/tutorials/pauli-correlation-encoding-for-qaoa)

</details>

<details>
  <summary>**Оцінка спостережуваних величин**</summary>

Ці посібники зосереджені на оцінці фізично значущих величин, таких як енергія або значення кореляцій, шляхом підготовки квантових станів і вимірювання спостережуваних величин. Методи включають як варіаційні підходи, так і підходи на основі троттеризованих схем, які забезпечують баланс між виразністю схеми та ефективністю глибини схеми. Акцент робиться на робочих процесах, які зменшують вимоги до квантових ресурсів, зберігаючи точність, та уможливлюють практичну оцінку спостережуваних величин у хімічних і фізичних системах.

* [Оцінка енергії основного стану ланцюжка Гайзенберґа за допомогою VQE](/tutorials/spin-chain-vqe)

</details>

<details>
  <summary>**Алгоритми для відмовостійких обчислень**</summary>

У цьому розділі представлені алгоритми з чітко визначеними теоретичними гарантіями, розроблені для виконання на майбутньому квантовому обладнанні з корекцією помилок. Схеми або накладні витрати на семплювання для цих алгоритмів масштабуються таким чином, що не є ефективним за глибиною, і тому з більшою ймовірністю продемонструють квантову перевагу, коли існуватимуть відмовостійкі квантові комп'ютери. Ці посібники ілюструють, як методи працюють в ідеалізованих умовах, і демонструють приклади малого масштабу.

* [Алгоритм Ґровера](/tutorials/grovers-algorithm)

</details>
