Побудуй [стан Белла](https://en.wikipedia.org/wiki/Bell_state) (два заплутаних кубіти) і виконай його трьома способами:

1. **Ідеальна симуляція** — ідеальні результати, не потрібен обліковий запис
2. **Шумна симуляція** — симулює реальний пристрій, не потрібен обліковий запис
3. **Реальне квантове обладнання** — потрібен безкоштовний обліковий запис IBM Quantum (кроки налаштування нижче)
## Побудова схеми {#build-the-circuit}

```python
# Added by doQumentation — required packages for this notebook
```

```python
from qiskit import QuantumCircuit

qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

qc.draw(output="mpl")
```

## Варіант 1: Ідеальна симуляція (не потрібен обліковий запис) {#option-1-ideal-simulation-no-account-needed}

Використовує `StatevectorSampler` — локальний симулятор з ідеальними результатами без шуму.

```python
from qiskit.primitives import StatevectorSampler

result = StatevectorSampler().run([qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
from qiskit.visualization import plot_histogram
plot_histogram(counts)
```

## Варіант 2: Шумна симуляція (не потрібен обліковий запис) {#option-2-noisy-simulation-no-account-needed}

Використовує `FakeManilaV2` — локальний симулятор, що імітує реальний квантовий пристрій IBM, включаючи його характеристики шуму. Спочатку схему необхідно транспілювати (адаптувати) до набору вентилів пристрою та з'єднання кубітів.

```python
from qiskit_ibm_runtime import SamplerV2
from qiskit_ibm_runtime.fake_provider import FakeManilaV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

backend = FakeManilaV2()
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```

## Варіант 3: Реальне квантове обладнання {#option-3-real-quantum-hardware}

Потрібен безкоштовний обліковий запис IBM Quantum. Щоб його налаштувати:

1. Зареєструйся на [quantum.cloud.ibm.com/registration](https://quantum.cloud.ibm.com/registration) — кредитна картка не потрібна протягом перших 30 днів
2. Увійди на [quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) та обери регіон **us-east** (обов'язково для безкоштовного Open Plan)
3. Створи екземпляр (безкоштовний Open Plan) на сторінці [Instances](https://quantum.cloud.ibm.com/instances), якщо у тебе його ще немає
4. Створи API-ключ на [quantum.cloud.ibm.com](https://quantum.cloud.ibm.com) (або на [cloud.ibm.com/iam/apikeys](https://cloud.ibm.com/iam/apikeys))
5. Скопіюй свій **CRN** (Cloud Resource Name) зі сторінки [Instances](https://quantum.cloud.ibm.com/instances)

Якщо ти ще не зберіг свої облікові дані в цій сесії Binder, виконай комірку нижче. Заміни `<your-api-key>` на API-ключ з кроку 4, а `<your-crn>` — на CRN з кроку 5.

```python
from qiskit_ibm_runtime import QiskitRuntimeService

QiskitRuntimeService.save_account(
    channel="ibm_quantum_platform",
    token="<your-api-key>",
    instance="<your-crn>",
    set_as_default=True,
    overwrite=True,
)
```

**Примітка:** Завдання на реальному обладнанні можуть зайняти певний час залежно від черги. Якщо комірка все ще виконується, ти можеш перевірити статус завдання та побачити результати на [quantum.cloud.ibm.com/workloads](https://quantum.cloud.ibm.com/workloads?user=me).

```python
from qiskit_ibm_runtime import QiskitRuntimeService, SamplerV2
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager

service = QiskitRuntimeService()
backend = service.least_busy(operational=True, simulator=False)
print(f"Running on {backend.name}")

pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

result = SamplerV2(mode=backend).run([isa_qc], shots=1024).result()
counts = result[0].data.meas.get_counts()
counts
```

```python
plot_histogram(counts)
```
