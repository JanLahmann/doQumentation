# Огляд доступних планів {#overview-of-available-plans}

IBM Quantum Platform пропонує безкоштовний план Open і платні плани, описані нижче. Повні деталі дивись на сторінці [IBM Quantum access plans](https://www.ibm.com/quantum/pricing).

Якщо тобі потрібен доступ до кількох планів, можна створити кілька екземплярів і призначити їм різні плани. Наприклад, можна мати екземпляр плану Open і екземпляр плану Pay-As-You-Go. Бажані екземпляри можна вказати в ініціалізаторі `QiskitRuntimeService` або методі `QiskitRuntimeService.save_account()`. Приклад:

```python
  from qiskit_ibm_runtime import QiskitRuntimeService
  service = QiskitRuntimeService.save_account(
    token="<your-API_KEY>", # Use the 44-character API_KEY you created and saved from the IBM Quantum Platform Home dashboard.
    # Your token is confidential. Do not share your key in public code.
    instance="<IBM Cloud CRN or instance name>", # Optionally specify the instance to use.
    plans_preference="['open', 'premium']", # Optionally set the types of plans to prioritize.  This is ignored if the instance is specified.
    # Additionally, instances of a certain plan type are excluded if the plan name is not specified.
    region="us-east", # Optionally set the region to prioritize. Accepted values are 'us-east' or 'eu-de'. This is ignored if the instance is specified.
    name="<account-name>", # Optionally name this set of account credentials.
    set_as_default=True, # Optionally set these as your default credentials.
  )
```

## Open Plan {#open-plan}

Запускай квантові схеми на найкращих QPU у світі безкоштовно (до 10 хвилин квантового часу в ковзному вікні 28 днів). Відстежувати використання можна на сторінках [Platform dashboard](https://quantum.cloud.ibm.com/) і [Workloads](https://quantum.cloud.ibm.com/workloads). Open Plan доступний лише в регіоні us-east.

Цей план рекомендовано тим, хто тільки вивчає квантові обчислення та досліджує технології IBM Quantum.

## Flex Plan {#flex-plan}

Попередньо придбай певну кількість хвилин (не менше 400) для запуску квантових задач — їх потрібно використати протягом одного року з моменту покупки. Докуповувати хвилини можна будь-коли.

Після покупки часу необхідно розподілити хвилини між наявними або новими екземплярами плану Flex для свого акаунту. Установлюй або змінюй ліміти розподілу на [сторінці Instances](https://quantum.cloud.ibm.com/instances) або задай розподіл під час [створення нового екземпляра Flex Plan.](/guides/instances) Якщо всі хвилини вже розподілено між наявними екземплярами, новий екземпляр для Flex Plan неможливо створити, допоки не буде придбано більше хвилин або не знято частину хвилин з іншого екземпляра. Докладніше читай у розділі [Встановлення лімітів розподілу екземплярів](allocation-limits).

Якщо тобі потрібно більше 10 000 хвилин, варто розглянути підписку Premium Plan. Якщо купувати 400 хвилин небажано — розглянь Pay-As-You-Go Plan.

[Звернись до IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum), щоб підписатися на Flex Plan.

## Pay-As-You-Go Plan {#pay-as-you-go-plan}

Запускай квантові схеми на найкращих QPU у світі та плати лише за фактично використаний квантовий час.

Цей план рекомендовано тим, хто виконує дослідницькі проєкти з квантової корисності та тестує бізнес-моделі з гнучким доступом. Якщо ти плануєш використовувати щонайменше 400 хвилин протягом наступного року, рекомендуємо замість цього скористатися Flex Plan.

## Premium Plan {#premium-plan}

Запускай квантові схеми на найкращих QPU у світі в рамках корпоративної підписки на квантовий час. Крім того, деякі функції, зокрема [Qiskit Functions](functions) і [Qiskit Transpiler as a Service](qiskit-transpiler-service), доступні користувачам Premium Plan.

Цей план рекомендовано тим, хто хоче реалізовувати стратегічну квантову дорожню карту та розробляти квантові алгоритми й застосунки у великому масштабі. [Звернись до IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum), щоб дізнатися більше.

## On-Prem Plan {#on-prem-plan}

Отримай доступ до повністю виділеної локальної квантової системи, яку обслуговує та підтримує IBM Quantum.

Цей сервіс рекомендовано тим, хто досліджує класично-квантові алгоритми й застосунки та потребує повного контролю над своїми ресурсами й даними. [Звернись до IBM](https://www.ibm.com/account/reg/us-en/signup?formid=MAIL-quantum), щоб дізнатися більше.
