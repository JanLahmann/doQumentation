## Виведені з експлуатації сімейства процесорів {#retired-processor-families}

- **Canary**: Сімейство Canary складалося з невеликих конструкцій, що містили від 5 до 16 кубітів.
- **Falcon**: Сімейство пристроїв Falcon забезпечувало цінну платформу для схем середнього масштабу, а також слугувало платформою для демонстрації вдосконалень продуктивності та масштабованості перед їх переносом на більші пристрої.
- **Egret**: Egret впровадив інновації у вигляді перестроюваних зв'язувачів на 33-кубітній платформі, що забезпечило більш швидкі та точні двокубітні вентилі.
- **Hummingbird**: Використовуючи розташування кубітів у формі важкого шестикутника, сімейство Hummingbird підтримувало до 65 кубітів.
- **Osprey**: Osprey був майже вчетверо більшим за Eagle — 433 кубіти. Більший розмір чіпа вимагав додаткових удосконалень у корпусуванні пристрою, а також спеціальних гнучких кабелів у кріостаті для задоволення більших вимог введення/виведення в межах тієї самої трасування проводки.
- **Eagle**: Сімейство процесорів Eagle використовувало більш масштабовані технології пакування, ніж попередні покоління. Зокрема, сигнали проходили через кілька шарів чіпа, що забезпечувало щільне введення/виведення без втрати продуктивності.

## Отримання завдання з виведеного з експлуатації QPU {#retrieve-a-job-from-a-retired-qpu}

Використовуй наступний код для отримання завдання з виведеного з експлуатації QPU.

<Admonition type="note">
Передача Cloud Resource Name (CRN) у наведеному нижче коді є необов'язковою. Якщо CRN передано, доступні лише бекенди та завдання з цього екземпляра. Якщо екземпляр не вказано, доступні всі бекенди та завдання з усіх екземплярів у твоєму обліковому записі. У цьому випадку, коли вказано бекенд, автоматично знаходиться та використовується екземпляр, який має доступ до цього бекенда.
</Admonition>

```python
from qiskit_ibm_runtime import QiskitRuntimeService

# Load your IBM Quantum account. Replace "<IBM Cloud CRN>" with your desired instance's CRN.
service = QiskitRuntimeService(channel="ibm_quantum_platform", instance="<IBM Cloud CRN>")

# Retrieve a single job by id
job = service.job(<job_id>)

# Retrieve a batch of jobs. Filtering options can be found in the QiskitRuntimeService.jobs api reference
jobs = service.jobs(backend_name=<backend_name>)
```

<Admonition type="note" title="Deprecated provider packages">
  `service.jobs()` returns jobs run from both `qiskit-ibm-runtime` and the deprecated `qiskit-ibm-provider` package. Jobs submitted by the older (also deprecated) `qiskit-ibmq-provider` package are no longer available.
</Admonition>
