# Журналювання для Qiskit Runtime {#logging-for-qiskit-runtime}

Сервіси IBM Cloud&reg;, такі як Qiskit Runtime, генерують журнали платформи, які ти можеш використовувати для розслідування аномальної активності та критичних дій у своєму акаунті, а також для усунення несправностей.

Ти можеш скористатися [IBM Cloud Logs Routing,](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about) — сервісом платформи, — щоб маршрутизувати журнали платформи у своєму акаунті до потрібного місця призначення, налаштувавши тенант, який визначає, куди надсилаються журнали. Докладніше див. у розділі [About Logs Routing.](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about)

Ти можеш використовувати [IBM Cloud Logs](https://cloud.ibm.com/docs/cloud-logs) для візуалізації та сповіщень щодо журналів платформи, що генеруються у твоєму акаунті та маршрутизуються службою IBM Cloud Logs Routing до екземпляра IBM Cloud Logs.

## Розташування, де генеруються журнали платформи {#locations-where-platform-logs-are-generated}

IBM Cloud Container Registry генерує журнали платформи, що відображаються у твоїх екземплярах журналювання. У наведеній нижче таблиці перелічено розташування, де ввімкнено автоматичне збирання журналів сервісу Container Registry.

| Розташування               | Чи доступні журнали сервісів платформи? |
|----------------------------|----------------------------------------|
| Washington, D.C. (us-east) |                  Так                   |
| Frankfurt (eu-de)          |                  Так                   |

IBM Qiskit Runtime генерує журнали сервісів платформи, що відображаються у твоїх екземплярах журналювання, у таких конкретних випадках:

Інформацію про запуск інтерфейсу IBM Cloud Logs дивись у розділі **Launching the UI** документації [IBM Cloud Logs.](https://cloud.ibm.com/docs/cloud-logs?topic=cloud-logs-instance-launch)

## Поля за типом журналу {#fields-by-log-type}

Інформацію про поля, що входять до кожного журналу платформи, дивись у розділі [Fields that are included in platform logs.](https://cloud.ibm.com/docs/logs-router?topic=logs-router-about-platform-logs#about-platform-logs-2)

Нижче наведено поля, що включаються до запису журналу.

| Поле            | Тип      | Опис                                                                                      |
|-----------------|----------|-------------------------------------------------------------------------------------------|
| logSourceCRN    | Required | Визначає акаунт та екземпляр журналу потоку, де публікується запис.                       |
| saveServiceCopy | Required | Визначає, чи зберігає IBM копію запису в операційних цілях.                               |
| message         | Required | Опис згенерованого журналу.                                                               |
