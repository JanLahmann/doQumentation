## Отримати допомогу з IBM Quantum Platform та Qiskit {#get-help-with-ibm-quantum-platform-and-qiskit}

**Premium Plan, On-Prem Plan, Direct Access, Flex Plan, Pay-As-You-Go Plan, спеціальні програми та Open Plan (за винятком пробних облікових записів):**

Усі запити на підтримку потрібно подавати через [IBM Cloud&reg; Support Center](https://cloud.ibm.com/unifiedsupport/cases/form). Залежно від типу твого облікового запису ти можеш відкрити Cloud Case (Topic: Qiskit Runtime) або Technical Issue (Topic: Report-an-Issue::Qiskit Runtime). Переконайся, що в перемикачі облікового запису вибрано відповідний обліковий запис IBM Cloud (твій інституційний хмарний обліковий запис або будь-який непробний обліковий запис).

Щоб переглянути всі відкриті заявки IBM Cloud або відкрити нову заявку, відвідай [IBM Cloud Support Center](https://cloud.ibm.com/unifiedsupport/cases).

Звертайся з будь-яких технічних питань, зокрема щодо проблем із платформою, помилок завдань, інструкцій із Qiskit або ширших питань квантових обчислень. Наша команда включає технічних експертів, готових надати допомогу з усього цього спектру тем.

**Підтримка спільноти:**

Маєш пробний обліковий запис або шукаєш інші варіанти підтримки? [Приєднуйся до спільноти Qiskit у Slack](https://qisk.it/join-slack), щоб отримати допомогу від нашої великої спільноти користувачів та експертів. У Slack ти знайдеш тематичні канали, як-от #qiskit-runtime, #ibm-quantum-platform, #qiskit-transpiler, #qiskit-ecosystem та багато інших.

> **Note:** This survey is provided by IBM Quantum and relates to the original English content. To give feedback on doQumentation's website, translations, or code execution, please [open a GitHub issue](https://github.com/JanLahmann/doQumentation/issues).

Також можна скористатися [Stack Exchange](https://quantumcomputing.stackexchange.com/?tags=qiskit) для отримання підтримки від ширшої спільноти з питань квантових обчислень та програмування.

## Проблеми зі створенням облікового запису або входом? {#having-trouble-creating-an-account-or-logging-in}

**Якщо при створенні облікового запису IBM Cloud&reg; виникає помилка,** спочатку перевір, чи маєш ти доступ до [Support Center](https://cloud.ibm.com/unifiedsupport/cases/form) на IBM Cloud.

- Якщо маєш доступ до сторінки «Create Case», вибери Account (Managing your account and resources) як категорію, а потім Account verification як Subtopic.
- Якщо не маєш доступу, відкрий заявку на сторінці [Create an Account, Login or Billing Request](https://watson.service-now.com/x_ibmwc_open_case_app.do#!/create).

Якщо під час спроби створити обліковий запис виникає помилка, а **твоя країна є в списку [доступності для особистого використання](https://cloud.ibm.com/docs/account?topic=account-account-getting-started#signup-personalaccts),** розглянь такі варіанти:

1. Ти можеш користуватися ресурсами [Документації](/docs) та [Навчання](/learning) без облікового запису.
2. Якщо ти студент або викладач акредитованого навчального закладу, що присвоює наукові ступені, ти можеш створити [обліковий запис SkillsBuild Technology Access](https://github.com/academic-initiative/documentation/blob/main/academic-initiative/how-to/How-to-register-with-the-IBM-Academic-Initiative/readme.md) (дивись [SkillsBuild Technology Access FAQ](https://academic.ibm.com/a2mt/faq#/) для отримання додаткової інформації).
3. [Створи пробний обліковий запис](https://quantum.cloud.ibm.com/registration) для 30 днів безкоштовного доступу (зверни увагу, що після закінчення пробного періоду ти, найімовірніше, втратиш доступ).

**Якщо ти бачиш повідомлення «This account is inactive»,** відвідай [IBM Cloud Docs](https://cloud.ibm.com/docs/account?topic=account-ts_accnt_inactive), щоб дізнатися про причини та способи виправлення, або перейди до [IBM Cloud](https://cloud.ibm.com), щоб перевірити, чи є незавершені дії в твоєму акаунті.

<Admonition type="caution" title="Примітка">
Якщо ти є **власником** облікового запису Pay-As-You-Go або підписки, потрібно створити обліковий запис IBM Cloud і надати платіжні дані для верифікації.

Якщо ти **запрошений приєднатися** до облікового запису, просто прийми запрошення електронною поштою і виконай [зазначені кроки](/guides/cloud-setup-invited), щоб отримати доступ без надання платіжних даних.

**Власники пробних облікових записів** можуть створити екземпляр Open Plan на 30 днів без надання платіжних даних. Після закінчення пробного періоду користувачі повинні надати платіжні дані для верифікації та продовження використання облікового запису. З користувачів *не знімають кошти*, якщо вони продовжують використовувати безкоштовний Open Plan, але можуть перейти на інші платні плани в цей час.

Для отримання додаткової інформації про облікові записи IBM Cloud відвідай сторінку [Запитань і відповідей про облікові записи IBM Cloud](https://cloud.ibm.com/docs/account?topic=account-accountfaqs).
</Admonition>
