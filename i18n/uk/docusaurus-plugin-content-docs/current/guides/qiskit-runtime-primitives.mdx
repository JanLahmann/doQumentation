# Вступ до примітивів {#introduction-to-primitives}

Примітиви були створені для спрощення найпоширеніших завдань квантових комп'ютерів: а саме, вибірки квантових станів та обчислення очікуваних значень. Перші примітиви Qiskit Runtime ([`EstimatorV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/estimator-v2) та [`SamplerV2`](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/sampler-v2)) — це реалізації базових класів [примітивів Qiskit](/guides/primitives). Вони надають більш досконалу реалізацію (наприклад, включаючи пом'якшення помилок) як хмарний сервіс і використовуються для доступу до апаратного забезпечення IBM Quantum&reg;.

Найновіший примітив Qiskit Runtime, [Executor](https://qiskit.github.io/qiskit-ibm-runtime/stubs/qiskit_ibm_runtime.Executor.html) (у бета-версії), надає інтерфейс нижчого рівня, що забезпечує більшу видимість і контроль без шкоди для продуктивності.
