# Відкритий код {#open-source}

З 2016 року IBM Quantum&reg; використовує силу інновацій, що живляться спільнотою, роблячи ресурси квантових обчислень доступними для всіх через хмару.
Ця революційна ініціатива включає відкриття вихідного коду ключових частин стеку квантового програмного забезпечення та представлення Qiskit — універсального фреймворку для квантових обчислень.
Сьогодні Qiskit є найпопулярнішим квантовим фреймворком, і частково завдяки своїй яскравій спільноті.
Спільнота Qiskit — це динамічна та привітна група розробників, науковців і ентузіастів, які співпрацюють задля розвитку програмного забезпечення, досліджень, освіти та бізнесу через квантові обчислення.
Роблячи цю трансформаційну технологію відкритою та доступною, ми прискорюємо інновації та відкриваємо нові застосування.

В міру розвитку квантових технологій внески з відкритих проєктів [екосистеми Qiskit](https://www.ibm.com/quantum/ecosystem) та розробників відіграють вирішальну роль у формуванні її майбутнього.

Приєднавшись до спільноти Qiskit, ти можеш брати участь у розвитку квантових технологій.
Незалежно від того, чи ти досвідчений квантовий програміст, чи тільки робиш перший крок у своїй квантовій подорожі, Qiskit забезпечує підтримку та натхнення, необхідні для розширення меж можливого.
Ми вітаємо будь-які внески — від коду у пул-реквестах до підтримки спільноти у [Slack](https://qisk.it/join-slack), — забезпечуючи кожному можливість зробити свій внесок у розвиток квантових технологій.
