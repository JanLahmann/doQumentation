# Оновлення з плану Open {#upgrade-from-the-open-plan}

Цей посібник призначений для організацій, які хочуть перейти з IBM Quantum&reg; Open Plan на IBM Cloud&reg; Pay-As-You-Go Plan. Щоб налаштувати хмарний акаунт, створити екземпляр сервісу та керувати користувачами, виконай наведені нижче кроки.

Якщо ти вперше налаштовуєш сервіс як окрема особа або якщо адміністратор запросив тебе до екземпляра, дотримуйся кроків у розділі [Встановлення Qiskit](./install-qiskit).

## Перед початком роботи {#before-you-begin}

За потреби виконай кроки з розділу [Налаштування для роботи з IBM Cloud](./cloud-setup#cloud), щоб завершити наступне:

- Зареєструй акаунт IBM Cloud.
- Створи екземпляр IBM Quantum.

<span id="update-cloud-account"></span>
## Оновлення акаунту IBM Cloud {#upgrade-your-ibm-cloud-account}

Пропусти цей крок, якщо у тебе вже є акаунт IBM Cloud Pay-as-you-go. Усі нові акаунти IBM Cloud автоматично налаштовуються як Pay-as-you-go.

Тобі необхідно оновити акаунт IBM Cloud у таких випадках:

- У тебе пробний (Trial) акаунт.
- У тебе акаунт Lite (застарілий), створений до 25 жовтня 2021 року.

Після оновлення ти зможеш продовжувати використовувати будь-які екземпляри, створені з акаунтом Lite. Щоб перейти на акаунт Pay-As-You-Go, виконай такі кроки:

1. Перейди до Manage > [Account](https://cloud.ibm.com/account) у консолі IBM Cloud.
2. Вибери **Account settings** і натисни **Add credit card**.
3. Введи платіжну інформацію та натисни **Next**, щоб надіслати дані.
4. Після обробки платіжної інформації твій акаунт буде оновлено.
   <Admonition type="note">
   З тебе не стягуватиметься плата, якщо ти не створиш екземпляр плану Pay-As-You-Go і не використаєш його для надсилання завдань на QPU.
   </Admonition>

## Доступ до плану Pay-As-You-Go {#access-your-pay-as-you-go-plan}

Щоб використовувати IBM Cloud Pay-As-You-Go Plan, створи новий [екземпляр](/guides/instances) і призначте його до плану Pay-As-You-Go. <ins>За всі завдання, надіслані на QPU через цей екземпляр, буде стягнуто плату.</ins> Щоб використовувати QPU безкоштовно, потрібно використовувати екземпляр, призначений до плану Open або іншого безкоштовного плану (якщо він у тебе є).

Рекомендується налаштувати [ліміт витрат](/guides/manage-cost#cost-limit) на екземплярі та ознайомитися з іншими способами керування витратами при використанні плану Pay-As-You-Go.
