# Встановлення клієнта Qiskit Runtime {#install-the-qiskit-runtime-client}

Qiskit Runtime — це хмарний сервіс для виконання квантових обчислень на апаратному забезпеченні IBM Quantum&reg;. Цей посібник показує, як встановити `qiskit-ibm-runtime` — клієнтський пакет для доступу до сервісу Qiskit Runtime.

Перш ніж встановлювати клієнт Qiskit Runtime, виконай інструкції з таких посібників:

1. [Встанови Qiskit](/guides/install-qiskit).
2. [Налаштуй свій акаунт IBM Cloud](cloud-setup).

Запусти цю команду для встановлення Qiskit Runtime:

```shell
pip install qiskit-ibm-runtime
```

<Admonition type="note" title="Залишайся в курсі останніх версій">
Регулярно перевіряй [примітки до випусків Qiskit Runtime](https://docs.quantum.ibm.com/api/qiskit-ibm-runtime/release-notes), щоб бачити нові релізи. Ми рекомендуємо часто оновлювати вимоги для `qiskit-ibm-runtime`, запускаючи `pip install -U qiskit-ibm-runtime`.
</Admonition>
