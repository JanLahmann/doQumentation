# Посібник з міграції на Qiskit 1.0 {#qiskit-10-migration-guide}

Qiskit 1.0 приносить нові гарантії стабільності та покращення продуктивності, але
порушує сумісність з усіма попередніми версіями Qiskit. Переглянь
наступні посібники з оновлення свого середовища та коду для роботи з Qiskit v1.0.

- [Встановлення нового пакету Qiskit v1.0](/guides/qiskit-1.0-installation) — для оновлення Python-середовищ і залежностей проєкту.
- [Зміни у функціях Qiskit v1.0](/guides/qiskit-1.0-features) — для оновлення коду.
- [Зміни пакетів, що порушують сумісність у Qiskit v1.0](/guides/metapackage-migration) — для розуміння нової та старої структури пакетів.

Докладніше про переваги Qiskit v1.0 читай у [блозі IBM&reg;](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
