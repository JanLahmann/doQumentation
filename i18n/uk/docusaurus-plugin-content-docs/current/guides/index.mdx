# Вступ {#introduction}

## Вступ до Qiskit та IBM Quantum {#introduction-to-qiskit-and-ibm-quantum}

Ласкаво просимо до документації Qiskit, його пов'язаних пакетів та IBM Quantum&reg; Platform. Ця документація включає практичні посібники для початку роботи з нашими інструментами, навчальні матеріали для конкретних випадків використання з наскрізними прикладами та колекцію довідників API.

Qiskit надає модульну та розширювану платформу для квантових досліджень і розробки в галузі алгоритмів, високопродуктивних обчислень та квантової інформатики. За його допомогою дослідники можуть створювати, оптимізувати та виконувати квантові робочі процеси за допомогою спеціалізованих доповнень, програмних інструментів та великих ресурсів. Через IBM Quantum Platform користувачі можуть отримати доступ до сервісів квантових обчислень, таких як Qiskit Runtime та каталог Qiskit Functions, для ефективного виконання робочих навантажень на флоті квантових комп'ютерів IBM&reg;.

![Графік, що показує різні компоненти IBM Quantum Platform та Qiskit.](/docs/images/guides/index/overview-image.svg)

Окрім Qiskit та його пов'язаних пакетів існує [екосистема Qiskit](https://www.ibm.com/quantum/ecosystem), каталог проєктів з відкритим вихідним кодом, які взаємодіють з Qiskit для розширення його функціональності.
