# Доповнення Qiskit {#qiskit-addons}

Доповнення Qiskit — це набір дослідницьких можливостей для розробки алгоритмів у промисловому масштабі. Ці можливості базуються на потужній основі інструментів Qiskit для створення та запуску квантових алгоритмів. Вони надаються як модульні програмні компоненти, які можна підключити до [робочого процесу](/guides/intro-to-patterns) для масштабування або розробки нових квантових алгоритмів.

## Доповнення для відображення {#addons-for-mapping}

### Наближена квантова компіляція з тензорними мережами {#approximate-quantum-compilation-with-tensor-networks}

Наближена квантова компіляція з тензорними мережами (AQC-Tensor) дає змогу будувати високоточні схеми зі зменшеною глибиною.

- Відвідай [репозиторій GitHub.](https://github.com/Qiskit/qiskit-addon-aqc-tensor)
- Читай [документацію.](/guides/qiskit-addons-aqc)
- Читай туторіал із використання AQC для [покращення часової еволюції за Троттером.](/tutorials/approximate-quantum-compilation-for-time-evolution)

### Формули мультипродукту {#multi-product-formulas}

Формули мультипродукту (MPF) зменшують похибку Троттера в динаміці Гамільтоніана шляхом зваженого комбінування кількох виконань схем.

- Відвідай [репозиторій GitHub.](https://github.com/Qiskit/qiskit-addon-mpf)
- Читай [документацію.](https://qiskit.github.io/qiskit-addon-mpf)

### Оптимізаційний маппер {#optimization-mapper}

Доповнення Optimization Mapper містить функціональність для моделювання задач оптимізації: формулювання їх у абстрактних моделях і перетворення в представлення, зрозумілі квантовому комп'ютеру.

- Читай [документацію](https://qiskit.github.io/qiskit-addon-opt-mapper/) з інструкціями та довідником API.
- Відвідай [репозиторій GitHub.](https://github.com/qiskit/qiskit-addon-opt-mapper)

## Доповнення для оптимізації {#addons-for-optimizing}

### Зворотне поширення операторів {#operator-backpropagation}

Зворотне поширення операторів (OBP) зменшує глибину схеми, обрізаючи операції з кінця, ціною більшої кількості вимірювань операторів.

- Відвідай [репозиторій GitHub.](https://github.com/Qiskit/qiskit-addon-obp)
- Читай [документацію.](/guides/qiskit-addons-obp)
- Читай туторіал із використання [OBP для покращення очікуваних значень.](/tutorials/operator-back-propagation)

### Розрізання схем {#circuit-cutting}

Розрізання схем зменшує глибину транспільованих схем, розкладаючи заплутувальні гейти між несуміжними кубітами.

- Відвідай [репозиторій GitHub.](https://github.com/Qiskit/qiskit-addon-cutting)
- Читай [документацію.](https://qiskit.github.io/qiskit-addon-cutting)

## Доповнення для постобробки {#addons-for-post-processing}

### Квантова діагоналізація на основі вибірок {#sample-based-quantum-diagonalization}

Квантова діагоналізація на основі вибірок (SQD) виконує класичну постобробку зашумлених квантових вибірок, щоб отримати точніші оцінки власних значень Гамільтоніанів квантових систем — наприклад, у задачах хімії.

- Відвідай [репозиторій GitHub.](https://github.com/Qiskit/qiskit-addon-sqd)
- Читай [документацію.](/guides/qiskit-addons-sqd)
- Читай туторіал із [квантової діагоналізації на основі вибірок для хімічного Гамільтоніана.](/tutorials/sample-based-quantum-diagonalization)

### SQD для HPC {#sqd-for-hpc}

Це доповнення є реалізацією SQD, готовою до використання в HPC-середовищах. Написано за стандартами сучасного C++17 і призначено для створення єдиного скомпільованого бінарника для роботи з MPI.

- Відвідай [довідник API](https://qiskit.github.io/qiskit-addon-sqd-hpc/).
- Переглянь [репозиторій GitHub.](https://github.com/qiskit/qiskit-addon-sqd-hpc)

## Доповнення для усунення помилок {#addons-for-error-mitigation}

### M3 (mthree) {#m3-mthree}

Matrix-free Measurement Mitigation (M3) — це пакет для масштабованого усунення помилок вимірювань у квантових обчисленнях, який підтримує паралельні обчислення.

- Відвідай [документацію](https://qiskit.github.io/qiskit-addon-mthree/) з інструкціями, туторіалами та довідником API.
- Відвідай [сторінку GitHub](https://github.com/Qiskit/qiskit-addon-mthree) пакету.

### Затінені світлові конуси {#shaded-lightcones}

Доповнення shaded lightcones використовує поширення Паулі для зменшення кількості членів похибок у моделі шумів, які потрібно усунути, з урахуванням специфіки цільового спостережуваного. Це зменшує накладні витрати на вибірку у робочих процесах імовірнісного скасування помилок (PEC).

- Відвідай [документацію](https://qiskit.github.io/qiskit-addon-slc/) з інструкціями, туторіалами та довідником API.

### Поглинання поширеного шуму {#propagated-noise-absorption}

Поглинання поширеного шуму (PNA) використовує поширення Паулі, щоб включити інформацію з моделі шумів у цільове спостережуване. Вимірювання цього модифікованого спостережуваного фактично усуває шум, представлений моделлю.

- Відвідай [документацію](https://qiskit.github.io/qiskit-addon-pna/apidocs/qiskit_addon_pna.html) з інструкціями, туторіалами та довідником API.
