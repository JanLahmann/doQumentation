## Встановлення Qiskit Runtime {#install-qiskit-runtime}

Виконай ці кроки для встановлення Qiskit Runtime:

1. Клонуй репозиторій Qiskit Runtime.

```bash
git clone https://github.com/Qiskit/qiskit-ibm-runtime.git
```

2. Перейди до каталогу `qiskit-ibm-runtime`.

```bash
cd qiskit-ibm-runtime
```

3. Встанови `qiskit-runtime`. Рекомендуємо використовувати [віртуальне середовище](https://docs.python.org/3/library/venv.html), щоб не забруднювати глобальну інсталяцію Python.

  * **Стандартна установка**:

    ```bash
    pip install .
    ```

  * **Редагований режим**: У цьому режимі тобі не потрібно перевстановлювати Qiskit при змінах коду проєкту.

    ```bash
    pip install -e .
    ```

    У редагованому режимі скомпільовані розширення збираються в _режимі налагодження_ без оптимізацій.

4. (Необов'язково) Якщо хочеш запускати тести або перевірки коду, встанови вимоги для розробника. Рекомендуємо використовувати [віртуальне середовище](https://docs.python.org/3/library/venv.html), щоб не забруднювати глобальну інсталяцію Python.

```bash
pip install -e ".[dev]"
```
