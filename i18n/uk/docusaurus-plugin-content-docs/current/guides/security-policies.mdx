# Політики безпеки репозиторіїв {#repository-security-policies}

Надзвичайно важливо підтримувати свої системи якомога більш захищеними. Тому настійно рекомендується використовувати інструменти безпеки, як-от [Dependabot](https://github.com/dependabot), які автоматично сповіщають тебе, коли залежності застарівають. Ознайомся з політиками безпеки для таких репозиторіїв GitHub, пов'язаних з IBM Quantum&reg;:

- [Qiskit](https://github.com/Qiskit/qiskit/security)
- [Qiskit IBM Runtime](https://github.com/Qiskit/qiskit-ibm-runtime/security)
- [Доповнення Qiskit для розрізання схем](https://github.com/Qiskit/qiskit-addon-cutting/security)
- [Доповнення Qiskit — Fermionic mapper](https://github.com/Qiskit/qiskit-fermions/blob/main/SECURITY.md)
- [Доповнення Qiskit — Multi-product formulas (MPF)](https://github.com/Qiskit/qiskit-addon-mpf/security)
- [Доповнення Qiskit — Operator backpropagation (OBP)](https://github.com/Qiskit/qiskit-addon-obp/security)
- [Доповнення Qiskit — Optimization mapper](https://github.com/Qiskit/qiskit-addon-opt-mapper/blob/main/SECURITY.md)
- [Доповнення Qiskit — Pauli propagation](https://github.com/Qiskit/pauli-prop/blob/main/SECURITY.md)
- [Доповнення Qiskit — Propagated noise absorption (PNA)](https://github.com/Qiskit/qiskit-addon-pna/blob/main/SECURITY.md)
- [Утиліти доповнень Qiskit](https://github.com/Qiskit/qiskit-addon-utils/security)
- [Доповнення Qiskit — Sample-based quantum diagonalization (SQD)](https://github.com/Qiskit/qiskit-addon-sqd/security)
- [Доповнення Qiskit — Shaded lightcones](https://github.com/Qiskit/qiskit-addon-slc/blob/main/SECURITY.md)
