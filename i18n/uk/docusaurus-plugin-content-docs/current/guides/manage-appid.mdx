# Керування користувачами постачальника ідентифікаторів {#manage-id-provider-users}

Ти можеш інтегруватися зі своїм зовнішнім постачальником ідентифікаторів (IDP), щоб безпечно автентифікувати зовнішніх користувачів у своєму обліковому записі IBM Cloud. Використовуючи зовнішнього постачальника, ти можеш надати користувачам своєї компанії можливість використовувати єдиний вхід (SSO). Ти можеш підключити свій хмарний обліковий запис до зовнішнього IDP одним із таких способів:

- Використати [постачальника послуг IBM Cloud&reg; SAML (SP)](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-cloud-saml)
- Використати [App ID](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-appid) (може стягуватися плата)
- [Об'єднати своїх користувачів із усіма продуктами IBM&reg; через IBMid](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#federate-ibmid)

<Admonition type="tip">
Після підключення IDP ти можеш налаштувати динамічні правила, щоб не доводилося вручну додавати кожного користувача до кожної групи при вступі. Дивись [Увімкнення автентифікації від зовнішнього постачальника ідентифікаторів](https://cloud.ibm.com/docs/account?topic=account-ibm-idp-integration#app-id-dynamic-rules) для отримання інструкцій.
</Admonition>
